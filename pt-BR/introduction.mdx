## O que é Dodo Payments?

Dodo Payments é o **motor tudo-em-um** para lançar, escalar e monetizar globalmente. Projetado para a nova geração de produtos digitais SaaS e IA, cuidamos de tudo, desde **pagamentos até faturamento e distribuição** - tudo isso sem engenharia extra.

<Info>
**Construído para Produtos SaaS e de IA**: Gerencie pagamentos únicos, assinaturas, créditos, assentos, tokens ou qualquer métrica de uso com nossa infraestrutura otimizada de faturamento.
</Info>

<Tip>
  <strong>Conheça Sentra: Seu assistente de faturamento movido a IA</strong><br/>
  Sentra é um agente de IA que integra, analisa e age sobre faturamento e pagamentos diretamente no seu IDE. Faça perguntas, construa integrações ou planeje arquiteturas de cobrança — tudo descrevendo o que você deseja.

### Servidores MCP para Assistentes de IA

Conecte seu assistente de codificação de IA (Cursor, Claude Desktop, Windsurf, VS Code) diretamente aos pagamentos Dodo com nossos servidores MCP.

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  Busca semântica na documentação da Dodo Payments — obtenha respostas instantâneas sobre recursos, APIs e melhores práticas
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  Execute operações de API diretamente — crie pagamentos, gerencie assinaturas e trate reembolsos por meio do seu assistente de IA
</Card>
</CardGroup>

{/* LOCKED_PATTERN_8ee343b87e057252cc4a8a376b190970 */}
Adicione ambos os servidores à sua configuração MCP (`~/.cursor/mcp.json` ou configuração Claude Desktop):

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
Requer Node.js 18+. Veja a [documentação do servidor MCP](/developer-resources/mcp-server) para instruções detalhadas de configuração para todos os clientes suportados.
</Info>
</Accordion>
