## Visão Geral

As Habilidades do Agente são capacidades reutilizáveis que ampliam o que os assistentes de codificação de IA conseguem fazer ao trabalhar com a Dodo Payments. Elas fornecem conhecimento procedural e melhores práticas que ajudam os agentes a realizar tarefas específicas de forma mais eficaz — desde integrar fluxos de checkout até configurar cobrança baseada em uso.

<Info>
**Pense nas habilidades como plugins para o seu assistente de IA.** Elas ensinam ao seu agente como implementar corretamente os recursos da Dodo Payments, seguindo nossos padrões recomendados e melhores práticas.
</Info>

As habilidades funcionam com qualquer agente de IA compatível com MCP, incluindo:
- **Claude Code** - Anthropic's CLI coding assistant
- **OpenCode** - Open source AI coding agent
- **Cursor** - AI-first code editor
- **Other MCP clients** - Any agent supporting the skills protocol

## Instalação

Escolha seu método de instalação preferido com base no seu assistente de codificação de IA.

<Tabs>
<Tab title="Skills CLI">
Instale as habilidades usando o CLI universal de habilidades que funciona com qualquer agente compatível com MCP:

```bash
# Install all Dodo Payments skills
npx skills add dodopayments/skills
```

Ou instale habilidades individuais conforme necessário:

```bash
# Best practices guide
npx skills add dodopayments/skills/dodo-best-practices

# Webhook integration
npx skills add dodopayments/skills/webhook-integration

# Subscription billing
npx skills add dodopayments/skills/subscription-integration

# Checkout flows
npx skills add dodopayments/skills/checkout-integration

# Usage-based billing
npx skills add dodopayments/skills/usage-based-billing

# BillingSDK components
npx skills add dodopayments/skills/billing-sdk

# License key management
npx skills add dodopayments/skills/license-keys
```

<Tip>
Instale apenas as habilidades que você precisa para manter seu agente focado e reduzir a sobrecarga de contexto.
</Tip>
</Tab>

<Tab title="Claude Code">
Adicione o marketplace de habilidades da Dodo Payments ao Claude Code:

```bash
/plugin marketplace add dodopayments/skills
```

Em seguida, instale plugins individuais:

```bash
# Install best practices
/plugin install dodo-best-practices

# Install webhook integration
/plugin install webhook-integration

# Install subscription integration
/plugin install subscription-integration

# Install checkout integration
/plugin install checkout-integration

# Install usage-based billing
/plugin install usage-based-billing

# Install BillingSDK
/plugin install billing-sdk

# Install license keys
/plugin install license-keys
```

</Tab>

<Tab title="OpenCode">
As habilidades ficam automaticamente disponíveis quando configuradas nas suas configurações do OpenCode. Adicione o repositório de habilidades da Dodo Payments à sua configuração:

```json
{
  "skills": {
    "repositories": [
      "dodopayments/skills"
    ]
  }
}
```

</Tab>
</Tabs>

## Usando habilidades

Uma vez instaladas, seu assistente de IA pode aproveitar essas habilidades quando você pedir para implementar recursos da Dodo Payments. Aqui estão alguns prompts de exemplo:

### Melhores práticas

```
Help me integrate Dodo Payments into my Next.js app following best practices
```

### Webhooks

```
Set up webhook handlers for payment and subscription events
```

### Assinaturas

```
Implement a subscription flow with free trial and multiple pricing tiers
```

### Checkout

```
Create a checkout session for my SaaS product
```

### Cobrança baseada em uso

```
Add usage-based billing to track API calls for my AI product
```

### BillingSDK

```
Add a pricing page using BillingSDK components
```

### Chaves de licença

```
Implement license key validation for my desktop app
```

## Como as habilidades funcionam

As habilidades fornecem ao seu assistente de IA:

1. **Conhecimento procedural** - Instruções passo a passo para implementar recursos
2. **Modelos de código** - Padrões de código prontos para produção e trechos
3. **Melhores práticas** - Abordagens recomendadas para segurança, tratamento de erros e UX
4. **Consciência de contexto** - Entendimento de como diferentes recursos da Dodo Payments funcionam juntos

Quando você pede ao seu assistente de IA para implementar um recurso, ele usa a habilidade relevante para:
- Entender a abordagem correta de implementação
- Gerar código preciso e funcional
- Seguir as convenções da Dodo Payments
- Lidar com casos extremos e erros de forma adequada

<Note>
As habilidades complementam o [MCP Server](/developer-resources/mcp-server) e o [Sentra](/developer-resources/sentra). Use o MCP para operações de API, o Sentra para integração guiada e as Habilidades para capacidades aprimoradas do assistente de IA.
</Note>

## Comparação com outras ferramentas

| Ferramenta | Propósito | Ideal para |
|------|---------|----------|
| **Agent Skills** | Aprimorar o conhecimento do assistente de IA | Desenvolvimento assistido por IA com qualquer cliente MCP |
| **MCP Server** | Executar operações de API | Acesso direto à API por assistentes de IA |
| **Sentra** | Assistente de integração guiada | Integração interativa de cobrança no IDE |
