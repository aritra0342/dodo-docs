## Introdução

O plugin Dodo Payments para WooCommerce fornece uma integração perfeita com sua loja, permitindo que você aceite pagamentos através de múltiplos métodos, incluindo UPI, cartões, internet banking e carteiras digitais. Este guia abrangente irá orientá-lo pelo processo de instalação e configuração.

## Guia de Instalação Passo a Passo

1. Baixe o arquivo zip mais recente do plugin de pagamento WooCommerce.

    <Card title="Baixar Plugin" icon="download" href="https://github.com/dodopayments/dodo-woocommerce/releases">
    Obtenha a versão mais recente do nosso plugin WooCommerce
    </Card>

    Alternativamente, você pode instalar diretamente do diretório de plugins do WordPress:
    
    <Card title="Diretório de Plugins do WordPress" icon="wordpress" href="https://wordpress.org/plugins/dodo-payments-for-woocommerce/">
    Instale Dodo Payments para WooCommerce do WordPress.org
    </Card>

2. Faça login no seu site WordPress como administrador (assumimos que você já tem o WooCommerce configurado)
3. Navegue até **Plugins → Adicionar Novo Plugin**
    
    <Frame>
        <img src="/images/woocommerce/1.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
4. Clique no botão **Enviar Plugin** e um diálogo de seleção de arquivo aparecerá

    <Frame>
        <img src="/images/woocommerce/2.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/3.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
5. Clique em **Procurar…** e selecione o arquivo zip que você baixou
6. Clique no botão **Instalar Agora** para iniciar o processo de instalação
    
    <Frame>
        <img src="/images/woocommerce/4.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
7. Após a conclusão da instalação, clique em **Ativar Plugin** ou ative-o na seção **Plugins Instalados**
    
    <Frame>
        <img src="/images/woocommerce/5.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
8. O plugin está agora instalado, mas a configuração ainda é necessária. Continue para o guia de configuração abaixo.

## Guia de Configuração

### Configurando Chaves de API e Webhooks

1. Navegue até **WooCommerce → Configurações → Pagamentos** ou clique no botão **Pagamentos** na barra lateral esquerda abaixo do item de menu WooCommerce
    
    <Frame>
        <img src="/images/woocommerce/6.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/7.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
2. Ative o provedor de pagamento **Dodo Payments** se ainda não estiver ativado, em seguida, clique em **Gerenciar** para configurar o plugin
3. Você verá várias opções de configuração, cada uma com texto explicativo útil. Comece configurando sua Chave de API ao Vivo
    
    <Frame>
        <img src="/images/woocommerce/8.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
4. Faça login no seu painel Dodo Payments em **Modo Ao Vivo**, em seguida, navegue até **Dodo Payments (Modo Ao Vivo) > Desenvolvedor > Chaves de API** ou visite [este link direto](https://app.dodopayments.com/developer/api-keys) e clique em **Adicionar Chave de API**
    
    <Frame>
        <img src="/images/woocommerce/9.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
5. Dê um nome descritivo à sua Chave de API e clique em **Criar**
    
    <Frame>
        <img src="/images/woocommerce/10.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
6. Copie a Chave de API gerada e cole-a no campo **Chave de API ao Vivo** nas configurações do seu plugin WooCommerce
    
    <Frame>
        <img src="/images/woocommerce/11.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/12.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
7. Em seguida, configure a Chave de Assinatura do Webhook para habilitar a sincronização do status de pagamento entre Dodo Payments e WooCommerce
8. Role até o final da página de configurações do plugin e copie a URL do webhook exibida lá
    
    <Frame>
        <img src="/images/woocommerce/13.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
9. Retorne ao seu painel Dodo Payments e navegue até **Dodo Payments (Modo Ao Vivo) > Desenvolvedor > Webhooks** e clique em **Adicionar Webhook**
    
    <Frame>
        <img src="/images/woocommerce/14.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
10. Cole a URL que você copiou no passo 8 no diálogo e clique em **Adicionar Webhook**
    
    <Frame>
        <img src="/images/woocommerce/15.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
11. Após criar o webhook, clique no ícone de olho ao lado da **Chave de Assinatura** redigida para revelá-la e copiá-la
    
    <Frame>
        <img src="/images/woocommerce/16.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
12. Cole a Chave de Assinatura no campo **Chave de Assinatura do Webhook ao Vivo** nas configurações do seu plugin e salve as alterações
    
    <Frame>
        <img src="/images/woocommerce/17.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>
    
13. Revise as configurações restantes, como **Categoria Fiscal Global** e **Todos os Preços são Inclusivos de Imposto**, pois essas opções determinam como os produtos são sincronizados do WooCommerce para Dodo Payments. Observe que **Chave de API de Teste** e **Chave de Assinatura do Webhook de Teste** são necessárias apenas se você planeja usar o Modo de Teste
    
    <Frame>
        <img src="/images/woocommerce/18.webp" alt="Plugin Dodo Payments para WooCommerce" />
    </Frame>

## Principais Recursos

- **Experiência de Checkout Sem Costura**: Redirecione os clientes para uma página de checkout segura e otimizada
- **Múltiplos Métodos de Pagamento**: Suporte para várias opções de pagamento locais e internacionais
- **Atualizações de Status em Tempo Real**: Sincronização instantânea do status do pedido via webhooks
- **Registros de Transação Detalhados**: Histórico abrangente de pagamentos no seu painel WooCommerce
- **Suporte a Múltiplas Moedas**: Aceite pagamentos em várias moedas

## Solução de Problemas

### Problemas Comuns

1. **Falhas no Processamento de Pagamentos**
   - Verifique se as chaves de API estão configuradas corretamente
   - Certifique-se de que a URL do webhook está configurada corretamente
   - Verifique se seu certificado SSL é válido

2. **Problemas de Sincronização de Webhook**
   - Confirme se a chave de assinatura do webhook corresponde
   - Revise os logs do servidor para possíveis erros
   - Verifique se seu servidor pode receber solicitações de webhook de entrada

## Melhores Práticas de Manutenção

1. Atualize regularmente o plugin para correções de segurança e novos recursos
2. Teste todos os fluxos de pagamento no modo sandbox antes de processar transações ao vivo
3. Mantenha o WordPress e o WooCommerce atualizados para as versões mais recentes
4. Implemente backups regulares do site
5. Monitore os logs de transação para qualquer atividade incomum
6. Armazene as chaves de API com segurança e nunca as compartilhe publicamente
