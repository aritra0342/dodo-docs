## Introdução

O plugin Dodo Payments para WooCommerce fornece uma integração perfeita com sua loja, permitindo que você aceite pagamentos através de múltiplos métodos, incluindo UPI, cartões, internet banking e carteiras digitais. Este guia abrangente irá orientá-lo pelo processo de instalação e configuração.

## Guia de Configuração

### Configurando Chaves de API e Webhooks

1. Vá até **WooCommerce → Configurações → Pagamentos** ou clique no botão **Pagamentos** na barra lateral esquerda, abaixo do item do menu WooCommerce
    
    <Frame>
        <img src="/images/woocommerce/6.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/7.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
2. Ative o provedor de pagamento **Dodo Payments**, se ainda não estiver ativado, e clique em **Gerenciar** para configurar o plugin
3. Você verá várias opções de configuração, cada uma com textos explicativos úteis. Comece configurando sua Chave de API em produção
    
    <Frame>
        <img src="/images/woocommerce/8.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
4. Faça login no painel da Dodo Payments em **Modo Ao Vivo**, depois vá para **Dodo Payments (Modo Ao Vivo) > Desenvolvedor > Chaves de API** ou visite [este link direto](https://app.dodopayments.com/developer/api-keys) e clique em **Adicionar chave de API**
    
    <Frame>
        <img src="/images/woocommerce/9.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
5. Dê à sua chave de API um nome descritivo e clique em **Criar**
    
    <Frame>
        <img src="/images/woocommerce/10.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
6. Copie a Chave de API gerada e cole-a no campo **Chave de API Ao Vivo** nas configurações do plugin do WooCommerce
    
    <Frame>
        <img src="/images/woocommerce/11.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/12.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
7. Em seguida, configure a Chave de Assinatura do Webhook para habilitar a sincronização do status de pagamento entre a Dodo Payments e o WooCommerce
8. Role até o final da página de configurações do plugin e copie a URL do webhook exibida lá
    
    <Frame>
        <img src="/images/woocommerce/13.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
9. Retorne ao painel da Dodo Payments e vá para **Dodo Payments (Modo Ao Vivo) > Desenvolvedor > Webhooks** e clique em **Adicionar webhook**
    
    <Frame>
        <img src="/images/woocommerce/14.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
10. Cole a URL copiada no passo 8 na caixa de diálogo e clique em **Adicionar webhook**
    
    <Frame>
        <img src="/images/woocommerce/15.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
11. Após criar o webhook, clique no ícone de olho ao lado da **Chave de Assinatura** redigida para revelá-la e copiá-la
    
    <Frame>
        <img src="/images/woocommerce/16.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
12. Cole a Chave de Assinatura no campo **Chave de Assinatura de Webhook Ao Vivo** nas configurações do plugin e salve as alterações
    
    <Frame>
        <img src="/images/woocommerce/17.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
13. Revise as configurações restantes, como **Categoria Fiscal Global** e **Todos os preços incluem impostos**, pois essas opções determinam como os produtos são sincronizados do WooCommerce para a Dodo Payments. Observe que **Chave de API de Teste** e **Chave de Assinatura de Webhook de Teste** são necessárias apenas se você planeja usar o Modo de Teste

## Principais Recursos

- **Experiência de Checkout Sem Costura**: Redirecione os clientes para uma página de checkout segura e otimizada
- **Múltiplos Métodos de Pagamento**: Suporte para várias opções de pagamento locais e internacionais
- **Atualizações de Status em Tempo Real**: Sincronização instantânea do status do pedido via webhooks
- **Registros de Transação Detalhados**: Histórico abrangente de pagamentos no seu painel WooCommerce
- **Suporte a Múltiplas Moedas**: Aceite pagamentos em várias moedas

## Solução de Problemas

### Problemas Comuns

1. **Falhas no Processamento de Pagamentos**
   - Verifique se as chaves de API estão configuradas corretamente
   - Certifique-se de que a URL do webhook está configurada corretamente
   - Verifique se seu certificado SSL é válido

2. **Problemas de Sincronização de Webhook**
   - Confirme se a chave de assinatura do webhook corresponde
   - Revise os logs do servidor para possíveis erros
   - Verifique se seu servidor pode receber solicitações de webhook de entrada

## Melhores Práticas de Manutenção

1. Atualize regularmente o plugin para correções de segurança e novos recursos
2. Teste todos os fluxos de pagamento no modo sandbox antes de processar transações ao vivo
3. Mantenha o WordPress e o WooCommerce atualizados para as versões mais recentes
4. Implemente backups regulares do site
5. Monitore os logs de transação para qualquer atividade incomum
6. Armazene as chaves de API com segurança e nunca as compartilhe publicamente
