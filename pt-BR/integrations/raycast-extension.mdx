Seu painel Dodo Payments, a poucos toques de distância. Use o Raycast para visualizar rapidamente pagamentos, assinaturas, clientes, produtos, códigos de desconto, chaves de licença e reembolsos sem abrir o painel.

## Instalação

<Card title="Instalar na Raycast Store →" icon="puzzle-piece" href="https://www.raycast.com/dodopayments/dodo-payments">
  Abra a página da extensão Dodo Payments na Raycast Store.
</Card>

<Info>
Requer Raycast no macOS e uma chave de API do Dodo Payments.
</Info>

## Configuração

<Steps>
<Step title="Obtenha sua chave de API">
1. Faça login no seu painel Dodo Payments.
2. Vá para Configurações › Chaves de API.
3. Copie sua chave de API.
</Step>

<Step title="Configure a Extensão">
1. Abra o Raycast.
2. Pesquise por "Dodo Payments" e selecione qualquer comando.
3. Quando solicitado, insira sua chave de API no campo Chave de API.
4. Escolha o modo da API:
   - Teste: use para desenvolvimento e testes.
   - Produção: use para dados de produção.
5. Salve as configurações.
</Step>
</Steps>

<Tip>
Comece no modo Teste para validar sua configuração, depois mude para Produção.
</Tip>

<Warning>
Trate sua chave de API como uma senha. Não a compartilhe ou armazene em texto simples.
</Warning>

## Use a extensão

- **Pesquisar**: abra o Raycast, digite "Dodo Payments" e escolha um comando.
- **Filtrar**: refine os resultados digitando partes de IDs, e-mails, nomes ou status.
- **Abrir detalhes**: pressione Enter em um item da lista para ver mais informações.

## Solução de Problemas

<AccordionGroup>
<Accordion title="Falha na autenticação ou erros 401/403">
Confirme se a chave da API e o modo (Teste vs Produção) correspondem aos dados do seu painel. Reabra o comando e reinsira a chave, se necessário.
</Accordion>
<Accordion title="Nenhum resultado aparece">
Certifique-se de que o modo selecionado tenha dados. Tente alternar entre Teste e Produção.
</Accordion>
<Accordion title="Limites de taxa ou timeouts">
Aguarde um momento e tente novamente. Se o problema persistir, verifique sua conectividade de rede.
</Accordion>
