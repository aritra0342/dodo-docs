## Visão Geral

O Plugin Framer do Dodo Payments integra perfeitamente o processamento de pagamentos em seus sites Framer. Crie experiências de checkout personalizáveis com suporte para ambientes de teste e produção.

<Card title="Plugin Oficial" icon="link" href="https://www.framer.com/marketplace/plugins/dodo-payments">
Acesse o plugin oficial do Dodo Payments no marketplace do Framer.
</Card>

## Guia de Configuração

<Steps>
<Step title="Gerar Chave de API">
Visite o [Painel do Dodo Payments](https://app.dodopayments.com/developer/api-keys) e crie uma nova chave de API.

<Tabs>

<Tab title="Modo Ao Vivo">
Para uso em produção com processamento de pagamento real.
</Tab>

<Tab title="Modo de Teste">
Perfeito para desenvolvimento e testes sem cobranças reais.
</Tab>

</Tabs>

<Check>
Copie sua chave de API com segurança - você precisará dela para o próximo passo.
</Check>
</Step>

<Step title="Autenticar Plugin">
1. Abra o plugin Dodo Payments no Framer
2. Navegue até a página de Autenticação
3. Cole sua chave de API
4. Selecione o modo apropriado (Teste ou Ao Vivo)
5. Clique em **Continuar** para autenticar

<Warning>
Mantenha suas chaves de API seguras e nunca as compartilhe publicamente.
</Warning>
</Step>

<Step title="Gerenciar Produtos">
Após a autenticação, você verá sua lista de produtos:

- **Produtos existentes**: Exibidos automaticamente
- **Sem produtos**: Clique no botão **Mais (+)** ao lado de 'Produtos'

<Info>
Você será redirecionado para o Painel do Dodo Payments para criar produtos com preços, descrições e configurações.
</Info>
</Step>

<Step title="Adicionar Botões de Compra">
1. Selecione um produto da sua lista disponível
2. Clique no botão `Add Buy Button`
3. Um botão de compra com o texto `Comprar {product_name}` será adicionado ao seu canvas

<Warning>
O botão Adicionar Botão de Compra será desativado se você não tiver as permissões corretas do projeto.
</Warning>
</Step>
</Steps>

## Personalização

<Steps>
<Step title="Selecionar Experiência de Checkout">
    Escolha sua experiência de checkout preferida:
    - **Overlay (Embutido)**: O checkout aparece como um overlay dentro do seu projeto Framer, mantendo os usuários em seu site.
    - **Hospedado**: Os usuários são redirecionados para uma página de checkout hospedada pelo Dodo Payments para uma experiência simplificada.
</Step>

<Step title="Personalizar Opções de Aparência">
    Personalize sua experiência de checkout com as seguintes configurações de aparência:
    - **Tema do Checkout**: Escolha entre temas claros ou escuros para combinar com a estética da sua marca.
    - **Rótulo do Botão**: Personalize o texto (por exemplo, 'Comprar Agora', 'Adquirir', 'Começar').
    - **Estilos do Botão**: Ajuste cores, fontes, tamanhos e outras propriedades de estilo.
    - **URL de Retorno**: Defina uma página do Framer para a qual o usuário retornará após o checkout.
</Step>
</Steps>

## Solução de Problemas

<AccordionGroup>
<Accordion title="Problemas de Autenticação">
**Problema**: O plugin não autentica

**Soluções**:
- Verifique se sua chave de API está correta
- Certifique-se de que está usando o modo correto (Teste/Ao Vivo)
- Verifique sua conexão com a internet
</Accordion>

<Accordion title="Problemas na Criação de Botões">
**Problema**: O botão Adicionar Botão de Compra está desativado

**Soluções**:
- Verifique se você tem as permissões corretas em seu projeto Framer
- Certifique-se de que está autenticado corretamente
- Tente atualizar o plugin
</Accordion>

<Accordion title="Problemas na Exibição de Produtos">
**Problema**: Produtos não estão aparecendo

**Soluções**:
- Certifique-se de que você criou produtos em seu painel do Dodo Payments
- Verifique se sua chave de API tem as permissões corretas
- Tente re-autenticar
</Accordion>
</AccordionGroup>
