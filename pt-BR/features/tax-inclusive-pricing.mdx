## O Que É Preço com Impostos Inclusos?

O Preço com Impostos Inclusos permite que você incorpore os impostos aplicáveis diretamente no preço do produto exibido. Os clientes veem um valor final único de forma antecipada, proporcionando uma experiência mais transparente e amigável para conversões. No checkout, o Dodo calcula a parte do preço sem impostos e o valor do imposto com base na localização do cliente, e os detalha claramente na fatura.

<Info>
O imposto final ainda varia de acordo com a localização do cliente e a categoria de imposto. O checkout e as faturas mostram a parte do imposto calculada a partir do preço exibido.
</Info>

## Principais Benefícios

- **Clareza para os clientes**: Um preço final exibido de forma antecipada
- **Conformidade por região**: O imposto é calculado por localização e detalhado
- **Faturamento preciso**: As faturas mostram o valor líquido e o imposto derivado do preço
- **Amigável para conversões**: Redução de surpresas no checkout

## Configuração do Produto

Crie ou edite um produto no seu painel, depois ative o preço com impostos inclusos nas configurações de preços.

#### Preços

- **Preço** (obrigatório): Preço base que é exibido para o cliente.
- **Preço com Impostos Inclusos**: Ative para tratar o preço exibido como incluindo impostos. O Dodo calculará o valor líquido (antes dos impostos) e a parte do imposto no checkout com base na localização do cliente e na categoria de imposto.

<Warning>
Apenas compras futuras refletem as configurações de impostos inclusos atualizadas. Compras existentes e assinaturas ativas não são alteradas retroativamente.
</Warning>

<Tip>
Use uma cópia de produto consistente, como “Impostos inclusos”, para que os clientes entendam os totais rapidamente.
</Tip>

## Exemplos de Integração

### Criar um produto com preços com impostos inclusos

```typescript
const product = await client.products.create({
  name: 'Pro Plan',
  description: 'All features included',
  price: {
    currency: 'USD',
    price: 10000
    tax_inclusive: true
  }
});
```

### Atualizar um produto existente para ativar impostos inclusos

```typescript
await client.products.update(product.id, {
  price: {
    tax_inclusive: true
  }
});
```

## Melhores Práticas

- **Escolha a categoria de imposto correta** para garantir o cálculo correto por região.
- **Comunique os totais** de forma clara com rótulos “Impostos Inclusos” na cópia do produto.
- **Teste no sandbox** para validar.
