Os pagamentos com cartão são a base dos pagamentos online, aceitos globalmente e confiáveis para clientes em todo o mundo. A Dodo Payments oferece suporte a todas as principais bandeiras com proteção antifraude integrada e conformidade com PCI.

## Configuração

Use estes valores em `allowed_payment_method_types`:

| Type | Description |
| :--- | :---------- |
| `credit` | All credit cards |
| `debit` | All debit cards |

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  allowed_payment_method_types: ['credit', 'debit'],
  return_url: 'https://example.com/success'
});
```

<Tip>
Inclua tanto `credit` quanto `debit`, a menos que haja um motivo específico para excluir um deles. Cartões de débito são preferidos por muitos clientes e frequentemente têm taxas menores.
</Tip>

## Autenticação 3D Secure

3D Secure (3DS) adiciona uma camada de autenticação que reduz fraudes e estornos ao verificar a identidade do titular do cartão.

### Quando o 3DS é Acionado

O 3DS é acionado automaticamente quando:
- Exigido pela bandeira do cartão
- Exigido por regulamentações regionais (por exemplo, PSD2 na Europa)
- A transação é sinalizada como de alto risco

### Forçar 3DS

Você pode exigir 3DS em todas as transações:

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  force_3ds: true,
  return_url: 'https://example.com/success'
});
```

<Note>
Ativar 3DS para todas as transações reduz fraudes, mas pode diminuir ligeiramente a conversão, pois alguns clientes desistem durante a autenticação.
</Note>

Os clientes podem salvar seus cartões para acelerar compras futuras.

### Habilitar Cartões Salvos

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  show_saved_payment_methods: true,
  customer: { customer_id: 'cus_existing_123' },
  return_url: 'https://example.com/success'
});
```

### Compras com Um Clique

```javascript
// Get customer's saved payment methods
const methods = await client.customers.getPaymentMethods('cus_123');

// Use saved card for instant checkout
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  customer: { customer_id: 'cus_123' },
  payment_method_id: methods[0].payment_method_id,
  confirm: true,
  return_url: 'https://example.com/success'
});
```

## Solução de Problemas

<AccordionGroup>
<Accordion title="Card declined">
**Causas:** Fundos insuficientes, cartão vencido, CVV incorreto, proteção antifraude do banco.

**Solução:** Peça ao cliente para verificar os dados ou tentar outro cartão.
</Accordion>

<Accordion title="3DS authentication failed">
**Causas:** Cliente desistiu, sistema bancário indisponível, tempo limite.

**Solução:** Tente novamente ou peça ao cliente que entre em contato com o banco.
</Accordion>

<Accordion title="Card not supported">
**Causas:** Cartão regional não compatível, restrições de pré-pago.

**Solução:** O cliente deve tentar outro cartão de uma rede principal.
</Accordion>
</AccordionGroup>
