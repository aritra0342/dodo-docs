## O que são Carteiras de Clientes?

As carteiras dos clientes são contas de saldo monetário que armazenam fundos reais para seus usuários. Cada cliente recebe uma automaticamente ao criar a conta. Você pode usar essas carteiras para:

- **Armazenar fundos pré-pagos** para pagamentos futuros de assinaturas
- **Tratar reembolsos** como saldo na carteira em vez de reembolsos no cartão
- **Emitir saldos promocionais** como bônus de boas-vindas ou recompensas de fidelidade
- **Aplicar fundos da carteira em faturas** automaticamente durante a cobrança
- **Acompanhar transações monetárias** com um histórico detalhado do razão

<Info>
Todo cliente recebe automaticamente uma carteira quando você cria a conta. As carteiras suportam moedas USD e INR com saldos monetários separados para cada uma.
</Info>

<Warning>
**Carteiras dos Clientes ≠ Cobrança Baseada em Créditos**

As carteiras dos clientes mantêm **saldos monetários reais** (USD, INR) que podem ser aplicados a faturas e pagamentos de assinaturas.

Se você deseja acompanhar unidades virtuais de uso (chamadas de API, tokens, horas de computação), veja [Credit-Based Billing](/features/credit-based-billing) em vez disso.
</Warning>

## Como Funciona

As carteiras dos clientes mantêm fundos que podem ser aplicados às compras. Quando um cliente tem uma fatura ou cobrança recorrente de assinatura, o saldo da carteira é verificado primeiro. Quaisquer fundos disponíveis são aplicados automaticamente à fatura antes de cobrar o método de pagamento principal.

### Configuração Automática

Quando você cria um novo cliente, o Dodo Payments cria automaticamente uma carteira com saldo zero. Ela está pronta para receber fundos imediatamente por meio da nossa API.

### Suporte Multimoeda

Cada carteira pode manter saldos em diferentes moedas:

<ResponseField name="USD Balance" type="integer">
Saldo em Dólares Americanos (armazenado em centavos)
</ResponseField>

<ResponseField name="INR Balance" type="integer">
Saldo em Rúpias Indianas (armazenado em paise)
</ResponseField>

<Info>
Atualmente, apenas saldos em **USD** e **INR** estão disponíveis. Mais moedas chegando em breve.
</Info>

## Exemplos do Mundo Real

### Reembolso para Carteira

Quando um cliente solicita um reembolso, você pode adicionar o valor ao saldo da carteira dele em vez de realizar um reembolso tradicional no cartão. Isso mantém os fundos dentro do seu ecossistema para compras futuras.

```javascript
async function refundToWallet(customerId, refundAmount, originalPaymentId) {
  await client.customers.wallets.ledgerEntries.create(customerId, {
    amount: refundAmount, // Amount in cents
    currency: 'USD',
    entry_type: 'credit',
    reason: `Refund for payment ${originalPaymentId}`,
    idempotency_key: `refund_${originalPaymentId}`
  });
}
```

### Bônus de Boas-Vindas / Saldo Promocional

Ofereça aos novos clientes um bônus monetário de boas-vindas para incentivar a primeira compra.

```javascript
async function addWelcomeBonus(customerId) {
  await client.customers.wallets.ledgerEntries.create(customerId, {
    amount: 1000, // $10.00 promotional balance
    currency: 'USD',
    entry_type: 'credit',
    reason: 'Welcome bonus - $10 promotional balance',
    idempotency_key: `welcome_${customerId}`
  });
}
```

### Pagamento de Assinatura pela Carteira

Deduz o fundo da carteira para cobrir uma cobrança de assinatura ou uma compra manual.

```javascript
async function deductForPurchase(customerId, purchaseAmount, purchaseId) {
  try {
    await client.customers.wallets.ledgerEntries.create(customerId, {
      amount: purchaseAmount,
      currency: 'USD',
      entry_type: 'debit',
      reason: `Subscription charge - monthly billing`,
      idempotency_key: `charge_${purchaseId}`
    });
  } catch (error) {
    if (error.status === 400) {
      console.log('Insufficient wallet balance');
    }
  }
}
```

## Melhores Práticas

### Evite Transações Duplicadas

Use chaves de idempotência para garantir que você não adicione ou deduza fundos duas vezes por engano no mesmo evento.

```javascript
async function addFundsSafely(customerId, amount, reason) {
  const idempotencyKey = `${reason}_${customerId}_${Date.now()}`;
  
  try {
    const result = await client.customers.wallets.ledgerEntries.create(customerId, {
      amount: amount,
      currency: 'USD',
      entry_type: 'credit',
      reason: reason,
      idempotency_key: idempotencyKey
    });
    
    return { success: true, wallet: result };
  } catch (error) {
    if (error.status === 409) {
      // Transaction already processed
      return { success: true, wallet: null, duplicate: true };
    }
    
    throw error;
  }
}
```

### Verifique os Saldos Antes de Cobrar

Confirme se os clientes têm fundos suficientes antes de tentar processar grandes transações usando a carteira.

```javascript
async function checkBalanceBeforeOperation(customerId, requiredAmount) {
  const wallets = await client.customers.wallets.list(customerId);
  const usdWallet = wallets.items.find(w => w.currency === 'USD');
  
  if (!usdWallet || usdWallet.balance < requiredAmount) {
    throw new Error('Insufficient funds for this operation');
  }
  
  return usdWallet.balance;
}
```
