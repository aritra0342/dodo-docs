## Introdução

A geração de Chave de API permite que os comerciantes autentiquem e integrem seus sistemas com a plataforma da Dodo Payments de forma segura para acesso contínuo às funcionalidades de pagamento, produto e assinatura.

**Nota :** Chaves de API criadas no modo de Teste serão limitadas ao ambiente de modo de teste e vice-versa para o modo ao vivo
