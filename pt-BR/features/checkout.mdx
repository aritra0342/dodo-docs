## Moeda Adaptativa

A Moeda Adaptativa permite que os clientes paguem em sua moeda local preferida, melhorando a confiança e as taxas de conversão.

### Como Funciona

1. **Ativar**: Ative a Moeda Adaptativa nas Configurações do Seu Negócio
2. **Selecionar**: Os clientes podem trocar de moeda diretamente no checkout
3. **Converter**: Os preços são convertidos dinamicamente usando taxas de câmbio em tempo real
4. **Exibir**: O valor final a ser pago é mostrado de forma transparente antes do pagamento

## Cálculo Automático de Impostos

Os impostos são calculados automaticamente com base na localização de cobrança do cliente, garantindo conformidade com os requisitos de GST, VAT e impostos sobre vendas sem configuração manual.

### Como Funciona o Cálculo de Impostos

<Steps>
<Step title="Detecção de Localização">
  As regras fiscais são aplicadas com base no país do cliente (e região, quando aplicável).
</Step>

<Step title="Atualizações Dinâmicas">
  O valor do imposto é atualizado automaticamente quando:
  - O país muda
  - O endereço é atualizado
</Step>

<Step title="Exibição Transparente">
  A divisão final do imposto é mostrada claramente antes do pagamento.
</Step>
</Steps>

<Tip>
O cálculo de impostos é totalmente automatizado. Nenhuma configuração manual é necessária para bens digitais padrão e produtos SaaS.
</Tip>

## Suporte a ID Fiscal Empresarial

Para empresas registradas, o checkout permite que os clientes insiram seu ID Fiscal Empresarial (por exemplo, número de VAT/GST).

### O Que Acontece Quando um ID Fiscal É Inserido

- A elegibilidade para impostos é validada em tempo real
- Isenções fiscais aplicáveis ou regras de reversão são aplicadas
- O valor do imposto é atualizado instantaneamente no checkout

## Códigos de Desconto

Os clientes podem aplicar códigos de desconto ou promoções que você criou no painel diretamente na página de checkout.

### Experiência de Checkout

1. O cliente insere o código de desconto
2. O desconto é validado instantaneamente
3. O preço atualizado e as economias são mostrados claramente

<Frame>
  <img src="/images/checkout/c4.png" alt="Entrada do código de desconto no checkout" style={{ width: '70%' }}/>
</Frame>

### Integração de API

Pré-aplique um código de desconto ou habilite o campo de entrada de desconto:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  discount_code: 'WELCOME20', // Pre-apply a code
  feature_flags: {
    allow_discount_code: true // Show discount input field
  },
  return_url: 'https://yoursite.com/return'
});
```

## Coleta Inteligente de Endereços

O checkout suporta entrada de endereço flexível para conclusão mais rápida.

### Opções Disponíveis

| Opção | Descrição |
|--------|-------------|
| **Autocompletar Endereço do Google** | Seleção rápida com autocompletar |
| **Entrada Manual** | Controle total para endereços completos |
| **Seleção de País** | Direciona a lógica de impostos e conformidade |

<Tip>
A coleta de endereços equilibra velocidade, precisão e cobertura global para maximizar a conversão enquanto garante conformidade.
</Tip>

## Política de Privacidade e Aceitação de Termos

Para garantir transparência legal e de conformidade:

- Os links da [Política de Privacidade](https://dodopayments.com/privacy-policy) e dos [Termos do Comprador](https://dodopayments.com/buyer-terms) são claramente exibidos no checkout
- Os clientes reconhecem explicitamente isso antes de concluir o pagamento

<Info>
Isso ajuda a atender aos requisitos globais de proteção ao consumidor e privacidade de dados, incluindo conformidade com o GDPR.
</Info>

## Configuração da Sessão de Checkout

Controle o comportamento do checkout usando a API de Sessões de Checkout:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  customer: {
    email: 'customer@example.com',
    name: 'Jane Doe'
  },
  billing_currency: 'EUR', // Set specific currency
  discount_code: 'PROMO10',
  feature_flags: {
    allow_discount_code: true
  },
  return_url: 'https://yoursite.com/return',
  metadata: {
    order_ref: 'ORD-12345'
  }
});
```
