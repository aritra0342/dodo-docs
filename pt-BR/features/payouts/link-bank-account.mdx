## Processo para Vincular uma Conta Bancária

1. Vá para a seção **Configurações de Pagamento** no painel da Dodo Payments e selecione a aba **Informações Bancárias**.
2. Selecione a **localização da sua conta bancária** (Índia, EUA, Europa, Resto do Mundo).
3. Preencha o formulário com os detalhes precisos conforme descrito abaixo.
4. Faça o upload de um **Extrato Bancário** que atenda aos requisitos de visibilidade para verificação.

## Campos de Informações Bancárias com Base no País

### **1. Índia**

| Campo | Descrição |
| --- | --- |
| **Nome do Titular da Conta** | Nome exato conforme os detalhes da conta bancária |
| **Nome do Banco** | Nome do banco |
| **Categoria da Conta Bancária** | Tipo de conta bancária (Pessoal ou Empresarial) |
| **Moeda da Conta Bancária** | Código de 3 letras da moeda local (ex: INR) |
| **Número da Conta Bancária** | Número único da conta bancária |
| **Código IFSC** | Código IFSC conforme o extrato bancário |
| **Upload do Extrato Bancário** | Faça o upload de um extrato bancário visível mostrando todos os detalhes relevantes |

### **Diretrizes Adicionais**

- **Visibilidade do Extrato Bancário**: Todos os extratos bancários enviados devem ter detalhes da conta claros e legíveis. Evite cortar ou ocultar qualquer informação, pois isso pode atrasar o processo de verificação.
- **Precisão do Nome da Conta**: O Nome do Titular da Conta deve corresponder exatamente ao nome nos documentos bancários enviados para evitar discrepâncias durante o processo de verificação.
- **Códigos SWIFT/BIC e Números de Roteamento**: Certifique-se de que os códigos SWIFT/BIC e os Números de Roteamento estão corretos para transferências internacionais e domésticas para facilitar pagamentos sem problemas.
- **Códigos de Moeda**: Sempre use o código de moeda de 3 letras correto (ex: INR para Índia, USD para EUA, EUR para Europa) conforme a moeda local da conta bancária.
Você pode usar o seguinte link para identificar seu código de moeda se tiver alguma dúvida: [https://www.iban.com/currency-codes](https://www.iban.com/currency-codes)
