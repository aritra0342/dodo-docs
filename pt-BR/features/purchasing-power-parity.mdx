## Introdução

[ParityDeals](https://www.paritydeals.com/) é uma plataforma que permite a otimização automática de preços com base na localização geográfica do cliente, comumente conhecida como precificação de "paridade de poder de compra".

Com o ParityDeals, você também pode oferecer [descontos automáticos baseados em tempo](https://www.paritydeals.com/features/time-based-discounts/) e [promoções baseadas em feriados](https://www.paritydeals.com/features/holiday-based-discounts/) ao longo do ano.

<Info>
Embora você não possa incorporar diretamente o banner do ParityDeals em uma loja Dodo Payments (já que scripts personalizados não são suportados), você ainda pode implementar o ParityDeals de forma eficaz em seu próprio site ou aplicativo.
</Info>

## Como Funciona

Quando o ParityDeals é integrado ao seu site, visitantes elegíveis verão um banner que inclui um código de desconto baseado em localização que podem usar no checkout.

<Frame>
  <img src="/images/ppp/1.png" alt="Exemplo de banner de desconto do ParityDeals" />
</Frame>

Esses códigos de desconto são criados e gerenciados automaticamente pelo ParityDeals e aparecerão no seu **Painel do Dodo Payments** em **Vendas → Descontos**.

## Guia de Configuração Passo a Passo

### 1. Inscreva-se no ParityDeals

Visite [app.paritydeals.com](https://app.paritydeals.com/) e crie uma conta.

### 2. Crie Ofertas

Após fazer login, localize a opção para criar ofertas. No menu suspenso, selecione:

**→ "Criar Ofertas via Dodo Payments"**

<Frame>
  <img src="/images/ppp/2.png" alt="Criando ofertas via Dodo Payments" />
</Frame>

### 3. Gere uma Chave de API do Dodo Payments

Faça login no seu **painel do Dodo Payments** e navegue até:

**Desenvolvedores → Chaves de API**

Clique em **"Criar Nova API"** e nomeie-a de forma descritiva, como "ParityDeals".

<Frame>
  <img src="/images/ppp/3.png" alt="Criando uma nova chave de API no Dodo Payments" />
</Frame>

### 4. Conecte Sua Chave de API ao ParityDeals

Copie a chave de API que você criou e cole-a na interface do ParityDeals para estabelecer a conexão com sua conta do Dodo Payments.

<Frame>
  <img src="/images/ppp/4.png" alt="Conectando a chave de API ao ParityDeals" />
</Frame>

### 5. Selecione Produtos

Uma vez conectado, o ParityDeals buscará seu catálogo de produtos do Dodo Payments. Selecione os produtos que deseja associar às suas ofertas.

<Frame>
  <img src="/images/ppp/5.png" alt="Selecionando produtos para o ParityDeals" />
</Frame>

### 6. Configure as Configurações da Oferta

Insira a URL do seu site e, opcionalmente, selecione páginas-alvo específicas onde o banner de desconto deve aparecer.

Em seguida, escolha seu tipo de oferta preferido:

- Ofertas baseadas em localização
- Ofertas de feriado
- Ofertas relâmpago baseadas em tempo

<Frame>
  <img src="/images/ppp/6.png" alt="Configurando as configurações da oferta" />
</Frame>

### 7. Defina Regras Baseadas em País

Selecione os países para os quais deseja oferecer descontos localizados e defina a porcentagem de desconto para cada um. O ParityDeals fornece padrões recomendados que você pode ajustar conforme necessário.

<Frame>
  <img src="/images/ppp/7.png" alt="Definindo regras de desconto baseadas em país" />
</Frame>

### 8. Personalize o Banner

Desenhe seu banner de desconto ao:

- Personalizar o texto
- Selecionar cores e layout
- Escolher a posição do banner (parte superior ou inferior da página)

Este banner será exibido para visitantes elegíveis com base nas regras que você configurou.

<Frame>
  <img src="/images/ppp/8.png" alt="Personalizando o banner de desconto" />
</Frame>

### 9. Adicione o Script do ParityDeals ao Seu Site

Após salvar seu negócio, o ParityDeals gerará um script. Adicione este script ao código do seu site (normalmente na seção `<head>`).

Este script detecta automaticamente as localizações dos visitantes e exibe o banner de desconto apropriado.

<Frame>
  <img src="/images/ppp/9.png" alt="Script de integração do ParityDeals" />
</Frame>

## Tudo Pronto!

Agora você está pronto para oferecer:

- **Ofertas baseadas em localização** - Ajuste os preços com base no poder de compra
- **Descontos com contagem regressiva/sensíveis ao tempo** - Crie urgência com ofertas por tempo limitado
- **Promoções de feriado** - Aproveite as tendências de compras sazonais

Tudo isso é automatizado através do ParityDeals e funciona perfeitamente com o Dodo Payments.

Para mais informações detalhadas, consulte a [Documentação do ParityDeals](https://www.paritydeals.com/docs/) ou entre em contato com a equipe de suporte deles.
