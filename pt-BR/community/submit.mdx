---
title: Envie seu projeto
description: Como adicionar seu projeto de código aberto ao diretório da
  comunidade Dodo Payments.
icon: plus
---

<Steps>
{/* LOCKED_PATTERN_b8e49197111d4f91963d876b7d3cbd9b */}
- O projeto é open-source com licença permissiva.
- É relevante para Dodo Payments (SDK, plugin, integração, template).
- Possui um README com instalação, uso e status de manutenção.
</Step>

{/* LOCKED_PATTERN_29bfb0d9a10f5f1cc614e31786e69c52 */}
Edite `community/projects.mdx` e adicione um novo `<Card>` na guia mais relevante.

```mdx
<Card title="PROJECT_NAME (LANG)" href="REPO_URL" icon="code">
  One-sentence description of your project.
  <br/>By <a href="GITHUB_PROFILE_URL">@github_username</a>
  <br/><br/>`LANGUAGE` `FRAMEWORK` `CATEGORY`
</Card>
```

</Step>

{/* LOCKED_PATTERN_5390269a28150786d344085d66aa0ef8 */}
Se preferir, abra uma issue usando o modelo "Community Project Submission" com:
- URL do repositório, descrição, categoria (SDK/Plugin/Boilerplate)
- Linguagem/framework principal
- Link do trecho de instalação/uso
- Status de manutenção e última atualização
</Step>

{/* LOCKED_PATTERN_03e556e84db70a2199e5921d65cd601a */}
Revisamos quanto à relevância, clareza e segurança. Entradas aprovadas são mescladas e publicadas.
</Step>
</Steps>

<Warning>
Podemos remover entradas que estejam sem manutenção, inseguras, enganosas ou violem a política.
</Warning>
