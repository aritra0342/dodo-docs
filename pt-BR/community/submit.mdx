---
title: Envie seu projeto
description: Como adicionar seu projeto de código aberto ao diretório da
  comunidade Dodo Payments.
icon: plus
---

<Steps>
<Step title="Verifique a elegibilidade">
- O projeto é de código aberto com uma licença permissiva.
- É relevante para o Dodo Payments (SDK, plugin, integração, template).
- Possui um README com instalação, uso e status de manutenção.
</Step>

<Step title="Adicionar entrada via PR">
Edite `community/projects.mdx` e adicione um novo `<Card>` na aba mais relevante.

```mdx
<Card title="PROJECT_NAME (LANG)" href="REPO_URL" icon="code">
  One-sentence description of your project.
  <br/>By <a href="GITHUB_PROFILE_URL">@github_username</a>
  <br/><br/>`LANGUAGE` `FRAMEWORK` `CATEGORY`
</Card>
```

</Step>

<Step title="Abra uma issue de submissão (opcional)">
Se preferir, abra uma issue usando o template "Submissão de Projeto da Comunidade" com:
- URL do repositório, descrição, categoria (SDK/Plugin/Boilerplate)
- Linguagem/framework principal
- Link do snippet de instalação/uso
- Status de manutenção e última atualização
</Step>

<Step title="Revisar e publicar">
Nós revisamos a relevância, clareza e segurança. Entradas aprovadas são mescladas e publicadas.
</Step>
</Steps>

<Warning>
Podemos remover entradas que estão sem manutenção, inseguras, enganosas ou que violam a política.
</Warning>
