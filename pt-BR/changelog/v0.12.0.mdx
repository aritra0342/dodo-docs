## Novos Recursos
- **SDKs Oficiais**: Introduzidos SDKs oficiais do Dodo Payments para **Node.js**, **Python** e **Go**. Consulte a página dos [SDKs](/developer-resources/dodo-payments-sdks) para mais informações.
- **Campo de Metadados**: Adicionado um campo `Metadata` para pagamentos e assinaturas para armazenar informações adicionais. Consulte a página de [Metadados](/api-reference/metadata) para mais detalhes.
- **APIs de Clientes**: APIs de clientes expandidas para incluir os seguintes endpoints. Agora você pode passar `customer_id` em pagamentos e assinaturas para evitar duplicação de clientes:
    - **Criar Cliente**: [/api-reference/customers/create-customer](/api-reference/customers/create-customer)
    - **Atualizar Cliente**: [/api-reference/customers/patch-customer](/api-reference/customers/patch-customer)
