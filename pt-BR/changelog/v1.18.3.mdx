## Novos Recursos 🚀

- **API de Mudança de Plano de Assinatura**: Lançada uma nova API que permite atualizações ou reduções de assinaturas entre diferentes planos de produtos de forma contínua. Para detalhes de implementação, veja a [referência da API](/api-reference/subscriptions/change-plan).

- **Orientação de Integração Aprimorada**: Melhoradas as dicas na barra lateral para fornecer uma direção mais clara sobre os próximos passos na sua jornada de integração com o Dodo Payments.

  <Frame>
    <img alt="Dicas na Barra Lateral" src="/images/changelog/sidebar-nudges.png" />
  </Frame>

- **Centro de Preferências de Notificação**: Adicionada uma seção dedicada para gerenciar suas configurações de notificação. Acesse clicando no ícone de sino no canto superior direito e, em seguida, selecionando o ícone de configurações.

  <Frame>
    <img alt="Preferências de Notificação" src="/images/changelog/notif-pref.png" />
  </Frame>

- **Informações sobre Moeda Adaptativa Aprimoradas**: Adicionados novos campos à resposta de pagamento, incluindo `settlement_amount`, `settlement_currency`, e `settlement_tax` para fornecer maior transparência sobre os detalhes de liquidação.

- **Melhorias Gerais**: Várias pequenas melhorias e correções de bugs para melhorar o desempenho geral da plataforma.
