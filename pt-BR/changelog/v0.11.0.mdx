## Novos Recursos
- **E-mails Automatizados**: Adicionados e-mails automatizados para pagamentos, assinaturas, reembolsos e disputas para manter comerciantes e clientes informados.
- **Log de Eventos de Webhook**: Adicionado log de eventos de webhook no Painel de Pagamentos Dodo para rastrear eventos de webhook e solucionar problemas. Navegue até `Settings > Webhooks > Webhook Logs` para acessar os logs.
- **Guia de Geração de Chave de Licença**: Adicionado um novo guia para implementar a geração de chave de licença para produtos digitais. Consulte o [guia](/features/license-keys) para mais informações.

- Várias correções e melhorias menores.
