## Novos Recursos
- **Automated Emails**: Adicionados e-mails automatizados para pagamentos, assinaturas, reembolsos e disputas para manter os comerciantes e clientes informados.
- **Webhook Event Log**: Adicionado registro de eventos de webhook no Painel Dodo Payments para acompanhar eventos de webhook e solucionar problemas. Navegue até `Settings > Webhooks > Webhook Logs` para acessar os registros.
- **License Key Generation Guide**: Adicionado um novo guia para implementar a geração de chaves de licença para produtos digitais. Consulte o [guia](/features/license-keys) para mais informações.
