## Novos Recursos 🚀

- **Primeiro Pagamento da Assinatura**: O recurso mais solicitado está aqui! Agora, haverá uma única transação para o primeiro pagamento, combinando o mandato e o valor real da assinatura.

- **Método de Pagamento UPI QR**: Adicionado UPI QR como método de pagamento para o mercado indiano, permitindo que os clientes paguem usando códigos QR UPI.
    
    <img src="/images/changelog/upi-qr.png" alt="Recurso UPI QR"/>

- **CPF na Finalização da Compra**: Os clientes agora podem adicionar seu CPF na página de finalização da compra para fins de faturamento e tributação.

- **Reestruturação do Painel**: Todo o Painel de Pagamentos Dodo foi reestruturado para ser mais intuitivo e amigável ao usuário.

- **Guias de Integração Autoatendidas**: Novos usuários agora podem começar a usar o Dodo Payments com guias de integração autoatendidas.

- **Suporte multilíngue**: Adicionamos suporte para mais de 14 idiomas na página de checkout, permitindo que os clientes paguem no idioma preferido. Os idiomas suportados incluem:
    - English (en)
    - العربية (ar) // Arabic
    - Català (ca) // Catalan
    - 中文 (cn) // Chinese
    - Deutsch (de) // German
    - Español (es) // Spanish
    - Français (fr) // French
    - עברית (he) // Hebrew
    - Italiano (it) // Italian
    - 日本語 (ja) // Japanese
    - Nederlands (nl) // Dutch
    - Polski (pl) // Polish
    - Português (pt) // Portuguese
    - Svenska (sv) // Swedish

- **Reformulação do PDF da Fatura**: Os PDFs das faturas receberam um novo design, mais claro e eficaz.

- **Widget de Ajuda Centralizado**: Adicionado um widget de Ajuda centralizado para agilizar e melhorar o suporte.
    
    <img src="/images/changelog/help-widget.png" alt="Widget de Ajuda"/>

## Melhorias e Correções de Bugs 🔧

- **Limitação de Taxa**: Adicionada limitação de taxa na API de faturas para prevenir abusos e garantir a estabilidade do sistema.

- **Metadados em Links de Pagamento Estáticos**: Adicionado suporte para metadados em links de pagamento estáticos. Para mais detalhes, consulte a [documentação](/developer-resources/integration-guide#1-static-payment-links).

- **Melhorias e Correções Menores**: Várias melhorias e correções menores para melhorar o desempenho geral e a experiência do usuário.
