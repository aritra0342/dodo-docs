## Novos Recursos 🚀

- **Formulários de Verificação Modernos**: Reformulamos os formulários de verificação com uma interface moderna e amigável, apresentando instruções mais claras, uma hierarquia visual aprimorada e campos de formulário intuitivos para um processo de verificação mais suave.

  <Frame>
    <img alt="Verification Forms" src="/images/changelog/verification_forms.png" />
  </Frame>

- **Novos SDKs para Java e Kotlin**: Lançamos SDKs nativos para Java e Kotlin para suportar o desenvolvimento Android e aplicações Java empresariais. Os SDKs incluem documentação abrangente, código de exemplo e suporte total para recursos de Dodo Payments, incluindo pagamentos, assinaturas e reembolsos. Disponível no GitHub: [Java SDK](https://github.com/dodopayments/dodopayments-java) e [Kotlin SDK](https://github.com/dodopayments/dodopayments-kotlin).

- **Integração com Afterpay/Clearpay**: Adicionamos suporte para Afterpay/Clearpay como um método de pagamento Buy Now, Pay Later (BNPL), permitindo que os clientes dividam os pagamentos em 4 parcelas sem juros.

- **Google Sign-In**: Implementamos o Google Sign-In como uma nova opção de autenticação, permitindo que os usuários façam login usando suas contas do Google. Esta integração compatível com OAuth 2.0 suporta contas pessoais e de workspace do Google para maior segurança e conveniência.
