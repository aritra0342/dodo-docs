## Novos Recursos 🚀

- **Checkout 2.0**: Apresentando uma experiência de checkout completamente redesenhada com:
    - Interface moderna e simplificada para melhorar a experiência do usuário
    - Novo Sistema de Coleta de Consultas que armazena parâmetros no armazenamento de sessão com IDs únicos (por exemplo, `sess_1a2b3c4d`)
    - Disponibilidade de dados persistentes durante todo o fluxo de checkout
    - Mais melhorias planejadas para lançamentos futuros!
  
    <Frame><img src="/images/changelog/checkout_2.png" alt="Checkout 2.0"/></Frame>

- **Métodos de Pagamento Configuráveis**: Nova configuração `allowed_payment_methods` permite que os comerciantes personalizem as opções de pagamento disponíveis durante o checkout. Veja a documentação sobre [Métodos de Pagamento Permitidos](/features/payment-methods#configuring-payment-methods) para detalhes de implementação.

- **Cálculo Dinâmico de Impostos**: Cálculo em tempo real das taxas de impostos com base nos endereços de cobrança dos clientes para preços mais precisos nos links de pagamento estáticos.

- **Assinaturas Sob Demanda**: Novos endpoints de API para faturamento flexível de assinaturas (disponível para comerciantes selecionados). Saiba mais no guia de [Assinaturas Sob Demanda](/features/subscription#enabling-on-demand-subscriptions).

- **Atualizações Adicionais**:
    - Otimizações de desempenho
    - Correções de bugs
    - Melhorias gerais do sistema
