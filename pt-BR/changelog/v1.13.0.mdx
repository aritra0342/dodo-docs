## Novos Recursos 🚀

- **[Suporte a Moeda Adaptativa](/features/adaptive-currency)** 💱
  - Exibe automaticamente os preços nas moedas locais dos clientes com base em sua localização.
  - Os comerciantes recebem pagamentos em sua moeda preferida, enquanto os clientes pagam em sua moeda local.
  - Estrutura de taxa de conversão transparente variando de 2% a 4%, dependendo do valor do pedido.
  - Suporte para mais de 25 novas moedas foi adicionado:
    - **Oriente Médio**: AED (Dirham dos EAU), SAR (Riyal Saudita)
    - **Ásia-Pacífico**: AUD (Dólar Australiano), CNY (Yuan Chinês), HKD (Dólar de Hong Kong), IDR (Rupia Indonésia), KRW (Won Sul-Coreano), SGD (Dólar de Cingapura), TWD (Novo Dólar de Taiwan)
    - **Europa**: ALL (Lek Albanês), CHF (Franco Suíço), CZK (Coroa Checa), DKK (Coroa Dinamarquesa), EUR (Euro), GBP (Libra Esterlina), HUF (Forint Húngaro), NOK (Coroa Norueguesa), PLN (Zloty Polonês), RON (Leu Romeno), RSD (Dinar Sérvio), SEK (Coroa Sueca), TRY (Lira Turca)
    - **Américas**: BRL (Real Brasileiro), CAD (Dólar Canadense), MXN (Peso Mexicano)

- **[Servidor MCP do Dodo Payments](/developer-resources/mcp-server)** 🤖
  - Apresentando o novo Servidor do Protocolo de Contexto de Modelo (MCP) para integração perfeita de modelos de IA.
  - Fornece uma interface padronizada para interagir com a API do DodoPayments.
  - Instalação simples via `npx` ou através da integração do cliente MCP.
  - Oferece um conjunto abrangente de ferramentas para gerenciar pagamentos, assinaturas e mais.
  - Inclui recursos avançados, como filtragem de ferramentas e controles de segurança aprimorados.

- [Aplicativo Móvel](/features/mobile-app): Agora disponível para iOS e Android, trazendo a experiência completa do painel para seu dispositivo móvel. Os principais recursos incluem:
  - Notificações em tempo real para transações, pagamentos e renovações de assinaturas
  - Acesso completo ao painel com rastreamento de receita e gráficos de desempenho
  - Capacidades completas de gerenciamento de assinaturas e produtos
  - Operações comerciais incluindo troca entre modo de teste/ao vivo e gerenciamento de verificações
  - Ferramentas para desenvolvedores com gerenciamento de chaves de API e monitoramento de webhooks
  Baixe agora na [App Store](https://apps.apple.com/in/app/dodo-payments/id6743651329) ou [Google Play](https://play.google.com/store/apps/details?id=com.dodopayments.app).

- **GPT Personalizado para Dodo Payments**: Disponível no ChatGPT Marketplace, treinado em nossa documentação para ajudar com perguntas ou problemas técnicos. Acesse aqui, https://dodo.pe/custom-gpt

- **Dodo Dash**: Experimente a emoção do nosso jogo de corrida infinita, Dodo Dash! Guie um ousado pássaro dodo enquanto ele corre, salta e desvia de obstáculos para sobreviver. Experimente aqui, https://dash.dodopayments.com.

- Várias outras correções de bugs e melhorias foram implementadas.
