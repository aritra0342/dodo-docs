## Novos Recursos 🚀

### API de Sessões de Checkout
Apresentamos nossa nova **API de Sessões de Checkout** - uma solução de checkout hospedada e poderosa que fornece um fluxo de pagamento completo tanto para compras únicas quanto para assinaturas. Esta nova API substitui a abordagem anterior de Links Dinâmicos com flexibilidade aprimorada e uma melhor experiência do usuário.

**Principais Benefícios:**
- **Integração Simplificada**: Não é necessário fornecer endereços de cobrança completos antecipadamente - nós lidamos com os detalhes ausentes automaticamente
- **Personalização Aprimorada**: Controle total sobre métodos de pagamento, moedas, temas e flags de recursos
- **Melhor UX**: Pré-preencha as informações do cliente para reduzir a fricção no checkout e melhorar as taxas de conversão
- **Configuração Flexível**: Suporte para carrinhos de múltiplos produtos, testes de assinatura, códigos de desconto e mais

**O que Mudou:**
- **Antes**: Era necessário fornecer informações completas de endereço de cobrança ao criar links de pagamento
- **Agora**: Passe qualquer informação do cliente que você tenha disponível - nós coletaremos o restante durante o checkout
- **Migração**: Mudança simples de endpoint da API sem necessidade de manuseio adicional
