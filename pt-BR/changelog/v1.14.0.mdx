## Novos Recursos 🚀

- **Novos Métodos de Pagamento Europeus**: Expandimos nossas opções de pagamento com quatro novos métodos de pagamento europeus:
  - **iDEAL** - Países Baixos
  - **Bancontact** - Bélgica
  - **Przelewy24 (P24)** - Polônia
  - **EPS** - Áustria

  <Frame>
    <img alt="Supported European Payment Methods" src="/images/transactions/payments/european.png" />
  </Frame>

  <Note>
    Estes métodos de pagamento estão atualmente limitados a:
      * Moeda Euro (EUR) apenas
      * Pagamentos únicos apenas (não disponíveis para assinaturas)

    Para obter informações detalhadas sobre esses métodos de pagamento, visite nossa documentação de [Métodos de Pagamento](/features/payment-methods).
  </Note>

- **Atualizações de Endereço de Cobrança de Assinatura**: Adicionada a capacidade de atualizar endereços de cobrança para assinaturas através de:
  * Interface do portal do cliente
  * Endpoint da API de patch de assinatura
