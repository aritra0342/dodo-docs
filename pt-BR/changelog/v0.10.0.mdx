## Novos Recursos
- **Atlas de Código Aberto**: Um aplicativo de demonstração escrito em Next.js para mostrar a integração dos Dodo Payments. Explore o [Repositório do GitHub](https://github.com/dodopayments/dodo-checkout-demo) e a documentação [aqui](/developer-resources/integration-guide).
- **Seção de Solicitações de Recursos**: Introduzida uma seção de solicitações de recursos no painel do Dodo Payments para priorizar o roadmap do produto com base no feedback dos usuários.
- **API de Logs de Solicitações de Webhook**: Adicionadas APIs para listar e detalhar logs de solicitações de webhook.
- **Links de Pagamento Estáticos**: Adicionados mais parâmetros de consulta aos links de pagamento estáticos para fornecer mais flexibilidade no processo de checkout. Para mais detalhes, clique [aqui](/developer-resources/integration-guide#1-static-payment-links).
- **API de Lista de Assinaturas**: Adicionada uma API para buscar todas as assinaturas. Consulte a [Referência da API](/api-reference/subscriptions/get-subscriptions-1) para detalhes.

## Melhorias Menores e Correções de Bugs
- Aprimoradas as respostas da API de Produto e Assinatura com parâmetros adicionais.
- Implementado limite de taxa para solicitações de API, atualmente definido para 20 solicitações por segundo.
- Melhorada a documentação para ser mais amigável e fácil de navegar.
- Várias correções de bugs e melhorias menores.
