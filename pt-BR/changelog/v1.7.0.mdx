## Novos Recursos 🚀

- **Página de Assinaturas Dedicada**: Gerencie as assinaturas dos clientes facilmente com a nova página dedicada. Navegue até `Sales > Subscriptions` no painel.
  
    <Frame><img src="/images/changelog/subscriptions.png" alt="Página de Assinaturas"/></Frame>

- **Detalhes do Cliente Aprimorados**: Veja assinaturas, pagamentos e reembolsos de cada cliente em um só lugar. Vá para `Sales > Customers > arrow icon` no painel.

    <Frame><img src="/images/changelog/customer_info.png" alt="Detalhes do Cliente"/></Frame>

- **Informações Detalhadas sobre Pagamentos**: Obtenha mais insights sobre valores de pagamento, ciclos e datas. Visite `Business > Payouts` no painel.

    <Frame><img src="/images/changelog/payouts_update.png" alt="Informações sobre Pagamentos"/></Frame>

- **SDK Oficial para Ruby**: Integre o Dodo Payments em suas aplicações Ruby sem esforço. Confira o [SDK Ruby](https://github.com/dodopayments/dodopayments-ruby) para mais detalhes.

- **Filtros de API Aprimorados**: Adicionados filtros como `customer_id`, `status`, etc., para APIs de lista relevantes. Consulte a documentação da API para mais informações.

- **Melhorias Menores e Correções de Bugs**: Várias outras melhorias menores e correções de bugs foram implementadas.
