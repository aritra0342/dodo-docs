## Novos Recursos 🚀

- **Addons de Assinatura**: Introduzido suporte para addons a produtos de assinatura. Na seção Produtos, você encontrará uma nova aba Addons onde pode criar addons como "Assentos". Uma vez criados, você pode editar seus produtos de assinatura para incluir esses addons. Este recurso está disponível tanto no painel quanto via API.
  - [Criar Addon](/api-reference/addons/create-addon)
  - [Listar Addons](/api-reference/addons/list-addons)
  - [Obter Addon](/api-reference/addons/get-addon)
  - [Atualizar Addon](/api-reference/addons/update-addon)
  - [Atualizar Imagens do Addon](/api-reference/addons/update-addon-images)

  Com addons, você desbloqueia novas opções de cobrança, como cobrança baseada em assentos ou cobrança variável por cliente. Seja criativo com addons para projetar modelos de cobrança flexíveis para o seu negócio.

  </Frame>

- **Gerenciamento de Disputas Aprimorado**: Melhorado o gerenciamento de disputas para ser mais robusto e mais fácil de gerenciar através do seu painel de pagamentos Dodo.

- **Dados de Resposta Expandidos**: Adicionados detalhes de cobrança do cliente às respostas de pagamento, respostas de assinatura e webhooks.

- **Melhorias Gerais**: Várias melhorias menores e correções de bugs para melhorar o desempenho geral da plataforma.
