---
title: Países Suportados
openapi: get /checkout/supported_countries
description: Recupere uma lista de países que são suportados pelos pagamentos
  Dodo para seus clientes.
---

<Tip> Você receberá uma lista de códigos de país [ISO Alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) representando os países que **atualmente suportamos para pagamentos**. Você pode ver a lista completa de países suportados [aqui](/miscellaneous/list-of-countries-we-accept-payments-from). Esta lista é atualizada regularmente à medida que expandimos nossa cobertura. </Tip>

<Tip> Nota: As opções disponíveis incluem códigos ISO Alpha-2 para **todos os países**, não apenas aqueles que atualmente suportamos. Consulte a lista de países suportados para confirmar a disponibilidade. </Tip>
