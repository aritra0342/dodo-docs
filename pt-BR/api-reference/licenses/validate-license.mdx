<Info>
**Não é necessária chave de API**: Este é um endpoint público que não requer autenticação. Você pode chamá-lo diretamente de aplicativos de cliente, softwares para desktop ou CLIs para validar chaves de licença sem expor suas credenciais de API.
</Info>
