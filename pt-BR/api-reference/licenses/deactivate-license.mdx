---
title: Desativar Licença
description: Este endpoint permite que você desative uma licença para o usuário.
openapi: post /licenses/deactivate
---

<Info>
**Nenhuma chave de API necessária**: Este é um endpoint público que não exige autenticação. Você pode chamá-lo diretamente de aplicações cliente, softwares de desktop ou CLIs para desativar chaves de licença sem expor suas credenciais de API.
</Info>
