**Nenhuma Chave de API Necessária**: Este é um endpoint público que não requer autenticação. Você pode chamá-lo diretamente de aplicativos cliente, software de desktop ou CLIs para desativar chaves de licença sem expor suas credenciais de API.
</Info>
