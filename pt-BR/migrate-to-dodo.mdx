**Mudando de outro provedor de pagamento para Dodo Payments?** Nossa ferramenta de migração facilita a transferência de seus **produtos, clientes e códigos de desconto sem perder nenhum dado.** O processo é seguro, guiado e pode ser feito em apenas alguns minutos.

Atualmente, suportamos migrações de **Lemon Squeezy, Stripe, Polar.sh e Paddle**.

## Comece

Primeiro, instale a ferramenta de migração:

```bash
npm install -g dodo-migrate
```

<Note>
Você precisará do Node.js ≥ 18 instalado em seu computador.
</Note>

## Migre Seus Dados

### Se você está vindo do Lemon Squeezy:

```bash
dodo-migrate lemonsqueezy
```

### Se você está vindo do Stripe:

```bash
dodo-migrate stripe
```

### Se você está vindo do Polar.sh:

```bash
dodo-migrate polar
```

### Se você está vindo do Paddle:

```bash
dodo-migrate paddle
```

A ferramenta irá guiá-lo por tudo passo a passo. Você precisará:
- Da chave API do seu provedor (Lemon Squeezy, Stripe, Polar.sh, Paddle, etc.)
- Da sua chave API do Dodo Payments  
- Se você deseja testar primeiro (recomendado) ou ir ao vivo
- Qual marca do Dodo Payments você deseja migrar

## Por Que Usar Nossa Ferramenta de Migração?

- **Sem perda de dados** - Tudo é transferido com segurança
- **Teste primeiro** - Experimente em modo de teste antes de ir ao vivo
- **Processo guiado** - Nós o guiamos em cada etapa
- **Retome se necessário** - Se algo der errado, você pode continuar de onde parou

## Uso Avançado

Se você preferir executar a migração sem prompts, pode fornecer todos os detalhes antecipadamente:

### Migração do Lemon Squeezy:

```bash
dodo-migrate lemonsqueezy \
  --provider-api-key=lsq_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX
```

### Migração do Stripe:

```bash
dodo-migrate stripe \
  --provider-api-key=sk_test_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX \
  --migrate-types=products,coupons
```

### Migração do Polar.sh:

```bash
dodo-migrate polar \
  --provider-api-key=polar_org_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX \
  --migrate-types=products,discounts,customers
```

### Migração do Paddle:

```bash
dodo-migrate paddle \
  --provider-api-key=paddle_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX
```
