**Chuyển từ nhà cung cấp thanh toán khác sang Dodo Payments?** Công cụ di chuyển của chúng tôi giúp bạn dễ dàng chuyển **sản phẩm, khách hàng và mã giảm giá mà không mất bất kỳ dữ liệu nào.** Quy trình này an toàn, có hướng dẫn và có thể thực hiện chỉ trong vài phút.

Chúng tôi hiện hỗ trợ di chuyển từ **Lemon Squeezy, Stripe, Polar.sh và Paddle**.

## Bắt Đầu

Đầu tiên, cài đặt công cụ di chuyển:

```bash
npm install -g dodo-migrate
```

<Note>
Bạn sẽ cần cài đặt Node.js ≥ 18 trên máy tính của bạn.
</Note>

## Di Chuyển Dữ Liệu Của Bạn

### Nếu bạn đến từ Lemon Squeezy:

```bash
dodo-migrate lemonsqueezy
```

### Nếu bạn đến từ Stripe:

```bash
dodo-migrate stripe
```

### Nếu bạn đến từ Polar.sh:

```bash
dodo-migrate polar
```

### Nếu bạn đến từ Paddle:

```bash
dodo-migrate paddle
```

Công cụ sẽ hướng dẫn bạn từng bước. Bạn sẽ cần:
- Khóa API của nhà cung cấp (Lemon Squeezy, Stripe, Polar.sh, Paddle, v.v.)
- Khóa API Dodo Payments của bạn  
- Bạn có muốn thử nghiệm trước (được khuyến nghị) hay đi vào hoạt động ngay
- Thương hiệu Dodo Payments nào bạn muốn di chuyển đến

## Tại Sao Nên Sử Dụng Công Cụ Di Chuyển Của Chúng Tôi?

- **Không mất dữ liệu** - Mọi thứ được chuyển an toàn
- **Thử nghiệm trước** - Thử nghiệm trong chế độ thử nghiệm trước khi đi vào hoạt động
- **Quy trình có hướng dẫn** - Chúng tôi sẽ hướng dẫn bạn qua từng bước
- **Tiếp tục nếu cần** - Nếu có điều gì đó không ổn, bạn có thể tiếp tục từ nơi bạn đã dừng lại

## Sử Dụng Nâng Cao

Nếu bạn muốn chạy quá trình di chuyển mà không có thông báo, bạn có thể cung cấp tất cả thông tin trước:

### Di chuyển từ Lemon Squeezy:

```bash
dodo-migrate lemonsqueezy \
  --provider-api-key=lsq_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX
```

### Di chuyển từ Stripe:

```bash
dodo-migrate stripe \
  --provider-api-key=sk_test_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX \
  --migrate-types=products,coupons
```

### Di chuyển từ Polar.sh:

```bash
dodo-migrate polar \
  --provider-api-key=polar_org_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX \
  --migrate-types=products,discounts,customers
```

### Di chuyển từ Paddle:

```bash
dodo-migrate paddle \
  --provider-api-key=paddle_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX
```
