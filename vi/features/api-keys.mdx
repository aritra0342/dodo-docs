## Giới thiệu

Việc tạo Khóa API cho phép các thương nhân xác thực một cách an toàn và tích hợp hệ thống của họ với nền tảng Dodo Payments để truy cập liền mạch vào các chức năng thanh toán, sản phẩm và đăng ký.

**Lưu ý :** Các Khóa API được tạo trong chế độ Kiểm tra sẽ bị giới hạn trong môi trường chế độ kiểm tra và ngược lại cho chế độ trực tiếp
