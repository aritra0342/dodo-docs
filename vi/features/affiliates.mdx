## Giới thiệu

Hướng dẫn này sẽ hướng dẫn bạn cách theo dõi các giới thiệu liên kết, xử lý các sự kiện hoa hồng và tăng doanh thu của bạn với các đối tác liên kết đáng tin cậy thông qua Dodo Payments. Chúng tôi cung cấp tích hợp với các nền tảng liên kết hàng đầu bao gồm **Affonso** và **Dub Partners** để giúp bạn quản lý và theo dõi các chương trình liên kết của mình một cách liền mạch.

## Tùy chọn tích hợp

Dodo Payments tích hợp với hai nền tảng quản lý liên kết hàng đầu:

- **[Affonso](https://affonso.io)**: Một nền tảng quản lý liên kết toàn diện với theo dõi hoa hồng tự động, báo cáo theo thời gian thực và bảng điều khiển liên kết chuyên dụng.
- **[Dub Partners](https://dub.co)**: Sử dụng quản lý liên kết và theo dõi chuyển đổi của Dub để cung cấp cho chương trình liên kết của bạn các phân tích phân bổ chi tiết.

Chọn nền tảng phù hợp nhất với nhu cầu của bạn, hoặc sử dụng cả hai cho các trường hợp sử dụng khác nhau.

## Tích hợp Affonso

Chúng tôi đã hợp tác với Affonso, một nền tảng quản lý liên kết hàng đầu để thúc đẩy doanh số của bạn. Làm theo các bước dưới đây để kết nối tài khoản Dodo Payments của bạn với Affonso và bắt đầu theo dõi doanh số từ liên kết.

<Steps>
  <Step title="Kết nối với Dodo Payments">
    Để bắt đầu, hãy kết nối tài khoản Dodo Payments của bạn bằng cách thêm khóa API của bạn.

    ### Cách lấy khóa API Dodo Payments của bạn

    - Đăng nhập vào **Bảng điều khiển Dodo Payments** của bạn
    - Đi đến phần **Nhà phát triển** trong menu bên trái
    - Nhấp vào **Khóa API**
    - Chọn **Thêm Khóa API**
    - Đặt tên cho nó (ví dụ: "Tích hợp Affonso")
    - Nhấp vào **Tạo** và **sao chép** khóa API của bạn

    ### Thêm khóa API của bạn vào Affonso

    - Mở **Bảng điều khiển Affonso qua** [affonso.io](https://affonso.io/)
    - Đi đến **Chi tiết Chương trình > Kết nối**
    - Nhấp vào thẻ **tích hợp Dodo Payments**
    - Dán khóa API Dodo Payments của bạn
    - Nhấp vào **Kết nối**

    <Callout type="info">
      Khóa API của bạn sẽ được mã hóa an toàn bởi Affonso.
    </Callout>
  </Step>

  <Step title="Thêm Webhook vào Dodo Payments">
    Khi đã kết nối, bạn sẽ nhận được một URL webhook duy nhất từ Affonso. Điều này cho phép theo dõi thanh toán và kích hoạt hoa hồng liên kết.

    ### Thêm URL Webhook trong Dodo Payments

    - Trong Bảng điều khiển Dodo Payments của bạn, đi đến **Nhà phát triển → Webhooks**
    - Nhấp vào **Thêm Webhook**
    - Dán **URL webhook Affonso** của bạn (từ cài đặt chương trình của bạn)
    - Lưu webhook

    ### Thêm bí mật Webhook trong Affonso

    - Sau khi lưu webhook trong Dodo Payments, **sao chép khóa ký** được cung cấp
    - Dán lại vào Affonso dưới **cài đặt webhook Dodo Payments**

    <Callout type="info">
      Điều này cho phép Affonso xác minh an toàn các sự kiện đến từ Dodo Payments.
    </Callout>
  </Step>

  <Step title="Thêm mã theo dõi">
    Đặt pixel Affonso trên trang web của bạn để bắt đầu theo dõi lượt truy cập và chuyển đổi từ liên kết.

    ```html
    <!-- Place in <head> tag -->
    <script async defer
    src="https://affonso.io/js/pixel.min.js"
    data-affonso="YOUR_PUBLIC_PROGRAM_ID"
    data-cookie_duration="YOUR_COOKIE_DURATION">
    </script>
    ```

    Mã này:
    - Theo dõi lượt truy cập liên kết
    - Đặt một cookie với ID giới thiệu (`affonso_referral`)
    - Kích hoạt theo dõi đăng ký và thanh toán

    <Callout type="info">
      Đặt mã này trên tất cả các miền mà người dùng truy cập hoặc chuyển đổi (ví dụ: trang tiếp thị, ứng dụng).
    </Callout>
  </Step>

  <Step title="Theo dõi đăng ký người dùng">
    Để có cái nhìn rõ ràng về quy trình và tối ưu hóa hiệu suất liên kết, hãy theo dõi đăng ký người dùng như là khách hàng tiềm năng.

    ```js
    // After successful registration
    window.Affonso.signup(userEmail);
    ```

    Điều này sẽ:
    - Ghi lại một giới thiệu mới như là **KHÁCH HÀNG TIỀM NĂNG** nếu người dùng đến qua liên kết liên kết
    - Hiển thị nó trong bảng điều khiển của bạn và thống kê của liên kết

    <Callout type="info">
      Thực hành tốt nhất: Kích hoạt điều này sau khi đăng ký thành công hoặc xác nhận kép (DOI).
    </Callout>
  </Step>

  <Step title="Chuyển dữ liệu giới thiệu đến Dodo Payments">
    Để phân bổ hoa hồng một cách chính xác, hãy chuyển ID giới thiệu từ Affonso đến Dodo Payments trong quá trình thanh toán.

    ### Tích hợp thanh toán

    ```js
    import DodoPayments from 'dodopayments';

    const referralId = window.affonso_referral;

    const client = new DodoPayments({
      bearerToken: process.env['DODO_PAYMENTS_API_KEY'],
    });

    async function main() {
      const payment = await client.payments.create({
        payment_link: true,
        billing: { city: 'city', country: 'AF', state: 'state', street: 'street', zipcode: 0 },
        customer: { email: 'email@email.com', name: 'name' },
        product_cart: [{ product_id: 'product_id', quantity: 0 }],
        metadata: {
          affonso_referral: referralId,
        },
      });

      console.log(payment.payment_id);
    }

    main();
    ```

    ### Tích hợp đăng ký

    ```js
    import DodoPayments from 'dodopayments';

    const referralId = window.affonso_referral;

    const client = new DodoPayments({
      bearerToken: process.env['DODO_PAYMENTS_API_KEY'],
    });

    async function main() {
      const subscription = await client.subscriptions.create({
        billing: { city: 'city', country: 'IN', state: 'state', street: 'street', zipcode: 89789 },
        customer: { customer_id: 'customer_id' },
        product_id: 'product_id',
        payment_link: true,
        return_url: 'https://example.com/success',
        quantity: 1,
        metadata: {
          affonso_referral: referralId,
        },
      });

      console.log(subscription.subscription_id);
    }

    main();
    ```

    ### Liên kết thanh toán

    ```js
    // Original Dodo Payments Link
    const originalLink = 'https://checkout.dodopayments.com/buy/<YOUR_PRODUCT_ID>';

    // Append referral ID as reference_id
    const linkWithReferral = `${originalLink}?metadata_affonso_referral=${window.affonso_referral}`;
    ```

    <Callout type="info">
      Để biết thêm chi tiết, xem [tài liệu API Dodo Payments](https://docs.dodopayments.com/api-reference/introduction)
    </Callout>
  </Step>

  <Step title="Kiểm tra tích hợp của bạn">
    Trước khi công khai, hãy kiểm tra thiết lập của bạn với các liên kết liên kết trực tiếp và các giao dịch thử nghiệm.

    **Danh sách kiểm tra thử nghiệm:**
    - ✅ Tạo một liên kết và tạo một liên kết theo dõi
    - ✅ Truy cập trang web của bạn qua liên kết liên kết
    - ✅ Đảm bảo mã Affonso được tải và cookie `affonso_referral` được thiết lập
    - ✅ Hoàn thành một đăng ký (tùy chọn)
    - ✅ Hoàn thành một thanh toán qua Dodo Payments
    - ✅ Xác nhận theo dõi và phân bổ trong bảng điều khiển Affonso của bạn
  </Step>
</Steps>

## Tích hợp Dub Partners

Bạn cũng có thể sử dụng [Dub Partners](https://dub.co) để theo dõi các giới thiệu và chuyển đổi liên kết. Tính năng theo dõi chuyển đổi của Dub cho phép bạn phân bổ doanh số cho các liên kết liên kết của bạn, cung cấp phân tích chi tiết về các đối tác nào tạo ra nhiều doanh thu nhất.

<Info>
Để biết hướng dẫn thiết lập chi tiết, xem [hướng dẫn tích hợp Dub](/integrations/dub) của chúng tôi, hướng dẫn cách theo dõi các sự kiện chuyển đổi bán hàng và phân bổ doanh thu cho các liên kết liên kết của bạn.
</Info>

### Thiết lập nhanh với Dub Partners

1. **Kích hoạt theo dõi chuyển đổi** trong bảng điều khiển Dub của bạn cho các liên kết liên kết
2. **Ghi lại ID nhấp chuột Dub** (`dub_id` cookie) khi tạo các phiên thanh toán
3. **Lưu ID nhấp chuột** trong siêu dữ liệu thanh toán cùng với thông tin khách hàng
4. **Cấu hình tích hợp webhook** để gửi dữ liệu bán hàng đến Dub khi thanh toán thành công

Tích hợp tự động theo dõi:
- Thanh toán một lần
- Kích hoạt đăng ký
- Thanh toán đăng ký định kỳ
