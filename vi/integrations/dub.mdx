## Giới thiệu

[Dub](https://dub.co) là một nền tảng quản lý liên kết mạnh mẽ giúp bạn tạo, chia sẻ và theo dõi các liên kết ngắn. Bằng cách tích hợp Dodo Payments với Dub, bạn có thể tự động theo dõi các sự kiện chuyển đổi bán hàng khi khách hàng hoàn tất mua hàng, cho phép bạn đo lường ROI của các chiến dịch tiếp thị và chương trình giới thiệu của mình.

Một sự kiện "bán hàng" được ghi lại trong Dub khi một khách hàng:
- Hoàn tất một khoản thanh toán một lần
- Đăng ký một gói trả phí
- Thực hiện một khoản thanh toán đăng ký định kỳ

<Info>
Tích hợp này yêu cầu tài khoản Dub với theo dõi chuyển đổi được bật trên các liên kết của bạn.
</Info>

<Tip>
**Tích hợp Chương trình Cộng tác viên**: Tích hợp này cũng hoạt động trơn tru với **Dub Partners** để theo dõi giới thiệu và hoa hồng cho cộng tác viên. Sử dụng theo dõi chuyển đổi của Dub để quy doanh số cho các liên kết cộng tác viên của bạn và đo hiệu suất đối tác. Tìm hiểu thêm về cách thiết lập chương trình cộng tác viên trong [Hướng dẫn tính năng Cộng tác viên](/features/affiliates).
</Tip>

Dub theo dõi khách truy cập thông qua một ID nhấp chuột duy nhất (`dub_id`) được lưu trữ trong cookie khi người dùng nhấp vào các liên kết ngắn của bạn. Để gán doanh thu cho các liên kết của bạn, bạn cần:

Dub theo dõi khách truy cập thông qua một ID nhấp chuột duy nhất (`dub_id`) được lưu trong cookie khi người dùng nhấp vào liên kết ngắn của bạn trên Dub. Để quy doanh số cho các liên kết của bạn, bạn cần:

1. **Thu thập ID nhấp chuột của Dub** từ cookie `dub_id` khi tạo các phiên thanh toán
2. **Lưu ID nhấp chuột** trong metadata thanh toán của bạn cùng với ID bên ngoài của khách hàng
3. **Gửi dữ liệu doanh số cho Dub** khi thanh toán thành công bằng API Track của họ

## Điều kiện tiên quyết

Trước khi thiết lập tích hợp này, hãy đảm bảo bạn đã có:

1. Một [tài khoản Dub](https://dub.co) với một không gian làm việc
2. Theo dõi chuyển đổi được kích hoạt cho các liên kết của bạn
3. Khóa API Dub của bạn (có sẵn trong bảng điều khiển Dub của bạn dưới Cài đặt → Khóa API)

Theo dõi doanh thu khi các khoản thanh toán thành công:

```javascript basic_sale.js icon="js" expandable
function handler(webhook) {
  if (webhook.eventType === "payment.succeeded") {
    const payment = webhook.payload.data;

    // Only send to Dub if click ID exists in metadata
    if (payment.metadata && payment.metadata.dub_click_id) {
      webhook.payload = {
        clickId: payment.metadata.dub_click_id,
        externalId: payment.metadata.dub_external_id || payment.customer.customer_id,
        amount: payment.total_amount, // Ensure the amount is in cents
        currency: payment.currency || "USD",
        paymentProcessor: "dodo",
        invoiceId: payment.payment_id,
        metadata: {
          customer_email: payment.customer.email,
          customer_name: payment.customer.name,
          product_id: payment.product_cart ? payment.product_cart.map(product => product.product_id).join(', ') : undefined,
        },
      };
    } else {
      // Cancel dispatch if no click ID (organic traffic)
      webhook.cancel = true;
    }
  }
  return webhook;
}
```

```javascript basic_sale.js icon="js" expandable
function handler(webhook) {
```
```javascript subscription_sale.js icon="js" expandable
  const data = webhook.payload.data;

  // Track initial subscription activation
  if (webhook.eventType === "subscription.active") {
    if (data.metadata && data.metadata.dub_click_id) {
      webhook.payload = {
        clickId: data.metadata.dub_click_id,

          product_id: data.product_id,

          customer_email: data.customer.email,

      // Cancel dispatch if no click ID (organic traffic)
```

```typescript Next.js (Server Action)
'use server';

import { Dub } from 'dub';

const dub = new Dub();

export async function trackSale(

  currency: string
) {

    amount: amount,
    currency: currency,
    paymentProcessor: 'dodo',
    invoiceId: paymentId,
  });
}
```
