<Info>
  **Điều kiện tiên quyết**: Vui lòng cài đặt Node.js (phiên bản 19 hoặc cao hơn) trước khi tiếp tục.
</Info>

Thực hiện các bước sau để cài đặt và chạy Mintlify trên hệ điều hành của bạn:

**Bước 1**: Cài đặt Mintlify:

<CodeGroup>

  ```bash npm
  npm i -g mintlify
  ```

```bash yarn
yarn global add mintlify
```

</CodeGroup>

**Bước 2**: Điều hướng đến thư mục docs (nơi chứa tệp `mint.json`) và thực hiện lệnh sau:

```bash
mintlify dev
```

Một bản xem trước cục bộ của tài liệu của bạn sẽ có sẵn tại `http://localhost:3000`.

### Cổng tùy chỉnh

Theo mặc định, Mintlify sử dụng cổng 3000. Bạn có thể tùy chỉnh cổng mà Mintlify chạy bằng cách sử dụng cờ `--port`. Để chạy Mintlify trên cổng 3333, ví dụ, hãy sử dụng lệnh này:

```bash
mintlify dev --port 3333
```

Nếu bạn cố gắng chạy Mintlify trên một cổng đã được sử dụng, nó sẽ sử dụng cổng tiếp theo có sẵn:

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Các phiên bản Mintlify

Vui lòng lưu ý rằng mỗi phiên bản CLI được liên kết với một phiên bản cụ thể của Mintlify. Nếu trang web cục bộ của bạn không khớp với phiên bản sản xuất, vui lòng cập nhật CLI:

<CodeGroup>

```bash npm
npm i -g mintlify@latest
```

```bash yarn
yarn global upgrade mintlify
```

</CodeGroup>

## Xác thực liên kết

CLI có thể hỗ trợ xác thực các liên kết tham chiếu được thực hiện trong tài liệu của bạn. Để xác định bất kỳ liên kết hỏng nào, hãy sử dụng lệnh sau:

```bash
mintlify broken-links
```

## Định dạng mã

Chúng tôi đề xuất sử dụng các tiện ích mở rộng trên IDE của bạn để nhận diện và định dạng MDX. Nếu bạn là người dùng VSCode, hãy xem xét tiện ích mở rộng [MDX VSCode](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) để tô sáng cú pháp, và [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) để định dạng mã.

## Khắc phục sự cố

<AccordionGroup>
  <Accordion title='Lỗi: Không thể tải mô-đun "sharp" bằng cách sử dụng runtime darwin-arm64'>

    Điều này có thể do phiên bản node đã lỗi thời. Hãy thử các bước sau:
    1. Gỡ cài đặt phiên bản mintlify hiện tại: `npm remove -g mintlify`
    2. Nâng cấp lên Node v19 hoặc cao hơn.
    3. Cài đặt lại mintlify: `npm install -g mintlify`
  </Accordion>

  <Accordion title="Vấn đề: Gặp lỗi không xác định">
  
    Giải pháp: Đi đến thư mục gốc của thiết bị của bạn và xóa thư mục \~/.mintlify. Sau đó, chạy lại `mintlify dev`.
  </Accordion>
</AccordionGroup>
