## Sự kiện Webhook Đăng Ký

Các sự kiện webhook sau đây có sẵn để theo dõi các thay đổi trong vòng đời đăng ký:

| Sự kiện | Mô tả |
|-------|-------------|
| `subscription.active` | Đăng ký đã được kích hoạt thành công |
| `subscription.updated` | Đối tượng đăng ký đã được cập nhật (bất kỳ thay đổi nào về trường đều kích hoạt sự kiện này) |
| `subscription.on_hold` | Đăng ký bị tạm hoãn do gia hạn không thành công |
| `subscription.renewed` | Đăng ký được gia hạn cho kỳ thanh toán tiếp theo |
| `subscription.plan_changed` | Kế hoạch đăng ký đã được nâng cấp, hạ cấp hoặc sửa đổi |
| `subscription.cancelled` | Đăng ký đã bị hủy |
| `subscription.failed` | Việc tạo đăng ký đã thất bại trong quá trình tạo ủy quyền |
| `subscription.expired` | Đăng ký đã đạt đến cuối kỳ hạn |

### Sử Dụng `subscription.updated` cho Đồng Bộ Thời Gian Thực

Webhook `subscription.updated` sẽ được kích hoạt bất cứ khi nào có bất kỳ trường đăng ký nào thay đổi, cho phép bạn giữ trạng thái ứng dụng của mình đồng bộ mà không cần polling:

```javascript
app.post('/webhooks/dodo', async (req, res) => {
  const event = req.body;
  
  if (event.type === 'subscription.updated') {
    const subscription = event.data;
    
    // Sync subscription changes to your database
    await syncSubscription(subscription.subscription_id, {
      status: subscription.status,
      next_billing_date: subscription.next_billing_date,
      metadata: subscription.metadata,
      // ... other fields you want to track
    });
    
    console.log(`Subscription ${subscription.subscription_id} updated`);
  }
  
  res.json({ received: true });
});
```

<Tip>
Đăng ký nhận thông báo thời gian thực về bất kỳ thay đổi nào của đăng ký bằng cách sử dụng `subscription.updated`, loại bỏ nhu cầu polling API để cập nhật.
</Tip>
