## URL Môi Trường

Sử dụng URL cơ sở phù hợp cho môi trường của bạn:

- Chế độ Kiểm Tra: [`https://test.dodopayments.com`](https://test.dodopayments.com)
- Chế độ Trực Tiếp: [`https://live.dodopayments.com`](https://live.dodopayments.com)

Tìm hiểu thêm về [Chế độ Kiểm Tra so với Chế độ Trực Tiếp](/miscellaneous/test-mode-vs-live-mode).

## Các Bước Tích Hợp

Thực hiện các bước sau để tích hợp Dodo Payments:

<Steps>
  1. [Thiết lập tài khoản của bạn](/quickstart) và nhận khóa API của bạn
  2. Chọn phương pháp tích hợp của bạn:
     - [Theo dõi hướng dẫn của chúng tôi](/developer-resources/integration-tutorial)
     - [Sử dụng SDK của chúng tôi](/developer-resources/dodo-payments-sdks)
     - [Tích hợp API trực tiếp](/developer-resources/integration-guide)
  3. [Thiết lập webhooks](/developer-resources/webhooks/intents/webhook-events-guide) để nhận cập nhật theo thời gian thực
  4. [Kiểm tra tích hợp của bạn](/miscellaneous/testing-process)
  5. Đi vào hoạt động!
</Steps>
