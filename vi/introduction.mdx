## Dodo Payments là gì?

Dodo Payments là **công cụ tất cả trong một** để khởi động, mở rộng và kiếm tiền trên toàn cầu. Được thiết kế cho thế hệ sản phẩm kỹ thuật số SaaS và AI mới, chúng tôi xử lý mọi thứ từ **thanh toán đến lập hóa đơn đến phân phối** - tất cả mà không cần thêm kỹ thuật.

<Info>
**Xây dựng cho SaaS và các sản phẩm AI**: Xử lý thanh toán một lần, đăng ký, tín dụng, chỗ ngồi, token hoặc bất kỳ chỉ số sử dụng nào với cơ sở hạ tầng lập hóa đơn được tối ưu hóa của chúng tôi.
</Info>

<Tip>
  <strong>Gặp Sentra: Trợ lý lập hóa đơn được AI hỗ trợ</strong><br/>
  Sentra là một tác nhân AI tích hợp, phân tích và hành động dựa trên hóa đơn và thanh toán ngay trong IDE của bạn. Hỏi câu hỏi, xây dựng tích hợp hoặc lên kế hoạch kiến trúc lập hóa đơn — tất cả chỉ bằng cách mô tả những gì bạn muốn.
  <a href="/developer-resources/sentra">
    Bắt đầu với Sentra &rarr;
  </a>
</Tip>

### Máy chủ MCP cho Trợ lý AI

Kết nối trợ lý lập trình AI của bạn (Cursor, Claude Desktop, Windsurf, VS Code) trực tiếp với Dodo Payments thông qua các máy chủ MCP của chúng tôi.

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  Tìm kiếm ngữ nghĩa trên toàn bộ tài liệu Dodo Payments — nhận câu trả lời nhanh về tính năng, API và các thực hành tốt nhất
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  Thực thi hoạt động API trực tiếp — tạo thanh toán, quản lý đăng ký và xử lý hoàn tiền thông qua trợ lý AI của bạn
</Card>
</CardGroup>

{/* LOCKED_PATTERN_8ee343b87e057252cc4a8a376b190970 */}
Thêm cả hai máy chủ vào cấu hình MCP của bạn (`~/.cursor/mcp.json` hoặc cấu hình Claude Desktop):

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
Yêu cầu Node.js 18+. Xem [tài liệu MCP Server](/developer-resources/mcp-server) để biết hướng dẫn thiết lập chi tiết cho tất cả các khách hàng được hỗ trợ.
</Info>
</Accordion>
