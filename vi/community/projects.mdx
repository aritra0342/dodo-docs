      <br/>Vi dịch vụ thanh toán sẵn sàng cho môi trường sản xuất được xây dựng bằng FastAPI, xử lý thanh toán cho nhiều ứng dụng sử dụng Dodo Payments. 
      <br/>Các tính năng bao gồm <br/> `credit management`, `subscription support`, `webhook processing with idempotency`, `complete audit trails`, `MixPanel Analytics`, `Backend-to-backend architecture with API key authentication`.
      <br/><br/>Bởi <a href="https://github.com/aniketkarne">@aniketkarne</a> (Aniket Karne)
