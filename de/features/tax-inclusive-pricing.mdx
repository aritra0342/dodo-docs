## Was ist preisgestaltung inklusive Steuern?

Die Preisgestaltung inklusive Steuern ermöglicht es Ihnen, die anwendbaren Steuern direkt in den angezeigten Produktpreis einzubetten. Kunden sehen einen endgültigen Betrag im Voraus für ein transparenteres, konversionsfreundliches Erlebnis. An der Kasse berechnet Dodo den steuerfreien Anteil und den Steuerbetrag aus dem angezeigten Preis basierend auf dem Standort des Kunden und bricht diese klar auf der Rechnung auf.

<Info>
Die endgültige Steuer variiert weiterhin je nach Kundenstandort und Steuerkategorie. Kasse und Rechnungen zeigen den berechneten Steueranteil, der sich aus dem angezeigten Preis ergibt.
</Info>

## Hauptvorteile

- **Klarheit für Kunden**: Ein endgültiger Preis wird im Voraus angezeigt
- **Einhaltung nach Region**: Steuer wird pro Standort berechnet und aufgeschlüsselt
- **Genauere Rechnungsstellung**: Rechnungen zeigen Nettobetrag und Steuer, die aus dem Preis abgeleitet sind
- **Konversionsfreundlich**: Weniger Überraschungen an der Kasse

## Produktsetup

Erstellen oder bearbeiten Sie ein Produkt in Ihrem Dashboard und aktivieren Sie die steuerlich inklusive Preisgestaltung unter den Preiseinstellungen.

#### Preise

- **Preis** (erforderlich): Basispreis, der dem Kunden angezeigt wird.
- **Preisgestaltung inklusive Steuern**: Aktivieren Sie den Schalter, um den angezeigten Preis als steuerlich inklusive zu behandeln. Dodo berechnet den Nettobetrag (vor Steuern) und den Steueranteil an der Kasse basierend auf dem Standort des Kunden und der Steuerkategorie.

<Warning>
Nur zukünftige Käufe spiegeln die aktualisierten steuerinklusive Einstellungen wider. Bestehende Käufe und aktive Abonnements werden nicht rückwirkend geändert.
</Warning>

<Tip>
Verwenden Sie konsistente Produkttexte wie “Tax inclusive”, damit Kunden die Gesamtsumme auf einen Blick verstehen.
</Tip>

## Integrationsbeispiele

### Erstellen Sie ein Produkt mit steuerlich inklusiver Preisgestaltung

```typescript
const product = await client.products.create({
  name: 'Pro Plan',
  description: 'All features included',
  price: {
    currency: 'USD',
    price: 10000
    tax_inclusive: true
  }
});
```

### Aktualisieren Sie ein bestehendes Produkt, um die Steuer inklusive zu aktivieren

```typescript
await client.products.update(product.id, {
  price: {
    tax_inclusive: true
  }
});
```

## Best Practices

- **Wählen Sie die richtige Steuerkategorie** aus, um eine korrekte Berechnung pro Region sicherzustellen.
- **Kommunizieren Sie die Gesamtsummen** klar mit „Steuer inklusive“-Labels im Produkttext.
- **Testen Sie im Sandbox** zur Validierung.
