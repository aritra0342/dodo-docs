Wir verwenden [Featurebase](https://dodopayments.featurebase.app/), um Funktionsanfragen zu verwalten und unsere öffentliche Roadmap zu pflegen. Diese Plattform ermöglicht es Ihnen, Ideen einzureichen, für Funktionen abzustimmen, die Sie sehen möchten, und den Entwicklungsstatus bevorstehender Verbesserungen zu verfolgen.

<Card title="Besuchen Sie unsere öffentliche Roadmap" icon="external-link" href="https://dodopayments.featurebase.app/">
Sehen Sie sich alle Funktionsanfragen an, stimmen Sie für Ideen ab und verfolgen Sie unsere Produkt-Roadmap auf Featurebase.
</Card>

## **Fordern Sie eine neue Funktion an**

Wenn Sie eine Idee für eine Funktion haben, die Ihr Erlebnis mit Dodo Payments verbessern würde, können Sie eine Funktionsanfrage direkt auf Featurebase einreichen.

<img src="/images/feature-request/Feature.png" alt="Funktionsanfrage"/>

- **So fordern Sie eine Funktion an**:
    1. Besuchen Sie unser [Featurebase-Portal](https://dodopayments.featurebase.app/).
    2. Klicken Sie auf **"Eine Idee posten"** oder **"Eine Funktionsanfrage einreichen"**.
    3. Geben Sie die folgenden Details an:
        - **Funktionsname**: Ein prägnanter Titel für Ihre Funktionsidee.
        - **Funktionsbeschreibung**: Eine kurze Erklärung der Funktion, einschließlich warum sie wichtig ist und wie sie Ihrem Unternehmen helfen wird.
    4. Reichen Sie die Anfrage ein.
- **Tipps für effektive Anfragen**:
    - Seien Sie klar und prägnant.
    - Heben Sie die geschäftlichen Auswirkungen der Funktion hervor.
    - Geben Sie Beispiele an, wenn möglich.
    - Fügen Sie Anwendungsfälle hinzu, die den Wert demonstrieren.

## **Für bestehende Funktionsanfragen abstimmen**

Wir priorisieren Funktionen, die den meisten Händlern zugutekommen. Sie können bestehende Funktionsanfragen auf Featurebase durchsuchen und für die abstimmen, die Sie wertvoll finden.

- **So stimmen Sie ab**:
    1. Besuchen Sie unser [Featurebase-Portal](https://dodopayments.featurebase.app/).
    2. Durchsuchen Sie die Liste der eingereichten Anfragen.
    3. Klicken Sie auf die **Upvote**-Schaltfläche (👍) neben der/dem Funktion(en), die Sie unterstützen.
- **Vorteile des Abstimmens**:
    - Beeinflussen Sie die Entwicklungspriorität für Funktionen, die Ihnen wichtig sind.
    - Helfen Sie unserem Team, Funktionen zu identifizieren, die eine breite Nachfrage haben.
    - Erhalten Sie Benachrichtigungen, wenn Funktionen, für die Sie abgestimmt haben, veröffentlicht werden.

## **Die Produkt-Roadmap einsehen**

Bleiben Sie über den Status der Funktionsentwicklung durch unsere öffentliche Roadmap auf Featurebase informiert.

- **Roadmap-Funktionen**:
    - **Geplant**: Funktionen, die genehmigt wurden und für die Entwicklung eingeplant sind.
    - **In Arbeit**: Funktionen, an denen unser Team derzeit arbeitet.
    - **Abgeschlossen**: Funktionen, die erfolgreich implementiert und live sind.
- **So greifen Sie zu**:
    - Besuchen Sie unser [Featurebase-Portal](https://dodopayments.featurebase.app/).
    - Navigieren Sie zum Abschnitt **Roadmap**, um Funktionen nach Status anzuzeigen.
    - Durchsuchen Sie die Funktionskategorien und deren aktuellen Status.
    - Filtern Sie nach Status, Kategorie oder suchen Sie nach bestimmten Funktionen.

<Tip>
Legen Sie unser [Featurebase-Portal](https://dodopayments.featurebase.app/) als Lesezeichen ab, um über neue Funktionen informiert zu bleiben und für Ideen abzustimmen, die Ihnen wichtig sind.
</Tip>
