## Adaptive Währung

Adaptive Währung ermöglicht es Kunden, in ihrer bevorzugten lokalen Währung zu bezahlen, was Vertrauen und Konversionsraten verbessert.

### So funktioniert es

1. **Aktivieren**: Aktivieren Sie die Adaptive Währung in Ihren Geschäftseinstellungen
2. **Auswählen**: Kunden können Währungen direkt beim Checkout wechseln
3. **Konvertieren**: Preise werden dynamisch mit Echtzeit-Wechselkursen umgerechnet
4. **Anzeigen**: Der endgültige zu zahlende Betrag wird transparent vor der Zahlung angezeigt

## Automatische Steuerberechnung

Steuern werden automatisch basierend auf dem Rechnungsstandort des Kunden berechnet, um die Einhaltung von GST-, VAT- und Umsatzsteueranforderungen ohne manuelle Einrichtung sicherzustellen.

### So funktioniert die Steuerberechnung

<Steps>
<Step title="Standorterkennung">
  Steuerregeln werden basierend auf dem Land des Kunden (und der Region, wo zutreffend) angewendet.
</Step>

<Step title="Dynamische Updates">
  Der Steuerbetrag wird automatisch aktualisiert, wenn:
  - Das Land wechselt
  - Die Adresse aktualisiert wird
</Step>

<Step title="Transparente Anzeige">
  Die endgültige Steueraufstellung wird klar vor der Zahlung angezeigt.
</Step>
</Steps>

<Tip>
Die Steuerberechnung ist vollständig automatisiert. Keine manuelle Konfiguration für Standard-Digitalgüter und SaaS-Produkte erforderlich.
</Tip>

## Unterstützung der Unternehmenssteuer-ID

Für registrierte Unternehmen ermöglicht der Checkout den Kunden, ihre Unternehmenssteuer-ID (z. B. VAT/GST-Nummer) einzugeben.

### Was passiert, wenn eine Steuer-ID eingegeben wird

- Die Steuerberechtigung wird in Echtzeit validiert
- Anwendbare Steuerbefreiungen oder Umkehrsteuerregeln werden angewendet
- Der Steuerbetrag wird sofort beim Checkout aktualisiert

</Info>

## Rabattcodes

Kunden können Rabatt- oder Aktionscodes, die Sie im Dashboard erstellt haben, direkt auf der Checkout-Seite anwenden.

### Checkout-Erlebnis

1. Der Kunde gibt den Rabattcode ein
2. Der Rabatt wird sofort validiert
3. Der aktualisierte Preis und die Ersparnisse werden klar angezeigt

<Frame>
  <img src="/images/checkout/c4.png" alt="Eingabe des Rabattcodes beim Checkout" style={{ width: '70%' }}/>
</Frame>

### API-Integration

Wenden Sie einen Rabattcode vorab an oder aktivieren Sie das Eingabefeld für Rabatte:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  discount_code: 'WELCOME20', // Pre-apply a code
  feature_flags: {
    allow_discount_code: true // Show discount input field
  },
  return_url: 'https://yoursite.com/return'
});
```

## Intelligente Adressensammlung

Der Checkout unterstützt eine flexible Adresseneingabe für eine schnellere Abwicklung.

### Verfügbare Optionen

| Option | Beschreibung |
|--------|-------------|
| **Google Adress-Autovervollständigung** | Schnelle Auswahl mit Autovervollständigung |
| **Manuelle Eingabe** | Volle Kontrolle für vollständige Adressen |
| **Länderauswahl** | Steuert Steuer- und Compliance-Logik |

<Tip>
Die Adressensammlung balanciert Geschwindigkeit, Genauigkeit und globale Abdeckung, um die Konversion zu maximieren und gleichzeitig die Einhaltung sicherzustellen.
</Tip>

## Konfiguration der Checkout-Sitzung

Steuern Sie das Checkout-Verhalten mit der Checkout Sessions API:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  customer: {
    email: 'customer@example.com',
    name: 'Jane Doe'
  },
  billing_currency: 'EUR', // Set specific currency
  discount_code: 'PROMO10',
  feature_flags: {
    allow_discount_code: true
  },
  return_url: 'https://yoursite.com/return',
  metadata: {
    order_ref: 'ORD-12345'
  }
});
```
