## Einführung

[ParityDeals](https://www.paritydeals.com/) ist eine Plattform, die eine automatische Preisoptimierung basierend auf dem geografischen Standort eines Kunden ermöglicht, allgemein bekannt als "Kaufkraftparität"-Preisgestaltung.

Mit ParityDeals können Sie auch [automatische zeitbasierte Rabatte](https://www.paritydeals.com/features/time-based-discounts/) und [feiertagsbasierte Aktionen](https://www.paritydeals.com/features/holiday-based-discounts/) im Laufe des Jahres anbieten.

<Info>
Auch wenn Sie das ParityDeals-Banner nicht direkt in einen Dodo Payments-Shop einbetten können (da benutzerdefinierte Skripte nicht unterstützt werden), können Sie ParityDeals dennoch effektiv auf Ihrer eigenen Website oder Anwendung implementieren.
</Info>

## So funktioniert es

Wenn ParityDeals in Ihre Website integriert ist, sehen berechtigte Besucher ein Banner, das einen standortbasierten Rabattcode enthält, den sie beim Checkout verwenden können.

<Frame>
  <img src="/images/ppp/1.png" alt="ParityDeals discount banner example" />
</Frame>

Diese Rabattcodes werden automatisch von ParityDeals erstellt und verwaltet und erscheinen in Ihrem **Dodo Payments Dashboard** unter **Verkäufe → Rabatte**.

## Schritt-für-Schritt-Setup-Anleitung

### 1. Registrieren Sie sich bei ParityDeals

Besuchen Sie [app.paritydeals.com](https://app.paritydeals.com/) und erstellen Sie ein Konto.

### 2. Erstellen Sie Angebote

Nach dem Anmelden finden Sie die Option, Angebote zu erstellen. Wählen Sie im Dropdown-Menü:

**→ "Angebote über Dodo Payments erstellen"**

<Frame>
  <img src="/images/ppp/2.png" alt="Erstellen von Deals über Dodo Payments" />
</Frame>

### 3. Generieren Sie einen Dodo Payments API-Schlüssel

Melden Sie sich bei Ihrem **Dodo Payments Dashboard** an und navigieren Sie zu:

**Entwickler → API-Schlüssel**

Klicken Sie auf **"Neuen API-Schlüssel erstellen"** und benennen Sie ihn beschreibend, z. B. "ParityDeals".

<Frame>
  <img src="/images/ppp/3.png" alt="Erstellen eines neuen API-Schlüssels in Dodo Payments" />
</Frame>

### 4. Verbinden Sie Ihren API-Schlüssel mit ParityDeals

Kopieren Sie den API-Schlüssel, den Sie erstellt haben, und fügen Sie ihn in die ParityDeals-Oberfläche ein, um die Verbindung zu Ihrem Dodo Payments-Konto herzustellen.

<Frame>
  <img src="/images/ppp/4.png" alt="Verbinden des API-Schlüssels mit ParityDeals" />
</Frame>

### 5. Wählen Sie Produkte aus

Sobald die Verbindung hergestellt ist, wird ParityDeals Ihren Dodo Payments-Produktkatalog abrufen. Wählen Sie die Produkte aus, die Sie mit Ihren Angeboten verknüpfen möchten.

<Frame>
  <img src="/images/ppp/5.png" alt="Auswahl von Produkten für ParityDeals" />
</Frame>

### 6. Konfigurieren Sie die Angebotsoptionen

Geben Sie die URL Ihrer Website ein und wählen Sie optional spezifische Zielseiten aus, auf denen das Rabattbanner angezeigt werden soll.

Wählen Sie dann Ihren bevorzugten Angebotstyp:

- Standortbasierte Angebote
- Feiertagsangebote
- Zeitbasierte Blitzangebote

<Frame>
  <img src="/images/ppp/6.png" alt="Konfigurieren der Deal-Einstellungen" />
</Frame>

### 7. Legen Sie länderbasierte Regeln fest

Wählen Sie die Länder aus, für die Sie lokalisierte Rabatte anbieten möchten, und definieren Sie den Rabattprozentsatz für jedes. ParityDeals bietet empfohlene Standardwerte, die Sie nach Bedarf anpassen können.

<Frame>
  <img src="/images/ppp/7.png" alt="Festlegen länderspezifischer Rabattregeln" />
</Frame>

### 8. Passen Sie das Banner an

Gestalten Sie Ihr Rabattbanner, indem Sie:

- Den Text anpassen
- Farben und Layout auswählen
- Die Bannerposition (oben oder unten auf der Seite) wählen

Dieses Banner wird basierend auf Ihren konfigurierten Regeln den berechtigten Besuchern angezeigt.

<Frame>
  <img src="/images/ppp/8.png" alt="Anpassen des Rabattbanners" />
</Frame>

### 9. Fügen Sie das ParityDeals-Skript zu Ihrer Website hinzu

Nachdem Sie Ihren Deal gespeichert haben, generiert ParityDeals ein Skript. Fügen Sie dieses Skript in Ihren Website-Code ein (typischerweise im Abschnitt `<head>`).

Dieses Skript erkennt automatisch die Standorte der Besucher und zeigt das entsprechende Rabattbanner an.

<Frame>
  <img src="/images/ppp/9.png" alt="ParityDeals-Integrationsskript" />
</Frame>

## Alles bereit!

Sie sind jetzt bereit, anzubieten:

- **Standortbasierte Angebote** - Preise basierend auf der Kaufkraft anpassen
- **Countdown-/zeitlich begrenzte Rabatte** - Dringlichkeit mit zeitlich begrenzten Angeboten schaffen
- **Feiertagsaktionen** - Von saisonalen Einkaufstrends profitieren

All dies wird automatisiert über ParityDeals und funktioniert nahtlos mit Dodo Payments.

Für detailliertere Informationen konsultieren Sie die [ParityDeals-Dokumentation](https://www.paritydeals.com/docs/) oder kontaktieren Sie deren Support-Team.
