<Info>
  **Voraussetzung**: Bitte installieren Sie Node.js (Version 19 oder höher), bevor Sie fortfahren.
</Info>

Befolgen Sie diese Schritte, um Mintlify auf Ihrem Betriebssystem zu installieren und auszuführen:

**Schritt 1**: Installieren Sie Mintlify:

<CodeGroup>

  ```bash npm
  npm i -g mintlify
  ```

```bash yarn
yarn global add mintlify
```

</CodeGroup>

**Schritt 2**: Navigieren Sie zum Verzeichnis der Dokumentation (wo sich die `mint.json` Datei befindet) und führen Sie den folgenden Befehl aus:

```bash
mintlify dev
```

Eine lokale Vorschau Ihrer Dokumentation ist unter `http://localhost:3000` verfügbar.

### Benutzerdefinierte Ports

Standardmäßig verwendet Mintlify den Port 3000. Sie können den Port, auf dem Mintlify läuft, anpassen, indem Sie das `--port` Flag verwenden. Um Mintlify beispielsweise auf Port 3333 auszuführen, verwenden Sie diesen Befehl:

```bash
mintlify dev --port 3333
```

Wenn Sie versuchen, Mintlify auf einem bereits verwendeten Port auszuführen, wird der nächste verfügbare Port verwendet:

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Mintlify-Versionen

Bitte beachten Sie, dass jede CLI-Version mit einer bestimmten Version von Mintlify verknüpft ist. Wenn Ihre lokale Website nicht mit der Produktionsversion übereinstimmt, aktualisieren Sie bitte die CLI:

<CodeGroup>

```bash npm
npm i -g mintlify@latest
```

```bash yarn
yarn global upgrade mintlify
```

</CodeGroup>

## Links validieren

Die CLI kann bei der Validierung von Referenzlinks in Ihrer Dokumentation helfen. Um defekte Links zu identifizieren, verwenden Sie den folgenden Befehl:

```bash
mintlify broken-links
```

## Fehlersuche

<AccordionGroup>
  <Accordion title='Fehler: Das "sharp"-Modul konnte nicht mit der darwin-arm64-Laufzeit geladen werden'>

    Dies kann an einer veralteten Version von Node liegen. Versuchen Sie Folgendes:
    1. Entfernen Sie die derzeit installierte Version von mintlify: `npm remove -g mintlify`
    2. Aktualisieren Sie auf Node v19 oder höher.
    3. Installieren Sie mintlify erneut: `npm install -g mintlify`
  </Accordion>

  <Accordion title="Problem: Unbekannter Fehler aufgetreten">
  
    Lösung: Gehen Sie zum Stammverzeichnis Ihres Geräts und löschen Sie den \~/.mintlify-Ordner. Führen Sie danach `mintlify dev` erneut aus.
  </Accordion>
</AccordionGroup>
