## Übersicht

Agent Skills sind wiederverwendbare Funktionen, die die Möglichkeiten von KI-Coding-Assistenten bei der Arbeit mit Dodo Payments erweitern. Sie bieten prozedurales Wissen und Best Practices, die den Agenten helfen, spezifische Aufgaben effektiver zu erledigen – von der Integration von Checkout-Flows bis hin zur Einrichtung nutzungsbasierter Abrechnung.

<Info>
**Betrachte Skills als Plugins für deinen KI-Assistenten.** Sie lehren deinen Agenten, wie Dodo Payments-Funktionen korrekt implementiert werden, unter Einhaltung unserer empfohlenen Muster und Best Practices.
</Info>

Skills funktionieren mit jedem MCP-kompatiblen KI-Agenten, einschließlich:
- **Claude Code** - Anthropics CLI-Coding-Assistent
- **OpenCode** - Open Source KI-Coding-Agent
- **Cursor** - KI-first Code-Editor
- **Andere MCP-Clients** - Jeder Agent, der das Skills-Protokoll unterstützt

## Installation

Wählen Sie Ihre bevorzugte Installationsmethode basierend auf Ihrem KI-Coding-Assistenten.

<Tabs>
<Tab title="Skills CLI">
Installiere Skills mithilfe der universellen Skills-CLI, die mit jedem MCP-kompatiblen Agenten funktioniert:

```bash
# Install all Dodo Payments skills
npx skills add dodopayments/skills
```

Oder installieren Sie individuelle Skills nach Bedarf:

```bash
# Best practices guide
npx skills add dodopayments/skills/dodo-best-practices

# Webhook integration
npx skills add dodopayments/skills/webhook-integration

# Subscription billing
npx skills add dodopayments/skills/subscription-integration

# Checkout flows
npx skills add dodopayments/skills/checkout-integration

# Usage-based billing
npx skills add dodopayments/skills/usage-based-billing

# BillingSDK components
npx skills add dodopayments/skills/billing-sdk

# License key management
npx skills add dodopayments/skills/license-keys
```

<Tip>
Installiere nur die Skills, die du benötigst, um deinen Agenten fokussiert zu halten und den Kontext-Aufwand zu verringern.
</Tip>
</Tab>

<Tab title="Claude Code">
Füge den Dodo Payments Skills-Marktplatz zu Claude Code hinzu:

```bash
/plugin marketplace add dodopayments/skills
```

Installieren Sie dann individuelle Plugins:

```bash
# Install best practices
/plugin install dodo-best-practices

# Install webhook integration
/plugin install webhook-integration

# Install subscription integration
/plugin install subscription-integration

# Install checkout integration
/plugin install checkout-integration

# Install usage-based billing
/plugin install usage-based-billing

# Install BillingSDK
/plugin install billing-sdk

# Install license keys
/plugin install license-keys
```

</Tab>

<Tab title="OpenCode">
Skills sind automatisch verfügbar, wenn sie in deinen OpenCode-Einstellungen konfiguriert sind. Füge das Dodo Payments Skills-Repository zu deiner Konfiguration hinzu:

```json
{
  "skills": {
    "repositories": [
      "dodopayments/skills"
    ]
  }
}
```

</Tab>
</Tabs>

## Verwendung von Skills

Sobald sie installiert sind, kann Ihr KI-Assistent diese Skills nutzen, wenn Sie ihn bitten, Dodo Payments-Funktionen zu implementieren. Hier sind einige Beispielaufforderungen:

### Best Practices

```
Help me integrate Dodo Payments into my Next.js app following best practices
```

### Webhooks

```
Set up webhook handlers for payment and subscription events
```

### Abonnements

```
Implement a subscription flow with free trial and multiple pricing tiers
```

### Checkout

```
Create a checkout session for my SaaS product
```

### Verbrauchsabhängige Abrechnung

```
Add usage-based billing to track API calls for my AI product
```

### BillingSDK

```
Add a pricing page using BillingSDK components
```

### Lizenzschlüssel

```
Implement license key validation for my desktop app
```

## Wie Skills funktionieren

Skills geben Ihrem KI-Assistenten:

1. **Prozedurales Wissen** - Schritt-für-Schritt-Anleitungen zur Implementierung von Funktionen
2. **Code-Vorlagen** - Produktionsbereite Code-Muster und Snippets
3. **Best Practices** - Empfohlene Ansätze für Sicherheit, Fehlerbehandlung und UX
4. **Kontextbewusstsein** - Verständnis dafür, wie verschiedene Dodo Payments-Funktionen zusammenarbeiten

Wenn Sie Ihren KI-Assistenten bitten, eine Funktion zu implementieren, verwendet er den entsprechenden Skill, um:
- Den richtigen Implementierungsansatz zu verstehen
- Genauen, funktionierenden Code zu generieren
- Dodo Payments-Konventionen zu befolgen
- Randfälle und Fehler ordnungsgemäß zu behandeln

<Note>
Skills ergänzen den [MCP Server](/developer-resources/mcp-server) und [Sentra](/developer-resources/sentra). Nutze MCP für API-Operationen, Sentra für geführte Integration und Skills für erweiterte Fähigkeiten deines KI-Assistenten.
</Note>
