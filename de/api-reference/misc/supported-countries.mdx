---
title: Unterstützte Länder
openapi: get /checkout/supported_countries
description: Rufen Sie eine Liste von Ländern ab, die von Dodo Payments für Ihre
  Kunden unterstützt werden.
---

<Tip> Sie erhalten eine Liste von [ISO Alpha-2](https://de.wikipedia.org/wiki/ISO_3166-1_alpha-2) Ländercodes, die die Länder repräsentieren, die wir **derzeit für Zahlungen unterstützen**. Sie können die vollständige Liste der unterstützten Länder [hier](/miscellaneous/list-of-countries-we-accept-payments-from) einsehen. Diese Liste wird regelmäßig aktualisiert, während wir unsere Abdeckung erweitern. </Tip>

<Tip> Hinweis: Die verfügbaren Optionen umfassen ISO Alpha-2-Codes für **alle Länder**, nicht nur für die, die wir derzeit unterstützen. Bitte beziehen Sie sich auf die Liste der unterstützten Länder, um die Verfügbarkeit zu bestätigen. </Tip>
