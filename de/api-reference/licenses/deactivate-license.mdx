<Info>
**Kein API-Schlüssel erforderlich**: Dies ist ein öffentlicher Endpunkt, der keine Authentifizierung erfordert. Sie können ihn direkt von Client-Anwendungen, Desktop-Software oder CLIs aufrufen, um Lizenzschlüssel zu deaktivieren, ohne Ihre API-Anmeldeinformationen offenzulegen.
</Info>
