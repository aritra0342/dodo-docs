---
title: Lizenz deaktivieren
description: Dieser Endpunkt ermöglicht es Ihnen, eine Lizenz für den Benutzer
  zu deaktivieren.
openapi: post /licenses/deactivate
---

<Info>
**Es ist kein API-Schlüssel erforderlich**: Dies ist ein öffentlicher Endpunkt, der keine Authentifizierung benötigt. Sie können ihn direkt aus Client-Anwendungen, Desktop-Software oder CLIs aufrufen, um Lizenzschlüssel zu deaktivieren, ohne Ihre API-Zugangsdaten preiszugeben.
</Info>
