**Wechseln Sie von einem anderen Zahlungsanbieter zu Dodo Payments?** Unser Migrationstool erleichtert es, Ihre **Produkte, Kunden und Rabattcodes zu übertragen, ohne Daten zu verlieren.** Der Prozess ist sicher, geleitet und kann in nur wenigen Minuten durchgeführt werden.

Derzeit unterstützen wir Migrationen von **Lemon Squeezy, Stripe, Polar.sh und Paddle**.

## Erste Schritte

Zuerst installieren Sie das Migrationstool:

```bash
npm install -g dodo-migrate
```

<Note>
Sie benötigen Node.js ≥ 18, das auf Ihrem Computer installiert ist.
</Note>

## Migrieren Sie Ihre Daten

### Wenn Sie von Lemon Squeezy kommen:

```bash
dodo-migrate lemonsqueezy
```

### Wenn Sie von Stripe kommen:

```bash
dodo-migrate stripe
```

### Wenn Sie von Polar.sh kommen:

```bash
dodo-migrate polar
```

### Wenn Sie von Paddle kommen:

```bash
dodo-migrate paddle
```

Das Tool führt Sie Schritt für Schritt durch alles. Sie benötigen:
- Den API-Schlüssel Ihres Anbieters (Lemon Squeezy, Stripe, Polar.sh, Paddle usw.)
- Ihren Dodo Payments API-Schlüssel  
- Ob Sie zuerst testen möchten (empfohlen) oder live gehen möchten
- Welche Dodo Payments-Marke Sie migrieren möchten

## Warum unser Migrationstool verwenden?

- **Kein Datenverlust** - Alles wird sicher übertragen
- **Zuerst testen** - Probieren Sie es im Testmodus aus, bevor Sie live gehen
- **Geleiteter Prozess** - Wir führen Sie durch jeden Schritt
- **Fortsetzen, wenn nötig** - Wenn etwas schiefgeht, können Sie dort weitermachen, wo Sie aufgehört haben

## Erweiterte Nutzung

Wenn Sie die Migration ohne Eingabeaufforderungen durchführen möchten, können Sie alle Details im Voraus angeben:

### Lemon Squeezy Migration:

```bash
dodo-migrate lemonsqueezy \
  --provider-api-key=lsq_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX
```

### Stripe Migration:

```bash
dodo-migrate stripe \
  --provider-api-key=sk_test_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX \
  --migrate-types=products,coupons
```

### Polar.sh Migration:

```bash
dodo-migrate polar \
  --provider-api-key=polar_org_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX \
  --migrate-types=products,discounts,customers
```

### Paddle Migration:

```bash
dodo-migrate paddle \
  --provider-api-key=paddle_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX
```
