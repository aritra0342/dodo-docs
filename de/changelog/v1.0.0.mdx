## Neue Funktionen 🚀

- **Kundenportal**: Die `Customer Portal` Funktion wurde eingeführt, die es Kunden ermöglicht, ihre Abonnements, Rechnungen und Zahlungsmethoden zu verwalten. Sie können statische oder dynamische Kundenportal-Links für Ihre Kunden über das Dodo Payments Dashboard oder über APIs erstellen. Navigieren Sie im Dashboard zu `Sales > Customers > Share Invite`.

- **Kundenportal-Sitzungs-API**: Die `Customer Portal Session` API wurde hinzugefügt, um eine Sitzung des Kundenportals für einen bestimmten Kunden zu erstellen. Beachten Sie, dass die Sitzung nur 24 Stunden gültig ist. Weitere Details finden Sie in der [API-Dokumentation](/api-reference/customers/create-customer-portal-session).

- **Sonstiges**: Kleinere Verbesserungen und Fehlerbehebungen.
