## Neue Funktionen 🚀

- **Checkout 2.0**: Einführung eines vollständig neu gestalteten Checkout-Erlebnisses mit:
    - Vereinfachter, moderner Oberfläche für eine verbesserte Benutzererfahrung
    - Neuem Query Collector-System, das Parameter mit eindeutigen IDs (z. B. `sess_1a2b3c4d`) im Session Storage speichert
    - Durchgängiger Datenverfügbarkeit im gesamten Checkout-Prozess
    - Weiteren Verbesserungen für zukünftige Releases!
  
    <Frame><img src="/images/changelog/checkout_2.png" alt="Checkout 2.0"/></Frame>

- **Konfigurierbare Zahlungsmethoden**: Neue `allowed_payment_methods`-Konfiguration ermöglicht Händlern die Anpassung der verfügbaren Zahlungsmöglichkeiten während des Checkouts. Siehe die [Allowed Payment Methods](/features/payment-methods#configuring-payment-methods)-Dokumentation für Implementierungsdetails.

- **Dynamische Steuerberechnung**: Echtzeitberechnung von Steuersätzen basierend auf den Rechnungsadressen der Kunden für genauere Preisangaben auf den statischen Zahlungslinks.

- **Bedarfsorientierte Abonnements**: Neue API-Endpunkte für flexible Abrechnungen von Abonnements (verfügbar für ausgewählte Händler). Erfahren Sie mehr im [On-Demand Subscriptions](/features/subscription#enabling-on-demand-subscriptions) Leitfaden.

- **Zusätzliche Updates**:
    - Leistungsoptimierungen
    - Fehlerbehebungen
    - Allgemeine Systemverbesserungen
