## Neue Funktionen 🚀

### Analytics-Diagrammfreigabe

Teilen Sie Ihre Analytics-Diagramme als Bilder mit anpassbaren Branding-Optionen. Klicken Sie auf die Schaltfläche „Teilen“ in jedem Diagramm, um ein Bild mit mehreren Farbvarianten zu generieren, die mit Ihrer Markenidentität übereinstimmen.

### Mehrere Teilrückerstattungen

Verarbeiten Sie mehrere Teilrückerstattungen für eine einzelne Zahlung über die API und das Dodo Payments Dashboard, um mehr Flexibilität bei der Bearbeitung von Rückerstattungsanfragen von Kunden zu bieten.

### Kumulative Umsatzanalysen

Verfolgen Sie das Wachstum Ihres Unternehmens mit neuen kumulativen Umsatzdiagrammen im Analyse-Dashboard, die umfassende Einblicke in Ihre Umsatztrends im Laufe der Zeit bieten.

### Next.js-Adapter

Integrieren Sie Dodo Payments einfach in Ihre Next.js-Anwendung mit unserem offiziellen Adapter. Richten Sie die Zahlungsabwicklung in nur wenigen Codezeilen ein. Sie können auch unser fertiges LLM-Prompt mit KI-Coding-Agenten (wie Cursor, Windsurf, Lovable und anderen) verwenden, um den Integrationsprozess zu automatisieren.

[Sehen Sie sich das @dodopayments/nextjs-Paket auf npm an](https://www.npmjs.com/package/@dodopayments/nextjs)

### Context7-Integration

Greifen Sie über Context7 MCP auf die Dokumentation von Dodo Payments zu, damit KI-Coding-Agenten umfassenden Kontext zur Zahlungsabwicklung nutzen können. Für detaillierte Informationen besuchen Sie die [Context7-Dokumentation](https://context7.com/llmstxt/dodopayments-llms-full.txt).
