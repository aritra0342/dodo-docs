## Neue Funktionen 🚀

- Die Adaptive Währungsfunktion unterstützt jetzt zusätzliche Währungen, was die lokalisierten Zahlungserfahrungen in weiteren Regionen verbessert. Neu unterstützte Währungen sind:
  - MVR (Maldivische Rufiyaa)
  - MUR (Mauritische Rupie)
  - ZAR (Südafrikanischer Rand)
  - PHP (Philippinischer Peso)
  - FJD (Fidschianischer Dollar)
  - QAR (Katar-Riyal)
  - MYR (Malaysischer Ringgit)

Für umfassende Details zur Funktionalität der adaptiven Währung und ihren Vorteilen besuchen Sie bitte die [Dokumentation zur adaptiven Währung](/features/adaptive-currency).

- Die Unterstützung der türkischen Sprache wurde zur Dodo Payments Checkout-Seite hinzugefügt.

- Ein neuer Prorationsmodus, `difference_immediately`, wurde eingeführt. Dieser Modus berechnet die Differenz sofort, unabhängig von der verbleibenden Zeit im Abrechnungszyklus, wenn Pläne über die API geändert werden. [Dokumentation](/developer-resources/subscription-integration-guide).
