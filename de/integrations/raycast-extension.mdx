Ihr Dodo Payments-Dashboard, nur wenige Tasteneingaben entfernt. Verwenden Sie Raycast, um schnell Zahlungen, Abonnements, Kunden, Produkte, Rabattcodes, Lizenzschlüssel und Rückerstattungen anzuzeigen, ohne das Dashboard zu öffnen.

## Installation

<Card title="Aus dem Raycast Store installieren →" icon="puzzle-piece" href="https://www.raycast.com/dodopayments/dodo-payments">
  Öffnen Sie die Dodo Payments-Erweiterungsseite im Raycast Store.
</Card>

<Info>
Erfordert Raycast auf macOS und einen Dodo Payments API-Schlüssel.
</Info>

## Einrichtung

<Steps>
<Step title="Holen Sie sich Ihren API-Schlüssel">
1. Melden Sie sich bei Ihrem Dodo Payments-Dashboard an.
2. Gehen Sie zu Einstellungen › API-Schlüssel.
3. Kopieren Sie Ihren API-Schlüssel.
</Step>

<Step title="Konfigurieren Sie die Erweiterung">
1. Öffnen Sie Raycast.
2. Suchen Sie nach "Dodo Payments" und wählen Sie einen Befehl aus.
3. Geben Sie bei Aufforderung Ihren API-Schlüssel im Feld API-Schlüssel ein.
4. Wählen Sie den API-Modus:
   - Test: verwenden Sie ihn für Entwicklung und Tests.
   - Live: verwenden Sie ihn für Produktionsdaten.
5. Speichern Sie die Einstellungen.
</Step>
</Steps>

<Tip>
Beginnen Sie im Testmodus, um Ihre Einrichtung zu validieren, und wechseln Sie dann zu Live.
</Tip>

<Warning>
Behandeln Sie Ihren API-Schlüssel wie ein Passwort. Teilen Sie ihn nicht und speichern Sie ihn nicht im Klartext.
</Warning>

## Verwenden Sie die Erweiterung

- **Suchen**: Öffnen Sie Raycast, geben Sie "Dodo Payments" ein und wählen Sie einen Befehl aus.
- **Filtern**: Verfeinern Sie die Ergebnisse, indem Sie Teile von IDs, E-Mails, Namen oder Status eingeben.
- **Details öffnen**: Drücken Sie die Eingabetaste auf einem Listenelement, um weitere Informationen anzuzeigen.

## Fehlersuche

<AccordionGroup>
<Accordion title="Authentifizierung fehlgeschlagen oder 401/403-Fehler">
Bestätigen Sie, dass der API-Schlüssel und der Modus (Test vs. Live) mit Ihren Dashboard-Daten übereinstimmen. Öffnen Sie den Befehl erneut und geben Sie den Schlüssel bei Bedarf erneut ein.
</Accordion>
<Accordion title="Es erscheinen keine Ergebnisse">
Stellen Sie sicher, dass Ihr ausgewählter Modus Daten hat. Versuchen Sie, zwischen Test und Live zu wechseln.
</Accordion>
<Accordion title="Ratenlimits oder Zeitüberschreitungen">
Warten Sie einen Moment und versuchen Sie es erneut. Wenn das Problem weiterhin besteht, überprüfen Sie Ihre Netzwerkverbindung.
</Accordion>
