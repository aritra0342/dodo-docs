## Übersicht

Das Dodo Payments Framer Plugin integriert nahtlos die Zahlungsabwicklung in Ihre Framer-Websites. Erstellen Sie anpassbare Checkout-Erlebnisse mit Unterstützung für Test- und Produktionsumgebungen.

<Card title="Offizielles Plugin" icon="link" href="https://www.framer.com/marketplace/plugins/dodo-payments">
Zugriff auf das offizielle Dodo Payments Plugin im Framer-Marktplatz.
</Card>

## Funktionen

<CardGroup cols={2}>
<Card title="Gehosteter oder Overlay-Checkout" icon="credit-card">
Wählen Sie einen Vollbild-Flow oder ein Inline-Overlay, das sich nativ auf Ihrer Website anfühlt.
</Card>

<Card title="Drag-and-Drop-Setup" icon="hand-pointer">
Fügen Sie die Komponente hinzu, fügen Sie Ihre Schlüssel ein, veröffentlichen Sie.
</Card>

Banklevel-Akquisition auf sechs Kontinenten, lokale Zahlungsmethoden integriert.
</Card>

</Card>
</CardGroup>

## Einrichtungsanleitung

<Steps>
<Step title="API-Schlüssel generieren">
Besuchen Sie das [Dodo Payments Dashboard](https://app.dodopayments.com/developer/api-keys) und erstellen Sie einen neuen API-Schlüssel.

<Tabs>

<Tab title="Live-Modus">
Für die Produktion mit echter Zahlungsabwicklung.
</Tab>

<Tab title="Testmodus">
Perfekt für Entwicklung und Tests ohne echte Gebühren.
</Tab>

</Tabs>

<Check>
Kopieren Sie Ihren API-Schlüssel sicher - Sie benötigen ihn für den nächsten Schritt.
</Check>
</Step>

<Step title="Plugin authentifizieren">
1. Öffnen Sie das Dodo Payments Plugin in Framer
2. Navigieren Sie zur Authentifizierungsseite
3. Fügen Sie Ihren API-Schlüssel ein
4. Wählen Sie den entsprechenden Modus (Test oder Live)
5. Klicken Sie auf **Weiter**, um sich zu authentifizieren

<Warning>
Halten Sie Ihre API-Schlüssel sicher und teilen Sie sie niemals öffentlich.
</Warning>
</Step>

<Step title="Produkte verwalten">
Nach der Authentifizierung sehen Sie Ihre Produktliste:

- **Vorhandene Produkte**: Automatisch angezeigt
- **Keine Produkte**: Klicken Sie auf die **Plus (+)**-Schaltfläche neben 'Produkte'

<Info>
Sie werden zum Dodo Payments Dashboard weitergeleitet, um Produkte mit Preisen, Beschreibungen und Einstellungen zu erstellen.
</Info>
</Step>

<Step title="Kauf-Buttons hinzufügen">
1. Wählen Sie ein Produkt aus Ihrer verfügbaren Liste aus
2. Klicken Sie auf die `Add Buy Button` Schaltfläche
3. Ein Kauf-Button mit dem Text 'Kaufen \{product_name\}' wird zu Ihrer Leinwand hinzugefügt

<Warning>
Der Kauf-Button wird deaktiviert, wenn Sie nicht über die richtigen Projektberechtigungen verfügen.
</Warning>
</Step>
</Steps>

## Anpassung

<Steps>
<Step title="Checkout-Erlebnis auswählen">
    Wählen Sie Ihr bevorzugtes Checkout-Erlebnis:
    - **Overlay (Eingebettet)**: Der Checkout erscheint als Overlay innerhalb Ihres Framer-Projekts und hält die Benutzer auf Ihrer Website.
    - **Gehostet**: Benutzer werden zu einer von Dodo Payments gehosteten Checkout-Seite weitergeleitet, um ein optimiertes Erlebnis zu bieten.
</Step>

<Step title="Erscheinungsbild anpassen">
    Passen Sie Ihr Checkout-Erlebnis mit den folgenden Erscheinungseinstellungen an:
    - **Checkout-Thema**: Wählen Sie zwischen hellen oder dunklen Themen, um Ihrer Markenästhetik zu entsprechen.
    - **Button-Beschriftung**: Passen Sie den Text an (z. B. 'Jetzt kaufen', 'Kaufen', 'Loslegen').
    - **Button-Stile**: Passen Sie Farben, Schriftarten, Größen und andere Stil-Eigenschaften an.
    - **Rückkehr-URL**: Legen Sie eine Framer-Seite fest, zu der der Benutzer nach dem Checkout zurückkehren soll.
</Step>
</Steps>

## Fehlersuche

<AccordionGroup>
<Accordion title="Authentifizierungsprobleme">
**Problem**: Plugin kann nicht authentifiziert werden

**Lösungen**:
- Überprüfen Sie, ob Ihr API-Schlüssel korrekt ist
- Stellen Sie sicher, dass Sie den richtigen Modus verwenden (Test/Live)
- Überprüfen Sie Ihre Internetverbindung
</Accordion>

<Accordion title="Button-Erstellungsprobleme">
**Problem**: Kauf-Button ist deaktiviert

**Lösungen**:
- Überprüfen Sie, ob Sie die richtigen Berechtigungen in Ihrem Framer-Projekt haben
- Stellen Sie sicher, dass Sie sich ordnungsgemäß authentifiziert haben
- Versuchen Sie, das Plugin zu aktualisieren
</Accordion>

<Accordion title="Produktanzeigeprobleme">
**Problem**: Produkte werden nicht angezeigt

**Lösungen**:
- Stellen Sie sicher, dass Sie Produkte in Ihrem Dodo Payments Dashboard erstellt haben
- Überprüfen Sie, ob Ihr API-Schlüssel die richtigen Berechtigungen hat
- Versuchen Sie, sich erneut zu authentifizieren
</Accordion>
</AccordionGroup>
