---
title: Reichen Sie Ihr Projekt ein
description: So fügen Sie Ihr Open-Source-Projekt dem Dodo Payments
  Community-Verzeichnis hinzu.
icon: plus
---
<Steps>
{/* LOCKED_PATTERN_b8e49197111d4f91963d876b7d3cbd9b */}
- Das Projekt ist Open Source mit einer permissiven Lizenz.
- Es ist relevant für Dodo Payments (SDK, Plugin, Integration, Vorlage).
- Es verfügt über eine README mit Installations-, Nutzungs- und Wartungsstatus.
</Step>

{/* LOCKED_PATTERN_29bfb0d9a10f5f1cc614e31786e69c52 */}
Bearbeiten Sie `community/projects.mdx` und fügen Sie unter dem relevantesten Tab ein neues `<Card>` hinzu.

```mdx
<Card title="PROJECT_NAME (LANG)" href="REPO_URL" icon="code">
  One-sentence description of your project.
  <br/>By <a href="GITHUB_PROFILE_URL">@github_username</a>
  <br/><br/>`LANGUAGE` `FRAMEWORK` `CATEGORY`
</Card>
```

</Step>

{/* LOCKED_PATTERN_5390269a28150786d344085d66aa0ef8 */}
Wenn Sie möchten, eröffnen Sie ein Issue mit der Vorlage "Community Project Submission" mit:
- Repo-URL, Beschreibung, Kategorie (SDK/Plugin/Boilerplate)
- Primäre Sprache/Framework
- Installations-/Nutzungssnippet-Link
- Wartungsstatus und letzte Aktualisierung
</Step>

{/* LOCKED_PATTERN_03e556e84db70a2199e5921d65cd601a */}
Wir prüfen auf Relevanz, Klarheit und Sicherheit. Genehmigte Einträge werden zusammengeführt und veröffentlicht.
</Step>
</Steps>

<Warning>
Wir können Einträge entfernen, die nicht gepflegt, unsicher, irreführend oder gegen die Richtlinien verstoßen.
</Warning>
