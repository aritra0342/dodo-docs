---
title: Reichen Sie Ihr Projekt ein
description: So fügen Sie Ihr Open-Source-Projekt dem Dodo Payments
  Community-Verzeichnis hinzu.
icon: plus
---

<Steps>
<Step title="Überprüfen Sie die Berechtigung">
- Das Projekt ist Open Source mit einer permissiven Lizenz.
- Es ist relevant für Dodo Payments (SDK, Plugin, Integration, Vorlage).
- Hat ein README mit Installations-, Nutzungs- und Wartungsstatus.
</Step>

<Step title="Eintrag über PR hinzufügen">
Bearbeiten `community/projects.mdx` und einen neuen `<Card>` unter dem relevantesten Tab hinzufügen.

```mdx
<Card title="PROJECT_NAME (LANG)" href="REPO_URL" icon="code">
  One-sentence description of your project.
  <br/>By <a href="GITHUB_PROFILE_URL">@github_username</a>
  <br/><br/>`LANGUAGE` `FRAMEWORK` `CATEGORY`
</Card>
```

</Step>

<Step title="Einreichungsproblem öffnen (optional)">
Wenn Sie möchten, öffnen Sie ein Problem mit der Vorlage "Community Project Submission" mit:
- Repo-URL, Beschreibung, Kategorie (SDK/Plugin/Boilerplate)
- Primäre Sprache/Framework
- Installations-/Nutzungs-Snippet-Link
- Wartungsstatus und letztes Update
</Step>

<Step title="Überprüfen & veröffentlichen">
Wir überprüfen auf Relevanz, Klarheit und Sicherheit. Genehmigte Einträge werden zusammengeführt und veröffentlicht.
</Step>
</Steps>

<Warning>
Wir können Einträge entfernen, die nicht gewartet, unsicher, irreführend oder gegen die Richtlinien verstoßen.
</Warning>
