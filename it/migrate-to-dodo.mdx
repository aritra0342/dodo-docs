**Stai passando da un altro fornitore di pagamento a Dodo Payments?** Il nostro strumento di migrazione rende facile spostare i tuoi **prodotti, clienti e codici sconto senza perdere dati.** Il processo è sicuro, guidato e può essere completato in pochi minuti.

Attualmente supportiamo le migrazioni da **Lemon Squeezy, Stripe, Polar.sh e Paddle**.

## Inizia

Per prima cosa, installa lo strumento di migrazione:

```bash
npm install -g dodo-migrate
```

<Note>
Avrai bisogno di Node.js ≥ 18 installato sul tuo computer.
</Note>

## Migra i Tuoi Dati

### Se provieni da Lemon Squeezy:

```bash
dodo-migrate lemonsqueezy
```

### Se provieni da Stripe:

```bash
dodo-migrate stripe
```

### Se provieni da Polar.sh:

```bash
dodo-migrate polar
```

### Se provieni da Paddle:

```bash
dodo-migrate paddle
```

Lo strumento ti guiderà attraverso ogni passaggio. Avrai bisogno di:
- La chiave API del tuo fornitore (Lemon Squeezy, Stripe, Polar.sh, Paddle, ecc.)
- La tua chiave API di Dodo Payments  
- Se desideri testare prima (consigliato) o andare in produzione
- Quale brand di Dodo Payments migrare

## Perché Usare il Nostro Strumento di Migrazione?

- **Nessuna perdita di dati** - Tutto viene trasferito in sicurezza
- **Test prima** - Provalo in modalità test prima di andare in produzione
- **Processo guidato** - Ti guidiamo attraverso ogni passaggio
- **Riprendi se necessario** - Se qualcosa va storto, puoi riprendere da dove hai lasciato

## Utilizzo Avanzato

Se preferisci eseguire la migrazione senza richieste, puoi fornire tutti i dettagli in anticipo:

### Migrazione da Lemon Squeezy:

```bash
dodo-migrate lemonsqueezy \
  --provider-api-key=lsq_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX
```

### Migrazione da Stripe:

```bash
dodo-migrate stripe \
  --provider-api-key=sk_test_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX \
  --migrate-types=products,coupons
```

### Migrazione da Polar.sh:

```bash
dodo-migrate polar \
  --provider-api-key=polar_org_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX \
  --migrate-types=products,discounts,customers
```

### Migrazione da Paddle:

```bash
dodo-migrate paddle \
  --provider-api-key=paddle_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX
```
