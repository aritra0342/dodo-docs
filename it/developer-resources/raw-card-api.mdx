<Note>
Questa guida tratta esclusivamente **pagamenti una tantum** con **carte, Apple Pay e Google Pay**.
</Note>

## Endpoint API

| Environment | Base URL | Confirm Payment Endpoint |
| --- | --- | --- |
| **Test Mode** | `https://test.dodopayments.com` | `https://api.custom.hs.dodopayments.com/payments/<payment_id>/confirm` |
| **Live Mode** | `https://live.dodopayments.com` | `https://api.hs.dodopayments.com/payments/<payment_id>/confirm` |

## Flusso di pagamento con carta passo dopo passo

### Passaggio 1: ottenere la chiave segreta

Inizia creando un pagamento utilizzando la seguente API:

- **Payments**: [Create Payment API](/api-reference/payments/post-payments)

Questa API restituisce un **`secret_key`**, necessario per i passaggi successivi.

### Passaggio 2: confermare il pagamento

Conferma il pagamento chiamando l’endpoint `payments confirm`:

- **Test Mode**: `POST https://api.custom.hs.dodopayments.com/payments/<payment_id>/confirm`
- **Live Mode**: `POST https://api.hs.dodopayments.com/payments/<payment_id>/confirm`

Includi i seguenti dettagli nel corpo della richiesta:

- `client_secret`: chiave segreta del Passaggio 1
- `payment_method`: impostare su `card`
- `payment_method_data`: dati della carta e fatturazione
- `browser_info`: dati specifici del browser (user agent, dettagli dello schermo, ecc.)
- `payment_type`: impostare esplicitamente su `normal` (per pagamenti una tantum)

Intestazioni richieste:

- Ottieni la tua chiave pubblicabile dal supporto DodoPayments
- `api-key`: la tua chiave pubblicabile (ottienila dal team Dodo Payments)
- `content-type`: `application/json`

**Esempio di corpo richiesta:**

```json
{
  "client_secret": "<secret_key>",
  "payment_method": "card",
  "payment_method_data": {
    "card": {
      "card_number": "<card_number>",
      "card_exp_month": "<exp_month>",
      "card_exp_year": "<exp_year>",
      "card_holder_name": "<cardholder_name>",
      "card_cvc": "<cvc>",
      "card_network": "<network>"
    },
    "billing": {
      "address": {
        "first_name": "<first_name>",
        "last_name": "<last_name>"
      }
    }
  },
  "browser_info": {
    "user_agent": "<user_agent>",
    "accept_header": "<accept_header>",
    "language": "<language>",
    "color_depth": "<color_depth>",
    "screen_height": "<screen_height>",
    "screen_width": "<screen_width>",
    "time_zone": "<time_zone>",
    "java_enabled": true,
    "java_script_enabled": true,
    "ip_address": "<ip_address_of_client>",

    "os_type": "<os_type>",

```

### Gestione della risposta

- Se la risposta **`status`** è **`requires_customer_action`**, reindirizza il cliente all’URL fornito nel campo **`next_action`** per l’autenticazione successiva.

<Warning>
Garantisci il rispetto delle linee guida PCI quando gestisci dati di carte grezzi.
</Warning>

## Integrazione con Apple Pay

Per utilizzare Apple Pay, imposta `payment_method` su `wallet` e includi i dati Apple Pay in `payment_method_data`.

### Struttura del corpo richiesta

```json
{
  "client_secret": "<secret_key>",
  "payment_method": "wallet",
  "payment_method_data": {
    "wallet": {
      "apple_pay": {
        "payment_data": "<decrypted_payment_data>",
        "payment_method": {
          "display_name": "Visa 1234",
          "network": "Visa",
          "type": "debit"
        },
        "transaction_identifier": "UNIQUE_TRANSACTION_ID_FROM_APPLE"
      }
    }
  },
  "browser_info": {
    "user_agent": "<user_agent>",
    "accept_header": "<accept_header>",
    "language": "<language>",
    "color_depth": "<color_depth>",
    "screen_height": "<screen_height>",
    "screen_width": "<screen_width>",
    "time_zone": "<time_zone>",
    "java_enabled": true,
    "java_script_enabled": true,
    "ip_address": "<ip_address_of_client>",
    "device_model": "<device_model>",
    "os_type": "<os_type>",
    "os_version": "<version>"
  },
  "payment_type": "normal"
}
```

### Riferimento campi Apple Pay

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `payment_data` | string | Yes | Questo enum rappresenta i dati di pagamento Apple Pay, che possono essere cifrati o decifrati. |
| `payment_method.display_name` | string | Yes | Nome mostrato sul pulsante Apple Pay (ad esempio, "Visa 1234") |
| `payment_method.network` | string | Yes | Rete della carta (ad esempio, "Visa", "Mastercard", "Amex") |
| `payment_method.type` | string | Yes | Tipo di metodo di pagamento (ad esempio, "debit", "credit") |
| `payment_method.card_exp_month` | string | No | Mese di scadenza della carta (ad esempio, "12") |
| `payment_method.card_exp_year` | string | No | Anno di scadenza della carta (ad esempio, "25") |
| `transaction_identifier` | string | Yes | Identificativo univoco della transazione fornito da Apple Pay |

## Integrazione con Google Pay

Per utilizzare Google Pay, imposta `payment_method` su `wallet` e includi i dati Google Pay in `payment_method_data`.

### Struttura del corpo richiesta

```json
{
  "client_secret": "<secret_key>",
  "payment_method": "wallet",
  "payment_method_data": {
    "wallet": {
      "google_pay": {
        "type": "CARD",
        "description": "Visa ending in 1234",
        "info": {
          "card_network": "VISA",
          "card_details": "1234"
        },
        "tokenization_data": {
          "type": "PAYMENT_GATEWAY",
          "token": "<encrypted_payment_token>"
        }
      }
    }
  },
  "browser_info": {
    "user_agent": "<user_agent>",
    "accept_header": "<accept_header>",
    "language": "<language>",
    "color_depth": "<color_depth>",
    "screen_height": "<screen_height>",
    "screen_width": "<screen_width>",
    "time_zone": "<time_zone>",
    "java_enabled": true,
    "java_script_enabled": true,
    "ip_address": "<ip_address_of_client>",
    "device_model": "<device_model>",
    "os_type": "<os_type>",
    "os_version": "<version>"
  },
  "payment_type": "normal"
}
```

### Riferimento campi Google Pay

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `type` | string | Yes | Tipo di metodo di pagamento (tipicamente "CARD") |
| `description` | string | Yes | Descrizione rivolta all’utente del metodo di pagamento |
| `info.card_network` | string | Yes | Rete della carta (ad esempio, "VISA", "MASTERCARD", "AMEX") |
| `info.card_details` | string | Yes | Ultime 4 cifre della carta |
| `info.assurance_details` | object | No | Dettagli sull’assicurazione dell’autenticazione |
| `tokenization_data.type` | string | Yes | Tipo di tokenizzazione (ad esempio, "PAYMENT_GATEWAY") |
| `tokenization_data.token` | string | Yes | Token di pagamento criptato da Google Pay |
