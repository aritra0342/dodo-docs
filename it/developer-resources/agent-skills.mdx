## Panoramica

Le Agent Skills sono capacità riutilizzabili che ampliano ciò che gli assistenti di codifica IA riescono a fare quando lavorano con Dodo Payments. Forniscono conoscenze procedurali e best practice che aiutano gli agenti a portare a termine compiti specifici in modo più efficace, dall'integrazione dei flussi di checkout all'impostazione della fatturazione basata sull'utilizzo.

<Info>
**Considera le skill come plugin per il tuo assistente IA.** Insegnano al tuo agente come implementare correttamente le funzionalità di Dodo Payments, seguendo i modelli consigliati e le pratiche ottimali.
</Info>

Le skill funzionano con qualsiasi agente IA compatibile con MCP, inclusi:
- **Claude Code** - assistente CLI per la codifica di Anthropic
- **OpenCode** - agente di codifica IA open source
- **Cursor** - editor di codice pensato per l'IA
- **Altri client MCP** - qualsiasi agente che supporta il protocollo delle skill

## Installazione

Scegli il metodo di installazione preferito in base al tuo assistente di codifica IA.

<Tabs>
<Tab title="Skills CLI">
Installa le skill utilizzando l'interfaccia a riga di comando universale delle skill che funziona con qualsiasi agente compatibile con MCP:

```bash
# Install all Dodo Payments skills
npx skills add dodopayments/skills
```

Oppure installa singole skill secondo le necessità:

```bash
# Best practices guide
npx skills add dodopayments/skills/dodo-best-practices

# Webhook integration
npx skills add dodopayments/skills/webhook-integration

# Subscription billing
npx skills add dodopayments/skills/subscription-integration

# Checkout flows
npx skills add dodopayments/skills/checkout-integration

# Usage-based billing
npx skills add dodopayments/skills/usage-based-billing

# BillingSDK components
npx skills add dodopayments/skills/billing-sdk

# License key management
npx skills add dodopayments/skills/license-keys
```

<Tip>
Installa solo le skill di cui hai bisogno per mantenere il tuo agente concentrato e ridurre il sovraccarico di contesto.
</Tip>
</Tab>

<Tab title="Claude Code">
Aggiungi il marketplace delle skill di Dodo Payments a Claude Code:

```bash
/plugin marketplace add dodopayments/skills
```

Quindi installa i singoli plugin:

```bash
# Install best practices
/plugin install dodo-best-practices

# Install webhook integration
/plugin install webhook-integration

# Install subscription integration
/plugin install subscription-integration

# Install checkout integration
/plugin install checkout-integration

# Install usage-based billing
/plugin install usage-based-billing

# Install BillingSDK
/plugin install billing-sdk

# Install license keys
/plugin install license-keys
```

</Tab>

<Tab title="OpenCode">
Le skill sono disponibili automaticamente quando sono configurate nelle impostazioni di OpenCode. Aggiungi il repository delle skill di Dodo Payments alla tua configurazione:

```json
{
  "skills": {
    "repositories": [
      "dodopayments/skills"
    ]
  }
}
```

</Tab>
</Tabs>

## Uso delle skill

Una volta installate, il tuo assistente IA può sfruttare queste skill quando gli chiedi di implementare funzionalità di Dodo Payments. Ecco alcuni prompt di esempio:

### Best Practices

```
Help me integrate Dodo Payments into my Next.js app following best practices
```

### Webhooks

```
Set up webhook handlers for payment and subscription events
```

### Subscriptions

```
Implement a subscription flow with free trial and multiple pricing tiers
```

### Checkout

```
Create a checkout session for my SaaS product
```

### Usage-Based Billing

```
Add usage-based billing to track API calls for my AI product
```

### BillingSDK

```
Add a pricing page using BillingSDK components
```

### License Keys

```
Implement license key validation for my desktop app
```

## Come funzionano le skill

Le skill forniscono al tuo assistente IA:
1. **Conoscenze procedurali** - Istruzioni passo dopo passo per implementare le funzionalità
2. **Modelli di codice** - Pattern e snippet pronti per la produzione
3. **Best Practice** - Approcci consigliati per sicurezza, gestione degli errori e UX
4. **Consapevolezza del contesto** - Comprensione di come funzionano insieme le diverse funzionalità di Dodo Payments

Quando chiedi al tuo assistente IA di implementare una funzionalità, utilizza la skill pertinente per:
- Comprendere l'approccio di implementazione corretto
- Generare codice accurato e funzionante
- Seguire le convenzioni di Dodo Payments
- Gestire adeguatamente edge case ed errori

<Note>
Le skill completano il [MCP Server](/developer-resources/mcp-server) e [Sentra](/developer-resources/sentra). Usa MCP per le operazioni API, Sentra per integrazioni guidate e le skill per capacità avanzate del tuo assistente IA.
</Note>
