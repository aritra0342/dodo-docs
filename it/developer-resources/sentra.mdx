## Panoramica

Sentra aggiunge intelligenza ed esecuzione sopra gli stack core di Dodo Payments. È un agente AI che integra, analizza e agisce su fatturazione e pagamenti. Sentra collega SDK, API e adattatori attraverso il tuo stack (Auth, DB, CRM, ecc.), evidenzia approfondimenti sui ricavi come fattori di crescita e esegue azioni sui clienti come rimborsi, aggiornamenti e crediti.

<Info>
Sentra è attualmente in Beta. Le funzionalità, le API e le integrazioni possono cambiare rapidamente. Accogliamo con favore feedback e segnalazioni di bug mentre iteriamo!
</Info>

## Configurazione

Una volta installato, devi configurare Sentra con la tua chiave API di Dodo Payments e le impostazioni ambientali.

<Steps>
<Step title="Get Your API Key">
1. Accedi alla tua [dashboard Dodo Payments](https://dashboard.dodopayments.com).
2. Vai su **Impostazioni** › **API Keys**.
3. Copia la tua chiave API.
</Step>

<Step title="Configure Sentra">
1. Apri il tuo IDE (VS Code, Cursor o Windsurf).
2. Apri il Command Palette (`Cmd+Shift+P` su macOS / `Ctrl+Shift+P` su Windows/Linux).
3. Digita "Sentra: Configure" e selezionalo.
4. Inserisci la tua chiave API di Dodo Payments quando richiesto.

<Frame>
  <img
    src="/images/sentra/setup.png"
    alt="Sentra configuration screen with API key input and environment selection"
    width="50%"
  />
</Frame>
</Step>

<Step title="Select Environment Mode">
Scegli la modalità di ambiente:
- **Test Mode**: per sviluppo e test
- **Live Mode**: per l'uso in produzione

<Warning>
Inizia con la modalità Test per convalidare la configurazione prima di passare alla modalità Live per i dati di produzione.
</Warning>
</Step>

<Step title="Set Model Reasoning Effort">
Seleziona il livello di sforzo di ragionamento del modello preferito:
- **Low**: risposte più rapide, adatto a compiti semplici
- **Medium**: prestazioni bilanciate (consigliato)
- **High**: analisi più approfondita, adatto a integrazioni complesse
</Step>

<Step title="Complete Setup">
Clicca **Let's go!** per completare la configurazione e iniziare a usare Sentra.

<Check>
Le tue chiavi API sono archiviate in modo sicuro nello Secret Storage del tuo IDE e non vengono mai esposte nel tuo codice.
</Check>
</Step>
</Steps>

## Come Funziona

Sentra segue un semplice processo in quattro fasi da prompt a produzione:

<Steps>
<Step title="Prompt">
Scrivi ciò che desideri costruire. Ad esempio: "Aggiungi fatturazione basata sull'utilizzo con una prova gratuita per il mio piano Pro."

<Frame>
  <img
    src="/images/sentra/welcome.png"
    alt="Sentra interface showing prompt input with example task"
    width="50%"
  />
</Frame>
</Step>

<Step title="Plan">
Sentra genera le modifiche (hook SDK, oggetti, configurazione) e presenta un piano per la tua revisione.
</Step>

<Step title="Approve">
Esamina il piano generato e conferma le modifiche che desideri che Sentra implementi.
</Step>

<Step title="Execute">
Sentra applica le modifiche, le testa e conferma il successo.
</Step>
</Steps>

## Utilizzare Sentra

### Modalità

Sentra offre tre modalità di interazione accessibili dall'interfaccia:

- **Chiedi**: Fai domande su Dodo Payments, modelli di fatturazione, utilizzo delle API e migliori pratiche
- **Costruisci**: Integra fatturazione e UI nella tua applicazione con assistenza guidata
- **Piano**: Progetta architetture di fatturazione scalabili con guida architettonica

### Esempi di Compiti

Ecco alcuni compiti comuni che puoi realizzare con Sentra:

- **Integra Pagina Prezzi**: Configura una pagina prezzi completa con elenchi di prodotti e checkout
- **Implementa Webhook**: Configura gestori di webhook per eventi di pagamento
- **Aggiungi Flusso di Abbonamento**: Crea gestione degli abbonamenti con capacità di aggiornamento/diminuzione
- **Imposta Fatturazione Basata sull'Uso**: Configura misuratori e ingestione di eventi per il tracciamento dell'uso
- **Crea Portale Clienti**: Costruisci un portale self-service per la gestione della fatturazione

### Impostazioni di Auto-Approva

Puoi configurare le impostazioni di auto-approvazione per semplificare il tuo flusso di lavoro:

- **Nessuna**: Esamina e approva manualmente tutte le modifiche (raccomandato per la produzione)
- **Basso Rischio**: Auto-approva modifiche semplici e a basso rischio
- **Tutte**: Auto-approva tutte le modifiche (usa con cautela)

## Modelli di Fatturazione Supportati

Sentra supporta tutti i modelli di fatturazione di Dodo Payments:

- **Pagamenti una tantum**: Transazioni di acquisto singole
- **Abbonamenti**: Fatturazione ricorrente con più intervalli
- **Fatturazione Basata sull'Uso**: Fatturazione misurata in base al consumo
- **Fatturazione Basata sui Risultati**: Modelli di prezzo pay-per-result

## Per Chi È

Sentra collega ingegneria e business, in modo che ogni team possa spedire monetizzazione senza attriti:

- **Sviluppatori**: Spedisci monetizzazione più velocemente all'interno del tuo IDE
- **Fondatori e PM**: Prototipa modelli di fatturazione senza lavoro profondo sull'infrastruttura
- **RevOps e Finanza**: Usa azioni guidate senza bisogno di ingegneri
- **Supporto e Successo**: Risolvi casi di fatturazione in modo sicuro, con approvazioni dove necessario

## Risoluzione dei Problemi

<AccordionGroup>
<Accordion title="Extension not appearing after installation">
Se Sentra non appare nel tuo IDE dopo l'installazione, prova quanto segue:

1. **Riavvia l'editor**: chiudi e riapri completamente VS Code, Cursor o Windsurf
2. **Controlla l'installazione**: verifica che l'estensione sia installata andando su Estensioni e cercando "Sentra"
3. **Ricarica la finestra**: usa il Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`) ed esegui "Developer: Reload Window"

<Check>
Riavviare l'editor risolve il problema nella maggior parte dei casi.
</Check>
</Accordion>

<Accordion title="Invalid API key or authentication errors">
- Verifica che la tua chiave API sia corretta e attiva nella dashboard di Dodo Payments
- Assicurati di utilizzare la modalità di ambiente corretta (Test vs Live) che corrisponde alla tua chiave API
- Controlla che la tua chiave API abbia le autorizzazioni necessarie
- Prova a rigenerare la tua chiave API se il problema persiste
</Accordion>

<Accordion title="Configuration not saving">
- Assicurati di aver completato tutti i campi obbligatori nella schermata di configurazione
- Verifica che il tuo IDE abbia il permesso di accedere allo Secret Storage
- Prova a riconfigurare Sentra tramite il Command Palette
</Accordion>

<Accordion title="Plan generation fails">
- Verifica che la tua chiave API sia valida e abbia accesso alle risorse necessarie
- Controlla la tua connessione internet
- Assicurati di utilizzare un modello di fatturazione supportato
- Prova a riformulare il tuo prompt con dettagli più specifici
</Accordion>

<Accordion title="Execution errors">
- Esamina attentamente il piano generato prima di approvarlo
- Verifica che la struttura del tuo progetto corrisponda alle aspettative di Sentra
- Controlla che tutte le dipendenze siano installate
- Esamina il pannello di output dell'IDE per messaggi di errore dettagliati
</Accordion>
</AccordionGroup>

<Warning>
Se continui a riscontrare problemi dopo aver riavviato l'editor, contatta il supporto fornendo dettagli sulla versione del tuo IDE e sui messaggi di errore specifici che visualizzi.
</Warning>
