## Introduzione

Il plugin WooCommerce di Dodo Payments fornisce un'integrazione senza soluzione di continuità con il tuo negozio, consentendoti di accettare pagamenti tramite più metodi, tra cui UPI, carte, bonifici bancari e portafogli digitali. Questa guida completa ti guiderà attraverso il processo di installazione e configurazione.

## Guida all'Installazione Passo-Passo

1. Scarica l'ultima versione zip del plugin di pagamento WooCommerce.

    <Card title="Scarica Plugin" icon="download" href="https://github.com/dodopayments/dodo-woocommerce/releases">
    Ottieni l'ultima versione del nostro plugin WooCommerce
    </Card>

    In alternativa, puoi installare direttamente dalla directory dei plugin di WordPress:
    
    <Card title="Directory Plugin WordPress" icon="wordpress" href="https://wordpress.org/plugins/dodo-payments-for-woocommerce/">
    Installa Dodo Payments per WooCommerce da WordPress.org
    </Card>

2. Accedi al tuo sito WordPress come amministratore (presumiamo che tu abbia già configurato WooCommerce)
3. Naviga su **Plugin → Aggiungi Nuovo Plugin**
    
    <Frame>
        <img src="/images/woocommerce/1.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
4. Clicca sul pulsante **Carica Plugin** e apparirà una finestra di selezione file

    <Frame>
        <img src="/images/woocommerce/2.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/3.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
5. Clicca su **Sfoglia…** e seleziona il file zip che hai scaricato
6. Clicca sul pulsante **Installa Ora** per avviare il processo di installazione
    
    <Frame>
        <img src="/images/woocommerce/4.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
7. Dopo che l'installazione è completata, clicca su **Attiva Plugin** o attivalo dalla sezione **Plugin Installati**
    
    <Frame>
        <img src="/images/woocommerce/5.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
8. Il plugin è ora installato, ma è ancora necessaria la configurazione. Continua con la guida alla configurazione qui sotto.

## Guida alla Configurazione

### Impostazione delle Chiavi API e dei Webhook

1. Naviga su **WooCommerce → Impostazioni → Pagamenti** o clicca sul pulsante **Pagamenti** nella barra laterale sinistra sotto l'elemento di menu WooCommerce
    
    <Frame>
        <img src="/images/woocommerce/6.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/7.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
2. Abilita il fornitore di pagamento **Dodo Payments** se non è già abilitato, quindi clicca su **Gestisci** per configurare il plugin
3. Vedrai varie opzioni di configurazione, ognuna con un testo esplicativo utile. Inizia impostando la tua Chiave API Live
    
    <Frame>
        <img src="/images/woocommerce/8.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
4. Accedi al tuo dashboard di Dodo Payments in **Modalità Live**, quindi naviga su **Dodo Payments (Modalità Live) > Sviluppatore > Chiavi API** oppure visita [questo link diretto](https://app.dodopayments.com/developer/api-keys) e clicca su **Aggiungi Chiave API**
    
    <Frame>
        <img src="/images/woocommerce/9.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
5. Dai alla tua Chiave API un nome descrittivo e clicca su **Crea**
    
    <Frame>
        <img src="/images/woocommerce/10.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
6. Copia la Chiave API generata e incollala nel campo **Chiave API Live** nelle impostazioni del tuo plugin WooCommerce
    
    <Frame>
        <img src="/images/woocommerce/11.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/12.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
7. Successivamente, imposta la Chiave di Firma del Webhook per abilitare la sincronizzazione dello stato dei pagamenti tra Dodo Payments e WooCommerce
8. Scorri fino in fondo alla pagina delle impostazioni del plugin e copia l'URL del webhook visualizzato lì
    
    <Frame>
        <img src="/images/woocommerce/13.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
9. Torna al tuo dashboard di Dodo Payments e naviga su **Dodo Payments (Modalità Live) > Sviluppatore > Webhook** e clicca su **Aggiungi Webhook**
    
    <Frame>
        <img src="/images/woocommerce/14.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
10. Incolla l'URL che hai copiato nel passaggio 8 nella finestra di dialogo e clicca su **Aggiungi Webhook**
    
    <Frame>
        <img src="/images/woocommerce/15.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
11. Dopo aver creato il webhook, clicca sull'icona a forma di occhio accanto alla **Chiave di Firma** redatta per rivelarla e copiarla
    
    <Frame>
        <img src="/images/woocommerce/16.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
12. Incolla la Chiave di Firma nel campo **Chiave di Firma Webhook Live** nelle impostazioni del tuo plugin e salva le modifiche
    
    <Frame>
        <img src="/images/woocommerce/17.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>
    
13. Rivedi le impostazioni rimanenti come **Categoria Fiscale Globale** e **Tutti i Prezzi sono Inclusivi di Tassa**, poiché queste opzioni determinano come i prodotti si sincronizzano da WooCommerce a Dodo Payments. Nota che **Chiave API di Test** e **Chiave di Firma Webhook di Test** sono necessarie solo se prevedi di utilizzare la Modalità di Test
    
    <Frame>
        <img src="/images/woocommerce/18.webp" alt="Plugin Dodo Payments per WooCommerce" />
    </Frame>

## Caratteristiche Chiave

- **Esperienza di Checkout Senza Soluzione di Continuità**: Reindirizza i clienti a una pagina di checkout sicura e ottimizzata
- **Molteplici Metodi di Pagamento**: Supporto per varie opzioni di pagamento locali e internazionali
- **Aggiornamenti di Stato in Tempo Reale**: Sincronizzazione istantanea dello stato degli ordini tramite webhook
- **Registri di Transazione Dettagliati**: Storia dei pagamenti completa nel tuo dashboard WooCommerce
- **Supporto Multi-Valuta**: Accetta pagamenti in più valute

## Risoluzione dei Problemi

### Problemi Comuni

1. **Errori di Elaborazione dei Pagamenti**
   - Verifica che le chiavi API siano configurate correttamente
   - Assicurati che l'URL del webhook sia impostato correttamente
   - Controlla che il tuo certificato SSL sia valido

2. **Problemi di Sincronizzazione del Webhook**
   - Conferma che la chiave di firma del webhook corrisponda
   - Rivedi i log del server per eventuali errori
   - Verifica che il tuo server possa ricevere richieste webhook in arrivo

## Migliori Pratiche di Manutenzione

1. Aggiorna regolarmente il plugin per patch di sicurezza e nuove funzionalità
2. Testa tutti i flussi di pagamento in modalità sandbox prima di elaborare transazioni live
3. Mantieni WordPress e WooCommerce aggiornati all'ultima versione
4. Implementa backup regolari del sito
5. Monitora i log delle transazioni per eventuali attività insolite
6. Conserva le chiavi API in modo sicuro e non condividerle mai pubblicamente
