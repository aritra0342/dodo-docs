## Panoramica

Il plugin Dodo Payments per Framer integra senza soluzione di continuità il processamento dei pagamenti nei tuoi siti web Framer. Crea esperienze di checkout personalizzabili con supporto per ambienti di test e produzione.

<Card title="Plugin Ufficiale" icon="link" href="https://www.framer.com/marketplace/plugins/dodo-payments">
Accedi al plugin ufficiale di Dodo Payments nel marketplace di Framer.
</Card>

## Guida all'Impostazione

<Steps>
<Step title="Genera Chiave API">
Visita il [Dashboard di Dodo Payments](https://app.dodopayments.com/developer/api-keys) e crea una nuova chiave API.

<Tabs>

<Tab title="Modalità Live">
Per uso in produzione con elaborazione dei pagamenti reali.
</Tab>

<Tab title="Modalità Test">
Perfetto per sviluppo e test senza addebiti reali.
</Tab>

</Tabs>

<Check>
Copia la tua chiave API in modo sicuro - ne avrai bisogno per il passaggio successivo.
</Check>
</Step>

<Step title="Autenticare il Plugin">
1. Apri il plugin Dodo Payments in Framer
2. Naviga alla pagina di Autenticazione
3. Incolla la tua chiave API
4. Seleziona la modalità appropriata (Test o Live)
5. Clicca su **Continua** per autenticare

<Warning>
Tieni le tue chiavi API al sicuro e non condividerle mai pubblicamente.
</Warning>
</Step>

<Step title="Gestisci Prodotti">
Dopo l'autenticazione, vedrai l'elenco dei tuoi prodotti:

- **Prodotti esistenti**: Visualizzati automaticamente
- **Nessun prodotto**: Clicca sul pulsante **Più (+)** accanto a 'Prodotti'

<Info>
Verrai reindirizzato al Dashboard di Dodo Payments per creare prodotti con prezzi, descrizioni e impostazioni.
</Info>
</Step>

<Step title="Aggiungi pulsanti di acquisto">
1. Seleziona un prodotto dalla tua lista disponibile
2. Clicca sul pulsante `Add Buy Button`
3. Un pulsante di acquisto con il testo 'Acquista \{product_name\}' verrà aggiunto al tuo canvas

<Warning>
Il pulsante Aggiungi Pulsante di Acquisto sarà disabilitato se non hai i permessi corretti per il progetto.
</Warning>
</Step>
</Steps>

## Personalizzazione

<Steps>
<Step title="Seleziona Esperienza di Checkout">
    Scegli la tua esperienza di checkout preferita:
    - **Overlay (Incorporato)**: Il checkout appare come un overlay all'interno del tuo progetto Framer, mantenendo gli utenti sul tuo sito.
    - **Ospitato**: Gli utenti vengono reindirizzati a una pagina di checkout ospitata da Dodo Payments per un'esperienza semplificata.
</Step>

<Step title="Personalizza Opzioni di Aspetto">
    Personalizza la tua esperienza di checkout con le seguenti impostazioni di aspetto:
    - **Tema di Checkout**: Scegli tra temi chiari o scuri per abbinarsi all'estetica del tuo marchio.
    - **Etichetta del Pulsante**: Personalizza il testo (es. 'Acquista Ora', 'Acquista', 'Inizia').
    - **Stili del Pulsante**: Regola colori, caratteri, dimensioni e altre proprietà di stile.
    - **URL di Ritorno**: Imposta una pagina Framer per cui l'utente tornerà dopo il checkout.
</Step>
</Steps>

## Risoluzione dei Problemi

<AccordionGroup>
<Accordion title="Problemi di Autenticazione">
**Problema**: Il plugin non si autentica

**Soluzioni**:
- Verifica che la tua chiave API sia corretta
- Assicurati di utilizzare la modalità giusta (Test/Live)
- Controlla la tua connessione internet
</Accordion>

<Accordion title="Problemi di Creazione Pulsante">
**Problema**: Aggiungi Pulsante di Acquisto disabilitato

**Soluzioni**:
- Verifica di avere i permessi corretti nel tuo progetto Framer
- Assicurati di essere autenticato correttamente
- Prova a ricaricare il plugin
</Accordion>

<Accordion title="Problemi di Visualizzazione Prodotti">
**Problema**: Prodotti non visualizzati

**Soluzioni**:
- Assicurati di avere prodotti creati nel tuo dashboard di Dodo Payments
- Controlla che la tua chiave API abbia i permessi corretti
- Prova a ri-autenticarti
</Accordion>
</AccordionGroup>
