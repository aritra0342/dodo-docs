## Panoramica

Il plugin Dodo Payments per Framer integra senza soluzione di continuità il processamento dei pagamenti nei tuoi siti web Framer. Crea esperienze di checkout personalizzabili con supporto per ambienti di test e produzione.

## Guida all'Impostazione

<Steps>
{/* LOCKED_PATTERN_386a8c9ba27e2f520f417976ae2d7c92 */}
Visita la [Dodo Payments Dashboard](https://app.dodopayments.com/developer/api-keys) e crea una nuova chiave API.

<Tabs>

{/* LOCKED_PATTERN_ab5fd62157a874c8d97a623e17ed6dd9 */}
Per l'utilizzo in produzione con elaborazione dei pagamenti reali.
</Tab>

{/* LOCKED_PATTERN_0893a30648532a977fd32fa4cfe5aa5c */}
Perfetto per sviluppo e test senza costi reali.
</Tab>

</Tabs>

<Check>
Copia la tua chiave API in modo sicuro: ti servirà per il passaggio successivo.
</Check>
</Step>

{/* LOCKED_PATTERN_918ab01af8a91f24ca04bebd360e16d7 */}
1. Apri il plugin Dodo Payments in Framer
2. Vai alla pagina di Autenticazione
3. Incolla la tua chiave API
4. Seleziona la modalità appropriata (Test o Live)
5. Fai clic su **Continue** per autenticarti

<Warning>
Mantieni al sicuro le tue chiavi API e non condividerle mai pubblicamente.
</Warning>
</Step>

{/* LOCKED_PATTERN_1f6cf950f4953c2b98fed68f92e5abd3 */}
Dopo l'autenticazione, vedrai la lista dei tuoi prodotti:

- **Prodotti esistenti**: Visualizzati automaticamente
- **Nessun prodotto**: Clicca sul pulsante **Più (+)** accanto a 'Prodotti'

<Info>
Verrai reindirizzato alla dashboard di Dodo Payments per creare prodotti con prezzi, descrizioni e impostazioni.
</Info>
</Step>

{/* LOCKED_PATTERN_d0430ab947518d2b45d1080b6954c396 */}
1. Seleziona un prodotto dalla tua lista disponibile
2. Fai clic sul pulsante `Add Buy Button`
3. Un pulsante di acquisto con il testo `Buy {product_name}` verrà aggiunto alla tua tela

<Warning>
Il pulsante Add Buy Button sarà disabilitato se non hai le autorizzazioni corrette per il progetto.
</Warning>
</Step>
</Steps>

## Personalizzazione

<Steps>
{/* LOCKED_PATTERN_029f7fc64f4faf96b9708137acbff21d */}
    Scegli l'esperienza di checkout che preferisci:
    - **Overlay (Embedded)**: Il checkout appare come sovrapposizione all'interno del tuo progetto Framer, mantenendo l'utente sul tuo sito.
    - **Hosted**: Gli utenti vengono reindirizzati a una pagina di checkout ospitata da Dodo Payments per un'esperienza semplificata.
</Step>

{/* LOCKED_PATTERN_fe74d27da37271770f88b3905e82bc15 */}
    Personalizza l'esperienza di checkout con le seguenti impostazioni di aspetto:
    - **Checkout Theme**: Scegli tra temi chiari o scuri per allinearti all'estetica del tuo brand.
    - **Button Label**: Personalizza il testo (es. 'Buy Now', 'Purchase', 'Get Started').
    - **Button Styles**: Regola colori, font, dimensioni e altre proprietà di stile.
    - **Return URL**: Imposta una pagina Framer a cui l'utente tornerà dopo il checkout.
</Step>
</Steps>

## Risoluzione dei Problemi

<AccordionGroup>
{/* LOCKED_PATTERN_e28ee1b2828d589507a5b4e5ba006e0b */}
**Problema**: il plugin non si autentica

**Soluzioni**:
- Verifica che la tua chiave API sia corretta
- Assicurati di usare la modalità giusta (Test/Live)
- Controlla la tua connessione a internet
</Accordion>

{/* LOCKED_PATTERN_6cf97621a010d8ea59e32f4f5ac4ba4e */}
**Problema**: Add Buy Button è disabilitato

**Soluzioni**:
- Verifica di avere le autorizzazioni corrette nel tuo progetto Framer
- Assicurati di essere autenticato correttamente
- Prova a ricaricare il plugin
</Accordion>

{/* LOCKED_PATTERN_44dde71818d2e482b3429adba145540c */}
**Problema**: i prodotti non vengono visualizzati

**Soluzioni**:
- Assicurati di aver creato prodotti nella dashboard di Dodo Payments
- Controlla che la tua chiave API abbia le autorizzazioni corrette
- Prova a ri-autenticarti
</Accordion>
</AccordionGroup>
