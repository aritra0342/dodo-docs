Il tuo dashboard di Dodo Payments, a pochi tasti di distanza. Usa Raycast per visualizzare rapidamente pagamenti, abbonamenti, clienti, prodotti, codici sconto, chiavi di licenza e rimborsi senza aprire il dashboard.

## Installazione

<Card title="Installa dal Raycast Store →" icon="puzzle-piece" href="https://www.raycast.com/dodopayments/dodo-payments">
  Apri la pagina dell'estensione Dodo Payments nel Raycast Store.
</Card>

<Info>
Richiede Raycast su macOS e una chiave API di Dodo Payments.
</Info>

## Configurazione

<Steps>
<Step title="Ottieni la tua chiave API">
1. Accedi al tuo dashboard di Dodo Payments.
2. Vai su Impostazioni › Chiavi API.
3. Copia la tua chiave API.
</Step>

<Step title="Configura l'estensione">
1. Apri Raycast.
2. Cerca "Dodo Payments" e seleziona un comando qualsiasi.
3. Quando richiesto, inserisci la tua chiave API nel campo Chiave API.
4. Scegli la modalità API:
   - Test: usa per sviluppo e test.
   - Live: usa per dati di produzione.
5. Salva le impostazioni.
</Step>
</Steps>

<Tip>
Inizia in modalità Test per convalidare la tua configurazione, poi passa a Live.
</Tip>

<Warning>
Tratta la tua chiave API come una password. Non condividerla o memorizzarla in testo semplice.
</Warning>

## Usa l'estensione

- **Cerca**: apri Raycast, digita "Dodo Payments" e scegli un comando.
- **Filtra**: restringi i risultati digitando parti di ID, email, nomi o stati.
- **Apri dettagli**: premi Invio su un elemento dell'elenco per visualizzare ulteriori informazioni.

## Risoluzione dei problemi

<AccordionGroup>
<Accordion title="Autenticazione fallita o errori 401/403">
Conferma che la chiave API e la modalità (Test vs Live) corrispondano ai dati del tuo dashboard. Riapri il comando e reinserisci la chiave se necessario.
</Accordion>
<Accordion title="Nessun risultato appare">
Assicurati che la modalità selezionata abbia dati. Prova a passare tra Test e Live.
</Accordion>
<Accordion title="Limiti di frequenza o timeout">
Aspetta un momento e riprova. Se il problema persiste, controlla la tua connettività di rete.
</Accordion>
