Il tuo dashboard di Dodo Payments, a pochi tasti di distanza. Usa Raycast per visualizzare rapidamente pagamenti, abbonamenti, clienti, prodotti, codici sconto, chiavi di licenza e rimborsi senza aprire il dashboard.

## Installazione

{/* LOCKED_PATTERN_689eae9422ae1bb6dca6aba8d59d9dd9 */}
  Apri la pagina dell'estensione Dodo Payments nel Raycast Store.
</Card>

<Info>
Richiede Raycast su macOS e una chiave API di Dodo Payments.
</Info>

## Configurazione

<Steps>
{/* LOCKED_PATTERN_92afe5a5f9799e8778da662cdecfb3a0 */}
1. Accedi alla dashboard di Dodo Payments.
2. Vai su Impostazioni › Chiavi API.
3. Copia la tua chiave API.
</Step>

{/* LOCKED_PATTERN_62acba3447424cd114a190a47207d390 */}
1. Apri Raycast.
2. Cerca "Dodo Payments" e seleziona un comando.
3. Quando richiesto, inserisci la tua chiave API nel campo API Key.
4. Scegli la modalità API:
   - Test: da usare per sviluppo e test.
   - Live: da usare per i dati di produzione.
5. Salva le impostazioni.
</Step>
</Steps>

<Tip>
Inizia in modalità Test per convalidare la configurazione, poi passa a Live.
</Tip>

<Warning>
Tratta la tua chiave API come una password. Non condividerla né conservarla in chiaro.
</Warning>

## Usa l'estensione

- **Cerca**: apri Raycast, digita "Dodo Payments" e scegli un comando.
- **Filtra**: restringi i risultati digitando parti di ID, email, nomi o stati.
- **Apri dettagli**: premi Invio su un elemento dell'elenco per visualizzare ulteriori informazioni.
