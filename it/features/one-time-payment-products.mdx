## Cosa sono i pagamenti una tantum?

I pagamenti una tantum sono acquisti fissi e anticipati - ideali per:

- **Licenze software**: Sblocco a vita o di versioni principali
- **Download digitali**: E-book, modelli, grafiche o plugin
- **Servizi**: Audit, pacchetti di onboarding o consulenze una tantum
- **Corsi e biglietti**: Corsi, workshop o sessioni dal vivo

## Vantaggi principali

- **Checkout semplice**: Nessun rinnovo o logica di ripartizione
- **Evasione istantanea**: Consegna immediata di licenze o file
- **Prezzi flessibili**: Sconti e tasse supportati
- **Sviluppatore‑first**: API chiare per creazione, rimborsi e webhook

### Configurazione del prodotto

Configura i campi nel dashboard per definire come appare e si vende il tuo prodotto.

#### Dettagli del prodotto

- **Nome del prodotto** (obbligatorio): Nome visualizzato nel checkout e nelle fatture
- **Descrizione del prodotto** (obbligatoria): Dichiarazione di valore chiara che i clienti vedono prima di pagare
- **Immagine del prodotto** (obbligatoria): PNG/JPG/WebP fino a 3 MB
- **Marca**: Associa il prodotto a un marchio specifico per la tematizzazione e le email
- **Categoria fiscale** (obbligatoria): Scegli la categoria corretta per determinare le regole fiscali

<Tip>
Scegli la categoria fiscale più precisa per garantire la corretta riscossione delle tasse per regione.
</Tip>

#### Prezzi

- **Tipo di prezzo**: Scegli <b>Pagamento unico</b>
- **Prezzo** (obbligatorio): Carica fissa con valuta
- **Sconto applicabile (%)**: Percentuale di sconto opzionale visualizzata nel checkout e nelle fatture

<Warning>
La modifica del prezzo del prodotto influisce solo sui nuovi acquisti. I clienti esistenti non sono interessati.
</Warning>

#### Impostazioni avanzate

- **Prezzi inclusivi di tasse**: Visualizza i prezzi inclusivi delle tasse applicabili
- **Genera chiavi di licenza**: Emissione di una chiave unica dopo l'acquisto. Vedi <a href="/features/license-keys">Chiavi di licenza</a>
- **Consegna di prodotti digitali**: Consegna automatica di file o contenuti. Vedi <a href="/features/digital-product-delivery">Consegna di prodotti digitali</a>
- **Metadati**: Allegare coppie chiave-valore personalizzate per tagging interno o integrazioni. Vedi <a href="/api-reference/metadata">Metadati</a>

<Tip>
Usa i metadati per memorizzare identificatori dal tuo sistema (ad esempio orderId) per riconciliare in seguito eventi e ricevute.
</Tip>

## Casi d'uso comuni

- **Strumenti basati su licenza**: Sblocca funzionalità con una chiave una tantum
- **Asset digitali**: Temi, modelli, set di icone o media
- **Servizi professionali**: Implementazione o configurazione a tariffa fissa
- **Educazione**: Biglietti per corsi e workshop una tantum

## Esempi di integrazione

### Sessioni di checkout (pagamento unico)
Crea una sessione di checkout con un prodotto una tantum nel carrello:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_one_time', quantity: 1 }
  ]
});
```

## Migliori pratiche

- **Chiarire i deliverables**: Comunica ai compratori esattamente cosa ricevono e come viene consegnato
- **Consegna istantanea**: Utilizza chiavi di licenza o consegna digitale per accesso immediato
- **Traccia con metadati**: Memorizza i tuoi ID per una facile riconciliazione

<Info>
I pagamenti una tantum sono la strada più veloce per vendere prodotti a prezzo fisso. Parti in modo semplice, verifica la consegna e adatta la tua offerta in base alla conversione e al feedback.
</Info>
