## Panoramica

La funzione di **Consegna di Prodotti Digitali** consente ai commercianti di consegnare automaticamente file digitali ai clienti dopo acquisti riusciti. Che tu stia vendendo e-book, modelli di design, software o pacchetti multimediali, puoi allegare contenuti scaricabili ai tuoi prodotti, fornendo accesso immediato tramite email e il portale clienti.

Questa funzione supporta sia il caricamento di file ospitati che i link di download esterni, offrendo flessibilità per vari casi d'uso.

## Caratteristiche Principali

<CardGroup cols={2}>
<Card title="Caricamento File" icon="upload">

## Processo di Configurazione

<Steps>
<Step title="Naviga a Prodotti">
Vai alla tua dashboard e seleziona **Prodotti > Crea Nuovo** o modifica un prodotto esistente. Naviga a **Impostazioni Avanzate → Consegna di Prodotti Digitali**.
</Step>

<Step title="Configura la Consegna dei File">
Scegli una delle seguenti opzioni:

- **Carica File**: Aggiungi fino a 10 file direttamente al tuo prodotto
- **Link di Download Esterno**: Incolla un link sicuro ospitato esternamente (es. Dropbox, Google Drive)

<Frame>
<img src="/images/digital-product-delivery/1.png" alt="Interfaccia di configurazione della consegna di prodotti digitali che mostra opzioni di caricamento file e link esterni" />
</Frame>
</Step>

<Step title="Salva Configurazione">
Clicca su **Aggiorna Prodotto** per rendere attive le impostazioni di consegna dei prodotti digitali.

<Warning>
I file caricati sono serviti tramite URL statici: chiunque abbia il link può accedere al file. Considera servizi esterni con autenticazione per contenuti sensibili.
</Warning>
</Step>
</Steps>

## Esperienza del Cliente

### Conferma di Acquisto

Dopo una transazione riuscita, i clienti ricevono un'email di conferma d'acquisto contenente link per il download dei loro prodotti digitali.

### Accesso al Portale Clienti

I clienti possono accedere ai loro prodotti digitali tramite il Portale Clienti:

- Accedi per visualizzare tutti gli acquisti passati
- Scarica file o accedi a link esterni per ciascun prodotto
- Accesso persistente ai contenuti digitali acquistati

## Considerazioni Importanti

- **Sicurezza**: I link di download statici non sono autenticati per ordine. I clienti con il link possono ri-condividere lo stesso link con altre parti. Per contenuti sicuri o con licenza, considera di integrare servizi esterni che supportano accesso protetto o autenticato.
- **Aggiornamenti dei File**: L'aggiornamento di un file o di un link esterno influenzerà solo i futuri acquisti. Gli acquisti passati non saranno influenzati e i clienti potranno ancora accedere a quei file.
- **Rimborsi**: L'emissione di un rimborso non revoca l'accesso ai file. I clienti che hanno già scaricato un file lo avranno ancora sul loro dispositivo.
- **Migliore Pratica**: Per il controllo delle versioni o l'accesso limitato nel tempo, considera di consegnare i tuoi contenuti tramite URL scaduti o firmati ospitati esternamente.
