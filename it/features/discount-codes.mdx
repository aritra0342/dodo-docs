## Cosa Sono i Codici Sconto?

I codici sconto sono token promozionali che riducono i totali degli ordini al checkout. Sono ideali per:

- **Campagne stagionali**: Black Friday, lanci di prodotti o anniversari
- **Offerte di acquisizione**: Incentivi per il primo acquisto o premi per referral
- **Fidelizzazione**: Premi per il recupero o la fedeltà per i clienti esistenti
- **Offerte B2B**: Prezzi contrattati o negoziati tramite codici privati

## Vantaggi Chiave

- **Sconti flessibili**: Percentuale o importo fisso di sconto
- **Controllo mirato**: Restrizioni per prodotto e cicli di abbonamento
- **Governance delle campagne**: Date di scadenza e limiti di utilizzo
- **Checkout senza soluzione di continuità**: Supporto per campo UI e API tramite sessioni di checkout

## Creazione di Codici Sconto

Crea codici sconto nel tuo dashboard di Dodo Payments, quindi applicali nel checkout ospitato o tramite API.

### Configurazione del Dashboard

- **Nome Sconto** (obbligatorio): Nome interno e di visualizzazione nel dashboard
- **Codice** (obbligatorio): La stringa che i clienti inseriscono al checkout
- **Tipo & Importo** (obbligatorio): Imposta un valore percentuale o fisso, oppure genera un codice casuale utilizzando il pulsante fornito
- **Data di Scadenza** (opzionale): Data dopo la quale il codice diventa non valido
- **Limite di Utilizzo** (opzionale): Numero massimo di riscatti totali tra tutti i clienti
- **Restrizione Prodotto** (opzionale): Limita l'applicabilità a prodotti selezionati
- **Limite Ciclo di Abbonamento** (opzionale): Numero di cicli di fatturazione a cui si applica lo sconto

<Tip>
Usa limiti di ciclo per prezzi introduttivi sugli abbonamenti (es. "50% di sconto per 3 mesi").
</Tip>

## Esperienza di Checkout

1. Gli acquirenti inseriscono il codice nel campo di checkout.
2. Gli sconti idonei vengono applicati e i totali si aggiornano immediatamente.

<Info>
Nei Checkout Sessions trasmetti `discount_code` per pre-applicare un codice e imposta `feature_flags.allow_discount_code` per mostrare il campo di inserimento.
</Info>

## Casi d'Uso Comuni

- **Offerte introduttive**: Promozioni di lancio a tempo limitato per nuovi prodotti
- **Bulk o B2B**: Sconti contrattati per set di prodotti selezionati
- **Giocattoli di fidelizzazione**: Codici di recupero in flussi di prevenzione del churn
- **Campagne stagionali**: Promozioni basate su festività o eventi

## Esempi di Integrazione

### Applica uno sconto nelle Sessioni di Checkout
Pre-applica uno sconto e mostra l'interfaccia utente per l'inserimento del codice.

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  discount_code: 'BLACKFRIDAY2024',
  customer: { email: 'user@example.com', name: 'Jane Doe' },
  return_url: 'https://yoursite.com/return'
});
```

### Abilita l'inserimento dello sconto senza pre-applicazione
Consenti ai clienti di inserire un codice al checkout senza passarne uno in anticipo.

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  feature_flags: {
    allow_discount_code: true
  },
  return_url: 'https://yoursite.com/return'
});
```

## Migliori Pratiche

- **Nomina chiaramente**: Usa codici riconoscibili che corrispondano ai nomi delle campagne
- **Limita nel tempo**: Aggiungi scadenze per creare urgenza e prevenire abusi
- **Definisci saggiamente**: Limita a prodotti specifici per evitare perdite di margine
- **Valida in anticipo**: Controlla l'applicabilità del codice prima di confermare il checkout
- **Monitora l'impatto**: Tieni traccia dell'uso e della conversione per campagna

<Info>
I codici sconto sono potenti leve per acquisizione e fidelizzazione. Inizia con offerte semplici e ben denominate, convalida accuratamente e itera in base alle prestazioni.
</Info>
