I pagamenti con carta sono la base dei pagamenti online, accettati globalmente e fidati dai clienti in tutto il mondo. Dodo Payments supporta tutte le principali reti di carte con protezione antifrode integrata e conformità PCI.

## Configurazione

Usa questi valori in `allowed_payment_method_types`:

| Tipo | Descrizione |
| :--- | :---------- |
| `credit` | Tutte le carte di credito |
| `debit` | Tutte le carte di debito |

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  allowed_payment_method_types: ['credit', 'debit'],
  return_url: 'https://example.com/success'
});
```

<Tip>
Includi sia `credit` che `debit` a meno che tu non abbia un motivo specifico per escluderne uno. Le carte di debito sono preferite da molti clienti e spesso hanno costi inferiori.
</Tip>

## Autenticazione 3D Secure

Il 3D Secure (3DS) aggiunge uno strato di autenticazione che riduce le frodi e i chargeback verificando l'identità del titolare della carta.

### Quando viene attivato il 3DS

Il 3DS viene attivato automaticamente quando:
- Richiesto dalla rete di carte
- Richiesto da regolamenti regionali (es. PSD2 in Europa)
- La transazione è segnalata come ad alto rischio

### Forza 3DS

Puoi richiedere il 3DS su tutte le transazioni:

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  force_3ds: true,
  return_url: 'https://example.com/success'
});
```

<Note>
Abilitare il 3DS per tutte le transazioni riduce le frodi ma potrebbe leggermente diminuire il tasso di conversione, poiché alcuni clienti abbandonano durante l'autenticazione.
</Note>

### Abilitare Carte Salvate

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  show_saved_payment_methods: true,
  customer: { customer_id: 'cus_existing_123' },
  return_url: 'https://example.com/success'
});
```

### Acquisti con un clic

```javascript
// Get customer's saved payment methods
const methods = await client.customers.getPaymentMethods('cus_123');

// Use saved card for instant checkout
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  customer: { customer_id: 'cus_123' },
  payment_method_id: methods[0].payment_method_id,
  confirm: true,
  return_url: 'https://example.com/success'
});
```

</AccordionGroup>

## Risoluzione dei Problemi

<AccordionGroup>
<Accordion title="Carta rifiutata">
**Cause:** Fondi insufficienti, carta scaduta, CVV errato, protezione antifrode della banca.

**Soluzione:** Chiedi al cliente di verificare i dettagli o prova con un'altra carta.
</Accordion>

<Accordion title="Autenticazione 3DS non riuscita">
**Cause:** Abbandono del cliente, sistema bancario non disponibile, timeout.

**Soluzione:** Ripeti o chiedi al cliente di contattare la propria banca.
</Accordion>

<Accordion title="Carta non supportata">
**Cause:** Carta regionale non supportata, restrizioni sui prepagati.

**Soluzione:** Il cliente dovrebbe provare un'altra carta di una rete principale.
</Accordion>
</AccordionGroup>
