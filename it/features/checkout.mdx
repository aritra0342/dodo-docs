## Valuta Adattiva

La Valuta Adattiva consente ai clienti di pagare nella loro valuta locale preferita, migliorando la fiducia e i tassi di conversione.

### Come Funziona

1. **Abilita**: Abilita la Valuta Adattiva dalle Impostazioni Aziendali
2. **Seleziona**: I clienti possono cambiare valuta direttamente al checkout
3. **Converti**: I prezzi vengono convertiti dinamicamente utilizzando tassi di cambio in tempo reale
4. **Visualizza**: L'importo finale da pagare viene mostrato in modo trasparente prima del pagamento

## Calcolo Automatico delle Tasse

Le tasse vengono calcolate automaticamente in base alla posizione di fatturazione del cliente, garantendo la conformità con i requisiti di GST, VAT e imposta sulle vendite senza configurazione manuale.

### Come Funziona il Calcolo delle Tasse

<Steps>
<Step title="Rilevamento della Posizione">
  Le regole fiscali vengono applicate in base al paese del cliente (e alla regione, se applicabile).
</Step>

<Step title="Aggiornamenti Dinamici">
  L'importo delle tasse si aggiorna automaticamente quando:
  - Il paese cambia
  - L'indirizzo viene aggiornato
</Step>

<Step title="Visualizzazione Trasparente">
  La ripartizione finale delle tasse viene mostrata chiaramente prima del pagamento.
</Step>
</Steps>

<Tip>
Il calcolo delle tasse è completamente automatizzato. Non è necessaria alcuna configurazione manuale per beni digitali standard e prodotti SaaS.
</Tip>

## Supporto per il Codice Fiscale Aziendale

Per le aziende registrate, il checkout consente ai clienti di inserire il proprio Codice Fiscale Aziendale (ad es. numero VAT/GST).

### Cosa Succede Quando Viene Inserito un Codice Fiscale

- L'idoneità fiscale viene convalidata in tempo reale
- Vengono applicate esenzioni fiscali o regole di inversione dell'onere
- L'importo delle tasse si aggiorna istantaneamente al checkout

## Codici Sconto

I clienti possono applicare codici sconto o promozionali che hai creato nel dashboard direttamente nella pagina di checkout.

### Esperienza di Checkout

1. Il cliente inserisce il codice sconto
2. Lo sconto viene convalidato istantaneamente
3. Il prezzo aggiornato e i risparmi vengono mostrati chiaramente

<Frame>
  <img src="/images/checkout/c4.png" alt="Inserimento del codice sconto al checkout" style={{ width: '70%' }}/>
</Frame>

### Integrazione API

Applica preventivamente un codice sconto o abilita il campo di input per lo sconto:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  discount_code: 'WELCOME20', // Pre-apply a code
  feature_flags: {
    allow_discount_code: true // Show discount input field
  },
  return_url: 'https://yoursite.com/return'
});
```

## Raccolta Intelligente degli Indirizzi

Il checkout supporta l'inserimento flessibile degli indirizzi per un completamento più veloce.

### Opzioni Disponibili

| Opzione | Descrizione |
|--------|-------------|
| **Google Address Autofill** | Selezione rapida con completamento automatico |
| **Inserimento Manuale** | Controllo completo per indirizzi completi |
| **Selezione del Paese** | Guida la logica fiscale e di conformità |

<Tip>
La raccolta degli indirizzi bilancia velocità, accuratezza e copertura globale per massimizzare la conversione garantendo al contempo la conformità.
</Tip>

## Configurazione della Sessione di Checkout

Controlla il comportamento del checkout utilizzando l'API delle Sessioni di Checkout:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  customer: {
    email: 'customer@example.com',
    name: 'Jane Doe'
  },
  billing_currency: 'EUR', // Set specific currency
  discount_code: 'PROMO10',
  feature_flags: {
    allow_discount_code: true
  },
  return_url: 'https://yoursite.com/return',
  metadata: {
    order_ref: 'ORD-12345'
  }
});
```
