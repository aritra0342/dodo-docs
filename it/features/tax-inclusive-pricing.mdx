## Che Cos'è il Prezzo Inclusivo di Tassa?

Il Prezzo Inclusivo di Tassa consente di incorporare le tasse applicabili direttamente nel prezzo del prodotto visualizzato. I clienti vedono un importo finale unico in anticipo per un'esperienza più trasparente e favorevole alla conversione. Al momento del checkout, Dodo calcola la parte esclusa dalla tassa e l'importo della tassa dal prezzo visualizzato in base alla posizione del cliente e la categoria fiscale, e le suddivide chiaramente sulla fattura.

<Info>
L'imposta finale varia ancora in base alla posizione del cliente e alla categoria fiscale. Il checkout e le fatture mostrano la parte d'imposta calcolata derivata dal prezzo visualizzato.
</Info>

## Vantaggi Chiave

- **Chiarezza per i clienti**: Un prezzo finale mostrato in anticipo
- **Conformità per regione**: La tassa è calcolata per posizione e suddivisa
- **Fatturazione accurata**: Le fatture mostrano l'importo netto e la tassa derivata dal prezzo
- **Favorisce la conversione**: Riduzione delle sorprese al checkout

## Configurazione del Prodotto

Crea o modifica un prodotto nel tuo dashboard, quindi abilita il prezzo inclusivo di tassa nelle impostazioni di prezzo.

#### Prezzi

- **Prezzo** (obbligatorio): Prezzo base che viene visualizzato al cliente.
- **Prezzo Inclusivo di Tassa**: Attiva per trattare il prezzo visualizzato come inclusivo di tassa. Dodo calcolerà l'importo netto (pre-tassa) e la parte di tassa al momento del checkout in base alla posizione del cliente e alla categoria fiscale.

<Warning>
Solo gli acquisti futuri riflettono le impostazioni aggiornate con imposta inclusa. Gli acquisti esistenti e gli abbonamenti attivi non vengono modificati retroattivamente.
</Warning>

<Tip>
Usa un testo prodotto coerente come “Tax inclusive” affinché i clienti comprendano i totali a colpo d'occhio.
</Tip>

## Esempi di Integrazione

### Crea un prodotto con prezzo inclusivo di tassa

```typescript
const product = await client.products.create({
  name: 'Pro Plan',
  description: 'All features included',
  price: {
    currency: 'USD',
    price: 10000
    tax_inclusive: true
  }
});
```

### Aggiorna un prodotto esistente per attivare l'inclusione della tassa

```typescript
await client.products.update(product.id, {
  price: {
    tax_inclusive: true
  }
});
```

## Migliori Pratiche

- **Scegli la giusta categoria fiscale** per garantire un calcolo corretto per regione.
- **Comunica i totali** chiaramente con etichette “Inclusivo di tassa” nel testo del prodotto.
- **Testa in sandbox** per convalidare.
