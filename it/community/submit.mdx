---
title: Invia il tuo progetto
description: Come aggiungere il tuo progetto open-source al directory della
  comunità Dodo Payments.
icon: plus
---
<Steps>
{/* LOCKED_PATTERN_b8e49197111d4f91963d876b7d3cbd9b */}
- Il progetto è open-source con licenza permissiva.
- È rilevante per Dodo Payments (SDK, plugin, integrazione, template).
- Ha un README con installazione, utilizzo e stato di manutenzione.
</Step>

{/* LOCKED_PATTERN_29bfb0d9a10f5f1cc614e31786e69c52 */}
Modifica `community/projects.mdx` e aggiungi un nuovo `<Card>` sotto la scheda più rilevante.

```mdx
<Card title="PROJECT_NAME (LANG)" href="REPO_URL" icon="code">
  One-sentence description of your project.
  <br/>By <a href="GITHUB_PROFILE_URL">@github_username</a>
  <br/><br/>`LANGUAGE` `FRAMEWORK` `CATEGORY`
</Card>
```

</Step>

{/* LOCKED_PATTERN_5390269a28150786d344085d66aa0ef8 */}
Se preferisci, apri un issue utilizzando il template "Community Project Submission" con:
- URL del repository, descrizione, categoria (SDK/Plugin/Boilerplate)
- Linguaggio/framework principale
- Link allo snippet di installazione/utilizzo
- Stato della manutenzione e ultimo aggiornamento
</Step>

{/* LOCKED_PATTERN_03e556e84db70a2199e5921d65cd601a */}
Valutiamo rilevanza, chiarezza e sicurezza. Le voci approvate vengono unite e pubblicate.
</Step>
</Steps>

<Warning>
Possiamo rimuovere voci non manutenute, insicure, fuorvianti o in violazione della policy.
</Warning>
