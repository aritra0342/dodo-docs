## Nuove Funzionalità 🚀

- **Checkout 2.0**: Introduciamo un’esperienza di checkout completamente riprogettata con:
    - Interfaccia moderna e snella per migliorare l’esperienza utente
    - Nuovo Query Collector System che memorizza i parametri nella session storage con ID univoci (es. `sess_1a2b3c4d`)
    - Disponibilità persistente dei dati per l’intero flusso di checkout
    - Ulteriori miglioramenti previsti per le versioni future!
  
    <Frame><img src="/images/changelog/checkout_2.png" alt="Checkout 2.0"/></Frame>

- **Metodi di pagamento configurabili**: La nuova configurazione `allowed_payment_methods` consente ai commercianti di personalizzare le opzioni di pagamento disponibili durante il checkout. Consulta la documentazione su [Allowed Payment Methods](/features/payment-methods#configuring-payment-methods) per i dettagli di implementazione.

- **Calcolo Dinamico delle Tasse**: Calcolo in tempo reale delle aliquote fiscali basato sugli indirizzi di fatturazione dei clienti per una determinazione dei prezzi più accurata sui link di pagamento statici.

- **Abbonamenti On-Demand**: Nuovi endpoint API per la fatturazione flessibile degli abbonamenti (disponibile per alcuni commercianti). Scopri di più nella guida sugli [Abbonamenti On-Demand](/features/subscription#enabling-on-demand-subscriptions).

- **Aggiornamenti Aggiuntivi**:
    - Ottimizzazioni delle prestazioni
    - Correzioni di bug
    - Miglioramenti generali del sistema
