## Nuove Funzionalità 🚀

- **Checkout 2.0**: Introduzione di un'esperienza di checkout completamente ridisegnata che include:
    - Interfaccia moderna e semplificata per un'esperienza utente migliorata
    - Nuovo Sistema di Raccolta Query che memorizza i parametri nello storage di sessione con ID unici (ad es., `sess_1a2b3c4d`)
    - Disponibilità persistente dei dati durante l'intero flusso di checkout
    - Ulteriori miglioramenti previsti per le future versioni!
  
    <Frame><img src="/images/changelog/checkout_2.png" alt="Checkout 2.0"/></Frame>

- **Metodi di Pagamento Configurabili**: La nuova configurazione `allowed_payment_methods` consente ai commercianti di personalizzare le opzioni di pagamento disponibili durante il checkout. Consulta la documentazione sui [Metodi di Pagamento Consentiti](/features/payment-methods#configuring-payment-methods) per i dettagli sull'implementazione.

- **Calcolo Dinamico delle Tasse**: Calcolo in tempo reale delle aliquote fiscali basato sugli indirizzi di fatturazione dei clienti per una determinazione dei prezzi più accurata sui link di pagamento statici.

- **Abbonamenti On-Demand**: Nuovi endpoint API per la fatturazione flessibile degli abbonamenti (disponibile per alcuni commercianti). Scopri di più nella guida sugli [Abbonamenti On-Demand](/features/subscription#enabling-on-demand-subscriptions).

- **Aggiornamenti Aggiuntivi**:
    - Ottimizzazioni delle prestazioni
    - Correzioni di bug
    - Miglioramenti generali del sistema
