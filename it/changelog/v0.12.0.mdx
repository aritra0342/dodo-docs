## Nuove funzionalità
- **SDK ufficiali**: Introdotti gli SDK ufficiali di Dodo Payments per **Node.js**, **Python** e **Go**. Consulta la pagina degli [SDK](/developer-resources/dodo-payments-sdks) per maggiori informazioni.
- **Campo Metadata**: Aggiunto un campo `Metadata` per pagamenti e abbonamenti per memorizzare informazioni aggiuntive. Consulta la pagina dei [Metadata](/api-reference/metadata) per maggiori dettagli.
- **API Cliente**: Ampliate le API cliente per includere i seguenti endpoint. Ora puoi passare `customer_id` in pagamenti e abbonamenti per evitare la duplicazione dei clienti:
    - **Crea cliente**: [/api-reference/customers/create-customer](/api-reference/customers/create-customer)
    - **Aggiorna cliente**: [/api-reference/customers/patch-customer](/api-reference/customers/patch-customer)
