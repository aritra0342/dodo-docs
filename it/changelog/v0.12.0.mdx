## Nuove Funzionalità
- **SDK Ufficiali**: Introdotti SDK ufficiali di Dodo Payments per **Node.js**, **Python** e **Go**. Consulta la pagina [SDKs](/developer-resources/dodo-payments-sdks) per ulteriori informazioni.
- **Campo Metadata**: Aggiunto un campo `Metadata` per pagamenti e abbonamenti per memorizzare informazioni aggiuntive. Consulta la pagina [Metadata](/api-reference/metadata) per maggiori dettagli.
- **API Clienti**: Ampliate le API clienti per includere i seguenti endpoint. Ora puoi passare `customer_id` nei pagamenti e negli abbonamenti per prevenire la duplicazione dei clienti:
    - **Crea Cliente**: [/api-reference/customers/create-customer](/api-reference/customers/create-customer)
    - **Aggiorna Cliente**: [/api-reference/customers/patch-customer](/api-reference/customers/patch-customer)
