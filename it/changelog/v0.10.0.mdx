## Nuove Funzionalità
- **Atlas Open Source**: Un'applicazione demo scritta in Next.js per mostrare l'integrazione di Dodo Payments. Esplora il [Repository GitHub](https://github.com/dodopayments/dodo-checkout-demo) e la documentazione [qui](/developer-resources/integration-guide).
- **Sezione per Richieste di Funzionalità**: Introdotta una sezione per richieste di funzionalità nel dashboard di Dodo Payments per dare priorità alla roadmap del prodotto in base al feedback degli utenti.
- **API dei Log delle Richieste Webhook**: Aggiunte API per elencare e dettagliare i log delle richieste webhook.
- **Link di Pagamento Statici**: Aggiunti più parametri di query ai link di pagamento statici per fornire maggiore flessibilità nel processo di checkout. Per ulteriori dettagli, clicca [qui](/developer-resources/integration-guide#1-static-payment-links).
- **API della Lista degli Abbonamenti**: Aggiunta un'API per recuperare tutti gli abbonamenti. Consulta il [Riferimento API](/api-reference/subscriptions/get-subscriptions-1) per i dettagli.

## Miglioramenti Minori e Correzioni di Bug
- Migliorate le risposte delle API di Prodotto e Abbonamento con parametri aggiuntivi.
- Implementato il rate limiting per le richieste API, attualmente impostato a 20 richieste al secondo.
- Migliorata la documentazione per renderla più user-friendly e facile da navigare.
- Varie correzioni di bug minori e miglioramenti.
