## Nuove Funzionalità 🚀

### API Checkout Sessions
Presentiamo la nostra nuova **API Checkout Sessions** - una soluzione di checkout potente e ospitata che fornisce un flusso di pagamento completo sia per acquisti una tantum che per abbonamenti. Questa nuova API sostituisce il precedente approccio Dynamic Links con maggiore flessibilità e un'esperienza utente migliorata.

**Vantaggi Chiave:**
- **Integrazione Semplificata**: Non è necessario fornire indirizzi di fatturazione completi in anticipo - gestiamo automaticamente i dettagli mancanti
- **Personalizzazione Migliorata**: Controllo completo su metodi di pagamento, valute, temi e flag delle funzionalità
- **Migliore UX**: Pre-compila le informazioni del cliente per ridurre l'attrito al checkout e migliorare i tassi di conversione
- **Configurazione Flessibile**: Supporto per carrelli multi-prodotto, prove di abbonamento, codici sconto e altro

**Cosa è Cambiato:**
- **Prima**: Richiesta di informazioni complete sull'indirizzo di fatturazione durante la creazione dei link di pagamento
- **Ora**: Passa qualsiasi informazione del cliente tu abbia disponibile - raccoglieremo il resto durante il checkout
- **Migrazione**: Cambiamento semplice dell'endpoint API senza necessità di ulteriori gestioni
