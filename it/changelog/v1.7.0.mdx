## Nuove Funzionalità 🚀

- **Pagina dedicata agli abbonamenti**: gestisci facilmente gli abbonamenti dei clienti con la nuova pagina dedicata. Vai a `Sales > Subscriptions` nella dashboard.
  
    <Frame><img src="/images/changelog/subscriptions.png" alt="Pagina Abbonamenti"/></Frame>

- **Dettagli cliente migliorati**: visualizza abbonamenti, pagamenti e rimborsi di ogni cliente in un unico posto. Vai a `Sales > Customers > arrow icon` nella dashboard.

    <Frame><img src="/images/changelog/customer_info.png" alt="Dettagli Cliente"/></Frame>

- **Informazioni dettagliate sui pagamenti**: ottieni maggiori informazioni sugli importi, i cicli e le date dei pagamenti. Visita `Business > Payouts` nella dashboard.

    <Frame><img src="/images/changelog/payouts_update.png" alt="Informazioni Pagamenti"/></Frame>

- **SDK Ruby Ufficiale**: Integra Dodo Payments nelle tue applicazioni Ruby senza sforzo. Dai un'occhiata all'[SDK Ruby](https://github.com/dodopayments/dodopayments-ruby) per ulteriori dettagli.

- **Filtri API migliorati**: aggiunti filtri come `customer_id`, `status`, ecc., alle API di elenco pertinenti. Consulta la documentazione API per maggiori informazioni.

- **Miglioramenti Minori e Correzioni di Bug**: Sono stati implementati vari altri miglioramenti minori e correzioni di bug.
