## Nuove Funzionalità 🚀

- **Pagina Dedicata agli Abbonamenti**: Gestisci facilmente gli abbonamenti dei clienti con la nuova pagina dedicata. Naviga a `Sales > Subscriptions` nel dashboard.
  
    <Frame><img src="/images/changelog/subscriptions.png" alt="Pagina Abbonamenti"/></Frame>

- **Dettagli Migliorati sui Clienti**: Visualizza abbonamenti, pagamenti e rimborsi per ogni cliente in un unico posto. Vai a `Sales > Customers > arrow icon` nel dashboard.

    <Frame><img src="/images/changelog/customer_info.png" alt="Dettagli Cliente"/></Frame>

- **Informazioni Dettagliate sui Pagamenti**: Ottieni maggiori informazioni sugli importi, i cicli e le date dei pagamenti. Visita `Business > Payouts` nel dashboard.

    <Frame><img src="/images/changelog/payouts_update.png" alt="Informazioni sui Pagamenti"/></Frame>

- **SDK Ruby Ufficiale**: Integra Dodo Payments nelle tue applicazioni Ruby senza sforzo. Dai un'occhiata all'[SDK Ruby](https://github.com/dodopayments/dodopayments-ruby) per ulteriori dettagli.

- **Filtri API Migliorati**: Aggiunti filtri come `customer_id`, `status`, ecc., alle API di elenco pertinenti. Consulta la documentazione API per ulteriori informazioni.

- **Miglioramenti Minori e Correzioni di Bug**: Sono stati implementati vari altri miglioramenti minori e correzioni di bug.
