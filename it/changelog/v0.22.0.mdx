## Nuove Funzionalità 🚀

- **Primo Pagamento dell'Abbonamento**: La funzionalità più richiesta è qui! Ora ci sarà una singola transazione per il primo pagamento, combinando il mandato e l'importo effettivo dell'abbonamento.

- **Metodo di Pagamento UPI QR**: Aggiunto UPI QR come metodo di pagamento per il mercato indiano, consentendo ai clienti di pagare utilizzando codici QR UPI.
    
    <img src="/images/changelog/upi-qr.png" alt="Funzionalità UPI QR"/>

- **ID Fiscale al Checkout**: I clienti possono ora aggiungere il proprio ID fiscale nella pagina di checkout per scopi di fatturazione e fiscali.

- **Ristrutturazione del Dashboard**: L'intero Dashboard di Dodo Payments è stato ristrutturato per essere più intuitivo e user-friendly.

- **Guide di Onboarding Self-Service**: I nuovi utenti possono ora iniziare a utilizzare Dodo Payments con guide di onboarding self-service.

- **Supporto multilingue**: Aggiunto supporto per oltre 14 lingue nella pagina di pagamento, permettendo ai clienti di pagare nella lingua preferita. Le lingue supportate includono:
    - English (en)
    - العربية (ar) // Arabic
    - Català (ca) // Catalan
    - 中文 (cn) // Chinese
    - Deutsch (de) // German
    - Español (es) // Spanish
    - Français (fr) // French
    - עברית (he) // Hebrew
    - Italiano (it) // Italian
    - 日本語 (ja) // Japanese
    - Nederlands (nl) // Dutch
    - Polski (pl) // Polish
    - Português (pt) // Portuguese
    - Svenska (sv) // Swedish

- **Restyling del PDF della Fattura**: I PDF delle fatture hanno ricevuto un nuovo design, più chiaro ed efficace.

- **Widget di Aiuto Centralizzato**: Aggiunto un widget di aiuto centralizzato per semplificare e migliorare il supporto.
    
    <img src="/images/changelog/help-widget.png" alt="Widget di Aiuto"/>

## Miglioramenti e Correzioni di Bug 🔧

- **Limitazione della Velocità**: Aggiunta limitazione della velocità nell'API delle fatture per prevenire abusi e garantire la stabilità del sistema.

- **Metadati nei Link di Pagamento Statici**: Aggiunto supporto per i metadati nei link di pagamento statici. Per ulteriori dettagli, fare riferimento alla [documentazione](/developer-resources/integration-guide#1-static-payment-links).

- **Miglioramenti e Correzioni Minori**: Vari miglioramenti e correzioni minori per migliorare le prestazioni complessive e l'esperienza utente.
