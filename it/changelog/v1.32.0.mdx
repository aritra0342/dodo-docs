## Nuove Funzionalità 🚀

- **Integrazione del Programma Affiliati**: Lanciata l'integrazione con Affonso, che ti consente di creare e gestire programmi di affiliazione direttamente tramite Dodo Payments. Traccia i riferimenti, gestisci le commissioni e aumenta le tue entrate con partnership di affiliazione affidabili. Scopri di più nella nostra documentazione sugli [Affiliati](/features/affiliates).
  <Frame>
    <img src="/images/affiliates/cover.jpeg" alt="Immagine di Copertura Affiliati"/>
  </Frame>

- **Supporto per più Webhook**: Capacità di webhook migliorate per supportare più endpoint, consentendo di inviare notifiche di eventi a diversi sistemi contemporaneamente.

- **Prevenzione delle Frodi Migliorata**: Aggiunta una nuova impostazione aziendale per forzare l'autenticazione 3D Secure per tutte le transazioni, contribuendo a ridurre le frodi e i chargeback.

- **Cancellazione della Pianificazione degli Abbonamenti**: Introdotta la possibilità di pianificare le cancellazioni degli abbonamenti per avere effetto alla fine del periodo di fatturazione, disponibile tramite la nostra API.

- **Link di Pagamento Statici**: Aggiunti due ulteriori parametri di query per i link di pagamento statici:
  - `paymentCurrency`: Specifica la valuta di pagamento. Il valore predefinito è impostato in base al paese di fatturazione.
  - `showCurrencySelector`: Mostra il selettore di valuta. Il valore predefinito è true.
