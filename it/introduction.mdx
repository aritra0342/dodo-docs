## Cos'è Dodo Payments?

Dodo Payments è il **motore all-in-one** per lanciare, scalare e monetizzare a livello mondiale. Progettato per la nuova generazione di prodotti digitali SaaS e AI, gestiamo tutto, dai **pagamenti alla fatturazione alla distribuzione** - tutto senza ingegneria aggiuntiva.

<Info>
**Progettato per SaaS e prodotti basati su AI**: gestisci pagamenti una tantum, abbonamenti, crediti, postazioni, token o qualsiasi metrica di utilizzo con la nostra infrastruttura di fatturazione ottimizzata.
</Info>

<Tip>
  <strong>Incontra Sentra: il tuo assistente di fatturazione alimentato dall'AI</strong><br/>
  Sentra è un agente AI che si integra, analizza e agisce su fatturazione e pagamenti direttamente nel tuo IDE. Fai domande, crea integrazioni o progetta architetture di fatturazione - tutto descrivendo ciò di cui hai bisogno.
  <a href="/developer-resources/sentra">
    Inizia con Sentra &rarr;
  </a>
</Tip>

### Server MCP per Assistenti AI

Collega il tuo assistente di codifica AI (Cursor, Claude Desktop, Windsurf, VS Code) direttamente a Dodo Payments con i nostri server MCP.

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  Ricerca semantica attraverso la documentazione di Dodo Payments - ottieni risposte immediate su funzionalità, API e best practice
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  Esegui operazioni API direttamente: crea pagamenti, gestisci abbonamenti e gestisci rimborsi tramite il tuo assistente AI
</Card>
</CardGroup>

{/* LOCKED_PATTERN_8ee343b87e057252cc4a8a376b190970 */}
Aggiungi entrambi i server alla tua configurazione MCP (`~/.cursor/mcp.json` o configurazione Claude Desktop):

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
Richiede Node.js 18+. Consulta la [documentazione MCP Server](/developer-resources/mcp-server) per istruzioni dettagliate sull'installazione per tutti i client supportati.
</Info>
</Accordion>
