title: Kirim proyek Anda
description: Cara menambahkan proyek open-source Anda ke direktori komunitas Dodo Payments.
icon: plus
---

<Steps>
{/* LOCKED_PATTERN_b8e49197111d4f91963d876b7d3cbd9b */}
- Proyek ini bersifat sumber terbuka dengan lisensi permisif.
- Relevan dengan Dodo Payments (SDK, plugin, integrasi, template).
- Memiliki README dengan status instalasi, penggunaan, dan pemeliharaan.
</Step>

{/* LOCKED_PATTERN_29bfb0d9a10f5f1cc614e31786e69c52 */}
Edit `community/projects.mdx` dan tambahkan `<Card>` di bawah tab yang paling relevan.

```mdx
<Card title="PROJECT_NAME (LANG)" href="REPO_URL" icon="code">
  One-sentence description of your project.
  <br/>By <a href="GITHUB_PROFILE_URL">@github_username</a>
  <br/><br/>`LANGUAGE` `FRAMEWORK` `CATEGORY`
</Card>
```

</Step>

{/* LOCKED_PATTERN_5390269a28150786d344085d66aa0ef8 */}
Jika Anda lebih suka, buka issue menggunakan template "Community Project Submission" dengan:
- URL repo, deskripsi, kategori (SDK/Plugin/Boilerplate)
- Bahasa/kerangka kerja utama
- Tautan cuplikan instalasi/penggunaan
- Status pemeliharaan dan pembaruan terakhir
</Step>

{/* LOCKED_PATTERN_03e556e84db70a2199e5921d65cd601a */}
Kami meninjau relevansi, kejelasan, dan keamanan. Entri yang disetujui digabung dan diterbitkan.
</Step>
</Steps>

<Warning>
Kami dapat menghapus entri yang tidak dipelihara, tidak aman, menyesatkan, atau melanggar kebijakan.
</Warning>
