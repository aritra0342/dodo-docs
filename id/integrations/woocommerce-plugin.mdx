## Pendahuluan

Plugin WooCommerce Dodo Payments menyediakan integrasi yang mulus dengan toko Anda, memungkinkan Anda untuk menerima pembayaran melalui berbagai metode termasuk UPI, kartu, perbankan internet, dan dompet digital. Panduan komprehensif ini akan memandu Anda melalui proses instalasi dan konfigurasi.

## Panduan Instalasi Langkah-demi-Langkah

1. Unduh zip terbaru dari plugin pembayaran WooCommerce.

    <Card title="Unduh Plugin" icon="download" href="https://github.com/dodopayments/dodo-woocommerce/releases">
    Dapatkan versi terbaru dari plugin WooCommerce kami
    </Card>

    Sebagai alternatif, Anda dapat menginstal langsung dari direktori plugin WordPress:
    
    <Card title="Direktori Plugin WordPress" icon="wordpress" href="https://wordpress.org/plugins/dodo-payments-for-woocommerce/">
    Instal Dodo Payments untuk WooCommerce dari WordPress.org
    </Card>

2. Masuk ke situs WordPress Anda sebagai administrator (kami anggap Anda sudah mengatur WooCommerce)
3. Navigasikan ke **Plugins → Tambah Plugin Baru**
    
    <Frame>
        <img src="/images/woocommerce/1.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
4. Klik tombol **Unggah Plugin** dan dialog pemilihan file akan muncul

    <Frame>
        <img src="/images/woocommerce/2.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/3.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
5. Klik **Jelajahi…** dan pilih file zip yang Anda unduh
6. Klik tombol **Instal Sekarang** untuk memulai proses instalasi
    
    <Frame>
        <img src="/images/woocommerce/4.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
7. Setelah instalasi selesai, klik **Aktifkan Plugin** atau aktifkan dari bagian **Plugin yang Diinstal**
    
    <Frame>
        <img src="/images/woocommerce/5.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
8. Plugin sekarang sudah terinstal, tetapi konfigurasi masih diperlukan. Lanjutkan ke panduan pengaturan di bawah.

## Panduan Konfigurasi

### Menyiapkan Kunci API dan Webhook

1. Navigasikan ke **WooCommerce → Pengaturan → Pembayaran** atau klik tombol **Pembayaran** di sidebar kiri di bawah item menu WooCommerce
    
    <Frame>
        <img src="/images/woocommerce/6.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/7.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
2. Aktifkan penyedia pembayaran **Dodo Payments** jika belum diaktifkan, lalu klik **Kelola** untuk mengonfigurasi plugin
3. Anda akan melihat berbagai opsi konfigurasi, masing-masing dengan teks penjelasan yang membantu. Mulailah dengan menyiapkan Kunci API Live Anda
    
    <Frame>
        <img src="/images/woocommerce/8.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
4. Masuk ke dasbor Dodo Payments Anda dalam **Mode Live**, lalu navigasikan ke **Dodo Payments (Mode Live) > Pengembang > Kunci API** atau kunjungi [tautan langsung ini](https://app.dodopayments.com/developer/api-keys) dan klik **Tambahkan Kunci API**
    
    <Frame>
        <img src="/images/woocommerce/9.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
5. Beri nama deskriptif untuk Kunci API Anda dan klik **Buat**
    
    <Frame>
        <img src="/images/woocommerce/10.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
6. Salin Kunci API yang dihasilkan dan tempelkan ke dalam kolom **Kunci API Live** di pengaturan plugin WooCommerce Anda
    
    <Frame>
        <img src="/images/woocommerce/11.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/12.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
7. Selanjutnya, siapkan Kunci Penandatangan Webhook untuk mengaktifkan sinkronisasi status pembayaran antara Dodo Payments dan WooCommerce
8. Gulir ke bagian bawah halaman pengaturan plugin dan salin URL webhook yang ditampilkan di sana
    
    <Frame>
        <img src="/images/woocommerce/13.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
9. Kembali ke dasbor Dodo Payments Anda dan navigasikan ke **Dodo Payments (Mode Live) > Pengembang > Webhook** dan klik **Tambahkan Webhook**
    
    <Frame>
        <img src="/images/woocommerce/14.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
10. Tempelkan URL yang Anda salin di langkah 8 ke dalam dialog dan klik **Tambahkan Webhook**
    
    <Frame>
        <img src="/images/woocommerce/15.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
11. Setelah membuat webhook, klik ikon mata di sebelah **Kunci Penandatangan** yang disembunyikan untuk menampilkan dan menyalinnya
    
    <Frame>
        <img src="/images/woocommerce/16.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
12. Tempelkan Kunci Penandatangan ke dalam kolom **Kunci Penandatangan Webhook Live** di pengaturan plugin Anda dan simpan perubahan
    
    <Frame>
        <img src="/images/woocommerce/17.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>
    
13. Tinjau pengaturan yang tersisa seperti **Kategori Pajak Global** dan **Semua Harga Termasuk Pajak**, karena opsi ini menentukan bagaimana produk disinkronkan dari WooCommerce ke Dodo Payments. Perhatikan bahwa **Kunci API Uji** dan **Kunci Penandatangan Webhook Uji** hanya diperlukan jika Anda berencana untuk menggunakan Mode Uji
    
    <Frame>
        <img src="/images/woocommerce/18.webp" alt="Plugin Dodo Payments untuk WooCommerce" />
    </Frame>

## Fitur Utama

- **Pengalaman Checkout yang Mulus**: Arahkan pelanggan ke halaman checkout yang aman dan dioptimalkan
- **Berbagai Metode Pembayaran**: Dukungan untuk berbagai opsi pembayaran lokal dan internasional
- **Pembaruan Status Waktu Nyata**: Sinkronisasi status pesanan instan melalui webhook
- **Rekaman Transaksi yang Detail**: Riwayat pembayaran yang komprehensif di dasbor WooCommerce Anda
- **Dukungan Multi-mata Uang**: Terima pembayaran dalam berbagai mata uang

## Pemecahan Masalah

### Masalah Umum

1. **Kegagalan Pemrosesan Pembayaran**
   - Verifikasi kunci API telah dikonfigurasi dengan benar
   - Pastikan URL webhook telah diatur dengan benar
   - Periksa bahwa sertifikat SSL Anda valid

2. **Masalah Sinkronisasi Webhook**
   - Konfirmasi kunci penandatangan webhook cocok
   - Tinjau log server untuk potensi kesalahan
   - Verifikasi server Anda dapat menerima permintaan webhook yang masuk

## Praktik Terbaik Pemeliharaan

1. Secara teratur perbarui plugin untuk patch keamanan dan fitur baru
2. Uji semua alur pembayaran dalam mode sandbox sebelum memproses transaksi langsung
3. Jaga WordPress dan WooCommerce tetap diperbarui ke versi terbaru
4. Lakukan cadangan situs secara teratur
5. Pantau log transaksi untuk aktivitas yang tidak biasa
6. Simpan kunci API dengan aman dan jangan pernah membagikannya secara publik
