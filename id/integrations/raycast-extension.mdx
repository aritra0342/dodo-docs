Dasbor Pembayaran Dodo Anda, hanya beberapa ketukan saja. Gunakan Raycast untuk dengan cepat melihat pembayaran, langganan, pelanggan, produk, kode diskon, kunci lisensi, dan pengembalian dana tanpa membuka dasbor.

## Instal

<Card title="Instal dari Toko Raycast →" icon="puzzle-piece" href="https://www.raycast.com/dodopayments/dodo-payments">
  Buka halaman ekstensi Pembayaran Dodo di Toko Raycast.
</Card>

<Info>
Memerlukan Raycast di macOS dan kunci API Pembayaran Dodo.
</Info>

## Pengaturan

<Steps>
<Step title="Dapatkan Kunci API Anda">
1. Masuk ke dasbor Pembayaran Dodo Anda.
2. Pergi ke Pengaturan › Kunci API.
3. Salin kunci API Anda.
</Step>

<Step title="Konfigurasi Ekstensi">
1. Buka Raycast.
2. Cari "Pembayaran Dodo" dan pilih perintah apa pun.
3. Ketika diminta, masukkan kunci API Anda di kolom Kunci API.
4. Pilih mode API:
   - Uji: gunakan untuk pengembangan dan pengujian.
   - Langsung: gunakan untuk data produksi.
5. Simpan pengaturan.
</Step>
</Steps>

<Tip>
Mulailah dalam mode Uji untuk memvalidasi pengaturan Anda, lalu beralih ke Langsung.
</Tip>

<Warning>
Perlakukan kunci API Anda seperti kata sandi. Jangan membagikannya atau menyimpannya dalam bentuk teks biasa.
</Warning>

## Gunakan ekstensi

- **Cari**: buka Raycast, ketik "Pembayaran Dodo", dan pilih perintah.
- **Saring**: sempitkan hasil dengan mengetik bagian dari ID, email, nama, atau status.
- **Buka detail**: tekan Enter pada item daftar untuk melihat informasi lebih lanjut.

## Pemecahan Masalah

<AccordionGroup>
<Accordion title="Autentikasi gagal atau kesalahan 401/403">
Konfirmasi bahwa kunci API dan mode (Uji vs Langsung) cocok dengan data dasbor Anda. Buka kembali perintah dan masukkan kunci jika diperlukan.
</Accordion>
<Accordion title="Tidak ada hasil yang muncul">
Pastikan mode yang Anda pilih memiliki data. Cobalah beralih antara Uji dan Langsung.
</Accordion>
<Accordion title="Batasan laju atau waktu habis">
Tunggu sebentar dan coba lagi. Jika masalah berlanjut, periksa konektivitas jaringan Anda.
</Accordion>
</AccordionGroup>
