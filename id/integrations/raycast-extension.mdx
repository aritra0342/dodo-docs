Dasbor Pembayaran Dodo Anda, hanya beberapa ketukan saja. Gunakan Raycast untuk dengan cepat melihat pembayaran, langganan, pelanggan, produk, kode diskon, kunci lisensi, dan pengembalian dana tanpa membuka dasbor.

## Instal

<Card title="Install from Raycast Store →" icon="puzzle-piece" href="https://www.raycast.com/dodopayments/dodo-payments">
  Buka halaman ekstensi Dodo Payments di Raycast Store.
</Card>

<Info>
Membutuhkan Raycast di macOS dan kunci API Dodo Payments.
</Info>

## Pengaturan

<Steps>
<Step title="Get your API Key">
1. Masuk ke dasbor Dodo Payments Anda.
2. Buka Settings › API Keys.
3. Salin kunci API Anda.
</Step>

<Step title="Configure the Extension">
1. Buka Raycast.
2. Cari "Dodo Payments" dan pilih salah satu perintah.
3. Saat diminta, masukkan kunci API Anda di kolom API Key.
4. Pilih mode API:
   - Test: gunakan untuk pengembangan dan pengujian.
   - Live: gunakan untuk data produksi.
5. Simpan pengaturannya.
</Step>
</Steps>

<Tip>
Mulai dalam mode Test untuk memvalidasi pengaturan Anda, lalu beralih ke Live.
</Tip>

<Warning>
Perlakukan kunci API Anda seperti kata sandi. Jangan membagikannya atau menyimpannya dalam teks biasa.
</Warning>

## Gunakan ekstensi

- **Cari**: buka Raycast, ketik "Pembayaran Dodo", dan pilih perintah.
- **Saring**: sempitkan hasil dengan mengetik bagian dari ID, email, nama, atau status.
- **Buka detail**: tekan Enter pada item daftar untuk melihat informasi lebih lanjut.

## Pemecahan Masalah

<AccordionGroup>
<Accordion title="Authentication failed or 401/403 errors">
Konfirmasi bahwa kunci API dan mode (Test vs Live) sesuai dengan data di dasbor Anda. Buka kembali perintah dan masukkan ulang kunci jika diperlukan.
</Accordion>
<Accordion title="No results appear">
Pastikan mode yang dipilih memiliki data. Cobalah beralih antara Test dan Live.
</Accordion>
<Accordion title="Rate limits or timeouts">
Tunggu sebentar dan coba lagi. Jika masalah berlanjut, periksa koneksi jaringan Anda.
</Accordion>
</AccordionGroup>
