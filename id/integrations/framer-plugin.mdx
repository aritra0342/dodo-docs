## Ikhtisar

Plugin Framer Dodo Payments mengintegrasikan pemrosesan pembayaran ke dalam situs web Framer Anda dengan mulus. Buat pengalaman checkout yang dapat disesuaikan dengan dukungan untuk lingkungan pengujian dan produksi.

<Card title="Plugin Resmi" icon="link" href="https://www.framer.com/marketplace/plugins/dodo-payments">
Akses plugin Dodo Payments resmi di pasar Framer.
</Card>

## Panduan Pengaturan

<Steps>
<Step title="Hasilkan Kunci API">
Kunjungi [Dasbor Dodo Payments](https://app.dodopayments.com/developer/api-keys) dan buat kunci API baru.

<Tabs>

<Tab title="Mode Langsung">
Untuk penggunaan produksi dengan pemrosesan pembayaran nyata.
</Tab>

<Tab title="Mode Uji">
Sempurna untuk pengembangan dan pengujian tanpa biaya nyata.
</Tab>

</Tabs>

<Check>
Salin kunci API Anda dengan aman - Anda akan membutuhkannya untuk langkah berikutnya.
</Check>
</Step>

<Step title="Autentikasi Plugin">
1. Buka plugin Dodo Payments di Framer
2. Navigasikan ke halaman Autentikasi
3. Tempel kunci API Anda
4. Pilih mode yang sesuai (Uji atau Langsung)
5. Klik **Lanjutkan** untuk mengautentikasi

<Warning>
Jaga kunci API Anda tetap aman dan jangan pernah membagikannya secara publik.
</Warning>
</Step>

<Step title="Kelola Produk">
Setelah autentikasi, Anda akan melihat daftar produk Anda:

- **Produk yang Ada**: Ditampilkan secara otomatis
- **Tidak ada produk**: Klik tombol **Plus (+)** di sebelah 'Produk'

<Info>
Anda akan diarahkan ke Dasbor Dodo Payments untuk membuat produk dengan harga, deskripsi, dan pengaturan.
</Info>
</Step>

<Step title="Tambahkan Tombol Beli">
1. Pilih produk dari daftar yang tersedia
2. Klik tombol `Add Buy Button`
3. Tombol beli dengan teks 'Beli \{product_name\}' akan ditambahkan ke kanvas Anda

<Warning>
Tombol Tambah Beli akan dinonaktifkan jika Anda tidak memiliki izin proyek yang benar.
</Warning>
</Step>
</Steps>

## Kustomisasi

<Steps>
<Step title="Pilih Pengalaman Checkout">
    Pilih pengalaman checkout yang Anda inginkan:
    - **Overlay (Tersemat)**: Checkout muncul sebagai overlay dalam proyek Framer Anda, menjaga pengguna tetap di situs Anda.
    - **Tuan Rumah**: Pengguna diarahkan ke halaman checkout yang dihosting Dodo Payments untuk pengalaman yang lebih lancar.
</Step>

<Step title="Kustomisasi Opsi Penampilan">
    Sesuaikan pengalaman checkout Anda dengan pengaturan penampilan berikut:
    - **Tema Checkout**: Pilih antara tema terang atau gelap untuk mencocokkan estetika merek Anda.
    - **Label Tombol**: Sesuaikan teks (misalnya, 'Beli Sekarang', 'Pembelian', 'Mulai').
    - **Gaya Tombol**: Sesuaikan warna, font, ukuran, dan properti gaya lainnya.
    - **URL Kembali**: Atur halaman Framer untuk pengguna kembali setelah checkout.
</Step>
</Steps>

## Pemecahan Masalah

<AccordionGroup>
<Accordion title="Masalah Autentikasi">
**Masalah**: Plugin tidak dapat diautentikasi

**Solusi**:
- Verifikasi bahwa kunci API Anda benar
- Pastikan Anda menggunakan mode yang tepat (Uji/Langsung)
- Periksa koneksi internet Anda
</Accordion>

<Accordion title="Masalah Pembuatan Tombol">
**Masalah**: Tambah Tombol Beli dinonaktifkan

**Solusi**:
- Verifikasi Anda memiliki izin yang benar dalam proyek Framer Anda
- Pastikan Anda telah diautentikasi dengan benar
- Coba segarkan plugin
</Accordion>

<Accordion title="Masalah Tampilan Produk">
**Masalah**: Produk tidak muncul

**Solusi**:
- Pastikan Anda telah membuat produk di dasbor Dodo Payments Anda
- Periksa bahwa kunci API Anda memiliki izin yang benar
- Coba autentikasi ulang
</Accordion>
</AccordionGroup>
