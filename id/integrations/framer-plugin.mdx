## Ikhtisar

Plugin Framer Dodo Payments mengintegrasikan pemrosesan pembayaran ke dalam situs web Framer Anda dengan mulus. Buat pengalaman checkout yang dapat disesuaikan dengan dukungan untuk lingkungan pengujian dan produksi.

## Panduan Pengaturan

<Steps>
{/* LOCKED_PATTERN_386a8c9ba27e2f520f417976ae2d7c92 */}
Kunjungi [Dodo Payments Dashboard](https://app.dodopayments.com/developer/api-keys) dan buat kunci API baru.

<Tabs>

{/* LOCKED_PATTERN_ab5fd62157a874c8d97a623e17ed6dd9 */}
Untuk penggunaan produksi dengan pemrosesan pembayaran nyata.
</Tab>

{/* LOCKED_PATTERN_0893a30648532a977fd32fa4cfe5aa5c */}
Sempurna untuk pengembangan dan pengujian tanpa biaya nyata.
</Tab>

</Tabs>

<Check>
Salin kunci API Anda dengan aman - Anda akan membutuhkannya untuk langkah berikutnya.
</Check>
</Step>

{/* LOCKED_PATTERN_918ab01af8a91f24ca04bebd360e16d7 */}
1. Buka plugin Dodo Payments di Framer
2. Arahkan ke halaman Autentikasi
3. Tempel kunci API Anda
4. Pilih mode yang sesuai (Test atau Live)
5. Klik **Continue** untuk mengautentikasi

<Warning>
Jaga kunci API Anda tetap aman dan jangan pernah membagikannya secara publik.
</Warning>
</Step>

{/* LOCKED_PATTERN_1f6cf950f4953c2b98fed68f92e5abd3 */}
Setelah autentikasi, Anda akan melihat daftar produk Anda:

- **Produk yang Ada**: Ditampilkan secara otomatis
- **Tidak ada produk**: Klik tombol **Plus (+)** di sebelah 'Produk'

<Info>
Anda akan diarahkan ke Dodo Payments Dashboard untuk membuat produk dengan harga, deskripsi, dan pengaturan.
</Info>
</Step>

{/* LOCKED_PATTERN_d0430ab947518d2b45d1080b6954c396 */}
1. Pilih produk dari daftar yang tersedia
2. Klik tombol `Add Buy Button`
3. Tombol beli dengan teks `Buy {product_name}` akan ditambahkan ke kanvas Anda

<Warning>
Tombol Tambah Beli akan dinonaktifkan jika Anda tidak memiliki izin proyek yang benar.
</Warning>
</Step>
</Steps>

## Kustomisasi

<Steps>
{/* LOCKED_PATTERN_029f7fc64f4faf96b9708137acbff21d */}
    Pilih pengalaman checkout yang Anda sukai:
    - **Overlay (Embedded)**: Checkout muncul sebagai overlay dalam proyek Framer Anda, menjaga pengguna tetap di situs Anda.
    - **Hosted**: Pengguna diarahkan ke halaman checkout yang dihosting Dodo Payments untuk pengalaman yang lebih lancar.
</Step>

{/* LOCKED_PATTERN_fe74d27da37271770f88b3905e82bc15 */}
    Sesuaikan pengalaman checkout Anda dengan pengaturan tampilan berikut:
    - **Checkout Theme**: Pilih antara tema terang atau gelap untuk menyesuaikan estetika merek Anda.
    - **Button Label**: Sesuaikan teks (misal, 'Buy Now', 'Purchase', 'Get Started').
    - **Button Styles**: Sesuaikan warna, font, ukuran, dan properti gaya lainnya.
    - **Return URL**: Tetapkan halaman Framer agar pengguna kembali setelah checkout.
</Step>
</Steps>

## Pemecahan Masalah

<AccordionGroup>
{/* LOCKED_PATTERN_e28ee1b2828d589507a5b4e5ba006e0b */}
**Masalah**: Plugin gagal mengautentikasi

**Solusi**:
- Verifikasi kunci API Anda benar
- Pastikan Anda menggunakan mode yang benar (Test/Live)
- Periksa koneksi internet Anda
</Accordion>

{/* LOCKED_PATTERN_6cf97621a010d8ea59e32f4f5ac4ba4e */}
**Masalah**: Tombol Tambah Beli dinonaktifkan

**Solusi**:
- Verifikasi Anda memiliki izin yang benar di proyek Framer Anda
- Pastikan Anda terautentikasi dengan benar
- Coba muat ulang plugin
</Accordion>

{/* LOCKED_PATTERN_44dde71818d2e482b3429adba145540c */}
**Masalah**: Produk tidak muncul

**Solusi**:
- Pastikan Anda telah membuat produk di dashboard Dodo Payments
- Periksa bahwa kunci API Anda memiliki izin yang benar
- Coba autentikasi ulang
</Accordion>
</AccordionGroup>
