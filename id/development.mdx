</Info>

Ikuti langkah-langkah ini untuk menginstal dan menjalankan Mintlify di sistem operasi Anda:

**Langkah 1**: Instal Mintlify:

<CodeGroup>

  ```bash npm
  npm i -g mintlify
  ```

```bash yarn
yarn global add mintlify
```

</CodeGroup>

**Langkah 2**: Arahkan ke direktori dokumen (di mana file `mint.json` berada) dan jalankan perintah berikut:

```bash
mintlify dev
```

Pratinjau lokal dari dokumentasi Anda akan tersedia di `http://localhost:3000`.

### Port Kustom

Secara default, Mintlify menggunakan port 3000. Anda dapat menyesuaikan port tempat Mintlify berjalan dengan menggunakan flag `--port`. Untuk menjalankan Mintlify di port 3333, misalnya, gunakan perintah ini:

```bash
mintlify dev --port 3333
```

Jika Anda mencoba menjalankan Mintlify di port yang sudah digunakan, ia akan menggunakan port berikutnya yang tersedia:

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Versi Mintlify

Harap dicatat bahwa setiap rilis CLI terkait dengan versi tertentu dari Mintlify. Jika situs web lokal Anda tidak sesuai dengan versi produksi, harap perbarui CLI:

<CodeGroup>

```bash npm
npm i -g mintlify@latest
```

```bash yarn
yarn global upgrade mintlify
```

</CodeGroup>

## Memvalidasi Tautan

CLI dapat membantu memvalidasi tautan referensi yang dibuat dalam dokumentasi Anda. Untuk mengidentifikasi tautan yang rusak, gunakan perintah berikut:

```bash
mintlify broken-links
```

## Pemformatan Kode

Kami menyarankan untuk menggunakan ekstensi di IDE Anda untuk mengenali dan memformat MDX. Jika Anda pengguna VSCode, pertimbangkan [ekstensi MDX VSCode](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) untuk penyorotan sintaks, dan [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) untuk pemformatan kode.

## Pemecahan Masalah

<AccordionGroup>
  <Accordion title='Kesalahan: Tidak dapat memuat modul "sharp" menggunakan runtime darwin-arm64'>

    Ini mungkin disebabkan oleh versi node yang usang. Coba langkah-langkah berikut:
    1. Hapus versi mintlify yang saat ini terinstal: `npm remove -g mintlify`
    2. Tingkatkan ke Node v19 atau lebih tinggi.
    3. Instal ulang mintlify: `npm install -g mintlify`
  </Accordion>

  <Accordion title="Masalah: Mengalami kesalahan yang tidak diketahui">
  
    Solusi: Pergi ke root perangkat Anda dan hapus folder \~/.mintlify. Setelah itu, jalankan `mintlify dev` lagi.
  </Accordion>
</AccordionGroup>

Penasaran tentang apa yang berubah di versi CLI? [Lihat changelog CLI.](https://www.npmjs.com/package/mintlify?activeTab=versions)
