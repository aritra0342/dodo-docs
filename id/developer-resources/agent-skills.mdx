## Ikhtisar

Keterampilan Agen adalah kemampuan yang dapat digunakan kembali yang meningkatkan apa yang bisa dilakukan asisten pengkodean AI saat bekerja dengan Dodo Payments. Mereka menyediakan pengetahuan prosedural dan praktik terbaik yang membantu agen menyelesaikan tugas tertentu dengan lebih efektif - mulai dari mengintegrasikan alur checkout hingga mengatur penagihan berbasis penggunaan.

<Info>
**Anggap keterampilan sebagai plugin untuk asisten AI Anda.** Mereka mengajari agen Anda cara mengimplementasikan fitur Dodo Payments dengan benar, mengikuti pola rekomendasi dan praktik terbaik kami.
</Info>

Keterampilan bekerja dengan agen AI yang kompatibel MCP, termasuk:
- **Claude Code** - asisten pengkodean CLI dari Anthropic
- **OpenCode** - agen pengkodean AI sumber terbuka
- **Cursor** - editor kode berbasis AI
- **Klien MCP lainnya** - Agen apa pun yang mendukung protokol keterampilan

## Instalasi

Pilih metode instalasi yang diinginkan berdasarkan asisten pengkodean AI Anda.

<Tabs>
<Tab title="Skills CLI">
Pasang keterampilan menggunakan CLI keterampilan universal yang bekerja dengan agen kompatibel MCP apa pun:

```bash
# Install all Dodo Payments skills
npx skills add dodopayments/skills
```

Atau instal keterampilan individu sesuai kebutuhan:

```bash
# Best practices guide
npx skills add dodopayments/skills/dodo-best-practices

# Webhook integration
npx skills add dodopayments/skills/webhook-integration

# Subscription billing
npx skills add dodopayments/skills/subscription-integration

# Checkout flows
npx skills add dodopayments/skills/checkout-integration

# Usage-based billing
npx skills add dodopayments/skills/usage-based-billing

# BillingSDK components
npx skills add dodopayments/skills/billing-sdk

# License key management
npx skills add dodopayments/skills/license-keys
```

<Tip>
Pasang hanya keterampilan yang Anda perlukan untuk menjaga fokus agen Anda dan mengurangi beban konteks.
</Tip>
</Tab>

<Tab title="Claude Code">
Tambahkan marketplace keterampilan Dodo Payments ke Claude Code:

```bash
/plugin marketplace add dodopayments/skills
```

Kemudian instal plugin individu:

```bash
# Install best practices
/plugin install dodo-best-practices

# Install webhook integration
/plugin install webhook-integration

# Install subscription integration
/plugin install subscription-integration

# Install checkout integration
/plugin install checkout-integration

# Install usage-based billing
/plugin install usage-based-billing

# Install BillingSDK
/plugin install billing-sdk

# Install license keys
/plugin install license-keys
```

</Tab>

<Tab title="OpenCode">
Keterampilan otomatis tersedia ketika dikonfigurasi di pengaturan OpenCode Anda. Tambahkan repositori keterampilan Dodo Payments ke konfigurasi Anda:

```json
{
  "skills": {
    "repositories": [
      "dodopayments/skills"
    ]
  }
}
```

</Tab>
</Tabs>

## Menggunakan Keterampilan

Setelah terinstal, asisten AI Anda dapat memanfaatkan keterampilan ini ketika Anda memintanya untuk menerapkan fitur Dodo Payments. Berikut adalah beberapa contoh prompt:

### Praktik Terbaik

```
Help me integrate Dodo Payments into my Next.js app following best practices
```

### Webhooks

```
Set up webhook handlers for payment and subscription events
```

### Langganan

```
Implement a subscription flow with free trial and multiple pricing tiers
```

### Checkout

```
Create a checkout session for my SaaS product
```

### Penagihan Berbasis Penggunaan

```
Add usage-based billing to track API calls for my AI product
```

### BillingSDK

```
Add a pricing page using BillingSDK components
```

### Kunci Lisensi

```
Implement license key validation for my desktop app
```

## Cara Keterampilan Bekerja

Keterampilan memberikan asisten AI Anda:

1. **Pengetahuan Prosedural** - Instruksi langkah demi langkah untuk menerapkan fitur
2. **Template Kode** - Pola dan potongan kode siap pakai
3. **Praktik Terbaik** - Pendekatan yang direkomendasikan untuk keamanan, penanganan kesalahan, dan UX
4. **Kesadaran Konteks** - Pemahaman bagaimana berbagai fitur Dodo Payments bekerja sama

Saat Anda meminta asisten AI Anda untuk menerapkan suatu fitur, ia menggunakan keterampilan yang relevan untuk:
- Memahami pendekatan implementasi yang benar
- Menghasilkan kode yang akurat dan berfungsi dengan baik
- Mengikuti konvensi Dodo Payments
- Menangani kasus ekstrem dan kesalahan dengan tepat

<Note>
Keterampilan melengkapi [MCP Server](/developer-resources/mcp-server) dan [Sentra](/developer-resources/sentra). Gunakan MCP untuk operasi API, Sentra untuk integrasi terpandu, dan Keterampilan untuk kemampuan asisten AI yang ditingkatkan.
</Note>
