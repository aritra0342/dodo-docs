## Ikhtisar

Sebuah boilerplate siap produksi untuk mengintegrasikan Dodo Payments ke dalam aplikasi Expo/React Native. Template ini mencakup pembuatan sesi checkout, penanganan pembayaran, dan konfigurasi TypeScript yang tepat untuk membantu Anda mulai menerima pembayaran di aplikasi mobile Anda dengan cepat.

<Info>
Boilerplate ini menggunakan Expo SDK dengan TypeScript dan menyertakan implementasi contoh untuk sesi checkout dan pemrosesan pembayaran.
</Info>

## Prasyarat

- Node.js 18+
- Expo CLI atau EAS CLI
- Akun Dodo Payments (API key dari dashboard)
- Simulator iOS atau Emulator Android (atau perangkat fisik)

## Memulai dengan Cepat

<Steps>
{/* LOCKED_PATTERN_3810046f56651f3ba0e1684e009cd2e3 */}
Clone repositori boilerplate dan masuk ke direktori proyek:

```bash
git clone https://github.com/dodopayments/expo-boilerplate.git
cd expo-boilerplate
```

</Step>

{/* LOCKED_PATTERN_985ffb13172d607786ae5265635e02cf */}
Dapatkan kredensial Anda dari dasbor Dodo Payments dalam Mode Tes:

- **API Key**: Navigasikan ke **Developer → API Keys** di [dashboard](https://app.dodopayments.com) Anda

<Tip>
Mulailah dengan Mode Tes untuk menguji integrasi Anda dengan aman tanpa memproses pembayaran nyata.
</Tip>
</Step>

{/* LOCKED_PATTERN_f8f1c4a54590eca96bda1740ea80800b */}
Buat file `.env` di direktori root:

```bash
touch .env
```

Tambahkan kredensial Dodo Payments Anda:

```bash .env
EXPO_PUBLIC_DODO_API_KEY=your_test_api_key_here
```

<Warning>
Jangan pernah meng-commit file `.env` ke kontrol versi. File tersebut sudah termasuk dalam `.gitignore`.
</Warning>
</Step>

{/* LOCKED_PATTERN_5e9c44b19c1da0e28673e0b0f87e9162 */}
Pasang paket npm yang diperlukan:

```bash
npm install
```

</Step>

{/* LOCKED_PATTERN_e0b1f8ca8b418cb641bfae6fc3ff28bc */}
Jalankan server pengembangan Expo:

```bash
npx expo start
```

Pindai kode QR dengan aplikasi Expo Go di perangkat Anda, atau tekan `i` untuk simulator iOS atau `a` untuk emulator Android.

<Check>
Anda siap memulai pengembangan! Boilerplate ini mencakup layar dan komponen contoh untuk membantu Anda memahami integrasi.
</Check>
</Step>
</Steps>

## Struktur Proyek

```
expo-boilerplate/
├── app/
│   ├── (tabs)/
│   │   └── index.tsx          # Home screen with examples
│   └── checkout.tsx           # Checkout screen
├── components/
│   └── PaymentButton.tsx      # Payment button component
├── lib/
│   └── dodo.ts                # Dodo Payments client setup
├── .env.example               # Environment variables template
└── package.json
```

## Membangun untuk Produksi

<Steps>
{/* LOCKED_PATTERN_8da0a64f26eca1e80276896a6999817f */}

```bash
npm install -g eas-cli
```

</Step>

{/* LOCKED_PATTERN_68d4159ac59cdefe3384c2094bbecbd9 */}

```bash
eas build:configure
```

</Step>

{/* LOCKED_PATTERN_46d8869736258162472d6c37d4e5967b */}
Untuk iOS:

```bash
eas build --platform ios
```

Untuk Android:

```bash
eas build --platform android
```

</Step>
</Steps>
