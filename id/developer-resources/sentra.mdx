## Ikhtisar

Sentra menambahkan kecerdasan dan eksekusi di atas tumpukan inti Dodo Payments. Ini adalah agen AI yang mengintegrasikan, menganalisis, dan bertindak pada penagihan dan pembayaran. Sentra menghubungkan SDK, API, dan adaptor di seluruh tumpukan Anda (Auth, DB, CRM, dll.), menampilkan wawasan pendapatan seperti pendorong pertumbuhan, dan mengeksekusi tindakan pelanggan seperti pengembalian dana, peningkatan, dan kredit.

<Info>
Sentra saat ini dalam versi Beta. Fitur, API, dan integrasi dapat berubah dengan cepat. Kami menyambut masukan dan laporan bug saat kami mengembangkannya!
</Info>

## Instal

Sentra tersedia sebagai ekstensi untuk VS Code, Cursor, dan Windsurf. Pilih IDE Anda di bawah ini:

<Tabs>
<Tab title="VS Code">

## Pengaturan

Setelah diinstal, Anda perlu mengonfigurasi Sentra dengan kunci API Dodo Payments dan pengaturan lingkungan Anda.

<Steps>
<Step title="Get Your API Key">
1. Masuk ke [dasbor Dodo Payments](https://dashboard.dodopayments.com).
2. Arahkan ke **Settings** › **API Keys**.
3. Salin kunci API Anda.
</Step>

<Step title="Configure Sentra">
1. Buka IDE Anda (VS Code, Cursor, atau Windsurf).
2. Buka Command Palette (`Cmd+Shift+P` pada macOS / `Ctrl+Shift+P` pada Windows/Linux).
3. Ketik "Sentra: Configure" dan pilih opsi tersebut.
4. Masukkan kunci API Dodo Payments Anda saat diminta.

<Frame>
  <img
    src="/images/sentra/setup.png"
    alt="Layar konfigurasi Sentra dengan masukan kunci API dan pemilihan lingkungan"
    width="50%"
  />
</Frame>
</Step>

<Step title="Select Environment Mode">
Pilih mode lingkungan Anda:
- **Test Mode**: Untuk pengembangan dan pengujian
- **Live Mode**: Untuk penggunaan produksi

<Warning>
Mulailah dengan mode Test untuk memvalidasi pengaturan Anda sebelum beralih ke mode Live untuk data produksi.
</Warning>
</Step>

<Step title="Set Model Reasoning Effort">
Pilih tingkat usaha penalaran model yang Anda sukai:
- **Low**: Respon lebih cepat, cocok untuk tugas sederhana
- **Medium**: Kinerja seimbang (direkomendasikan)
- **High**: Analisis lebih mendalam, cocok untuk integrasi kompleks
</Step>

<Step title="Complete Setup">
Klik **Let's go!** untuk menyelesaikan pengaturan dan mulai menggunakan Sentra.

<Check>
Kunci API Anda disimpan dengan aman di Secret Storage IDE Anda dan tidak pernah terekspos dalam kode Anda.
</Check>
</Step>
</Steps>

## Menggunakan Sentra

### Mode

Sentra menawarkan tiga mode interaksi yang dapat diakses dari antarmuka:

- **Tanya**: Ajukan pertanyaan tentang Dodo Payments, model penagihan, penggunaan API, dan praktik terbaik
- **Bangun**: Integrasikan penagihan dan UI ke dalam aplikasi Anda dengan bantuan terarah
- **Rencanakan**: Rancang arsitektur penagihan yang dapat diskalakan dengan panduan arsitektural

### Tugas Contoh

Berikut adalah beberapa tugas umum yang dapat Anda capai dengan Sentra:

- **Integrasikan Halaman Harga**: Siapkan halaman harga lengkap dengan daftar produk dan checkout
- **Implementasikan Webhook**: Konfigurasikan pengelola webhook untuk peristiwa pembayaran
- **Tambahkan Alur Langganan**: Buat manajemen langganan dengan kemampuan peningkatan/penurunan
- **Siapkan Penagihan Berbasis Penggunaan**: Konfigurasikan meteran dan pengambilan peristiwa untuk pelacakan penggunaan
- **Buat Portal Pelanggan**: Bangun portal layanan mandiri pelanggan untuk manajemen penagihan

### Pengaturan Persetujuan Otomatis

Anda dapat mengonfigurasi pengaturan persetujuan otomatis untuk memperlancar alur kerja Anda:

- **Tidak Ada**: Tinjau dan setujui semua perubahan secara manual (disarankan untuk produksi)
- **Risiko Rendah**: Persetujuan otomatis untuk perubahan sederhana dan berisiko rendah
- **Semua**: Persetujuan otomatis untuk semua perubahan (gunakan dengan hati-hati)

## Model Penagihan yang Didukung

Sentra mendukung semua model penagihan Dodo Payments:

- **Pembayaran Sekali**: Transaksi pembelian tunggal
- **Langganan**: Penagihan berulang dengan beberapa interval
- **Penagihan Berbasis Penggunaan**: Penagihan terukur berdasarkan konsumsi
- **Penagihan Berdasarkan Hasil**: Model harga bayar-per-hasil

## Siapa yang Cocok untuk Ini

Sentra menjembatani teknik dan bisnis, sehingga setiap tim dapat mengirim monetisasi tanpa gesekan:

- **Pengembang**: Kirim monetisasi lebih cepat di dalam IDE Anda
- **Pendiri & PM**: Prototipe model penagihan tanpa pekerjaan infrastruktur yang mendalam
- **RevOps & Keuangan**: Gunakan tindakan terjaga tanpa memerlukan insinyur
- **Dukungan & Sukses**: Selesaikan kasus penagihan dengan aman, dengan persetujuan di mana diperlukan

## Pemecahan Masalah

<AccordionGroup>
<Accordion title="Extension not appearing after installation">
Jika Sentra tidak muncul di IDE Anda setelah instalasi, coba hal berikut:

1. **Mulai ulang editor Anda**: Tutup dan buka kembali VS Code, Cursor, atau Windsurf sepenuhnya
2. **Periksa instalasi**: Pastikan ekstensi terinstal dengan pergi ke Ekstensi dan mencari "Sentra"
3. **Muat ulang jendela**: Gunakan Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`) dan jalankan "Developer: Reload Window"

<Check>
Memulai ulang editor biasanya menyelesaikan masalah.
</Check>
</Accordion>

<Accordion title="Invalid API key or authentication errors">
- Verifikasi bahwa kunci API Anda benar dan aktif di dasbor Dodo Payments
- Pastikan Anda menggunakan mode lingkungan yang sesuai (Test vs Live) yang cocok dengan kunci API Anda
- Periksa bahwa kunci API Anda memiliki izin yang diperlukan
- Coba buat ulang kunci API jika masalah berlanjut
</Accordion>

<Accordion title="Configuration not saving">
- Pastikan Anda telah mengisi semua bidang yang diperlukan di layar konfigurasi
- Periksa bahwa IDE Anda memiliki izin untuk mengakses Secret Storage
- Coba konfigurasikan ulang Sentra melalui Command Palette
</Accordion>

<Accordion title="Plan generation fails">
- Verifikasi kunci API Anda valid dan memiliki akses ke sumber daya yang diperlukan
- Periksa koneksi internet Anda
- Pastikan Anda menggunakan model penagihan yang didukung
- Coba susun ulang prompt Anda dengan detail yang lebih spesifik
</Accordion>

<Accordion title="Execution errors">
- Tinjau rencana yang dihasilkan dengan cermat sebelum menyetujuinya
- Periksa bahwa struktur proyek Anda sesuai dengan ekspektasi Sentra
- Pastikan semua dependensi terinstal
- Periksa panel output IDE untuk pesan kesalahan detail
</Accordion>
</AccordionGroup>

<Warning>
Jika Anda terus mengalami masalah setelah memulai ulang editor, hubungi dukungan dengan rincian versi IDE Anda dan pesan kesalahan spesifik yang muncul.
</Warning>
