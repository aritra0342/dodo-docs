## Fitur Baru 🚀

- **Pembayaran Pertama Langganan**: Fitur yang paling diminta telah hadir! Sekarang, akan ada satu transaksi untuk pembayaran pertama, menggabungkan mandat dan jumlah langganan yang sebenarnya.

- **Metode Pembayaran UPI QR**: Menambahkan UPI QR sebagai metode pembayaran untuk pasar India, memungkinkan pelanggan membayar menggunakan kode QR UPI.
    
    <img src="/images/changelog/upi-qr.png" alt="Fitur UPI QR"/>

- **ID Pajak di Checkout**: Pelanggan sekarang dapat menambahkan ID Pajak mereka di halaman checkout untuk keperluan penagihan dan pajak.

- **Restrukturisasi Dasbor**: Seluruh Dasbor Dodo Payments telah direstrukturisasi agar lebih intuitif dan ramah pengguna.

- **Panduan Onboarding Mandiri**: Pengguna baru sekarang dapat memulai dengan Dodo Payments menggunakan panduan onboarding mandiri.

- **Dukungan Multi-Bahasa**: Menambahkan dukungan untuk 14+ bahasa di halaman checkout, memungkinkan pelanggan membayar dalam bahasa pilihan mereka. Bahasa yang didukung meliputi:
    - English (en)
    - العربية (ar) // Arab
    - Català (ca) // Katalan
    - 中文 (cn) // Mandarin
    - Deutsch (de) // Jerman
    - Español (es) // Spanyol
    - Français (fr) // Prancis
    - עברית (he) // Ibrani
    - Italiano (it) // Italia
    - 日本語 (ja) // Jepang
    - Nederlands (nl) // Belanda
    - Polski (pl) // Polandia
    - Português (pt) // Portugis
    - Svenska (sv) // Swedia

- **Perombakan PDF Faktur**: PDF faktur telah diberikan desain baru yang lebih jelas dan efektif.

- **Widget Bantuan Terpusat**: Menambahkan widget Bantuan terpusat untuk memperlancar dan meningkatkan dukungan.
    
    <img src="/images/changelog/help-widget.png" alt="Widget Bantuan"/>

## Peningkatan dan Perbaikan Bug 🔧

- **Pembatasan Laju**: Menambahkan pembatasan laju di API faktur untuk mencegah penyalahgunaan dan memastikan stabilitas sistem.

- **Metadata di Tautan Pembayaran Statis**: Menambahkan dukungan untuk metadata di tautan pembayaran statis. Untuk detail lebih lanjut, lihat [dokumentasi](/developer-resources/integration-guide#1-static-payment-links).

- **Peningkatan dan Perbaikan Kecil**: Berbagai peningkatan dan perbaikan kecil untuk meningkatkan kinerja keseluruhan dan pengalaman pengguna.
