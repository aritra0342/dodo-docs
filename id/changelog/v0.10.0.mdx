## Fitur Baru
- **Atlas Sumber Terbuka**: Aplikasi demo yang ditulis dalam Next.js untuk menunjukkan integrasi Dodo Payments. Jelajahi [Repositori GitHub](https://github.com/dodopayments/dodo-checkout-demo) dan dokumentasi [di sini](/developer-resources/integration-guide).
- **Bagian Permintaan Fitur**: Memperkenalkan bagian permintaan fitur di dasbor Dodo Payments untuk memprioritaskan peta jalan produk berdasarkan umpan balik pengguna.
- **API Log Permintaan Webhook**: Menambahkan API untuk listing dan merinci log permintaan webhook.
- **Tautan Pembayaran Statis**: Menambahkan lebih banyak parameter kueri ke tautan pembayaran statis untuk memberikan lebih banyak fleksibilitas dalam proses checkout. Untuk detail lebih lanjut, klik [di sini](/developer-resources/integration-guide#1-static-payment-links).
- **API Daftar Langganan**: Menambahkan API untuk mengambil semua langganan. Lihat [Referensi API](/api-reference/subscriptions/get-subscriptions-1) untuk detailnya.

## Perbaikan Kecil dan Perbaikan Bug
- Meningkatkan respons API Produk dan Langganan dengan parameter tambahan.
- Menerapkan pembatasan laju untuk permintaan API, saat ini diatur ke 20 permintaan per detik.
- Meningkatkan dokumentasi agar lebih ramah pengguna dan lebih mudah dinavigasi.
- Berbagai perbaikan bug kecil dan peningkatan.
