## Fitur Baru 🚀

- **Pembayaran Nol Dolar**: Menambahkan dukungan untuk pembayaran $0 dan kode diskon 100% untuk produk satu kali, memungkinkan penawaran produk gratis dan kampanye promosi.

- **Ambang Pembayaran Kustom**: Menambahkan kemampuan untuk menetapkan jumlah pembayaran minimum untuk dompet USD, membantu pedagang mengoptimalkan pemrosesan pembayaran dan mengurangi biaya untuk transaksi kecil. Akses fitur ini di `Payouts ▸ USD Wallet ▸ Edit Payout Threshold`.
  
  <Frame>
    <img src="/images/changelog/payout-threshold.png" alt="Payout Threshold Cover Image"/>
  </Frame>

- **Standarisasi API**: Menerapkan kode dan pesan kesalahan API yang distandarisasi sebagai bagian dari inisiatif perbaikan API yang sedang berlangsung. Peningkatan ini membuat API lebih konsisten dan ramah pengembang. Untuk informasi lebih lanjut, kunjungi [Dokumentasi Kode Kesalahan](/api-reference/error-codes).

- **Standarisasi Penolakan Transaksi**: Memperkenalkan kode dan pesan penolakan transaksi yang seragam, memberikan wawasan yang lebih jelas tentang alasan kegagalan dan membedakan antara kesalahan pengguna dan masalah sistem. Pelajari lebih lanjut di [Dokumentasi Kegagalan Transaksi](/api-reference/transaction-failures).
