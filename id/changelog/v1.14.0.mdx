## Fitur Baru 🚀

- **Metode Pembayaran Eropa Baru**: Kami telah memperluas opsi pembayaran kami dengan empat metode pembayaran Eropa baru:
  - **iDEAL** - Belanda
  - **Bancontact** - Belgia
  - **Przelewy24 (P24)** - Polandia
  - **EPS** - Austria

    Untuk informasi lebih lanjut tentang metode pembayaran ini, kunjungi dokumentasi kami [Metode Pembayaran](/features/payment-methods).
  </Note>

  * Antarmuka portal pelanggan
  * Endpoint API langganan patch
