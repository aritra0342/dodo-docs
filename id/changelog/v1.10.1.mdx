## Fitur Baru 🚀

- **Checkout 2.0**: Memperkenalkan pengalaman checkout yang sepenuhnya dirancang ulang dengan fitur:
    - Antarmuka modern yang ramping untuk meningkatkan pengalaman pengguna
    - Sistem Pengumpul Query baru yang menyimpan parameter dalam penyimpanan sesi dengan ID unik (misalnya, `sess_1a2b3c4d`)
    - Ketersediaan data yang persisten sepanjang alur checkout
    - Lebih banyak peningkatan direncanakan untuk rilis mendatang!
  
    <Frame><img src="/images/changelog/checkout_2.png" alt="Checkout 2.0"/></Frame>

- **Metode Pembayaran yang Dapat Dikonfigurasi**: Konfigurasi baru `allowed_payment_methods` memungkinkan pedagang untuk menyesuaikan opsi pembayaran yang tersedia selama checkout. Lihat dokumentasi [Metode Pembayaran yang Diizinkan](/features/payment-methods#configuring-payment-methods) untuk detail implementasi.

- **Perhitungan Pajak Dinamis**: Perhitungan tarif pajak secara real-time berdasarkan alamat penagihan pelanggan untuk harga yang lebih akurat pada tautan pembayaran statis.

- **Langganan Sesuai Permintaan**: Titik akhir API baru untuk penagihan langganan yang fleksibel (tersedia untuk pedagang tertentu). Pelajari lebih lanjut di panduan [Langganan Sesuai Permintaan](/features/subscription#enabling-on-demand-subscriptions).

- **Pembaruan Tambahan**:
    - Optimisasi kinerja
    - Perbaikan bug
    - Peningkatan sistem secara umum
