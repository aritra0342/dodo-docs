## Fitur Baru 🚀

- **Customer Portal**: Memperkenalkan fitur `Customer Portal`, yang memungkinkan pelanggan untuk mengelola langganan, faktur, dan metode pembayaran mereka. Anda dapat membuat tautan Customer Portal statis atau dinamis untuk pelanggan Anda melalui Dasbor Dodo Payments atau melalui API. Di dasbor, navigasikan ke `Sales > Customers > Share Invite`.

- **Customer Portal Session API**: Menambahkan API `Customer Portal Session` untuk membuat sesi Customer Portal untuk pelanggan tertentu. Perhatikan bahwa sesi ini hanya berlaku selama 24 jam. Periksa [API Reference](/api-reference/customers/create-customer-portal-session) untuk detail lebih lanjut.

- **Lainnya**: Peningkatan kecil dan perbaikan bug.
