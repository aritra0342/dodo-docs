## Fitur Baru 🚀

- **Portal Pelanggan**: Memperkenalkan fitur `Customer Portal` yang memungkinkan pelanggan untuk mengelola langganan, faktur, dan metode pembayaran mereka. Anda dapat membuat tautan Portal Pelanggan statis atau dinamis untuk pelanggan Anda melalui Dasbor Pembayaran Dodo atau melalui API. Di dasbor, navigasikan ke `Sales > Customers > Share Invite`.

- **API Sesi Portal Pelanggan**: Menambahkan API `Customer Portal Session` untuk membuat sesi Portal Pelanggan untuk pelanggan tertentu. Perhatikan bahwa sesi ini hanya berlaku selama 24 jam. Periksa [Referensi API](/api-reference/customers/create-customer-portal-session) untuk detail lebih lanjut.

- **Lainnya**: Peningkatan kecil dan perbaikan bug.
