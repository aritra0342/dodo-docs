## New Features 🚀

- **Customer Portal**: Memperkenalkan fitur `Customer Portal`, yang memungkinkan pelanggan mengelola langganan, faktur, dan metode pembayaran mereka. Anda dapat membuat tautan Customer Portal statis atau dinamis untuk pelanggan melalui Dashboard Dodo Payments atau melalui API. Di dashboard, navigasikan ke `Sales > Customers > Share Invite`.

- **Customer Portal Session API**: Ditambahkan API `Customer Portal Session` untuk membuat sesi Customer Portal bagi pelanggan tertentu. Perlu dicatat bahwa sesi hanya berlaku selama 24 jam. Periksa [Referensi API](/api-reference/customers/create-customer-portal-session) untuk detail lebih lanjut.

- **Lainnya**: Peningkatan kecil dan perbaikan bug.
