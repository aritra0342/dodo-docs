## Fitur Baru
- **Email Otomatis**: Menambahkan email otomatis untuk pembayaran, langganan, pengembalian dana, dan sengketa untuk menjaga agar pedagang dan pelanggan tetap terinformasi.
- **Log Peristiwa Webhook**: Menambahkan log peristiwa webhook di Dasbor Pembayaran Dodo untuk melacak peristiwa webhook dan memecahkan masalah. Akses log di `Settings > Webhooks > Webhook Logs`.
- **Panduan Pembuatan Kunci Lisensi**: Menambahkan panduan baru untuk menerapkan pembuatan kunci lisensi untuk produk digital. Lihat [panduan](/features/license-keys) untuk informasi lebih lanjut.

- Berbagai perbaikan kecil dan peningkatan.
