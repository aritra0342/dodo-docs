## Fitur Baru
- **Email Otomatis**: Menambahkan email otomatis untuk pembayaran, langganan, pengembalian dana, dan sengketa agar pedagang dan pelanggan tetap mendapat informasi.
- **Log Peristiwa Webhook**: Menambahkan log peristiwa webhook di Dasbor Dodo Payments untuk melacak peristiwa webhook dan memecahkan masalah. Arahkan ke `Settings > Webhooks > Webhook Logs` untuk mengakses log.
- **Panduan Pembuatan Kunci Lisensi**: Menambahkan panduan baru untuk menerapkan pembuatan kunci lisensi untuk produk digital. Lihat [panduan](/features/license-keys) untuk informasi lebih lanjut.

- Berbagai perbaikan kecil dan peningkatan.
