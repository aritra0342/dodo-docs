## Fitur Baru 🚀

- **[Adaptive Currency Support](/features/adaptive-currency)** 💱
  - Secara otomatis menampilkan harga dalam mata uang lokal pelanggan berdasarkan lokasi mereka.
  - Pedagang menerima pembayaran dalam mata uang pilihan mereka, sementara pelanggan membayar dalam mata uang lokal mereka.
  - Struktur biaya konversi transparan antara 2% hingga 4%, tergantung nilai pesanan.
  - Dukungan untuk lebih dari 25 mata uang baru telah ditambahkan:
    - **Timur Tengah**: AED (Dirham UEA), SAR (Riyal Saudi)
    - **Asia Pasifik**: AUD (Dolar Australia), CNY (Yuan Tiongkok), HKD (Dolar Hong Kong), IDR (Rupiah Indonesia), KRW (Won Korea Selatan), SGD (Dolar Singapura), TWD (Dolar Taiwan Baru)
    - **Eropa**: ALL (Lek Albania), CHF (Franc Swiss), CZK (Koruna Ceko), DKK (Krone Denmark), EUR (Euro), GBP (Pound Inggris), HUF (Forint Hongaria), NOK (Krone Norwegia), PLN (Zloty Polandia), RON (Leu Rumania), RSD (Dinar Serbia), SEK (Krona Swedia), TRY (Lira Turki)
    - **Amerika**: BRL (Real Brasil), CAD (Dolar Kanada), MXN (Peso Meksiko)

- **[Dodo Payments MCP Server](/developer-resources/mcp-server)** 🤖
  - Memperkenalkan Server Model Context Protocol (MCP) baru untuk integrasi model AI yang mulus.
  - Menyediakan antarmuka standar untuk berinteraksi dengan API DodoPayments.
  - Instalasi sederhana melalui `npx` atau melalui integrasi klien MCP.
  - Menawarkan rangkaian alat lengkap untuk mengelola pembayaran, langganan, dan lainnya.
  - Menyertakan fitur lanjutan seperti penyaringan alat dan kontrol keamanan yang ditingkatkan.

- [Aplikasi Seluler](/features/mobile-app): Sekarang tersedia untuk iOS dan Android, membawa pengalaman dasbor lengkap ke perangkat seluler Anda. Fitur utama termasuk:
  - Notifikasi waktu nyata untuk transaksi, pembayaran, dan pembaruan langganan
  - Akses penuh ke dasbor dengan pelacakan pendapatan dan grafik kinerja
  - Kemampuan manajemen langganan dan produk yang lengkap
  - Operasi bisnis termasuk pengalihan mode uji/hidup dan manajemen verifikasi
  - Alat pengembang dengan manajemen kunci API dan pemantauan webhook
  Unduh sekarang di [App Store](https://apps.apple.com/in/app/dodo-payments/id6743651329) atau [Google Play](https://play.google.com/store/apps/details?id=com.dodopayments.app).

- **GPT Kustom untuk Pembayaran Dodo**: Tersedia di Marketplace ChatGPT, dilatih berdasarkan dokumentasi kami untuk membantu dengan pertanyaan atau masalah teknis. Akses di sini, https://dodo.pe/custom-gpt

- **Dodo Dash**: Rasakan sensasi permainan pelari tanpa akhir kami, Dodo Dash! Pandu burung dodo yang berani saat ia berlari, melompat, dan menghindari rintangan untuk bertahan hidup. Coba di sini, https://dash.dodopayments.com.

- Berbagai perbaikan bug dan peningkatan lainnya telah diterapkan.
