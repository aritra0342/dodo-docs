### Adapter Next.js

Integrasikan Dodo Payments ke dalam aplikasi Next.js Anda dengan mudah menggunakan adapter resmi kami. Siapkan pemrosesan pembayaran hanya dalam beberapa baris kode. Anda juga dapat menggunakan prompt LLM siap pakai kami dengan agen pengkodean AI (seperti Cursor, Windsurf, Lovable, dan lainnya) untuk mengotomatiskan proses integrasi.

[Lihat paket @dodopayments/nextjs di npm](https://www.npmjs.com/package/@dodopayments/nextjs)

### Integrasi Context7

Akses dokumentasi Dodo Payments melalui MCP Context7, memungkinkan agen pengkodean AI untuk memanfaatkan konteks pemrosesan pembayaran yang komprehensif. Untuk informasi lebih lanjut, kunjungi [dokumentasi Context7](https://context7.com/llmstxt/dodopayments-llms-full.txt).
