## Fitur Baru 🚀

- **Halaman Langganan Khusus**: Kelola langganan pelanggan dengan mudah menggunakan halaman khusus yang baru. Navigasikan ke `Sales > Subscriptions` di dasbor.
  
    <Frame><img src="/images/changelog/subscriptions.png" alt="Halaman Langganan"/></Frame>

- **Detail Pelanggan yang Ditingkatkan**: Lihat langganan, pembayaran, dan pengembalian dana untuk setiap pelanggan di satu tempat. Pergi ke `Sales > Customers > arrow icon` di dasbor.

    <Frame><img src="/images/changelog/customer_info.png" alt="Detail Pelanggan"/></Frame>

- **Informasi Pembayaran yang Detail**: Dapatkan lebih banyak wawasan tentang jumlah pembayaran, siklus, dan tanggal. Kunjungi `Business > Payouts` di dasbor.

    <Frame><img src="/images/changelog/payouts_update.png" alt="Informasi Pembayaran"/></Frame>

- **SDK Ruby Resmi**: Integrasikan Dodo Payments ke dalam aplikasi Ruby Anda dengan mudah. Lihat [SDK Ruby](https://github.com/dodopayments/dodopayments-ruby) untuk informasi lebih lanjut.

- **Filter API yang Ditingkatkan**: Menambahkan filter seperti `customer_id`, `status`, dll., ke API daftar yang relevan. Lihat dokumentasi API untuk informasi lebih lanjut.

- **Perbaikan Kecil dan Perbaikan Bug**: Berbagai perbaikan kecil dan perbaikan bug lainnya telah diterapkan.
