## Fitur Baru 🚀

- **Halaman Langganan Khusus**: Kelola langganan pelanggan dengan mudah melalui halaman khusus baru. Arahkan ke `Sales > Subscriptions` di dasbor.
  
    <Frame><img src="/images/changelog/subscriptions.png" alt="Subscriptions Page"/></Frame>

- **Rincian Pelanggan yang Ditingkatkan**: Lihat langganan, pembayaran, dan pengembalian dana untuk setiap pelanggan dalam satu tempat. Pergi ke `Sales > Customers > arrow icon` di dasbor.

    <Frame><img src="/images/changelog/customer_info.png" alt="Customer Details"/></Frame>

- **Informasi Pembayaran yang Rinci**: Dapatkan wawasan lebih lanjut mengenai jumlah pembayaran, siklus, dan tanggal. Kunjungi `Business > Payouts` di dasbor.

    <Frame><img src="/images/changelog/payouts_update.png" alt="Payout Information"/></Frame>

- **SDK Ruby Resmi**: Integrasikan Dodo Payments ke dalam aplikasi Ruby Anda dengan mudah. Lihat [SDK Ruby](https://github.com/dodopayments/dodopayments-ruby) untuk informasi lebih lanjut.

- **Filter API yang Ditingkatkan**: Menambahkan filter seperti `customer_id`, `status`, dll., ke API daftar yang relevan. Lihat dokumentasi API untuk informasi lebih lanjut.

- **Perbaikan Kecil dan Perbaikan Bug**: Berbagai perbaikan kecil dan perbaikan bug lainnya telah diterapkan.
