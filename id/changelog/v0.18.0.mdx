## Fitur Baru 🚀

- **Dukungan Obrolan AI**: Sekarang Anda dapat mengobrol dengan asisten AI kami untuk mendapatkan bantuan dengan pertanyaan teknis Anda atau memeriksa dokumentasi dengan lebih efektif.
    <img src="/images/changelog/ai_chat.png" alt="Dukungan Obrolan AI Dodo Payments" />

- **API yang Ditingkatkan**: Menambahkan API untuk memfilter pembayaran, pengembalian dana, langganan, dan sengketa berdasarkan `customer_id`, periode waktu, dan status, memberikan lebih banyak fleksibilitas dalam pengambilan data.

- **Harga Pajak Inklusif**: Menambahkan dukungan untuk membuat harga produk termasuk pajak saat membuat/memperbarui produk. Jumlah pajak akan dihitung secara otomatis berdasarkan tarif pajak.

- **Pemilihan Negara Otomatis**: Mengimplementasikan pemilihan negara otomatis berbasis IP dalam tautan pembayaran statis. Negara akan dipilih secara otomatis berdasarkan alamat IP pengguna saat mereka membuka tautan pembayaran statis.

- **Penghapusan Parameter Kuery**: Parameter kuery sekarang dihapus segera setelah tautan pembayaran statis dimuat, mencegah pelanggan akhir mengubah parameter URL dan merusak tautan pembayaran statis.

## Peningkatan dan Perbaikan Bug 🔧

- **Peningkatan dan Perbaikan Kecil**: Berbagai peningkatan dan perbaikan kecil untuk meningkatkan kinerja keseluruhan dan pengalaman pengguna.
