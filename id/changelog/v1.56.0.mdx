## Fitur Baru 🚀

1. **Pengalaman Onboarding Baru**  
   Onboarding kini lebih efisien sehingga Anda dapat memulai lebih cepat dan mulai menerima pembayaran tanpa penundaan yang tidak perlu.
   - Aktivasi akun lebih cepat
   - Proses verifikasi yang disederhanakan

2. **Autentikasi Dua Faktor (2FA)**  
   Keamanan akun yang ditingkatkan dengan dukungan autentikasi dua faktor. Lindungi akun merchant Anda dan anggota tim dengan lapisan keamanan tambahan di luar kata sandi.
   - Amankan akun Anda dari akses yang tidak sah
   - Dukungan untuk aplikasi autentikator
   - Diperlukan untuk operasi sensitif

3. **Tautan Sesi di Checkout Overlay**  
   Sesi checkout kini terintegrasi dengan mulus dengan checkout overlay, memberikan pengalaman pembayaran yang lebih halus. Buat sesi checkout dan tampilkan dalam overlay tanpa mengalihkan pelanggan dari aplikasi Anda.

4. **C# SDK**  
   Dodo Payments C# SDK resmi kini tersedia, membawa dukungan .NET asli untuk mengintegrasikan fungsionalitas pembayaran ke dalam aplikasi C# dan .NET Anda.
   - Keamanan tipe penuh dengan antarmuka C#
   - Dukungan Async/await untuk semua operasi API
   - Tersedia di NuGet untuk instalasi yang mudah

   Pelajari lebih lanjut: [Dodo Payments SDKs](/developer-resources/dodo-payments-sdks)

5. **Verifikasi Webhook di Semua SDK**  
   Semua SDK kini menyertakan fungsi verifikasi webhook bawaan untuk memvalidasi acara webhook yang masuk dengan aman. Didukung oleh [perpustakaan webhook](https://github.com/standard-webhooks) standar industri, fungsi ini memudahkan untuk memverifikasi dan memproses payload webhook.
   
   Dua metode tersedia:
   - `unwrap()`: Memverifikasi tanda tangan webhook menggunakan kunci rahasia webhook Anda untuk penggunaan produksi
   - `unsafe_unwrap()`: Mengurai payload webhook tanpa verifikasi untuk pengujian dan pengembangan lokal
   
   Untuk menggunakan verifikasi webhook, berikan `DODO_PAYMENTS_WEBHOOK_KEY` saat menginisialisasi klien Dodo Payments. Kedua fungsi secara otomatis mengurai payload webhook dan mengekstrak jenis acara, membuat penanganan webhook lebih aman dan nyaman.

## Peningkatan & Pembaruan ✨

- **Perbarui nama pelanggan dalam detail penagihan langganan**  
  Nama pelanggan kini dapat dimodifikasi langsung dalam detail penagihan langganan, memudahkan untuk memperbaiki informasi atau memperbarui catatan.

- **API Dapatkan Sesi Checkout**  
  Ambil detail sesi checkout secara programatis untuk memeriksa status, memvalidasi sesi, atau mengintegrasikan dengan alur kerja analitik Anda.

- **Dukungan perubahan rencana untuk langganan berbasis penggunaan**  
  Perubahan rencana langganan kini sepenuhnya didukung untuk model penagihan berbasis penggunaan, memungkinkan peningkatan dan penurunan yang mulus sambil mempertahankan pelacakan penggunaan.

- **ID Pajak dalam respons langganan**  
  Nomor identifikasi pajak kini disertakan dalam respons API langganan, meningkatkan pelaporan kepatuhan dan pembuatan faktur.

- **Paksa bendera 3DS dalam API**  
  Kontrol 3D Secure yang ditingkatkan dengan bendera paksa 3DS kini tersedia dalam sesi checkout, pembayaran, dan API langganan untuk persyaratan autentikasi yang lebih kuat.

- **Filter buku besar saldo berdasarkan mata uang**  
  Dasbor kini mendukung penyaringan entri buku besar saldo berdasarkan mata uang, menyederhanakan rekonsiliasi untuk operasi multi-mata uang.

- **Ambang pembayaran dikurangi menjadi \$50**  
  Merchant kini dapat menetapkan ambang pembayaran serendah \$50, memberikan lebih banyak fleksibilitas untuk bisnis kecil dan akses yang lebih cepat ke dana.

- **Peningkatan kecil dan perbaikan bug**  
  Peningkatan umum dan perbaikan stabilitas di seluruh platform.
