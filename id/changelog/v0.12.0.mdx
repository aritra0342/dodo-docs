## Fitur Baru
- **Official SDKs**: Memperkenalkan SDK resmi Dodo Payments untuk **Node.js**, **Python**, dan **Go**. Lihat halaman [SDKs](/developer-resources/dodo-payments-sdks) untuk informasi lebih lanjut.
- **Metadata Field**: Menambahkan field `Metadata` untuk pembayaran dan langganan guna menyimpan informasi tambahan. Lihat halaman [Metadata](/api-reference/metadata) untuk detail lebih lanjut.
- **Customer APIs**: API pelanggan diperluas untuk mencakup endpoint berikut. Anda sekarang dapat menyertakan `customer_id` dalam pembayaran dan langganan untuk mencegah duplikasi pelanggan:
    - **Create Customer**: [/api-reference/customers/create-customer](/api-reference/customers/create-customer)
    - **Patch Customer**: [/api-reference/customers/patch-customer](/api-reference/customers/patch-customer)
