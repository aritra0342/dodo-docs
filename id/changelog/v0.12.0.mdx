## Fitur Baru
- **SDK Resmi**: Memperkenalkan SDK Dodo Payments resmi untuk **Node.js**, **Python**, dan **Go**. Lihat halaman [SDKs](/developer-resources/dodo-payments-sdks) untuk informasi lebih lanjut.
- **Field Metadata**: Menambahkan field `Metadata` untuk pembayaran dan langganan untuk menyimpan informasi tambahan. Lihat halaman [Metadata](/api-reference/metadata) untuk detail lebih lanjut.
- **API Pelanggan**: Memperluas API pelanggan untuk menyertakan endpoint berikut. Anda sekarang dapat melewatkan `customer_id` dalam pembayaran dan langganan untuk mencegah duplikasi pelanggan:
    - **Buat Pelanggan**: [/api-reference/customers/create-customer](/api-reference/customers/create-customer)
    - **Patch Pelanggan**: [/api-reference/customers/patch-customer](/api-reference/customers/patch-customer)
