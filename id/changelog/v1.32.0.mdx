## Fitur Baru 🚀

- **Integrasi Program Afiliasi**: Diluncurkan integrasi dengan Affonso, memungkinkan Anda untuk membuat dan mengelola program afiliasi langsung melalui Dodo Payments. Lacak rujukan, kelola komisi, dan tingkatkan pendapatan Anda dengan kemitraan afiliasi yang terpercaya. Pelajari lebih lanjut di dokumentasi kami [Afiliasi](/features/affiliates).
  <Frame>
    <img src="/images/affiliates/cover.jpeg" alt="Gambar Sampul Afiliasi"/>
  </Frame>

- **Dukungan Webhook Ganda**: Meningkatkan kemampuan webhook untuk mendukung beberapa endpoint, memungkinkan Anda mengirim notifikasi acara ke sistem yang berbeda secara bersamaan.

- **Peningkatan Pencegahan Penipuan**: Menambahkan pengaturan bisnis baru untuk memaksa autentikasi 3D Secure untuk semua transaksi, membantu mengurangi penipuan dan pengembalian dana.

- **Pembatalan Jadwal Langganan**: Memperkenalkan kemampuan untuk menjadwalkan pembatalan langganan yang akan berlaku pada akhir periode penagihan, tersedia melalui API kami.

- **Tautan Pembayaran Statis**: Menambahkan dua parameter kueri lagi untuk tautan pembayaran statis:
  - `paymentCurrency`: Menentukan mata uang pembayaran. Default diatur sesuai dengan negara penagihan.
  - `showCurrencySelector`: Menampilkan pemilih mata uang. Default adalah true.
