Kelola pembayaran, langganan, dan aktivitas bisnis Anda — kapan saja, di mana saja. **Aplikasi Mobile Dodo Payments** membawa kekuatan Dasbor Dodo ke ujung jari Anda, memungkinkan Anda tetap terhubung dengan pendapatan, pelanggan, dan aktivitas Anda secara real-time.

Apakah Anda sedang dalam perjalanan, bepergian, atau hanya jauh dari meja Anda, Anda tidak akan pernah melewatkan pembaruan penting lagi.

## Paritas dengan Dasbor Desktop

Semua fitur dari versi desktop sepenuhnya tersedia di aplikasi mobile, termasuk:

- Mode Uji dan Mode Langsung
- Progres dan pengiriman verifikasi
- Pengaturan pengembang dan visibilitas kunci API
- Konfigurasi Harga dan Diskon
