<Info>
Halaman Desain adalah pusat utama Anda untuk membangun identitas merek di semua permukaan yang dihadapi pelanggan: checkout, storefront, dan portal pelanggan. Perubahan diterapkan secara instan di ketiganya dan dipratinjau secara real time sebelum Anda menyimpannya.
</Info>

## Sorotan Utama

1. **Pusat Desain Terpadu** - Atur tampilan checkout, storefront, dan portal pelanggan dari satu halaman.
2. **Tema Bawaan** - Mulai dengan tema yang kurasi dan sesuaikan dari sana.
3. **Mode Terang & Gelap** - Tentukan palet warna terpisah untuk masing-masing mode.
4. **Pratinjau Langsung** - Lihat bagaimana perubahan muncul di checkout, portal pelanggan, dan storefront sebelum menyimpan.
5. **Penimpaan per-Seksi** - Sesuaikan bagian tertentu (checkout, storefront, portal pelanggan) tanpa memengaruhi bagian lainnya.
6. **Kontrol Programatik** - Timpa tema saat checkout melalui API atau Checkout SDK.

## Memulai

Navigasikan ke **Desain** di sidebar utama Merchant Dashboard Anda. Halaman Desain memiliki empat tab:

| Tab | Tujuan |
|-----|---------|
| **Umum** | Nama bisnis, logo, pemilihan tema, dan pengaturan lanjutan global |
| **Checkout** | Timpa pengaturan tema secara khusus untuk halaman checkout |
| **Storefront** | Timpa pengaturan tema dan konfigurasikan tata letak storefront |
| **Portal Pelanggan** | Timpa pengaturan tema untuk portal pelanggan |

## Pengaturan Umum

Tab **Umum** mengendalikan identitas merek dasar dan tema global Anda.

<Frame>
  <img src="/images/design/general-advanced-settings.jpg" alt="Tab Umum menunjukkan pengaturan lanjutan dengan tipografi, warna, dan konfigurasi batas"/>
</Frame>

### Konfigurasi Dasar

| Pengaturan | Deskripsi |
|---------|-------------|
| **Nama Bisnis** | Nama yang ditampilkan di checkout, storefront, dan portal pelanggan |
| **Logo Bisnis** | Logo merek Anda yang ditampilkan di semua halaman yang dihadapi pelanggan |
| **Tema** | Pilih tema bawaan sebagai titik awal Anda |

### Pengaturan Lanjutan

Perluas **Pengaturan Lanjutan** untuk mengakses kontrol granular:

<Frame>
  <img src="/images/design/general-color-settings.jpg" alt="Pengaturan lanjutan diperluas menunjukkan konfigurasi warna penuh untuk mode terang dan gelap"/>
</Frame>

#### Tipografi

| Pengaturan | Opsi | Deskripsi |
|---------|---------|-------------|
| **Font Utama** | Any Google Font | Font utama untuk judul dan teks utama |
| **Font Sekunder** | Any Google Font | Font aksen untuk label dan teks sekunder |
| **Ukuran Font** | `xs`, `sm`, `md`, `lg`, `xl`, `2xl` | Ukuran font dasar global |
| **Ketebalan Font** | `normal`, `medium`, `bold`, `extraBold` | Ketebalan font default |

#### Konfigurasi Warna

Tentukan palet warna terpisah untuk **Mode Terang** dan **Mode Gelap**. Setiap mode mendukung properti berikut:

| Properti | Deskripsi |
|----------|-------------|
| **Latar Belakang Utama** | Warna latar belakang utama |
| **Latar Belakang Sekunder** | Latar belakang sekunder (tab, kartu) |
| **Teks Utama** | Warna teks utama |
| **Teks Sekunder** | Warna teks yang lembut |
| **Tombol Utama** | Latar belakang tombol utama |
| **Teks Tombol Utama** | Warna teks pada tombol utama |
| **Tombol Sekunder** | Latar belakang tombol sekunder |
| **Teks Tombol Sekunder** | Warna teks pada tombol sekunder |
| **Border Utama** | Warna border utama |
| **Border Sekunder** | Warna border aksen |

#### Batas

| Pengaturan | Deskripsi |
|---------|-------------|
| **Radius Batas** | Mengontrol kelengkungan input, tombol, tab, dan kartu (misalnya, `0` untuk sudut tajam, `8px` untuk melengkung, `16px` untuk bentuk pil) |

## Penimpaan per-Seksi

Setiap tab seksi (Checkout, Storefront, Portal Pelanggan) menyertakan toggle **Timpa Tema**. Saat diaktifkan, Anda dapat menyesuaikan tampilan seksi tersebut secara mandiri tanpa mengikuti tema global.

### Tab Checkout

<Frame>
  <img src="/images/design/checkout-tab.jpg" alt="Tab Checkout menampilkan pratinjau penuh checkout dengan detail produk dan formulir pembayaran"/>
</Frame>

Timpa tipografi, warna, dan pengaturan batas secara khusus untuk pengalaman checkout. Pratinjau langsung menampilkan:
- Informasi produk dan harga
- Opsi checkout ekspres (Apple Pay, Google Pay)
- Pilihan metode pembayaran dan formulir kartu
- Input kode diskon dan rincian harga

### Tab Storefront

<Frame>
  <img src="/images/design/storefront-tab.jpg" alt="Tab Storefront menampilkan URL toko, kontrol branding, dan pratinjau storefront"/>
</Frame>

Atur pengaturan khusus storefront:

| Pengaturan | Deskripsi |
|---------|-------------|
| **URL Toko** | Alamat storefront Anda (misalnya, `store.dodopayments.com/your-store`) |
| **Nama Toko** | Aktifkan dan atur nama yang ditampilkan di storefront |
| **Logo Toko** | Aktifkan dan unggah logo untuk header storefront |
| **Banner Toko** | Aktifkan dan unggah gambar banner |
| **Deskripsi Produk** | Aktifkan untuk menampilkan atau menyembunyikan deskripsi produk di kartu |
| **Timpa Tema** | Aktifkan untuk menyesuaikan tipografi, warna, dan batas hanya untuk storefront |

### Tab Portal Pelanggan

<Frame>
  <img src="/images/design/customer-portal-tab.jpg" alt="Tab portal pelanggan menampilkan manajemen langganan, riwayat tagihan, dan pratinjau metode pembayaran"/>

Timpa tema untuk portal pelanggan, yang mencakup:
- Manajemen langganan
- Riwayat penagihan
- Metode pembayaran
- Dompet dan kredit
