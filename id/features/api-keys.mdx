## Pendahuluan

Generasi Kunci API memungkinkan pedagang untuk mengautentikasi dan mengintegrasikan sistem mereka dengan platform Dodo Payments secara aman untuk akses yang mulus ke fungsi pembayaran, produk, dan langganan.

**Catatan :** Kunci API yang dibuat dalam mode Uji akan dibatasi pada lingkungan mode uji dan sebaliknya untuk mode langsung
