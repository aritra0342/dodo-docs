## Mata Uang Adaptif

Mata Uang Adaptif memungkinkan pelanggan untuk membayar dalam mata uang lokal yang mereka pilih, meningkatkan kepercayaan dan tingkat konversi.

### Cara Kerjanya

1. **Aktifkan**: Aktifkan Mata Uang Adaptif dari Pengaturan Bisnis Anda
2. **Pilih**: Pelanggan dapat mengganti mata uang langsung di checkout
3. **Konversi**: Harga dikonversi secara dinamis menggunakan kurs FX waktu nyata
4. **Tampilkan**: Jumlah yang harus dibayar ditampilkan secara transparan sebelum pembayaran

## Perhitungan Pajak Otomatis

Pajak dihitung secara otomatis berdasarkan lokasi penagihan pelanggan, memastikan kepatuhan terhadap persyaratan GST, VAT, dan pajak penjualan tanpa pengaturan manual.

### Cara Kerja Perhitungan Pajak

<Steps>
<Step title="Deteksi Lokasi">
  Aturan pajak diterapkan berdasarkan negara pelanggan (dan wilayah jika berlaku).
</Step>

<Step title="Pembaruan Dinamis">
  Jumlah pajak diperbarui secara otomatis ketika:
  - Negara berubah
  - Alamat diperbarui
</Step>

<Step title="Tampilan Transparan">
  Rincian pajak akhir ditampilkan dengan jelas sebelum pembayaran.
</Step>
</Steps>

<Tip>
Perhitungan pajak sepenuhnya otomatis. Tidak ada konfigurasi manual yang diperlukan untuk barang digital standar dan produk SaaS.
</Tip>

## Dukungan ID Pajak Bisnis

Untuk bisnis terdaftar, checkout memungkinkan pelanggan untuk memasukkan ID Pajak Bisnis mereka (misalnya, nomor VAT/GST).

### Apa yang Terjadi Ketika ID Pajak Dimasukkan

- Kelayakan pajak divalidasi secara waktu nyata
- Pengecualian pajak yang berlaku atau aturan reverse-charge diterapkan
- Jumlah pajak diperbarui secara instan di checkout

## Kode Diskon

Pelanggan dapat menerapkan kode diskon atau promo yang telah Anda buat di dasbor langsung di halaman checkout.

### Pengalaman Checkout

1. Pelanggan memasukkan kode diskon
2. Diskon divalidasi secara instan
3. Harga dan penghematan yang diperbarui ditampilkan dengan jelas

<Frame>
  <img src="/images/checkout/c4.png" alt="Masukan kode diskon di checkout" style={{ width: '70%' }}/>
</Frame>

### Integrasi API

Terapkan kode diskon sebelumnya atau aktifkan kolom input diskon:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  discount_code: 'WELCOME20', // Pre-apply a code
  feature_flags: {
    allow_discount_code: true // Show discount input field
  },
  return_url: 'https://yoursite.com/return'
});
```

## Pengumpulan Alamat Cerdas

Checkout mendukung entri alamat yang fleksibel untuk penyelesaian yang lebih cepat.

### Opsi yang Tersedia

| Opsi | Deskripsi |
|--------|-------------|
| **Google Address Autofill** | Pemilihan cepat dengan autocomplete |
| **Entri Manual** | Kontrol penuh untuk alamat lengkap |
| **Pemilihan Negara** | Menggerakkan logika pajak dan kepatuhan |

<Tip>
Pengumpulan alamat menyeimbangkan kecepatan, akurasi, dan cakupan global untuk memaksimalkan konversi sambil memastikan kepatuhan.

## Checkout Koleksi

Koleksi Produk memungkinkan pengalaman checkout yang terpadu di mana pelanggan dapat melihat dan memilih dari beberapa produk terkait (misalnya, paket Starter, Pro, Enterprise) dalam satu checkout.

### Cara Kerjanya

1. **Semua produk ditampilkan**: Pelanggan dapat melihat setiap produk aktif dalam koleksi
2. **Produk pertama dipilih secara otomatis**: Produk pertama dalam koleksi secara otomatis dipilih
3. **Bandingkan opsi**: Pelanggan dapat membandingkan harga dan fitur sebelum memilih
4. **Pemilihan tunggal**: Setelah memilih produk, proses checkout dilanjutkan dengan alur pembayaran standar

### Membuat Checkout Koleksi

```typescript
const session = await client.checkoutSessions.create({
  product_collection_id: 'pdc_abc123',
  product_cart: [], // Required: pass an empty array for collection checkout
  return_url: 'https://yoursite.com/return'
});
```

<Warning>
Saat menggunakan `product_collection_id`, kirimkan array kosong `product_cart`. Kode diskon tidak dapat diterapkan sebelumnya saat pembuatan sesi.
</Warning>

## Konfigurasi Sesi Checkout

Kontrol perilaku checkout menggunakan API Sesi Checkout:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  customer: {
    email: 'customer@example.com',
    name: 'Jane Doe'
  },
  billing_currency: 'EUR', // Set specific currency
  discount_code: 'PROMO10',
  feature_flags: {
    allow_discount_code: true
  },
  return_url: 'https://yoursite.com/return',
  metadata: {
    order_ref: 'ORD-12345'
  }
});
```
