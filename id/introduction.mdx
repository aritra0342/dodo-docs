## Apa itu Dodo Payments?

Dodo Payments adalah **mesin all-in-one** untuk meluncurkan, mengembangkan, dan memonetisasi secara global. Dirancang untuk generasi baru produk digital SaaS dan AI, kami menangani semuanya mulai dari **pembayaran hingga penagihan hingga distribusi** - semua tanpa rekayasa tambahan.

<Info>
**Dibangun untuk Produk SaaS dan AI**: Tangani pembayaran satu kali, langganan, kredit, kursi, token, atau metrik penggunaan lainnya dengan infrastruktur penagihan kami yang dioptimalkan.
</Info>

<Tip>
  <strong>Kenali Sentra: Asisten penagihan bertenaga AI Anda</strong><br/>
  Sentra adalah agen AI yang mengintegrasikan, menganalisis, dan bertindak pada penagihan dan pembayaran langsung di IDE Anda. Ajukan pertanyaan, bangun integrasi, atau rencanakan arsitektur penagihan - semua dengan menjelaskan apa yang Anda inginkan.
  <a href="/developer-resources/sentra">
    Mulai dengan Sentra &rarr;
  </a>
</Tip>

Dapatkan Dodo Payments berjalan hanya dalam beberapa langkah:

<Steps>
<Step title="Buat Akun Anda">
  Daftar di [app.dodopayments.com](https://app.dodopayments.com/signup) menggunakan email Anda.

  <Check>
  Anda akan menerima email konfirmasi untuk memverifikasi akun Anda.
  </Check>
</Step>

<Step title="Tambahkan Produk Pertama Anda">
  Pilih model penagihan Anda dan tambahkan produk Anda untuk mulai menerima pembayaran.

### Server MCP untuk Asisten AI

Hubungkan asisten pengkodean AI Anda (Cursor, Claude Desktop, Windsurf, VS Code) langsung ke Dodo Payments dengan server MCP kami.

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  Pencarian semantik di dokumentasi Dodo Payments - dapatkan jawaban instan tentang fitur, API, dan praktik terbaik
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  Lakukan operasi API secara langsung - buat pembayaran, kelola langganan, dan tangani pengembalian dana melalui asisten AI Anda
</Card>
</CardGroup>

<Accordion title="Pengaturan MCP Cepat untuk Cursor/Claude Desktop">
Tambahkan kedua server ke konfigurasi MCP Anda (`~/.cursor/mcp.json` atau konfigurasi Claude Desktop):

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
Membutuhkan Node.js 18+. Lihat [dokumentasi Server MCP](/developer-resources/mcp-server) untuk instruksi pengaturan rinci untuk semua klien yang didukung.
</Info>
</Accordion>
