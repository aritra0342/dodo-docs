## Apa itu Dodo Payments?

Dodo Payments adalah **mesin all-in-one** untuk meluncurkan, mengembangkan, dan memonetisasi secara global. Dirancang untuk generasi baru produk digital SaaS dan AI, kami menangani semuanya mulai dari **pembayaran hingga penagihan hingga distribusi** - semua tanpa rekayasa tambahan.

<Info>
**Dibuat untuk Produk SaaS dan AI**: Tangani pembayaran sekali, langganan, kredit, kursi, token, atau metrik penggunaan apa pun dengan infrastruktur penagihan kami yang dioptimalkan.
</Info>

<Tip>
  <strong>Temui Sentra: Asisten penagihan bertenaga AI Anda</strong><br/>
  Sentra adalah agen AI yang mengintegrasikan, menganalisis, dan mengambil tindakan terhadap penagihan dan pembayaran langsung di IDE Anda. Ajukan pertanyaan, bangun integrasi, atau rencanakan arsitektur penagihan — semuanya dengan mendeskripsikan apa yang Anda inginkan.
  <a href="/developer-resources/sentra">
    Mulai dengan Sentra &rarr;
  </a>
</Tip>

### Server MCP untuk Asisten AI

Hubungkan asisten pengkodean AI Anda (Cursor, Claude Desktop, Windsurf, VS Code) langsung ke Dodo Payments dengan server MCP kami.

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  Pencarian semantik di seluruh dokumentasi Dodo Payments - dapatkan jawaban instan tentang fitur, API, dan praktik terbaik
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  Jalankan operasi API secara langsung - buat pembayaran, kelola langganan, dan tangani pengembalian dana melalui asisten AI Anda
</Card>
</CardGroup>

{/* LOCKED_PATTERN_8ee343b87e057252cc4a8a376b190970 */}
Tambahkan kedua server ke konfigurasi MCP Anda (`~/.cursor/mcp.json` atau Claude Desktop config):

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
Membutuhkan Node.js 18+. Lihat [dokumentasi Server MCP](/developer-resources/mcp-server) untuk petunjuk pengaturan terperinci untuk semua klien yang didukung.
</Info>
</Accordion>
