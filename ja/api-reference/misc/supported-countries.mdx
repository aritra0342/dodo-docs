---

<Tip> 現在**支払いに対応している**国を表す[ISO Alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2)の国コードの一覧を受け取ります。対応国の完全な一覧は[こちら](/miscellaneous/list-of-countries-we-accept-payments-from)からご確認いただけます。対応範囲を拡大するにつれて、この一覧は定期的に更新されます。 </Tip>

<Tip>注: 利用可能なオプションには、現在対応している国だけでなく、**すべての国**のISO Alpha-2コードが含まれます。対応状況は対応国の一覧でご確認ください。 </Tip>
