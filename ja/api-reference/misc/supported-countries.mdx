title: サポートされている国
openapi: get /checkout/supported_countries
description: お客様のためにDodo Paymentsがサポートしている国のリストを取得します。
---

<Tip> 現在私たちが**支払いをサポートしている**国を表す[ISO Alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2)国コードのリストを受け取ります。サポートされている国の完全なリストは[こちら](/miscellaneous/list-of-countries-we-accept-payments-from)でご覧いただけます。このリストは、カバレッジを拡大するにつれて定期的に更新されます。 </Tip>

<Tip> 注: 利用可能なオプションには、**すべての国**のISO Alpha-2コードが含まれており、現在サポートしている国だけではありません。利用可能性を確認するには、サポートされている国のリストを参照してください。 </Tip>
