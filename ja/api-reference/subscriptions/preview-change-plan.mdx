---
openapi: post /subscriptions/{subscription_id}/change-plan/preview
title: プラン変更のプレビュー
description: サブスクリプションプランの変更を確定する前に、その影響をプレビューします。実際の変更を行うことなく、即時の請求額と新しいサブスクリプションの詳細を返します。
---

<Info>
このエンドポイントを使用すると、サブスクリプションをアップグレードまたはダウングレードした際に顧客に請求される金額を正確に表示できるため、透明性が向上し、サポートの問い合わせが減少します。
</Info>

## ユースケース

- **チェックアウト確認**: 顧客がプラン変更を確認する前に、按分請求額を表示
- **価格計算機**: アプリケーション内でアップグレード/ダウングレード計算機を構築
- **顧客セルフサービス**: 顧客が正確な価格でプランオプションを探索できるようにする

## レスポンスフィールド

プレビューレスポンスには以下が含まれます:

| フィールド | 説明 |
|------------|------|
| `immediate_charge` | ライン項目と要約を含め、直ちに作成される請求 |
| `new_plan` | プラン変更後の状態を示す完全なサブスクリプションオブジェクト |

<Tip>
`immediate_charge.summary` には請求される合計金額が含まれています。これを使用して、プラン変更を顧客が確認する前に価格を表示してください。
</Tip>
