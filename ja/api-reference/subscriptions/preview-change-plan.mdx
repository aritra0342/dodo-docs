openapi: post /subscriptions/{subscription_id}/change-plan/preview
title: プラン変更のプレビュー
description: サブスクリプションプランの変更を確定する前に、その影響をプレビューします。実際の変更を行うことなく、即時の請求額と新しいサブスクリプションの詳細を返します。
---

<Info>
このエンドポイントを使用して、顧客がサブスクリプションをアップグレードまたはダウングレードする際に正確に請求される金額を示し、透明性を向上させ、サポートリクエストを減少させます。
</Info>

## ユースケース

- **チェックアウト確認**: 顧客がプラン変更を確認する前に、按分請求額を表示
- **価格計算機**: アプリケーション内でアップグレード/ダウングレード計算機を構築
- **顧客セルフサービス**: 顧客が正確な価格でプランオプションを探索できるようにする

## レスポンスフィールド

プレビューレスポンスには以下が含まれます:

| フィールド | 説明 |
|-------|-------------|
| `immediate_charge` | 行項目と概要を含む、即座に作成される請求 |
| `new_plan` | プラン変更後の見た目を示す完全なサブスクリプションオブジェクト |

<Tip>
`immediate_charge.summary` には、請求される合計金額が含まれています。これを使用して、顧客がプラン変更を確認する前に価格を表示します。
</Tip>
