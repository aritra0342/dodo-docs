## Dodo Paymentsとは？

Dodo Paymentsは、**オールインワンエンジン**で、世界中での立ち上げ、スケール、収益化を実現します。新世代のデジタルSaaSおよびAI製品向けに設計されており、**決済から請求、配布まで**すべてを処理します - 追加のエンジニアリングなしで。

<Info>
**Built for SaaS and AI Products**: 当社の最適化された請求インフラで、単発の支払い、サブスクリプション、クレジット、席数、トークン、または任意の利用メトリクスを処理できます。
</Info>

<Tip>
  <strong>Sentraに会いましょう：AI搭載の請求アシスタント</strong><br/>
  SentraはIDE上で請求と支払いを直接統合、分析、実行するAIエージェントです。質問したり、統合を構築したり、請求アーキテクチャを計画したり—いずれも希望する内容を説明するだけで行えます。
  <a href="/developer-resources/sentra">
    Sentraを始める &rarr;
  </a>
</Tip>

### AIアシスタント用MCPサーバー

AIコーディングアシスタント（Cursor、Claude Desktop、Windsurf、VS Code）を、当社のMCPサーバーを介してDodo Paymentsに直接接続します。

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  Dodo Paymentsのドキュメント全体をセマンティック検索—機能、API、ベストプラクティスについてすぐに回答を得られます
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  AIアシスタント経由でAPI操作を直接実行—支払いの作成、サブスクリプション管理、返金処理など
</Card>
</CardGroup>

{/* LOCKED_PATTERN_8ee343b87e057252cc4a8a376b190970 */}
両方のサーバーをMCP構成 (`~/.cursor/mcp.json` または Claude Desktop config) に追加してください:

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
Node.js 18+ が必要です。対応クライアントすべての詳細なセットアップ手順は [MCP Server documentation](/developer-resources/mcp-server) を参照してください。
</Info>
</Accordion>
