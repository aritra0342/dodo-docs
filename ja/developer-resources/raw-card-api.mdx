<Note>
このガイドでは、**カード、Apple Pay、Google Pay** を使用した**ワンタイム支払い**のみを扱います。
</Note>

## API エンドポイント

| 環境 | ベース URL | 支払い確定エンドポイント |
| --- | --- | --- |
| **テストモード** | `https://test.dodopayments.com` | `https://api.custom.hs.dodopayments.com/payments/<payment_id>/confirm` |
| **ライブモード** | `https://live.dodopayments.com` | `https://api.hs.dodopayments.com/payments/<payment_id>/confirm` |

## ステップバイステップのカード支払いフロー

### ステップ 1: シークレットキーの取得

以下の API を使って支払いを作成します。

- **Payments**: [Create Payment API](/api-reference/payments/post-payments)

この API は、以降のステップで必要となる **`secret_key`** を返します。

### ステップ 2: 支払いの確定

以下の `payments confirm` エンドポイントを呼び出して支払いを確定します。

- **テストモード**: `POST https://api.custom.hs.dodopayments.com/payments/<payment_id>/confirm`
- **ライブモード**: `POST https://api.hs.dodopayments.com/payments/<payment_id>/confirm`

リクエストボディには以下の内容を含めてください。

- `client_secret`: ステップ 1 のシークレットキー
- `payment_method`: `card` に設定
- `payment_method_data`: カードおよび請求情報
- `browser_info`: ブラウザ固有のデータ（ユーザーエージェント、画面情報など）
- `payment_type`: 明示的に `normal` に設定（ワンタイム支払いの場合）

必要なヘッダー:

- DodoPayments サポートから公開可能キーを取得
- `api-key`: あなたの公開可能キー（Dodo Payments チームから取得）
- `content-type`: `application/json`

**リクエストボディの例:**

```json
{
  "client_secret": "<secret_key>",
  "payment_method": "card",
  "payment_method_data": {
    "card": {
      "card_number": "<card_number>",
      "card_exp_month": "<exp_month>",
      "card_exp_year": "<exp_year>",
      "card_holder_name": "<cardholder_name>",
      "card_cvc": "<cvc>",
      "card_network": "<network>"
    },
    "billing": {
      "address": {
        "first_name": "<first_name>",
        "last_name": "<last_name>"
      }
    }
  },
  "browser_info": {
    "user_agent": "<user_agent>",
    "accept_header": "<accept_header>",
    "language": "<language>",
    "color_depth": "<color_depth>",
    "screen_height": "<screen_height>",
    "screen_width": "<screen_width>",
    "time_zone": "<time_zone>",
    "java_enabled": true,
    "java_script_enabled": true,
    "ip_address": "<ip_address_of_client>",
    "device_model": "<device_model>",
    "os_type": "<os_type>",
    "os_version": "<version>"
  },
  "payment_type": "normal"
}
```

### レスポンスの処理

- レスポンスの **`status`** が **`requires_customer_action`** の場合、**`next_action`** フィールドに記載された URL に顧客をリダイレクトし、追加の認証を行ってください。

<Warning>
生のカードデータを扱う際には PCI 準拠ガイドラインを遵守してください。
</Warning>

## Apple Pay 連携

Apple Pay を利用するには、`payment_method` を `wallet` に設定し、Apple Pay データを `payment_method_data` に含めます。

### リクエストボディ構造

```json
{
  "client_secret": "<secret_key>",
  "payment_method": "wallet",
  "payment_method_data": {
    "wallet": {
      "apple_pay": {
        "payment_data": "<decrypted_payment_data>",
        "payment_method": {
          "display_name": "Visa 1234",
          "network": "Visa",
          "type": "debit"
        },
        "transaction_identifier": "UNIQUE_TRANSACTION_ID_FROM_APPLE"
      }
    }
  },
  "browser_info": {
    "user_agent": "<user_agent>",
    "accept_header": "<accept_header>",
    "language": "<language>",
    "color_depth": "<color_depth>",
    "screen_height": "<screen_height>",
    "screen_width": "<screen_width>",
    "time_zone": "<time_zone>",
    "java_enabled": true,
    "java_script_enabled": true,
    "ip_address": "<ip_address_of_client>",
    "device_model": "<device_model>",
    "os_type": "<os_type>",
    "os_version": "<version>"
  },
  "payment_type": "normal"
}
```

### Apple Pay フィールド参考

| フィールド | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `payment_data` | string | 必須 | 暗号化済みまたは復号済みの Apple Pay 支払いデータを表す列挙型です。 |
| `payment_method.display_name` | string | 必須 | Apple Pay ボタンに表示される名称（例: "Visa 1234"） |
| `payment_method.network` | string | 必須 | カードネットワーク（例: "Visa", "Mastercard", "Amex"） |
| `payment_method.type` | string | 必須 | 支払い方法タイプ（例: "debit", "credit"） |
| `payment_method.card_exp_month` | string | 任意 | カードの有効月（例: "12"） |
| `payment_method.card_exp_year` | string | 任意 | カードの有効年（例: "25"） |
| `transaction_identifier` | string | 必須 | Apple Pay による固有のトランザクション識別子 |

## Google Pay 連携

Google Pay を利用するには、`payment_method` を `wallet` に設定し、Google Pay データを `payment_method_data` に含めます。

### リクエストボディ構造

```json
{
  "client_secret": "<secret_key>",
  "payment_method": "wallet",
  "payment_method_data": {
    "wallet": {
      "google_pay": {
        "type": "CARD",
        "description": "Visa ending in 1234",
        "info": {
          "card_network": "VISA",
          "card_details": "1234"
        },
        "tokenization_data": {
          "type": "PAYMENT_GATEWAY",
          "token": "<encrypted_payment_token>"
        }
      }
    }
  },
  "browser_info": {
    "user_agent": "<user_agent>",
    "accept_header": "<accept_header>",
    "language": "<language>",
    "color_depth": "<color_depth>",
    "screen_height": "<screen_height>",
    "screen_width": "<screen_width>",
    "time_zone": "<time_zone>",
    "java_enabled": true,
    "java_script_enabled": true,
    "ip_address": "<ip_address_of_client>",
    "device_model": "<device_model>",
    "os_type": "<os_type>",
    "os_version": "<version>"
  },
  "payment_type": "normal"
}
```

### Google Pay フィールド参考

| フィールド | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `type` | string | 必須 | 支払い方法タイプ（通常は "CARD"） |
| `description` | string | 必須 | 利用者に表示される支払い方法の説明 |
| `info.card_network` | string | 必須 | カードネットワーク（例: "VISA", "MASTERCARD", "AMEX"） |
| `info.card_details` | string | 必須 | カードの下4桁 |
| `info.assurance_details` | object | 任意 | 認証保証に関する詳細 |
| `tokenization_data.type` | string | 必須 | トークン化タイプ（例: "PAYMENT_GATEWAY"） |
| `tokenization_data.token` | string | 必須 | Google Pay からの暗号化済み支払いトークン |
