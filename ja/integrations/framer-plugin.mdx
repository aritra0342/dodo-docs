## 概要

Dodo Payments Framer プラグインは、Framer ウェブサイトに支払い処理をシームレスに統合します。テスト環境と本番環境の両方をサポートしたカスタマイズ可能なチェックアウト体験を作成します。

<Card title="公式プラグイン" icon="link" href="https://www.framer.com/marketplace/plugins/dodo-payments">
Framer マーケットプレイスで公式の Dodo Payments プラグインにアクセスします。
</Card>

## セットアップガイド

<Steps>
<Step title="API キーを生成する">
[Dodo Payments ダッシュボード](https://app.dodopayments.com/developer/api-keys)にアクセスし、新しい API キーを作成します。

<Tabs>

<Tab title="ライブモード">
実際の支払い処理を伴う本番用。
</Tab>

<Tab title="テストモード">
実際の請求がない開発およびテストに最適です。
</Tab>

</Tabs>

<Check>
API キーを安全にコピーしてください - 次のステップで必要になります。
</Check>
</Step>

<Step title="プラグインを認証する">
1. Framer で Dodo Payments プラグインを開く
2. 認証ページに移動する
3. API キーを貼り付ける
4. 適切なモードを選択する（テストまたはライブ）
5. 認証するには **続行** をクリックします。

<Warning>
API キーを安全に保管し、公開しないでください。
</Warning>
</Step>

<Step title="製品を管理する">
認証後、製品リストが表示されます：

- **既存の製品**：自動的に表示されます
- **製品がない**：'製品' の隣にある **プラス (+)** ボタンをクリックします。

<Info>
製品の価格、説明、および設定を作成するために Dodo Payments ダッシュボードにリダイレクトされます。
</Info>
</Step>

<Step title="購入ボタンを追加する">
1. 利用可能なリストから製品を選択します
2. `Add Buy Button` ボタンをクリックします
3. 'Buy \{product_name\}' というテキストの購入ボタンがキャンバスに追加されます。

<Warning>
正しいプロジェクト権限がない場合、購入ボタンの追加は無効になります。
</Warning>
</Step>
</Steps>

## カスタマイズ

<Steps>
<Step title="チェックアウト体験を選択する">
    お好みのチェックアウト体験を選択してください：
    - **オーバーレイ（埋め込み）**：チェックアウトが Framer プロジェクト内のオーバーレイとして表示され、ユーザーがサイトに留まります。
    - **ホスティング**：ユーザーは Dodo Payments のホストされたチェックアウトページにリダイレクトされ、スムーズな体験が得られます。
</Step>

<Step title="外観オプションをカスタマイズする">
    次の外観設定でチェックアウト体験をカスタマイズします：
    - **チェックアウトテーマ**：ブランドの美学に合わせて、明るいテーマまたは暗いテーマを選択します。
    - **ボタンラベル**：テキストをカスタマイズします（例：'今すぐ購入'、'購入'、'始める'）。
    - **ボタンスタイル**：色、フォント、サイズ、その他のスタイリングプロパティを調整します。
    - **戻り URL**：チェックアウト後にユーザーが戻る Framer ページを設定します。
</Step>
</Steps>

## トラブルシューティング

<AccordionGroup>
<Accordion title="認証の問題">
**問題**：プラグインが認証されない

**解決策**：
- API キーが正しいことを確認する
- 正しいモード（テスト/ライブ）を使用していることを確認する
- インターネット接続を確認する
</Accordion>

<Accordion title="ボタン作成の問題">
**問題**：購入ボタンの追加が無効

**解決策**：
- Framer プロジェクトで正しい権限があることを確認する
- 正しく認証されていることを確認する
- プラグインを更新してみる
</Accordion>

<Accordion title="製品表示の問題">
**問題**：製品が表示されない

**解決策**：
- Dodo Payments ダッシュボードで製品が作成されていることを確認する
- API キーに正しい権限があることを確認する
- 再認証を試みる
</Accordion>
</AccordionGroup>
