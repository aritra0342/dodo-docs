## はじめに

Dodo Payments WooCommerceプラグインは、あなたのストアとシームレスに統合され、UPI、カード、ネットバンキング、デジタルウォレットなど、複数の方法で支払いを受け入れることができます。この包括的なガイドでは、インストールと設定のプロセスを説明します。

## ステップバイステップのインストールガイド

1. WooCommerce支払いプラグインの最新のzipファイルをダウンロードします。

    <Card title="プラグインをダウンロード" icon="download" href="https://github.com/dodopayments/dodo-woocommerce/releases">
    WooCommerceプラグインの最新バージョンを取得
    </Card>

    あるいは、WordPressプラグインディレクトリから直接インストールすることもできます：
    
    <Card title="WordPressプラグインディレクトリ" icon="wordpress" href="https://wordpress.org/plugins/dodo-payments-for-woocommerce/">
    WordPress.orgからWooCommerce用のDodo Paymentsをインストール
    </Card>

2. 管理者としてWordPressウェブサイトにログインします（WooCommerceがすでに設定されていると仮定します）
3. **プラグイン → 新しいプラグインを追加**に移動します
    
    <Frame>
        <img src="/images/woocommerce/1.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
4. **プラグインをアップロード**ボタンをクリックすると、ファイル選択ダイアログが表示されます。

    <Frame>
        <img src="/images/woocommerce/2.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/3.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
5. **参照…**をクリックし、ダウンロードしたzipファイルを選択します。
6. **今すぐインストール**ボタンをクリックしてインストールプロセスを開始します。
    
    <Frame>
        <img src="/images/woocommerce/4.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
7. インストールが完了したら、**プラグインを有効化**をクリックするか、**インストール済みプラグイン**セクションから有効化します。
    
    <Frame>
        <img src="/images/woocommerce/5.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
8. プラグインはインストールされましたが、設定がまだ必要です。以下のセットアップガイドに進んでください。

## 設定ガイド

### APIキーとWebhookの設定

1. **WooCommerce → 設定 → 支払い**に移動するか、WooCommerceメニュー項目の下にある左サイドバーの**支払い**ボタンをクリックします。
    
    <Frame>
        <img src="/images/woocommerce/6.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/7.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
2. **Dodo Payments**支払いプロバイダーがまだ有効でない場合は有効にし、次に**管理**をクリックしてプラグインを設定します。
3. 様々な設定オプションが表示され、それぞれに役立つ説明文があります。まず、Live APIキーを設定します。
    
    <Frame>
        <img src="/images/woocommerce/8.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
4. **Live Mode**でDodo Paymentsダッシュボードにログインし、**Dodo Payments (Live Mode) > Developer > API Keys**に移動するか、[この直接リンク](https://app.dodopayments.com/developer/api-keys)を訪れて**APIキーを追加**をクリックします。
    
    <Frame>
        <img src="/images/woocommerce/9.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
5. APIキーに説明的な名前を付けて**作成**をクリックします。
    
    <Frame>
        <img src="/images/woocommerce/10.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
6. 生成されたAPIキーをコピーし、WooCommerceプラグイン設定の**Live API Key**フィールドに貼り付けます。
    
    <Frame>
        <img src="/images/woocommerce/11.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/12.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
7. 次に、Dodo PaymentsとWooCommerce間の支払いステータスの同期を有効にするためにWebhook署名キーを設定します。
8. プラグイン設定ページの下部にスクロールし、そこに表示されているWebhook URLをコピーします。
    
    <Frame>
        <img src="/images/woocommerce/13.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
9. Dodo Paymentsダッシュボードに戻り、**Dodo Payments (Live Mode) > Developer > Webhooks**に移動し、**Webhookを追加**をクリックします。
    
    <Frame>
        <img src="/images/woocommerce/14.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
10. ステップ8でコピーしたURLをダイアログに貼り付け、**Webhookを追加**をクリックします。
    
    <Frame>
        <img src="/images/woocommerce/15.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
11. Webhookを作成した後、赤く表示された**Signing Key**の隣にある目のアイコンをクリックして、それを表示しコピーします。
    
    <Frame>
        <img src="/images/woocommerce/16.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
12. Signing Keyをプラグイン設定の**Live Webhook Signing Key**フィールドに貼り付け、変更を保存します。
    
    <Frame>
        <img src="/images/woocommerce/17.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
13. **Global Tax Category**や**All Prices are Tax Inclusive**などの残りの設定を確認します。これらのオプションは、WooCommerceからDodo Paymentsに製品が同期される方法を決定します。**Test API Key**と**Test Webhook Signing Key**は、テストモードを使用する予定がある場合のみ必要です。
    
    <Frame>
        <img src="/images/woocommerce/18.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>

## 主な機能

- **シームレスなチェックアウト体験**: 顧客を安全で最適化されたチェックアウトページにリダイレクト
- **複数の支払い方法**: 様々なローカルおよび国際的な支払いオプションをサポート
- **リアルタイムのステータス更新**: Webhookを介した即時の注文ステータス同期
- **詳細な取引記録**: WooCommerceダッシュボードでの包括的な支払い履歴
- **多通貨サポート**: 複数の通貨での支払いを受け入れ

## トラブルシューティング

### 一般的な問題

1. **支払い処理の失敗**
   - APIキーが正しく設定されていることを確認
   - Webhook URLが正しく設定されていることを確認
   - SSL証明書が有効であることを確認

2. **Webhook同期の問題**
   - Webhook署名キーが一致していることを確認
   - サーバーログを確認して潜在的なエラーを探す
   - サーバーが受信Webhookリクエストを受け取れることを確認

## メンテナンスのベストプラクティス

1. セキュリティパッチや新機能のためにプラグインを定期的に更新します。
2. 本番取引を処理する前に、サンドボックスモードで全ての支払いフローをテストします。
3. WordPressとWooCommerceを最新バージョンに更新します。
4. 定期的なサイトバックアップを実施します。
5. 取引ログを監視して異常な活動を探します。
6. APIキーを安全に保管し、決して公開しないでください。
