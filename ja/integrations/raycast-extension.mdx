Dodo Paymentsダッシュボードが数回のキー入力で利用可能です。Raycastを使用して、ダッシュボードを開かずに支払い、サブスクリプション、顧客、製品、割引コード、ライセンスキー、返金を迅速に表示します。

## インストール

<Card title="Install from Raycast Store →" icon="puzzle-piece" href="https://www.raycast.com/dodopayments/dodo-payments">
  Raycast Store で Dodo Payments 拡張機能ページを開きます。
</Card>

<Info>
macOS 上の Raycast と Dodo Payments API キーが必要です。
</Info>

## セットアップ

<Steps>
<Step title="Get your API Key">
1. Dodo Payments ダッシュボードにログインします。
2. 設定 › API キーに移動します。
3. API キーをコピーします。
</Step>

<Step title="Configure the Extension">
1. Raycast を開きます。
2. 「Dodo Payments」と検索し、任意のコマンドを選択します。
3. プロンプトが表示されたら、API キー フィールドに API キーを入力します。
4. API モードを選択します:
   - Test: 開発およびテストに使用します。
   - Live: 本番データに使用します。
5. 設定を保存します。
</Step>
</Steps>

<Tip>
Test モードでセットアップを検証し、その後 Live に切り替えてください。
</Tip>

<Warning>
API キーはパスワードと同じように取り扱ってください。共有したりプレーンテキストで保存したりしないでください。
</Warning>

## 拡張機能の使用

- **検索**：Raycastを開き、「Dodo Payments」と入力し、コマンドを選択します。
- **フィルタ**：ID、メール、名前、またはステータスの一部を入力して結果を絞り込みます。
- **詳細を開く**：リスト項目でEnterを押して、詳細情報を表示します。

## トラブルシューティング

<AccordionGroup>
<Accordion title="Authentication failed or 401/403 errors">
API キーとモード (Test と Live) がダッシュボードのデータと一致していることを確認してください。必要であればコマンドを再度開いてキーを再入力してください。
</Accordion>
<Accordion title="No results appear">
選択したモードにデータがあることを確認してください。Test と Live を切り替えて試してください。
</Accordion>
<Accordion title="Rate limits or timeouts">
少し待ってからもう一度お試しください。問題が解決しない場合はネットワーク接続を確認してください。
</Accordion>
</AccordionGroup>
