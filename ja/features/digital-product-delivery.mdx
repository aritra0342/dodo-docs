## 概要

**デジタル製品の配信**機能により、商人は成功した購入後にデジタルファイルを自動的に顧客に配信できます。電子書籍、デザインテンプレート、ソフトウェア、メディアパックなどを販売している場合、製品にダウンロード可能なコンテンツを添付し、メールおよび顧客ポータルを通じて即座にアクセスを提供できます。

この機能は、ホストされたファイルのアップロードと外部ダウンロードリンクの両方をサポートしており、さまざまなユースケースに柔軟性を提供します。

## 主な機能

<CardGroup cols={2}>
<Card title="File Upload" icon="upload">
PDF、ZIP、画像、ビデオなどのデジタルファイルを顧客に直接アップロードして提供します。
</Card>

<Card title="Multiple Files" icon="files">
製品ごとに最大10個のファイルを添付して、包括的なデジタル製品の提供を行えます。
</Card>

<Card title="External Links" icon="link">
Dropbox や Google Drive などの外部のダウンロードリンクを、ファイルアップロードの代替手段として提供できます。
</Card>

<Card title="Automatic Delivery" icon="download">
顧客は支払い完了後、自動的にメールでダウンロード手順を受け取ります。
</Card>
</CardGroup>

## セットアッププロセス

<Steps>
<Step title="Navigate to Products">
ダッシュボードにアクセスし、**Products > Create New** を選択するか既存の製品を編集します。**Advanced Settings → Digital Product Delivery** に移動します。
</Step>

<Step title="Configure File Delivery">
以下のオプションのいずれかを選択します:

- **ファイルをアップロード**：製品に最大10ファイルを直接追加
- **外部ダウンロードリンク**：外部にホストされた安全なリンクを貼り付けます（例：Dropbox、Google Drive）

<Frame>
<img src="/images/digital-product-delivery/1.png" alt="Digital product delivery configuration interface showing file upload and external link options" />
</Frame>
</Step>

<Step title="Save Configuration">
**Update Product** をクリックしてデジタル製品の配信設定を反映させます。

<Warning>
アップロードされたファイルは静的な URL を通じて提供されるため、リンクを知っている人なら誰でもファイルにアクセスできます。機密性の高いコンテンツの場合は、認証付きの外部サービスの利用を検討してください。
</Warning>
</Step>
</Steps>

## 顧客体験

### 購入確認

成功した取引の後、顧客はデジタル製品のダウンロードリンクを含む購入確認メールを受け取ります。

<Frame>
<img src="/images/digital-product-delivery/2.png" alt="Purchase confirmation email showing download links for digital products" />
</Frame>

### 顧客ポータルへのアクセス

顧客は顧客ポータルを通じてデジタル製品にアクセスできます：

- 過去の購入を表示するためにログイン
- 各製品のファイルをダウンロードまたは外部リンクにアクセス
- 購入したデジタルコンテンツへの持続的なアクセス

<Frame>
<img src="/images/digital-product-delivery/3.png" alt="Customer portal interface showing available digital products for download" />
</Frame>

<Check>
顧客は確認メールから直接ファイルをダウンロードするか、ポータルを通じていつでもアクセスできます。
</Check>

## 重要な考慮事項

- **セキュリティ**：静的ダウンロードリンクは注文認証されていません。リンクを持っている顧客は、同じリンクを他の人と再共有できます。安全またはライセンスされたコンテンツには、ゲート付きまたは認証されたアクセスをサポートする外部サービスとの統合を検討してください。
- **ファイルの更新**：ファイルまたは外部リンクを更新すると、今後の購入にのみ影響します。過去の購入には影響せず、顧客はそれらのファイルに引き続きアクセスできます。
- **返金**：返金を発行してもファイルへのアクセスは取り消されません。すでにファイルをダウンロードした顧客は、デバイスにそのファイルを保持します。
- **ベストプラクティス**：バージョン管理や期間限定アクセスのために、外部にホストされた期限付きまたは署名付きURLを介してコンテンツを配信することを検討してください。
