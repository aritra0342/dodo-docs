## 適応通貨

適応通貨により、顧客は好みの現地通貨で支払うことができ、信頼性とコンバージョン率が向上します。

### 仕組み

1. **有効化**: ビジネス設定から適応通貨を有効にします
2. **選択**: 顧客はチェックアウト時に通貨を直接切り替えることができます
3. **変換**: 価格はリアルタイムの為替レートを使用して動的に変換されます
4. **表示**: 支払い前に最終的な支払額が透明に表示されます

## 自動税計算

税金は顧客の請求先住所に基づいて自動的に計算され、GST、VAT、および売上税の要件に準拠します。手動設定は不要です。

### 税計算の仕組み

<Steps>
<Step title="所在地検出">
  税ルールは顧客の国（および該当する地域）に基づいて適用されます。
</Step>

<Step title="動的更新">
  税額は次の場合に自動的に更新されます:
  - 国が変更されたとき
  - 住所が更新されたとき
</Step>

<Step title="透明な表示">
  最終的な税の内訳が支払い前に明確に表示されます。
</Step>
</Steps>

<Tip>
税計算は完全に自動化されています。標準的なデジタル商品およびSaaS製品に対して手動設定は不要です。
</Tip>

## ビジネスタックスIDサポート

登録されたビジネスの場合、チェックアウトでは顧客がビジネスタックスID（例：VAT/GST番号）を入力できます。

### 税IDが入力された場合の処理

- 税の適格性がリアルタイムで検証されます
- 適用可能な税の免除または逆課税ルールが適用されます
- チェックアウト時に税額が即座に更新されます

</Info>

## 割引コード

顧客はダッシュボードで作成した割引またはプロモーションコードをチェックアウトページで直接適用できます。

### チェックアウト体験

1. 顧客が割引コードを入力します
2. 割引が即座に検証されます
3. 更新された価格と節約額が明確に表示されます

<Frame>
  <img src="/images/checkout/c4.png" alt="チェックアウト時の割引コード入力" style={{ width: '70%' }}/>
</Frame>

### API統合

割引コードを事前に適用するか、割引入力フィールドを有効にします:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  discount_code: 'WELCOME20', // Pre-apply a code
  feature_flags: {
    allow_discount_code: true // Show discount input field
  },
  return_url: 'https://yoursite.com/return'
});
```

## スマート住所収集

チェックアウトは、迅速な完了のために柔軟な住所入力をサポートしています。

### 利用可能なオプション

| オプション | 説明 |
|--------|-------------|
| **Google住所自動入力** | オートコンプリートによる迅速な選択 |
| **手動入力** | 完全な住所のための完全なコントロール |
| **国の選択** | 税金およびコンプライアンスロジックを駆動します |

<Tip>
住所収集は、スピード、正確性、グローバルなカバレッジのバランスを取り、コンバージョンを最大化しつつコンプライアンスを確保します。
</Tip>

## プライバシーポリシーと利用規約の承認

法的およびコンプライアンスの透明性を確保するために:

- [プライバシーポリシー](https://dodopayments.com/privacy-policy)および[バイヤー利用規約](https://dodopayments.com/buyer-terms)のリンクがチェックアウト時に明確に表示されます
- 顧客は支払いを完了する前にこれらを明示的に承認します

<Info>
これは、GDPRコンプライアンスを含む、世界的な消費者保護およびデータプライバシー要件を満たすのに役立ちます。
</Info>

## チェックアウトセッションの設定

チェックアウトの動作をCheckout Sessions APIを使用して制御します:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  customer: {
    email: 'customer@example.com',
    name: 'Jane Doe'
  },
  billing_currency: 'EUR', // Set specific currency
  discount_code: 'PROMO10',
  feature_flags: {
    allow_discount_code: true
  },
  return_url: 'https://yoursite.com/return',
  metadata: {
    order_ref: 'ORD-12345'
  }
});
```
