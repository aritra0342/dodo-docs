<Info>
適応通貨を使用すると、ローカライズされた価格を表示し、顧客の現地通貨での支払いを受け付けることができます。これにより、チェックアウト時の摩擦が減り、信頼が築かれます。
</Info>

## 適応通貨とは？

適応通貨は、製品の価格を固定のグローバル通貨（例：USD）ではなく、顧客の現地通貨で表示します。有効にすると、チェックアウトはサポートされている国の検出された現地通貨にデフォルト設定され、USDに切り替えるオプションがあります。

## 主な利点

- **ローカライズされた支払い体験**: 価格はデフォルトで現地通貨で表示されます。
- **より多くの支払い方法**: 現地通貨で利用可能な支払い方法が解放されます。
- **同通貨での返金**: 顧客が支払った通貨で返金します。

## 適応通貨を有効にする

<Steps>
<Step title="ビジネス設定を開く">
マーチャントダッシュボードにログインし、ビジネス → ビジネス設定に移動します。
</Step>

<Step title="適応価格設定を切り替える">
適応価格設定のトグルを有効にします。いつでも無効にできます。

<Warning>
変更は今後の取引にのみ適用されます。
</Warning>
</Step>

<Step title="確認して保存">
設定を保存します。

<Check>
テストチェックアウトがサポートされている場合、現地通貨で価格が表示されることを確認します。
</Check>
</Step>
</Steps>

## 顧客体験

1. **検出**: システムは請求先住所に基づいて顧客の国を検出します。
2. **通貨選択**: 国がサポートされている場合、価格はデフォルトで現地通貨で表示されます。顧客はUSDに切り替えることができます。
3. **支払い方法**: 適用可能な場合、ローカライズされた支払い方法が表示されます。
4. **チェックアウト**: 選択した通貨で支払いが完了します。

## 変換と手数料

適応通貨は、最新の為替レートを使用して顧客の現地通貨で請求します。

- **あなたが支払う**: 0%
- **顧客が支払う**: 注文金額に基づいて2〜4%

チェックアウト時に適用される段階的手数料:

- 500ドル未満の注文には4%
- 500ドルから1,500ドルの注文には3%
- 1,500ドルを超える注文には2%

<Info>
適応通貨の手数料は、商人ではなく顧客が負担します。
</Info>

## 返金と調整

Dodo Paymentsは、顧客が元々支払った通貨で返金を行い、最新の為替レートを使用します。グローバルなUSD金額は、ダッシュボード、請求書、および返金で固定されます。これにより、顧客は為替変動に応じて元の現地通貨金額よりも多くまたは少なく受け取る可能性があります。

<Info>
適応通貨の手数料は一般的にFX手数料であり、返金されません。
</Info>

**返金の例**

1. あなたは適応通貨を有効にして100 USDで製品を販売します。
2. カナダの顧客は、1 USDあたり1.37 CADの為替レートで137 CADを見て購入を完了します。
3. 私たちは支払いを処理し、137 CADを100 USDに変換してあなたの決済にします。
4. 後に、為替レートが1 USDあたり1.40 CADに変わり、あなたは全額返金を発行します。
5. 私たちは100 USDを差し引き、顧客に140 CADを返金します。

## 請求書と課税

- 請求書には決済通貨の金額のみが表示されます。
- 税金とプラットフォーム手数料は決済通貨の金額に基づいて計算されます。
- 例: 10ドルの販売が36 AEDに変換されても、ダッシュボードと請求書には10ドルとして表示されます。

<Info>
すべての金額はDodo Paymentsの内部丸めロジックに従って丸められます。
</Info>

## 統合例

**請求通貨を持つチェックアウトセッション**

`billing_currency`を渡して、セッションの請求通貨を明示的に設定します。

<Info>
適応価格設定が無効になっている場合、`billing_currency`は無視されます。
</Info>

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    {
      product_id: 'prod_one_time_or_subscription',
      quantity: 1
    }
  ],
  billing_currency: 'AED',
  return_url: 'https://example.com/return'
});
```
