## はじめに

APIキーの生成により、商人はDodo Paymentsのプラットフォームと安全に認証し、システムを統合して、支払い、製品、およびサブスクリプション機能にシームレスにアクセスできます。

**注意 :** テストモードで作成されたAPIキーは、テストモード環境に制限され、ライブモードの場合はその逆になります。
