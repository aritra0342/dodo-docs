<CardGroup cols={2}>
<Card title="製品カタログ" img="/images/our-features/Product Catalog.webp" href="/features/product-catalog/overview" >
 この機能により、商人は一回限りの購入やサブスクリプションベースの製品を含む製品提供を管理できます。柔軟な価格オプションと請求サイクルを使用して、製品を作成、構成、管理できます。
</Card>
<Card title="支払い" img="/images/our-features/Payments.webp" href="/features/transactions/payments" >
 支払い機能はDodo Paymentsの基盤であり、エンドツーエンドの支払い処理を行います。このセクションでは、支払いがどのように処理され、管理されるかについての完全な透明性を提供します。
</Card>
<Card title="返金" img="/images/our-features/Refunds.webp" href="/features/transactions/refunds" >
 この機能は、商人が返金リクエストを整理されたタイムリーな方法で管理および開始するのを助けます。返金機能は、顧客満足を確保しながら、ビジネスの財務操作に対するコントロールを維持するのに役立ちます。
</Card>
<Card title="請求書発行" img="/images/our-features/Invoicing.webp" href ="/features/invoice-generation">
 Dodo Paymentsは自動請求書発行を提供し、成功した取引の後に請求書を自動化することで請求プロセスを簡素化します。商人は、ビジネスのブランディングに合わせて請求書テンプレートをカスタマイズすることもできます。
</Card>
<Card title="支払い" img="/images/our-features/Payouts.webp" href="/features/payouts/overview" >
支払い機能は、Dodo Paymentsアカウントから銀行口座への資金移動プロセスを簡素化します。このセクションでは、商人が自分の収益を完全に管理し、関連する手数料を理解できるようにします。
</Card>

<Card title="分析と報告" img="/images/our-features/Reports.webp" href="/features/analytics-and-reporting" >
分析と報告機能は、商人にビジネスのパフォーマンスに関する詳細な洞察を提供します。Dodo Paymentsのダッシュボードから、収益、顧客活動、取引成功率などに関する包括的なレポートを表示できます。
</Card>
<Card title="顧客" img="/images/our-features/Customers.webp" href="/features/customers" >
顧客ページは、商人にすべての顧客詳細の中央ビューを提供し、顧客記録をダウンロードできるようにします。
</Card>
<Card title="詐欺検出とデータセキュリティ" img="/images/our-features/Fraud.webp" href="/features/fraud-detection">
私たちの包括的なプラットフォームは、取引が安全であることを保証するために、すべての標準的なセキュリティおよび詐欺検出の慣行に従っています。
</Card>
<Card title="ライセンスキー" img="/images/our-features/license-key.png" href="/features/license-keys">
ライセンスキー管理機能は、商人がデジタル製品やサービスを保護するために、ユニークなライセンスキーを生成、管理、追跡するのを助けるように設計されています。
</Card>
<Card title="Webhook" img="/images/our-features/Webhook.png" href="/developer-resources/webhooks">
Webhookは、Dodo Payments内で特定のイベントが発生したときにリアルタイム通知を提供するメカニズムを提供し、アプリケーションとのシームレスな自動化と統合を可能にします。
</Card>
<Card title="アカウント概要と支払いウォレット" img="/images/our-features/account-summary.png"  href="/features/account-summary-payout-wallet">
アカウント概要は、支払い、返金、支払い、取引手数料、税金を含むすべての財務取引の詳細な内訳を提供します。これにより、商人の収益と控除に対する完全な可視性が確保されます。
</Card>
<Card title="APIキー生成" img="/images/our-features/api-key-generation.png"  href="/features/api-keys">
APIキー生成により、商人はDodo Paymentsのプラットフォームと安全に認証し、システムを統合して、支払い、製品、サブスクリプション機能にシームレスにアクセスできます。
</Card>
<Card title="Atlasデモ製品" img="/images/our-features/atlas-demo.png" href="/features/atlas-demo-website">
Atlasデモ製品は、Dodo Paymentsによるデモサイトで、デジタル製品（ソフトウェアライセンス、デジタルダウンロード、オンラインメンバーシップなど）が私たちのチェックアウトソリューションを使用してどのように販売されるかを示しています。商人は、サブスクリプションと一回限りの購入フローの両方を実際に見ることができます。
</Card>
</CardGroup>
