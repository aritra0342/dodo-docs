私たちは[Featurebase](https://dodopayments.featurebase.app/)を使用して機能リクエストを管理し、公開ロードマップを維持しています。このプラットフォームでは、アイデアを提出し、見たい機能に投票し、今後の改善の開発状況を追跡できます。

<Card title="公開ロードマップを訪問" icon="external-link" href="https://dodopayments.featurebase.app/">
すべての機能リクエストを表示し、アイデアに投票し、Featurebaseで製品ロードマップを追跡してください。
</Card>

## **新しい機能をリクエストする**

Dodo Paymentsの体験を向上させる機能のアイデアがある場合は、Featurebaseで直接機能リクエストを提出できます。

<img src="/images/feature-request/Feature.png" alt="機能リクエスト"/>

- **機能をリクエストする方法**:
    1. 私たちの[Featurebaseポータル](https://dodopayments.featurebase.app/)にアクセスします。
    2. **「アイデアを投稿」**または**「機能リクエストを提出」**をクリックします。
    3. 次の詳細を提供します:
        - **機能名**: 機能アイデアの簡潔なタイトル。
        - **機能説明**: 機能の簡単な説明、重要性、ビジネスにどのように役立つかを含めます。
    4. リクエストを提出します。
- **効果的なリクエストのためのヒント**:
    - 明確かつ簡潔に。
    - 機能のビジネスへの影響を強調します。
    - 適用可能な場合は例を提供します。
    - 価値を示すユースケースを含めます。

## **既存の機能リクエストに投票する**

私たちは、最も多くの商人に利益をもたらす機能を優先します。Featurebaseで既存の機能リクエストを閲覧し、価値があると思うものに投票できます。

- **投票する方法**:
    1. 私たちの[Featurebaseポータル](https://dodopayments.featurebase.app/)にアクセスします。
    2. 提出されたリクエストのリストを閲覧します。
    3. 支持する機能の横にある**アップボート**ボタン (👍) をクリックします。
- **投票の利点**:
    - あなたが気にかける機能の開発優先順位に影響を与えます。
    - 私たちのチームが広く需要のある機能を特定するのを助けます。
    - あなたが投票した機能がリリースされたときに通知を受け取ります。

## **製品ロードマップを表示する**

Featurebaseの公開ロードマップを通じて、機能開発の状況を把握してください。

- **ロードマップの機能**:
    - **計画中**: 承認され、開発が予定されている機能。
    - **進行中**: 現在私たちのチームが作業中の機能。
    - **完了**: 成功裏に実装され、稼働中の機能。
- **アクセス方法**:
    - 私たちの[Featurebaseポータル](https://dodopayments.featurebase.app/)にアクセスします。
    - ステータス別に機能を表示するために**ロードマップ**セクションに移動します。
    - 機能カテゴリとその現在のステータスを閲覧します。
    - ステータス、カテゴリでフィルタリングするか、特定の機能を検索します。

<Tip>
新機能の更新を受け取り、あなたにとって重要なアイデアに投票するために、私たちの[Featurebaseポータル](https://dodopayments.featurebase.app/)をブックマークしてください。
</Tip>
