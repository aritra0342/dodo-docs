使用量に基づく請求は、顧客が実際に使用した分（APIコール、ストレージ、AIトークン、または定義した任意のメトリック）に対して請求することを可能にします。顧客は消費に対して支払い、固定料金はありません。

<Info>
**最適な用途**: API、AIサービス、インフラストラクチャプラットフォーム、および使用量に応じて価値がスケールする製品。
</Info>

## 仕組み

<Steps>
<Step title="使用量イベントを送信">
請求可能なアクションが発生したときに、アプリケーションがイベントを送信します。

```javascript
{
  "event_id": "evt_123",
  "customer_id": "cus_abc",
  "event_name": "api.call",
  "metadata": { "endpoint": "/v1/users" }
}
```

</Step>

<Step title="メーターが使用量を集計">
メーターは、Count、Sum、Max、またはLast集計を使用してイベントを請求可能な数量に集計します。
</Step>

<Step title="自動請求">
単位ごとの価格と無料の閾値を設定します。顧客は各サイクルで自動的に請求されます。
</Step>
</Steps>

## コア概念

### イベント

イベントは、アプリケーション内の使用アクションを表します。各イベントには以下が含まれます:
- **イベント名**: 使用の種類 (例: `api.call`)
- **顧客ID**: 誰が使用したか
- **メタデータ**: 集計のための追加プロパティ

### メーター

メーターは、イベントを請求可能な数量に集計します:
- **カウント**: 総イベント数（APIコール）
- **合計**: 値を加算（総バイト数）
- **最大**: 最高値（ピークユーザー数）
- **最新**: 最も最近の値

<img src="/images/usage-based/UBB-2.png" alt="メーター作成インターフェース" />
</Frame>

### 使用量に基づく価格設定のある製品

単位ごとの価格とオプションの無料閾値を設定します:

<Frame>
<img src="/images/usage-based/UBB-4.png" alt="価格設定の構成" />
</Frame>

**例**: 2,500コール - 1,000無料 = 1,500 × $0.02 = $30.00

### 顧客クレジット

顧客に各請求サイクルごとにリフレッシュされる使用量クレジットを提供します。クレジットは有料使用の前に適用されます。

## クイックスタート

<Steps>
<Step title="メーターを作成">
ダッシュボードで: **メーター** → **メーターを作成**

1. イベント名を設定します (例: `api.call`)
2. 集計を選択します (カウント、合計、最大、最後)
3. オプションのフィルターを追加します
</Step>

<Step title="製品に追加">
メーターを価格設定付きの製品にリンクします:

<Frame>
<img src="/images/usage-based/UBB-5.png" alt="製品にメーターを追加" />
</Frame>

1. **使用量に基づく請求**を選択
2. メーターを選択
3. 単位ごとの価格と無料閾値を設定
</Step>

<Step title="イベントを送信">

```javascript
await fetch('https://test.dodopayments.com/events/ingest', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${process.env.DODO_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    events: [{
      event_id: "unique_event_id",
      customer_id: "cus_abc123",
      event_name: "api.call",
      metadata: { endpoint: "/v1/users" }
    }]
  })
});
```

</Step>

<Step title="使用量を監視">
<Frame>
<img src="/images/guides/usage-based-billing/meter-quantities-chart.png" alt="メーターダッシュボード" />
</Frame>

メーターのダッシュボードをチェックして、イベントと使用量の集計を確認します。顧客は各サイクルで自動的に請求されます。
</Step>
</Steps>

## ユースケース

<CardGroup cols={2}>
<Card title="API & インフラストラクチャ" icon="cloud">
APIコール、計算時間、帯域幅、ストレージを追跡します。例: 10,000無料の後、$0.001/コール。
</Card>

<Card title="AI使用量追跡" icon="brain">
トークン、生成、処理時間を追跡します。例: 標準画像は\$0.04、HDは\$0.08。
</Card>

<Card title="データ & 分析" icon="chart-line">
処理されたデータ、実行されたクエリ、分析されたレコードに基づいて請求します。例: 10GB無料の後、$5/GB。
</Card>

<Card title="コミュニケーション" icon="message">
メッセージ、分、転送されたデータをカウントします。例: 送信されたSMSは$0.01。
</Card>
</CardGroup>
