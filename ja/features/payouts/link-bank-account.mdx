## 銀行口座のリンクプロセス

1. Dodo Paymentsダッシュボードの**支払い設定**セクションに移動し、**銀行情報**タブを選択します。
2. **銀行口座の所在地**を選択します（インド、アメリカ、ヨーロッパ、その他の地域）。
3. 以下に示す正確な詳細でフォームを記入します。
4. 検証のために、可視性要件を満たす**銀行明細書**をアップロードします。

## 国別の銀行情報フィールド

### **1. インド**

| フィールド | 説明 |
| --- | --- |
| **口座名義人名** | 銀行口座の詳細に記載されている正確な名前 |
| **銀行名** | 銀行の名前 |
| **銀行口座の種類** | 銀行口座の種類（個人または法人） |
| **銀行口座通貨** | 現地通貨の3文字コード（例：INR） |
| **銀行口座番号** | 一意の銀行口座番号 |
| **IFSCコード** | 銀行口座明細書に記載されているIFSCコード |
| **銀行明細書のアップロード** | すべての関連詳細が表示されている可視の銀行明細書をアップロードします |

### **追加ガイドライン**

- **銀行明細書の可視性**: アップロードされたすべての銀行明細書には、明確で読みやすい口座詳細が含まれている必要があります。情報を切り取ったり隠したりしないでください。これにより、検証プロセスが遅れる可能性があります。
- **口座名の正確性**: 口座名義人名は、提出された銀行書類に記載されている名前と正確に一致する必要があります。これにより、検証プロセス中の不一致を避けることができます。
- **SWIFT/BICおよびルーティング番号**: 国際および国内送金を円滑に行うために、SWIFT/BICおよびルーティング番号が正確であることを確認してください。
- **通貨コード**: 銀行口座の現地通貨に応じて、常に正しい3文字の通貨コード（例：インドの場合はINR、アメリカの場合はUSD、ヨーロッパの場合はEUR）を使用してください。
通貨コードに疑問がある場合は、次のリンクを使用して確認できます: [https://www.iban.com/currency-codes](https://www.iban.com/currency-codes)
