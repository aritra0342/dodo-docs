## インド向け

### 登録済み事業

インドの登録済み事業は、異なるタイプに分類され、それぞれに検証のための特定の書類が必要です。

#### サポートされている登録済み事業の種類：

* プライベートリミテッド

* 有限責任事業組合 (LLP)

* パートナーシップ

* 個人事業主

#### 登録済み事業に必要な書類：

| 事業タイプ               | 書類名                     | 説明                                                          |
| ----------------------- | -------------------------- | ------------------------------------------------------------ |
| **プライベートリミテッド** | 事業PAN番号                | 登録された事業のPAN番号。                                     |
|                         | GSTIN                      | 税務目的のためのGST識別番号。                                |
|                         | CIN                        | 会社識別番号                                                |
|                         | DBA名                     | 銀行口座明細に表示される「Doing Business As」名。            |
| **LLP**                 | 事業PAN番号                | 登録されたLLPのPAN番号。                                    |
|                         | GSTIN                      | 税務目的のためのGST識別番号。                                |
|                         | LLP設立書類               | LLP登録を証明する書類。                                      |
|                         | DBA名                     | 銀行口座明細に表示される「Doing Business As」名。            |
| **パートナーシップ**     | パートナーシップPAN番号     | パートナーシップ事業のPAN番号。                              |
|                         | パートナーシップ契約       | パートナーシップの条件と利害関係者の役割を概説する公式契約。 |
|                         | DBA名                     | 銀行口座明細に表示される「Doing Business As」名。            |
| **個人事業主**         | DBA名                     | 銀行口座明細に表示される「Doing Business As」名。            |

***

### 未登録事業

インドの未登録事業には、法的に法人化されていないため、異なる書類が必要です。

#### 未登録事業に必要な書類：

| 書類名         | 説明                                                                    |
| ------------- | ------------------------------------------------------------------------ |
| PAN番号       | 事業主の個人PAN                                                       |
| DBA名        | 銀行口座明細に表示される「Doing Business As」名。                      |
| 住所          | 証明書に基づく**通り、都市、州、郵便番号**を含む完全な住所。            |
| 国籍         | 個人または法人の所有者の市民権の国。                                   |

## 主要利害関係者情報 - インド

Dodo Paymentsは、事業に関連する各主要利害関係者の詳細情報を必要とします。各利害関係者は、コンプライアンス要件の一環として**PANカード**を提供する必要があります。最大4人の主要利害関係者の書類を受け付けます。4人を超える主要利害関係者がいる場合は、書類をメールで共有してください。

### 主要利害関係者とは？

以下の者が会社の主要利害関係者となることができます：

#### 実質的所有者

実質的所有者とは、単独または共同で、または他の会社、LLP、法的な法人を通じて、関連ユーザーの株式、資本、または利益の25％以上を所有または権利を有する自然人、または他の手段を通じて関連ユーザーを支配する者を指します。これには、取締役の大多数を任命する権利を享受することや、株主権や経営権を通じてその管理や方針決定を制御することが含まれます。

関連ユーザーに実質的所有者が1人以上いる場合は、すべての必要な書類を提供してください。

#### 事業取締役

インドの会社法に従って、関連ユーザーの取締役会に任命され、現在も取締役として勤務している個人。

すべての取締役のPANを提供してください。

#### 事業代表者

Dodo Paymentsサービスの利用に関して、関連ユーザーを代表して行動する権限を持つ従業員または取締役。

#### 共同創業者

事業が登録されていない場合、登録後に過半数の株式（25％以上）を所有するか、取締役になる共同創業者の書類を提供してください。登録後に変更があった場合は、これを更新できます。

## 税務書類

### 米国市民向け

米国市民は、商人の種類に関係なく、**W-9フォーム**を提出する必要があります。このフォームは、IRSに納税者識別情報を報告するために使用されます。

#### 米国市民に必要な税務書類：

| 市民権の状況 | 商人の種類         | 必要なフォーム | 説明                                         |
| -------------- | ------------------- | ------------- | ------------------------------------------- |
| **米国市民**   | 個人および事業     | W-9           | 納税者識別を報告するためのIRSフォーム。   |

* **指示**：米国市民は、W-9フォームをダウンロードし、記入し、すべての詳細が明確に表示されるように署名したW-9フォームをアップロードしてください。

* **ダウンロードリンク**： [https://www.irs.gov/pub/irs-pdf/fw9.pdf](https://www.irs.gov/pub/irs-pdf/fw9.pdf)

***

### 非米国市民向け

非米国市民は、商人の種類に基づいて異なるフォームを提出する必要があります。具体的には、個人か登録された事業体かによります。

#### 非米国市民 - 個人

非米国の個人商人は、**W-8BENフォーム**を提出する必要があります。このフォームは、IRSに情報を報告し、適用される税条約の下での利益を請求するために使用されます。

##### 非米国個人に必要な税務書類：

| 市民権の状況   | 商人の種類 | 必要なフォーム | 説明                                                       |
| ---------------- | ----------- | ------------- | --------------------------------------------------------- |
| **非米国市民**   | 個人        | W-8BEN        | 非米国個人が税条約の利益を請求するためのIRSフォーム。   |

* **指示**：非米国の個人は、W-8BENフォームをダウンロードし、記入し、すべての詳細が明確に表示されるように署名したW-8BENフォームをアップロードしてください。

* **ダウンロードリンク**： [https://www.irs.gov/pub/irs-pdf/fw8ben.pdf](https://www.irs.gov/pub/irs-pdf/fw8ben.pdf)

#### 非米国市民 - 事業体

非米国の事業体は、**W-8BEN-Eフォーム**を提出する必要があります。このフォームは、外国の事業体が外国の地位を請求し、適用される場合は所得税条約の下での源泉徴収税率の軽減を請求するために使用されます。

##### 非米国事業体に必要な税務書類：

| 市民権の状況   | 商人の種類     | 必要なフォーム | 説明                                                                  |
| ---------------- | --------------- | ------------- | -------------------------------------------------------------------- |
| **非米国市民**   | 事業体         | W-8BEN-E      | 非米国の事業体が税条約の利益を請求するためのIRSフォーム（該当する場合）。 |

* **指示**：非米国の事業体は、W-8BEN-Eフォームをダウンロードし、記入し、すべての詳細が明確に表示されるように署名したW-8BEN-Eフォームをアップロードしてください。

* **ダウンロードリンク**： [https://www.irs.gov/pub/irs-pdf/fw8bene.pdf](https://www.irs.gov/pub/irs-pdf/fw8bene.pdf)
