      <br/><br/>作成者 <a href="https://github.com/codeplugtech">@codeplugtech</a>
      <br/><br/>`PHP` `Laravel` `SDK`
    </Card>
    <Card title="dodo-payments-rs (Rust)" href="https://github.com/PiyushXCoder/dodo-payments-rs" icon="rust">

      <br/><br/>`Rust` `SDK`

      <br/>Dodo Paymentsを使用して複数のアプリケーションの支払いを処理するために構築された、プロダクション対応の支払いマイクロサービス。 
      <br/>機能には <br/> `credit management`, `subscription support`, `webhook processing with idempotency`, `complete audit trails`, `MixPanel Analytics`, `Backend-to-backend architecture with API key authentication`.
      <br/><br/>作成者 <a href="https://github.com/aniketkarne">@aniketkarne</a> (アニケット・カルネ)
