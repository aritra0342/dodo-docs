---
title: プロジェクトを提出する
description: オープンソースプロジェクトをDodo Paymentsコミュニティディレクトリに追加する方法。
icon: plus
---

<Steps>
{/* LOCKED_PATTERN_b8e49197111d4f91963d876b7d3cbd9b */}
- プロジェクトは寛容なライセンスの下でオープンソースです。
- Dodo Payments（SDK、プラグイン、統合、テンプレート）に関連しています。
- インストール、使用法、メンテナンス状況を含む README があります。
</Step>

{/* LOCKED_PATTERN_29bfb0d9a10f5f1cc614e31786e69c52 */}
最も関連性の高いタブの下で `community/projects.mdx` を編集し、新しい `<Card>` を追加してください。

```mdx
<Card title="PROJECT_NAME (LANG)" href="REPO_URL" icon="code">
  One-sentence description of your project.
  <br/>By <a href="GITHUB_PROFILE_URL">@github_username</a>
  <br/><br/>`LANGUAGE` `FRAMEWORK` `CATEGORY`
</Card>
```

</Step>

{/* LOCKED_PATTERN_5390269a28150786d344085d66aa0ef8 */}
必要であれば、「Community Project Submission」テンプレートを使って issue を開いてください。
- リポジトリ URL（Repo URL）、説明、カテゴリ（SDK/Plugin/Boilerplate）
- 主な言語/フレームワーク
- インストール/使用法スニペットへのリンク
- メンテナンス状況と最終更新
</Step>

{/* LOCKED_PATTERN_03e556e84db70a2199e5921d65cd601a */}
関連性、明瞭性、安全性を精査します。承認されたエントリはマージされ、公開されます。
</Step>
</Steps>

<Warning>
メンテナンスされていない、セキュリティに問題がある、誤解を招く、またはポリシーに違反しているエントリは削除される場合があります。
</Warning>
