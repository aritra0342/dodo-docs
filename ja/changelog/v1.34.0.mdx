## 新機能 🚀

- **Digital Product Delivery**: 自動化されたフルフィルメントを備えた合理化されたデジタル製品配信システムにより、支払い完了後すぐに顧客がデジタル商品を受け取れるようにします。詳細は[Digital Product Delivery]ドキュメントをご覧ください。
  
  <Frame>
    <img src="/images/digital-product-delivery/1.png" alt="Digital product delivery configuration interface showing file upload and external link options" />
  </Frame>

- **React Native SDK (ベータ)**: React Native SDKをベータ版でリリースし、カスタマイズ可能なUIコンポーネントと画面を使用して、ネイティブのAndroidおよびiOSアプリで安全な支払い処理を可能にします。詳細は、[React Native SDK](/developer-resources/react-native-integration)のドキュメントをご覧ください。

- **New Proration Mode**: `full_immediately` proration modeを導入し、サブスクリプションの変更が発生した際に即時に全額請求を行えるようにすることで、ビジネスニーズに合わせたより柔軟な請求オプションを提供します。

- **サブスクリプションの適応通貨**: 顧客の所在地とローカル通貨の好みに基づいて価格を自動的に調整する適応通貨サポートを備えたサブスクリプション管理を強化しました。

- **即時オンデマンド処理**: オンデマンド料金の処理速度を改善し、ほぼ瞬時に行えるようにし、取引時間を短縮し、迅速な支払い確認を提供します。

- **検証フォーム管理**: Dodo Paymentsダッシュボードの検証セクションにおいて、提出されたすべての検証書類を一元管理できる包括的な検証フォームデータ表示機能を追加しました。

- **サブスクリプションキャンセルの強化**: 請求期間の終了時にサブスクリプションをキャンセルする機能を実装し、ダッシュボードと顧客ポータルの両方で利用可能になり、ユーザーエクスペリエンスと請求の透明性が向上しました。

- **一般的な改善**: プラットフォームの全体的な安定性とユーザーエクスペリエンスを向上させるためのさまざまな小さな修正とパフォーマンスの改善。
