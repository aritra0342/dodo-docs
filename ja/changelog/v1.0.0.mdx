## 新機能 🚀

- **Customer Portal**: `Customer Portal` 機能を導入し、顧客がサブスクリプション、請求書、支払い方法を管理できるようになりました。Dodo Payments ダッシュボードまたは API を通じて、静的または動的な Customer Portal リンクを顧客向けに作成できます。ダッシュボードでは `Sales > Customers > Share Invite` に移動してください。

- **Customer Portal Session API**: 特定の顧客のために Customer Portal のセッションを作成するための `Customer Portal Session` API を追加しました。セッションは24時間のみ有効であることに注意してください。詳細は [API Reference](/api-reference/customers/create-customer-portal-session) をご確認ください。

- **その他**: 小さな改善とバグ修正。
