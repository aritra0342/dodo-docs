## 新機能 🚀

- **サブスクリプションの初回支払い**: 最もリクエストの多かった機能が登場しました！これで、初回支払いはmandateと実際のサブスクリプション金額を組み合わせた単一のトランザクションになります。

- **UPI QR支払い方法**: インド市場向けにUPI QRを支払い方法として追加し、顧客がUPI QRコードを使用して支払えるようになりました。
    
    <img src="/images/changelog/upi-qr.png" alt="UPI QR機能"/>

- **チェックアウト時の税ID**: 顧客は請求書および税務目的のために、チェックアウトページで税IDを追加できるようになりました。

- **ダッシュボードの再構成**: Dodo Paymentsダッシュボード全体が再構成され、より直感的で使いやすくなりました。

- **セルフサービスのオンボーディングガイド**: 新しいユーザーは、セルフサービスのオンボーディングガイドを使用してDodo Paymentsを始めることができます。

- **マルチ言語サポート**: チェックアウトページで14以上の言語をサポートし、顧客が希望する言語で支払いができるようになりました。対応言語は以下の通りです：
    - English (en)
    - العربية (ar) // アラビア語
    - Català (ca) // カタルーニャ語
    - 中文 (cn) // 中国語
    - Deutsch (de) // ドイツ語
    - Español (es) // スペイン語
    - Français (fr) // フランス語
    - עברית (he) // ヘブライ語
    - Italiano (it) // イタリア語
    - 日本語 (ja) // 日本語
    - Nederlands (nl) // オランダ語
    - Polski (pl) // ポーランド語
    - Português (pt) // ポルトガル語
    - Svenska (sv) // スウェーデン語

- **請求書PDFの改良**: 請求書PDFが新しく、より明確で効果的なデザインに変更されました。

- **中央集約型ヘルプウィジェット**: サポートを効率化し改善するために、中央集約型のヘルプウィジェットを追加しました。
    
    <img src="/images/changelog/help-widget.png" alt="ヘルプウィジェット"/>

## 改善点とバグ修正 🔧

- **レート制限**: 請求書APIにレート制限を追加し、悪用を防ぎ、システムの安定性を確保しました。

- **静的支払いリンクのメタデータ**: 静的支払いリンクにメタデータのサポートを追加しました。詳細については、[ドキュメント](/developer-resources/integration-guide#1-static-payment-links)を参照してください。

- **マイナーな改善と修正**: 全体的なパフォーマンスとユーザーエクスペリエンスを向上させるためのさまざまなマイナーな改善と修正が行われました。
