Card payments are accepted globally and trusted by customers worldwide. Dodo Payments supports all major card networks with built-in fraud protection and PCI compliance.

## Supported Networks

**Global:** Visa, Mastercard, American Express, Discover, JCB, UnionPay, Diners Club

**Regional:** Interac (Canada), Cartes Bancaires (France), Korean Local Cards, Rupay (India)

## Configuration

Use `credit` and `debit` in `allowed_payment_method_types`:

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  allowed_payment_method_types: ['credit', 'debit'],
  return_url: 'https://example.com/success'
});
```

## Features

<AccordionGroup>
<Accordion title="3D Secure Authentication">
3DS adds an authentication layer that reduces fraud and chargebacks. It's automatically triggered when required by card networks or regulations (e.g., PSD2 in Europe).

Force 3DS on all transactions:
```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  force_3ds: true,
  return_url: 'https://example.com/success'
});
```
</Accordion>

<Accordion title="Saved Payment Methods">
Customers can save cards for faster checkouts. Cards are tokenized and PCI compliant.

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  show_saved_payment_methods: true,
  customer: { customer_id: 'cus_existing_123' },
  return_url: 'https://example.com/success'
});
```
</Accordion>

<Accordion title="Security & Compliance">
- **PCI DSS Level 1** — Highest certification level
- **Tokenization** — Card numbers never stored
- **Fraud Prevention** — Real-time scoring, AVS, CVV validation
</Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
<Accordion title="Card declined">
Common causes: insufficient funds, expired card, incorrect CVV, bank fraud protection. Ask customer to verify details or try a different card.
</Accordion>

<Accordion title="3DS authentication failed">
Customer may have abandoned authentication or bank's system was unavailable. Retry or suggest contacting the bank.
</Accordion>

<Accordion title="Card not supported">
Some regional or prepaid cards have limitations. Customer should try a major network card.
</Accordion>
</AccordionGroup>
