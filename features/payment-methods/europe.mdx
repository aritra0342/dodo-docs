European customers strongly prefer local payment methods that integrate with their banking systems. Offering these methods can increase conversion rates by 20-40% in target markets.

## Supported Methods

| Method | Country | Market Share | API Type | Subscriptions |
| :----- | :------ | :----------- | :------- | :-----------: |
| iDEAL | Netherlands | ~60% | `ideal` | No |
| Bancontact | Belgium | ~50% | `bancontact_card` | No |
| EPS | Austria | ~30% | `eps` | No |
| Multibanco | Portugal | ~40% | `multibanco` | No |

<Note>
All European methods require EUR currency and country-specific billing addresses.
</Note>

## Configuration

### Single Country

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  allowed_payment_method_types: ['ideal', 'credit', 'debit'],
  billing_currency: 'EUR',
  billing_address: { country: 'NL', zipcode: '1012JS' },
  return_url: 'https://example.com/success'
});
```

### Multi-Country

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  allowed_payment_method_types: [
    'ideal',           // Netherlands
    'bancontact_card', // Belgium
    'eps',             // Austria
    'multibanco',      // Portugal
    'credit', 'debit'  // Fallback
  ],
  billing_currency: 'EUR',
  return_url: 'https://example.com/success'
});
```

Dodo automatically shows only relevant methods based on customer location.

## Method Details

<AccordionGroup>
<Accordion title="iDEAL (Netherlands)">
Direct bank transfers for Dutch customers. Connects to all major Dutch banks: ABN AMRO, ING, Rabobank, SNS, Bunq, Triodos, and more.

- Real-time payment confirmation
- Bank-level security
- No chargebacks
</Accordion>

<Accordion title="Bancontact (Belgium)">
Belgium's national payment scheme. Works with existing Belgian debit cards and Payconiq mobile app.

- Instant confirmation
- No additional registration needed
- Mobile app support
</Accordion>

<Accordion title="EPS (Austria)">
Electronic Payment Standard for Austrian bank transfers. Integrates with Erste Bank, Bank Austria, Raiffeisen, BAWAG, Volksbank.

- Real-time confirmation
- High consumer trust
- No chargebacks
</Accordion>

<Accordion title="Multibanco (Portugal)">
Portugal's interbank network supporting online banking, mobile, and ATM payments.

- **Online/Mobile:** Instant confirmation
- **ATM:** Customer receives reference (entity + reference number) to pay at any ATM within 24-72 hours

<Warning>
ATM payments may have delays. Use webhooks for async confirmation.
</Warning>
</Accordion>
</AccordionGroup>

## Testing

Use test API keys and set billing address country to match the payment method:
- `NL` for iDEAL
- `BE` for Bancontact
- `AT` for EPS
- `PT` for Multibanco

Follow the simulated bank authentication flow in the test environment.

## Troubleshooting

<AccordionGroup>
<Accordion title="Method not appearing">
Check: (1) billing country matches method's country, (2) currency is EUR, (3) method included in `allowed_payment_method_types`.
</Accordion>

<Accordion title="Bank authentication failed">
Customer cancelled or bank system unavailable. Retry or try a different payment method.
</Accordion>

<Accordion title="Multibanco payment pending">
Expected for ATM payments. Customer received reference but hasn't paid yet. Wait for webhook confirmation (expires in 24-72 hours).
</Accordion>
</AccordionGroup>
