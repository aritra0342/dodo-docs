## 介绍

[ParityDeals](https://www.paritydeals.com/) 是一个平台，能够根据客户的地理位置自动优化定价，通常称为“购买力平价”定价。

通过ParityDeals，您还可以全年提供[基于时间的自动折扣](https://www.paritydeals.com/features/time-based-discounts/)和[基于假期的促销](https://www.paritydeals.com/features/holiday-based-discounts/)。

<Info>
虽然您无法直接在 Dodo Payments 商店页面上嵌入 ParityDeals 横幅（因为不支持自定义脚本），但仍可以在您自己的网站或应用程序中有效使用 ParityDeals。
</Info>

## 工作原理

当ParityDeals集成到您的网站时，符合条件的访客将看到一个横幅，其中包含他们可以在结账时使用的基于位置的折扣代码。

<Frame>
  <img src="/images/ppp/1.png" alt="ParityDeals discount banner example" />
</Frame>

这些折扣代码由ParityDeals自动创建和管理，并将在您的**Dodo Payments仪表板**的**销售 → 折扣**下显示。

## 分步设置指南

### 1. 在ParityDeals上注册

访问 [app.paritydeals.com](https://app.paritydeals.com/) 并创建一个账户。

### 2. 创建交易

登录后，找到创建交易的选项。从下拉菜单中选择：

**→ "通过Dodo Payments创建交易"**

<Frame>
  <img src="/images/ppp/2.png" alt="Creating deals via Dodo Payments" />
</Frame>

### 3. 生成Dodo Payments API密钥

登录到您的**Dodo Payments仪表板**，并导航到：

**开发者 → API密钥**

点击**"创建新API"**，并给它起一个描述性的名称，比如"ParityDeals"。

<Frame>
  <img src="/images/ppp/3.png" alt="Creating a new API key in Dodo Payments" />
</Frame>

### 4. 将您的API密钥连接到ParityDeals

复制您创建的API密钥，并将其粘贴到ParityDeals界面中，以建立与您的Dodo Payments账户的连接。

<Frame>
  <img src="/images/ppp/4.png" alt="Connecting API key to ParityDeals" />
</Frame>

### 5. 选择产品

连接后，ParityDeals将获取您的Dodo Payments产品目录。选择您想要与交易关联的产品。

<Frame>
  <img src="/images/ppp/5.png" alt="Selecting products for ParityDeals" />
</Frame>

### 6. 配置交易设置

输入您的网站URL，并可选择特定的目标页面，在这些页面上显示折扣横幅。

然后选择您首选的交易类型：

- 基于位置的交易
- 假期交易
- 基于时间的闪购交易

<Frame>
  <img src="/images/ppp/6.png" alt="Configuring deal settings" />
</Frame>

### 7. 设置基于国家的规则

选择您希望提供本地化折扣的国家，并为每个国家定义折扣百分比。ParityDeals提供推荐的默认值，您可以根据需要进行调整。

<Frame>
  <img src="/images/ppp/7.png" alt="Setting country-based discount rules" />
</Frame>

### 8. 自定义横幅

通过以下方式设计您的折扣横幅：

- 自定义文本
- 选择颜色和布局
- 选择横幅位置（页面顶部或底部）

该横幅将根据您配置的规则显示给符合条件的访客。

<Frame>
  <img src="/images/ppp/8.png" alt="Customizing the discount banner" />
</Frame>

### 9. 将ParityDeals脚本添加到您的网站

保存您的活动后，ParityDeals 会生成一段脚本。将此脚本添加到您的网站代码中（通常放在 `<head>` 部分）。

该脚本会自动检测访客位置并显示适当的折扣横幅。

<Frame>
  <img src="/images/ppp/9.png" alt="ParityDeals integration script" />
</Frame>

## 一切准备就绪！

您现在可以提供：

- **基于位置的交易** - 根据购买力调整定价
- **倒计时/时间敏感折扣** - 通过限时优惠创造紧迫感
- **假期促销** - 利用季节性购物趋势

所有这些都通过ParityDeals自动化，并与Dodo Payments无缝协作。

有关更详细的信息，请参阅[ParityDeals文档](https://www.paritydeals.com/docs/)或联系他们的支持团队。
