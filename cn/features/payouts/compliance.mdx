## 针对印度

### 注册企业

印度的注册企业被分类为不同类型，每种类型需要特定的文件进行验证。

#### 支持的注册企业类型：

* 私人有限公司

* 有限责任合伙企业 (LLP)

* 合伙企业

* 个体经营

#### 注册企业所需文件：

| 企业类型           | 文件名称                  | 描述                                                          |
| ------------------- | ------------------------ | ------------------------------------------------------------ |
| **私人有限公司**     | 企业 PAN 号码            | 注册企业的 PAN 号码。                                        |
|                     | GSTIN                    | 用于税务目的的 GST 识别号码。                                |
|                     | CIN                      | 公司识别号码                                                |
|                     | DBA 名称                 | 在银行对账单上显示的“以某名义经营”的名称。                  |
| **LLP**             | 企业 PAN 号码            | 注册 LLP 的 PAN 号码。                                       |
|                     | GSTIN                    | 用于税务目的的 GST 识别号码。                                |
|                     | LLP 注册文件            | 证明 LLP 注册的文件。                                      |
|                     | DBA 名称                 | 在银行对账单上显示的“以某名义经营”的名称。                  |
| **合伙企业**         | 合伙企业 PAN 号码       | 合伙企业的 PAN 号码。                                        |
|                     | 合伙协议                 | 概述合伙条款和利益相关者角色的正式协议。                    |
|                     | DBA 名称                 | 在银行对账单上显示的“以某名义经营”的名称。                  |
| **个体经营**         | DBA 名称                 | 在银行对账单上显示的“以某名义经营”的名称。                  |

***

### 未注册企业

对于印度的未注册企业，由于它们不是合法注册的实体，因此需要不同的文件。

#### 未注册企业所需文件：

| 文件名称       | 描述                                                                    |
| --------------- | ---------------------------------------------------------------------- |
| PAN 号码        | 企业主的个人 PAN                                                      |
| DBA 名称       | 在银行对账单上显示的“以某名义经营”的名称。                            |
| 地址           | 包括 **街道、城市、州和邮政编码** 的完整地址，需提供证明。            |
| 国籍           | 个人或实体所有者的国籍。                                            |

***

## 关键利益相关者信息 - 印度

Dodo Payments 需要每个与企业相关的关键利益相关者的详细信息。每个利益相关者必须提供他们的 **PAN 卡** 作为合规要求的一部分。我们接受最多 4 个关键利益相关者的文件。如果您有超过 4 个关键利益相关者，请通过电子邮件与我们分享文件。

### 谁是关键利益相关者？

以下人员可以是公司的关键利益相关者：

#### 实益拥有者

实益拥有者是指任何自然人，无论是单独行动还是共同行动，或通过其他公司、LLP 或法律实体，拥有或有权拥有连接用户 25% 或更多的股份、资本或利润，或通过其他方式控制连接用户，包括享有任命大多数董事的权利或通过股权或管理权或通过股东协议或投票协议控制其管理或政策决策。

如果连接用户有多个实益拥有者，请提供所有人的必要文件。

#### 企业董事

根据印度公司法，任命并目前担任连接用户董事会董事的个人。

请提供所有董事的 PAN。

#### 企业代表

连接用户的员工或董事，授权代表其在使用 Dodo Payments 服务时行事。

#### 联合创始人

如果企业未注册，请提供与公司相关的所有联合创始人的文件，他们将拥有大多数股份（超过 25%）或在注册后成为公司的董事。如果有任何更改，可以在注册后更新。

## 税务表格

### 针对美国公民

美国公民，无论其商户类型如何，都需要提交 **W-9 表格**。该表格用于向 IRS 报告纳税人识别信息。

#### 美国公民所需的税务表格：

| 国籍状态         | 商户类型               | 所需表格 | 描述                                         |
| ------------------ | --------------------- | --------- | ------------------------------------------- |
| **美国公民**       | 个人和企业           | W-9       | 用于报告纳税人识别的 IRS 表格。            |

* **说明**：美国公民应下载、填写并上传签署的 W-9 表格，确保所有细节清晰可见。

* **下载链接**：[https://www.irs.gov/pub/irs-pdf/fw9.pdf](https://www.irs.gov/pub/irs-pdf/fw9.pdf)

***

### 针对非美国公民

非美国公民必须根据其商户类型提交不同的表格，具体取决于他们是个人还是注册的商业实体。

#### 非美国公民 - 个人

非美国个人商户需要提交 **W-8BEN 表格**。该表格用于向 IRS 报告信息并根据适用的税收条约申请优惠。

##### 非美国个人所需的税务表格：

| 国籍状态         | 商户类型               | 所需表格 | 描述                                                       |
| ------------------ | --------------------- | --------- | --------------------------------------------------------- |
| **非美国公民**     | 个人                  | W-8BEN    | 用于非美国个人申请税收条约优惠的 IRS 表格。            |

* **说明**：非美国个人应下载、填写并上传签署的 W-8BEN 表格，确保所有细节清晰可见。

* **下载链接**：[https://www.irs.gov/pub/irs-pdf/fw8ben.pdf](https://www.irs.gov/pub/irs-pdf/fw8ben.pdf)

#### 非美国公民 - 商业实体

非美国商业实体需要提交 **W-8BEN-E 表格**。该表格由外国实体用于申请外国身份，并在适用时根据所得税条约申请减免预扣税率。

##### 非美国商业实体所需的税务表格：

| 国籍状态         | 商户类型               | 所需表格 | 描述                                                                  |
| ------------------ | --------------------- | --------- | -------------------------------------------------------------------- |
| **非美国公民**     | 商业实体             | W-8BEN-E  | 用于非美国实体申请税收条约优惠（如适用）的 IRS 表格。            |

* **说明**：非美国商业实体应下载、填写并上传签署的 W-8BEN-E 表格，确保所有细节清晰可见。

* **下载链接**：[https://www.irs.gov/pub/irs-pdf/fw8bene.pdf](https://www.irs.gov/pub/irs-pdf/fw8bene.pdf)
