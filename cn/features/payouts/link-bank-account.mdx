## 链接银行账户流程

1. 在Dodo Payments仪表板中转到**付款设置**部分，并选择**银行信息**选项卡
2. 选择**您的银行账户位置**（印度、美国、欧洲、其他地区）。
3. 按照下面的说明填写准确的详细信息。
4. 上传符合可见性要求的**银行对账单**以进行验证。

## 根据国家的银行信息字段

### **1. 印度**

| 字段 | 描述 |
| --- | --- |
| **账户持有人姓名** | 与银行账户详细信息一致的确切姓名 |
| **银行名称** | 银行的名称 |
| **银行账户类别** | 银行账户类型（个人或商业） |
| **银行账户货币** | 本地货币的3字母代码（例如，INR） |
| **银行账户号码** | 唯一的银行账户号码 |
| **IFSC代码** | 根据银行对账单的IFSC代码 |
| **银行对账单上传** | 上传显示所有相关详细信息的可见银行对账单 |

### **附加指南**

- **银行对账单可见性**：所有上传的银行对账单应具有清晰可读的账户详细信息。避免裁剪或遮挡任何信息，因为这可能会延迟验证过程。
- **账户名称准确性**：账户持有人姓名应与提交的银行文件上的姓名完全一致，以避免在验证过程中出现差异。
- **SWIFT/BIC和路由号码**：确保SWIFT/BIC和路由号码准确无误，以便于国际和国内转账，确保顺利付款。
- **货币代码**：始终使用正确的3字母货币代码（例如，印度为INR，美国为USD，欧洲为EUR），以符合银行账户的本地货币。
如果您有任何疑问，可以使用以下链接来识别您的货币代码： [https://www.iban.com/currency-codes](https://www.iban.com/currency-codes)
