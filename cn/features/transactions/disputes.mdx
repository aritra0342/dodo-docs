## 介绍

争议部分提供了所有提出的争议的详细视图，以及这些争议的当前状态。您需要在您的端管理争议解决，但仪表板将提供状态、争议原因、争议 ID 的可见性，以方便访问信息。

## 查看和理解争议

<Steps>
{/* LOCKED_PATTERN_e4a7709e0b3add883f6a7f68d7d8fc0c */}
    在仪表板中转到 <strong>交易</strong> 页面并选择 <strong>争议</strong> 选项卡。此页面显示与您的账户相关的所有争议列表。
</Step>

{/* LOCKED_pattern_66079061e39a2d134af31a1f91372bfa */}
    点击任何争议条目旁的 <strong>眼睛图标</strong> 打开详细视图。在此处，您可以查看争议原因、状态、争议 ID 以及其他相关信息。
</Step>

{/* LOCKED_PATTERN_2a5fe9b244e5c59c326ec03271abe83d */}
    在争议详细信息中，找到 <strong>关联支付</strong> 字段。点击此处的 <strong>眼睛图标</strong>，查看与争议相关的完整支付详细信息。
</Step>
</Steps>

## 回应争议

当您的交易被提出争议时，请按照以下步骤有效回应：

1. **检查您的仪表板**
   - 导航到交易 → 争议部分
   - 查找新的争议条目

2. **查看电子邮件通知**
   - 检查您注册的电子邮件以获取争议创建通知

3. **选择您的回应**
   - 您可以选择接受或反驳争议
   
   <Frame>
       <img src="/images/disputes/1.png" alt="争议创建" />
   </Frame>

4. **如果接受**
   - 不需要进一步操作
   - 您将失去争议

5. **如需反驳**
   - 准备提交证据
   - 遵循以下最佳实践

       <img src="/images/disputes/2.png" alt="争议创建" />
   </Frame>

## Visa 快速争议解决 (RDR)

Visa 快速争议解决 (RDR) 是一个自动争议预防工具，可在合格争议 **成为正式退单** 之前解决。当持卡人与其银行发起争议时，如果交易金额低于您配置的阈值，RDR 会拦截争议并自动代表您发起退款，从而防止退单的发生。

### RDR 的重要性

争议是数字企业面临的最昂贵且最耗时的挑战之一。除了直接的财务影响外，过高的争议率可能带来严重后果：

{/* LOCKED_PATTERN_bd3b9ce11ef978f59c6eb5461169b62d */}
{/* LOCKED_PATTERN_7bfcbc7069cd77e396d8396a2c50beb6 */}
  每起争议都会让您承担交易金额 **以及** 不可退还的争议费用，无论您是胜诉还是败诉。
</Card>
{/* LOCKED_PATTERN_ff6b0cb3e188d562719344330e286e36 */}
  卡网络会监控您的争议与交易比例。Visa 的阈值为 **0.9%** —— 超过此值会使您进入监控计划，并面临不断升级的罚款。
</Card>
{/* LOCKED_PATTERN_71ae679822adfc7721452d692f952c43 */}
  持续较高的争议率可能导致更高的处理费用、准备金要求，甚至支付处理商终止您的账户。
</Card>
{/* LOCKED_PATTERN_9d15686e617b83741dfb528d4c3cd022 */}
  每起争议都需要收集证据、准备回应并进行后续跟进 —— 耗费大量本可用于构建产品的宝贵时间。
</Card>
</CardGroup>

{/* LOCKED_PATTERN_4907e9f6f7fbd509120d7a87afc829e9 */}
当您的争议率超过 **0.9%** 或一个月内收到超过 **100 起争议** 时，Visa 的争议监控计划 (VDMP) 将启动。若商家持续留在计划中，罚款每月递增，可高达 **每月 50,000 美元以上**。
{/* LOCKED_PATTERN_176d815432e7554ac558e8631b2bc397 */}

### RDR 的工作原理

<Steps>
{/* LOCKED_PATTERN_16b3bf8926d33e9c464c5064858dd86c */}
  客户联系其银行，就其对账单上的 Visa 交易提出争议。
</Step>

{/* LOCKED_PATTERN_ee59f99bfff31c37bafd82751c977301 */}
  在争议正式成为退单之前，Visa 的 RDR 系统会根据您配置的规则检查该交易是否符合自动解决条件。
</Step>

{/* LOCKED_PATTERN_d003c64f69b831bbc41f643c2afed21a */}
  如果交易金额在您的阈值内，Visa 网络会自动向持卡人发起退款。争议即时解决。
</Step>

{/* LOCKED_PATTERN_58faf14a5cf9add7680997fa855a0d01 */}
  由于争议在成为正式退单之前已解决，因此 **不会计入 Visa 的争议率**。您可以避免争议费用及对账户的负面影响。
</Step>
</Steps>

<Info>
通过 RDR 自动退款的交易将在您的 Dodo Payments 仪表板中显示为 **争议失败**。这是预期行为 —— 为防止正式退单，系统已自动为持卡人退款，“失败”状态表示资金已退还给持卡人。
</Info>

### 主要优势

| 优势 | 描述 |
|---------|-------------|
| **更低的争议率** | RDR 解决的争议不会计入 Visa 的争议比例，有助于保持在 0.9% 阈值以下 |
| **无需争议费用** | 由于争议未成为正式退单，可避免不可退还的争议费用（通常每起 15-25 美元） |
| **零运营负担** | 无需收集证据、撰写回应或追踪争议截止日期 —— 解决过程全自动 |
| **更好的客户体验** | 持卡人立即收到退款，而不是等待数周才能解决争议 |
| **保护账户状态** | 避免进入 Visa 的监控计划，防止罚款升级和处理限制 |


### 配置

RDR 通过 **美元阈值** 进行配置 —— 任何在该金额或以下的 Visa 争议将自动通过退款解决。超过阈值的争议将按照正常争议流程处理，您可以提交证据进行反驳。


<Info>
例如，设定阈值为 100 美元：75 美元的争议将自动解决（退款），而 150 美元的争议则会继续正常流程，供您审核并回应。
</Info>
