<Card title="产品目录" img="/images/our-features/Product Catalog.webp" href="/features/product-catalog/overview" >
 此功能允许商家管理他们的产品供应，无论是一次性购买还是基于订阅的产品。您可以创建、配置和管理具有灵活定价选项和计费周期的产品。
</Card>
<Card title="支付" img="/images/our-features/Payments.webp" href="/features/transactions/payments" >
 支付功能是 Dodo Payments 的核心，处理端到端的支付处理。此部分提供有关支付如何处理和管理的全面透明度。
</Card>
<Card title="退款" img="/images/our-features/Refunds.webp" href="/features/transactions/refunds" >
 此功能帮助商家以有序和及时的方式管理和发起退款请求。退款功能有助于确保客户满意，同时保持对财务操作的业务控制。
</Card>
<Card title="开票" img="/images/our-features/Invoicing.webp" href ="/features/invoice-generation">
 Dodo Payments 提供自动开票功能，通过在每次成功交易后自动生成发票来简化计费过程。商家还可以自定义发票模板，以与其业务品牌保持一致。
</Card>
<Card title="支付" img="/images/our-features/Payouts.webp" href="/features/payouts/overview" >
支付功能简化了将资金从您的 Dodo Payments 账户转移到您的银行账户的过程。此部分确保商家对其收入拥有完全控制，并了解任何相关费用。
</Card>

<Card title="分析与报告" img="/images/our-features/Reports.webp" href="/features/analytics-and-reporting" >
分析与报告功能为商家提供了对其业务表现的详细洞察。查看有关收入、客户活动、交易成功率等的全面报告，所有这些都可以通过 Dodo Payments 仪表板访问。
</Card>
<Card title="客户" img="/images/our-features/Customers.webp" href="/features/customers" >
客户页面为商家提供了所有客户详细信息的中央视图，并允许商家下载客户记录。
</Card>
<Card title="欺诈检测与数据安全" img="/images/our-features/Fraud.webp" href="/features/fraud-detection">
我们的综合平台遵循所有标准安全和欺诈检测实践，以确保交易的安全性。
</Card>
<Card title="许可证密钥" img="/images/our-features/license-key.png" href="/features/license-keys">
许可证密钥管理功能旨在帮助商家通过生成、管理和跟踪唯一的许可证密钥来保护其数字产品和服务。
</Card>
<Card title="Webhooks" img="/images/our-features/Webhook.png" href="/developer-resources/webhooks">
Webhooks 提供了一种机制，用于在 Dodo Payments 中发生特定事件时进行实时通知，从而实现与您的应用程序的无缝自动化和集成。
</Card>
<Card title="账户摘要与支付钱包" img="/images/our-features/account-summary.png"  href="/features/account-summary-payout-wallet">
账户摘要提供所有财务交易的详细分解，包括支付、退款、支付、交易费用和税费。这确保商家对其收入和扣除项有全面的可见性。
</Card>
<Card title="API 密钥生成" img="/images/our-features/api-key-generation.png"  href="/features/api-keys">
API 密钥生成允许商家安全地验证和集成其系统与 Dodo Payments 平台，以便无缝访问支付、产品和订阅功能。
</Card>
<Card title="Atlas 演示产品" img="/images/our-features/atlas-demo.png" href="/features/atlas-demo-website">
Atlas 演示产品是 Dodo Payments 的一个演示网站，展示了如何使用我们的结账解决方案销售数字产品（如软件许可证、数字下载或在线会员）。商家可以看到订阅和一次性购买流程的实际操作。
</Card>
</CardGroup>
