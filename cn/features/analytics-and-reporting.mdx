## 企业级分析

主分析仪表板提供跨整个业务的全面洞察。可在侧边栏 **Business > Analytics** 下访问。

### 收入分析

跟踪收入与支付活动，以了解收入趋势并识别表现最佳的市场。

{/* LOCKED_PATTERN_d4aa93b632e40589a23822a7121a4bbb */}
  <img src="/images/analytics/revenue.png" alt="Total revenue and revenue trend chart" />
</Frame>

- **总收入：** 即时查看历史总收入与近期趋势。
- **收入趋势：** 30 天图表帮助你发现收入的高峰或下降。

{/* LOCKED_PATTERN_63818b5832852a76b80158a21d97e657 */}
  <img src="/images/analytics/revenue-2.png" alt="Top revenue countries and growth rate chart" />
</Frame>

- **增长率：** 跟踪月度收入增长以衡量业务势头。
- **地理洞察：** 可视化收入最高的五个国家/地区。

<Tip>
利用这些洞察将营销和产品投入集中在高绩效区域和增长期。
</Tip>

### 收入细分

通过详细分解跟踪你的经常性收入指标：

- **预计 ARR**：带有趋势分析的年化经常性收入预测
- **MRR 细分与增长率**：包含环比增长指标的月度经常性收入分析
- **新 MRR 图表**：新客户收入获取的可视化呈现
- **扩展 MRR 图表**：跟踪现有客户升级与扩展带来的收入增长
- **流失 MRR 图表**：监控因客户取消或降级而流失的收入
- **净新增 MRR 图表**：综合新、扩展与流失 MRR 的净收入影响

{/* LOCKED_PATTERN_b02415e4d71ec542c6aea99b61ab297d */}
  <img src="/images/analytics/revenue-breakdown.png" alt="Revenue Breakdown dashboard showing projected ARR, MRR breakdown, and growth rates" />
</Frame>

### 留存分析

监控客户留存与流失模式：

- **客户流失率：** 取消订阅的客户比例
- **收入流失率：** 因取消与降级而流失的收入比例
- **流失率趋势：** 随时间追踪客户与收入流失模式的历史趋势
- **用户留存矩阵：** 显示不同时间段客户留存率的队列分析

{/* LOCKED_PATTERN_9b338d767cb175cebc8a05f2fe9acfa6 */}
  <img src="/images/analytics/retention.png" alt="Retention Analytics dashboard showing customer churn rate, revenue churn rate, and churn rate trends" />
</Frame>

### 客户分析

分析客户获取、留存和消费模式，以优化客户策略。

{/* LOCKED_PATTERN_423875f0ac3863c748d167e3d5a769f0 */}
  <img src="/images/analytics/customer.png" alt="Customer analytics: ARPU, active customers, customer split, top customers" />
</Frame>

- **ARPU（每用户平均收入）：** 了解每位客户平均贡献金额。
- **活跃客户：** 查看有多少客户正在与你的业务互动。
- **客户构成：** 即刻查看新客户与回头客的比例。
- **重点客户：** 识别消费最高的客户以便进行精准沟通。

<Tip>
监测 ARPU 和客户构成以优化留存与获客策略。
</Tip>

### 交易成功与失败

监控支付与退款成功率，快速识别并解决交易问题。

{/* LOCKED_PATTERN_b5047757528029f4b0abcda56b75e348 */}
  <img src="/images/analytics/success-rate.png" alt="Payment and refund success rates, failure reasons, and payment breakdown" />
</Frame>

- **支付成功率：** 跟踪成功支付尝试所占比例。
- **退款成功率：** 监控退款处理成功的数量。
- **失败原因：** 快速识别常见问题（如卡片过期、不支持的方式）。

  {/* LOCKED_PATTERN_ae96e7c89f29b1aafa638bfc6a5aace4 */}
    获取所有可能的交易失败代码及其原因。
  </Card>
- **支付细分：** 可视化展示成功与失败交易的分布。

<Warning>
如果你发现成功率下降或失败激增，请立即查看失败原因并迅速处理，以最大限度地减少收入损失。
</Warning>

### 筛选与自定义

你可以按以下方式筛选分析数据：
- **产品类型**（一次性与订阅）
- **日期范围**（可定制）
- **支付方式**

<Tip>
使用筛选器深入了解特定产品、时间段或支付方式的精准洞察。
</Tip>

## 产品级分析

除了整体业务指标外，你还可以查看**每个单独产品**的详细分析。导航到 **Products > [Select a Product]** 访问该产品的专属分析仪表板。

{/* LOCKED_PATTERN_e2a07d7bc80e7418b404823a54c36811 */}
  <img src="/images/product-analytics/overview.png" alt="Product analytics overview tab with total revenue, successful payments, and revenue trend chart" />
</Frame>

产品分析仪表板分为五个选项卡：

{/* LOCKED_PATTERN_6d744560e4135463c359b094ae69cd5f */}
{/* LOCKED_PATTERN_6e4704d1898defaa77c750aeddaeee7d */}
收入摘要、趋势图与交易历史。
</Card>
{/* LOCKED_PATTERN_5d07d2e523844b62c983ff62b4723a83 */}
按照支出排名的重点客户及完整分解表。
</Card>
{/* LOCKED_PATTERN_3325fe8a85f123d25260486f5c90ce7b */}
客户与收入流失率及趋势分析。
</Card>
{/* LOCKED_PATTERN_198764c6a6244eda5acbe24bd1917ee7 */}
包含状态与计费详情的所有订阅。
</Card>
{/* LOCKED_PATTERN_8c5a811104c87f90458e8aa5dba1b1a1 */}
月度经常性收入细分——新、扩展、流失与净值。
</Card>
</CardGroup>

所有选项卡都支持**日期筛选**（默认为全部时间），以便你聚焦任意时期。

### 概览选项卡

概览选项卡提供该产品财务表现的高层快照。

- **总收入：** 该产品产生的累计收入。
- **成功支付次数：** 成功交易的总数量。
- **收入趋势图：** 30 天折线图显示收入变化。使用 **Show Daily** 按钮在累计与每日视图之间切换。

图表下方的**交易**表列出该产品的每笔支付：

| 列 | 描述 |
|--------|-------------|
| Payment ID | 交易的唯一标识 |
| Status | `Successful` 或 `Failed` |
| Amount | 含币种的交易金额 |
| Date (UTC) | 交易时间戳 |
| Payment Method | 使用的卡、UPI 或其他方式 |
| Refund | 发起退款的选项 |

{/* LOCKED_PATTERN_fae68d50781afa79a0cf421ce53bfe39 */}
  <img src="/images/product-analytics/overview-transactions.png" alt="Product transactions table with payment IDs, statuses, amounts, and refund actions" />
</Frame>

使用 **Filter by status** 按钮将列表缩小到成功或失败交易。

### 客户选项卡

查看是谁在购买该产品以及他们的支出情况。

{/* LOCKED_PATTERN_9673cca2a71ee06402a10916fa1f1576 */}
  <img src="/images/product-analytics/customers.png" alt="Customers tab showing top 3 customers ranked by revenue with a detailed breakdown table" />
</Frame>

- **本月重点客户：** 可视化 podium 展示依据总支出排名的**前三名客户**，显示收入金额、姓名与邮箱。
- **客户表：** 详细列出所有客户及其客户 ID 和该产品带来的总收入。

### 留存选项卡

了解该产品的订阅者保留情况。

{/* LOCKED_PATTERN_6ee5749dbf26d589b6efe9d2cf7b2924 */}
  <img src="/images/product-analytics/retention.png" alt="Retention tab showing customer churn rate, revenue churn rate, and churn rate trend chart" />
</Frame>

- **客户流失率：** 取消订阅的用户比例。
- **收入流失率：** 因取消或降级流失的收入比例。
- **流失率趋势：** 双折线图同时跟踪两项指标随时间变化，帮助你发现趋势并采取纠正措施。

### 订阅用户选项卡

列出与该产品关联的所有订阅。

{/* LOCKED_PATTERN_25fcf28f1b00f18c88e6ffcad1a8a379 */}
  <img src="/images/product-analytics/subscribers.png" alt="Subscribers tab with subscription IDs, customer emails, statuses, start dates, and next billing information" />
</Frame>

| 列 | 描述 |
|--------|-------------|
| Subscription ID | 订阅的唯一标识 |
| Customer Email | 订阅用户的邮箱地址 |
| Status | `Active`、`Failed`、`On hold` 或 `Pending` |
| Product Name | 该订阅所属的产品 |
| Start Date | 订阅开始时间 |
| Next Billing Date | 下次续订日期 |
| Next Billing Amount | 下次续费金额（不含税） |

### MRR 选项卡

详细分解产品的月度经常性收入。

{/* LOCKED_PATTERN_d953379143cca55583375c5f6e73807e */}
  <img src="/images/product-analytics/mrr.png" alt="MRR tab with stacked bar chart showing new, expansion, and churned MRR by month" />
</Frame>

**MRR 图表**以堆叠柱状图形式逐月可视化经常性收入构成。

图表下方的四张指标卡提供详细洞察：

{/* LOCKED_PATTERN_4da84c6504cd5ce9e8066df60cf31c89 */}
  <img src="/images/product-analytics/mrr-breakdown.png" alt="MRR breakdown showing New MRR, Expansion MRR, Churned MRR, and Net New MRR with trend charts" />
</Frame>

| 指标 | 描述 |
|--------|-------------|
| **新 MRR** | 本月新增订阅带来的收入，并提供环比变化 |
| **扩展 MRR** | 现有订阅用户升级或新增服务带来的附加收入 |
| **流失 MRR** | 因取消或降级订阅而失去的收入 |
| **净新增 MRR** | 净结果：新 MRR + 扩展 MRR − 流失 MRR |

每张卡片都包含趋势迷你图和相对于上月的百分比变化指标。

## 报表

导航至 **Business > Reports** 访问报表套件。

可用报表包括：

- **支付报表：** 所有已处理支付的详细记录，包含交易状态跟踪
- **退款报表：** 所有退款活动的综合日志，含详细退款信息
- **付款报表：** 汇总支付到你账户的款项
- **按产品销售报表：** 提供收入与业务增长指标的销售表现洞察
- **支付失败报表：** 总结失败支付尝试及失败原因
- **客户报表：** 按客户组织的支付汇总
- **争议报表：** 支付争议摘要及解决状态
- **账户摘要报表：** 完整的账户概览与摘要

{/* LOCKED_PATTERN_9b478cf20d5e690c27633758fcb7bf52 */}
  <img src="/images/analytics/reports.png" alt="Reports 2.0 dashboard showing various report types" />
</Frame>

### 付款报表

详细的付款报表清晰展示每笔付款的费用与交易详情。可在 **Business > Payouts** 中访问。

{/* LOCKED_PATTERN_0c3c42c5ebca67e1f69862355c9b3cce */}
  <img src="/images/analytics/payout.png" alt="Payout Report dashboard showing detailed payout information" />
</Frame>

<Steps>
{/* LOCKED_PATTERN_134f913d83c9e55b8f400bc9528af87d */}

## 最佳实践

<Tip>
- 定期查看分析内容，及早发现趋势与异常。
- 利用筛选器比较不同产品、时间段或地域的表现。
- 深入产品级分析，了解哪些产品贡献最大收入、哪些流失率最高。
- 若数据异常或丢失，优先检查筛选器与日期范围设置。
</Tip>

<Warning>
如发现持续性差异或缺失数据，请联系 <a href="mailto:support@dodopayments.com">support@dodopayments.com</a> 寻求帮助。
</Warning>
