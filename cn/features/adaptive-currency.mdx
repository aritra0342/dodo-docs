<Info>
自适应货币允许您显示本地化价格并以客户的本地货币接受付款。这减少了结账时的摩擦并建立了信任。
</Info>

## 什么是自适应货币？

自适应货币以客户的本地货币显示产品价格，而不是固定的全球货币（例如，美元）。启用后，结账默认使用支持国家的检测到的本地货币，并可以切换到美元。

## 主要好处

- **本地化支付体验**：价格默认以本地货币显示。
- **更多支付方式**：解锁可用于本地货币的支付方式。
- **同币种退款**：以客户支付的货币退款。

## 启用自适应货币

<Steps>
<Step title="打开业务设置">
登录到您的商户仪表板，转到业务 → 业务设置。
</Step>

<Step title="切换自适应定价">
启用自适应定价切换。您可以随时禁用它。

<Warning>
更改仅适用于未来的交易。
</Warning>
</Step>

<Step title="确认并保存">
保存您的设置。

<Check>
验证测试结账在支持的情况下以您的本地货币显示价格。
</Check>
</Step>
</Steps>

## 客户体验

1. **检测**：系统根据账单地址检测客户的国家。
2. **货币选择**：如果国家被支持，价格默认以本地货币显示。客户可以切换到美元。
3. **支付方式**：在适用的地方显示本地化支付方式。
4. **结账**：以所选货币完成支付。

## 转换和费用

自适应货币以客户的本地货币收费，使用最新的汇率。

- **您支付**：0%
- **您的客户支付**：根据订单价值2–4%

结账时应用的分层费用：

- 订单金额低于500美元时为4%
- 500到1500美元的订单为3%
- 超过1500美元的订单为2%

<Info>
自适应货币费用由客户承担，而不是商户。
</Info>

## 退款和调整

Dodo Payments以客户最初支付的货币发放退款，使用最新的汇率。全球美元金额在您的仪表板、发票和退款中保持固定。这意味着客户可能会根据外汇变化收到比原本地货币金额更多或更少的退款。

<Info>
自适应货币费用通常是外汇费用，不予退款。
</Info>

**示例退款**

1. 您以100美元的价格销售一件产品，并启用了自适应货币。
2. 一位加拿大客户以1.37加元兑换1美元的汇率看到137加元并完成购买。
3. 我们处理付款，将137加元转换为100美元进行结算。
4. 后来，汇率变为1.40加元兑换1美元，您发放全额退款。
5. 我们扣除100美元，并向客户退款140加元。

## 发票和税务

- 发票仅显示结算货币金额。
- 税费和平台费用根据结算货币金额计算。
- 示例：10美元的销售转换为36阿联酋迪拉姆，仪表板和发票中仍显示为10美元。

<Info>
所有金额均根据Dodo Payments的内部四舍五入逻辑进行四舍五入。
</Info>

## 集成示例

**带有账单货币的结账会话**

传递 `billing_currency` 以明确设置会话的计费货币。

<Info>
当自适应定价被禁用时，`billing_currency` 将被忽略。
</Info>

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    {
      product_id: 'prod_one_time_or_subscription',
      quantity: 1
    }
  ],
  billing_currency: 'AED',
  return_url: 'https://example.com/return'
});
```
