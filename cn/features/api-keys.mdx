## 介绍

API 密钥生成允许商家安全地验证并将其系统与 Dodo Payments 平台集成，以无缝访问支付、产品和订阅功能。

**注意：** 在测试模式下创建的 API 密钥将仅限于测试模式环境，反之亦然适用于实时模式。
