## 新功能 🚀

- **订阅的首次付款**：最受欢迎的功能来了！现在，首次付款将合并为一笔交易，包括授权和实际的订阅金额。

- **UPI二维码支付方式**：为印度市场新增UPI二维码作为支付方式，允许客户使用UPI二维码进行支付。
    
    <img src="/images/changelog/upi-qr.png" alt="UPI QR功能"/>

- **结账时的税号**：客户现在可以在结账页面添加他们的税号，以便开具发票和税务用途。

- **仪表板重构**：整个Dodo Payments仪表板已重构，以更直观和用户友好的方式呈现。

- **自助入门指南**：新用户现在可以使用自助入门指南开始使用Dodo Payments。

- **多语言支持**：在结账页面新增对14种以上语言的支持，允许客户使用他们首选的语言进行支付。支持的语言包括：
    - 英语 (en)
    - العربية (ar) // 阿拉伯语
    - Català (ca) // 加泰罗尼亚语
    - 中文 (cn) // 中文
    - Deutsch (de) // 德语
    - Español (es) // 西班牙语
    - Français (fr) // 法语
    - עברית (he) // 希伯来语
    - Italiano (it) // 意大利语
    - 日本語 (ja) // 日语
    - Nederlands (nl) // 荷兰语
    - Polski (pl) // 波兰语
    - Português (pt) // 葡萄牙语
    - Svenska (sv) // 瑞典语

- **发票PDF改版**：发票PDF已进行了新的、更加清晰和有效的设计。

- **集中帮助小部件**：新增集中帮助小部件，以简化和改善支持。

## 改进和错误修复 🔧

- **速率限制**：在发票API中添加了速率限制，以防止滥用并确保系统稳定性。

- **静态支付链接中的元数据**：在静态支付链接中添加了对元数据的支持。有关更多详细信息，请参阅[文档](/developer-resources/integration-guide#1-static-payment-links)。

- **小幅增强和修复**：进行了各种小幅增强和修复，以改善整体性能和用户体验.
