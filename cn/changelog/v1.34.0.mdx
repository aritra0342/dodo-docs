## 新功能 🚀

- **Digital Product Delivery**：流畅的数字产品交付系统，自动完成履约，确保客户在支付成功后立即收到数字商品。更多信息请参阅我们的[Digital Product Delivery](/features/digital-product-delivery)文档。

  </Frame>

- **React Native SDK（Beta）**：发布了我们的React Native SDK beta版本，使得在原生Android和iOS应用中进行安全支付处理成为可能，支持可自定义的UI组件和屏幕。有关更多信息，请参阅我们的[React Native SDK](/developer-resources/react-native-integration)文档。

- **New Proration Mode**：引入了 `full_immediately` 计费模式，在订阅变更时允许立即收取全额费用，为您的业务需求提供更灵活的计费选项。

- **订阅的自适应货币**：增强了订阅管理，支持自适应货币，根据客户位置和当地货币偏好自动调整定价。

- **即时按需处理**：改善了按需收费的处理速度，使其几乎瞬时完成，减少交易时间并提供更快的支付确认。

- **验证表单管理**：在Dodo Payments仪表板的验证部分中添加了全面的验证表单数据查看功能，允许您在一个集中位置查看和管理所有提交的验证文件。

- **订阅取消增强**：实现了在账单周期结束时取消订阅的功能，现在在仪表板和客户门户中均可用，以改善用户体验和账单透明度。

- **一般改进**：各种小修复和性能改进，以增强整体平台的稳定性和用户体验。
