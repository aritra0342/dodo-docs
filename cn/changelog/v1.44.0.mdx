## 新功能 🚀

- 自适应货币功能现在支持更多货币，进一步增强了更多地区的本地化支付体验。新支持的货币包括：
  - MVR（马尔代夫卢菲亚）
  - MUR（毛里求斯卢比）
  - ZAR（南非兰特）
  - PHP（菲律宾比索）
  - FJD（斐济元）
  - QAR（卡塔尔里亚尔）
  - MYR（马来西亚林吉特）

有关自适应货币功能及其好处的详细信息，请访问[自适应货币文档](/features/adaptive-currency)。

- Dodo Payments 结账页面已添加土耳其语支持。

- 引入了一种新的按比例计费模式，`difference_immediately`。此模式在通过 API 更改计划时，无论计费周期剩余时间如何，都会立即收取差额。[文档](/developer-resources/subscription-integration-guide)。
