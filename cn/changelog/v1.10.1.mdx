## 新功能 🚀

- **Checkout 2.0**：引入了全新设计的结账体验，特点包括：
    - 精简现代的界面，提升用户体验
    - 新的 Query Collector 系统，可将参数以唯一 ID（例如 `sess_1a2b3c4d`）存储在会话存储中
    - 整个结账流程中持续提供数据可用性
    - 未来版本计划推出更多增强功能！
  
    <Frame><img src="/images/changelog/checkout_2.png" alt="Checkout 2.0"/></Frame>

- **可配置支付方式**：新的 `allowed_payment_methods` 配置允许商户在结账时自定义可用的支付选项。有关实施详情，请参阅[允许的支付方式](/features/payment-methods#configuring-payment-methods)文档。

- **动态税率计算**: 基于客户账单地址的实时税率计算，以便在静态支付链接上提供更准确的定价。

- **按需订阅**: 新的 API 端点用于灵活的订阅计费（适用于部分商家）。在 [按需订阅](/features/subscription#enabling-on-demand-subscriptions) 指南中了解更多信息。

- **其他更新**:
    - 性能优化
    - 错误修复
    - 一般系统改进
