## 新功能 🚀

### 基于使用的计费 
根据实际消费精确计费。定义计量器，摄取事件，并自动将使用量转换为费用。

- **灵活的聚合**：计数、求和（按属性）、唯一、最大、最后
- **事件过滤**：使用丰富的比较器进行AND/OR逻辑，仅包含相关事件
- **产品关联**：将计量器附加到具有单价和可选免费阈值的产品
- **分析**：跟踪趋势、事件流和每个客户的消费

### 100% 订阅折扣
对任何数量的计费周期应用100%的折扣，以支持免费期、赞助或促销活动。发票反映全额折扣，按比例分配遵循您的计划设置。

### 小改进

- 退款收据 API 现在可以公开访问。查看文档: [/api-reference/refunds/get-refund-receipt](/api-reference/refunds/get-refund-receipt).
- 在获取支付 API 响应和支付 Webhook 中添加了 `checkout session ID`，以帮助关联支付与其来源结账会话。
