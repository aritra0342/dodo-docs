## 新功能 🚀

- **客户门户**: 引入了 `Customer Portal` 功能，允许客户管理他们的订阅、发票和支付方式。您可以通过 Dodo Payments Dashboard 或 API 为您的客户创建静态或动态的客户门户链接。在仪表板上，导航到 `Sales > Customers > Share Invite`。

- **客户门户会话 API**: 添加了 `Customer Portal Session` API，以为特定客户创建客户门户的会话。请注意，该会话仅在 24 小时内有效。有关更多详细信息，请查看 [API 参考](/api-reference/customers/create-customer-portal-session)。

- **其他**：小幅改进和错误修复。
