## 新功能 🚀

### 增强的分析仪表板

我们扩展了分析仪表板，提供全面的收入和留存洞察，帮助您更好地理解业务表现。

#### 收入细分分析

跟踪您的经常性收入指标，提供详细的细分：

- **预计ARR**：年度经常性收入预测及趋势分析
- **MRR细分和增长率**：月度经常性收入分析及环比增长指标  
- **新MRR图表**：新客户收入获取的可视化表示
- **扩展MRR图表**：跟踪现有客户升级和扩展带来的收入增长
- **流失MRR图表**：监控因客户取消和降级而损失的收入
- **净新MRR图表**：结合新、扩展和流失MRR的净收入影响

#### 留存分析

监控客户留存和流失模式：

- **客户流失率**：取消订阅的客户百分比
- **收入流失率**：因取消和降级而损失的收入百分比
- **流失率趋势**：历史跟踪客户和收入流失模式
- **用户留存矩阵**：显示不同时间段客户留存率的队列分析

### 报告2.0

我们彻底改造了报告部分，提供更全面和详细的报告能力。请前往 **业务 > 报告** 访问新的报告套件。

可用的报告包括：

- **支付报告**：所有处理支付的详细记录及交易状态跟踪
- **退款报告**：所有退款活动的综合日志及详细退款信息
- **支付汇款报告**：对您账户的汇款摘要
- **按产品销售报告**：销售表现洞察及收入和业务增长指标
- **支付失败报告**：失败支付尝试的摘要及失败原因
- **客户报告**：按客户组织的支付摘要
- **争议报告**：支付争议摘要及解决状态
- **账户摘要报告**：完整的账户概述和摘要

<Steps>
<Step title="打开远程MCP服务器">
    在浏览器中访问 <a href="https://mcp.dodopayments.com" target="_blank">https://mcp.dodopayments.com</a>。
</Step>
<Step title="复制MCP JSON配置">
    在网站上，复制为您的客户提供的MCP JSON配置。
</Step>
<Step title="配置您的AI应用程序">
    将配置粘贴到您应用程序的MCP设置中（例如，Cursor或Claude）。
    <Tip>
    对于Cursor，您的配置应如下所示：
    </Tip>

    ```json
    {
      "mcpServers": {
        "dodopayments_api": {
          "command": "npx",
          "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
        }
      }
    }
    ```

</Step>
<Step title="完成身份验证和环境设置">
    添加JSON配置后，弹出窗口将提示您：
    - 输入您的Dodo Payments API密钥
    - 选择环境（测试或生产）
    - 在配置选项中选择您的MCP客户端
</Step>
<Step title="登录并批准">
    - 点击登录并批准按钮
</Step>
<Step title="验证成功连接">
    设置完成后，您将看到成功屏幕。
    <Check>
    您现在可以使用Dodo Payments MCP服务器！
    </Check>
</Step>
</Steps>
