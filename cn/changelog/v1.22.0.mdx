## 新功能 🚀

- **统一仪表板搜索**：我们在整个仪表板中实现了强大的搜索功能。您现在可以快速找到：
  - **标识符**：支付、退款、争议、客户、折扣、产品和附加组件 ID
  - **名称**：产品、附加组件和客户
  - **联系信息**：客户电子邮件地址
  - **元数据**：支付和订阅元数据
  - **导航**：Dodo Payments 仪表板中的所有页面

  <Frame>
    <img alt="Unified Dashboard Search" src="/images/changelog/search.png" />
  </Frame>

- **附加组件计划变更**：增加了对包含附加组件的订阅升级和降级的支持。

- **扩展的支付卡详情**：支付响应现在包括全面的卡信息：
  - `card_issuing_country`：发卡国家
  - `card_last_four`：卡号后四位
  - `card_brand`：卡品牌（Visa、Mastercard 等）
  - `card_type`：卡类型（信用卡、借记卡、预付卡）

- **平台增强**：对平台进行了各种性能优化、错误修复和小幅改进。
