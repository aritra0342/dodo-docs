## 新功能 🚀

### 分析图表共享

将您的分析图表作为图像共享，并提供可自定义的品牌选项。点击任何图表上的分享按钮以生成与您的品牌形象相匹配的多种颜色变体的图像。

  <img src="/images/changelog/share-chart.png" alt="分析图表共享界面，显示分享按钮和颜色变体选项" />
</Frame>

### 多个部分退款

通过 API 和 Dodo Payments Dashboard 处理单笔支付的多个部分退款，为处理客户退款请求提供更大的灵活性。

### 累计收入分析

通过分析仪表板中的新累计收入图表跟踪您的业务增长，提供对您收入趋势的全面洞察。

<Frame>
  <img src="/images/changelog/revenue-chart.png" alt="显示收入增长趋势的累计收入图表" />
</Frame>

### Next.js 适配器

轻松将 Dodo Payments 集成到您的 Next.js 应用程序中，使用我们的官方适配器。只需几行代码即可设置支付处理。您还可以使用我们现成的 LLM 提示与 AI 编码代理（如 Cursor、Windsurf、Lovable 等）自动化集成过程。

[查看 npm 上的 @dodopayments/nextjs 包](https://www.npmjs.com/package/@dodopayments/nextjs)

### Context7 集成

通过 Context7 MCP 访问 Dodo Payments 文档，使 AI 编码代理能够利用全面的支付处理上下文。有关详细信息，请访问 [Context7 文档](https://context7.com/llmstxt/dodopayments-llms-full.txt)。
