## 新功能 🚀

- **专属订阅页面**：通过新的专属页面轻松管理客户订阅。请在仪表板上导航至 `Sales > Subscriptions`。
  
    <Frame><img src="/images/changelog/subscriptions.png" alt="订阅页面"/></Frame>

- **增强的客户详情**：在一个地方查看每位客户的订阅、付款和退款。请在仪表板上转至 `Sales > Customers > arrow icon`。

    <Frame><img src="/images/changelog/customer_info.png" alt="客户详情"/></Frame>

- **详尽的付款信息**：深入了解付款金额、周期和日期。请访问仪表板上的 `Business > Payouts`。

    <Frame><img src="/images/changelog/payouts_update.png" alt="付款信息"/></Frame>

- **官方Ruby SDK**：轻松将Dodo Payments集成到您的Ruby应用程序中。有关更多详细信息，请查看 [Ruby SDK](https://github.com/dodopayments/dodopayments-ruby)。

- **增强的 API 筛选器**：在相关列表 API 中新增了 `customer_id`、`status` 等筛选器。有关更多信息，请参考 API 文档。

- **小改进和错误修复**：实施了各种其他小改进和错误修复。
