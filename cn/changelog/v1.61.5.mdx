## 新功能 🚀

1. **EUR & GBP 定价 - 全面推出**  
   对 € EUR 和 £ GBP 的本地支持现已全面推出，全球所有企业均可无缝进行这些主要货币的定价和支付收款。
   
   现在您可以：
   - 用 € 和 £ 定价产品
   - 本地收款这些货币
   - 直接以 € 和 £ 结算，无需外汇费用
   
   <Check>
   结果：在英国和欧洲的成功率更高，接受度更好。
   </Check>

2. **支付方式管理**  
   增强了对订阅和客户的支付方式管理能力。您现在可以更新现有订阅的支付方式，并检索与客户关联的所有支付方式，使处理过期卡、失败支付和客户偏好更改变得更加容易。
   
   **更新订阅的支付方式：**
   - 在不中断的情况下更新活跃订阅的支付方式
   - 通过更新支付方式自动重新激活 `on_hold` 订阅
   - 使用现有的保存支付方式或添加新的支付方式
   - 在重新激活订阅时自动创建剩余欠款的收费
   
   **获取客户支付方式：**
   - 列出客户的所有保存支付方式
   - 在更新订阅时使用支付方式 ID
   - 通过显示可用的支付选项改善客户体验
   
   了解更多：[更新支付方式 API](/api-reference/subscriptions/update-payment-method) | [列出支付方式 API](/api-reference/customers/get-customer-payment-methods)

3. **客户和退款对象中的元数据支持**  
   现在客户和退款对象中可用元数据，扩展了您存储自定义信息和与外部系统集成的能力。
   - 将元数据附加到客户以进行 CRM 集成和自定义分类
   - 存储退款的额外上下文，以便更好地跟踪和报告
   - 与其他支持元数据的 Dodo Payments 对象保持一致
   
   了解更多：[元数据指南](/api-reference/metadata)

4. **结账会话客户编辑功能标志**  
   结账会话中的新功能标志为您提供了对客户在结账过程中可以编辑内容的细粒度控制，提高了灵活性和用户体验。
   - `allow_customer_editing_name`: 允许客户编辑他们的姓名
   - `allow_customer_editing_email`: 允许修改电子邮件地址
   - `allow_customer_editing_street`: 允许编辑街道地址
   - `allow_customer_editing_city`: 允许编辑城市字段
   - `allow_customer_editing_state`: 允许编辑州/省
   - `allow_customer_editing_zipcode`: 允许编辑邮政编码
   - `allow_customer_editing_country`: 允许更改国家选择
   
   这些标志提供了对结账体验的细致控制，使您能够根据业务需求自定义客户可以修改的字段。

5. **结账许可证密钥和文件指示器**  
   结账页面现在在购买许可证密钥或数字文件时显示清晰的指示器，提高了透明度和客户信心。
   - 许可证密钥购买的视觉指示器
   - 清晰的文件下载指示器
   - 在结账过程中增强产品类型的可见性

6. **GitHub OAuth 登录页面**  
   通过 GitHub OAuth 集成增强了身份验证选项，为开发人员和技术用户提供了一种熟悉且安全的登录方式。
   - 使用 GitHub 账户登录
   - 为开发者账户简化身份验证
   - 为团队访问提供额外的安全层
