## 新特性 🚀

- **Overlay Checkout Beta**：我们很高兴地宣布我们新的 overlay checkout 体验的 beta 版本发布。这个现代的 TypeScript 库提供了一种无缝的方式，将我们的支付 overlay 集成到您的 web 应用中，支持实时事件处理和可定制主题。主要特性包括：
  - **实时事件处理**：监听支付创建、成功和失败等结账事件
  - **TypeScript 支持**：完全类型化的 API，提升开发者体验
  - **可定制主题**：支持明亮和黑暗主题
  - **浏览器兼容性**：在所有现代浏览器中均可使用，包括 Chrome、Firefox、Safari、Edge 和 IE11+
  - **PCI 合规性**：在提供嵌入式体验的同时保持安全标准

- **新的订阅事件**：新增对 `subscription.plan_changed` webhook 事件的支持，该事件在订阅升级、降级或修改时触发。此事件提供：
  - 更改后的完整订阅详情
  - 之前和新的计划信息
  - 按比例计费调整
  - 更新后的下次计费日期

  此事件补充了我们现有的订阅事件（`subscription.active`，`subscription.on_hold` 等），以提供对订阅生命周期的全面可见性。

- **平台增强**：对平台进行了各种性能优化、错误修复和小幅改进。
