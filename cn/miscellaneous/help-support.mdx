## **Dodo Payments 集中支持系统**

Dodo Payments 集中支持系统提供统一的支持体验，提供多种渠道来协助商户。

商户可以通过 Dodo Payments 仪表板上的 "获取支持" 图标访问支持系统，进入一个专门的支持着陆页，提供四个主要选项：

## 1. AI 驱动的聊天支持

AI 聊天功能提供即时的自动化支持，使用 AI 驱动的聊天机器人。它旨在通过以下方式协助用户：

- 回答技术和业务查询
- 提供 API 集成和设置的指导
- 为产品文档提供导航支持

此功能确保快速解决常见查询，减少对直接人工支持的需求。

📌 最适合：即时自助支持和一般故障排除。

## 2. 通过 Discord 的社区支持

Dodo Payments Discord 社区使商户能够与其他用户和 Dodo Payments 团队互动。通过此平台，用户可以：

- 讨论最佳实践并分享经验
- 寻求其他商户的点对点支持
- 在互动环境中与 Dodo Payments 支持团队互动

此功能促进了以社区为驱动的协作支持体验。

📌 最适合：同行支持、较长查询和与其他商户的网络交流。

## 3. 专属聊天支持

聊天支持功能为更复杂或特定于账户的查询提供了与 Dodo Payments 支持团队的直接联系。关键方面包括：

- 支持通过 Intercom 管理，商户可以提交详细查询。
- 响应时间为 24-48 个工作小时。
- 如果 72 小时内没有用户响应，未解决的工单将自动关闭。

此功能确保商户在需要人工干预的关键问题上获得个性化支持。

📌 最适合：与账户相关的问题、技术升级和个性化故障排除。

## **4. 自助文档**

Dodo Payments 文档门户作为商户寻找详细产品和集成信息的综合资源。门户包括：

- 详细的功能说明
- 分步实施指南
- API 文档和集成手册
- 常见问题解答 (FAQs)

此功能为商户提供结构化的、自主学习的机会，以独立导航平台。

📌 最适合：深入学习、实施指导和技术文档。

## **支持选项概述**

| **支持渠道** | **目的** | **响应时间** |
| --- | --- | --- |
| **AI 驱动的聊天** | 自动响应，文档指导 | 即时 |
| **Discord 社区** | 同行支持，社区讨论 | 社区驱动 |
| **聊天支持** | 针对复杂问题的个性化支持 | 24-48 小时 |
| **自助文档** | 产品指南，API 参考，教程 | 自主学习 |

商户可以根据查询的复杂性和所需的响应时间选择最合适的支持渠道。如需进一步帮助，请通过仪表板访问 Dodo Payments 支持中心。
