## 什么是 Dodo Payments？

Dodo Payments 是一个 **一体化引擎**，用于全球启动、扩展和货币化。专为新一代数字 SaaS 和 AI 产品设计，我们处理从 **支付到账单到分发** 的所有事务 - 所有这些都无需额外的工程。

<Info>
**为 SaaS 和 AI 产品打造**：使用我们优化的计费基础设施处理一次性支付、订阅、积分、席位、代币或任何使用指标。
</Info>

<Tip>
  <strong>认识 Sentra：您的 AI 驱动计费助理</strong><br/>
  Sentra 是一个 AI 代理，能在您的 IDE 中直接集成、分析并处理计费与支付。只需描述您的需求，就可以提问、构建集成或规划计费架构。

    开始使用 Sentra &rarr;
  </a>
</Tip>

### AI 助手的 MCP 服务器

将您的 AI 编码助手（Cursor、Claude Desktop、Windsurf、VS Code）直接连接到 Dodo Payments，使用我们的 MCP 服务器。

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  对 Dodo Payments 文档进行语义搜索——迅速获取有关功能、API 和最佳实践的即时答案
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  直接执行 API 操作——通过您的 AI 助手创建支付、管理订阅并处理退款
</Card>
</CardGroup>

{/* LOCKED_PATTERN_8ee343b87e057252cc4a8a376b190970 */}
将两个服务器添加到您的 MCP 配置中 (`~/.cursor/mcp.json` 或 Claude Desktop 配置)：

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
需要 Node.js 18+。请参阅 [MCP Server documentation](/developer-resources/mcp-server) 以获取适用于所有支持客户端的详细设置说明。
</Info>
</Accordion>
