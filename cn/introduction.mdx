## 什么是 Dodo Payments？

Dodo Payments 是一个 **一体化引擎**，用于全球启动、扩展和货币化。专为新一代数字 SaaS 和 AI 产品设计，我们处理从 **支付到账单到分发** 的所有事务 - 所有这些都无需额外的工程。

<Info>
**为 SaaS 和 AI 产品构建**：使用我们优化的计费基础设施处理一次性支付、订阅、积分、席位、代币或任何使用指标。
</Info>

<Tip>
  <strong>认识 Sentra：您的 AI 驱动账单助手</strong><br/>
  Sentra 是一个 AI 代理，可以直接在您的 IDE 中集成、分析和处理账单和支付。提出问题、构建集成或规划账单架构 - 只需描述您想要的内容。
  <a href="/developer-resources/sentra">
    开始使用 Sentra &rarr;
  </a>
</Tip>

## 快速入门指南

只需几个步骤即可开始使用 Dodo Payments：

<Steps>
<Step title="创建您的账户">
  使用您的电子邮件在 [app.dodopayments.com](https://app.dodopayments.com/signup) 注册。

  <Check>
  您将收到一封确认电子邮件以验证您的账户。
  </Check>
</Step>

<Step title="添加您的第一个产品">
  选择您的计费模型并添加您的产品以开始接受支付。

  <CardGroup cols={3}>
  <Card title="一次性付款" icon="box" href="/features/one-time-payment-products">
    非常适合数字下载、课程或单次购买
  </Card>
  
  <Card title="订阅产品" icon="repeat" href="/features/subscription">
    适合SaaS、会员或定期服务
  </Card>
  <Card title="基于使用的计费" icon="chart-bar" href="/features/usage-based-billing">
    根据API调用、积分、令牌或任何使用指标向客户收费 - 非常适合AI和SaaS
  </Card>
  </CardGroup>
</Step>

<Step title="选择您的集成方法">
  选择最适合您需求和技术要求的集成方法。

  <Tabs>
  <Tab title="无代码（最快）">

    - 从您的仪表板创建 **支付链接**

    - 与您的后端、工作流和数据模型深度集成

### AI 助手的 MCP 服务器

将您的 AI 编码助手（Cursor、Claude Desktop、Windsurf、VS Code）直接连接到 Dodo Payments，使用我们的 MCP 服务器。

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  在 Dodo Payments 文档中进行语义搜索 - 获取有关功能、API 和最佳实践的即时答案
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  直接执行 API 操作 - 通过您的 AI 助手创建支付、管理订阅和处理退款
</Card>
</CardGroup>

<Accordion title="Cursor/Claude Desktop 的快速 MCP 设置">
将两个服务器添加到您的 MCP 配置中 (`~/.cursor/mcp.json` 或 Claude Desktop 配置)：

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
需要 Node.js 18 及以上版本。请参阅 [MCP 服务器文档](/developer-resources/mcp-server) 获取所有支持客户端的详细设置说明。
</Info>
</Accordion>
