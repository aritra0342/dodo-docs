<Tip> 您将收到我们**当前支持付款**的国家/地区的[ISO Alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2)国家代码列表。您可以在[此处](/miscellaneous/list-of-countries-we-accept-payments-from)查看完整的支持国家列表。随着我们扩大服务范围，该列表会定期更新。</Tip>

<Tip> 注意：可用的选项包括**所有国家**的 ISO Alpha-2 代码，而不仅仅是我们当前支持的国家。请参考支持国家列表以确认可用性。</Tip>
