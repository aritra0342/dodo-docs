---
title: 支持的国家
openapi: get /checkout/supported_countries
description: 获取 Dodo Payments 支持的国家列表，以便您的客户使用。
---

<Tip> 您将收到一个包含 [ISO Alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) 国家代码的列表，这些代码代表我们 **当前支持的支付国家**。您可以在 [这里](/miscellaneous/list-of-countries-we-accept-payments-from) 查看完整的支持国家列表。随着我们覆盖范围的扩大，此列表会定期更新。 </Tip>

<Tip> 注意：可用选项包括 **所有国家** 的 ISO Alpha-2 代码，而不仅仅是我们当前支持的国家。请参考支持的国家列表以确认可用性。 </Tip>
