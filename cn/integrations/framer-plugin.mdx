## 概述

Dodo Payments Framer 插件无缝集成支付处理到您的 Framer 网站中。创建可自定义的结账体验，支持测试和生产环境。

## 设置指南

<Steps>
{/* LOCKED_PATTERN_386a8c9ba27e2f520f417976ae2d7c92 */}
访问 [Dodo Payments Dashboard](https://app.dodopayments.com/developer/api-keys)，并创建一个新的 API 密钥。

<Tabs>

{/* LOCKED_PATTERN_ab5fd62157a874c8d97a623e17ed6dd9 */}
用于具有真实支付处理的生产环境。
</Tab>

{/* LOCKED_PATTERN_0893a30648532a977fd32fa4cfe5aa5c */}
非常适合无真实收费的开发与测试。
</Tab>

</Tabs>

<Check>
安全地复制您的 API 密钥——下一步将需要它。
</Check>
</Step>

{/* LOCKED_PATTERN_918ab01af8a91f24ca04bebd360e16d7 */}
1. 打开 Framer 中的 Dodo Payments 插件
2. 转到认证页面
3. 粘贴您的 API 密钥
4. 选择适当的模式（测试或实时）
5. 点击 **Continue** 进行认证

<Warning>
确保您的 API 密钥安全，切勿公开共享。
</Warning>
</Step>

{/* LOCKED_PATTERN_1f6cf950f4953c2b98fed68f92e5abd3 */}
认证后，您将看到产品列表：

- **现有产品**：自动显示
- **没有产品**：点击“产品”旁边的 **加号 (+)** 按钮

<Info>
系统会将您重定向到 Dodo Payments 仪表板，在那里创建具有定价、描述和设置的产品。
</Info>
</Step>

{/* LOCKED_PATTERN_d0430ab947518d2b45d1080b6954c396 */}
1. 从可用列表中选择一个产品
2. 点击 `Add Buy Button` 按钮
3. 一个带有文本 `Buy {product_name}` 的购买按钮将被添加到画布上

<Warning>
如果您没有正确的项目权限，“添加购买按钮”将被禁用。
</Warning>
</Step>
</Steps>

## 自定义

<Steps>
{/* LOCKED_PATTERN_029f7fc64f4faf96b9708137acbff21d */}
    选择您偏好的结账体验：
    - **Overlay (Embedded)**：结账作为内嵌覆盖层出现在 Framer 项目中，让用户留在您的网站上。
    - **Hosted**：用户会被重定向到由 Dodo Payments 托管的结账页面，以获得更流畅的体验。
</Step>

{/* LOCKED_PATTERN_fe74d27da37271770f88b3905e82bc15 */}
    通过以下外观设置自定义结账体验：
    - **Checkout Theme**：在浅色或深色主题之间选择，以匹配您的品牌美感。
    - **Button Label**：自定义文本（例如 “Buy Now”、“Purchase”、“Get Started”）。
    - **Button Styles**：调整颜色、字体、尺寸及其他样式属性。
    - **Return URL**：为用户设置结账后返回的 Framer 页面。
</Step>
</Steps>

## 故障排除

- 验证您的 API 密钥是否正确

</Accordion>
