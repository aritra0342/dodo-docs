## 介绍

Dodo Payments WooCommerce 插件为您的商店提供无缝集成，使您能够通过多种方式接受支付，包括 UPI、卡片、网银和数字钱包。本综合指南将引导您完成安装和配置过程。

## 配置指南

### 设置 API 密钥和 Webhook

1. 转到 **WooCommerce → 设置 → 支付**，或点击 WooCommerce 菜单项下侧边栏的 **支付** 按钮
    
    <Frame>
        <img src="/images/woocommerce/6.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/7.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
2. 如果尚未启用 **Dodo Payments** 支付提供商，请启用，然后点击 **管理** 配置插件
3. 您将看到多个配置选项，每个都有说明性文本。首先设置您的 Live API Key
    
    <Frame>
        <img src="/images/woocommerce/8.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
4. 以 Live 模式登录您的 Dodo Payments 仪表板，然后转到 **Dodo Payments (Live Mode) > Developer > API Keys**，或访问 [此直达链接](https://app.dodopayments.com/developer/api-keys) 并点击 **添加 API Key**
    
    <Frame>
        <img src="/images/woocommerce/9.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
5. 为您的 API Key 起一个描述性名称并点击 **创建**
    
    <Frame>
        <img src="/images/woocommerce/10.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
6. 复制生成的 API Key 并粘贴到 WooCommerce 插件设置中的 **Live API Key** 字段
    
    <Frame>
        <img src="/images/woocommerce/11.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/12.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
7. 接下来设置 Webhook Signing Key，以启用 Dodo Payments 与 WooCommerce 之间的支付状态同步
8. 滚动到插件设置页面底部并复制显示的 webhook URL
    
    <Frame>
        <img src="/images/woocommerce/13.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
9. 返回您的 Dodo Payments 仪表板，转到 **Dodo Payments (Live Mode) > Developer > Webhooks** 并点击 **添加 Webhook**
    
    <Frame>
        <img src="/images/woocommerce/14.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
10. 将第 8 步中复制的 URL 粘贴到对话框中并点击 **添加 Webhook**
    
    <Frame>
        <img src="/images/woocommerce/15.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
11. 创建 webhook 后，点击红色遮蔽的 **Signing Key** 旁边的眼睛图标以显示并复制它
    
    <Frame>
        <img src="/images/woocommerce/16.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
12. 将 Signing Key 粘贴到插件设置中的 **Live Webhook Signing Key** 字段并保存更改
    
    <Frame>
        <img src="/images/woocommerce/17.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
13. 查看其余设置，如 **Global Tax Category** 和 **All Prices are Tax Inclusive**，这些选项决定了产品如何从 WooCommerce 同步到 Dodo Payments。请注意，**Test API Key** 和 **Test Webhook Signing Key** 仅在计划使用测试模式时才需要
    
    <Frame>
        <img src="/images/woocommerce/18.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>

## 主要特点

- **无缝结账体验**：将客户重定向到安全、优化的结账页面
- **多种支付方式**：支持各种本地和国际支付选项
- **实时状态更新**：通过 Webhook 实现即时订单状态同步
- **详细交易记录**：在您的 WooCommerce 仪表板中提供全面的支付历史
- **多货币支持**：接受多种货币的支付

## 故障排除

### 常见问题

1. **支付处理失败**
   - 验证 API 密钥是否正确配置
   - 确保 webhook URL 正确设置
   - 检查您的 SSL 证书是否有效

2. **Webhook 同步问题**
   - 确认 webhook 签名密钥是否匹配
   - 检查服务器日志以查找潜在错误
   - 验证您的服务器是否可以接收传入的 webhook 请求
