您的 Dodo Payments 仪表板，几次按键即可访问。使用 Raycast 快速查看支付、订阅、客户、产品、折扣代码、许可证密钥和退款，而无需打开仪表板。

## 安装

<Card title="Install from Raycast Store →" icon="puzzle-piece" href="https://www.raycast.com/dodopayments/dodo-payments">
  在 Raycast 商店中打开 Dodo Payments 扩展页面。
</Card>

<Info>
  需要在 macOS 上使用 Raycast 以及 Dodo Payments API 密钥。
</Info>

## 设置

<Steps>
<Step title="Get your API Key">
1. 登录你的 Dodo Payments 仪表板。
2. 转到「设置 › API 密钥」。
3. 复制你的 API 密钥。
</Step>

<Step title="Configure the Extension">
1. 打开 Raycast。
2. 搜索「Dodo Payments」并选择任一命令。
3. 在提示时，在 API 密钥字段中输入你的 API 密钥。
4. 选择 API 模式：
   - 测试：用于开发和测试。
   - 生产：用于生产数据。
5. 保存设置。
</Step>
</Steps>

<Tip>
  从测试模式开始以验证你的设置，然后再切换到生产模式。
</Tip>

<Warning>
  将你的 API 密钥视为密码。不要分享或以明文形式存储它。
</Warning>

## 使用扩展

- **搜索**：打开 Raycast，输入 "Dodo Payments"，并选择一个命令。
- **过滤**：通过输入 ID、电子邮件、名称或状态的部分内容来缩小结果。
- **打开详细信息**：在列表项上按 Enter 以查看更多信息。

## 故障排除

<AccordionGroup>
<Accordion title="Authentication failed or 401/403 errors">
  确认 API 密钥和模式（测试与生产）与你的仪表板数据一致。如有需要，重新打开命令并重新输入密钥。
</Accordion>
<Accordion title="No results appear">
  确保所选模式有数据。尝试在测试和生产之间切换。
</Accordion>
<Accordion title="Rate limits or timeouts">
  等待片刻后再试。如果问题仍然存在，请检查你的网络连接。
</Accordion>
</AccordionGroup>
