您的 Dodo Payments 仪表板，几次按键即可访问。使用 Raycast 快速查看支付、订阅、客户、产品、折扣代码、许可证密钥和退款，而无需打开仪表板。

## 安装

<Card title="从 Raycast Store 安装 →" icon="puzzle-piece" href="https://www.raycast.com/dodopayments/dodo-payments">
  打开 Raycast Store 中的 Dodo Payments 扩展页面。
</Card>

<Info>
需要在 macOS 上运行 Raycast 和 Dodo Payments API 密钥。
</Info>

## 设置

<Steps>
<Step title="获取您的 API 密钥">
1. 登录到您的 Dodo Payments 仪表板。
2. 转到设置 › API 密钥。
3. 复制您的 API 密钥。
</Step>

<Step title="配置扩展">
1. 打开 Raycast。
2. 搜索 "Dodo Payments" 并选择任何命令。
3. 当提示时，在 API 密钥字段中输入您的 API 密钥。
4. 选择 API 模式：
   - 测试：用于开发和测试。
   - 生产：用于生产数据。
5. 保存设置。
</Step>
</Steps>

<Tip>
在测试模式下开始以验证您的设置，然后切换到生产模式。
</Tip>

<Warning>
将您的 API 密钥视为密码。请勿分享或以明文形式存储它。
</Warning>

## 使用扩展

- **搜索**：打开 Raycast，输入 "Dodo Payments"，并选择一个命令。
- **过滤**：通过输入 ID、电子邮件、名称或状态的部分内容来缩小结果。
- **打开详细信息**：在列表项上按 Enter 以查看更多信息。

## 故障排除

<AccordionGroup>
<Accordion title="身份验证失败或 401/403 错误">
确认 API 密钥和模式（测试与生产）与您的仪表板数据匹配。如有需要，请重新打开命令并重新输入密钥。
</Accordion>
<Accordion title="没有结果出现">
确保您选择的模式有数据。尝试在测试和生产之间切换。
</Accordion>
<Accordion title="速率限制或超时">
稍等片刻再试。如果问题仍然存在，请检查您的网络连接。
</Accordion>
</AccordionGroup>
