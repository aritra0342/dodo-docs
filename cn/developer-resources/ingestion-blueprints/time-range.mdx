<CardGroup cols={2}>
<Card title="无服务器函数" icon="function">
根据函数执行时间和内存使用情况计费。
</Card>
<Card title="容器运行时" icon="container-storage">
跟踪容器运行时间以进行基于使用的计费。
</Card>
<Card title="计算实例" icon="server">
监控虚拟机运行时间，并按分钟或小时收费。

非常适合基于计算时间、函数执行持续时间、容器运行时或任何基于时间的使用进行计费。
</Info>

## 快速开始

按时间持续时间跟踪资源使用：

<Steps>
<Step title="安装 SDK">

```bash
npm install @dodopayments/ingestion-blueprints
```

</Step>

<Step title="获取您的 API 密钥">
- **Dodo Payments API 密钥**：从 [Dodo Payments Dashboard](https://app.dodopayments.com/developer/api-keys) 获取
</Step>

<Step title="创建计量器">
在您的 [Dodo Payments Dashboard](https://app.dodopayments.com/) 中创建计量器：
- **事件名称**：`time_range_usage`（或您喜欢的名称）
- **聚合类型**：`sum` 以跟踪总持续时间
- **超属性**：`durationSeconds`，`durationMinutes`，或 `durationMs`
</Step>

<Step title="跟踪时间使用">
<CodeGroup>

```javascript Serverless Functions
import { Ingestion, trackTimeRange } from '@dodopayments/ingestion-blueprints';

const ingestion = new Ingestion({
  apiKey: process.env.DODO_PAYMENTS_API_KEY,
  environment: 'test_mode',
  eventName: 'function_execution'
});

// Track function execution time
const startTime = Date.now();

// Execute your function (example: image processing)
const result = await yourImageProcessingLogic();

const durationMs = Date.now() - startTime;

await trackTimeRange(ingestion, {
  customerId: 'customer_123',
  durationMs: durationMs
});
```

```javascript Container Runtime
import { Ingestion, trackTimeRange } from '@dodopayments/ingestion-blueprints';

const ingestion = new Ingestion({
  apiKey: process.env.DODO_PAYMENTS_API_KEY,
  environment: 'test_mode',
  eventName: 'container_runtime'
});

// Track container runtime in seconds
await trackTimeRange(ingestion, {
  customerId: 'customer_456',
  durationSeconds: 120
});
```

```javascript VM Instance Runtime
import { Ingestion, trackTimeRange } from '@dodopayments/ingestion-blueprints';

const ingestion = new Ingestion({
  apiKey: process.env.DODO_PAYMENTS_API_KEY,
  environment: 'test_mode',
  eventName: 'vm_runtime'
});

// Track VM runtime in minutes
await trackTimeRange(ingestion, {
  customerId: 'customer_789',
  durationMinutes: 60
});
```

</CodeGroup>
</Step>
</Steps>

## 配置

### 数据摄取配置

<ParamField path="apiKey" type="string" required>
您从仪表板获取的 Dodo Payments API 密钥。
</ParamField>

<ParamField path="environment" type="string" required>
环境模式：`test_mode` 或 `live_mode`。
</ParamField>

<ParamField path="eventName" type="string" required>
与您的计量器配置匹配的事件名称。
</ParamField>

### 跟踪时间范围选项

<ParamField path="customerId" type="string" required>
用于计费归属的客户 ID。
</ParamField>

<ParamField path="durationMs" type="number">
持续时间（毫秒）。用于亚秒精度。
</ParamField>

<ParamField path="durationSeconds" type="number">
持续时间（秒）。最常用于函数执行和短任务。
</ParamField>

<ParamField path="durationMinutes" type="number">
持续时间（分钟）。适用于运行时间较长的资源，如虚拟机。
</ParamField>

<ParamField path="metadata" type="object">
关于资源的可选元数据，如 CPU、内存、区域等。
</ParamField>

## 最佳实践

<Tip>
**选择正确的单位**：对于短操作使用毫秒，对于函数使用秒，对于运行时间较长的资源使用分钟。
</Tip>

<Warning>
**准确计时**：使用 `Date.now()` 或 `performance.now()` 进行准确的时间跟踪，特别是对于无服务器函数。
</Warning>
