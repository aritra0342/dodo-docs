## 环境 URL

使用适合您环境的基础 URL：

- 测试模式: [`https://test.dodopayments.com`](https://test.dodopayments.com)
- 生产模式: [`https://live.dodopayments.com`](https://live.dodopayments.com)

了解更多关于 [测试模式与生产模式](/miscellaneous/test-mode-vs-live-mode)。

## 集成步骤

按照以下步骤集成 Dodo Payments：

<Steps>
  1. [设置您的账户](/quickstart) 并获取您的 API 密钥
  2. 选择您的集成方法：
     - [按照我们的教程](/developer-resources/integration-tutorial)
     - [使用我们的 SDK](/developer-resources/dodo-payments-sdks)
     - [直接 API 集成](/developer-resources/integration-guide)
  3. [设置 Webhook](/developer-resources/webhooks/intents/webhook-events-guide) 以获取实时更新
  4. [测试您的集成](/miscellaneous/testing-process)
  5. 上线！
</Steps>
