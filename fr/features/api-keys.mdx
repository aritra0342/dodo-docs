## Introduction

La génération de Clé API permet aux commerçants d'authentifier et d'intégrer en toute sécurité leurs systèmes avec la plateforme Dodo Payments pour un accès fluide aux fonctionnalités de paiement, de produit et d'abonnement.

**Remarque :** Les Clés API créées en mode Test seront limitées à l'environnement de mode test et vice versa pour le mode en direct
