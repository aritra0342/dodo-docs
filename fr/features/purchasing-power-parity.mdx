## Introduction

[ParityDeals](https://www.paritydeals.com/) est une plateforme qui permet l'optimisation automatique des prix en fonction de la localisation géographique d'un client, communément appelée tarification par "parité du pouvoir d'achat".

Avec ParityDeals, vous pouvez également offrir des [réductions automatiques basées sur le temps](https://www.paritydeals.com/features/time-based-discounts/) et des [promotions basées sur les vacances](https://www.paritydeals.com/features/holiday-based-discounts/) tout au long de l'année.

<Info>
Bien que vous ne puissiez pas intégrer directement la bannière ParityDeals sur une vitrine Dodo Payments (les scripts personnalisés n'étant pas pris en charge), vous pouvez tout de même mettre en œuvre ParityDeals efficacement sur votre propre site ou application.
</Info>

## Comment ça fonctionne

Lorsque ParityDeals est intégré à votre site web, les visiteurs éligibles verront une bannière contenant un code de réduction basé sur la localisation qu'ils peuvent utiliser lors du paiement.

<Frame>
  <img src="/images/ppp/1.png" alt="Exemple de bannière de réduction ParityDeals" />
</Frame>

Ces codes de réduction sont automatiquement créés et gérés par ParityDeals et apparaîtront dans votre **Tableau de bord Dodo Payments** sous **Ventes → Réductions**.

## Guide d'installation étape par étape

### 1. Inscrivez-vous sur ParityDeals

Visitez [app.paritydeals.com](https://app.paritydeals.com/) et créez un compte.

### 2. Créez des offres

Après vous être connecté, localisez l'option pour créer des offres. Dans le menu déroulant, sélectionnez :

**→ "Créer des offres via Dodo Payments"**

<Frame>
  <img src="/images/ppp/2.png" alt="Créer des offres via Dodo Payments" />
</Frame>

### 3. Générez une clé API Dodo Payments

Connectez-vous à votre **tableau de bord Dodo Payments** et naviguez vers :

**Développeurs → Clés API**

Cliquez sur **"Créer une nouvelle API"** et nommez-la de manière descriptive, comme "ParityDeals".

<Frame>
  <img src="/images/ppp/3.png" alt="Créer une nouvelle clé API dans Dodo Payments" />
</Frame>

### 4. Connectez votre clé API à ParityDeals

Copiez la clé API que vous avez créée et collez-la dans l'interface ParityDeals pour établir la connexion avec votre compte Dodo Payments.

<Frame>
  <img src="/images/ppp/4.png" alt="Connecter la clé API à ParityDeals" />
</Frame>

### 5. Sélectionnez des produits

Une fois connecté, ParityDeals récupérera votre catalogue de produits Dodo Payments. Sélectionnez les produits que vous souhaitez associer à vos offres.

<Frame>
  <img src="/images/ppp/5.png" alt="Sélectionner des produits pour ParityDeals" />
</Frame>

### 6. Configurez les paramètres de l'offre

Entrez l'URL de votre site web et sélectionnez éventuellement des pages cibles spécifiques où la bannière de réduction doit apparaître.

Ensuite, choisissez votre type d'offre préféré :

- Offres basées sur la localisation
- Offres de vacances
- Offres flash basées sur le temps

<Frame>
  <img src="/images/ppp/6.png" alt="Configurer les paramètres de l'offre" />
</Frame>

### 7. Définissez des règles basées sur les pays

Sélectionnez les pays auxquels vous souhaitez offrir des réductions localisées et définissez le pourcentage de réduction pour chacun. ParityDeals fournit des valeurs par défaut recommandées que vous pouvez ajuster si nécessaire.

<Frame>
  <img src="/images/ppp/7.png" alt="Définir des règles de remise par pays" />
</Frame>

### 8. Personnalisez la bannière

Concevez votre bannière de réduction en :

- Personnalisant le texte
- Sélectionnant les couleurs et la mise en page
- Choisissant la position de la bannière (en haut ou en bas de la page)

Cette bannière sera affichée aux visiteurs éligibles en fonction de vos règles configurées.

<Frame>
  <img src="/images/ppp/8.png" alt="Personnaliser la bannière de remise" />
</Frame>

### 9. Ajoutez le script ParityDeals à votre site web

Après avoir enregistré votre offre, ParityDeals générera un script. Ajoutez ce script au code de votre site (généralement dans la section `<head>`).

Ce script détecte automatiquement les emplacements des visiteurs et affiche la bannière de réduction appropriée.

<Frame>
  <img src="/images/ppp/9.png" alt="Script d'intégration ParityDeals" />
</Frame>

## Tout est prêt !

Vous êtes maintenant prêt à offrir :

- **Offres basées sur la localisation** - Ajustez les prix en fonction du pouvoir d'achat
- **Réductions à durée limitée/sensibles au temps** - Créez un sentiment d'urgence avec des offres à durée limitée
- **Promotions de vacances** - Profitez des tendances d'achat saisonnières

Tout cela est automatisé via ParityDeals et fonctionne sans problème avec Dodo Payments.

Pour plus d'informations détaillées, consultez la [Documentation ParityDeals](https://www.paritydeals.com/docs/) ou contactez leur équipe de support.
