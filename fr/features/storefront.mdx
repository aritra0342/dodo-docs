## Points clés de la fonctionnalité

1. **Storefront de marque**
    - Personnalisez votre vitrine avec un nom de boutique, un logo et une image de couverture.
    - Partagez facilement l’URL de votre boutique (par exemple, `store.dodopayments.com/your-store`).
2. **Segmentation des produits**
    - Organise automatiquement vos produits en sections Paiement unique et Abonnement.
3. **Flux de paiement simple**
    - Les clients peuvent sélectionner un produit et une quantité, puis cliquer sur **« Acheter maintenant »** pour être redirigés vers le paiement Dodopayments.
4. **Publier/Dépublier**
    - Contrôlez les produits affichés sur votre boutique : publiez ou dépubliez rapidement depuis le tableau de bord marchand.
5. **Aucune configuration supplémentaire**
    - Aucun CMS séparé n’est nécessaire ; toute la gestion des produits se fait dans votre tableau de bord marchand, où vous gérez déjà les prix, descriptions et détails d’abonnement.
6. **Compatible mobile**
    - La mise en page de la vitrine est responsive, garantissant une expérience d’achat fluide sur téléphones, tablettes et ordinateurs de bureau.

## Configuration de la vitrine

Voici le processus étape par étape pour les marchands afin de configurer et de gérer leur vitrine :

<img src="/images/store-front/sf-1.png"/>

### Accéder aux paramètres de votre vitrine

1. Ouvrez la section Entreprise → Sélectionnez Vitrine

### Branding de la boutique

1. **Nom de la boutique** : Saisissez le nom public de votre boutique. Il apparaît en évidence en haut de votre vitrine.
2. **Logo** : Téléversez une image (par exemple PNG ou JPG) pour votre logo.
3. **Image de couverture** : Sélectionnez ou téléversez une bannière/image de couverture qui apparaîtra sur la page d’accueil de la vitrine.
4. **URL de la boutique** :
    - Générée automatiquement en fonction du nom de votre entreprise (par exemple, `business.dodopayments.com/my-store`).
    - Vous pouvez modifier cette URL si le système prend en charge les slugs personnalisés.
5. **Produits** : Vous pouvez sélectionner les produits que vous souhaitez afficher sur votre vitrine

### Enregistrer et voir votre boutique

<img src="/images/store-front/sf-2.png"/>

1. **Cliquez sur Publier** : Cliquez sur Publier pour confirmer la configuration de votre vitrine et vos décisions de publication de produits.
2. **Aperçu de la boutique** : Vous pouvez prévisualiser votre vitrine à l'URL générée.
3. **Partager l'URL** : Une fois satisfait, partagez le lien avec vos clients par e-mail, sur les réseaux sociaux ou sur votre site Web.

### Mode test

<img src="/images/store-front/sf-3.png"/>

Le mode test et le mode en direct ont des vitrines séparées. Vous pouvez voir l'indicateur de mode test en haut de votre site Web et de la vitrine générée par le mode test également.

Le mode test est pour vous permettre d'essayer le flux utilisateur et l'expérience d'achat sur la boutique. Veuillez partager le lien de la vitrine en mode en direct avec vos utilisateurs pour faciliter les achats réels.

## Expérience client

<img src="/images/store-front/sf-4.png"/>

### Arrivée sur la vitrine

- **Branding visuel** : Les clients voient immédiatement le nom de votre boutique, votre logo et votre image de couverture.
- **Produits organisés** : Deux catégories sont affichées : Paiement unique et Abonnement.

### Navigation dans les produits

- **Cartes de produits** : Chaque produit affiche un nom, une description et un prix. La description peut être consultée en cliquant sur l'icône i en bas à droite de la carte.
- **Sélection de la quantité** : Les clients peuvent choisir combien d'articles ils souhaitent acheter.
- **Fréquence d'abonnement et essai** : Les produits d'abonnement indiquent explicitement la fréquence de facturation (par exemple, mensuelle) et si un essai est disponible (par exemple, essai gratuit de 7 jours).

<img src="/images/store-front/sf-5.png"/>

### Processus d'achat

1. **Sélectionnez “Acheter maintenant”** : Une fois prêt, le client clique sur “Acheter” pour le produit souhaité, sélectionne la quantité et clique sur “Acheter maintenant”.
2. **Redirection vers le paiement Dodopayments** :
    - Le client sera dirigé vers le paiement Dodopayments.
    - Le client termine le processus de paiement sécurisé sur Dodopayments.
3. **Confirmation** :
    - Après un paiement réussi, Dodopayments affiche une confirmation.
    - En option, le client peut être redirigé vers une page “Merci” sur votre vitrine en fonction de l'URL de redirection saisie.
