## Processus de liaison d'un compte bancaire

1. Allez dans la section **Paramètres de paiement** du tableau de bord Dodo Payments et sélectionnez l'onglet **Informations bancaires**.
2. Sélectionnez **l'emplacement de votre compte bancaire** (Inde, États-Unis, Europe, Reste du monde).
3. Remplissez le formulaire avec des détails précis comme indiqué ci-dessous.
4. Téléchargez un **relevé bancaire** qui répond aux exigences de visibilité pour la vérification.

### **Directives supplémentaires**

- **Visibilité du relevé bancaire** : Tous les relevés bancaires téléchargés doivent avoir des détails de compte clairs et lisibles. Évitez de rogner ou d'obscurcir des informations, car cela pourrait retarder le processus de vérification.
- **Exactitude du nom du compte** : Le nom du titulaire du compte doit correspondre exactement au nom figurant sur les documents bancaires soumis pour éviter des divergences lors du processus de vérification.
- **SWIFT/BIC et numéros de routage** : Assurez-vous que les numéros SWIFT/BIC et de routage sont exacts pour les transferts internationaux et nationaux afin de faciliter les paiements sans problème.
- **Codes de devise** : Utilisez toujours le bon code de devise à 3 lettres (par ex., INR pour l'Inde, USD pour les États-Unis, EUR pour l'Europe) selon la devise locale du compte bancaire.
Vous pouvez utiliser le lien suivant pour identifier votre code de devise si vous avez des doutes : [https://www.iban.com/currency-codes](https://www.iban.com/currency-codes)
