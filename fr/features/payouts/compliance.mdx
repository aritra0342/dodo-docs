## Pour l'Inde

### Entreprises enregistrées

Les entreprises enregistrées en Inde sont classées en différents types, chacune nécessitant des documents spécifiques pour la vérification.

#### Types d'entreprises enregistrées prises en charge :

* Société à responsabilité limitée

* Partenariat à responsabilité limitée (LLP)

* Partenariat

* Entreprise individuelle

#### Documents requis pour les entreprises enregistrées :

| Type d'entreprise         | Nom du document            | Description                                                      |
| ------------------------- | -------------------------- | ---------------------------------------------------------------- |
| **Société à responsabilité limitée** | Numéro PAN de l'entreprise | Numéro PAN de l'entreprise enregistrée.                           |
|                           | GSTIN                      | Numéro d'identification GST à des fins fiscales.                  |
|                           | CIN                        | Numéro d'identification de l'entreprise                            |
|                           | Nom DBA                   | Nom "Doing Business As" tel qu'il apparaît sur les relevés bancaires. |
| **LLP**                   | Numéro PAN de l'entreprise | Numéro PAN de la LLP enregistrée.                                |
|                           | GSTIN                      | Numéro d'identification GST à des fins fiscales.                  |
|                           | Document d'incorporation de la LLP | Document certifiant l'enregistrement de la LLP.                  |
|                           | Nom DBA                   | Nom "Doing Business As" tel qu'il apparaît sur les relevés bancaires. |
| **Partenariat**           | Numéro PAN du partenariat   | Numéro PAN de la société de partenariat.                          |
|                           | Acte de partenariat        | Acte officiel décrivant les termes du partenariat et les rôles des parties prenantes. |
|                           | Nom DBA                   | Nom "Doing Business As" tel qu'il apparaît sur les relevés bancaires. |
| **Entreprise individuelle** | Nom DBA                   | Nom "Doing Business As" tel qu'il apparaît sur les relevés bancaires. |

***

### Entreprises non enregistrées

Pour les entreprises non enregistrées en Inde, une documentation différente est requise car elles ne sont pas des entités légalement constituées.

#### Documents requis pour les entreprises non enregistrées :

| Nom du document | Description                                                                    |
| --------------- | ------------------------------------------------------------------------------ |
| Numéro PAN      | Numéro PAN personnel du propriétaire de l'entreprise                           |
| Nom DBA        | Nom "Doing Business As" tel qu'il apparaît sur les relevés bancaires.         |
| Adresse         | Adresse complète incluant **Rue, Ville, État et Code Postal** selon preuve.   |
| Nationalité     | Pays de citoyenneté de l'individu ou de l'entité propriétaire.                |

***

## Informations sur les parties prenantes clés - Inde

Dodo Payments nécessite des informations détaillées pour chaque partie prenante clé associée à l'entreprise. Chaque partie prenante doit fournir sa **carte PAN** dans le cadre des exigences de conformité. Nous acceptons la documentation pour jusqu'à 4 parties prenantes clés. Si vous avez plus de 4 parties prenantes clés, veuillez nous envoyer la documentation par e-mail.

### Qui est une partie prenante clé ?

Les personnes suivantes peuvent être des parties prenantes clés dans l'entreprise :

#### Propriétaire(s) bénéficiaire(s)

Le propriétaire bénéficiaire est toute personne physique qui, agissant seule ou ensemble, ou par l'intermédiaire d'autres entreprises, LLP ou personnes juridiques, possède ou a droit à 25 % ou plus des actions ou du capital ou des bénéfices de l'utilisateur connecté, ou qui exerce un contrôle sur l'utilisateur connecté par d'autres moyens, y compris le droit de nommer la majorité de ses administrateurs ou de contrôler sa gestion ou ses décisions politiques par le biais de droits d'actionnariat ou de gestion ou par le biais d'accords d'actionnaires ou d'accords de vote.

Si l'utilisateur connecté a plus d'un propriétaire bénéficiaire, veuillez nous fournir les documents requis pour tous.

#### Directeur(s) d'entreprise

Individus nommés et actuellement en fonction en tant que directeurs au conseil d'administration de l'utilisateur connecté conformément aux lois indiennes sur les sociétés.

Veuillez nous fournir les PAN de tous les directeurs.

#### Représentant d'entreprise

Employé ou directeur de l'utilisateur connecté autorisé à agir en son nom en relation avec l'utilisation des services de Dodo Payments.

#### Co-fondateurs

Si l'entreprise n'est pas enregistrée, veuillez fournir la documentation de tous les co-fondateurs associés à l'entreprise qui détiendront une majorité des parts (plus de 25 %) ou deviendront directeurs dans l'entreprise après l'enregistrement. Cela peut être mis à jour après l'enregistrement si des modifications sont apportées.

Les entreprises enregistrées en dehors de l'Inde doivent soumettre des documents en fonction de leur type d'entreprise à des fins de vérification.

| Nom du document                   | Description                                                 |

## Formulaires fiscaux

### Pour les citoyens américains

Les citoyens américains, quel que soit leur type de commerçant, doivent soumettre le **Formulaire W-9**. Ce formulaire est utilisé pour signaler les informations d'identification fiscale à l'IRS.

#### Formulaire fiscal requis pour les citoyens américains :

| Statut de citoyenneté | Type de commerçant         | Formulaire requis | Description                                     |
| --------------------- | ------------------------- | ---------------- | ----------------------------------------------- |
| **Citoyen américain**  | Individuel & Entreprise   | W-9              | Formulaire IRS pour signaler l'identification fiscale. |

* **Instructions** : Les citoyens américains doivent télécharger, remplir et télécharger le formulaire W-9 signé avec tous les détails clairement visibles.

* **Lien de téléchargement** : [https://www.irs.gov/pub/irs-pdf/fw9.pdf](https://www.irs.gov/pub/irs-pdf/fw9.pdf)

***

### Pour les non-citoyens américains

Les non-citoyens américains doivent soumettre différents formulaires en fonction de leur type de commerçant, spécifiquement s'ils sont un individu ou une entité commerciale enregistrée.

#### Non-citoyens américains - Individuel

Les commerçants individuels non américains doivent soumettre le **Formulaire W-8BEN**. Ce formulaire est utilisé pour signaler des informations à l'IRS et revendiquer des avantages en vertu d'un traité fiscal applicable.

##### Formulaire fiscal requis pour les individus non américains :

| Statut de citoyenneté   | Type de commerçant | Formulaire requis | Description                                                     |
| ----------------------- | ------------------ | ---------------- | --------------------------------------------------------------- |
| **Non-citoyen américain** | Individuel         | W-8BEN           | Formulaire IRS pour les individus non américains pour revendiquer des avantages fiscaux en vertu d'un traité. |

* **Instructions** : Les individus non américains doivent télécharger, remplir et télécharger le formulaire W-8BEN signé avec tous les détails clairement visibles.

* **Lien de téléchargement** : [https://www.irs.gov/pub/irs-pdf/fw8ben.pdf](https://www.irs.gov/pub/irs-pdf/fw8ben.pdf)

#### Non-citoyens américains - Entités commerciales

Les entités commerciales non américaines doivent soumettre le **Formulaire W-8BEN-E**. Ce formulaire est utilisé par les entités étrangères pour revendiquer un statut étranger et, le cas échéant, un taux de retenue réduit en vertu d'un traité fiscal.

##### Formulaire fiscal requis pour les entités commerciales non américaines :

| Statut de citoyenneté   | Type de commerçant   | Formulaire requis | Description                                                                  |
| ----------------------- | ------------------- | ---------------- | ---------------------------------------------------------------------------- |
| **Non-citoyen américain** | Entité commerciale   | W-8BEN-E        | Formulaire IRS pour les entités non américaines pour revendiquer des avantages fiscaux en vertu d'un traité (le cas échéant). |

* **Instructions** : Les entités commerciales non américaines doivent télécharger, remplir et télécharger le formulaire W-8BEN-E signé avec tous les détails clairement visibles.

* **Lien de téléchargement** : [https://www.irs.gov/pub/irs-pdf/fw8bene.pdf](https://www.irs.gov/pub/irs-pdf/fw8bene.pdf)
