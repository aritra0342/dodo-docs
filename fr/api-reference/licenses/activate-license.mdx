---
title: Activer la licence
description: Cet endpoint vous permet d'activer une licence pour l'utilisateur.
openapi: post /licenses/activate
---

<Info>
**Aucune clé d’API requise** : Il s’agit d’un point de terminaison public qui ne nécessite pas d’authentification. Vous pouvez l’appeler directement depuis des applications clientes, des logiciels de bureau ou des CLIs pour activer des clés de licence sans exposer vos identifiants d’API.
</Info>
