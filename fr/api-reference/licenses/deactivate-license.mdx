---
title: Désactiver la licence
description: Cet endpoint vous permet de désactiver une licence pour l'utilisateur.
openapi: post /licenses/deactivate
---

<Info>
**Aucune clé API requise** : Il s'agit d'un point de terminaison public qui ne nécessite aucune authentification. Vous pouvez l'appeler directement depuis des applications clientes, des logiciels bureautiques ou des interfaces en ligne de commande pour désactiver des clés de licence sans exposer vos identifiants API.
</Info>
