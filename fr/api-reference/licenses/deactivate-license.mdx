---
title: Désactiver la licence
description: Cet endpoint vous permet de désactiver une licence pour l'utilisateur.
openapi: post /licenses/deactivate
---

<Info>
**Aucune clé API requise** : Il s'agit d'un endpoint public qui ne nécessite pas d'authentification. Vous pouvez l'appeler directement depuis des applications clientes, des logiciels de bureau ou des CLIs pour désactiver des clés de licence sans exposer vos identifiants API.
</Info>
