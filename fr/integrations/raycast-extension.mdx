Votre tableau de bord Dodo Payments, à quelques frappes de clavier. Utilisez Raycast pour visualiser rapidement les paiements, abonnements, clients, produits, codes de réduction, clés de licence et remboursements sans ouvrir le tableau de bord.

## Installer

<Card title="Installer depuis le Raycast Store →" icon="puzzle-piece" href="https://www.raycast.com/dodopayments/dodo-payments">
  Ouvrez la page de l'extension Dodo Payments dans le Raycast Store.
</Card>

<Info>
Nécessite Raycast sur macOS et une clé API Dodo Payments.
</Info>

## Configuration

<Steps>
<Step title="Obtenez votre clé API">
1. Connectez-vous à votre tableau de bord Dodo Payments.
2. Allez dans Paramètres › Clés API.
3. Copiez votre clé API.
</Step>

<Step title="Configurer l'extension">
1. Ouvrez Raycast.
2. Recherchez "Dodo Payments" et sélectionnez une commande.
3. Lorsque vous y êtes invité, entrez votre clé API dans le champ Clé API.
4. Choisissez le mode API :
   - Test : à utiliser pour le développement et les tests.
   - Live : à utiliser pour les données de production.
5. Enregistrez les paramètres.
</Step>
</Steps>

<Tip>
Commencez en mode Test pour valider votre configuration, puis passez en mode Live.
</Tip>

<Warning>
Traitez votre clé API comme un mot de passe. Ne la partagez pas et ne la stockez pas en texte clair.
</Warning>

## Utiliser l'extension

- **Recherche** : ouvrez Raycast, tapez "Dodo Payments" et choisissez une commande.
- **Filtrer** : affinez les résultats en tapant des parties d'IDs, d'emails, de noms ou de statuts.
- **Ouvrir les détails** : appuyez sur Entrée sur un élément de la liste pour voir plus d'informations.

## Dépannage

<AccordionGroup>
<Accordion title="Échec d'authentification ou erreurs 401/403">
Confirmez que la clé API et le mode (Test vs Live) correspondent aux données de votre tableau de bord. Rouvrez la commande et réentrez la clé si nécessaire.
</Accordion>
<Accordion title="Aucun résultat n'apparaît">
Assurez-vous que le mode sélectionné a des données. Essayez de basculer entre Test et Live.
</Accordion>
<Accordion title="Limites de taux ou délais d'attente">
Attendez un moment et réessayez. Si le problème persiste, vérifiez votre connectivité réseau.
</Accordion>
</AccordionGroup>
