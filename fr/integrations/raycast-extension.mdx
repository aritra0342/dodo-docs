Votre tableau de bord Dodo Payments, à quelques frappes de clavier. Utilisez Raycast pour visualiser rapidement les paiements, abonnements, clients, produits, codes de réduction, clés de licence et remboursements sans ouvrir le tableau de bord.

## Installer

{/* LOCKED_PATTERN_689eae9422ae1bb6dca6aba8d59d9dd9 */}
  Ouvrez la page de l’extension Dodo Payments dans le Raycast Store.
</Card>

<Info>
  Nécessite Raycast sur macOS et une clé API Dodo Payments.
</Info>

## Configuration

<Steps>
{/* LOCKED_PATTERN_92afe5a5f9799e8778da662cdecfb3a0 */}
1. Connectez-vous à votre tableau de bord Dodo Payments.
2. Allez dans Paramètres › Clés API.
3. Copiez votre clé API.
</Step>

{/* LOCKED_PATTERN_62acba3447424cd114a190a47207d390 */}
1. Ouvrez Raycast.
2. Recherchez « Dodo Payments » et sélectionnez une commande.
3. Lorsque cela est demandé, saisissez votre clé API dans le champ API Key.
4. Choisissez le mode de l’API :
   - Test : utilisez-le pour le développement et les tests.
   - Live : utilisez-le pour les données de production.
5. Enregistrez les réglages.
</Step>
</Steps>

<Tip>
Commencez en mode Test pour valider votre configuration, puis passez en Live.
</Tip>

<Warning>
Traitez votre clé API comme un mot de passe. Ne la partagez pas et ne la stockez pas en clair.
</Warning>

## Utiliser l'extension

- **Recherche** : ouvrez Raycast, tapez "Dodo Payments" et choisissez une commande.
- **Filtrer** : affinez les résultats en tapant des parties d'IDs, d'emails, de noms ou de statuts.
- **Ouvrir les détails** : appuyez sur Entrée sur un élément de la liste pour voir plus d'informations.
