## Introduction

Le plugin WooCommerce Dodo Payments offre une intégration transparente avec votre boutique, vous permettant d'accepter des paiements par plusieurs méthodes, y compris UPI, cartes, banque en ligne et portefeuilles numériques. Ce guide complet vous guidera à travers le processus d'installation et de configuration.

## Guide d'Installation Étape par Étape

1. Téléchargez le dernier zip du plugin de paiement WooCommerce.

    <Card title="Télécharger le Plugin" icon="download" href="https://github.com/dodopayments/dodo-woocommerce/releases">
    Obtenez la dernière version de notre plugin WooCommerce
    </Card>

    Alternativement, vous pouvez installer directement depuis le répertoire des plugins WordPress :
    
    <Card title="Répertoire des Plugins WordPress" icon="wordpress" href="https://wordpress.org/plugins/dodo-payments-for-woocommerce/">
    Installez Dodo Payments pour WooCommerce depuis WordPress.org
    </Card>

2. Connectez-vous à votre site WordPress en tant qu'administrateur (nous supposons que vous avez déjà configuré WooCommerce)
3. Accédez à **Plugins → Ajouter un Nouveau Plugin**
    
    <Frame>
        <img src="/images/woocommerce/1.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
4. Cliquez sur le bouton **Télécharger le Plugin** et une boîte de dialogue de sélection de fichier apparaîtra

    <Frame>
        <img src="/images/woocommerce/2.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/3.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
5. Cliquez sur **Parcourir…** et sélectionnez le fichier zip que vous avez téléchargé
6. Cliquez sur le bouton **Installer Maintenant** pour commencer le processus d'installation
    
    <Frame>
        <img src="/images/woocommerce/4.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
7. Une fois l'installation terminée, cliquez sur **Activer le Plugin** ou activez-le depuis la section **Plugins Installés**
    
    <Frame>
        <img src="/images/woocommerce/5.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
8. Le plugin est maintenant installé, mais une configuration est encore nécessaire. Continuez avec le guide de configuration ci-dessous.

## Guide de Configuration

### Configuration des Clés API et Webhooks

1. Accédez à **WooCommerce → Paramètres → Paiements** ou cliquez sur le bouton **Paiements** dans la barre latérale gauche sous l'élément de menu WooCommerce
    
    <Frame>
        <img src="/images/woocommerce/6.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/7.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
2. Activez le fournisseur de paiement **Dodo Payments** s'il n'est pas déjà activé, puis cliquez sur **Gérer** pour configurer le plugin
3. Vous verrez diverses options de configuration, chacune avec un texte explicatif utile. Commencez par configurer votre Clé API en Direct
    
    <Frame>
        <img src="/images/woocommerce/8.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
4. Connectez-vous à votre tableau de bord Dodo Payments en **Mode Direct**, puis accédez à **Dodo Payments (Mode Direct) > Développeur > Clés API** ou visitez [ce lien direct](https://app.dodopayments.com/developer/api-keys) et cliquez sur **Ajouter une Clé API**
    
    <Frame>
        <img src="/images/woocommerce/9.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
5. Donnez à votre Clé API un nom descriptif et cliquez sur **Créer**
    
    <Frame>
        <img src="/images/woocommerce/10.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
6. Copiez la Clé API générée et collez-la dans le champ **Clé API en Direct** dans les paramètres de votre plugin WooCommerce
    
    <Frame>
        <img src="/images/woocommerce/11.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/12.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
7. Ensuite, configurez la Clé de Signature Webhook pour activer la synchronisation de l'état des paiements entre Dodo Payments et WooCommerce
8. Faites défiler jusqu'en bas de la page des paramètres du plugin et copiez l'URL du webhook affichée là
    
    <Frame>
        <img src="/images/woocommerce/13.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
9. Retournez à votre tableau de bord Dodo Payments et accédez à **Dodo Payments (Mode Direct) > Développeur > Webhooks** et cliquez sur **Ajouter un Webhook**
    
    <Frame>
        <img src="/images/woocommerce/14.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
10. Collez l'URL que vous avez copiée à l'étape 8 dans la boîte de dialogue et cliquez sur **Ajouter un Webhook**
    
    <Frame>
        <img src="/images/woocommerce/15.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
11. Après avoir créé le webhook, cliquez sur l'icône de l'œil à côté de la **Clé de Signature** masquée pour la révéler et la copier
    
    <Frame>
        <img src="/images/woocommerce/16.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
12. Collez la Clé de Signature dans le champ **Clé de Signature Webhook en Direct** dans les paramètres de votre plugin et enregistrez les modifications
    
    <Frame>
        <img src="/images/woocommerce/17.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>
    
13. Passez en revue les paramètres restants tels que **Catégorie de Taxe Globale** et **Tous les Prix Incluent la Taxe**, car ces options déterminent comment les produits se synchronisent de WooCommerce à Dodo Payments. Notez que la **Clé API de Test** et la **Clé de Signature Webhook de Test** ne sont nécessaires que si vous prévoyez d'utiliser le Mode Test
    
    <Frame>
        <img src="/images/woocommerce/18.webp" alt="Plugin Dodo Payments pour WooCommerce" />
    </Frame>

## Fonctionnalités Clés

- **Expérience de Paiement Transparente** : Redirigez les clients vers une page de paiement sécurisée et optimisée
- **Multiples Méthodes de Paiement** : Support pour diverses options de paiement locales et internationales
- **Mises à Jour de Statut en Temps Réel** : Synchronisation instantanée de l'état des commandes via des webhooks
- **Dossiers de Transactions Détaillés** : Historique de paiement complet dans votre tableau de bord WooCommerce
- **Support Multi-Devises** : Acceptez des paiements dans plusieurs devises

## Dépannage

### Problèmes Courants

1. **Échecs de Traitement des Paiements**
   - Vérifiez que les clés API sont correctement configurées
   - Assurez-vous que l'URL du webhook est correctement configurée
   - Vérifiez que votre certificat SSL est valide

2. **Problèmes de Synchronisation des Webhooks**
   - Confirmez que la clé de signature du webhook correspond
   - Consultez les journaux du serveur pour d'éventuelles erreurs
   - Vérifiez que votre serveur peut recevoir des requêtes webhook entrantes

## Meilleures Pratiques de Maintenance

1. Mettez régulièrement à jour le plugin pour des correctifs de sécurité et de nouvelles fonctionnalités
2. Testez tous les flux de paiement en mode sandbox avant de traiter des transactions en direct
3. Gardez WordPress et WooCommerce à jour avec les dernières versions
4. Mettez en œuvre des sauvegardes régulières du site
5. Surveillez les journaux de transactions pour toute activité inhabituelle
6. Conservez les clés API en toute sécurité et ne les partagez jamais publiquement
