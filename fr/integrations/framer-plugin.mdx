## Aperçu

Le plugin Framer Dodo Payments intègre parfaitement le traitement des paiements dans vos sites Web Framer. Créez des expériences de paiement personnalisables avec un support pour les environnements de test et de production.

## Guide d'Installation

<Steps>
{/* LOCKED_PATTERN_386a8c9ba27e2f520f417976ae2d7c92 */}
Visitez le [Tableau de bord Dodo Payments](https://app.dodopayments.com/developer/api-keys) et créez une nouvelle clé API.

<Tabs>

{/* LOCKED_PATTERN_ab5fd62157a874c8d97a623e17ed6dd9 */}
Pour une utilisation en production avec un traitement réel des paiements.
</Tab>

{/* LOCKED_PATTERN_0893a30648532a977fd32fa4cfe5aa5c */}
Parfait pour le développement et les tests sans frais réels.
</Tab>

</Tabs>

<Check>
Copiez votre clé API en toute sécurité : vous en aurez besoin pour l’étape suivante.
</Check>
</Step>

{/* LOCKED_PATTERN_918ab01af8a91f24ca04bebd360e16d7 */}
1. Ouvrez le plugin Dodo Payments dans Framer
2. Accédez à la page Authentification
3. Collez votre clé API
4. Sélectionnez le mode approprié (Test ou Live)
5. Cliquez sur **Continuer** pour vous authentifier

<Warning>
Gardez vos clés API sécurisées et ne les partagez jamais publiquement.
</Warning>
</Step>

{/* LOCKED_PATTERN_1f6cf950f4953c2b98fed68f92e5abd3 */}
Après authentification, vous verrez la liste de vos produits :

- **Produits existants** : Affichés automatiquement
- **Aucun produit** : Cliquez sur le bouton **Plus (+)** à côté de 'Produits'

<Info>
Vous serez redirigé vers le Tableau de bord Dodo Payments pour créer des produits avec tarifications, descriptions et paramètres.
</Info>
</Step>

{/* LOCKED_PATTERN_d0430ab947518d2b45d1080b6954c396 */}
1. Sélectionnez un produit dans votre liste disponible
2. Cliquez sur le bouton `Add Buy Button`
3. Un bouton d’achat avec le texte `Buy {product_name}` sera ajouté à votre canevas

<Warning>
Le bouton Ajouter un achat sera désactivé si vous n’avez pas les autorisations de projet appropriées.
</Warning>
</Step>
</Steps>

## Personnalisation

<Steps>
{/* LOCKED_PATTERN_029f7fc64f4faf96b9708137acbff21d */}
    Choisissez votre expérience de paiement préférée :
    - **Overlay (intégré)** : le paiement s’affiche en superposition dans votre projet Framer, en maintenant les utilisateurs sur votre site.
    - **Hosted** : les utilisateurs sont redirigés vers une page de paiement hébergée par Dodo Payments pour une expérience fluide.
</Step>

{/* LOCKED_PATTERN_fe74d27da37271770f88b3905e82bc15 */}
    Personnalisez votre expérience de paiement avec les paramètres d’apparence suivants :
    - **Thème du checkout** : choisissez entre des thèmes clairs ou sombres pour correspondre à l’esthétique de votre marque.
    - **Libellé du bouton** : personnalisez le texte (par exemple, « Buy Now », « Purchase », « Get Started »).
    - **Styles de bouton** : ajustez les couleurs, polices, tailles et autres propriétés de style.
    - **URL de retour** : définissez une page Framer vers laquelle l’utilisateur revient après le paiement.
</Step>
</Steps>

## Dépannage

<AccordionGroup>
{/* LOCKED_PATTERN_e28ee1b2828d589507a5b4e5ba006e0b */}
**Problème** : le plugin ne s’authentifie pas

**Solutions** :
- Vérifiez que votre clé API est correcte
- Assurez-vous d’utiliser le bon mode (Test/Live)
- Vérifiez votre connexion Internet
</Accordion>

{/* LOCKED_PATTERN_6cf97621a010d8ea59e32f4f5ac4ba4e */}
**Problème** : le bouton Ajouter un achat est désactivé

**Solutions** :
- Vérifiez que vous avez les autorisations appropriées dans votre projet Framer
- Assurez-vous d’être correctement authentifié
- Essayez de rafraîchir le plugin
</Accordion>

{/* LOCKED_PATTERN_44dde71818d2e482b3429adba145540c */}
**Problème** : les produits ne s’affichent pas

**Solutions** :
- Assurez-vous d’avoir créé des produits dans votre tableau de bord Dodo Payments
- Vérifiez que votre clé API dispose des autorisations appropriées
- Essayez de vous réauthentifier
</Accordion>
</AccordionGroup>
