## Aperçu

Sentra ajoute de l'intelligence et de l'exécution au-dessus des piles de base de Dodo Payments. C'est un agent IA qui intègre, analyse et agit sur la facturation et les paiements. Sentra connecte les SDK, les API et les adaptateurs à travers votre pile (Auth, DB, CRM, etc.), met en lumière des informations sur les revenus comme les moteurs de croissance, et exécute des actions clients telles que des remboursements, des mises à niveau et des crédits.

<Info>
Sentra est actuellement en version bêta. Les fonctionnalités, API et intégrations peuvent évoluer rapidement. Nous accueillons volontiers vos retours et rapports de bugs au fil de nos itérations !
</Info>

## Configuration

Une fois installé, vous devez configurer Sentra avec votre clé API Dodo Payments et les paramètres d'environnement.

<Steps>
{/* LOCKED_PATTERN_414ce797076c31d13ae054f651b366fa */}
1. Connectez-vous à votre [tableau de bord Dodo Payments](https://dashboard.dodopayments.com).
2. Accédez à **Settings** › **API Keys**.
3. Copiez votre clé API.
</Step>

{/* LOCKED_PATTERN_2760e1f00720e8d27cb7b397504bdc4c */}
1. Ouvrez votre IDE (VS Code, Cursor ou Windsurf).
2. Ouvrez la palette de commandes (`Cmd+Shift+P` sur macOS / `Ctrl+Shift+P` sur Windows/Linux).
3. Tapez « Sentra: Configure » et sélectionnez la commande.
4. Saisissez votre clé API Dodo Payments lorsqu’elle est demandée.


<Frame>
  <img
    src="/images/sentra/setup.png"
    alt="Écran de configuration de Sentra avec saisie de la clé API et sélection de l’environnement"
    width="50%"
  />
</Frame>
</Step>

{/* LOCKED_PATTERN_1ae7200a035571a906b3c221b7fbc5d8 */}
Choisissez votre mode d’environnement :
- **Mode test** : pour le développement et les tests
- **Mode production** : pour l’utilisation en production


<Warning>
Commencez par le mode test pour valider votre configuration avant de passer en mode production pour les données en direct.
</Warning>
</Step>

{/* LOCKED_PATTERN_0325bd4f4bdf576270b329c2c00316f1 */}
Sélectionnez votre niveau d’effort de raisonnement du modèle :
- **Low** : réponses plus rapides, adapté aux tâches simples
- **Medium** : performance équilibrée (recommandé)
- **High** : analyse plus approfondie, adapté aux intégrations complexes
</Step>

{/* LOCKED_PATTERN_17f7fd2030813d6293cfcc62786a0f48 */}
Cliquez sur **Let’s go!** pour terminer la configuration et commencer à utiliser Sentra.


<Check>

</Check>
</Step>

## Utiliser Sentra

### Modes

Sentra propose trois modes d'interaction accessibles depuis l'interface :

- **Demander** : Posez des questions sur Dodo Payments, les modèles de facturation, l'utilisation de l'API et les meilleures pratiques
- **Construire** : Intégrez la facturation et l'interface utilisateur dans votre application avec une assistance guidée
- **Planifier** : Concevez des architectures de facturation évolutives avec des conseils architecturaux

### Tâches d'Exemple

Voici quelques tâches courantes que vous pouvez accomplir avec Sentra :

- **Intégrer la Page de Tarification** : Configurez une page de tarification complète avec des listes de produits et un processus de paiement
- **Implémenter un Webhook** : Configurez des gestionnaires de webhook pour les événements de paiement
- **Ajouter un Flux d'Abonnement** : Créez une gestion d'abonnement avec des capacités de mise à niveau/déclassement
- **Configurer la Facturation Basée sur l'Utilisation** : Configurez des compteurs et l'ingestion d'événements pour le suivi de l'utilisation
- **Créer un Portail Client** : Construisez un portail d'auto-service pour la gestion de la facturation

### Paramètres d'Auto-Approvisionnement

Vous pouvez configurer des paramètres d'auto-approvisionnement pour rationaliser votre flux de travail :

- **Aucun** : Examinez et approuvez toutes les modifications manuellement (recommandé pour la production)
- **Faible Risque** : Auto-approuvez les modifications simples et à faible risque
- **Toutes** : Auto-approuvez toutes les modifications (à utiliser avec prudence)

## Modèles de Facturation Pris en Charge

Sentra prend en charge tous les modèles de facturation de Dodo Payments :

- **Paiements Uniques** : Transactions d'achat uniques
- **Abonnements** : Facturation récurrente avec plusieurs intervalles
- **Facturation Basée sur l'Utilisation** : Facturation mesurée basée sur la consommation
- **Facturation Basée sur les Résultats** : Modèles de tarification au résultat

## Pour Qui Est-ce ?

Sentra fait le lien entre l'ingénierie et les affaires, afin que chaque équipe puisse expédier la monétisation sans friction :

- **Développeurs** : Expédiez la monétisation plus rapidement dans votre IDE
- **Fondateurs & PMs** : Prototypage de modèles de facturation sans travail d'infrastructure approfondi
- **RevOps & Finance** : Utilisez des actions encadrées sans avoir besoin d'ingénieurs
- **Support & Succès** : Résolvez les cas de facturation en toute sécurité, avec des approbations là où c'est nécessaire
