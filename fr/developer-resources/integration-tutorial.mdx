## Prérequis

Avant de commencer, assurez-vous d'avoir :

- [Créé un compte Dodo Payments](/quickstart)
- [Obtenu vos clés API](/api-reference/introduction#authentication)
- Des connaissances de base en Next.js et React
- Node.js installé (v14 ou supérieur)

## Configuration du Boilerplate

Clonez notre boilerplate Next.js pour commencer rapidement :

```bash
git clone https://github.com/dodopayments/dodo-nextjs-minimal-boilerplate
cd dodo-nextjs-minimal-boilerplate
npm install
```

<Note>
Le boilerplate inclut toutes les dépendances nécessaires et la configuration de base pour l'intégration de Dodo Payments.
</Note>

## Intégration de Webhook

Configurez des webhooks pour gérer les événements asynchrones :

```typescript
import { verifyWebhookSignature } from '@dodopayments/node';

export default async function handler(req, res) {
  const signature = req.headers['dodo-signature'];
  const payload = req.body;
  
  try {
    verifyWebhookSignature({
      payload,
      signature,
      secret: process.env.WEBHOOK_SECRET
    });
    
    // Handle webhook event
    switch (payload.type) {
      case 'payment.succeeded':
        // Handle successful payment
        break;
      case 'subscription.created':
        // Handle new subscription
        break;
      // ... handle other events
    }
    
    res.status(200).json({ received: true });
  } catch (err) {
    res.status(400).json({ error: 'Invalid signature' });
  }
}
```
