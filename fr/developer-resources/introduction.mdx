## URLs d'environnement

Utilisez l'URL de base appropriée pour votre environnement :

- Mode Test : [`https://test.dodopayments.com`](https://test.dodopayments.com)
- Mode Live : [`https://live.dodopayments.com`](https://live.dodopayments.com)

En savoir plus sur [Mode Test vs Mode Live](/miscellaneous/test-mode-vs-live-mode).

## Étapes d'intégration

Suivez ces étapes pour intégrer Dodo Payments :

<Steps>
  1. [Configurez votre compte](/quickstart) et obtenez vos clés API
  2. Choisissez votre méthode d'intégration :
     - [Suivez notre tutoriel](/developer-resources/integration-tutorial)
     - [Utilisez nos SDK](/developer-resources/dodo-payments-sdks)
     - [Intégration API directe](/developer-resources/integration-guide)
  3. [Configurez des webhooks](/developer-resources/webhooks/intents/webhook-events-guide) pour des mises à jour en temps réel
  4. [Testez votre intégration](/miscellaneous/testing-process)
  5. Mettez en ligne !
</Steps>
