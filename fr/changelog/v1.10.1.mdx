## Nouvelles fonctionnalités 🚀

- **Checkout 2.0** : Présente une expérience de paiement entièrement repensée comprenant :
    - Interface moderne épurée pour améliorer l'expérience utilisateur
    - Nouveau système de collecte de requêtes qui stocke les paramètres dans le stockage de session avec des identifiants uniques (e.g., `sess_1a2b3c4d`)
    - Disponibilité persistante des données tout au long du processus de paiement
    - D'autres améliorations prévues pour les prochaines versions !
  
    <Frame><img src="/images/changelog/checkout_2.png" alt="Checkout 2.0"/></Frame>

- **Configurable Payment Methods** : La nouvelle configuration `allowed_payment_methods` permet aux commerçants de personnaliser les options de paiement disponibles pendant le paiement. Consultez la documentation [Méthodes de paiement autorisées](/features/payment-methods#configuring-payment-methods) pour les détails de mise en œuvre.

- **Calcul dynamique des taxes** : Calcul en temps réel des taux de taxe en fonction des adresses de facturation des clients pour des prix plus précis sur les liens de paiement statiques.

- **Abonnements à la demande** : Nouveaux points de terminaison API pour une facturation d'abonnement flexible (disponible pour certains commerçants). En savoir plus dans le guide sur les [Abonnements à la demande](/features/subscription#enabling-on-demand-subscriptions).

- **Mises à jour supplémentaires** :
    - Optimisations de performance
    - Corrections de bogues
    - Améliorations générales du système
