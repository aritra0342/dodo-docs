## Nouvelles fonctionnalités 🚀

- **Checkout 2.0** : Introduction d'une expérience de paiement entièrement repensée avec :
    - Interface moderne et simplifiée pour une meilleure expérience utilisateur
    - Nouveau système de collecte de requêtes qui stocke les paramètres dans le stockage de session avec des ID uniques (par exemple, `sess_1a2b3c4d`)
    - Disponibilité des données persistantes tout au long du processus de paiement
    - D'autres améliorations prévues pour les futures versions !
  
    <Frame><img src="/images/changelog/checkout_2.png" alt="Checkout 2.0"/></Frame>

- **Méthodes de paiement configurables** : La nouvelle configuration `allowed_payment_methods` permet aux commerçants de personnaliser les options de paiement disponibles lors du paiement. Consultez la documentation sur les [Méthodes de paiement autorisées](/features/payment-methods#configuring-payment-methods) pour les détails d'implémentation.

- **Calcul dynamique des taxes** : Calcul en temps réel des taux de taxe en fonction des adresses de facturation des clients pour des prix plus précis sur les liens de paiement statiques.

- **Abonnements à la demande** : Nouveaux points de terminaison API pour une facturation d'abonnement flexible (disponible pour certains commerçants). En savoir plus dans le guide sur les [Abonnements à la demande](/features/subscription#enabling-on-demand-subscriptions).

- **Mises à jour supplémentaires** :
    - Optimisations de performance
    - Corrections de bogues
    - Améliorations générales du système
