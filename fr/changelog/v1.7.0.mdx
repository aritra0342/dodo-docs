## Nouvelles fonctionnalités 🚀

- **Page dédiée aux abonnements** : Gérez facilement les abonnements des clients avec la nouvelle page dédiée. Accédez à `Sales > Subscriptions` sur le tableau de bord.
  
    <Frame><img src="/images/changelog/subscriptions.png" alt="Page des abonnements"/></Frame>

- **Détails clients améliorés** : Consultez les abonnements, paiements et remboursements pour chaque client en un seul endroit. Allez à `Sales > Customers > arrow icon` sur le tableau de bord.

    <Frame><img src="/images/changelog/customer_info.png" alt="Détails du client"/></Frame>

- **Informations détaillées sur les paiements** : Obtenez plus d'informations sur les montants, cycles et dates de paiement. Visitez `Business > Payouts` sur le tableau de bord.

    <Frame><img src="/images/changelog/payouts_update.png" alt="Informations sur les paiements"/></Frame>

- **SDK Ruby officiel** : Intégrez Dodo Payments dans vos applications Ruby sans effort. Consultez le [SDK Ruby](https://github.com/dodopayments/dodopayments-ruby) pour plus de détails.

- **Filtres API améliorés** : Ajout de filtres tels que `customer_id`, `status`, etc., aux API de liste pertinentes. Consultez la documentation API pour plus d'informations.

- **Améliorations mineures et corrections de bogues** : Diverses autres améliorations mineures et corrections de bogues ont été mises en œuvre.
