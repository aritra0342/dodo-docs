## Nouvelles fonctionnalités
- **SDKs officiels** : Introduction des SDKs officiels de Dodo Payments pour **Node.js**, **Python** et **Go**. Consultez la page des [SDKs](/developer-resources/dodo-payments-sdks) pour plus d'informations.
- **Champ de métadonnées** : Ajout d'un champ `Metadata` pour les paiements et les abonnements afin de stocker des informations supplémentaires. Consultez la page des [Métadonnées](/api-reference/metadata) pour plus de détails.
- **APIs Client** : Extension des APIs client pour inclure les points de terminaison suivants. Vous pouvez maintenant passer `customer_id` dans les paiements et les abonnements pour éviter la duplication des clients :
    - **Créer un client** : [/api-reference/customers/create-customer](/api-reference/customers/create-customer)
    - **Modifier un client** : [/api-reference/customers/patch-customer](/api-reference/customers/patch-customer)
