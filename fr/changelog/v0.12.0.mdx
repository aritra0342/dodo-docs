## Nouvelles fonctionnalités
- **SDK officiels** : Introduction des SDK officiels Dodo Payments pour **Node.js**, **Python** et **Go**. Consultez la page [SDKs](/developer-resources/dodo-payments-sdks) pour plus d’informations.
- **Champ Metadata** : Ajout d’un champ `Metadata` pour les paiements et abonnements afin de stocker des informations supplémentaires. Consultez la page [Metadata](/api-reference/metadata) pour plus de détails.
- **API clients** : Extension des API clients pour inclure les points de terminaison suivants. Vous pouvez désormais passer `customer_id` dans les paiements et abonnements pour éviter les doublons de clients :
    - **Créer un client** : [/api-reference/customers/create-customer](/api-reference/customers/create-customer)
    - **Modifier un client** : [/api-reference/customers/patch-customer](/api-reference/customers/patch-customer)
