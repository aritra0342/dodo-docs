## Nouvelles fonctionnalités 🚀

- **Remboursements partiels** : Nous sommes ravis de présenter notre nouvelle fonctionnalité de remboursements partiels, désormais disponible via notre API et le tableau de bord. Cette capacité puissante permet aux commerçants de traiter des remboursements précis pour des composants spécifiques d’un paiement, qu’il s’agisse d’un supplément de siège ou d’une partie d’un abonnement. La fonctionnalité comprend une validation intégrée pour éviter les erreurs et génère automatiquement des reçus de remboursement pour vos dossiers.
  <Frame>
    <img src="/images/changelog/partial_refunds.png" alt="Partial Refunds Cover Image"/>
  </Frame>

- **Support multi-marques** : Nous lançons le support multi-marques, vous permettant de gérer plusieurs marques et sites Web sous un seul compte d'entreprise vérifié. Cette fonctionnalité est idéale pour le lancement de nouvelles gammes de produits, la création de microsites régionaux ou la séparation de différentes offres tout en maintenant une opération rationalisée.

  Les principales caractéristiques incluent :
  - **Vérification rapide** : La plupart des marques sont approuvées dans les 24 heures ouvrables grâce à notre processus de révision de site Web rationalisé
  - **Expérience de paiement de marque** : Les clients voient le logo, le nom et la description de la marque spécifique lors du paiement
  - **Création de marque sans effort** : Créez des marques supplémentaires via Paramètres de l'entreprise ▸ Marques ▸ Ajouter une marque
  - **Analytique des transactions détaillées** : Chaque transaction indique clairement sa marque associée pour de meilleures informations commerciales
  - **Paiements simplifiés** : Tous les revenus se consolident dans votre compte existant sans exigences de vérification supplémentaires

  <Frame>
    <img src="/images/changelog/brands.png" alt="Multi-Brand Cover Image"/>
  </Frame>

  Pour créer des marques supplémentaires, accédez à Business `Settings ▸ Brands ▸ Add Brand` et renseignez les informations requises.

- **Améliorations de la plateforme** : Diverses optimisations de performance, corrections de bogues et améliorations mineures à travers la plateforme.
