## Nouvelles fonctionnalités 🚀

- **Formulaires de vérification modernes** : Refonte des formulaires de vérification avec une interface moderne et conviviale, présentant des instructions plus claires, une hiérarchie visuelle améliorée et des champs de formulaire intuitifs pour un processus de vérification plus fluide.

  <Frame>
    <img alt="Verification Forms" src="/images/changelog/verification_forms.png" />
  </Frame>

- **Nouveaux SDK Java et Kotlin** : Publication de SDK natifs pour Java et Kotlin pour soutenir le développement Android et les applications Java d'entreprise. Les SDK incluent une documentation complète, du code d'exemple, et un support total pour les fonctionnalités de Dodo Payments, y compris les paiements, les abonnements et les remboursements. Disponibles sur GitHub : [SDK Java](https://github.com/dodopayments/dodopayments-java) et [SDK Kotlin](https://github.com/dodopayments/dodopayments-kotlin).

- **Intégration Afterpay/Clearpay** : Ajout du support pour Afterpay/Clearpay en tant que méthode de paiement Buy Now, Pay Later (BNPL), permettant aux clients de diviser les paiements en 4 versements sans intérêt.

- **Google Sign-In** : Mise en œuvre de Google Sign-In comme nouvelle option d'authentification, permettant aux utilisateurs de se connecter en utilisant leurs comptes Google. Cette intégration conforme à OAuth 2.0 prend en charge à la fois les comptes Google personnels et ceux de l'espace de travail pour une sécurité et une commodité accrues.
