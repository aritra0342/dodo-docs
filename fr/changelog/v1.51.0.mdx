## Nouvelles fonctionnalités 🚀

### API de sessions de paiement
Nous vous présentons notre nouvelle **API de sessions de paiement** - une solution de paiement hébergée puissante qui fournit un flux de paiement complet pour les achats uniques et les abonnements. Cette nouvelle API remplace l'approche précédente des liens dynamiques avec une flexibilité améliorée et une meilleure expérience utilisateur.

**Avantages clés :**
- **Intégration simplifiée** : Pas besoin de fournir des adresses de facturation complètes à l'avance - nous gérons automatiquement les détails manquants
- **Personnalisation améliorée** : Contrôle total sur les méthodes de paiement, les devises, les thèmes et les indicateurs de fonctionnalités
- **Meilleure expérience utilisateur** : Pré-remplissez les informations client pour réduire les frictions lors du paiement et améliorer les taux de conversion
- **Configuration flexible** : Prise en charge des paniers multi-produits, des essais d'abonnement, des codes de réduction, et plus encore

**Ce qui a changé :**
- **Avant** : Nécessitait des informations complètes sur l'adresse de facturation lors de la création de liens de paiement
- **Maintenant** : Transmettez toutes les informations client que vous avez disponibles - nous collecterons le reste lors du paiement
- **Migration** : Changement simple de point de terminaison API sans traitement supplémentaire requis

### Améliorations mineures

- Les rapports d'exportation des paiements incluent désormais des métadonnées pour une analyse de données améliorée.
- Vous pouvez désormais mettre à jour la date de facturation suivante pour les abonnements directement via l'API.
