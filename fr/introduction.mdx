## Qu'est-ce que Dodo Payments ?

Dodo Payments est le **moteur tout-en-un** pour lancer, développer et monétiser à l'échelle mondiale. Conçu pour la nouvelle génération de produits numériques SaaS et IA, nous gérons tout, des **paiements à la facturation en passant par la distribution** - le tout sans ingénierie supplémentaire.

<Info>
**Conçu pour les produits SaaS et IA** : Gérez les paiements ponctuels, les abonnements, les crédits, les sièges, les jetons ou tout indicateur d'utilisation grâce à notre infrastructure de facturation optimisée.
</Info>

<Tip>
  <strong>Découvrez Sentra : votre assistant de facturation alimenté par l’IA</strong><br/>
  Sentra est un agent IA qui s’intègre, analyse et agit sur la facturation et les paiements directement dans votre IDE. Posez des questions, construisez des intégrations ou planifiez des architectures de facturation - tout simplement en décrivant ce que vous souhaitez.
  <a href="/developer-resources/sentra">
    Commencez avec Sentra &rarr;
  </a>
</Tip>

### Serveurs MCP pour Assistants IA

Connectez votre assistant de codage IA (Cursor, Claude Desktop, Windsurf, VS Code) directement à Dodo Payments avec nos serveurs MCP.

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  Recherche sémantique dans la documentation Dodo Payments - obtenez des réponses instantanées sur les fonctionnalités, les API et les meilleures pratiques
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  Exécutez des opérations API directement - créez des paiements, gérez les abonnements et traitez les remboursements via votre assistant IA
</Card>
</CardGroup>

{/* LOCKED_PATTERN_8ee343b87e057252cc4a8a376b190970 */}
Ajoutez les deux serveurs à votre configuration MCP (`~/.cursor/mcp.json` ou configuration Claude Desktop) :

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
Nécessite Node.js 18+. Consultez la [documentation MCP Server](/developer-resources/mcp-server) pour les instructions détaillées d’installation pour tous les clients pris en charge.
</Info>
</Accordion>
