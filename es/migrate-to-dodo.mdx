**¿Cambiando de otro proveedor de pagos a Dodo Payments?** Nuestra herramienta de migración facilita el traslado de tus **productos, clientes y códigos de descuento sin perder ningún dato.** El proceso es seguro, guiado y se puede realizar en solo unos minutos.

Actualmente, soportamos migraciones desde **Lemon Squeezy, Stripe, Polar.sh y Paddle**.

## Comenzar

Primero, instala la herramienta de migración:

```bash
npm install -g dodo-migrate
```

<Note>
Necesitarás Node.js ≥ 18 instalado en tu computadora.
</Note>

## Migra Tus Datos

### Si vienes de Lemon Squeezy:

```bash
dodo-migrate lemonsqueezy
```

### Si vienes de Stripe:

```bash
dodo-migrate stripe
```

### Si vienes de Polar.sh:

```bash
dodo-migrate polar
```

### Si vienes de Paddle:

```bash
dodo-migrate paddle
```

La herramienta te guiará a través de todo paso a paso. Necesitarás:
- La clave API de tu proveedor (Lemon Squeezy, Stripe, Polar.sh, Paddle, etc.)
- Tu clave API de Dodo Payments  
- Si deseas probar primero (recomendado) o ir en vivo
- A qué marca de Dodo Payments deseas migrar

## ¿Por Qué Usar Nuestra Herramienta de Migración?

- **Sin pérdida de datos** - Todo se transfiere de forma segura
- **Prueba primero** - Pruébalo en modo de prueba antes de ir en vivo
- **Proceso guiado** - Te guiamos a través de cada paso
- **Reanudar si es necesario** - Si algo sale mal, puedes continuar desde donde lo dejaste

## Uso Avanzado

Si prefieres ejecutar la migración sin indicaciones, puedes proporcionar todos los detalles de antemano:

### Migración de Lemon Squeezy:

```bash
dodo-migrate lemonsqueezy \
  --provider-api-key=lsq_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX
```

### Migración de Stripe:

```bash
dodo-migrate stripe \
  --provider-api-key=sk_test_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX \
  --migrate-types=products,coupons
```

### Migración de Polar.sh:

```bash
dodo-migrate polar \
  --provider-api-key=polar_org_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX \
  --migrate-types=products,discounts,customers
```

### Migración de Paddle:

```bash
dodo-migrate paddle \
  --provider-api-key=paddle_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX
```
