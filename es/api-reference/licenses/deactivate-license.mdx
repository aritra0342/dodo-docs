---
title: Desactivar Licencia
description: Este endpoint te permite desactivar una licencia para el usuario.
openapi: post /licenses/deactivate
---

<Info>
**No se requiere clave API**: Este es un endpoint público que no requiere autenticación. Puedes llamarlo directamente desde aplicaciones cliente, software de escritorio o CLIs para desactivar claves de licencia sin exponer tus credenciales de API.
</Info>
