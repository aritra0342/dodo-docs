---
title: Desactivar Licencia
description: Este endpoint te permite desactivar una licencia para el usuario.
openapi: post /licenses/deactivate
---

<Info>
**Sin clave API requerida**: este es un endpoint público que no requiere autenticación. Puedes invocarlo directamente desde aplicaciones cliente, software de escritorio o CLIs para desactivar claves de licencia sin exponer tus credenciales de API.
</Info>
