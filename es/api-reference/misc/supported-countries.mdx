---
title: Países Soportados
openapi: get /checkout/supported_countries
description: Recupera una lista de países que son soportados por Dodo Payments
  para tus clientes.
---

<Tip> Recibirás una lista de códigos de país [ISO Alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) que representan los países que **actualmente soportamos para pagos**. Puedes ver la lista completa de países soportados [aquí](/miscellaneous/list-of-countries-we-accept-payments-from). Esta lista se actualiza regularmente a medida que expandimos nuestra cobertura. </Tip>

<Tip> Nota: Las opciones disponibles incluyen códigos ISO Alpha-2 para **todos los países**, no solo para los que actualmente soportamos. Por favor, consulta la lista de países soportados para confirmar disponibilidad. </Tip>
