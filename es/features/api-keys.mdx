## Introducción

La generación de Claves API permite a los comerciantes autenticar e integrar de manera segura sus sistemas con la plataforma de Dodo Payments para un acceso fluido a las funcionalidades de pago, producto y suscripción.

**Nota :** Las Claves API creadas en modo de prueba estarán limitadas al entorno de modo de prueba y viceversa para el modo en vivo
