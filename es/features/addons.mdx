## ¿Qué Son los Complementos?

Los complementos son productos suplementarios que los clientes pueden comprar junto con su suscripción principal. Son perfectos para:

- **Facturación por asiento**: Miembros adicionales del equipo, licencias de usuario o usuarios concurrentes
- **Mejoras de características**: Características premium, análisis avanzados o soporte prioritario
- **Extensiones de uso**: Almacenamiento adicional, llamadas a la API o asignaciones de ancho de banda
- **Complementos de servicio**: Servicios profesionales, capacitación o horas de consulta

## Beneficios Clave

- **Modelos de Precios Flexibles**: Ofrece planes base con complementos opcionales para crear estructuras de precios sofisticadas. Puedes abordar diversos segmentos de clientes con mejoras que crecen a medida que cambian las necesidades de tus clientes.
- **Optimización de Ingresos**: Aumenta tu ingreso promedio por usuario (ARPU) presentando complementos relevantes. Esto permite oportunidades de venta adicional naturales a medida que los clientes añaden características con el tiempo.
- **Gestión Simplificada**: Gestiona todos los componentes de precios desde un solo panel de control. Los complementos se incluyen automáticamente en las sesiones de pago y en la gestión de suscripciones.
- **Elección del Cliente**: Permite a los clientes personalizar sus suscripciones seleccionando solo los complementos que necesitan, lo que mejora la satisfacción y reduce la deserción.

## Creando Complementos

Los complementos se crean como productos separados en tu panel de control de Dodo Payments y luego se adjuntan a tus productos de suscripción principales. Esta separación te permite:

- Reutilizar complementos en múltiples productos de suscripción
- Gestionar precios de forma independiente
- Rastrear el rendimiento de los complementos por separado
- Actualizar complementos sin afectar las suscripciones base

### Configuración de Complementos

Al crear complementos, puedes configurar:

- **Precios**: Establecer precios únicos o recurrentes para el complemento
- **Ciclo de facturación**: Debe coincidir con el ciclo de facturación de su suscripción
- **Límites de cantidad**: Establecer cantidades mínimas y máximas por cliente
- **Disponibilidad**: Controlar qué productos de suscripción pueden usar el complemento
- **Configuraciones fiscales**: Configurar categorías fiscales apropiadas

## Ejemplos de Integración

### Sesiones de Pago con Complementos
Al crear sesiones de pago, puedes incluir complementos con cantidades personalizadas:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    {
      product_id: 'your_subscription_id',
      quantity: 1,
      addons: [
        {
          addon_id: 'your_addon_id',
          quantity: 3 // 3 additional seats
        }
      ]
    }
  ],
  // ... other checkout options
});
```

### Cambios de Plan con Complementos
Modifica suscripciones existentes para añadir, eliminar o actualizar complementos:

```typescript
// Add add-ons to existing subscription
await client.subscriptions.changePlan('sub_123', {
  product_id: 'prod_new',
  quantity: 1,
  proration_billing_mode: 'difference_immediately',
  addons: [
    { addon_id: 'addon_123', quantity: 2 }
  ]
});

// Remove all existing add-ons
await client.subscriptions.changePlan('sub_123', {
  product_id: 'prod_new',
  quantity: 1,
  proration_billing_mode: 'difference_immediately',
  addons: [] // Empty array removes all existing add-ons
});
```

### Precios Dinámicos
Calcula costos totales dinámicamente en función de las selecciones de complementos:

```typescript
function calculateTotalCost(basePrice: number, addons: AddonSelection[]) {
  const addonTotal = addons.reduce((sum, addon) => 
    sum + (addon.price * addon.quantity), 0
  );
  return basePrice + addonTotal;
}
```

## Mejores Prácticas

- **Comienza simple**: Lanza con 2-3 complementos básicos y expande las opciones según la retroalimentación y el uso de los clientes.
- **Mantén claridad en los precios**: Comunica claramente los precios y el valor de los complementos, para que los clientes entiendan lo que obtienen por el costo adicional.
- **Prueba a fondo**: Valida las combinaciones de complementos para asegurarte de que los cálculos de precios sigan siendo precisos y que los flujos de pago funcionen sin problemas.

### Consideraciones de Diseño

- **Propuesta de Valor Clara**: Cada complemento debe tener un beneficio claro que los clientes puedan entender fácilmente
- **Agrupación Lógica**: Agrupa complementos relacionados en tu flujo de pago
- **Cantidades Flexibles**: Permite a los clientes ajustar las cantidades de complementos según sea necesario
- **Precios Transparentes**: Muestra los costos totales claramente a lo largo del proceso de pago

<Info>
Los complementos son una forma poderosa de crear modelos de precios flexibles y escalables que crecen con tus clientes. Comienza con casos de uso simples y amplía a medida que descubres qué funciona mejor para tu empresa y tus clientes.
</Info>
