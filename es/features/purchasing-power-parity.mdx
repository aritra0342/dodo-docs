## Introducción

[ParityDeals](https://www.paritydeals.com/) es una plataforma que permite la optimización automática de precios basada en la ubicación geográfica de un cliente, comúnmente conocida como precios de "paridad del poder adquisitivo".

Con ParityDeals, también puedes ofrecer [descuentos automáticos basados en el tiempo](https://www.paritydeals.com/features/time-based-discounts/) y [promociones basadas en festividades](https://www.paritydeals.com/features/holiday-based-discounts/) a lo largo del año.

<Info>
Aunque no puedes incrustar directamente el banner de ParityDeals en una tienda de Dodo Payments (ya que no se admiten scripts personalizados), aún puedes implementar ParityDeals de manera efectiva en tu propio sitio web o aplicación.
</Info>

## Cómo Funciona

Cuando ParityDeals está integrado en tu sitio web, los visitantes elegibles verán un banner que incluye un código de descuento basado en la ubicación que pueden usar al momento de pagar.

<Frame>
  <img src="/images/ppp/1.png" alt="Ejemplo de banner de descuento de ParityDeals" />
</Frame>

Estos códigos de descuento son creados y gestionados automáticamente por ParityDeals y aparecerán en tu **Tablero de Dodo Payments** bajo **Ventas → Descuentos**.

## Guía de Configuración Paso a Paso

### 1. Regístrate en ParityDeals

Visita [app.paritydeals.com](https://app.paritydeals.com/) y crea una cuenta.

### 2. Crea Ofertas

Después de iniciar sesión, localiza la opción para crear ofertas. Desde el menú desplegable, selecciona:

**→ "Crear Ofertas a través de Dodo Payments"**

<Frame>
  <img src="/images/ppp/2.png" alt="Creando ofertas a través de Dodo Payments" />
</Frame>

### 3. Genera una Clave API de Dodo Payments

Inicia sesión en tu **tablero de Dodo Payments** y navega a:

**Desarrolladores → Claves API**

Haz clic en **"Crear Nueva API"** y nómbrala algo descriptivo como "ParityDeals".

<Frame>
  <img src="/images/ppp/3.png" alt="Creando una nueva clave API en Dodo Payments" />
</Frame>

### 4. Conecta tu Clave API a ParityDeals

Copia la clave API que creaste y pégala en la interfaz de ParityDeals para establecer la conexión con tu cuenta de Dodo Payments.

<Frame>
  <img src="/images/ppp/4.png" alt="Conectando la clave API a ParityDeals" />
</Frame>

### 5. Selecciona Productos

Una vez conectado, ParityDeals obtendrá tu catálogo de productos de Dodo Payments. Selecciona los productos que deseas asociar con tus ofertas.

<Frame>
  <img src="/images/ppp/5.png" alt="Seleccionando productos para ParityDeals" />
</Frame>

### 6. Configura la Configuración de la Oferta

Ingresa la URL de tu sitio web y opcionalmente selecciona páginas específicas donde debería aparecer el banner de descuento.

Luego elige tu tipo de oferta preferido:

- Ofertas basadas en la ubicación
- Ofertas por festividades
- Ofertas relámpago basadas en el tiempo

<Frame>
  <img src="/images/ppp/6.png" alt="Configurando la configuración de la oferta" />
</Frame>

### 7. Establece Reglas Basadas en el País

Selecciona los países a los que deseas ofrecer descuentos localizados y define el porcentaje de descuento para cada uno. ParityDeals proporciona valores predeterminados recomendados que puedes ajustar según sea necesario.

<Frame>
  <img src="/images/ppp/7.png" alt="Estableciendo reglas de descuento basadas en el país" />
</Frame>

### 8. Personaliza el Banner

Diseña tu banner de descuento al:

- Personalizar el texto
- Seleccionar colores y diseño
- Elegir la posición del banner (parte superior o inferior de la página)

Este banner se mostrará a los visitantes elegibles según tus reglas configuradas.

<Frame>
  <img src="/images/ppp/8.png" alt="Personalizando el banner de descuento" />
</Frame>

### 9. Agrega el Script de ParityDeals a Tu Sitio Web

Después de guardar tu oferta, ParityDeals generará un script. Agrega este script al código de tu sitio web (típicamente en la sección `<head>`).

Este script detecta automáticamente las ubicaciones de los visitantes y muestra el banner de descuento apropiado.

<Frame>
  <img src="/images/ppp/9.png" alt="Script de integración de ParityDeals" />
</Frame>

## ¡Todo Listo!

Ahora estás listo para ofrecer:

- **Descuentos por cuenta regresiva/sensibles al tiempo** - Crea urgencia con ofertas por tiempo limitado
