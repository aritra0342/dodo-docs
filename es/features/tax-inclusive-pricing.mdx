## ¿Qué Son los Precios con Impuestos Incluidos?

Los Precios con Impuestos Incluidos te permiten incluir los impuestos aplicables directamente en el precio del producto que se muestra. Los clientes ven un monto final desde el principio para una experiencia más transparente y amigable para la conversión. En el proceso de pago, Dodo calcula la parte del precio sin impuestos y el monto del impuesto según la ubicación del cliente, y los desglosa claramente en la factura.

<Info>
El impuesto final aún varía según la ubicación del cliente y la categoría fiscal. El proceso de pago y las facturas muestran la parte del impuesto calculada derivada del precio mostrado.
</Info>

## Beneficios Clave

- **Claridad para los clientes**: Un precio final mostrado desde el principio
- **Cumplimiento por región**: El impuesto se calcula por ubicación y se desglosa
- **Facturación precisa**: Las facturas muestran el monto neto y el impuesto derivado del precio
- **Amigable para la conversión**: Sorprendentes reducidas en el proceso de pago

## Configuración del Producto

Crea o edita un producto en tu panel de control, luego habilita los precios con impuestos incluidos en la configuración de precios.

#### Precios

- **Precio** (requerido): Precio base que se muestra al cliente.
- **Precios con Impuestos Incluidos**: Activa para tratar el precio mostrado como con impuestos incluidos. Dodo calculará el monto neto (antes de impuestos) y la parte del impuesto en el proceso de pago según la ubicación del cliente y la categoría fiscal.

<Warning>
Solo las compras futuras reflejan la configuración de impuestos incluidos actualizada. Las compras existentes y las suscripciones activas no se cambian retroactivamente.
</Warning>

<Tip>
Usa un texto de producto consistente como “Impuestos incluidos” para que los clientes entiendan los totales de un vistazo.
</Tip>

## Ejemplos de Integración

### Crear un producto con precios con impuestos incluidos

```typescript
const product = await client.products.create({
  name: 'Pro Plan',
  description: 'All features included',
  price: {
    currency: 'USD',
    price: 10000
    tax_inclusive: true
  }
});
```

### Actualizar un producto existente para activar impuestos incluidos

```typescript
await client.products.update(product.id, {
  price: {
    tax_inclusive: true
  }
});
```

## Mejores Prácticas

- **Elige la categoría fiscal correcta** para asegurar un cálculo correcto por región.
- **Comunica los totales** claramente con etiquetas de “Impuestos Incluidos” en el texto del producto.
- **Prueba en sandbox** para validar.
