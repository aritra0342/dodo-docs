## Moneda Adaptativa

La Moneda Adaptativa permite a los clientes pagar en su moneda local preferida, mejorando la confianza y las tasas de conversión.

### Cómo Funciona

1. **Habilitar**: Habilita la Moneda Adaptativa desde la Configuración de tu Negocio
2. **Seleccionar**: Los clientes pueden cambiar de moneda directamente en el pago
3. **Convertir**: Los precios se convierten dinámicamente utilizando tasas de cambio en tiempo real
4. **Mostrar**: El monto final a pagar se muestra de manera transparente antes del pago

## Cálculo Automático de Impuestos

Los impuestos se calculan automáticamente en función de la ubicación de facturación del cliente, asegurando el cumplimiento de los requisitos de GST, VAT y impuestos sobre ventas sin configuración manual.

### Cómo Funciona el Cálculo de Impuestos

<Steps>
<Step title="Detección de Ubicación">
  Las reglas fiscales se aplican según el país del cliente (y la región donde sea aplicable).
</Step>

<Step title="Actualizaciones Dinámicas">
  El monto del impuesto se actualiza automáticamente cuando:
  - Cambia el país
  - Se actualiza la dirección
</Step>

<Step title="Visualización Transparente">
  El desglose final de impuestos se muestra claramente antes del pago.
</Step>
</Steps>

<Tip>
El cálculo de impuestos es completamente automatizado. No se requiere configuración manual para bienes digitales estándar y productos SaaS.
</Tip>

## Soporte para ID Fiscal Empresarial

Para empresas registradas, el pago permite a los clientes ingresar su ID Fiscal Empresarial (por ejemplo, número de VAT/GST).

### Qué Ocurre Cuando se Ingresa un ID Fiscal

- La elegibilidad fiscal se valida en tiempo real
- Se aplican exenciones fiscales o reglas de inversión de carga aplicables
- El monto del impuesto se actualiza instantáneamente en el pago

## Códigos de Descuento

Los clientes pueden aplicar códigos de descuento o promoción que has creado en el panel directamente en la página de pago.

### Experiencia de Pago

1. El cliente ingresa el código de descuento
2. El descuento se valida instantáneamente
3. El precio actualizado y los ahorros se muestran claramente

<Frame>
  <img src="/images/checkout/c4.png" alt="Ingreso de código de descuento en el pago" style={{ width: '70%' }}/>
</Frame>

### Integración API

Aplica un código de descuento previamente o habilita el campo de entrada de descuento:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  discount_code: 'WELCOME20', // Pre-apply a code
  feature_flags: {
    allow_discount_code: true // Show discount input field
  },
  return_url: 'https://yoursite.com/return'
});
```

## Finalización de la Colección

Las colecciones de productos permiten una experiencia de finalización unificada donde los clientes pueden ver y seleccionar entre múltiples productos relacionados (por ejemplo, planes Starter, Pro, Enterprise) en una única finalización.

### Cómo Funciona

1. **Todos los productos mostrados**: Los clientes ven cada producto activo en la colección
2. **Primer producto preseleccionado**: El primer producto de la colección se selecciona automáticamente
3. **Comparar opciones**: Los clientes pueden comparar precios y características antes de elegir
4. **Selección única**: Después de seleccionar un producto, la finalización continúa con el flujo de pago estándar

### Creando una Finalización de Colección

```typescript
const session = await client.checkoutSessions.create({
  product_collection_id: 'pdc_abc123',
  product_cart: [], // Required: pass an empty array for collection checkout
  return_url: 'https://yoursite.com/return'
});
```

<Warning>
Al utilizar `product_collection_id`, pasa un array vacío `product_cart`. Los códigos de descuento no pueden aplicarse previamente en la creación de sesión.
</Warning>

## Configuración de la Sesión de Finalización

Controla el comportamiento de la finalización utilizando la API de Sesiones de Finalización:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  customer: {
    email: 'customer@example.com',
    name: 'Jane Doe'
  },
  billing_currency: 'EUR', // Set specific currency
  discount_code: 'PROMO10',
  feature_flags: {
    allow_discount_code: true
  },
  return_url: 'https://yoursite.com/return',
  metadata: {
    order_ref: 'ORD-12345'
  }
});
```
