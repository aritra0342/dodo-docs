## Descripción General

Las habilidades del agente son capacidades reutilizables que mejoran lo que los asistentes de codificación AI pueden hacer al trabajar con Dodo Payments. Proporcionan conocimientos procedimentales y mejores prácticas que ayudan a los agentes a realizar tareas específicas de manera más efectiva: desde la integración de flujos de pago hasta la configuración de facturación basada en el uso.

<Info>
**Piensa en las skills como plugins para tu asistente de IA.** Enseñan a tu agente cómo implementar correctamente las funciones de Dodo Payments, siguiendo nuestros patrones recomendados y buenas prácticas.
</Info>

Las habilidades funcionan con cualquier agente AI compatible con MCP, incluyendo:
- **Claude Code** - Asistente de codificación CLI de Anthropic
- **OpenCode** - Agente de codificación AI de código abierto
- **Cursor** - Editor de código centrado en AI
- **Otros clientes MCP** - Cualquier agente que soporte el protocolo de habilidades

## Instalación

Elige tu método de instalación preferido según tu asistente de codificación AI.

<Tabs>
<Tab title="Skills CLI">
Instala las skills utilizando la CLI universal de skills que funciona con cualquier agente compatible con MCP:

```bash
# Install all Dodo Payments skills
npx skills add dodopayments/skills
```

O instala habilidades individuales según sea necesario:

```bash
# Best practices guide
npx skills add dodopayments/skills/dodo-best-practices

# Webhook integration
npx skills add dodopayments/skills/webhook-integration

# Subscription billing
npx skills add dodopayments/skills/subscription-integration

# Checkout flows
npx skills add dodopayments/skills/checkout-integration

# Usage-based billing
npx skills add dodopayments/skills/usage-based-billing

# BillingSDK components
npx skills add dodopayments/skills/billing-sdk

# License key management
npx skills add dodopayments/skills/license-keys
```

<Tip>
Instala solo las skills que necesites para mantener a tu agente enfocado y reducir la sobrecarga de contexto.
</Tip>
</Tab>

<Tab title="Claude Code">
Añade el marketplace de skills de Dodo Payments a Claude Code:

```bash
/plugin marketplace add dodopayments/skills
```

Luego instala complementos individuales:

```bash
# Install best practices
/plugin install dodo-best-practices

# Install webhook integration
/plugin install webhook-integration

# Install subscription integration
/plugin install subscription-integration

# Install checkout integration
/plugin install checkout-integration

# Install usage-based billing
/plugin install usage-based-billing

# Install BillingSDK
/plugin install billing-sdk

# Install license keys
/plugin install license-keys
```

</Tab>

<Tab title="OpenCode">
Las skills están disponibles automáticamente cuando se configuran en tus ajustes de OpenCode. Añade el repositorio de skills de Dodo Payments a tu configuración:

```json
{
  "skills": {
    "repositories": [
      "dodopayments/skills"
    ]
  }
}
```

</Tab>
</Tabs>

## Usando Habilidades

Una vez instaladas, tu asistente AI puede aprovechar estas habilidades cuando le pidas implementar características de Dodo Payments. Aquí hay algunos ejemplos de prompts:

### Mejores Prácticas

```
Help me integrate Dodo Payments into my Next.js app following best practices
```

### Webhooks

```
Set up webhook handlers for payment and subscription events
```

### Suscripciones

```
Implement a subscription flow with free trial and multiple pricing tiers
```

### Checkout

```
Create a checkout session for my SaaS product
```

### Facturación Basada en Uso

```
Add usage-based billing to track API calls for my AI product
```

### BillingSDK

```
Add a pricing page using BillingSDK components
```

### Claves de licencia

```
Implement license key validation for my desktop app
```

## Cómo Funcionan las Habilidades

Las habilidades proporcionan a tu asistente AI:

1. **Conocimientos Procedimentales** - Instrucciones paso a paso para implementar características
2. **Plantillas de Código** - Patrones y fragmentos de código listos para producción
3. **Mejores Prácticas** - Enfoques recomendados para seguridad, manejo de errores y UX
4. **Consciencia Contextual** - Comprensión de cómo funcionan juntas las diferentes características de Dodo Payments

Cuando le pides a tu asistente AI que implemente una característica, utiliza la habilidad relevante para:
- Comprender el enfoque de implementación correcto
- Generar código preciso y funcional
- Seguir las convenciones de Dodo Payments
- Manejar adecuadamente los casos extremos y los errores

<Note>
Las skills complementan el [MCP Server](/developer-resources/mcp-server) y [Sentra](/developer-resources/sentra). Utiliza MCP para operaciones de API, Sentra para integración guiada y Skills para potenciar las capacidades del asistente de IA.
</Note>
