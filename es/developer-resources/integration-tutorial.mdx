## Requisitos Previos

Antes de comenzar, asegúrate de tener:

- [Creado una cuenta de Dodo Payments](/quickstart)
- [Obtenido tus claves API](/api-reference/introduction#authentication)
- Conocimientos básicos de Next.js y React
- Node.js instalado (v14 o superior)

## Configuración del Boilerplate

Clona nuestro boilerplate de Next.js para comenzar rápidamente:

```bash
git clone https://github.com/dodopayments/dodo-nextjs-minimal-boilerplate
cd dodo-nextjs-minimal-boilerplate
npm install
```

<Note>
El boilerplate incluye todas las dependencias necesarias y la configuración básica para la integración de Dodo Payments.
</Note>

## Integración de Webhooks

Configura webhooks para manejar eventos asíncronos:

```typescript
import { verifyWebhookSignature } from '@dodopayments/node';

export default async function handler(req, res) {
  const signature = req.headers['dodo-signature'];
  const payload = req.body;
  
  try {
    verifyWebhookSignature({
      payload,
      signature,
      secret: process.env.WEBHOOK_SECRET
    });
    
    // Handle webhook event
    switch (payload.type) {
      case 'payment.succeeded':
        // Handle successful payment
        break;
      case 'subscription.created':
        // Handle new subscription
        break;
      // ... handle other events
    }
    
    res.status(200).json({ received: true });
  } catch (err) {
    res.status(400).json({ error: 'Invalid signature' });
  }
}
```
