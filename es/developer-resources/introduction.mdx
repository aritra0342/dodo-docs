## URLs del Entorno

Utiliza la URL base apropiada para tu entorno:

- Modo de Prueba: [`https://test.dodopayments.com`](https://test.dodopayments.com)
- Modo en Vivo: [`https://live.dodopayments.com`](https://live.dodopayments.com)

Aprende más sobre [Modo de Prueba vs Modo en Vivo](/miscellaneous/test-mode-vs-live-mode).

## Pasos de Integración

Sigue estos pasos para integrar Dodo Payments:

<Steps>
  1. [Configura tu cuenta](/quickstart) y obtén tus claves API
  2. Elige tu método de integración:
     - [Sigue nuestro tutorial](/developer-resources/integration-tutorial)
     - [Usa nuestros SDKs](/developer-resources/dodo-payments-sdks)
     - [Integración directa de API](/developer-resources/integration-guide)
  3. [Configura webhooks](/developer-resources/webhooks/intents/webhook-events-guide) para actualizaciones en tiempo real
  4. [Prueba tu integración](/miscellaneous/testing-process)
  5. ¡Lanza en vivo!
</Steps>
