## Introducción

El plugin de WooCommerce de Dodo Payments proporciona una integración sin problemas con tu tienda, permitiéndote aceptar pagos a través de múltiples métodos, incluyendo UPI, tarjetas, banca en línea y billeteras digitales. Esta guía completa te guiará a través del proceso de instalación y configuración.

## Guía de Instalación Paso a Paso

1. Descarga el último zip del plugin de pago de WooCommerce.

    <Card title="Descargar Plugin" icon="download" href="https://github.com/dodopayments/dodo-woocommerce/releases">
    Obtén la última versión de nuestro plugin de WooCommerce
    </Card>

    Alternativamente, puedes instalar directamente desde el directorio de plugins de WordPress:
    
    <Card title="Directorio de Plugins de WordPress" icon="wordpress" href="https://wordpress.org/plugins/dodo-payments-for-woocommerce/">
    Instala Dodo Payments para WooCommerce desde WordPress.org
    </Card>

2. Inicia sesión en tu sitio web de WordPress como administrador (asumimos que ya tienes WooCommerce configurado)
3. Navega a **Plugins → Añadir Nuevo Plugin**
    
    <Frame>
        <img src="/images/woocommerce/1.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
4. Haz clic en el botón **Subir Plugin** y aparecerá un cuadro de diálogo para seleccionar archivos

    <Frame>
        <img src="/images/woocommerce/2.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/3.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
5. Haz clic en **Examinar…** y selecciona el archivo zip que descargaste
6. Haz clic en el botón **Instalar Ahora** para comenzar el proceso de instalación
    
    <Frame>
        <img src="/images/woocommerce/4.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
7. Después de que se complete la instalación, haz clic en **Activar Plugin** o actívalo desde la sección de **Plugins Instalados**
    
    <Frame>
        <img src="/images/woocommerce/5.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
8. El plugin está ahora instalado, pero aún se requiere configuración. Continúa con la guía de configuración a continuación.

## Guía de Configuración

### Configuración de Claves API y Webhooks

1. Navega a **WooCommerce → Configuración → Pagos** o haz clic en el botón **Pagos** en la barra lateral izquierda debajo del elemento de menú de WooCommerce
    
    <Frame>
        <img src="/images/woocommerce/6.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/7.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
2. Habilita el proveedor de pagos **Dodo Payments** si no está habilitado, luego haz clic en **Gestionar** para configurar el plugin
3. Verás varias opciones de configuración, cada una con texto explicativo útil. Comienza configurando tu Clave API en Vivo
    
    <Frame>
        <img src="/images/woocommerce/8.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
4. Inicia sesión en tu panel de Dodo Payments en **Modo en Vivo**, luego navega a **Dodo Payments (Modo en Vivo) > Desarrollador > Claves API** o visita [este enlace directo](https://app.dodopayments.com/developer/api-keys) y haz clic en **Agregar Clave API**
    
    <Frame>
        <img src="/images/woocommerce/9.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
5. Dale a tu Clave API un nombre descriptivo y haz clic en **Crear**
    
    <Frame>
        <img src="/images/woocommerce/10.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
6. Copia la Clave API generada y pégala en el campo **Clave API en Vivo** en la configuración de tu plugin de WooCommerce
    
    <Frame>
        <img src="/images/woocommerce/11.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/12.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
7. A continuación, configura la Clave de Firma de Webhook para habilitar la sincronización del estado de pago entre Dodo Payments y WooCommerce
8. Desplázate hasta la parte inferior de la página de configuración del plugin y copia la URL del webhook que se muestra allí
    
    <Frame>
        <img src="/images/woocommerce/13.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
9. Regresa a tu panel de Dodo Payments y navega a **Dodo Payments (Modo en Vivo) > Desarrollador > Webhooks** y haz clic en **Agregar Webhook**
    
    <Frame>
        <img src="/images/woocommerce/14.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
10. Pega la URL que copiaste en el paso 8 en el cuadro de diálogo y haz clic en **Agregar Webhook**
    
    <Frame>
        <img src="/images/woocommerce/15.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
11. Después de crear el webhook, haz clic en el ícono de ojo junto a la **Clave de Firma** redactada para revelarla y copiarla
    
    <Frame>
        <img src="/images/woocommerce/16.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
12. Pega la Clave de Firma en el campo **Clave de Firma de Webhook en Vivo** en la configuración de tu plugin y guarda los cambios
    
    <Frame>
        <img src="/images/woocommerce/17.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>
    
13. Revisa las configuraciones restantes como **Categoría Fiscal Global** y **Todos los Precios son Incluidos Impuestos**, ya que estas opciones determinan cómo se sincronizan los productos de WooCommerce a Dodo Payments. Ten en cuenta que la **Clave API de Prueba** y la **Clave de Firma de Webhook de Prueba** solo son necesarias si planeas usar el Modo de Prueba
    
    <Frame>
        <img src="/images/woocommerce/18.webp" alt="Plugin de Dodo Payments para WooCommerce" />
    </Frame>

## Características Clave

- **Experiencia de Pago Sin Problemas**: Redirige a los clientes a una página de pago segura y optimizada
- **Múltiples Métodos de Pago**: Soporte para varias opciones de pago locales e internacionales
- **Actualizaciones de Estado en Tiempo Real**: Sincronización instantánea del estado del pedido a través de webhooks
- **Registros de Transacciones Detallados**: Historial de pagos completo en tu panel de WooCommerce
- **Soporte Multimoneda**: Acepta pagos en múltiples monedas

## Solución de Problemas

### Problemas Comunes

1. **Fallos en el Procesamiento de Pagos**
   - Verifica que las claves API estén configuradas correctamente
   - Asegúrate de que la URL del webhook esté configurada correctamente
   - Verifica que tu certificado SSL sea válido

2. **Problemas de Sincronización de Webhooks**
   - Confirma que la clave de firma del webhook coincida
   - Revisa los registros del servidor en busca de posibles errores
   - Verifica que tu servidor pueda recibir solicitudes de webhook entrantes

## Mejores Prácticas de Mantenimiento

1. Actualiza regularmente el plugin para parches de seguridad y nuevas características
2. Prueba todos los flujos de pago en modo sandbox antes de procesar transacciones en vivo
3. Mantén WordPress y WooCommerce actualizados a las últimas versiones
4. Implementa copias de seguridad regulares del sitio
5. Monitorea los registros de transacciones en busca de cualquier actividad inusual
6. Almacena las claves API de forma segura y nunca las compartas públicamente
