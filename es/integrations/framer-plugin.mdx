## Descripción general

El plugin de Dodo Payments para Framer integra sin problemas el procesamiento de pagos en tus sitios web de Framer. Crea experiencias de pago personalizables con soporte para entornos de prueba y producción.

<Card title="Plugin Oficial" icon="link" href="https://www.framer.com/marketplace/plugins/dodo-payments">
Accede al plugin oficial de Dodo Payments en el mercado de Framer.
</Card>

## Guía de Configuración

<Steps>
<Step title="Generar Clave API">
Visita el [Tablero de Dodo Payments](https://app.dodopayments.com/developer/api-keys) y crea una nueva clave API.

<Tabs>

<Tab title="Modo en Vivo">
Para uso en producción con procesamiento de pagos reales.
</Tab>

<Tab title="Modo de Prueba">
Perfecto para desarrollo y pruebas sin cargos reales.
</Tab>

</Tabs>

<Check>
Copia tu clave API de forma segura; la necesitarás para el siguiente paso.
</Check>
</Step>

<Step title="Autenticar Plugin">
1. Abre el plugin de Dodo Payments en Framer
2. Navega a la página de Autenticación
3. Pega tu clave API
4. Selecciona el modo apropiado (Prueba o Vivo)
5. Haz clic en **Continuar** para autenticar

<Warning>
Mantén tus claves API seguras y nunca las compartas públicamente.
</Warning>
</Step>

<Step title="Gestionar Productos">
Después de la autenticación, verás tu lista de productos:

- **Productos existentes**: Se muestran automáticamente
- **Sin productos**: Haz clic en el botón **Más (+)** junto a 'Productos'

<Info>
Serás redirigido al Tablero de Dodo Payments para crear productos con precios, descripciones y configuraciones.
</Info>
</Step>

<Step title="Agregar botones de compra">
1. Selecciona un producto de tu lista disponible
2. Haz clic en el `Add Buy Button` botón
3. Se añadirá un botón de compra con el texto 'Comprar \{product_name\}' a tu lienzo

<Warning>
El botón Agregar Botón de Compra estará deshabilitado si no tienes los permisos de proyecto correctos.
</Warning>
</Step>
</Steps>

## Personalización

<Steps>
<Step title="Seleccionar Experiencia de Pago">
    Elige tu experiencia de pago preferida:
    - **Superpuesto (Incrustado)**: El pago aparece como un superpuesto dentro de tu proyecto de Framer, manteniendo a los usuarios en tu sitio.
    - **Alojado**: Los usuarios son redirigidos a una página de pago alojada por Dodo Payments para una experiencia simplificada.
</Step>

<Step title="Personalizar Opciones de Apariencia">
    Personaliza tu experiencia de pago con las siguientes configuraciones de apariencia:
    - **Tema de Pago**: Elige entre temas claros u oscuros para que coincidan con la estética de tu marca.
    - **Etiqueta del Botón**: Personaliza el texto (por ejemplo, 'Comprar Ahora', 'Adquirir', 'Comenzar').
    - **Estilos del Botón**: Ajusta colores, fuentes, tamaños y otras propiedades de estilo.
    - **URL de Retorno**: Establece una página de Framer a la que el usuario regresará después del pago.
</Step>
</Steps>

## Solución de Problemas

<AccordionGroup>
<Accordion title="Problemas de Autenticación">
**Problema**: El plugin no se autentica

**Soluciones**:
- Verifica que tu clave API sea correcta
- Asegúrate de estar usando el modo correcto (Prueba/Vivo)
- Verifica tu conexión a Internet
</Accordion>

<Accordion title="Problemas al Crear Botones">
**Problema**: Agregar Botón de Compra está deshabilitado

**Soluciones**:
- Verifica que tengas los permisos correctos en tu proyecto de Framer
- Asegúrate de estar autenticado correctamente
- Intenta refrescar el plugin
</Accordion>

<Accordion title="Problemas de Visualización de Productos">
**Problema**: Los productos no se muestran

**Soluciones**:
- Asegúrate de haber creado productos en tu tablero de Dodo Payments
- Verifica que tu clave API tenga los permisos correctos
- Intenta re-autenticar
</Accordion>
</AccordionGroup>
