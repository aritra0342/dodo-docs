## Descripción general

El plugin de Dodo Payments para Framer integra sin problemas el procesamiento de pagos en tus sitios web de Framer. Crea experiencias de pago personalizables con soporte para entornos de prueba y producción.

## Guía de Configuración

<Steps>
{/* LOCKED_PATTERN_386a8c9ba27e2f520f417976ae2d7c92 */}
Visita el [Dodo Payments Dashboard](https://app.dodopayments.com/developer/api-keys) y crea una nueva clave de API.

<Tabs>

{/* LOCKED_PATTERN_ab5fd62157a874c8d97a623e17ed6dd9 */}
Para uso en producción con procesamiento real de pagos.
</Tab>

{/* LOCKED_PATTERN_0893a30648532a977fd32fa4cfe5aa5c */}
Perfecto para desarrollo y pruebas sin cargos reales.
</Tab>

</Tabs>

<Check>
Copia tu clave de API de forma segura: la necesitarás en el siguiente paso.
</Check>
</Step>

{/* LOCKED_PATTERN_918ab01af8a91f24ca04bebd360e16d7 */}
1. Abre el plugin de Dodo Payments en Framer
2. Ve a la página de Autenticación
3. Pega tu clave de API
4. Selecciona el modo adecuado (Test o Live)
5. Haz clic en **Continue** para autenticarte

<Warning>
Mantén tus claves de API seguras y nunca las compartas públicamente.
</Warning>
</Step>

{/* LOCKED_PATTERN_1f6cf950f4953c2b98fed68f92e5abd3 */}
Después de la autenticación, verás la lista de tus productos:

- **Productos existentes**: Se muestran automáticamente
- **Sin productos**: Haz clic en el botón **Más (+)** junto a 'Productos'

<Info>
Serás redirigido al Panel de Dodo Payments para crear productos con precios, descripciones y configuraciones.
</Info>
</Step>

{/* LOCKED_PATTERN_d0430ab947518d2b45d1080b6954c396 */}
1. Selecciona un producto de tu lista disponible
2. Haz clic en el botón `Add Buy Button`
3. Se agregará a tu lienzo un botón de compra con el texto `Buy {product_name}`

<Warning>
El botón Agregar compra estará deshabilitado si no tienes los permisos adecuados del proyecto.
</Warning>
</Step>
</Steps>

## Personalización

<Steps>
{/* LOCKED_PATTERN_029f7fc64f4faf96b9708137acbff21d */}
    Elige la experiencia de pago que prefieras:
    - **Overlay (Embedded)**: El proceso de pago aparece como una superposición dentro de tu proyecto de Framer, manteniendo a los usuarios en tu sitio.
    - **Hosted**: Los usuarios son redirigidos a una página de pago alojada por Dodo Payments para una experiencia simplificada.
</Step>

{/* LOCKED_PATTERN_fe74d27da37271770f88b3905e82bc15 */}
    Personaliza tu experiencia de pago con los siguientes ajustes de apariencia:
    - **Checkout Theme**: Elige entre temas claro u oscuro para que coincida con la estética de tu marca.
    - **Button Label**: Personaliza el texto (por ejemplo, 'Buy Now', 'Purchase', 'Get Started').
    - **Button Styles**: Ajusta colores, fuentes, tamaños y otras propiedades de estilo.
    - **Return URL**: Establece una página de Framer a la que el usuario regresará después del pago.
</Step>
</Steps>

## Solución de Problemas

<AccordionGroup>
{/* LOCKED_PATTERN_e28ee1b2828d589507a5b4e5ba006e0b */}
**Problema**: El plugin no se autentica

**Solutions**:
- Verifica que tu clave de API sea correcta
- Asegúrate de estar usando el modo adecuado (Test/Live)
- Verifica tu conexión a internet
</Accordion>

{/* LOCKED_PATTERN_6cf97621a010d8ea59e32f4f5ac4ba4e */}
**Problema**: El botón Agregar compra está deshabilitado

**Solutions**:
- Verifica que tengas los permisos correctos en tu proyecto de Framer
- Asegúrate de estar autenticado correctamente
- Intenta actualizar el plugin
</Accordion>

{/* LOCKED_PATTERN_44dde71818d2e482b3429adba145540c */}
**Problema**: Los productos no se muestran

**Solutions**:
- Asegúrate de tener productos creados en tu panel de Dodo Payments
- Comprueba que tu clave de API tenga los permisos correctos
- Intenta volver a autenticarte
</Accordion>
</AccordionGroup>
