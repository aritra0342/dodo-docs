Tu panel de Dodo Payments, a unos pocos pulsos de teclado. Usa Raycast para ver rápidamente pagos, suscripciones, clientes, productos, códigos de descuento, claves de licencia y reembolsos sin abrir el panel.

## Instalar

<Card title="Install from Raycast Store →" icon="puzzle-piece" href="https://www.raycast.com/dodopayments/dodo-payments">
  Abre la página de la extensión Dodo Payments en la Raycast Store.
</Card>

<Info>
  Requiere Raycast en macOS y una clave API de Dodo Payments.
</Info>

## Configuración

<Steps>
<Step title="Get your API Key">
1. Inicia sesión en tu panel de Dodo Payments.
2. Ve a Configuración › Claves API.
3. Copia tu clave API.
</Step>

<Step title="Configure the Extension">
1. Abre Raycast.
2. Busca "Dodo Payments" y selecciona cualquier comando.
3. Cuando se solicite, introduce tu clave API en el campo API Key.
4. Elige el modo de API:
   - Test: úsalo para desarrollo y pruebas.
   - Live: úsalo para datos de producción.
5. Guarda la configuración.
</Step>
</Steps>

<Tip>
Empieza en modo Test para validar tu configuración, luego cambia a Live.
</Tip>

<Warning>
Trata tu clave API como una contraseña. No la compartas ni la almacenes en texto sin formato.
</Warning>

## Usa la extensión

- **Buscar**: abre Raycast, escribe "Dodo Payments" y elige un comando.
- **Filtrar**: reduce los resultados escribiendo partes de IDs, correos electrónicos, nombres o estados.
- **Abrir detalles**: presiona Enter en un elemento de la lista para ver más información.

## Solución de problemas

<AccordionGroup>
<Accordion title="Authentication failed or 401/403 errors">
Confirma que la clave API y el modo (Test vs Live) coinciden con los datos de tu panel. Vuelve a abrir el comando y vuelve a introducir la clave si es necesario.
</Accordion>
<Accordion title="No results appear">
Asegúrate de que el modo seleccionado tenga datos. Prueba a cambiar entre Test y Live.
</Accordion>
<Accordion title="Rate limits or timeouts">
Espera un momento y vuelve a intentarlo. Si el problema persiste, comprueba tu conectividad de red.
</Accordion>
</AccordionGroup>
