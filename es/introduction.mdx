## ¿Qué es Dodo Payments?

Dodo Payments es el **motor todo en uno** para lanzar, escalar y monetizar a nivel mundial. Diseñado para la nueva generación de productos digitales SaaS e IA, manejamos todo, desde **pagos hasta facturación y distribución** - todo sin ingeniería adicional.

<Info>
**Construido para Productos SaaS e IA**: Maneja pagos únicos, suscripciones, créditos, asientos, tokens o cualquier métrica de uso con nuestra infraestructura de facturación optimizada.
</Info>

<Tip>
  <strong>Conoce a Sentra: Tu asistente de facturación impulsado por IA</strong><br/>
  Sentra es un agente de IA que integra, analiza y actúa sobre la facturación y los pagos directamente en tu IDE. Haz preguntas, construye integraciones o planifica arquitecturas de facturación, todo describiendo lo que necesitas.
  <a href="/developer-resources/sentra">
    Comienza con Sentra &rarr;
  </a>
</Tip>

## Guía de Inicio Rápido

Configura Dodo Payments en solo unos pasos:

<Steps>
<Step title="Crea Tu Cuenta">
  Regístrate en [app.dodopayments.com](https://app.dodopayments.com/signup) usando tu correo electrónico.

  <Check>
  Recibirás un correo electrónico de confirmación para verificar tu cuenta.
  </Check>
</Step>

<Step title="Agrega Tu Primer Producto">
  Elige tu modelo de facturación y agrega tu producto para comenzar a aceptar pagos.

  <CardGroup cols={3}>
  <Card title="Pago Único" icon="box" href="/features/one-time-payment-products">
    Perfecto para descargas digitales, cursos o compras únicas
  </Card>
  
  <Card title="Producto de Suscripción" icon="repeat" href="/features/subscription">

    Factura a los clientes por llamadas a la API, créditos, tokens o cualquier métrica de uso - ideal para IA y SaaS
  </Card>
  </CardGroup>
</Step>

<Step title="Elige Tu Método de Integración">
  Selecciona el enfoque de integración que mejor se adapte a tus necesidades y requisitos técnicos.

  <Tabs>
  <Tab title="Sin Código (Más Rápido)">
    **Perfecto para comenzar rápidamente:**
    
    - Crea **Enlaces de Pago** desde tu panel de control

  </Tab>

  <Tab title="Checkout en Superposición">
    **Mejor para pagos sin interrupciones y con la marca:**
    
    - Lanza el checkout como una superposición en tu sitio o aplicación
    - Mantén a los usuarios en tu página para una experiencia sin fricciones
    - Personaliza la apariencia para que coincida con tu marca

  </Tab>

    - Crea resúmenes de pedidos personalizados que se sincronicen con el checkout

    <Card title="Integración en Línea" icon="credit-card" href="/developer-resources/inline-checkout">

    - Integración profunda con tu backend, flujos de trabajo y modelos de datos
    - Soporte completo para webhooks, características avanzadas y automatización

### Servidores MCP para Asistentes de IA

Conecta tu asistente de codificación de IA (Cursor, Claude Desktop, Windsurf, VS Code) directamente a Dodo Payments con nuestros servidores MCP.

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  Búsqueda semántica en la documentación de Dodo Payments - obtén respuestas instantáneas sobre características, APIs y mejores prácticas
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  Ejecuta operaciones de API directamente - crea pagos, gestiona suscripciones y maneja reembolsos a través de tu asistente de IA
</Card>
</CardGroup>

<Accordion title="Configuración rápida de MCP para Cursor/Claude Desktop">
Agrega ambos servidores a tu configuración de MCP (`~/.cursor/mcp.json` o configuración de Claude Desktop):

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
Requiere Node.js 18+. Consulta la [documentación del servidor MCP](/developer-resources/mcp-server) para obtener instrucciones detalladas de configuración para todos los clientes compatibles.
</Info>
</Accordion>
