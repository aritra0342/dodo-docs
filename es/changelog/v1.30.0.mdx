## Nuevas Funciones 🚀

- **Pagos de Cero Dólares**: Se agregó soporte para pagos de $0 y códigos de descuento del 100% para productos de una sola vez, lo que permite ofertas de productos gratuitos y campañas promocionales.

- **Umbral de Pago Personalizado**: Se agregó la capacidad de establecer montos mínimos de pago para billeteras en USD, ayudando a los comerciantes a optimizar el procesamiento de pagos y reducir tarifas para transacciones pequeñas. Accede a esta función en `Payouts ▸ USD Wallet ▸ Edit Payout Threshold`.
  
  <Frame>
    <img src="/images/changelog/payout-threshold.png" alt="Imagen de Portada del Umbral de Pago"/>
  </Frame>

- **Estandarización de API**: Se implementaron códigos de error y mensajes de API estandarizados como parte de nuestra iniciativa continua de mejora de API. Esta mejora hace que la API sea más consistente y amigable para los desarrolladores. Para información detallada, visita nuestra [Documentación de Códigos de Error](/api-reference/error-codes).

- **Estandarización de Rechazos de Transacciones**: Se introdujeron códigos y mensajes de rechazo de transacciones unificados, proporcionando una visión más clara de las razones de fallo y diferenciando entre errores del usuario y problemas del sistema. Aprende más en nuestra [Documentación de Fallos de Transacciones](/api-reference/transaction-failures).
