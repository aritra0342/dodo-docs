## Nuevas Funciones 🚀

- **Reembolsos Parciales**: Estamos emocionados de presentar nuestra nueva función de Reembolsos Parciales, ahora disponible a través de nuestra API y Dashboard. Esta poderosa capacidad permite a los comerciantes procesar reembolsos precisos para componentes específicos de un pago, ya sea un complemento de asiento o una parte de una suscripción. La función incluye validación incorporada para prevenir errores y genera automáticamente recibos de reembolso para sus registros.
  <Frame>
    <img src="/images/changelog/partial_refunds.png" alt="Imagen de Portada de Reembolsos Parciales"/>
  </Frame>

- **Soporte para Múltiples Marcas**: Estamos lanzando el Soporte para Múltiples Marcas, lo que le permite gestionar múltiples marcas y sitios web bajo una única cuenta de negocio verificada. Esta función es ideal para lanzar nuevas líneas de productos, crear micrositios regionales o separar diferentes ofertas mientras se mantiene una operación simplificada.

  Las características clave incluyen:
  - **Verificación Rápida**: La mayoría de las marcas son aprobadas en un plazo de 24 horas hábiles a través de nuestro proceso de revisión de sitios web simplificado
  - **Experiencia de Pago de Marca**: Los clientes ven el logo, nombre y descriptor específicos de la marca durante el proceso de pago
  - **Creación de Marca Sin Problemas**: Cree marcas adicionales a través de Configuración del Negocio ▸ Marcas ▸ Agregar Marca
  - **Análisis de Transacciones Detallados**: Cada transacción indica claramente su marca asociada para mejores insights comerciales
  - **Pagos Simplificados**: Todos los ingresos se consolidan en su cuenta existente sin requisitos adicionales de verificación

  <Frame>
    <img src="/images/changelog/brands.png" alt="Imagen de Portada de Múltiples Marcas"/>
  </Frame>

  Para crear marcas adicionales, navegue a Negocio `Settings ▸ Brands ▸ Add Brand` y complete los detalles requeridos.

- **Mejoras en la Plataforma**: Varias optimizaciones de rendimiento, correcciones de errores y mejoras menores en toda la plataforma.
