## Nuevas funciones
- **SDKs oficiales**: Se introdujeron los SDKs oficiales de Dodo Payments para **Node.js**, **Python** y **Go**. Consulta la página de los [SDKs](/developer-resources/dodo-payments-sdks) para obtener más información.
- **Campo de metadatos**: Se agregó un campo `Metadata` para pagos y suscripciones con el fin de almacenar información adicional. Consulta la página de [Metadatos](/api-reference/metadata) para obtener más detalles.
- **APIs de clientes**: Se ampliaron las APIs de clientes para incluir los siguientes endpoints. Ahora puedes pasar `customer_id` en pagos y suscripciones para evitar la duplicación de clientes:
    - **Crear cliente**: [/api-reference/customers/create-customer](/api-reference/customers/create-customer)
    - **Actualizar cliente**: [/api-reference/customers/patch-customer](/api-reference/customers/patch-customer)
