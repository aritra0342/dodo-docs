## Nuevas Funciones
- **SDKs Oficiales**: Se introdujeron SDKs oficiales de Dodo Payments para **Node.js**, **Python** y **Go**. Consulta la página de [SDKs](/developer-resources/dodo-payments-sdks) para más información.
- **Campo de Metadatos**: Se añadió un campo `Metadata` para pagos y suscripciones para almacenar información adicional. Consulta la página de [Metadatos](/api-reference/metadata) para más detalles.
- **APIs de Clientes**: Se expandieron las APIs de clientes para incluir los siguientes endpoints. Ahora puedes pasar `customer_id` en pagos y suscripciones para prevenir la duplicación de clientes:
    - **Crear Cliente**: [/api-reference/customers/create-customer](/api-reference/customers/create-customer)
    - **Actualizar Cliente**: [/api-reference/customers/patch-customer](/api-reference/customers/patch-customer)
