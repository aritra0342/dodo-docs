## Nuevas Características
- **Atlas de Código Abierto**: Una aplicación de demostración escrita en Next.js para mostrar la integración de Dodo Payments. Explora el [Repositorio de GitHub](https://github.com/dodopayments/dodo-checkout-demo) y la documentación [aquí](/developer-resources/integration-guide).
- **Sección de Solicitudes de Características**: Se introdujo una sección de solicitudes de características en el panel de Dodo Payments para priorizar la hoja de ruta del producto según los comentarios de los usuarios.
- **API de Registros de Solicitudes de Webhook**: Se añadieron APIs para listar y detallar los registros de solicitudes de webhook.
- **Enlaces de Pago Estáticos**: Se añadieron más parámetros de consulta a los enlaces de pago estáticos para proporcionar más flexibilidad en el proceso de pago. Para más detalles, haz clic [aquí](/developer-resources/integration-guide#1-static-payment-links).
- **API de Lista de Suscripciones**: Se añadió una API para obtener todas las suscripciones. Consulta la [Referencia de API](/api-reference/subscriptions/get-subscriptions-1) para más detalles.

## Mejoras Menores y Correcciones de Errores
- Se mejoraron las respuestas de la API de Producto y Suscripción con parámetros adicionales.
- Se implementó limitación de tasa para las solicitudes de API, actualmente establecida en 20 solicitudes por segundo.
- Se mejoró la documentación para que sea más amigable y fácil de navegar.
- Varias correcciones de errores menores y mejoras.
