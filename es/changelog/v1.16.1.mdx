## Nuevas Funciones 🚀

- **Formularios de Verificación Modernos**: Rediseñamos los formularios de verificación con una interfaz moderna y fácil de usar que presenta instrucciones más claras, una jerarquía visual mejorada y campos de formulario intuitivos para un proceso de verificación más fluido.

  <Frame>
    <img alt="Formularios de Verificación" src="/images/changelog/verification_forms.png" />
  </Frame>

- **Nuevos SDKs de Java y Kotlin**: Lanzamos SDKs nativos para Java y Kotlin para soportar el desarrollo en Android y aplicaciones de Java empresarial. Los SDKs incluyen documentación completa, código de ejemplo y soporte total para las características de Dodo Payments, incluyendo pagos, suscripciones y reembolsos. Disponibles en GitHub: [Java SDK](https://github.com/dodopayments/dodopayments-java) y [Kotlin SDK](https://github.com/dodopayments/dodopayments-kotlin).

- **Integración de Afterpay/Clearpay**: Se agregó soporte para Afterpay/Clearpay como un método de pago de Compra Ahora, Paga Después (BNPL), permitiendo a los clientes dividir los pagos en 4 cuotas sin intereses.

- **Google Sign-In**: Se implementó Google Sign-In como una nueva opción de autenticación, permitiendo a los usuarios iniciar sesión utilizando sus cuentas de Google. Esta integración compatible con OAuth 2.0 soporta tanto cuentas personales como de trabajo de Google para una mayor seguridad y conveniencia.
