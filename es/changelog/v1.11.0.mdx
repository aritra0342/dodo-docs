## Nuevas Funciones 🚀

- **Autocompletado de Direcciones**: Proceso de pago optimizado con autocompletado inteligente de campos de dirección basado en la entrada del usuario.
  
    <Frame><img src="/images/changelog/address_autofill.png" alt="Autocompletado de Direcciones"/></Frame>

- **Centro de Notificaciones**: Mantente informado con una nueva pestaña de notificaciones dedicada para eventos y actualizaciones importantes del sistema.

    <Frame><img src="/images/changelog/notifications.png" alt="Notificaciones"/></Frame>

- **Generador de Consultas de Enlaces de Pago**: Crea y personaliza enlaces de pago de manera más eficiente con nuestro nuevo generador de consultas visual.

    <Frame><img src="/images/changelog/query_selector.png" alt="Generador de Consultas"/></Frame>

- **Métodos de Pago Guardados**: Habilita la visualización de métodos de pago utilizados anteriormente durante el proceso de pago configurando `show_saved_payment_methods: true` en la configuración del enlace de pago.

- **Control de Moneda de Facturación**: Impone monedas de facturación específicas para enlaces de pago configurando el parámetro `billing_currency`.

- **Mejoras**: Varias mejoras y correcciones de errores.
