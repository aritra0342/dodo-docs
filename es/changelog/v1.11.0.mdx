## Nuevas Funciones 🚀

- **Autocompletar dirección**: Proceso de pago optimizado con autocompletado inteligente del campo de dirección según la entrada del usuario.
  
    <Frame><img src="/images/changelog/address_autofill.png" alt="Autocompletar dirección"/></Frame>

- **Centro de Notificaciones**: Mantente informado con una nueva pestaña de notificaciones dedicada para eventos y actualizaciones importantes del sistema.

    <Frame><img src="/images/changelog/notifications.png" alt="Notificaciones"/></Frame>

- **Generador de Consultas de Enlaces de Pago**: Crea y personaliza enlaces de pago de manera más eficiente con nuestro nuevo generador de consultas visual.

    <Frame><img src="/images/changelog/query_selector.png" alt="Constructor de consultas"/></Frame>

- **Métodos de pago guardados**: Permite mostrar métodos de pago previamente utilizados durante el proceso de pago configurando `show_saved_payment_methods: true` en la configuración del enlace de pago.

- **Control de moneda de facturación**: Aplica monedas de facturación específicas para los enlaces de pago configurando el parámetro `billing_currency`.

- **Mejoras**: Varias mejoras y correcciones de errores.
