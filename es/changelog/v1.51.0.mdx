## Nuevas Funciones 🚀

### API de Sesiones de Pago
Presentamos nuestra nueva **API de Sesiones de Pago** - una solución de pago alojada y potente que proporciona un flujo de pago completo tanto para compras únicas como para suscripciones. Esta nueva API reemplaza el enfoque anterior de Enlaces Dinámicos con mayor flexibilidad y una mejor experiencia de usuario.

**Beneficios Clave:**
- **Integración Simplificada**: No es necesario proporcionar direcciones de facturación completas por adelantado - nosotros manejamos los detalles faltantes automáticamente
- **Personalización Mejorada**: Control total sobre métodos de pago, monedas, temas y banderas de características
- **Mejor UX**: Pre-completar la información del cliente para reducir la fricción en el proceso de pago y mejorar las tasas de conversión
- **Configuración Flexible**: Soporte para carritos de múltiples productos, pruebas de suscripción, códigos de descuento y más

**Qué Ha Cambiado:**
- **Antes**: Se requería información completa de la dirección de facturación al crear enlaces de pago
- **Ahora**: Pasa cualquier información del cliente que tengas disponible - nosotros recopilaremos el resto durante el proceso de pago
- **Migración**: Cambio simple de punto final de API sin manejo adicional requerido
