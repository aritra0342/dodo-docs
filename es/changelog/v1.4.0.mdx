## Nuevas Funciones 🚀

- **Función de Storefront**: Crea una tienda en línea simple, con marca y amigable para dispositivos móviles para mostrar tanto productos de Pago Único como productos de Suscripción. Esta función elimina la complejidad de construir un sitio web completo, permitiéndote ofrecer rápidamente tus productos o servicios en línea sin esfuerzos adicionales de desarrollo web.

    <Frame><img src="/images/store-front/sf-1.png"/></Frame>

- **Integración de Amazon Pay**: Se agregó Amazon Pay como método de pago para Pagos Únicos.

- **Mejoras y Correcciones**: Mejoras menores y correcciones de errores.
