## Nuevas Funciones 🚀

- **Integración del Programa de Afiliados**: Lanzamos la integración con Affonso, lo que te permite crear y gestionar programas de afiliados directamente a través de Dodo Payments. Realiza un seguimiento de las referencias, maneja comisiones y aumenta tus ingresos con asociaciones de afiliados de confianza. Aprende más en nuestra documentación de [Afiliados](/features/affiliates).
  <Frame>
    <img src="/images/affiliates/cover.jpeg" alt="Imagen de Portada de Afiliados"/>
  </Frame>

- **Soporte para Múltiples Webhooks**: Se mejoraron las capacidades de webhook para soportar múltiples endpoints, permitiéndote enviar notificaciones de eventos a diferentes sistemas simultáneamente.

- **Mejora en la Prevención de Fraudes**: Se agregó una nueva configuración empresarial para forzar la autenticación 3D Secure para todas las transacciones, ayudando a reducir fraudes y contracargos.

- **Cancelación de Programación de Suscripciones**: Se introdujo la capacidad de programar cancelaciones de suscripciones para que tengan efecto al final del período de facturación, disponible a través de nuestra API.

- **Enlaces de Pago Estáticos**: Se agregaron dos parámetros de consulta más para enlaces de pago estáticos:
  - `paymentCurrency`: Especifica la moneda de pago. El valor predeterminado se establece según el país de facturación.
  - `showCurrencySelector`: Muestra el selector de moneda. El valor predeterminado es verdadero.
