## Nuevas Funciones 🚀

- **Portal del Cliente**: Se introdujo la `Customer Portal` función, que permite a los clientes gestionar sus suscripciones, facturas y métodos de pago. Puedes crear enlaces estáticos o dinámicos del Portal del Cliente para tus clientes a través del Panel de Dodo Payments o mediante APIs. En el panel, navega a `Sales > Customers > Share Invite`.

- **API de Sesión del Portal del Cliente**: Se agregó la `Customer Portal Session` API para crear una sesión del Portal del Cliente para un cliente específico. Ten en cuenta que la sesión es válida solo por 24 horas. Consulta la [Referencia de API](/api-reference/customers/create-customer-portal-session) para más detalles.

- **Otros**: Mejoras menores y correcciones de errores.
