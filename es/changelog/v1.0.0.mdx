## Nuevas funciones 🚀

- **Customer Portal**: Se introdujo la característica `Customer Portal`, que permite a los clientes administrar sus suscripciones, facturas y métodos de pago. Puedes crear enlaces estáticos o dinámicos del Customer Portal para tus clientes a través del Panel de control de Dodo Payments o mediante APIs. En el panel, navega a `Sales > Customers > Share Invite`.

- **Customer Portal Session API**: Se añadió la API `Customer Portal Session` para crear una sesión del Customer Portal para un cliente específico. Ten en cuenta que la sesión sólo es válida por 24 horas. Consulta la [Referencia de API](/api-reference/customers/create-customer-portal-session) para más detalles.

- **Otros**: Mejoras menores y correcciones de errores.
