## Nuevas Funciones 🚀

- **Overlay Checkout Beta**: Estamos emocionados de anunciar el lanzamiento beta de nuestra nueva experiencia de pago en superposición. Esta moderna biblioteca de TypeScript proporciona una forma fluida de integrar nuestra superposición de pago en tu aplicación web con manejo de eventos en tiempo real y temas personalizables. Las características clave incluyen:
  - **Manejo de Eventos en Tiempo Real**: Escucha eventos de pago como creación, éxito y fallo
  - **Soporte para TypeScript**: APIs completamente tipadas para una mejor experiencia de desarrollador
  - **Temas Personalizables**: Soporte para temas claros y oscuros
  - **Compatibilidad con Navegadores**: Funciona en todos los navegadores modernos, incluyendo Chrome, Firefox, Safari, Edge e IE11+
  - **Cumplimiento PCI**: Mantiene los estándares de seguridad mientras proporciona una experiencia integrada

- **Nuevo Evento de Suscripción**: Se agregó soporte para el evento de webhook `subscription.plan_changed` que se activa cada vez que una suscripción es mejorada, degradada o modificada. Este evento proporciona:
  - Detalles completos de la suscripción después del cambio
  - Información del plan anterior y nuevo
  - Ajustes de facturación prorrateados
  - Fecha de facturación actualizada

  Este evento complementa nuestros eventos de suscripción existentes (`subscription.active`, `subscription.on_hold`, etc.) para proporcionar una visibilidad completa del ciclo de vida de la suscripción.

- **Mejoras en la Plataforma**: Varias optimizaciones de rendimiento, correcciones de errores y mejoras menores en toda la plataforma.
