## Nuevas Funciones 🚀

### Actualización Mayor de Webhooks

Obtén control, visibilidad y seguridad completos con nuestro sistema de webhooks completamente reinventado. Explora las nuevas capacidades a continuación.

<Steps>
{/* LOCKED_PATTERN_c69db7a67c22374bc38018f45df7cd33 */}
    - Crear, editar, pausar/reanudar y eliminar endpoints de webhook (URLs)
    - Seleccionar tipos de eventos para cada endpoint
    - Configurar encabezados personalizados por endpoint
    - Establecer límites de velocidad para cada endpoint
</Step>

{/* LOCKED_PATTERN_8cbecd2710208355be8248f15a8550e7 */}
    - Ver y rotar secretos de firma (claves HMAC) por endpoint
    - Regenerar secretos y ver las últimas marcas de tiempo de rotación
    - Copiar fragmentos de código o documentación para la verificación de firmas
</Step>

{/* LOCKED_PATTERN_dcecaa97684681ec209947fe21250620 */}
    - Ver registros de mensajes para cada endpoint (estado, marcas de tiempo)
    - Inspeccionar intentos individuales de mensajes con cuerpos de solicitud/respuesta y encabezados
    - Reproducir (reenviar) mensajes fallidos o pasados
    - Filtrar registros por tipo de evento, código de estado o rango de tiempo
</Step>

{/* LOCKED_PATTERN_f556e9022b1846586ac7518d86ce8f6f */}
    - Enviar eventos de prueba desde el portal usando cargas útiles simuladas
    - Inspeccionar esquemas de carga útil y ver los campos esperados
</Step>

{/* LOCKED_PATTERN_b9949e0bf83fcf83823944e5a02934ca */}
    - Rastrear estadísticas de entrega: recuentos de éxito/fallo, latencia y tendencias recientes de errores
    - Monitorizar el estado de salud de los endpoints (p. ej., fallando, saludable)
</Step>
</Steps>
