## Nuevas Funciones 🚀

### Actualización Mayor de Webhooks

Obtén control, visibilidad y seguridad completos con nuestro sistema de webhooks completamente reinventado. Explora las nuevas capacidades a continuación.

<Steps>
<Step title="Gestión de Endpoints y Suscripciones">
    - Crear, editar, pausar/reanudar y eliminar endpoints de webhook (URLs)
    - Seleccionar tipos de eventos para cada endpoint
    - Configurar encabezados personalizados por endpoint
    - Establecer límites de tasa para cada endpoint
</Step>

<Step title="Secretos y Seguridad">
    - Ver y rotar secretos de firma (claves HMAC) por endpoint
    - Regenerar secretos y ver las marcas de tiempo de la última rotación
    - Copiar fragmentos de código o documentación para la verificación de firmas
</Step>

<Step title="Historial de Entregas y Depuración">
    - Ver registros de mensajes para cada endpoint (estado, marcas de tiempo)
    - Inspeccionar intentos de mensajes individuales con cuerpos y encabezados de solicitud/respuesta
    - Reenviar (volver a enviar) mensajes fallidos o pasados
    - Filtrar registros por tipo de evento, código de estado o rango de tiempo
</Step>

<Step title="Pruebas y Solución de Problemas">
    - Enviar eventos de prueba desde el portal utilizando cargas útiles simuladas
    - Inspeccionar esquemas de carga útil y ver campos esperados
</Step>

<Step title="Monitoreo y Métricas">
    - Rastrear estadísticas de entrega: conteos de éxito/fallo, latencia y tendencias recientes de errores
    - Monitorear el estado de salud del endpoint (por ejemplo, fallando, saludable)
</Step>
</Steps>

<Check>
Ahora puedes gestionar, depurar y asegurar tus webhooks con mayor flexibilidad—todo desde el panel.
</Check>
