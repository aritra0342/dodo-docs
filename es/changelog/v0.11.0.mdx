## Nuevas funciones
- **Correos electrónicos automatizados**: Correos electrónicos automatizados añadidos para pagos, suscripciones, reembolsos y disputas para mantener informados a comerciantes y clientes.
- **Registro de eventos de webhook**: Registro de eventos de webhook añadido en el Panel de Dodo Payments para rastrear eventos de webhook y resolver problemas. Navega a `Settings > Webhooks > Webhook Logs` para acceder a los registros.
- **Guía de generación de claves de licencia**: Nueva guía añadida para implementar la generación de claves de licencia para productos digitales. Consulta la [guía](/features/license-keys) para obtener más información.

## Mejoras Menores y Correcciones de Errores
- Varias correcciones y mejoras menores.
