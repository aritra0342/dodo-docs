## Nuevas Funciones
- **Correos Electrónicos Automatizados**: Se añadieron correos electrónicos automatizados para pagos, suscripciones, reembolsos y disputas para mantener informados a comerciantes y clientes.
- **Registro de Eventos de Webhook**: Se añadió un registro de eventos de webhook en el Panel de Dodo Payments para rastrear eventos de webhook y solucionar problemas. Navega a `Settings > Webhooks > Webhook Logs` para acceder a los registros.
- **Guía de Generación de Claves de Licencia**: Se añadió una nueva guía para implementar la generación de claves de licencia para productos digitales. Consulta la [guía](/features/license-keys) para más información.

## Mejoras Menores y Correcciones de Errores
- Varias correcciones y mejoras menores.
