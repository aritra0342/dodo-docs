## Integración Fluida

- Seguimiento integral de la integración para monitorear el progreso del comerciante.
- Soporte para la reenvío de documentos y activación en modo en vivo tras la verificación.

## Gestión del Catálogo de Productos

- Creación, edición y eliminación fácil de productos de una sola vez y de suscripción.

## Enlace de Pago

- Generar enlaces de pago para productos de una sola vez y de suscripción.
- Incluye cantidades configurables y opciones de redirección para los viajes del cliente después del pago.

## Página de Pago Personalizable

- Experiencias de pago de marca con opciones de pago localizadas disponibles.
- Cálculo en tiempo real de impuestos y monto total mostrado en el pago.

## Generación Automática de Facturas

- Generación automática de facturas para pagos, reembolsos y pagos.

## Gestión de Reembolsos

- Gestión de reembolsos parciales y totales sin problemas desde el panel de control.
- Introducción de seguimiento de disputas para resoluciones efectivas.

## Pagos Transparentes

- Mecanismos de pago flexibles con políticas y tarifas transparentes.

## Análisis e Informes

- Panel de análisis avanzado para mostrar ingresos totales, desglose de transacciones, información del cliente y tipos de productos más vendidos.

## Generación de API y Webhooks

- Claves de API fáciles de generar para integrar Dodo Payments en tu sitio web o aplicación.
- Configuración de webhooks para notificaciones en tiempo real sobre transacciones, reembolsos y disputas.
