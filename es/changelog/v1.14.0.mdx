## Nuevas Funciones 🚀

- **Nuevos Métodos de Pago Europeos**: Hemos ampliado nuestras opciones de pago con cuatro nuevos métodos de pago europeos:
  - **iDEAL** - Países Bajos
  - **Bancontact** - Bélgica
  - **Przelewy24 (P24)** - Polonia
  - **EPS** - Austria

  <Frame>
    <img alt="Métodos de pago europeos compatibles" src="/images/transactions/payments/european.png" />
  </Frame>

  <Note>
    Estos métodos de pago están actualmente limitados a:
      * Solo moneda Euro (EUR)
      * Pagos únicos solamente (no disponibles para suscripciones)

    Para obtener información detallada sobre estos métodos de pago, visita nuestra documentación de [Payment Methods](/features/payment-methods).
  </Note>

- **Actualizaciones de Direcciones de Facturación de Suscripciones**: Se agregó la capacidad de actualizar direcciones de facturación para suscripciones a través de:
  * Interfaz del portal del cliente
  * Endpoint de API de suscripción de parche
