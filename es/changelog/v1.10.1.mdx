## Nuevas Funciones 🚀

- **Checkout 2.0**: Presentamos una experiencia de pago completamente rediseñada que incluye:
    - Interfaz moderna y simplificada para mejorar la experiencia del usuario
    - Nuevo Sistema de Recolección de Consultas que almacena parámetros en el almacenamiento de sesión con IDs únicos (por ejemplo, `sess_1a2b3c4d`)
    - Disponibilidad de datos persistentes a lo largo de todo el flujo de pago
    - ¡Más mejoras planeadas para futuras versiones!
  
    <Frame><img src="/images/changelog/checkout_2.png" alt="Checkout 2.0"/></Frame>

- **Métodos de Pago Configurables**: La nueva `allowed_payment_methods` configuración permite a los comerciantes personalizar las opciones de pago disponibles durante el proceso de pago. Consulta la documentación de [Métodos de Pago Permitidos](/features/payment-methods#configuring-payment-methods) para detalles de implementación.

- **Cálculo Dinámico de Impuestos**: Cálculo en tiempo real de las tasas de impuestos basado en las direcciones de facturación de los clientes para una fijación de precios más precisa en los enlaces de pago estáticos.

- **Suscripciones Bajo Demanda**: Nuevos endpoints de API para facturación de suscripciones flexible (disponible para comerciantes seleccionados). Aprende más en la guía de [Suscripciones Bajo Demanda](/features/subscription#enabling-on-demand-subscriptions).

- **Actualizaciones Adicionales**:
    - Optimización del rendimiento
    - Corrección de errores
    - Mejoras generales del sistema
