## Nuevas Funciones 🚀

- **Primer Pago de Suscripción**: ¡La función más solicitada está aquí! Ahora, habrá una única transacción para el primer pago, combinando el mandato y el monto real de la suscripción.

- **Método de Pago UPI QR**: Se agregó UPI QR como método de pago para el mercado indio, permitiendo a los clientes pagar utilizando códigos QR de UPI.
    
    <img src="/images/changelog/upi-qr.png" alt="Función UPI QR"/>

- **ID Fiscal en el Proceso de Pago**: Los clientes ahora pueden agregar su ID fiscal en la página de pago para fines de facturación e impuestos.

- **Reestructuración del Panel**: Todo el Panel de Dodo Payments ha sido reestructurado para ser más intuitivo y fácil de usar.

- **Guías de Incorporación Autogestionadas**: Los nuevos usuarios ahora pueden comenzar con Dodo Payments utilizando guías de incorporación autogestionadas.

- **Soporte Multilingüe**: Se agregó soporte para más de 14 idiomas en la página de pago, permitiendo a los clientes pagar en su idioma preferido. Los idiomas soportados incluyen:
    - English (en)
    - العربية (ar) // Árabe
    - Català (ca) // Catalán
    - 中文 (cn) // Chino
    - Deutsch (de) // Alemán
    - Español (es) // Español
    - Français (fr) // Francés
    - עברית (he) // Hebreo
    - Italiano (it) // Italiano
    - 日本語 (ja) // Japonés
    - Nederlands (nl) // Neerlandés
    - Polski (pl) // Polaco
    - Português (pt) // Portugués
    - Svenska (sv) // Sueco

- **Renovación del PDF de la Factura**: Los PDFs de las facturas han recibido un nuevo diseño, más claro y efectivo.

- **Widget de Ayuda Centralizado**: Se agregó un widget de Ayuda centralizado para agilizar y mejorar el soporte.
    
    <img src="/images/changelog/help-widget.png" alt="Widget de Ayuda"/>

## Mejoras y Correcciones de Errores 🔧

- **Limitación de Tasa**: Se agregó limitación de tasa en la API de facturas para prevenir abusos y asegurar la estabilidad del sistema.

- **Metadatos en Enlaces de Pago Estáticos**: Se agregó soporte para metadatos en enlaces de pago estáticos. Para más detalles, consulta la [documentación](/developer-resources/integration-guide#1-static-payment-links).

- **Mejoras y Correcciones Menores**: Varias mejoras y correcciones menores para mejorar el rendimiento general y la experiencia del usuario.
