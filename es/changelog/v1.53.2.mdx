## Nuevas Funciones 🚀

### Saldos de billetera de clientes
  Los saldos de billetera ahora se **aplican automáticamente a las facturas de suscripción recurrentes**, lo que facilita:

  - Recompensar a los clientes leales con fondos promocionales
  - Emitir compensaciones de servicio como saldo de billetera
  - Gestionar ajustes manuales sin interrumpir los ciclos de facturación

  <Tip>
  Utiliza los saldos de billetera de los clientes para:
  - Incentivos promocionales
  - Compensación por interrupciones del servicio
  - Saldos de cuentas prepagadas
  </Tip>

  Puedes **añadir o deducir fondos de la billetera de un cliente mediante la API**, lo que permite flujos de trabajo automatizados para soporte al cliente, retención o programas de fidelización.

  **Casos de uso de ejemplo:**
  - Financiar la billetera de un cliente después de una recomendación exitosa
  - Deductar saldo de billetera como parte de una ruta de actualización personalizada

  Explora las nuevas APIs de billetera de cliente y las actualizadas:  
  - [Obtener billeteras de clientes](/api-reference/customers/get-customer-wallets)  
  - [Listar entradas del libro mayor de la billetera de clientes](/api-reference/customers/get-customer-wallets-ledger-entries)  
  - [Crear entrada del libro mayor de billetera de cliente para añadir y deducir fondos](/api-reference/customers/post-customer-wallets-ledger-entries)

   <Info>
  La aplicación del saldo de la billetera durante el Checkout estará disponible en el próximo lanzamiento, lo que permitirá a los clientes canjear sus fondos de billetera directamente en el punto de compra.
  </Info>

## Mejoras y Actualizaciones ✨

- **Detalles del cliente en la API de Obtener Reembolso**  
  El punto final de recuperación de reembolsos ahora devuelve información del cliente asociada, agilizando la conciliación y los flujos de trabajo de soporte.  
  Aprende más: [Obtener API de Reembolso](/api-reference/refunds/get-refunds-1)

- **Límite de medidor aumentado a 50**  
  Ahora puedes crear hasta 50 medidores por cuenta, lo que permite un seguimiento de uso más granular a través de tus productos y características.  
  Aprende más: [Obtener API de Medidores](/api-reference/meters/get-meters)

- **Contexto de producto y cliente en la API de Activar Clave de Licencia**  
  Las respuestas de activación de licencia ahora incluyen contexto detallado de producto y cliente, mejorando las verificaciones de derechos y las integraciones posteriores.  
  Aprende más: [Activar API de Clave de Licencia](/api-reference/licenses/activate-license)

- **Las suscripciones se pueden degradar a \$0**  
  Ahora puedes degradar suscripciones a planes gratuitos o de $0, facilitando el soporte para pruebas, niveles comunitarios y patrocinios.  
  Gestiona suscripciones: [Actualizar API de Suscripción](/api-reference/subscriptions/patch-subscriptions)
