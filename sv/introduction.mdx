## Vad är Dodo Payments?

Dodo Payments är den **allt-i-ett-motor** för att lansera, skala och tjäna pengar globalt. Designad för den nya generationen av digitala SaaS- och AI-produkter, hanterar vi allt från **betalningar till fakturering till distribution** - allt utan extra ingenjörsarbete.

<Info>
**Byggd för SaaS och AI-produkter**: Hantera engångsbetalningar, prenumerationer, krediter, platser, tokens eller vilken användningsmetrik som helst med vår optimerade faktureringsinfrastruktur.
</Info>

<Tip>
  <strong>Möt Sentra: Din AI-drivna faktureringsassistent</strong><br/>
  Sentra är en AI-agent som integrerar, analyserar och agerar på fakturering och betalningar direkt i din IDE. Ställ frågor, bygg integrationer eller planera faktureringsarkitekturer - allt genom att beskriva vad du vill.
  <a href="/developer-resources/sentra">
    Kom igång med Sentra &rarr;
  </a>
</Tip>

## Snabbstartsguide

Kom igång med Dodo Payments på bara några steg:

<Steps>
<Step title="Skapa ditt konto">
  Registrera dig på [app.dodopayments.com](https://app.dodopayments.com/signup) med din e-postadress.

  <Check>
  Du kommer att få ett bekräftelsemail för att verifiera ditt konto.
  </Check>
</Step>

<Step title="Lägg till din första produkt">
  Välj din faktureringsmodell och lägg till din produkt för att börja acceptera betalningar.

  <CardGroup cols={3}>
  <Card title="Engångsbetalning" icon="box" href="/features/one-time-payment-products">
    Perfekt för digitala nedladdningar, kurser eller engångsköp
  </Card>
  
  <Card title="Prenumerationsprodukt" icon="repeat" href="/features/subscription">

    Fakturera kunder för API-anrop, krediter, tokens eller vilken användningsmetrik som helst - idealisk för AI och SaaS
  </Card>
  </CardGroup>
</Step>

<Step title="Välj din integrationsmetod">
  Välj den integrationsmetod som passar dina behov och tekniska krav bäst.

  <Tabs>
  <Tab title="No-Code (Snabbast)">

  <Tab title="Overlay Checkout">
    **Bäst för sömlösa, varumärkesanpassade betalningar:**

### MCP-servrar för AI-assistenter

Anslut din AI-kodassistent (Cursor, Claude Desktop, Windsurf, VS Code) direkt till Dodo Payments med våra MCP-servrar.

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  Semantisk sökning över Dodo Payments-dokumentation - få omedelbara svar om funktioner, API:er och bästa praxis
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  Utför API-operationer direkt - skapa betalningar, hantera prenumerationer och hantera återbetalningar genom din AI-assistent
</Card>
</CardGroup>

<Accordion title="Snabb MCP-installation för Cursor/Claude Desktop">
Lägg till båda servrarna i din MCP-konfiguration (`~/.cursor/mcp.json` eller Claude Desktop-konfiguration):

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
Kräver Node.js 18+. Se [MCP Server-dokumentationen](/developer-resources/mcp-server) för detaljerade installationsinstruktioner för alla stödda klienter.
</Info>
</Accordion>
