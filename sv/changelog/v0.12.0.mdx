## Nya Funktioner
- **Officiella SDK:er**: Introducerade officiella Dodo Payments SDK:er för **Node.js**, **Python** och **Go**. Se sidan [SDK:er](/developer-resources/dodo-payments-sdks) för mer information.
- **Metadatafält**: Lagt till ett `Metadata` fält för betalningar och prenumerationer för att lagra ytterligare information. Se sidan [Metadata](/api-reference/metadata) för mer detaljer.
- **Kund-API:er**: Utvidgade kund-API:er för att inkludera följande slutpunkter. Du kan nu skicka `customer_id` i betalningar och prenumerationer för att förhindra kundduplicering:
    - **Skapa Kund**: [/api-reference/customers/create-customer](/api-reference/customers/create-customer)
    - **Patcha Kund**: [/api-reference/customers/patch-customer](/api-reference/customers/patch-customer)
