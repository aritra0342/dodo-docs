## Nya Funktioner 🚀

- **Första Betalningen för Prenumerationer**: Den mest efterfrågade funktionen är här! Nu kommer det att finnas en enda transaktion för den första betalningen, som kombinerar mandatet och det faktiska prenumerationsbeloppet.

- **UPI QR Betalningsmetod**: Lagt till UPI QR som en betalningsmetod för den indiska marknaden, vilket gör att kunder kan betala med UPI QR-koder.
    
    <img src="/images/changelog/upi-qr.png" alt="UPI QR Funktion"/>

- **Skatte-ID vid Kassan**: Kunder kan nu lägga till sitt skatte-ID på kassasidan för fakturering och skatteändamål.

- **Omstrukturering av Instrumentpanelen**: Hela Dodo Payments-instrumentpanelen har omstrukturerats för att bli mer intuitiv och användarvänlig.

- **Självbetjäningsguider för Onboarding**: Nya användare kan nu komma igång med Dodo Payments med hjälp av självbetjäningsguider för onboarding.

- **Fler språk**: Lagt till stöd för 14+ språk på kassan, vilket gör det möjligt för kunder att betala på sitt föredragna språk. Stödda språk inkluderar:
    - English (en)
    - العربية (ar) // Arabic
    - Català (ca) // Catalan
    - 中文 (cn) // Chinese
    - Deutsch (de) // German
    - Español (es) // Spanish
    - Français (fr) // French
    - עברית (he) // Hebrew
    - Italiano (it) // Italian
    - 日本語 (ja) // Japanese
    - Nederlands (nl) // Dutch
    - Polski (pl) // Polish
    - Português (pt) // Portuguese
    - Svenska (sv) // Swedish

- **Makeover av Faktura-PDF**: Faktura-PDF:erna har fått en ny, tydligare och mer effektiv design.

- **Centraliserad Hjälpwidget**: Lagt till en centraliserad hjälpwidget för att effektivisera och förbättra supporten.
    
    <img src="/images/changelog/help-widget.png" alt="Hjälpwidget"/>

## Förbättringar och Bugfixar 🔧

- **Hastighetsbegränsning**: Lagt till hastighetsbegränsning i faktura-API:t för att förhindra missbruk och säkerställa systemstabilitet.

- **Metadata i Statisk Betalningslänkar**: Lagt till stöd för metadata i statiska betalningslänkar. För mer information, se [dokumentationen](/developer-resources/integration-guide#1-static-payment-links).

- **Mindre Förbättringar och Fixar**: Olika mindre förbättringar och fixar för att förbättra den övergripande prestandan och användarupplevelsen.
