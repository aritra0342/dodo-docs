## Nya Funktioner 🚀

### Checkout Sessions API
Vi presenterar vår nya **Checkout Sessions API** - en kraftfull, hostad checkout-lösning som erbjuder ett komplett betalningsflöde för både engångsköp och prenumerationer. Denna nya API ersätter den tidigare Dynamic Links-metoden med förbättrad flexibilitet och användarupplevelse.

**Nyckelfördelar:**
- **Förenklad Integration**: Ingen behov av att ange kompletta faktureringsadresser i förväg - vi hanterar saknade detaljer automatiskt
- **Förbättrad Anpassning**: Full kontroll över betalningsmetoder, valutor, teman och funktionsflaggor
- **Bättre UX**: Förifyll kundinformation för att minska friktion vid checkout och förbättra konverteringsgraden
- **Flexibel Konfiguration**: Stöd för multiproduktvagnar, prenumerationstester, rabattkoder och mer

**Vad har ändrats:**
- **Före**: Krävde komplett faktureringsadressinformation vid skapande av betalningslänkar
- **Nu**: Skicka vilken kundinformation du har tillgänglig - vi samlar in resten under checkout
- **Migration**: Enkel API-endpointändring utan ytterligare hantering krävs
