## Nya Funktioner 🚀

- **API för Ändring av Prenumerationsplan**: Lanserade ett nytt API som möjliggör sömlösa uppgraderingar eller nedgraderingar av prenumerationer mellan olika produktplaner. För implementationsdetaljer, se [API-referensen](/api-reference/subscriptions/change-plan).

- **Förbättrad Onboarding-Vägledning**: Förbättrade sidofältspåminnelser för att ge tydligare riktning om nästa steg i din Dodo Payments onboarding-resa.

  <Frame>
    <img alt="Sidofältspåminnelser" src="/images/changelog/sidebar-nudges.png" />
  </Frame>

- **Centrum för Meddelandeinställningar**: Lagt till en dedikerad sektion för att hantera dina meddelandeinställningar. Åtkomst genom att klicka på klockikonen i det övre högra hörnet och sedan välja inställningsikonen.

  <Frame>
    <img alt="Meddelandeinställningar" src="/images/changelog/notif-pref.png" />
  </Frame>

- **Förbättrad Adaptiv Valutainformation**: Lagt till nya fält i betalningssvaret inklusive `settlement_amount`, `settlement_currency`, och `settlement_tax` för att ge större transparens kring avräkningsdetaljer.

- **Allmänna Förbättringar**: Olika mindre förbättringar och buggfixar för att förbättra den övergripande plattformsprestandan.
