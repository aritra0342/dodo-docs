## Nya funktioner

### 1. **Kommunikationsinställningar**

   Ta kontroll över hur du tar emot aviseringar med våra omarbetade kommunikationsinställningar. Konfigurera e-post-, push- och in-app-aviseringar så att de passar ditt arbetsflöde.

   <Frame>
     <img src="/images/communication-preferences/overview.png" alt="Communication Preferences overview showing notification settings" />
   </Frame>

   **Huvudfunktioner**

   - **Omarbetade e-postmallar**: Fräscha, moderna e-postmallar med tydliga uppmaningar och bättre läsbarhet
   - **Push-aviseringar**: Få omedelbara notiser på din mobil vid kritiska händelser som nya försäljningar, tvister och utbetalningar
   - **Detaljerade kontroller**: Välj exakt vilka händelser som utlöser aviseringar och genom vilka kanaler
   - **Support för flera e-postadresser**: Lägg till flera e-postadresser för att ta emot aviseringar
   - **Kundens e-postinställningar**: Konfigurera vilka e-postmeddelanden dina kunder får

   **Aviseringskategorier**

   | Category | Description |
   |----------|-------------|
   | **Sales & Payments** | Nya försäljningar, lyckade betalningar, misslyckade betalningar |
   | **Payouts & Refunds** | Slutförda utbetalningar, återbetalningsförfrågningar, färdiga återbetalningar |
   | **Subscriptions** | Nya prenumerationer, förnyelser, avbokningar, ändringar av plan |
   | **Disputes** | Tvist öppnad, bevis krävs, tvist löst |

   <Check>
   Resultat: Håll dig informerad om ditt företag med aviseringar levererade precis hur och när du vill ha dem.
   </Check>

   Läs mer: [Guide till kommunikationsinställningar](/features/communication-preferences)

### 2. **Anpassade fält i kassasessioner**

   Samla in ytterligare information från kunder under kassan med anpassade fält. Lägg till textfält, rullgardiner, datum med mera för att få den data du behöver.

   **Stödda fälttyper**

   | Type | Description | Use Case |
   |------|-------------|----------|
   | `text` | Enradigt textfält | Företagsnamn, orderanteckningar |
   | `number` | Numeriskt fält | Kvantitet, ålder |
   | `email` | E-postfält med validering | Fakturakontaktens e-post |
   | `url` | URL-fält med validering | Webbplats, social profil |
   | `date` | Datumväljare | Leveransdatum, evenemangsdatum |
   | `dropdown` | Val från fördefinierade alternativ | Storlek, färg, plantyp |
   | `boolean` | Kryssruta för ja/nej-värden | Godkännande av villkor, nyhetsbrevssamtycke |

   **Så fungerar det**

   ```typescript
   const session = await client.checkoutSessions.create({
     product_cart: [{ product_id: 'prod_123', quantity: 1 }],
     custom_fields: [
       {
         name: 'company_name',
         label: 'Company Name',
         type: 'text',
         required: true
       },
       {
         name: 'team_size',
         label: 'Team Size',
         type: 'dropdown',
         options: ['1-10', '11-50', '51-200', '200+'],
         required: true
       },
       {
         name: 'newsletter',
         label: 'Subscribe to our newsletter',
         type: 'boolean',
         required: false
       }
     ],
     return_url: 'https://yoursite.com/success'
   });
   ```

   **Fördelar**

   - **Flexibel datainsamling**: Samla in all information som behövs för leverans eller personalisering
   - **Inbyggd validering**: E-post- och URL-fält validerar automatiskt inmatningsformatet
   - **Kontroll mellan obligatoriskt och valfritt**: Markera fält som obligatoriska eller valfria beroende på dina behov
   - **Webhook-leverans**: Värden från anpassade fält ingår i betalningswebhook-händelser

   <Info>
   Värden från anpassade fält returneras i `custom_fields`-objektet i svar för betalningar och kassasessioner.
   </Info>

   Läs mer: [Anpassade fält i kassasessioner](/developer-resources/checkout-session#custom-fields)

### 3. **Utökat språksupport för kassan**

   Kassasidan stöder nu **21 språk**, upp från 16. Nå fler kunder på deras modersmål för en bättre kassaupplevelse.

   **Nya språk som lagts till**

   | Language | Code |
   |----------|------|
   | Koreanska | `ko` |
   | Malajiska | `ms` |
   | Rumänska | `ro` |
   | Ryska | `ru` |
   | Thailändska | `th` |

   **Alla stödda språk**

   Arabiska, Kinesiska, Nederländska, Engelska, Franska, Tyska, Hindi, Indonesiska, Italienska, Japanska, Koreanska, Malajiska, Portugisiska, Rumänska, Ryska, Spanska, Svenska, Thailändska, Turkiska, Ukrainska, Vietnamesiska.

   **Tvinga ett specifikt språk**

   Använd `force_language`-parametern för att åsidosätta automatisk språkdetektion:

   ```typescript
   const session = await client.checkoutSessions.create({
     product_cart: [{ product_id: 'prod_123', quantity: 1 }],
     force_language: 'ko', // Force Korean language
     return_url: 'https://yoursite.com/success'
   });
   ```

   <Tip>
   Som standard identifierar kassasidan automatiskt kundens föredragna språk via webbläsarinställningarna. Använd `force_language` när du vill säkerställa konsekvens med din webbplats språk.
   </Tip>

   Läs mer: [Språkstöd för kassan](/features/checkout#supported-languages)
