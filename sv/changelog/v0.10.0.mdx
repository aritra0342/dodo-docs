## Nya Funktioner
- **Öppen Källkod Atlas**: En demotillämpning skriven i Next.js för att visa integrationen av Dodo Payments. Utforska [GitHub-repositoriet](https://github.com/dodopayments/dodo-checkout-demo) och dokumentationen [här](/developer-resources/integration-guide).
- **Funktion-Förfrågningssektion**: Introducerade en funktion-förfrågningssektion på Dodo Payments-instrumentpanelen för att prioritera produktens färdplan baserat på användarfeedback.
- **Webhook-Förfrågningsloggar API**: Lagt till API:er för att lista och detaljera webhook-förfrågningsloggar.
- **Statisk Betalningslänkar**: Lagt till fler frågeparametrar till de statiska betalningslänkarna för att ge mer flexibilitet i kassa-processen. För mer information, klicka [här](/developer-resources/integration-guide#1-static-payment-links).
- **Prenumerationslista API**: Lagt till ett API för att hämta alla prenumerationer. Se [API-referensen](/api-reference/subscriptions/get-subscriptions-1) för detaljer.

## Mindre Förbättringar och Buggfixar
- Förbättrade svaren från Produkt- och Prenumerations-API:erna med ytterligare parametrar.
- Implementerade hastighetsbegränsning för API-förfrågningar, för närvarande inställd på 20 förfrågningar per sekund.
- Förbättrade dokumentationen för att vara mer användarvänlig och lättare att navigera.
- Olika mindre buggfixar och förbättringar.
