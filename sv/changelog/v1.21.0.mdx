## Nya Funktioner 🚀

- **Prenumerationstillägg**: Introducerade stöd för tillägg till prenumerationsprodukter. I avsnittet Produkter hittar du en ny flik för Tillägg där du kan skapa tillägg som "Platser". När de har skapats kan du redigera dina prenumerationsprodukter för att inkludera dessa tillägg. Denna funktion är tillgänglig både på instrumentpanelen och via API.
  - [Skapa Tillägg](/api-reference/addons/create-addon)
  - [Lista Tillägg](/api-reference/addons/list-addons)
  - [Hämta Tillägg](/api-reference/addons/get-addon)
  - [Uppdatera Tillägg](/api-reference/addons/update-addon)
  - [Uppdatera Tilläggsbilder](/api-reference/addons/update-addon-images)

  Med tillägg låser du upp nya faktureringsalternativ som platsbaserad fakturering eller variabel fakturering per kund. Var kreativ med tillägg för att utforma flexibla faktureringsmodeller för ditt företag.

    <img alt="Tillägg" src="/images/changelog/addons.png" />
  </Frame>

- **Förbättrad Tvist Hantering**: Förbättrad hantering av tvister för att vara mer robust och enklare att hantera genom din Dodo Payments-instrumentpanel.

- **Utökade Svardata**: Lade till kundfakturadetaljer i betalningssvar, prenumerationssvar och webhooks.

- **Allmänna Förbättringar**: Olika mindre förbättringar och buggfixar för att förbättra den övergripande plattformsprestandan.
