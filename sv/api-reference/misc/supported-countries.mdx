---
title: Stödda länder
openapi: get /checkout/supported_countries
description: Hämta en lista över länder som stöds av Dodo Payments för dina kunder.
---

<Tip> Du kommer att få en lista över [ISO Alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) landskoder som representerar de länder vi **för närvarande stöder för betalningar**. Du kan se hela listan över stödda länder [här](/miscellaneous/list-of-countries-we-accept-payments-from). Denna lista uppdateras regelbundet när vi utökar vår täckning. </Tip>

<Tip> Obs: De tillgängliga alternativen inkluderar ISO Alpha-2-koder för **alla länder**, inte bara de vi för närvarande stöder. Vänligen hänvisa till listan över stödda länder för att bekräfta tillgänglighet. </Tip>
