---
title: Stödda länder
openapi: get /checkout/supported_countries
description: Hämta en lista över länder som stöds av Dodo Payments för dina kunder.
---
<Tip> Du kommer att få en lista över [ISO Alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) landskoder som representerar de länder vi **för närvarande stöder för betalningar**. Du kan se hela listan över stödjade länder [här](/miscellaneous/list-of-countries-we-accept-payments-from). Denna lista uppdateras regelbundet i takt med att vi utökar vår täckning. </Tip>

<Tip> Obs: De tillgängliga alternativen inkluderar ISO Alpha-2-koder för **alla länder**, inte bara dem vi för närvarande stödjer. Vänligen hänvisa till listan över stödjade länder för att bekräfta tillgängligheten. </Tip>
