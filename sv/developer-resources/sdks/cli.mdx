## Installation

### Go Install

```bash
go install github.com/dodopayments/dodopayments-cli/cmd/dodopayments@latest
```

### Från Källkod

```bash
git clone https://github.com/dodopayments/dodopayments-cli.git
cd dodopayments-cli
go install ./cmd/dodopayments
```

## Snabbstart

Ställ in din API-nyckel:

```bash
export DODO_PAYMENTS_API_KEY=your_api_key_here
```

Grundläggande kommandostruktur:

```bash
dodopayments [resource] [command] [flags]
```

Skapa en checkout-session:

```bash
dodopayments checkout-sessions create \
  --product-cart.product_id prod_123 \
  --product-cart.quantity 1 \
  --return-url https://yourdomain.com/return
```

## Globala Flaggor

| Flagga | Beskrivning |
|------|-------------|
| `--format` | Utdataformat: `auto`, `json`, `yaml`, `pretty`, `explore` |
| `--transform` | Filtrera svar med gjson-syntax |
| `--debug` | Aktivera detaljerad loggning |
| `--version`, `-v` | Visa CLI-version |

## Vanliga Kommandon

### Betalningar

```bash
# List payments
dodopayments payments list --limit 20 --format json

# Retrieve payment
dodopayments payments retrieve pay_123

# Get line items
dodopayments payments line-items pay_123
```

### Kunder

```bash
# Create customer
dodopayments customers create \
  --email customer@example.com \
  --name "John Doe"

# List customers
dodopayments customers list

# Update customer
dodopayments customers update cus_123 --name "Jane Doe"
```

### Prenumerationer

```bash
# Create subscription
dodopayments subscriptions create \
  --customer-id cus_123 \
  --product-id prod_456

# List subscriptions
dodopayments subscriptions list

# Get usage history
dodopayments subscriptions usage-history sub_123
```

### Produkter

```bash
# Create product
dodopayments products create \
  --name "Premium Plan" \
  --price.amount 2999 \
  --price.currency USD

# List products
dodopayments products list
```

## Användningsbaserad Fakturering

### Ingest Händelser

```bash
dodopayments usage-events ingest \
  --events.event_id api_call_12345 \
  --events.customer_id cus_abc123 \
  --events.event_name api_request \
  --events.timestamp 2024-01-15T10:30:00Z \
  --events.metadata.endpoint /api/v1/users
```

### Hantera Mätare

```bash
# Create meter
dodopayments meters create \
  --name "API Requests" \
  --event-name api_request \
  --aggregation count

# List meters
dodopayments meters list
```

## Utdataformat

### JSON (för skripting)

```bash
dodopayments payments list --format json | jq '.data[] | {id, amount}'
```

### Fin (för läsning)

```bash
dodopayments payments retrieve pay_123 --format pretty
```

### Interaktiv Utforskning

```bash
dodopayments payments list --format explore
```

### Transformera med gjson

```bash
# Extract all payment IDs
dodopayments payments list --transform "data.#.id"

# Filter and extract
dodopayments payments list --transform "data.#(amount>5000)#.id"
```

<Tip>
Använd `--format json` med `jq` för kraftfull databehandling i shell-skript.
</Tip>

## Skriptexempel

```bash
#!/bin/bash

# Export all payments to CSV
dodopayments payments list --format json | \
  jq -r '.data[] | [.id, .amount, .currency, .status] | @csv' > payments.csv

# Create customers from file
while IFS=',' read -r email name; do
  dodopayments customers create \
    --email "$email" \
    --name "$name" \
    --format json
done < customers.csv
```

## Shell Komplettering

Aktivera automatisk komplettering:

```bash
# Bash
eval "$(dodopayments completion bash)"

# Zsh
eval "$(dodopayments completion zsh)"

# Fish
dodopayments completion fish | source
```

## Tillgängliga Resurser

CLI tillhandahåller kommandon för 24 resurskategorier:

- `checkout-sessions` - Checkout-sessionhantering
- `payments` - Betalningsoperationer
- `subscriptions` - Återkommande fakturering
- `customers` - Kundkonton
- `products` - Produktkatalog
- `licenses` - Programvarulicenser
- `refunds` - Betalningsåterbetalningar
- `disputes` - Betalningsdispyter
- `webhooks` - Event-webhooks
- `meters` - Användningsmätare
- `usage-events` - Användningstracking
- `discounts` - Rabattkoder
- `payouts` - Säljarpayouts
- `brands` - Multi-brand-hantering
- Plus 10 ytterligare resurser

<Info>
Använd `dodopayments --help` för att se alla tillgängliga resurser och `dodopayments [resource] --help` för resurs-specifika kommandon.
</Info>
