## Översikt

Agentkunskaper är återanvändbara kapaciteter som förbättrar vad AI-kodassistenter kan göra när de arbetar med Dodo Payments. De tillhandahåller procedurkunskap och bästa praxis som hjälper agenter att utföra specifika uppgifter mer effektivt – från att integrera kassa-flöden till att konfigurera användningsbaserad fakturering.

<Info>
**Tänk på skills som tilläggsprogram för din AI-assistent.** De lär din agent hur man implementerar Dodo Payments-funktioner korrekt, enligt våra rekommenderade mönster och bästa praxis.
</Info>

Skills fungerar med alla MCP-kompatibla AI-agenter, inklusive:
- **Claude Code** – Anthropics CLI-kodassistent
- **OpenCode** – Öppen källkod AI-kodagent
- **Cursor** – AI-fokuserad kodredigerare
- **Other MCP clients** – Alla agenter som stöder skills-protokollet

## Installation

Välj din föredragna installationsmetod beroende på din AI-kodassistent.

<Tabs>
<Tab title="Skills CLI">
Installera skills med den universella skills-CLI:n som fungerar med alla MCP-kompatibla agenter:

```bash
# Install all Dodo Payments skills
npx skills add dodopayments/skills
```

Eller installera enskilda skills vid behov:

```bash
# Best practices guide
npx skills add dodopayments/skills/dodo-best-practices

# Webhook integration
npx skills add dodopayments/skills/webhook-integration

# Subscription billing
npx skills add dodopayments/skills/subscription-integration

# Checkout flows
npx skills add dodopayments/skills/checkout-integration

# Usage-based billing
npx skills add dodopayments/skills/usage-based-billing

# BillingSDK components
npx skills add dodopayments/skills/billing-sdk

# License key management
npx skills add dodopayments/skills/license-keys
```

<Tip>
Installera endast de skills du behöver för att hålla din agent fokuserad och minska kontextöverbörd.
</Tip>
</Tab>

<Tab title="Claude Code">
Lägg till Dodo Payments skills-marknadsplats i Claude Code:

```bash
/plugin marketplace add dodopayments/skills
```

Installera sedan enskilda plugins:

```bash
# Install best practices
/plugin install dodo-best-practices

# Install webhook integration
/plugin install webhook-integration

# Install subscription integration
/plugin install subscription-integration

# Install checkout integration
/plugin install checkout-integration

# Install usage-based billing
/plugin install usage-based-billing

# Install BillingSDK
/plugin install billing-sdk

# Install license keys
/plugin install license-keys
```

</Tab>

<Tab title="OpenCode">
Skills blir automatiskt tillgängliga när de är konfigurerade i dina OpenCode-inställningar. Lägg till Dodo Payments skills-repositoriet i din konfiguration:

```json
{
  "skills": {
    "repositories": [
      "dodopayments/skills"
    ]
  }
}
```

</Tab>
</Tabs>

## Använda skills

När de väl är installerade kan din AI-assistent utnyttja dessa skills när du ber den implementera Dodo Payments-funktioner. Här är några exempel på uppmaningar:

### Bästa praxis

```
Help me integrate Dodo Payments into my Next.js app following best practices
```

### Webhooks

```
Set up webhook handlers for payment and subscription events
```

### Prenumerationer

```
Implement a subscription flow with free trial and multiple pricing tiers
```

### Kassa

```
Create a checkout session for my SaaS product
```

### Användningsbaserad fakturering

```
Add usage-based billing to track API calls for my AI product
```

### BillingSDK

```
Add a pricing page using BillingSDK components
```

### Licensnycklar

```
Implement license key validation for my desktop app
```

## Hur skills fungerar

Skills ger din AI-assistent:

1. **Procedurkunskap** – Steg-för-steg-instruktioner för att implementera funktioner
2. **Kodmallar** – Produktionsklara kodmönster och snuttar
3. **Bästa praxis** – Rekommenderade tillvägagångssätt för säkerhet, felhantering och UX
4. **Kontextmedvetenhet** – Förståelse för hur olika Dodo Payments-funktioner samverkar

När du ber din AI-assistent implementera en funktion använder den relevant skill för att:
- Förstå rätt implementationsstrategi
- Generera korrekt, fungerande kod
- Följa Dodo Payments-konventioner
- Hantera edge cases och fel på rätt sätt

<Note>
Skills kompletterar [MCP Server](/developer-resources/mcp-server) och [Sentra](/developer-resources/sentra). Använd MCP för API-åtgärder, Sentra för vägledd integration och Skills för utökad kapacitet i AI-assistenter.
</Note>
