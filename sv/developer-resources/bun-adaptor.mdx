## Installation

<Steps>
<Step title="Install the package">
Kör följande kommando i projektets rotmapp:

```bash
bun add @dodopayments/bun
```

</Step>
<Step title="Set up environment variables">
Skapa en <code>.env</code>-fil i projektets rotmapp:

```env expandable
DODO_PAYMENTS_API_KEY=your-api-key
DODO_PAYMENTS_WEBHOOK_KEY=your-webhook-secret
DODO_PAYMENTS_ENVIRONMENT="test_mode" or "live_mode"
DODO_PAYMENTS_RETURN_URL=your-return-url
```

<Warning>
Lägg aldrig upp din <code>.env</code>-fil eller hemligheter i versionshantering.
</Warning>
</Step>
</Steps>

## Exempel på route-hanterare

<Info>
Alla exempel antar att du använder Buns inbyggda server med <code>Bun.serve()</code>.
</Info>

<Tabs>
<Tab title="Checkout Handler">
  <Info>
  Använd denna hanterare för att integrera Dodo Payments-kassan i din Bun-server. Stödjer statiska (GET), dynamiska (POST) och sessionbaserade (POST) betalflöden.
  </Info>
  <CodeGroup>

    ```typescript Bun Server Handler expandable
    import { Checkout } from '@dodopayments/bun';

    const staticCheckoutHandler = Checkout({
        bearerToken: process.env.DODO_PAYMENTS_API_KEY,
        returnUrl: process.env.DODO_PAYMENTS_RETURN_URL,
        environment: process.env.DODO_PAYMENTS_ENVIRONMENT,
        type: "static"
    });

    const sessionCheckoutHandler = Checkout({
        bearerToken: process.env.DODO_PAYMENTS_API_KEY,
        returnUrl: process.env.DODO_PAYMENTS_RETURN_URL,
        environment: process.env.DODO_PAYMENTS_ENVIRONMENT,
        type: "session"
    });

    const dynamicCheckoutHandler = Checkout({
        bearerToken: process.env.DODO_PAYMENTS_API_KEY,
        returnUrl: process.env.DODO_PAYMENTS_RETURN_URL,
        environment: process.env.DODO_PAYMENTS_ENVIRONMENT,
        type: "dynamic"
    });

    Bun.serve({
      port: 3000,
      fetch(request) {
        const url = new URL(request.url);
        
        if (url.pathname === "/api/checkout") {
          if (request.method === "GET") {
            return staticCheckoutHandler(request);
          }
          if (request.method === "POST") {
            return sessionCheckoutHandler(request);
            // or return dynamicCheckoutHandler(request);
          }
        }
        
        return new Response("Not Found", { status: 404 });
      },
    });
    ```

  </CodeGroup>

## Checkout Route Handler

<Info>
Dodo Payments stöder tre typer av betalflöden för att integrera betalningar på din webbplats; denna adaptor stöder alla typer.
</Info>

- **Statisk betalning:** Direkt delbara URL:er för snabb, kodfri betalningsinsamling.
- **Dynamisk betalning:** Generera betalningslänkar programmatisk med anpassade detaljer via API eller SDK:er.
- **Checkout-sessioner:** Skapa säkra, anpassningsbara kassaupplevelser med förkonfigurerade produktkorgar och kunduppgifter.

## Kundportalens route-hanterare

Customer Portal Route Handler gör det möjligt att sömlöst integrera Dodo Payments kundportal i din Bun-serverapplikation.

### Frågeparametrar
<ParamField query="customer_id" type="string" required>
Kund-ID för portal-sessionen (t.ex. <code>?customer_id=cus_123</code>).
</ParamField>
<ParamField query="send_email" type="boolean">
Om satt till <code>true</code> skickas ett e-postmeddelande till kunden med portalens länk.
</ParamField>

<Warning>
Returnerar 400 om <code>customer_id</code> saknas.
</Warning>

## Webhook Route Handler

- **Metod:** Endast POST-förfrågningar stöds. Andra metoder returnerar 405.
- **Signaturverifiering:** Verifierar webhook-signaturen med <code>webhookKey</code>. Returnerar 401 om verifieringen misslyckas.
- **Payload-validering:** Valideras med Zod. Returnerar 400 för ogiltiga payloads.
- **Felhäntering:**
  - 401: Ogiltig signatur
  - 400: Ogiltig payload
  - 500: Internt fel vid verifiering
- **Händelsehantering:** Anropar rätt händelsehanterare baserat på payload-typen.

typescript
import { Checkout, CustomerPortal, Webhooks } from "@dodopayments/bun";

const staticCheckoutHandler = Checkout({
  bearerToken: process.env.DODO_PAYMENTS_API_KEY,
  returnUrl: process.env.DODO_PAYMENTS_RETURN_URL,
  environment: process.env.DODO_PAYMENTS_ENVIRONMENT,
  type: "static",
});

const sessionCheckoutHandler = Checkout({
  bearerToken: process.env.DODO_PAYMENTS_API_KEY,
  returnUrl: process.env.DODO_PAYMENTS_RETURN_URL,
  environment: process.env.DODO_PAYMENTS_ENVIRONMENT,
  type: "session",
});

const customerPortalHandler = CustomerPortal({
  bearerToken: process.env.DODO_PAYMENTS_API_KEY,
  environment: process.env.DODO_PAYMENTS_ENVIRONMENT,
});

const webhookHandler = Webhooks({
  webhookKey: process.env.DODO_PAYMENTS_WEBHOOK_KEY,
  onPaymentSucceeded: async (payload) => {
    console.log("Payment succeeded:", payload);
    // Your business logic here
  },
  onSubscriptionActive: async (payload) => {
    console.log("Subscription activated:", payload);
    // Your business logic here
  },
});

Bun.serve({
  port: 3000,
  fetch(request) {
    const url = new URL(request.url);
    
    // Checkout routes
    if (url.pathname === "/api/checkout") {
      if (request.method === "GET") {
        return staticCheckoutHandler(request);
      }
      if (request.method === "POST") {
        return sessionCheckoutHandler(request);
      }
    }
    
    // Customer portal route
    if (url.pathname === "/api/customer-portal" && request.method === "GET") {
      return customerPortalHandler(request);
    }
    
    // Webhook route
    if (url.pathname === "/api/webhook/dodo-payments" && request.method === "POST") {
      return webhookHandler(request);
    }
    
    return new Response("Not Found", { status: 404 });
  },
});

console.log("Server running on http://localhost:3000");
