## Vad är tilläggsprodukter?

Tilläggsprodukter är kompletterande produkter som kunder kan köpa tillsammans med sin huvudsakliga prenumeration. De är perfekta för:

- **Användarbaserad fakturering**: Ytterligare teammedlemmar, användarlicenser eller samtidiga användare
- **Funktionsuppgraderingar**: Premiumfunktioner, avancerad analys eller prioriterat stöd
- **Utökad användning**: Extra lagring, API-anrop eller bandbreddsallokeringar
- **Tjänstetillägg**: Professionella tjänster, utbildning eller konsultationstimmar

## Viktiga fördelar

- **Flexibla prismodeller**: Erbjud basplaner med valfria tillägg för att skapa sofistikerade prissättningsstrukturer. Du kan tillgodose olika kundsegment med uppgraderingar som växer i takt med kundernas behov.
- **Intäktsoptimering**: Öka din genomsnittliga intäkt per användare (ARPU) genom att presentera relevanta tillägg. Detta skapar naturliga uppsäljningsmöjligheter när kunder lägger till funktioner över tid.
- **Förenklad hantering**: Hantera alla prisdelar från en och samma kontrollpanel. Tillägg inkluderas automatiskt i både kassan och prenumerationshanteringen.
- **Kundval**: Låt kunder skräddarsy sina prenumerationer genom att välja bara de tillägg de behöver, vilket ökar tillfredsställelsen och minskar churn.

## Skapa tillägg

Tillägg skapas som separata produkter i din Dodo Payments-kontrollpanel och kopplas sedan till dina huvudsakliga prenumerationsprodukter. Denna separation låter dig:

- Återanvända tillägg över flera prenumerationsprodukter
- Hantera prissättning oberoende
- Spåra tilläggsprestanda separat
- Uppdatera tillägg utan att påverka basprenumerationer

### Konfiguration av tillägg

När du skapar tillägg kan du konfigurera:

- **Prissättning**: Ange engångs- eller återkommande prissättning för tillägget
- **Faktureringscykel**: Måste matcha prenumerationens faktureringscykel
- **Kvantilimits**: Ange minimala och maximala kvantiteter per kund
- **Tillgänglighet**: Kontrollera vilka prenumerationsprodukter som kan använda tillägget
- **Skatteinställningar**: Konfigurera lämpliga skattkategorier

## Integrationsexempel

### Kassasessioner med tillägg
När du skapar kassasessioner kan du inkludera tillägg med anpassade kvantiteter:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    {
      product_id: 'your_subscription_id',
      quantity: 1,
      addons: [
        {
          addon_id: 'your_addon_id',
          quantity: 3 // 3 additional seats
        }
      ]
    }
  ],
  // ... other checkout options
});
```

### Planändringar med tillägg
Ändra befintliga prenumerationer för att lägga till, ta bort eller uppdatera tillägg:

```typescript
// Add add-ons to existing subscription
await client.subscriptions.changePlan('sub_123', {
  product_id: 'prod_new',
  quantity: 1,
  proration_billing_mode: 'difference_immediately',
  addons: [
    { addon_id: 'addon_123', quantity: 2 }
  ]
});

// Remove all existing add-ons
await client.subscriptions.changePlan('sub_123', {
  product_id: 'prod_new',
  quantity: 1,
  proration_billing_mode: 'difference_immediately',
  addons: [] // Empty array removes all existing add-ons
});
```

### Dynamisk prissättning
Beräkna totalkostnader dynamiskt baserat på valda tillägg:

```typescript
function calculateTotalCost(basePrice: number, addons: AddonSelection[]) {
  const addonTotal = addons.reduce((sum, addon) => 
    sum + (addon.price * addon.quantity), 0
  );
  return basePrice + addonTotal;
}
```

## Bästa praxis

- **Börja enkelt**: Lansera med 2–3 kärntillägg och utöka alternativen baserat på kundfeedback och användning.
- **Behåll prissklarhet**: Kommunicera tydligt tilläggspriser och värde, så att kunder förstår vad de får för den extra kostnaden.
- **Testa grundligt**: Validera tilläggskombinationer för att säkerställa att prisberäkningar förblir korrekta och att kassaflöden fungerar smidigt.

### Designöverväganden

- **Tydligt värdeförslag**: Varje tillägg bör ha en tydlig fördel som kunder lätt kan förstå
- **Logisk gruppering**: Gruppera relaterade tillägg tillsammans i ditt kassaflöde
- **Flexibla kvantiteter**: Låt kunder justera tilläggskvantiteter efter behov
- **Transparent prissättning**: Visa totalkostnader tydligt genom hela kassaprocessen

<Info>
Tillägg är ett kraftfullt sätt att skapa flexibla, skalbara prismodeller som växer med dina kunder. Börja med enkla användningsfall och utöka när du lär dig vad som fungerar bäst för din verksamhet och dina kunder.
</Info>
