## Adaptiv Valuta

Adaptiv Valuta gör det möjligt för kunder att betala i sin föredragna lokala valuta, vilket förbättrar förtroendet och konverteringsgraden.

### Hur Det Fungerar

1. **Aktivera**: Aktivera Adaptiv Valuta från dina Företagsinställningar
2. **Välj**: Kunder kan byta valuta direkt vid kassan
3. **Konvertera**: Priser konverteras dynamiskt med hjälp av realtids FX-kurser
4. **Visa**: Slutbeloppet visas transparent innan betalning

## Automatisk Skatteberäkning

Skatter beräknas automatiskt baserat på kundens faktureringsplats, vilket säkerställer efterlevnad av GST, VAT och försäljningsskattekrav utan manuell konfiguration.

### Hur Skatteberäkning Fungerar

<Steps>
<Step title="Platsdetektering">
  Skatteregler tillämpas baserat på kundens land (och region där det är tillämpligt).
</Step>

<Step title="Dynamiska Uppdateringar">
  Skattebeloppet uppdateras automatiskt när:
  - Landet ändras
  - Adressen uppdateras
</Step>

<Step title="Transparent Visning">
  Slutlig skatteuppdelning visas tydligt innan betalning.
</Step>
</Steps>

<Tip>
Skatteberäkning är helt automatiserad. Ingen manuell konfiguration krävs för standard digitala varor och SaaS-produkter.
</Tip>

## Stöd för Företags Skatte-ID

För registrerade företag tillåter kassan kunder att ange sitt Företags Skatte-ID (t.ex. VAT/GST-nummer).

### Vad Händer När Ett Skatte-ID Ange

- Skatteberättigande valideras i realtid
- Tillämpbara skattebefrielser eller omvända avgiftsregler tillämpas
- Skattebeloppet uppdateras omedelbart vid kassan

</Info>

## Rabattkoder

Kunder kan använda rabatt- eller kampanjkoder som du har skapat i instrumentpanelen direkt på kassasidan.

### Kassaupplevelse

1. Kunden anger rabattkoden
2. Rabatten valideras omedelbart
3. Uppdaterat pris och besparingar visas tydligt

<Frame>
  <img src="/images/checkout/c4.png" alt="Inmatning av rabattkod vid kassan" style={{ width: '70%' }}/>
</Frame>

### API-integration

Förhandsanvänd en rabattkod eller aktivera inmatningsfältet för rabattkoden:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  discount_code: 'WELCOME20', // Pre-apply a code
  feature_flags: {
    allow_discount_code: true // Show discount input field
  },
  return_url: 'https://yoursite.com/return'
});
```

## Smart Adressinsamling

Kassan stöder flexibel adressinmatning för snabbare slutförande.

### Tillgängliga Alternativ

| Alternativ | Beskrivning |
|--------|-------------|
| **Google Adress Autofyll** | Snabbt val med autocomplet |
| **Manuell Inmatning** | Full kontroll för kompletta adresser |
| **Landval** | Drivande skatte- och efterlevnadslogik |

<Tip>
Adressinsamling balanserar hastighet, noggrannhet och global täckning för att maximera konvertering samtidigt som efterlevnad säkerställs.
</Tip>

## Konfiguration av Kassasessioner

Kontrollera kassans beteende med Checkout Sessions API:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  customer: {
    email: 'customer@example.com',
    name: 'Jane Doe'
  },
  billing_currency: 'EUR', // Set specific currency
  discount_code: 'PROMO10',
  feature_flags: {
    allow_discount_code: true
  },
  return_url: 'https://yoursite.com/return',
  metadata: {
    order_ref: 'ORD-12345'
  }
});
```
