## Vad är Rabattkoder?

Rabattkoder är kampanjtokens som minskar orderbeloppet vid kassan. De är idealiska för:

- **Säsongs kampanjer**: Black Friday, produktlanseringar eller jubileer
- **Förvärvserbjudanden**: Incitament för första köp eller belöningar för hänvisningar
- **Behållning**: Återvinning eller lojalitetsbelöningar för befintliga kunder
- **B2B-erbjudanden**: Kontrakterade eller förhandlade priser via privata koder

## Nyckelfördelar

- **Flexibla rabatter**: Procentuell eller fast summa rabatt
- **Målstyrd kontroll**: Begränsa efter produkt och prenumerationscykler
- **Kampanjstyrning**: Utgångsdatum och användningsgränser
- **Smidig kassa**: UI-fält och API-stöd via kassa-sessioner

## Skapa Rabattkoder

Skapa rabattkoder i din Dodo Payments-instrumentpanel, och tillämpa dem i hostad kassa eller via API.

### Instrumentpanelens inställning

- **Rabattnamn** (obligatoriskt): Internt och visningsnamn i instrumentpanelen
- **Kod** (obligatoriskt): Strängen som kunderna anger vid kassan
- **Typ & Belopp** (obligatoriskt): Ställ in ett procentvärde eller fast belopp, eller generera en slumpmässig kod med den angivna knappen
- **Utgångsdatum** (valfritt): Datum efter vilket koden blir ogiltig
- **Användningsgräns** (valfritt): Max antal inlösen över alla kunder
- **Produktbegränsning** (valfritt): Begränsa tillämpning till utvalda produkter
- **Begränsning av prenumerationscykel** (valfritt): Antal faktureringscykler som rabatten gäller för

<Tip>
Använd cykellimiteringar för introduktionspriser på prenumerationer (t.ex. "50% rabatt i 3 månader").
</Tip>

## Kassaupplevelse

1. Köpare anger koden i kassafältet.
2. Berättigade rabatter tillämpas och totalsummor uppdateras omedelbart.

<Info>
I Checkout Sessions skickar du `discount_code` för att förapplicera en kod, och sätter `feature_flags.allow_discount_code` för att visa inmatningsfältet.
</Info>

## Vanliga Användningsfall

- **Introduktionserbjudanden**: Tidsbegränsade lanseringskampanjer för nya produkter
- **Bulk eller B2B**: Kontrakterade rabatter för utvalda produktset
- **Behållningsstrategier**: Återvinningskoder i churn-förebyggande arbetsflöden
- **Säsongs kampanjer**: Högtids- eller evenemangsbaserade kampanjer

## Integreringsexempel

### Tillämpa en rabatt i Kassa-sessioner
Förhandsanvänd en rabatt och visa kodinmatnings-UI:n.

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  discount_code: 'BLACKFRIDAY2024',
  customer: { email: 'user@example.com', name: 'Jane Doe' },
  return_url: 'https://yoursite.com/return'
});
```

### Aktivera rabattinmatning utan förhandsanvändning
Låt kunder ange en kod vid kassan utan att skicka en i förväg.

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  feature_flags: {
    allow_discount_code: true
  },
  return_url: 'https://yoursite.com/return'
});
```

## Bästa Praxis

- **Namnge tydligt**: Använd igenkännbara koder som matchar kampanjnamn
- **Tidsbegränsa**: Lägg till utgångsdatum för att skapa brådska och förhindra missbruk
- **Begränsa klokt**: Begränsa till specifika produkter för att undvika marginalförlust
- **Validera tidigt**: Kontrollera kodens tillämplighet innan kassan bekräftas
- **Övervaka påverkan**: Spåra användning och konvertering per kampanj

<Info>
Rabattkoder är kraftfulla verktyg för förvärv och behållning. Börja med enkla, väl namngivna erbjudanden, validera noggrant och iterera baserat på resultat.
</Info>
