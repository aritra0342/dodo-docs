## Vad är rabattkoder?

Rabattkoder är kampanjtoken som minskar orderbeloppet i kassan. De är idealiska för:

- **Säsongskampanjer**: Black Friday, produktlanseringar eller jubileer
- **Förvärvserbjudanden**: Incitament för första köp eller belöningar för hänvisningar
- **Behållning**: Återvinnings- eller lojalitetsbelöningar för befintliga kunder
- **B2B-erbjudanden**: Kontrakterade eller förhandlade priser via privata koder

## Nyckelfördelar

- **Flexibla rabatter**: Procentuell eller fast rabatt
- **Riktad kontroll**: Begränsa efter produkt och prenumerationscykler
- **Kampanjstyrning**: Utgångsdatum och användningsgränser
- **Sömlös kassa**: UI-fält och API-stöd via kassasessioner

## Skapa rabattkoder

Skapa rabattkoder i din Dodo Payments-instrumentpanel, och tillämpa dem sedan i värd-kassan eller via API.

### Instrumentpanelinställning

- **Rabattnamn** (obligatoriskt): Internt och visningsnamn i instrumentpanelen
- **Kod** (obligatoriskt): Strängen som kunderna anger i kassan
- **Typ & belopp** (obligatoriskt): Sätt en procentuell eller fast beloppsvärde, eller generera en slumpmässig kod med knappen som tillhandahålls
- **Utgångsdatum** (valfritt): Datum efter vilket koden blir ogiltig
- **Användningsgräns** (valfritt): Max antal inlösen över alla kunder
- **Produktbegränsning** (valfritt): Begränsa tillämplighet till utvalda produkter
- **Begränsning av prenumerationscykel** (valfritt): Antal faktureringscykler som rabatten gäller för

<Tip>
Använd cykelgränser för introduktionspriser på prenumerationer (t.ex. "50% rabatt i 3 månader").
</Tip>

## Kassaupplevelse

1. Köpare anger koden i kassafältet.
2. Berättigade rabatter tillämpas och totalsummor uppdateras omedelbart.

<Info>
I Kassa Sessioner, skicka `discount_code` för att förtillämpa en kod, och ställ in `feature_flags.allow_discount_code` för att visa inmatningsfältet.
</Info>

## Vanliga användningsfall

- **Introduktionserbjudanden**: Tidsbegränsade lanseringskampanjer för nya produkter
- **Bulk eller B2B**: Kontrakterade rabatter för utvalda produktset
- **Behållningsstrategier**: Återvinningskoder i churn-förebyggande arbetsflöden
- **Säsongskampanjer**: Högtids- eller evenemangsbaserade kampanjer

## Integreringsexempel

### Tillämpa en rabatt i Kassasessioner
Förtillämpa en rabatt och visa koden inmatnings-UI.

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  discount_code: 'BLACKFRIDAY2024',
  customer: { email: 'user@example.com', name: 'Jane Doe' },
  return_url: 'https://yoursite.com/return'
});
```

### Aktivera rabattinmatning utan förtillämpning
Låt kunder ange en kod i kassan utan att skicka en i förväg.

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  feature_flags: {
    allow_discount_code: true
  },
  return_url: 'https://yoursite.com/return'
});
```

## Bästa praxis

- **Namnge tydligt**: Använd igenkännbara koder som matchar kampanjnamn
- **Tidsbegränsa**: Lägg till utgångar för att skapa brådska och förhindra missbruk
- **Begränsa klokt**: Begränsa till specifika produkter för att undvika marginalförlust
- **Validera tidigt**: Kontrollera kodens tillämplighet innan kassan bekräftas
- **Övervaka påverkan**: Spåra användning och konvertering per kampanj

<Info>
Rabattkoder är kraftfulla verktyg för förvärv och behållning. Börja med enkla, väl namngivna erbjudanden, validera noggrant och iterera baserat på prestanda.
</Info>
