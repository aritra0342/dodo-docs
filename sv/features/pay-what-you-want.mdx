**Betala Vad Du Vill (PWYW)** låter kunder bestämma hur mycket de betalar—på eller över ett minimum som definieras av handlaren. Det ökar engagemanget, uppmuntrar samhällsstöd och ger flexibel prissättning för digitala produkter.

## Vad Är Betala Vad Du Vill?

PWYW är en prissättningsmodell där kunder sätter sitt eget pris inom de ramar du definierar. Det är idealiskt för:

- **Digitala produkter och media**: Mallar, e‑böcker, kurser, plugins
- **Experimentella lanseringar**: Tidig åtkomst, beta-releaser, tidsbegränsade kampanjer
- **Paket och tillägg**: Lägg till en PWYW-nivå för att öka deltagandet

## Nyckelfördelar

- **Kunddriven prissättning**: Köpare väljer vad de betalar
- **Intäktsgolv**: Tvinga fram ett minimum för att skydda marginalerna
- **Förankrade förväntningar**: Valfritt föreslaget pris för att vägleda val
- **Smidig kassa**: Värd för kassa och kvitton hanteras för dig
- **Utvecklarvänlig**: Tydliga API:er för skapande, återbetalningar och webhooks

<Warning>
PWYW är endast tillgängligt för **Single Payment** (engångs)produkter.
</Warning>

## Skapa Betala Vad Du Vill

Skapa en engångsprodukt och aktivera PWYW i din Dodo Payments-instrumentpanel. Avsnitten nedan motsvarar direkt skapelseformuläret.

### Produkt skapande

#### Produktdetaljer

- **Produktnamn** (obligatoriskt): Visningsnamn som visas i kassan och på fakturor
- **Produktbeskrivning** (obligatoriskt): Tydligt värdeuttalande som kunder ser innan de betalar
- **Produktbild** (obligatoriskt): PNG/JPG/WebP upp till 3 MB
- **Varumärke**: Koppla produkten till ett specifikt varumärke för temat och e-post
- **Skattekategori** (obligatoriskt): Välj rätt kategori för att bestämma skatteregler

<Tip>
Välj den mest korrekta skattekategorin för att säkerställa korrekt skatteuppbörd per region.
</Tip>

#### Prissättning

- **Prissättningstyp**: Välj <b>Engångsbetalning</b>
- **Betala Vad Du Vill** (växla): Aktivera för att låta köpare välja beloppet
- **Minimi pris** (obligatoriskt): Lägsta tillåtna belopp som en kund kan betala
- **Föreslaget pris** (valfritt): Rekommenderat belopp som visas på produktsidan

<Tip>
Använd metadata för att lagra identifierare från ditt system (t.ex. orderId) så att du senare kan stämma av händelser och kvitton.
</Tip>

## Vanliga Användningsfall

- **Digitala tillgångar**: Teman, mallar, ikonset, media eller plugins
- **Innehåll och utbildning**: E‑böcker, kurser, workshops med supporterprissättning
- **Gemenskap & donationer**: Insamlingar, öppen källkods sponsring, stöd till skapare
- **Lanseringsexperiment**: Betaåtkomst och tidsbegränsade kampanjer för att mäta efterfrågan

## Integreringsexempel

### Kassa sessioner (engångsbetalning, PWYW)
Lägg till en PWYW-produkt i kundvagnen. Kassan samlar in köparens belopp (respekterar ditt minimum) — inga extra fält krävs.

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_pwyw_demo', quantity: 1 }
  ],
  return_url: 'https://yoursite.com/checkout/success'
});
```

## Bästa Praxis

- **Välj ett rimligt minimum**: Täck kostnader samtidigt som deltagande är enkelt
- **Använd ett föreslaget pris**: Förankra förväntningar för att förbättra genomsnittligt ordervärde
- **Förklara värdet tydligt**: Berätta för köpare vad deras bidrag låser upp
- **A/B-test kampanjer**: Tidsbegränsa erbjudanden och experimentera med meddelanden
- **Övervaka distribution**: Spåra betalade belopp för att förfina minimum och föreslaget pris

<Info>
PWYW är ett flexibelt sätt att locka fler kunder samtidigt som du bevarar en intäktsnivå. Börja enkelt, testa grundligt och iterera utifrån konvertering, bidragsfördelning och återbetalningsfrekvens.
</Info>
