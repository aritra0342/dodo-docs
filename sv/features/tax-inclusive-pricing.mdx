## Vad är Skatteinkluderande Prissättning?

Skatteinkluderande Prissättning låter dig inkludera tillämpliga skatter direkt i det visade produktpriset. Kunder ser ett slutgiltigt belopp på förhand för en mer transparent och konverteringsvänlig upplevelse. Vid kassan beräknar Dodo den skattefria delen och skattebeloppet från det visade priset baserat på kundens plats och bryter tydligt ut dem på fakturan.

<Info>
Slutgiltig skatt varierar fortfarande beroende på kundens plats och skattekategori. Kassa och fakturor visar den beräknade skatteandelen som härstammar från det visade priset.
</Info>

## Nyckelfördelar

- **Tydlighet för kunder**: Ett slutgiltigt pris visas på förhand
- **Efterlevnad per region**: Skatt beräknas per plats och bryts ut
- **Korrekt fakturering**: Fakturor visar nettobelopp och skatt härledd från priset
- **Konverteringsvänligt**: Minskade överraskningar vid kassan

## Produktinställning

Skapa eller redigera en produkt i din instrumentpanel, aktivera sedan skatteinkluderande prissättning under prissättningsinställningar.

#### Prissättning

- **Pris** (obligatoriskt): Baspris som visas för kunden.
- **Skatteinkluderande Prissättning**: Slå på för att behandla det visade priset som skatteinkluderande. Dodo kommer att beräkna nettobeloppet (före skatt) och skatteandelen vid kassan baserat på kundens plats och skattekategori.

<Warning>
Endast framtida köp återspeglar uppdaterade skatteinkluderande inställningar. Befintliga köp och aktiva prenumerationer ändras inte retroaktivt.
</Warning>

<Tip>
Använd konsekvent produkttext som "Skatteinkluderande" så att kunderna förstår totalsummor vid en blick.
</Tip>

## Integreringsexempel

### Skapa en produkt med skatteinkluderande prissättning

```typescript
const product = await client.products.create({
  name: 'Pro Plan',
  description: 'All features included',
  price: {
    currency: 'USD',
    price: 10000
    tax_inclusive: true
  }
});
```

### Uppdatera en befintlig produkt för att slå på skatteinkluderande

```typescript
await client.products.update(product.id, {
  price: {
    tax_inclusive: true
  }
});
```

## Bästa praxis

- **Välj rätt skattekategori** för att säkerställa korrekt beräkning per region.
- **Kommunicera totalsummor** tydligt med "Skatteinkluderande" etiketter i produkttexten.
- **Testa i sandbox** för att validera.
