## Översikt

Dodo Payments genererar automatiskt professionella fakturor för varje lyckad transaktion, vilket eliminerar manuell fakturaskapande och säkerställer konsekvent dokumentation för ditt företag. Fakturor levereras omedelbart till både dig och dina kunder via e-post efter varje betalning.

<Info>
Fakturor genereras automatiskt för både engångsköp och återkommande prenumerationsbetalningar.
</Info>

## Ställa in fakturav branding

<Steps>
{/* LOCKED_PATTERN_a79a1165634b8d632bd8527f5b892b7b */}
Öppna Dodo Payments-instrumentpanelen och gå till **Inställningar** → **Företagsprofil**.
</Step>

{/* LOCKED_PATTERN_c10bb3e693d1a9bb6e874fcfd91040d9 */}
Ladda upp din företagslogotyp och ange ditt företagsnamn. Dessa uppgifter kommer att visas på alla genererade fakturor.

<Tip>
Använd en logotyp med hög upplösning (minst 300x300px) i PNG-format för bästa utskriftskvalitet.
</Tip>
</Step>

{/* LOCKED_PATTERN_08395c29a4a5c780a5cf3de53acf78b9 */}
Klicka på **Spara** för att tillämpa dina varumärkesinställningar. Alla framtida fakturor kommer automatiskt att använda dessa uppgifter.

<Check>
Din fakturamarknadsföring är nu konfigurerad och kommer att tillämpas på alla nya fakturor.
</Check>
</Step>
</Steps>

## Vad som ingår i fakturor

Varje genererad faktura innehåller omfattande transaktionsdetaljer:

- **Företagsinformation**: Ditt företagsnamn, logotyp och kontaktuppgifter
- **Kunduppgifter**: Kundens namn och e-postadress
- **Fakturanummer**: Unik identifierare för spårning och referens
- **Transaktionsdatum**: När betalningen behandlades
- **Betalningsdetaljer**: Betalt belopp, valuta och betalningsmetod
- **Produkt-/tjänstbeskrivning**: Specificerad lista över köpta produkter eller prenumerationer
- **Skatteinformation**: Tillämpade skatter och skatteidentifieringsnummer
- **Betalningsstatus**: Bekräftelse på lyckad betalning

<Note>
Fakturanummer genereras automatiskt och följer ett sekventiellt format för enkel uppföljning.
</Note>

## Hur fakturagenerering fungerar

<Tabs>
{/* LOCKED_PATTERN_1f557924c92ef240191a271a2acc0963 */}
När en kund slutför ett engångsköp:

1. Betalningen behandlas framgångsrikt
2. Fakturan genereras automatiskt med transaktionsdetaljer
3. Fakturan skickas via e-post till både kund och handlare
4. Fakturan visas i din dashboard under **Fakturor**-sektionen

Alla fakturor för engångsköp innehåller produktinformation, prissättning och betalningsbekräftelse.
</Tab>

{/* LOCKED_PATTERN_0c4b055467985bbff5951606e6691191 */}
För återkommande prenumerationsbetalningar:

1. Betalningen för prenumerationsförnyelse behandlas automatiskt
2. Fakturan genereras för faktureringsperioden
3. Fakturan skickas via e-post till kunden med prenumerationsdetaljer
4. Fakturan lagras i din dashboard för arkivering

Prenumerationsfakturor inkluderar faktureringsperioden, uppgifter om prenumerationsplanen och nästa förnyelsedatum.
</Tab>
</Tabs>

## Hantera fakturor

### Åtkomst till dina fakturor

Navigera till **Fakturor**-sektionen i din Dodo Payments dashboard för att se alla genererade fakturor. Du kan:

- **Visa fakturadetaljer**: Klicka på vilken faktura som helst för att se fullständig transaktionsinformation
- **Ladda ner PDF**: Exportera fakturor i PDF-format för dina register
- **Skicka igen till kund**: Skicka fakturan via e-post igen om det behövs
- **Filtrera och söka**: Hitta specifika fakturor efter datum, kund eller fakturanummer

<Tip>
Använd sök- och filteralternativen för att snabbt hitta fakturor för bokföring eller kundsupportändamål.
</Tip>
