Kortbetalningar är grunden för onlinebetalningar, accepterade globalt och betrodda av kunder världen över. Dodo Payments stöder alla stora kortnätverk med inbyggt bedrägeriskydd och PCI-efterlevnad.

## Konfigurering

Använd dessa värden i `allowed_payment_method_types`:

| Typ | Beskrivning |
| :--- | :---------- |
| `credit` | Alla kreditkort |
| `debit` | Alla betalkort |

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  allowed_payment_method_types: ['credit', 'debit'],
  return_url: 'https://example.com/success'
});
```

<Tip>
Inkludera både `credit` och `debit` om du inte har en specifik anledning att utesluta någon. Betalkort föredras av många kunder och har ofta lägre avgifter.
</Tip>

## 3D Secure Autentisering

3D Secure (3DS) lägger till ett autentiseringlager som minskar bedrägerier och återbetalningar genom att verifiera kortinnehavarens identitet.

### När 3DS Utlöses

3DS utlöses automatiskt när:
- Krävs av kortnätverket
- Krävs av regionala regler (t.ex. PSD2 i Europa)
- Transaktionen flaggas som hög risk

### Tvinga 3DS

Du kan kräva 3DS på alla transaktioner:

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  force_3ds: true,
  return_url: 'https://example.com/success'
});
```

<Note>
Att aktivera 3DS för alla transaktioner minskar bedrägerier men kan något minska konverteringen eftersom vissa kunder överger under autentiseringen.
</Note>

### Aktivera Sparade Kort

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  show_saved_payment_methods: true,
  customer: { customer_id: 'cus_existing_123' },
  return_url: 'https://example.com/success'
});
```

### Köp med Ett Klick

```javascript
// Get customer's saved payment methods
const methods = await client.customers.getPaymentMethods('cus_123');

// Use saved card for instant checkout
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  customer: { customer_id: 'cus_123' },
  payment_method_id: methods[0].payment_method_id,
  confirm: true,
  return_url: 'https://example.com/success'
});
```

## Bästa Praxis

<AccordionGroup>
<Accordion title="Acceptera alla stora nätverk">
Restriktera inte korttyper om det inte är nödvändigt. Kunder förväntar sig att deras föredragna kort ska fungera.
</Accordion>

<Accordion title="Visa kortlogotyper">
Visa Visa, Mastercard, Amex-logotyper på din utcheckning för att bygga förtroende.
</Accordion>

<Accordion title="Hantera avslag på ett smidigt sätt">
Visa tydliga felmeddelanden. Utsätt inte råa felkoder för kunder.
</Accordion>

<Accordion title="Aktivera sparade kort för återkommande kunder">
Sparade betalningsmetoder ökar avsevärt konverteringen för upprepade köp.
</Accordion>
</AccordionGroup>

## Felsökning

<AccordionGroup>
<Accordion title="Kort avslagits">
**Orsaker:** Otillräckliga medel, kortet har gått ut, fel CVV, bankens bedrägeriskydd.

**Lösning:** Be kunden att verifiera uppgifterna eller prova ett annat kort.
</Accordion>

<Accordion title="3DS-autentisering misslyckades">
**Orsaker:** Kunder övergav, bankens system var inte tillgängligt, timeout.

**Lösning:** Försök igen eller be kunden att kontakta sin bank.
</Accordion>

<Accordion title="Kort inte stöds">
**Orsaker:** Regionalt kort stöds inte, förskottsrestriktioner.

**Lösning:** Kunden bör prova ett annat kort från ett stort nätverk.
</Accordion>
</AccordionGroup>
