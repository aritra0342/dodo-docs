## Kundinformation

På **Kundsidan** kan du se en omfattande översikt över hela din kundbas. Tabellen visar alla kunder med följande information:

### Kolumner i Kundlistan

- **Customer Name**: Kundens fullständiga namn med en avatarindikator.
- **Customer Id**: En unik identifierare (t.ex., `cus_5fpsWAa...`) för varje kund.
- **Email**: Kundens e-postadress för kommunikation och portalåtkomst.
- **Phone Number**: Kundens telefonnummer (om det angivits).
- **Created At (UTC)**: Datum och tid då kundkontot skapades.
- **Invite**: Snabbknapp för att dela Customer Portal-inbjudningar.

### Sidfunktioner

- **Redigera kolumner**: Anpassa vilka kolumner som visas i kundtabellen.
- **Bygg rapport**: Generera anpassade rapporter baserade på dina kunddata.
- **Sök**: Hitta specifika kunder snabbt genom namn, e-post eller kund-ID.
- **Sidor**: Navigera genom din kundlista med justerbara rader per sida.

<Tip>
Använd funktionen "Redigera kolumner" för att anpassa din vy och visa bara den information som är mest relevant för ditt arbetsflöde.
</Tip>

### Om-sektion

Om-sektionen visar kärnkundinformation:

- **Namn**: Kundens fullständiga namn
- **E-post**: Kundens e-postadress
- **Kund-ID**: Unik identifierare för API och spårningsändamål
- **Skapat**: Tidsstämpel för när kundkontot skapades

## Kundportal

Kundportalen är ett säkert, värdat självbetjäningsområde där dina kunder kan hantera sina prenumerationer, se faktureringshistorik, få tillgång till licensnycklar och uppdatera sin profil—utan att kontakta support.

### Portalens Möjligheter

Kunder kan få tillgång till portalen för att:

- **Visa Faktureringshistorik**: Se alla tidigare transaktioner och ladda ner fakturor.
- **Hantera Prenumerationer**: Avbryta eller se prenumerationsdetaljer.
- **Hämta Licensnycklar**: Få tillgång till alla licensnycklar kopplade till deras köp.
- **Uppdatera Profil**: Ändra namn, e-post eller telefonnummer.

## Kundplånböcker

Kundplånböcker gör det möjligt för dig att bygga kreditbaserade faktureringssystem liknande OpenAI eller Claude. Varje kund får automatiskt en plånbok som kan hålla förbetalda krediter i flera valutor.

### Nyckelfunktioner

- **Stöd för flera valutor**: Spåra saldon i USD och INR separat.
- **Kreditbaserad fakturering**: Låt kunder köpa krediter i förskott och använda dem för dina tjänster.
- **Transaktionshistorik**: Visa komplett ledger över alla krediter och debiteringar.
- **API-Först Design**: Programmeringsmässigt lägga till, ta bort och kontrollera plånboks saldon.
