## Översikt

Funktionen för kunder i Dodo Payments erbjuder ett centralt nav för att hantera alla dina kundrelationer. Spåra varje kundinteraktion, se köphistorik, hantera prenumerationer, få tillgång till plånboks kreditledgers och möjliggöra självbetjäningsalternativ - allt från en enda instrumentpanel.

## Kundinformation

På **Kundsidan** kan du se en omfattande översikt över hela din kundbas. Tabellen visar alla kunder med följande information:

### Kolumner i Kundlistan

- **Kundnamn**: Kundens fullständiga namn med en avatarindikator.
- **Kund-ID**: En unik identifierare (t.ex. `cus_5fpsWAa...`) för varje kund.
- **E-post**: Kundens e-postadress för kommunikation och åtkomst till portalen.
- **Telefonnummer**: Kundens telefonnummer (om tillhandahållet).
- **Skapat (UTC)**: Datum och tid när kundkontot skapades.
- **Bjud in**: Snabbåtgärdsknapp för att dela inbjudningar till Kundportalen.

### Sidfunktioner

- **Redigera kolumner**: Anpassa vilka kolumner som visas i kundtabellen.
- **Bygg rapport**: Generera anpassade rapporter baserade på dina kunddata.
- **Sök**: Hitta specifika kunder snabbt genom namn, e-post eller kund-ID.
- **Sidor**: Navigera genom din kundlista med justerbara rader per sida.

<Tip>
Använd funktionen "Redigera kolumner" för att anpassa din vy och visa endast den information som är mest relevant för ditt arbetsflöde.
</Tip>

## Kunddetaljsidan

Klicka på valfri kund för att se deras detaljerade informationssida, som inkluderar:

<Frame>
<img src="/images/customer/customer-details.png" alt="Kunddetaljsida som visar kundinformation och transaktionshistorik"/>

### Om-sektion

Om-sektionen visar kärnkundinformation:

- **Namn**: Kundens fullständiga namn
- **E-post**: Kundens e-postadress
- **Kund-ID**: Unik identifierare för API och spårningsändamål
- **Skapat**: Tidsstämpel för när kundkontot skapades

### Kundaktivitetsflikar

Kunddetaljsidan inkluderar tre flikar för att visa olika aspekter av kundaktivitet:

<Tabs>
<Tab title="Transaktionshistorik">
Visa alla betalningstransaktioner för denna kund inklusive:

- **Betalnings-ID**: Unik transaktionsidentifierare
- **Status**: Betalningsstatus (Lyckad, Misslyckad, Pågående, Återbetalad)
- **Belopp**: Transaktionsbelopp
- **Datum (UTC)**: När transaktionen inträffade
- **Prissättningstyp**: Om det är ett engångsköp eller en prenumeration
- **Betalningsmetod**: Betalningsmetod som användes (Kort, UPI, etc.)
- **Kundens e-post**: E-post kopplad till transaktionen
- **Återbetalning**: Snabbåtgärd för att initiera återbetalningar
- **Åtgärder**: Ytterligare alternativ för att hantera transaktionen
</Tab>

<Tab title="Aktiva Prenumerationer">
Visa alla aktiva prenumerationer för denna kund:

- Detaljer om prenumerationsplan
- Faktureringscykel och nästa förnyelsedatum
- Prenumerationsstatus
- Alternativ för att hantera eller avbryta prenumerationer
</Tab>

<Tab title="Kreditledger">
Visa kundens plånboks kreditledger som visar:

- Alla kredit- och debettransaktioner
- Aktuellt plånboks saldo i varje valuta
- Transaktionsorsaker och tidsstämplar
- Detaljerad transaktionshistorik för avstämning
</Tab>
</Tabs>

<Note>
Kreditledger-fliken ger fullständig transparens i kundens plånbokstransaktioner, vilket gör det enkelt att spåra krediter, debiteringar och aktuella saldon.
</Note>

## Kundportal

Kundportalen är ett säkert, värdat självbetjäningsområde där dina kunder kan hantera sina prenumerationer, se faktureringshistorik, få tillgång till licensnycklar och uppdatera sin profil—utan att kontakta support.

### Portalens Möjligheter

Kunder kan få tillgång till portalen för att:

- **Visa Faktureringshistorik**: Se alla tidigare transaktioner och ladda ner fakturor.
- **Hantera Prenumerationer**: Avbryta eller se prenumerationsdetaljer.
- **Hämta Licensnycklar**: Få tillgång till alla licensnycklar kopplade till deras köp.
- **Uppdatera Profil**: Ändra namn, e-post eller telefonnummer.

## Kundplånböcker

Kundplånböcker gör det möjligt för dig att bygga kreditbaserade faktureringssystem liknande OpenAI eller Claude. Varje kund får automatiskt en plånbok som kan hålla förbetalda krediter i flera valutor.

### Nyckelfunktioner

- **Stöd för flera valutor**: Spåra saldon i USD och INR separat.
- **Kreditbaserad fakturering**: Låt kunder köpa krediter i förskott och använda dem för dina tjänster.
- **Transaktionshistorik**: Visa komplett ledger över alla krediter och debiteringar.
- **API-Först Design**: Programmeringsmässigt lägga till, ta bort och kontrollera plånboks saldon.
