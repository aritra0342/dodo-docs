## Kundinformation

På **Kundsidan** kan du se en omfattande översikt över hela din kundbas. Tabellen visar alla kunder med följande information:

### Kolumner i Kundlistan

- **Customer Name**: Kundens fullständiga namn med en avatarindikator.
- **Customer Id**: En unik identifierare (t.ex., `cus_5fpsWAa...`) för varje kund.
- **Email**: Kundens e-postadress för kommunikation och portalåtkomst.
- **Phone Number**: Kundens telefonnummer (om det angivits).
- **Created At (UTC)**: Datum och tid då kundkontot skapades.
- **Invite**: Snabbknapp för att dela Customer Portal-inbjudningar.

### Sidfunktioner

- **Redigera kolumner**: Anpassa vilka kolumner som visas i kundtabellen.
- **Bygg rapport**: Generera anpassade rapporter baserade på dina kunddata.
- **Sök**: Hitta specifika kunder snabbt genom namn, e-post eller kund-ID.
- **Sidor**: Navigera genom din kundlista med justerbara rader per sida.

<Tip>
Använd funktionen "Redigera kolumner" för att anpassa din vy och visa bara den information som är mest relevant för ditt arbetsflöde.
</Tip>

## Kunddetaljsidan

Klicka på valfri kund för att se deras detaljerade informationssida, som inkluderar:

<img src="/images/customer/customer-details.png" alt="Customer details page showing customer information and transaction history"/>

### Om-sektion

Om-sektionen visar kärnkundinformation:

- **Namn**: Kundens fullständiga namn
- **E-post**: Kundens e-postadress
- **Kund-ID**: Unik identifierare för API och spårningsändamål
- **Skapat**: Tidsstämpel för när kundkontot skapades

### Kundaktivitetsflikar

Kunddetaljsidan inkluderar tre flikar för att visa olika aspekter av kundaktivitet:

<Tabs>
<Tab title="Transactions History">
Visa alla betalningstransaktioner för denna kund inklusive:

- **Payment ID**: Unikt transaktions-ID
- **Status**: Betalningsstatus (Successful, Failed, Pending, Refunded)
- **Amount**: Transaktionsbelopp
- **Date (UTC)**: När transaktionen skedde
- **Pricing Type**: Om det är ett engångsköp eller prenumeration
- **Payment Method**: Betalningsmetod som användes (Card, UPI, etc.)
- **Customer Email**: E-post kopplad till transaktionen
- **Refund**: Snabbåtgärd för att initiera återbetalningar
- **Actions**: Ytterligare alternativ för att hantera transaktionen
</Tab>

<Tab title="Active Subscriptions">
Visa alla aktiva prenumerationer för denna kund:

- Detaljer om prenumerationsplanen
- Faktureringscykel och nästa förnyelsedatum
- Prenumerationsstatus
- Alternativ för att hantera eller avbryta prenumerationer
</Tab>

<Tab title="Wallet Ledger">
Visa kundens plånboks transaktionsbok som visar:

- Alla insättningar och uttag
- Aktuellt plånboksaldo i varje valuta (USD, INR)
- Transaktionsskäl och tidpunkter
- Detaljerad transaktionshistorik för avstämningar
Visa kundens kreditbok för plånboken som visar:

- Alla kredit- och debiteringstransaktioner
- Aktuellt plånboksaldo i varje valuta
- Transaktionsskäl och tidpunkter
- Detaljerad transaktionshistorik för avstämningar
</Tab>
</Tabs>

<Note>
Fliken Wallet Ledger ger full transparens i kundens plånbokstransaktioner, vilket gör det enkelt att följa insättningar, uttag och aktuella monetära saldon.
</Note>

## Kundportal

Kundportalen är ett säkert, hostat självbetjäningsområde där dina kunder kan hantera sina prenumerationer, se fakturahistorik, komma åt licensnycklar och uppdatera sin profil – utan att behöva kontakta support.

### Portalens funktioner

Kunder kan komma åt portalen för att:

- **Visa fakturahistorik**: Se alla tidigare transaktioner och ladda ner fakturor.
- **Hantera prenumerationer**: Avbryt eller visa prenumerationsdetaljer.
- **Hämta licensnycklar**: Få tillgång till alla licensnycklar kopplade till deras köp.
- **Uppdatera profilen**: Ändra namn, e-post eller telefonnummer.

## Kundplånböcker

Kundplånböcker är konton med förbetalda monetära saldon för dina kunder. Varje kund får automatiskt en plånbok som kan innehålla riktiga medel i flera valutor (USD och INR), som kan användas för prenumerationsfakturor, återbetalningssaldo eller laddas med kampanjbonusar.

### Viktiga funktioner

- **Flervalutastöd**: Spåra monetära saldon i USD och INR separat.
- **Förbetalda saldon**: Kunder kan hålla medel som tillämpas på framtida fakturor och prenumerationsbetalningar.
- **Transaktionshistorik**: Visa fullständig bok över alla insättningar och uttag.
- **API-först-design**: Lägg till, dra av och kontrollera plånbokssaldon programmatiskt.
- **Kreditbaserad fakturering**: Tillåt kunder att köpa krediter i förväg och spendera dem på dina tjänster.
- **Transaktionshistorik**: Visa fullständig bok över alla krediter och debiteringar.
- **API-först-design**: Lägg till, ta bort och kontrollera plånbokssaldon programmatiskt.
