## Dodo Payments क्या है?

Dodo Payments एक **सभी-में-एक इंजन** है जो वैश्विक स्तर पर लॉन्च, स्केल और मुद्रीकरण करने के लिए डिज़ाइन किया गया है। नए डिजिटल SaaS और AI उत्पादों की नई पीढ़ी के लिए डिज़ाइन किया गया, हम **भुगतान से लेकर बिलिंग और वितरण** तक सब कुछ संभालते हैं - बिना अतिरिक्त इंजीनियरिंग के।

<Info>
**SaaS और AI उत्पादों के लिए बनाया गया**: हमारे अनुकूलित बिलिंग अवसंरचना के साथ एक बार के भुगतान, सब्सक्रिप्शन, क्रेडिट, सीटें, टोकन, या किसी भी उपयोग मेट्रिक को संभालें।
</Info>

<Tip>
  <strong>Sentra से मिलें: आपका AI-संचालित बिलिंग सहायक</strong><br/>
  Sentra एक AI एजेंट है जो आपके IDE में सीधे बिलिंग और भुगतान को एकीकृत, विश्लेषण और कार्य करता है। प्रश्न पूछें, एकीकरण बनाएं, या बिलिंग आर्किटेक्चर की योजना बनाएं - सब कुछ यह बताकर कि आप क्या चाहते हैं।
  <a href="/developer-resources/sentra">
    Sentra के साथ शुरू करें &rarr;
  </a>
</Tip>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">

</Card>
</CardGroup>

<Accordion title="कर्सर/क्लॉड डेस्कटॉप के लिए त्वरित MCP सेटअप">
अपने MCP कॉन्फ़िगरेशन में दोनों सर्वरों को जोड़ें (`~/.cursor/mcp.json` या क्लॉड डेस्कटॉप कॉन्फ़िगरेशन):

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
Node.js 18+ की आवश्यकता है। सभी समर्थित क्लाइंट्स के लिए विस्तृत सेटअप निर्देशों के लिए [MCP सर्वर दस्तावेज़ीकरण](/developer-resources/mcp-server) देखें।
</Info>
</Accordion>

Dodo Payments को अपने पसंदीदा उपकरणों और प्लेटफार्मों से कनेक्ट करें ताकि कार्यप्रवाह को स्वचालित किया जा सके, सूचनाएँ भेजी जा सकें, और आपके भुगतान संचालन को बढ़ाया जा सके।

<Info>
**सब कुछ स्वचालित करें**: वास्तविक समय की सूचनाएँ सेट करें, कार्यप्रवाह को ट्रिगर करें, ग्राहक डेटा को समन्वयित करें, और हमारे बढ़ते एकीकरण पुस्तकालय के साथ अधिक करें।
</Info>
