## Dodo Payments क्या है?

Dodo Payments एक **सभी-में-एक इंजन** है जो वैश्विक स्तर पर लॉन्च, स्केल और मुद्रीकरण करने के लिए डिज़ाइन किया गया है। नए डिजिटल SaaS और AI उत्पादों की नई पीढ़ी के लिए डिज़ाइन किया गया, हम **भुगतान से लेकर बिलिंग और वितरण** तक सब कुछ संभालते हैं - बिना अतिरिक्त इंजीनियरिंग के।

<Info>
**SaaS और AI उत्पादों के लिए बना**: हमारी अनुकूलित बिलिंग इन्फ्रास्ट्रक्चर के साथ एक बार के भुगतान, सब्सक्रिप्शन, क्रेडिट, सीट, टोकन या किसी भी उपयोग मीट्रिक को संभालें।
</Info>

<Tip>
  <strong>Sentra से मिलें: आपका AI-संचालित बिलिंग सहायक</strong><br/>
  Sentra एक AI एजेंट है जो आपके IDE में सीधे बिलिंग और भुगतान को एकीकृत, विश्लेषण और कार्यान्वित करता है। प्रश्न पूछें, एकीकरण बनाएं, या बिलिंग संरचनाओं की योजना बनाएं — बस अपनी आवश्यकता का वर्णन करके।
  <a href="/developer-resources/sentra">
    Sentra के साथ शुरू करें &rarr;
  </a>
</Tip>

### एआई सहायक के लिए MCP सर्वर

अपने एआई कोडिंग सहायक (Cursor, Claude Desktop, Windsurf, VS Code) को सीधे Dodo Payments से हमारे MCP सर्वरों के माध्यम से कनेक्ट करें।

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  Dodo Payments दस्तावेज़ों में सिमेंटिक सर्च — फीचर्स, API, और सर्वोत्तम प्रथाओं के बारे में त्वरित उत्तर पाएं
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  सीधे API ऑपरेशंस को निष्पादित करें - भुगतान बनाएं, सब्सक्रिप्शन प्रबंधित करें, और रिफंड को संभालें अपने AI सहायक के माध्यम से
</Card>
</CardGroup>

{/* LOCKED_PATTERN_8ee343b87e057252cc4a8a376b190970 */}
दोनों सर्वर्स को अपनी MCP कॉन्फ़िगरेशन (`~/.cursor/mcp.json` या Claude Desktop कॉन्फ़िग) में जोड़ें:

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
Node.js 18+ आवश्यक है। सभी समर्थित क्लाइंट्स के लिए विस्तृत सेटअप निर्देशों के लिए [MCP Server documentation](/developer-resources/mcp-server) देखें।
</Info>
</Accordion>
