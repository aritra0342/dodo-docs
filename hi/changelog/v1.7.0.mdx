## नई विशेषताएँ 🚀

- **समर्पित सब्सक्रिप्शन पृष्ठ**: नए समर्पित पृष्ठ के साथ ग्राहक सब्सक्रिप्शन को आसानी से प्रबंधित करें। डैशबोर्ड पर `Sales > Subscriptions` पर जाएं।
  
    <Frame><img src="/images/changelog/subscriptions.png" alt="Subscriptions Page"/></Frame>

- **सुधारित ग्राहक विवरण**: प्रत्येक ग्राहक के लिए एक ही स्थान पर सब्सक्रिप्शन, भुगतान और रिफंड देखें। डैशबोर्ड पर `Sales > Customers > arrow icon` पर जाएं।

    <Frame><img src="/images/changelog/customer_info.png" alt="ग्राहक विवरण"/></Frame>

- **विस्तृत भुगतान जानकारी**: भुगतान राशि, चक्र और तिथियों पर अधिक जानकारी प्राप्त करें। डैशबोर्ड पर `Business > Payouts` पर जाएं।

    <Frame><img src="/images/changelog/payouts_update.png" alt="भुगतान जानकारी"/></Frame>

- **आधिकारिक Ruby SDK**: अपने Ruby अनुप्रयोगों में Dodo Payments को आसानी से एकीकृत करें। अधिक जानकारी के लिए [Ruby SDK](https://github.com/dodopayments/dodopayments-ruby) देखें।

- **सुधारित API फ़िल्टर**: संबंधित सूची API में `customer_id`, `status` आदि जैसे फ़िल्टर जोड़े गए हैं। अधिक जानकारी के लिए API दस्तावेज़ देखें।

- **छोटे सुधार और बग फिक्स**: विभिन्न अन्य छोटे सुधार और बग फिक्स लागू किए गए हैं।
