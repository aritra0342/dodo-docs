## नई विशेषताएँ 🚀

- **सब्सक्रिप्शन का पहला भुगतान**: सबसे अधिक अनुरोधित विशेषता यहाँ है! अब, पहले भुगतान के लिए एकल लेनदेन होगा, जो आदेश और वास्तविक सब्सक्रिप्शन राशि को मिलाता है।

- **UPI QR भुगतान विधि**: भारतीय बाजार के लिए UPI QR को भुगतान विधि के रूप में जोड़ा गया है, जिससे ग्राहक UPI QR कोड का उपयोग करके भुगतान कर सकते हैं।
    
    <img src="/images/changelog/upi-qr.png" alt="UPI QR Feature"/>

- **चेकआउट पर कर आईडी**: ग्राहक अब चालान और कर उद्देश्यों के लिए चेकआउट पृष्ठ पर अपनी कर आईडी जोड़ सकते हैं।

- **डैशबोर्ड पुनर्गठन**: पूरे डोडो पेमेंट्स डैशबोर्ड को अधिक सहज और उपयोगकर्ता के अनुकूल बनाने के लिए पुनर्गठित किया गया है।

- **स्व-सेवा ऑनबोर्डिंग गाइड**: नए उपयोगकर्ता अब स्व-सेवा ऑनबोर्डिंग गाइड का उपयोग करके डोडो पेमेंट्स के साथ शुरू कर सकते हैं।

- **बहु-भाषा समर्थन**: चेकआउट पृष्ठ पर 14+ भाषाओं का समर्थन जोड़ा गया है, जिससे ग्राहक अपनी पसंदीदा भाषा में भुगतान कर सकते हैं। समर्थित भाषाएँ शामिल हैं:
    - English (en)
    - العربية (ar) // Arabic
    - Català (ca) // Catalan
    - 中文 (cn) // Chinese
    - Deutsch (de) // German
    - Español (es) // Spanish
    - Français (fr) // French
    - עברית (he) // Hebrew
    - Italiano (it) // Italian
    - 日本語 (ja) // Japanese
    - Nederlands (nl) // Dutch
    - Polski (pl) // Polish
    - Português (pt) // Portuguese
    - Svenska (sv) // Swedish

- **चालान PDF मेकओवर**: चालान PDFs को एक नए, स्पष्ट और अधिक प्रभावी डिज़ाइन में दिया गया है।

- **केंद्रीकृत सहायता विजेट**: समर्थन को सुव्यवस्थित और सुधारने के लिए एक केंद्रीकृत सहायता विजेट जोड़ा गया है।
    
    <img src="/images/changelog/help-widget.png" alt="Help Widget"/>

## सुधार और बग फिक्स 🔧

- **रेट लिमिटिंग**: दुरुपयोग को रोकने और प्रणाली की स्थिरता सुनिश्चित करने के लिए चालान API में रेट लिमिटिंग जोड़ी गई है।

- **स्थिर भुगतान लिंक में मेटाडेटा**: स्थिर भुगतान लिंक में मेटाडेटा के लिए समर्थन जोड़ा गया है। अधिक विवरण के लिए, [दस्तावेज़](/developer-resources/integration-guide#1-static-payment-links) देखें।

- **छोटी सुधार और फिक्स**: समग्र प्रदर्शन और उपयोगकर्ता अनुभव में सुधार के लिए विभिन्न छोटी सुधार और फिक्स।
