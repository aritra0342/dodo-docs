---
title: अपने प्रोजेक्ट को सबमिट करें
description: अपने ओपन-सोर्स प्रोजेक्ट को डोडो पेमेंट्स कम्युनिटी डायरेक्टरी में कैसे जोड़ें।
icon: plus
---
<Steps>
<Step title="Check eligibility">
- यह परियोजना एक उदार लाइसेंस के साथ ओपन-सोर्स है।
- यह Dodo Payments (SDK, प्लगइन, एकीकरण, टेम्पलेट) से संबंधित है।
- इसमें इंस्टॉल, उपयोग, और रखरखाव स्थिति के साथ README है।
</Step>

<Step title="Add entry via PR">
`community/projects.mdx` को संपादित करें और सबसे प्रासंगिक टैब के अंतर्गत `<Card>` जोड़ें।

```mdx
<Card title="PROJECT_NAME (LANG)" href="REPO_URL" icon="code">
  One-sentence description of your project.
  <br/>By <a href="GITHUB_PROFILE_URL">@github_username</a>
  <br/><br/>`LANGUAGE` `FRAMEWORK` `CATEGORY`
</Card>
```

</Step>

<Step title="Open submission issue (optional)">
यदि आप चाहें तो "Community Project Submission" टेम्पलेट का उपयोग करके एक इशू खोलें जिसमें:
- Repo URL, विवरण, श्रेणी (SDK/Plugin/Boilerplate)
- प्रमुख भाषा/फ्रेमवर्क
- इंस्टॉल/उपयोग स्निपेट लिंक
- रखरखाव स्थिति और अंतिम अद्यतन
</Step>

<Step title="Review & publish">
हम प्रासंगिकता, स्पष्टता और सुरक्षा के लिए समीक्षा करते हैं। अनुमोदित प्रविष्टियों को मर्ज करके प्रकाशित किया जाता है।
</Step>
</Steps>

<Warning>
हम ऐसी प्रविष्टियों को हटा सकते हैं जो अप्रबंधित, असुरक्षित, भ्रामक या नीति का उल्लंघन करती हों।
</Warning>
