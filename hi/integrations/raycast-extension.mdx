आपका Dodo Payments डैशबोर्ड, कुछ कीस्ट्रोक्स की दूरी पर। Raycast का उपयोग करके भुगतान, सब्सक्रिप्शन, ग्राहक, उत्पाद, छूट कोड, लाइसेंस कुंजी और रिफंड जल्दी से देखें बिना डैशबोर्ड खोले।

## स्थापित करें

{/* LOCKED_PATTERN_689eae9422ae1bb6dca6aba8d59d9dd9 */}
  Raycast Store में Dodo Payments एक्सटेंशन पेज खोलें।
</Card>

<Info>
macOS पर Raycast और एक Dodo Payments API की आवश्यक है।
</Info>

## सेटअप

<Steps>
{/* LOCKED_PATTERN_92afe5a5f9799e8778da662cdecfb3a0 */}
1. अपने Dodo Payments डैशबोर्ड में लॉग इन करें।
2. Settings › API Keys पर जाएँ।
3. अपनी API की कॉपी करें।
</Step>

{/* LOCKED_PATTERN_62acba3447424cd114a190a47207d390 */}
1. Raycast खोलें।
2. "Dodo Payments" खोजें और कोई भी कमांड चुनें।
3. जब पूछा जाए, तो API Key फ़ील्ड में अपनी API की दर्ज करें।
4. API मोड चुनें:
   - Test: विकास और परीक्षण के लिए उपयोग करें।
   - Live: प्रोडक्शन डेटा के लिए उपयोग करें।
5. सेटिंग्स सहेजें।
</Step>
</Steps>

<Tip>
Test मोड में शुरू करें ताकि अपनी सेटअप को मान्य किया जा सके, फिर Live में स्विच करें।
</Tip>

<Warning>
अपनी API की को पासवर्ड की तरह मानें। इसे साझा न करें और प्लेन टेक्स्ट में संग्रहीत न करें।
</Warning>

## एक्सटेंशन का उपयोग करें

- **खोजें**: Raycast खोलें, "Dodo Payments" टाइप करें, और एक कमांड चुनें।
- **फिल्टर करें**: आईडी, ईमेल, नाम, या स्थिति के भागों को टाइप करके परिणामों को संकीर्ण करें।
- **विवरण खोलें**: अधिक जानकारी देखने के लिए सूची आइटम पर Enter दबाएं।
