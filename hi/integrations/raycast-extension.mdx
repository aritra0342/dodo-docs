आपका Dodo Payments डैशबोर्ड, कुछ कीस्ट्रोक्स की दूरी पर। Raycast का उपयोग करके भुगतान, सब्सक्रिप्शन, ग्राहक, उत्पाद, छूट कोड, लाइसेंस कुंजी और रिफंड जल्दी से देखें बिना डैशबोर्ड खोले।

## स्थापित करें

<Card title="Raycast स्टोर से स्थापित करें →" icon="puzzle-piece" href="https://www.raycast.com/dodopayments/dodo-payments">
  Raycast स्टोर में Dodo Payments एक्सटेंशन पृष्ठ खोलें।
</Card>

<Info>
macOS पर Raycast और एक Dodo Payments API कुंजी की आवश्यकता है।
</Info>

## सेटअप

<Steps>
<Step title="अपनी API कुंजी प्राप्त करें">
1. अपने Dodo Payments डैशबोर्ड में लॉग इन करें।
2. सेटिंग्स › API कुंजी पर जाएं।
3. अपनी API कुंजी कॉपी करें।
</Step>

<Step title="एक्सटेंशन कॉन्फ़िगर करें">
1. Raycast खोलें।
2. "Dodo Payments" के लिए खोजें और कोई भी कमांड चुनें।
3. जब पूछा जाए, तो API कुंजी फ़ील्ड में अपनी API कुंजी दर्ज करें।
4. API मोड चुनें:
   - टेस्ट: विकास और परीक्षण के लिए उपयोग करें।
   - लाइव: उत्पादन डेटा के लिए उपयोग करें।
5. सेटिंग्स सहेजें।
</Step>
</Steps>

<Tip>
अपने सेटअप को मान्य करने के लिए टेस्ट मोड में शुरू करें, फिर लाइव पर स्विच करें।
</Tip>

<Warning>
अपनी API कुंजी को पासवर्ड की तरह मानें। इसे साझा न करें या प्लेनटेक्स्ट में स्टोर न करें।
</Warning>

## एक्सटेंशन का उपयोग करें

- **खोजें**: Raycast खोलें, "Dodo Payments" टाइप करें, और एक कमांड चुनें।
- **फिल्टर करें**: आईडी, ईमेल, नाम, या स्थिति के भागों को टाइप करके परिणामों को संकीर्ण करें।
- **विवरण खोलें**: अधिक जानकारी देखने के लिए सूची आइटम पर Enter दबाएं।
