## परिचय

Dodo Payments WooCommerce प्लगइन आपके स्टोर के साथ एक सहज एकीकरण प्रदान करता है, जिससे आप UPI, कार्ड, नेट बैंकिंग और डिजिटल वॉलेट सहित कई तरीकों से भुगतान स्वीकार कर सकते हैं। यह व्यापक गाइड आपको स्थापना और कॉन्फ़िगरेशन प्रक्रिया के माध्यम से मार्गदर्शन करेगी।

## चरण-दर-चरण स्थापना गाइड

1. WooCommerce भुगतान प्लगइन का नवीनतम ज़िप डाउनलोड करें।

    <Card title="प्लगइन डाउनलोड करें" icon="download" href="https://github.com/dodopayments/dodo-woocommerce/releases">
    हमारे WooCommerce प्लगइन का नवीनतम संस्करण प्राप्त करें
    </Card>

    वैकल्पिक रूप से, आप सीधे WordPress प्लगइन निर्देशिका से स्थापित कर सकते हैं:
    
    <Card title="WordPress प्लगइन निर्देशिका" icon="wordpress" href="https://wordpress.org/plugins/dodo-payments-for-woocommerce/">
    WordPress.org से WooCommerce के लिए Dodo Payments स्थापित करें
    </Card>

2. अपने WordPress वेबसाइट में व्यवस्थापक के रूप में लॉग इन करें (हम मानते हैं कि आपने पहले से WooCommerce सेटअप कर लिया है)
3. **Plugins → Add New Plugin** पर जाएं
    
    <Frame>
        <img src="/images/woocommerce/1.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
4. **Upload Plugin** बटन पर क्लिक करें और एक फ़ाइल चयन संवाद प्रकट होगा

    <Frame>
        <img src="/images/woocommerce/2.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/3.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
5. **Browse…** पर क्लिक करें और उस ज़िप फ़ाइल का चयन करें जिसे आपने डाउनलोड किया है
6. स्थापना प्रक्रिया शुरू करने के लिए **Install Now** बटन पर क्लिक करें
    
    <Frame>
        <img src="/images/woocommerce/4.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
7. स्थापना पूरी होने के बाद, **Activate Plugin** पर क्लिक करें या इसे **Installed Plugins** अनुभाग से सक्रिय करें
    
    <Frame>
        <img src="/images/woocommerce/5.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
8. प्लगइन अब स्थापित है, लेकिन कॉन्फ़िगरेशन अभी भी आवश्यक है। नीचे सेटअप गाइड पर जारी रखें।

## कॉन्फ़िगरेशन गाइड

### API कुंजी और वेबहुक सेट करना

1. **WooCommerce → Settings → Payments** पर जाएं या WooCommerce मेनू आइटम के नीचे बाईं साइडबार में **Payments** बटन पर क्लिक करें
    
    <Frame>
        <img src="/images/woocommerce/6.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/7.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
2. यदि पहले से सक्षम नहीं है, तो **Dodo Payments** भुगतान प्रदाता को सक्षम करें, फिर प्लगइन कॉन्फ़िगर करने के लिए **Manage** पर क्लिक करें
3. आप विभिन्न कॉन्फ़िगरेशन विकल्प देखेंगे, प्रत्येक के साथ सहायक व्याख्यात्मक पाठ। अपने लाइव API कुंजी को सेट करने से शुरू करें
    
    <Frame>
        <img src="/images/woocommerce/8.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
4. अपने Dodo Payments डैशबोर्ड में **Live Mode** में लॉग इन करें, फिर **Dodo Payments (Live Mode) > Developer > API Keys** पर जाएं या [इस सीधे लिंक](https://app.dodopayments.com/developer/api-keys) पर जाएं और **Add API Key** पर क्लिक करें
    
    <Frame>
        <img src="/images/woocommerce/9.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
5. अपनी API कुंजी को एक वर्णनात्मक नाम दें और **Create** पर क्लिक करें
    
    <Frame>
        <img src="/images/woocommerce/10.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
6. उत्पन्न API कुंजी को कॉपी करें और इसे अपने WooCommerce प्लगइन सेटिंग्स में **Live API Key** फ़ील्ड में पेस्ट करें
    
    <Frame>
        <img src="/images/woocommerce/11.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/12.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
7. अगला, Dodo Payments और WooCommerce के बीच भुगतान स्थिति समन्वय सक्षम करने के लिए वेबहुक साइनिंग कुंजी सेट करें
8. प्लगइन सेटिंग्स पृष्ठ के नीचे स्क्रॉल करें और वहां प्रदर्शित वेबहुक URL को कॉपी करें
    
    <Frame>
        <img src="/images/woocommerce/13.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
9. अपने Dodo Payments डैशबोर्ड पर लौटें और **Dodo Payments (Live Mode) > Developer > Webhooks** पर जाएं और **Add Webhook** पर क्लिक करें
    
    <Frame>
        <img src="/images/woocommerce/14.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
10. चरण 8 में कॉपी किया गया URL संवाद में पेस्ट करें और **Add Webhook** पर क्लिक करें
    
    <Frame>
        <img src="/images/woocommerce/15.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
11. वेबहुक बनाने के बाद, छिपाए गए **Signing Key** के बगल में आंख के आइकन पर क्लिक करें ताकि इसे प्रकट किया जा सके और कॉपी किया जा सके
    
    <Frame>
        <img src="/images/woocommerce/16.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
12. साइनिंग कुंजी को अपने प्लगइन सेटिंग्स में **Live Webhook Signing Key** फ़ील्ड में पेस्ट करें और परिवर्तनों को सहेजें
    
    <Frame>
        <img src="/images/woocommerce/17.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
13. शेष सेटिंग्स की समीक्षा करें जैसे **Global Tax Category** और **All Prices are Tax Inclusive**, क्योंकि ये विकल्प यह निर्धारित करते हैं कि उत्पाद WooCommerce से Dodo Payments में कैसे समन्वयित होते हैं। ध्यान दें कि **Test API Key** और **Test Webhook Signing Key** केवल तभी आवश्यक हैं जब आप टेस्ट मोड का उपयोग करने की योजना बना रहे हों
    
    <Frame>
        <img src="/images/woocommerce/18.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    

## आप तैयार हैं!

आपका WooCommerce स्टोर अब Dodo Payments के साथ एकीकृत है! ग्राहक चेकआउट पर Dodo Payments का चयन कर सकते हैं ताकि सभी समर्थित भुगतान विधियों तक पहुंच प्राप्त कर सकें।

<Frame>
    <img src="/images/woocommerce/19.webp" alt="Dodo Payments checkout option in WooCommerce" />
</Frame>

जब ग्राहक Dodo Payments का चयन करते हैं, तो उन्हें अपने लेनदेन को पूरा करने के लिए Dodo Payments चेकआउट पृष्ठ पर सहजता से पुनर्निर्देशित किया जाएगा।

<Frame>
    <img src="/images/woocommerce/20.webp" alt="Dodo Payments checkout page" />

## प्रमुख विशेषताएँ

- **सहज चेकआउट अनुभव**: ग्राहकों को एक सुरक्षित, अनुकूलित चेकआउट पृष्ठ पर पुनर्निर्देशित करें
- **कई भुगतान विधियाँ**: विभिन्न स्थानीय और अंतरराष्ट्रीय भुगतान विकल्पों का समर्थन
- **वास्तविक समय स्थिति अपडेट**: वेबहुक के माध्यम से तात्कालिक आदेश स्थिति समन्वय
- **विस्तृत लेनदेन रिकॉर्ड**: आपके WooCommerce डैशबोर्ड में व्यापक भुगतान इतिहास
- **बहु-मुद्रा समर्थन**: कई मुद्राओं में भुगतान स्वीकार करें

## समस्या निवारण

### सामान्य समस्याएँ

1. **भुगतान प्रसंस्करण विफलताएँ**
   - सुनिश्चित करें कि API कुंजी सही ढंग से कॉन्फ़िगर की गई हैं
   - सुनिश्चित करें कि वेबहुक URL सही ढंग से सेटअप किया गया है
   - जांचें कि आपका SSL प्रमाणपत्र मान्य है

2. **वेबहुक समन्वय समस्याएँ**
   - सुनिश्चित करें कि वेबहुक साइनिंग कुंजी मेल खाती है
   - संभावित त्रुटियों के लिए सर्वर लॉग की समीक्षा करें
   - सुनिश्चित करें कि आपका सर्वर आने वाले वेबहुक अनुरोधों को प्राप्त कर सकता है

## रखरखाव सर्वोत्तम प्रथाएँ

1. सुरक्षा पैच और नई सुविधाओं के लिए नियमित रूप से प्लगइन को अपडेट करें
2. लाइव लेनदेन संसाधित करने से पहले सभी भुगतान प्रवाह को सैंडबॉक्स मोड में परीक्षण करें
3. WordPress और WooCommerce को नवीनतम संस्करणों में अपडेट रखें
4. नियमित साइट बैकअप लागू करें
5. किसी भी असामान्य गतिविधि के लिए लेनदेन लॉग की निगरानी करें
6. API कुंजी को सुरक्षित रूप से स्टोर करें और कभी भी उन्हें सार्वजनिक रूप से साझा न करें
