## अवलोकन

Dodo Payments Framer Plugin आपके Framer वेबसाइटों में भुगतान प्रसंस्करण को सहजता से एकीकृत करता है। परीक्षण और उत्पादन वातावरण दोनों के लिए समर्थन के साथ अनुकूलन योग्य चेकआउट अनुभव बनाएं।

<Card title="Official Plugin" icon="link" href="https://www.framer.com/marketplace/plugins/dodo-payments">
Framer मार्केटप्लेस पर आधिकारिक Dodo Payments प्लगइन तक पहुँचें।
</Card>

## सेटअप गाइड

<Steps>
<Step title="Generate API Key">
[Dodo Payments Dashboard](https://app.dodopayments.com/developer/api-keys) पर जाएँ और एक नया API कुंजी बनाएँ।

<Tabs>

<Tab title="Live Mode">
वास्तविक भुगतान प्रोसेसिंग के साथ उत्पादन उपयोग के लिए।
</Tab>

<Tab title="Test Mode">
कोई वास्तविक शुल्क नहीं होने पर विकास और परीक्षण के लिए उपयुक्त।
</Tab>

</Tabs>

<Check>
अपनी API कुंजी सुरक्षित तरीके से कॉपी करें - आपको अगले चरण के लिए इसकी आवश्यकता होगी।
</Check>
</Step>

<Step title="Authenticate Plugin">
1. Framer में Dodo Payments प्लगइन खोलें
2. ऑथेंटिकेशन पृष्ठ पर जाएँ
3. अपनी API कुंजी चिपकाएँ
4. उपयुक्त मोड (Test या Live) चुनें
5. ऑथेंटिकेट करने के लिए **Continue** पर क्लिक करें

<Warning>
अपनी API कुंजियों को सुरक्षित रखें और कभी भी उन्हें सार्वजनिक रूप से साझा न करें।
</Warning>
</Step>

<Step title="Manage Products">
ऑथेंटिकेशन के बाद, आप अपने उत्पादों की सूची देखेंगे:

- **मौजूदा उत्पाद**: स्वचालित रूप से प्रदर्शित होते हैं
- **कोई उत्पाद नहीं**: 'उत्पादों' के बगल में **प्लस (+)** बटन पर क्लिक करें

<Info>
आपको कीमतें, विवरण और सेटिंग्स के साथ उत्पाद बनाने के लिए Dodo Payments Dashboard पर पुनर्निर्देशित किया जाएगा।
</Info>
</Step>

<Step title="Add Buy Buttons">
1. अपनी उपलब्ध सूची में से कोई उत्पाद चुनें
2. `Add Buy Button` बटन पर क्लिक करें
3. आपके कैनवास पर `Buy {product_name}` टेक्स्ट के साथ एक खरीद बटन जुड़ जाएगा

<Warning>
सही प्रोजेक्ट अनुमतियाँ न होने पर Add Buy Button अक्षम होगा।
</Warning>
</Step>
</Steps>

## अनुकूलन

<Steps>
<Step title="Select Checkout Experience">
    अपनी पसंदीदा चेकआउट अनुभव चुनें:
    - **Overlay (Embedded)**: चेकआउट आपके Framer प्रोजेक्ट के भीतर एक ओवरले के रूप में दिखाई देता है, जिससे उपयोगकर्ता आपकी साइट पर ही बने रहते हैं।
    - **Hosted**: उपयोगकर्ताओं को एक सुगम अनुभव के लिए Dodo Payments होस्ट किए गए चेकआउट पेज पर पुनर्निर्देशित किया जाता है।
</Step>

<Step title="Customize Appearance Options">
    निम्नलिखित उपस्थिति सेटिंग्स के साथ अपना चेकआउट अनुभव अनुकूलित करें:
    - **Checkout Theme**: अपने ब्रांड की सौंदर्यशैली से मेल खाने के लिए हल्के या अंधेरे थीम के बीच चुनें।
    - **Button Label**: टेक्स्ट अनुकूलित करें (उदा., 'Buy Now', 'Purchase', 'Get Started')।
    - **Button Styles**: रंग, फ़ॉन्ट, आकार और अन्य स्टाइलिंग गुण समायोजित करें।
    - **Return URL**: चेकआउट के बाद उपयोगकर्ता को वापस लाने के लिए एक Framer पृष्ठ सेट करें।
</Step>
</Steps>

## समस्या निवारण

<AccordionGroup>
<Accordion title="Authentication Issues">
**समस्या**: प्लगइन ऑथेंटिकेट नहीं हो रहा

**समाधान**:
- पुष्टि करें कि आपकी API कुंजी सही है
- सुनिश्चित करें कि आप सही मोड (Test/Live) का उपयोग कर रहे हैं
- अपनी इंटरनेट कनेक्शन जांचें
</Accordion>

<Accordion title="Button Creation Issues">
**समस्या**: Add Buy Button अक्षम है

**समाधान**:
- जांचें कि आपके पास अपने Framer प्रोजेक्ट में सही अनुमतियाँ हैं
- सुनिश्चित करें कि आप सही ढंग से ऑथेंटिकेट हैं
- प्लगइन को रिफ्रेश करने की कोशिश करें
</Accordion>

<Accordion title="Product Display Issues">
**समस्या**: उत्पाद दिखाई नहीं दे रहे हैं

**समाधान**:
- सुनिश्चित करें कि आपने Dodo Payments डैशबोर्ड में उत्पाद बनाए हैं
- जांचें कि आपकी API कुंजी के पास सही अनुमतियाँ हैं
- फिर से ऑथेंटिकेट करने का प्रयास करें
</Accordion>
</AccordionGroup>
