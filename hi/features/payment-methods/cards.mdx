कार्ड भुगतान ऑनलाइन भुगतान की नींव हैं, जिन्हें वैश्विक स्तर पर स्वीकार किया जाता है और ग्राहकों द्वारा विश्वसनीयता मिलती है। Dodo Payments सभी प्रमुख कार्ड नेटवर्क का समर्थन करता है, जिसमें अंतर्निहित धोखाधड़ी सुरक्षा और PCI अनुपालन शामिल है।

## समर्थित कार्ड नेटवर्क

### वैश्विक नेटवर्क

| नेटवर्क | कवरेज |
| :------ | :------- |
| **विसा** | वैश्विक नेता, 4B+ कार्ड विश्व स्तर पर |
| **मास्टरकार्ड** | वैश्विक पहुंच, मजबूत सुरक्षा विशेषताएँ |
| **अमेरिकन एक्सप्रेस** | प्रीमियम कार्डधारक, उच्च खर्च |
| **डिस्कवर** | अमेरिका-आधारित, वैश्विक स्तर पर बढ़ रहा है |
| **JCB** | जापान में अग्रणी, एशिया में फ़ैल रहा है |
| **यूनियनपे** | चीन में प्रमुख, 8B+ कार्ड |
| **डाइनर्स क्लब** | प्रीमियम अंतरराष्ट्रीय यात्री |

### क्षेत्रीय नेटवर्क

| नेटवर्क | क्षेत्र |
| :------ | :----- |
| **Interac** | कनाडा का डेबिट नेटवर्क |
| **Cartes Bancaires** | फ्रांस का राष्ट्रीय नेटवर्क |
| **कोरियाई स्थानीय कार्ड** | कोरियाई घरेलू नेटवर्क |
| **Rupay** | भारत का राष्ट्रीय नेटवर्क |

## कॉन्फ़िगरेशन

`allowed_payment_method_types` में इन मानों का उपयोग करें:

| प्रकार | विवरण |
| :--- | :---------- |
| `credit` | सभी क्रेडिट कार्ड |
| `debit` | सभी डेबिट कार्ड |

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  allowed_payment_method_types: ['credit', 'debit'],
  return_url: 'https://example.com/success'
});
```

<Tip>
जब तक आपके पास किसी एक को बाहर रखने का विशिष्ट कारण न हो, दोनों `credit` और `debit` को शामिल करें। कई ग्राहक डेबिट कार्ड को प्राथमिकता देते हैं और उनके शुल्क अक्सर कम होते हैं।
</Tip>

## 3D सुरक्षा प्रमाणीकरण

3D सुरक्षा (3DS) एक प्रमाणीकरण परत जोड़ता है जो कार्डधारक की पहचान की पुष्टि करके धोखाधड़ी और चार्जबैक को कम करता है।

### जब 3DS सक्रिय होता है

3DS तब स्वचालित रूप से सक्रिय होता है जब:
- कार्ड नेटवर्क द्वारा आवश्यक है
- क्षेत्रीय नियमों द्वारा आवश्यक है (उदाहरण के लिए, यूरोप में PSD2)
- लेन-देन उच्च जोखिम के रूप में चिह्नित किया जाता है

### 3DS अनिवार्य करें

आप सभी लेनदेन पर 3DS को अनिवार्य कर सकते हैं:

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  force_3ds: true,
  return_url: 'https://example.com/success'
});
```

<Note>
सभी लेनदेन के लिए 3DS सक्षम करने से धोखाधड़ी कम होती है, लेकिन कुछ ग्राहक प्रमाणीकरण के दौरान लेनदेन छोड़ सकते हैं जिससे रूपांतरण थोड़ी कम हो सकती है।
</Note>

### सहेजे गए कार्ड सक्षम करें

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  show_saved_payment_methods: true,
  customer: { customer_id: 'cus_existing_123' },
  return_url: 'https://example.com/success'
});
```

### एक-क्लिक खरीदारी

```javascript
// Get customer's saved payment methods
const methods = await client.customers.getPaymentMethods('cus_123');

// Use saved card for instant checkout
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  customer: { customer_id: 'cus_123' },
  payment_method_id: methods[0].payment_method_id,
  confirm: true,
  return_url: 'https://example.com/success'
});
```
