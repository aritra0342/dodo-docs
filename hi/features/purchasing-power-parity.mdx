## परिचय

[ParityDeals](https://www.paritydeals.com/) एक ऐसा प्लेटफ़ॉर्म है जो ग्राहक के भौगोलिक स्थान के आधार पर स्वचालित मूल्य अनुकूलन की अनुमति देता है, जिसे सामान्यतः "खरीदने की शक्ति समानता" मूल्य निर्धारण के रूप में जाना जाता है।

ParityDeals के साथ, आप पूरे वर्ष [स्वचालित समय-आधारित छूट](https://www.paritydeals.com/features/time-based-discounts/) और [छुट्टी-आधारित प्रचार](https://www.paritydeals.com/features/holiday-based-discounts/) भी प्रदान कर सकते हैं।

<Info>
हालाँकि आप ParityDeals बैनर को सीधे Dodo Payments स्टोरफ्रंट में एम्बेड नहीं कर सकते (क्योंकि कस्टम स्क्रिप्ट समर्थित नहीं हैं), फिर भी आप अपने स्वयं के वेबसाइट या एप्लिकेशन पर ParityDeals को प्रभावी ढंग से लागू कर सकते हैं.
</Info>

## यह कैसे काम करता है

जब ParityDeals आपकी वेबसाइट में एकीकृत होता है, तो योग्य विज़िटर एक बैनर देखेंगे जिसमें एक स्थान-आधारित छूट कोड शामिल होता है जिसका वे चेकआउट पर उपयोग कर सकते हैं।

<Frame>
  <img src="/images/ppp/1.png" alt="ParityDeals discount banner example" />
</Frame>

ये छूट कोड स्वचालित रूप से ParityDeals द्वारा बनाए और प्रबंधित किए जाते हैं और आपके **Dodo Payments डैशबोर्ड** में **बिक्री → छूट** के तहत दिखाई देंगे।

## चरण-दर-चरण सेटअप गाइड

### 1. ParityDeals पर साइन अप करें

[app.paritydeals.com](https://app.paritydeals.com/) पर जाएं और एक खाता बनाएं।

### 2. डील्स बनाएं

साइन इन करने के बाद, डील बनाने का विकल्प खोजें। ड्रॉपडाउन मेनू से चुनें:

**→ "Dodo Payments के माध्यम से डील बनाएं"**

<Frame>
  <img src="/images/ppp/2.png" alt="Creating deals via Dodo Payments" />
</Frame>

### 3. Dodo Payments API कुंजी उत्पन्न करें

अपने **Dodo Payments डैशबोर्ड** में लॉग इन करें और नेविगेट करें:

**डेवलपर्स → API कुंजी**

**"नई API बनाएं"** पर क्लिक करें और इसे "ParityDeals" जैसे वर्णनात्मक नाम दें।

<Frame>
  <img src="/images/ppp/3.png" alt="Creating a new API key in Dodo Payments" />
</Frame>

### 4. अपनी API कुंजी को ParityDeals से कनेक्ट करें

आपकी बनाई गई API कुंजी को कॉपी करें और इसे ParityDeals इंटरफेस में पेस्ट करें ताकि आपके Dodo Payments खाते के साथ कनेक्शन स्थापित हो सके।

<Frame>
  <img src="/images/ppp/4.png" alt="Connecting API key to ParityDeals" />
</Frame>

### 5. उत्पाद चुनें

एक बार कनेक्ट होने के बाद, ParityDeals आपके Dodo Payments उत्पाद कैटलॉग को लाएगा। उन उत्पादों का चयन करें जिन्हें आप अपनी डील के साथ जोड़ना चाहते हैं।

<Frame>
  <img src="/images/ppp/5.png" alt="Selecting products for ParityDeals" />
</Frame>

### 6. डील सेटिंग्स कॉन्फ़िगर करें

अपनी वेबसाइट का URL दर्ज करें और वैकल्पिक रूप से उन विशिष्ट लक्षित पृष्ठों का चयन करें जहां छूट बैनर दिखाई देना चाहिए।

फिर अपनी पसंदीदा डील प्रकार चुनें:

- स्थान-आधारित डील्स
- छुट्टी डील्स
- समय-आधारित फ्लैश डील्स

<Frame>
  <img src="/images/ppp/6.png" alt="Configuring deal settings" />
</Frame>

### 7. देश-आधारित नियम सेट करें

उन देशों का चयन करें जहां आप स्थानीयकृत छूट प्रदान करना चाहते हैं और प्रत्येक के लिए छूट प्रतिशत निर्धारित करें। ParityDeals अनुशंसित डिफ़ॉल्ट प्रदान करता है जिन्हें आप आवश्यकतानुसार समायोजित कर सकते हैं।

<Frame>
  <img src="/images/ppp/7.png" alt="Setting country-based discount rules" />
</Frame>

### 8. बैनर को अनुकूलित करें

अपनी छूट बैनर को डिज़ाइन करें:

- पाठ को अनुकूलित करना
- रंग और लेआउट का चयन करना
- बैनर की स्थिति चुनना (पृष्ठ के शीर्ष या नीचे)

यह बैनर आपके कॉन्फ़िगर किए गए नियमों के आधार पर योग्य विज़िटर्स को प्रदर्शित किया जाएगा।

  <img src="/images/ppp/8.png" alt="Customizing the discount banner" />
</Frame>

### 9. अपनी वेबसाइट पर ParityDeals स्क्रिप्ट जोड़ें

<Frame>
  <img src="/images/ppp/9.png" alt="ParityDeals integration script" />

यह स्क्रिप्ट स्वचालित रूप से विज़िटर के स्थानों का पता लगाती है और उपयुक्त छूट बैनर प्रदर्शित करती है।

आपका सौदा सहेजने के बाद, ParityDeals एक स्क्रिप्ट बनाएगा। इस स्क्रिप्ट को अपनी वेबसाइट कोड में जोड़ें (आमतौर पर `<head>` अनुभाग में).

## सब कुछ सेट!

आप अब पेश करने के लिए तैयार हैं:

- **स्थान-आधारित डील्स** - खरीदने की शक्ति के आधार पर मूल्य समायोजित करें
- **गिनती/समय-संवेदनशील छूट** - सीमित समय के प्रस्तावों के साथ तात्कालिकता बनाएं
- **छुट्टी प्रचार** - मौसमी खरीदारी के रुझानों का लाभ उठाएं

इनमें से सभी स्वचालित रूप से ParityDeals के माध्यम से होते हैं और Dodo Payments के साथ निर्बाध रूप से काम करते हैं।

अधिक विस्तृत जानकारी के लिए, [ParityDeals दस्तावेज़ीकरण](https://www.paritydeals.com/docs/) देखें या उनकी सहायता टीम से संपर्क करें.
