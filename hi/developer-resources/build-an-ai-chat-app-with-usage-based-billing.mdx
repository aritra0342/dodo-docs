इस ट्यूटोरियल में, आप एक **AI चैट एप्लिकेशन** बनाएंगे जिसमें स्वचालित उपयोग-आधारित बिलिंग होगी। हम सब कुछ शून्य से बनाएंगे: बिलिंग मीटर, उत्पाद कॉन्फ़िगरेशन, और एप्लिकेशन कोड जो बातचीत को शक्ति प्रदान करता है और वास्तविक समय में टोकन उपयोग को ट्रैक करता है।

<Note>
यह ट्यूटोरियल एक पूर्ण कार्यशील एप्लिकेशन प्रदान करता है जिसमें बैकएंड और फ्रंटएंड दोनों शामिल हैं। चैट ऐप Google की Gemini AI का उपयोग करता है और टोकन उपयोग को स्वचालित रूप से ट्रैक करता है, इसलिए किसी भी मैन्युअल गिनती की आवश्यकता नहीं होती।
</Note>

इस ट्यूटोरियल के अंत तक, आपके पास एक कार्यशील चैट एप्लिकेशन होगा जो:
- Google Gemini (AI SDK) का उपयोग करके AI बातचीत को शक्ति प्रदान करता है
- स्वचालित रूप से टोकन उपयोग को ट्रैक करता है (कोई मैनुअल कोड नहीं)
- ग्राहकों को वास्तविक टोकन खपत के आधार पर चार्ज करता है
- एक सुंदर चैट इंटरफेस शामिल है

## हम क्या बना रहे हैं

आइए हम अपनी AI चैट सेवा को समझने से शुरू करें:

- **सेवा**: Google Gemini (AI SDK) का उपयोग करके AI-संचालित चैट
- **मूल्य निर्धारण मॉडल**: प्रति टोकन ($0.01 प्रति 1,000 टोकन)
- **फ्री टियर**: प्रति ग्राहक प्रति माह 10,000 मुफ्त टोकन
- **विशेषताएँ**: बातचीत का इतिहास, स्वचालित टोकन ट्रैकिंग

<Info>
शुरू करने से पहले, सुनिश्चित करें कि आपके पास है:
- एक Dodo Payments खाता
- एक Google AI API कुंजी ([aistudio से एक प्राप्त करें](https://aistudio.google.com/api-keys))
- Node.js v16+ स्थापित
</Info>

## चरण 3: अपना बिलिंग उत्पाद बनाएं

अब हमें एक उत्पाद बनाने की आवश्यकता है जो हमारे मूल्य निर्धारण को परिभाषित करता है ($0.01 प्रति 1,000 टोकन और 10,000 मुफ्त टोकन)। यह हमारे मीटर को वास्तविक बिलिंग से जोड़ता है।

<Tip>
**हम क्या बना रहे हैं**: "AI Chat Service" नामक एक उत्पाद जो टोकन उपयोग के आधार पर शुल्क लेगा और इसमें उदार फ्री टियर होगा।

<Steps>
{/* LOCKED_PATTERN_3c67254436118534dbff872da8ff045c */}
1. अपने Dodo Payments डैशबोर्ड में बाएं साइडबार में **Products** पर क्लिक करें
2. **Create Product** पर क्लिक करें
3. उत्पाद प्रकार के रूप में **Usage-Based** चुनें

यह Dodo Payments को बताता है कि बिलिंग मीटर उपयोग के आधार पर होगी, कोई निश्चित सदस्यता नहीं।

{/* LOCKED_PATTERN_65e0c7e6686dfabea9b37810fb8ee70f */}

**Description**: → `AI-powered chat service with automatic token-based billing`

**उत्पाद छवि**: एक प्रासंगिक छवि अपलोड करें

<Info>

अपने मीटर को कनेक्ट करने से पहले, सुनिश्चित करें कि आपने अपने उत्पाद के लिए **Usage Based Billing** को मूल्य प्रकार के रूप में चुना है।

इसके अलावा, यह सुनिश्चित करने के लिए कि ग्राहकों से केवल उनके उपयोग के आधार पर ही शुल्क लिया जाए और कोई आधार शुल्क न हो, **Fixed Price** को `0` पर सेट करें।

अब, आपने जो मीटर बनाया है उसे लिंक करें:

1. **संबंधित मीटर** अनुभाग में स्क्रॉल करें

4. पुष्टि करें कि यह आपके उत्पाद कॉन्फ़िगरेशन में दिखाई दे रहा है

<Check>

</Check>
</Step>

यहाँ एकीकृत बिलिंग के साथ पूरा AI चैट सर्वर है:

```typescript src/server.ts expandable
import express, { Request, Response } from 'express';
import { generateText } from 'ai';
import { google } from '@ai-sdk/google';
import { createLLMTracker } from '@dodopayments/ingestion-blueprints';
import 'dotenv/config';

const app = express();
app.use(express.json());
app.use(express.static('public'));

// Replace with your test customer ID
const CUSTOMER_ID = 'cus_123';

// Create tracker once with your meter event name
const llmTracker = createLLMTracker({
  apiKey: process.env.DODO_PAYMENTS_API_KEY!,
  environment: process.env.DODO_ENVIRONMENT as 'test_mode' | 'live_mode',
  eventName: 'ai_chat_usage', // Must match your meter configuration
});

// Chat endpoint with conversation support
app.post('/chat', async (req: Request, res: Response) => {
  try {
    const { messages } = req.body;

    if (!messages || !Array.isArray(messages)) {
      return res.status(400).json({ 
        error: 'Missing required field: messages (array)' 
      });
    }

    // Wrap AI SDK with automatic token tracking
    const trackedClient = llmTracker.wrap({
      client: { generateText },
      customerId: CUSTOMER_ID
    });

    // Generate AI response - tokens are automatically tracked!
    const response = await trackedClient.generateText({
      model: google('gemini-2.5-flash'),
      messages: messages
    });

    res.json({
      message: response.text,
      usage: {
        totalTokens: response.usage.totalTokens
      }
    });

  } catch (error: any) {
    console.error('Chat error:', error);
    res.status(500).json({ 
      error: 'Failed to process chat',
      details: error.message 
    });
  }
});

const PORT = process.env.PORT || 3000;
app.listen(PORT, () => {
```

```html public/index.html expandable
      async function sendMessage() {
        const input = document.getElementById("messageInput");
        const message = input.value.trim();

        if (!message) return;

        // Hide empty state
        document.getElementById("emptyState").style.display = "none";

        // Hide error
        document.getElementById("errorMessage").classList.remove("show");

        // Add user message to UI
        addMessage("user", message);

        // Add to conversation history
        conversationHistory.push({
          role: "user",
          content: message,
        });

        // Clear input
        input.value = "";
        input.style.height = "auto";

        // Show typing indicator
        showTypingIndicator();

        // Disable send button
        const sendBtn = document.getElementById("sendBtn");
        sendBtn.disabled = true;

        try {
          const response = await fetch("/chat", {
            method: "POST",
            headers: {
              "Content-Type": "application/json",
            },
            body: JSON.stringify({
              messages: conversationHistory,
            }),
          });

          const data = await response.json();

          if (!response.ok) {
            throw new Error(data.error || "Failed to get response");
          }

          // Hide typing indicator
          hideTypingIndicator();

          // Add assistant response to UI
          addMessage("assistant", data.message, data.usage);

          // Add to conversation history
          conversationHistory.push({
            role: "assistant",
            content: data.message,
          });
        } catch (error) {
          hideTypingIndicator();
          showError(error.message);
          // Remove the last user message from history since it failed
          conversationHistory.pop();
        } finally {
          sendBtn.disabled = false;
        }
```

## समस्या निवारण

सामान्य समस्याएँ और उनके समाधान:

<AccordionGroup>
{/* LOCKED_PATTERN_e3321682e5d5e50512cafdc18e426e4b */}
**संभावित कारण:**
- इवेंट नाम मीटर कॉन्फ़िगरेशन से बिल्कुल मेल नहीं खा रहा
- ग्राहक ID आपके खाते में मौजूद नहीं है
- API कुंजी अमान्य या समाप्त हो चुकी है
- नेटवर्क कनेक्टिविटी समस्या

**समाधान:**
1. सुनिश्चित करें कि इवेंट नाम मीटर कॉन्फ़िगरेशन से बिल्कुल मेल खाता है (केस-सेंसिटिव: `ai_chat_usage`)
2. जांचें कि ग्राहक ID Dodo Payments डैशबोर्ड में मौजूद है
3. सरल API कॉल के साथ API कुंजी का परीक्षण करें
4. त्रुटि संदेशों के लिए सर्वर लॉग्स देखें
</Accordion>

{/* LOCKED_PATTERN_58159e295d497ac31954f0840cbe188e */}
**संभावित कारण:**
- मॉडल उपयोग जानकारी वापस नहीं कर रहा
- SDK संस्करण गलत है

**समाधान:**
1. परीक्षण करें कि क्या मॉडल उपयोग लौटाता है:

```typescript
const response = await generateText({...});
console.log('Usage:', response.usage);
```
