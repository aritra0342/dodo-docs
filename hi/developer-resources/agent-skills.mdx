## अवलोकन

एजेंट स्किल्स पुन: उपयोग योग्य क्षमताएँ हैं जो डोडो पेमेंट्स के साथ काम करते समय AI कोडिंग सहायकों की क्षमताओं को बढ़ाती हैं। ये प्रक्रियात्मक ज्ञान और सर्वोत्तम प्रथाएँ प्रदान करती हैं जो एजेंटों को विशिष्ट कार्यों को अधिक प्रभावी ढंग से पूरा करने में मदद करती हैं — चेकआउट फ्लो को एकीकृत करने से लेकर उपयोग-आधारित बिलिंग सेट करने तक।

<Info>
**स्किल्स को अपने AI सहायक के लिए प्लगइन के रूप में सोचें।** ये आपके एजेंट को डोडो पेमेंट्स सुविधाओं को सही तरीके से लागू करने का तरीका सिखाती हैं, हमारे सुझाए गए पैटर्न और सर्वोत्तम प्रथाओं का पालन करते हुए।
</Info>

स्किल्स किसी भी MCP-समर्थित AI एजेंट के साथ काम करती हैं, जिसमें शामिल हैं:
- **Claude Code** - Anthropic का CLI कोडिंग सहायक
- **OpenCode** - ओपन सोर्स AI कोडिंग एजेंट
- **Cursor** - AI-प्रथम कोड संपादक
- **अन्य MCP क्लाइंट** - कोई भी एजेंट जो स्किल्स प्रोटोकॉल का समर्थन करता है

## स्थापना

अपने पसंदीदा AI कोडिंग सहायक के आधार पर स्थापना विधि चुनें।

<Tabs>
<Tab title="Skills CLI">
किसी भी MCP-समर्थित एजेंट के साथ काम करने वाले यूनिवर्सल स्किल्स CLI का उपयोग करके स्किल्स स्थापित करें:

```bash
# Install all Dodo Payments skills
npx skills add dodopayments/skills
```

या आवश्यकतानुसार व्यक्तिगत स्किल्स स्थापित करें:

```bash
# Best practices guide
npx skills add dodopayments/skills/dodo-best-practices

# Webhook integration
npx skills add dodopayments/skills/webhook-integration

# Subscription billing
npx skills add dodopayments/skills/subscription-integration

# Checkout flows
npx skills add dodopayments/skills/checkout-integration

# Usage-based billing
npx skills add dodopayments/skills/usage-based-billing

# BillingSDK components
npx skills add dodopayments/skills/billing-sdk

# License key management
npx skills add dodopayments/skills/license-keys
```

<Tip>
केवल उन स्किल्स को इंस्टॉल करें जिनकी आपको आवश्यकता है ताकि आपका एजेंट केंद्रित रहे और संदर्भ ओवरहेड कम हो।
</Tip>
</Tab>

<Tab title="Claude Code">
Claude Code में डोडो पेमेंट्स स्किल्स मार्केटप्लेस जोड़ें:

```bash
/plugin marketplace add dodopayments/skills
```

फिर व्यक्तिगत प्लगइन्स इंस्टॉल करें:

```bash
# Install best practices
/plugin install dodo-best-practices

# Install webhook integration
/plugin install webhook-integration

# Install subscription integration
/plugin install subscription-integration

# Install checkout integration
/plugin install checkout-integration

# Install usage-based billing
/plugin install usage-based-billing

# Install BillingSDK
/plugin install billing-sdk

# Install license keys
/plugin install license-keys
```

</Tab>

<Tab title="OpenCode">
जब आपके OpenCode सेटिंग्स में कॉन्फ़िगर किया जाए तो स्किल्स स्वचालित रूप से उपलब्ध हो जाती हैं। अपनी कॉन्फ़िगरेशन में डोडो पेमेंट्स स्किल्स रिपॉज़िटरी जोड़ें:

```json
{
  "skills": {
    "repositories": [
      "dodopayments/skills"
    ]
  }
}
```

</Tab>
</Tabs>

## स्किल्स का उपयोग

एक बार इंस्टॉल हो जाने पर, जब आप अपने AI सहायक से डोडो पेमेंट्स सुविधाएँ लागू करने के लिए कहेंगे, तो यह इन स्किल्स का लाभ उठा सकता है। यहाँ कुछ उदाहरण प्रम्प्ट दिए गए हैं:

### सर्वश्रेष्ठ प्रथाएँ

```
Help me integrate Dodo Payments into my Next.js app following best practices
```

### वेबहुक्स

```
Set up webhook handlers for payment and subscription events
```

### सब्सक्रिप्शन

```
Implement a subscription flow with free trial and multiple pricing tiers
```

### चेकआउट

```
Create a checkout session for my SaaS product
```

### उपयोग-आधारित बिलिंग

```
Add usage-based billing to track API calls for my AI product
```

### BillingSDK

```
Add a pricing page using BillingSDK components
```

### लाइसेंस कुंजी

```
Implement license key validation for my desktop app
```

## स्किल्स कैसे काम करती हैं

स्किल्स आपके AI सहायक को प्रदान करती हैं:

1. **प्रक्रियात्मक ज्ञान** - सुविधाओं को लागू करने के लिए चरण-वार निर्देश
2. **कोड टेम्प्लेट्स** - उत्पादन-तैयार कोड पैटर्न और स्निपेट्स
3. **सर्वश्रेष्ठ प्रथाएँ** - सुरक्षा, त्रुटि हैंडलिंग और UX के लिए सुझाए गए दृष्टिकोण
4. **संदर्भ जागरूकता** - यह समझना कि विभिन्न डोडो पेमेंट्स सुविधाएँ कैसे एक साथ काम करती हैं

जब आप अपने AI सहायक से कोई सुविधा लागू करने को कहते हैं, तो यह संबंधित स्किल का उपयोग करता है ताकि:
- सही कार्यान्वयन दृष्टिकोण को समझ सके
- सटीक, काम करने वाला कोड उत्पन्न कर सके
- डोडो पेमेंट्स कन्वेंशन्स का पालन कर सके
- एज केस और त्रुटियों को ठीक से संभाल सके

<Note>
स्किल्स [MCP Server](/developer-resources/mcp-server) और [Sentra](/developer-resources/sentra) की पूरक हैं। MCP का उपयोग API संचालन के लिए करें, Sentra का उपयोग मार्गदर्शित एकीकरण के लिए करें, और स्किल्स का उपयोग AI सहायक क्षमताओं को बढ़ाने के लिए करें।
</Note>
