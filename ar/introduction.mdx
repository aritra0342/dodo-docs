## ما هي مدفوعات دودو؟

مدفوعات دودو هي **محرك شامل** لإطلاق وتوسيع وتحقيق الربح عالميًا. مصممة للجيل الجديد من منتجات SaaS والذكاء الاصطناعي، نحن نتولى كل شيء من **المدفوعات إلى الفوترة إلى التوزيع** - كل ذلك دون هندسة إضافية.

<Info>
**مصممة لـ SaaS ومنتجات الذكاء الاصطناعي**: تعامل مع المدفوعات لمرة واحدة، الاشتراكات، الاعتمادات، المقاعد، الرموز، أو أي مقياس استخدام باستخدام بنيتنا التحتية المفوترة المحسّنة.
</Info>

<Tip>
  <strong>تعرف على سينترا: مساعد الفوترة المدعوم بالذكاء الاصطناعي</strong><br/>
  سينترا هو وكيل ذكاء اصطناعي يندمج ويحلّل ويتصرف في الفوترة والمدفوعات مباشرةً داخل بيئة تطوير البرمجيات الخاصة بك. اطرح الأسئلة، وابنِ التكاملات، أو خطّط لهياكل الفوترة - كل ذلك من خلال وصف ما تريده.
  <a href="/developer-resources/sentra">
    ابدأ مع سينترا &rarr;
  </a>
</Tip>

### خوادم MCP لمساعدي الذكاء الاصطناعي

قم بتوصيل مساعد البرمجة الذكي الخاص بك (Cursor، Claude Desktop، Windsurf، VS Code) مباشرةً إلى مدفوعات دودي باستخدام خوادم MCP الخاصة بنا.

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  بحث دلالي عبر توثيق Dodo Payments - احصل على إجابات فورية حول الميزات وواجهات البرمجة وأفضل الممارسات
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  نفّذ عمليات API مباشرة - أنشئ مدفوعات، أدِر الاشتراكات، وتعامَل مع التعويضات عبر مساعدك الذكي
</Card>
</CardGroup>

{/* LOCKED_PATTERN_8ee343b87e057252cc4a8a376b190970 */}
أضف الخادمين إلى تكوين MCP الخاص بك (`~/.cursor/mcp.json` أو تكوين Claude Desktop):

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
يتطلب Node.js 18+. راجع [توثيق خادم MCP](/developer-resources/mcp-server) للحصول على تعليمات إعداد مفصلة لجميع العملاء المدعومين.
</Info>
</Accordion>
