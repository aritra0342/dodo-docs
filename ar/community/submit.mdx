title: قدّم مشروعك
description: كيفية إضافة مشروعك مفتوح المصدر إلى دليل مجتمع مدفوعات دودو.
icon: plus
---
<Steps>
{/* LOCKED_PATTERN_b8e49197111d4f91963d876b7d3cbd9b */}
- المشروع مفتوح المصدر برخصة متساهلة.
- إنه مرتبط بـ Dodo Payments (SDK، مكون إضافي، تكامل، قالب).
- يوجد ملف README يتضمن التثبيت، والاستخدام، وحالة الصيانة.
</Step>

{/* LOCKED_PATTERN_29bfb0d9a10f5f1cc614e31786e69c52 */}
حرّر `community/projects.mdx` وأضف `<Card>` تحت علامة التبويب الأكثر صلة.


```mdx
<Card title="PROJECT_NAME (LANG)" href="REPO_URL" icon="code">
  One-sentence description of your project.
  <br/>By <a href="GITHUB_PROFILE_URL">@github_username</a>
  <br/><br/>`LANGUAGE` `FRAMEWORK` `CATEGORY`
</Card>
```

</Step>

{/* LOCKED_PATTERN_5390269a28150786d344085d66aa0ef8 */}
إذا رغبت، افتح مشكلة باستخدام قالب "إرسال مشروع المجتمع" مع:
- رابط المستودع، الوصف، الفئة (SDK/مكوّن إضافي/قالب جاهز)
- اللغة/الإطار الأساسي الرئيسي
- رابط مقتطف التثبيت/الاستخدام
- حالة الصيانة وآخر تحديث
</Step>

{/* LOCKED_PATTERN_03e556e84db70a2199e5921d65cd601a */}
نراجع الملاءمة والوضوح والسلامة. يتم دمج الإدخالات المعتمدة ونشرها.
</Step>
</Steps>

<Warning>
قد نحذف الإدخالات التي لم تتم صيانتها، أو غير آمنة، أو مضللة، أو تنتهك السياسة.
</Warning>
