## المقدمة

تقدم إضافة مدفوعات دودي لـ WooCommerce تكاملًا سلسًا مع متجرك، مما يمكنك من قبول المدفوعات من خلال طرق متعددة بما في ذلك UPI، والبطاقات، والخدمات المصرفية عبر الإنترنت، والمحافظ الرقمية. ستوجهك هذه الدليل الشامل خلال عملية التثبيت والتكوين.

## دليل التكوين

### إعداد مفاتيح API و Webhooks

1. انتقل إلى **WooCommerce → Settings → Payments** أو انقر على زر **Payments** في الشريط الجانبي الأيسر أسفل عنصر قائمة WooCommerce
    
    <Frame>
        <img src="/images/woocommerce/6.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/7.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
2. فعِّل مزود الدفع **Dodo Payments** إذا لم يكن مفعلًا بالفعل، ثم انقر على **Manage** لتكوين الإضافة
3. سترى خيارات إعداد مختلفة، كل منها مع نص توضيحي مفيد. ابدأ بإعداد مفتاح API الخاص بالوضع الفعلي
    
    <Frame>
        <img src="/images/woocommerce/8.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
4. سجِّل الدخول إلى لوحة Dodo Payments في **Live Mode**، ثم انتقل إلى **Dodo Payments (Live Mode) > Developer > API Keys** أو قم بزيارة [هذا الرابط المباشر](https://app.dodopayments.com/developer/api-keys) وانقر على **Add API Key**
    
    <Frame>
        <img src="/images/woocommerce/9.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
5. امنح مفتاح API اسمًا وصفيًا وانقر على **Create**
    
    <Frame>
        <img src="/images/woocommerce/10.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
6. انسخ مفتاح API المُنشأ والصقه في حقل **Live API Key** في إعدادات إضافة WooCommerce الخاصة بك
    
    <Frame>
        <img src="/images/woocommerce/11.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
    <Frame>
        <img src="/images/woocommerce/12.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
7. بعد ذلك، قم بإعداد مفتاح توقيع الويب هوك لتمكين تزامن حالة الدفع بين Dodo Payments وWooCommerce
8. انتقل إلى أسفل صفحة إعدادات الإضافة وقم بنسخ عنوان URL لويب هوك المعروض هناك
    
    <Frame>
        <img src="/images/woocommerce/13.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
9. ارجع إلى لوحة Dodo Payments الخاصة بك وانتقل إلى **Dodo Payments (Live Mode) > Developer > Webhooks** وانقر على **Add Webhook**
    
    <Frame>
        <img src="/images/woocommerce/14.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
10. الصق عنوان URL الذي نسخته في الخطوة 8 داخل مربع الحوار وانقر على **Add Webhook**
    
    <Frame>
        <img src="/images/woocommerce/15.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
11. بعد إنشاء الويب هوك، انقر على أيقونة العين بجانب **Signing Key** المحجوب لعرضه ونسخه
    
    <Frame>
        <img src="/images/woocommerce/16.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
12. الصق مفتاح التوقيع في حقل **Live Webhook Signing Key** في إعدادات الإضافة واحفظ التغييرات
    
    <Frame>
        <img src="/images/woocommerce/17.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>
    
13. راجع الإعدادات المتبقية مثل **Global Tax Category** و**All Prices are Tax Inclusive**، إذ تحدد هذه الخيارات كيفية مزامنة المنتجات من WooCommerce إلى Dodo Payments. لاحظ أن **Test API Key** و**Test Webhook Signing Key** مطلوبان فقط إذا كنت تخطط لاستخدام وضع الاختبار
    
    <Frame>
        <img src="/images/woocommerce/18.webp" alt="Dodo Payments for WooCommerce Plugin" />
    </Frame>

## الميزات الرئيسية

- **تجربة دفع سلسة**: إعادة توجيه العملاء إلى صفحة دفع آمنة ومحسّنة
- **طرق دفع متعددة**: دعم لخيارات الدفع المحلية والدولية المتنوعة
- **تحديثات حالة فورية**: مزامنة فورية لحالة الطلب عبر webhooks
- **سجلات معاملات مفصلة**: تاريخ شامل للمدفوعات في لوحة معلومات WooCommerce الخاصة بك
- **دعم متعدد العملات**: قبول المدفوعات بعملات متعددة

## استكشاف الأخطاء وإصلاحها

### مشكلات شائعة

1. **فشل معالجة المدفوعات**
   - تحقق من تكوين مفاتيح API بشكل صحيح
   - تأكد من إعداد عنوان URL الخاص بـ webhook بشكل صحيح
   - تحقق من أن شهادة SSL الخاصة بك صالحة

2. **مشكلات مزامنة Webhook**
   - تأكد من تطابق مفتاح توقيع webhook
   - راجع سجلات الخادم للبحث عن أخطاء محتملة
   - تحقق من أن خادمك يمكنه استقبال طلبات webhook الواردة

### موارد الدعم

للحصول على مساعدة إضافية:
- اتصل بنا على support@dodopayments.com
- راجع [وثائقنا الشاملة](https://docs.dodopayments.com)

## أفضل ممارسات الصيانة

1. قم بتحديث الإضافة بانتظام للحصول على تصحيحات الأمان والميزات الجديدة
2. اختبر جميع تدفقات الدفع في وضع الصندوق الرملي قبل معالجة المعاملات الحية
3. حافظ على تحديث WordPress وWooCommerce إلى أحدث الإصدارات
4. نفذ نسخ احتياطية منتظمة للموقع
5. راقب سجلات المعاملات لأي نشاط غير عادي
6. قم بتخزين مفاتيح API بشكل آمن ولا تشاركها علنًا
