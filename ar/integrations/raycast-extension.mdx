لوحة معلومات مدفوعات Dodo الخاصة بك، على بعد بضع ضغطات على لوحة المفاتيح. استخدم Raycast لعرض المدفوعات، الاشتراكات، العملاء، المنتجات، رموز الخصم، مفاتيح الترخيص، والمبالغ المستردة بسرعة دون فتح لوحة المعلومات.

## التثبيت

<Card title="تثبيت من متجر Raycast →" icon="puzzle-piece" href="https://www.raycast.com/dodopayments/dodo-payments">
  افتح صفحة إضافة مدفوعات Dodo في متجر Raycast.
</Card>

<Info>
يتطلب Raycast على macOS ومفتاح API لمدفوعات Dodo.
</Info>

## الإعداد

<Steps>
<Step title="احصل على مفتاح API الخاص بك">
1. قم بتسجيل الدخول إلى لوحة معلومات مدفوعات Dodo الخاصة بك.
2. انتقل إلى الإعدادات › مفاتيح API.
3. انسخ مفتاح API الخاص بك.
</Step>

<Step title="تكوين الإضافة">
1. افتح Raycast.
2. ابحث عن "مدفوعات Dodo" واختر أي أمر.
3. عند المطالبة، أدخل مفتاح API الخاص بك في حقل مفتاح API.
4. اختر وضع API:
   - اختبار: استخدمه للتطوير والاختبار.
   - مباشر: استخدمه لبيانات الإنتاج.
5. احفظ الإعدادات.
</Step>
</Steps>

<Tip>
ابدأ في وضع الاختبار للتحقق من إعدادك، ثم انتقل إلى الوضع المباشر.
</Tip>

<Warning>
عامل مفتاح API الخاص بك مثل كلمة مرور. لا تشاركه أو تخزنه بنص عادي.
</Warning>

## استخدام الإضافة

- **بحث**: افتح Raycast، اكتب "مدفوعات Dodo"، واختر أمرًا.
- **تصفية**: ضيق النتائج عن طريق كتابة أجزاء من المعرفات، أو البريد الإلكتروني، أو الأسماء، أو الحالات.
- **فتح التفاصيل**: اضغط على Enter على عنصر في القائمة لعرض المزيد من المعلومات.

## استكشاف الأخطاء وإصلاحها

<AccordionGroup>
<Accordion title="فشل المصادقة أو أخطاء 401/403">
تأكد من أن مفتاح API والوضع (اختبار مقابل مباشر) يتطابقان مع بيانات لوحة المعلومات الخاصة بك. أعد فتح الأمر وأعد إدخال المفتاح إذا لزم الأمر.
</Accordion>
<Accordion title="لا تظهر نتائج">
تأكد من أن الوضع المحدد لديك يحتوي على بيانات. حاول التبديل بين الاختبار والمباشر.
</Accordion>
<Accordion title="حدود المعدل أو انتهاء المهلة">
انتظر لحظة وحاول مرة أخرى. إذا استمرت المشكلة، تحقق من اتصال الشبكة لديك.
</Accordion>
</AccordionGroup>
