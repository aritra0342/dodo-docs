## نظرة عامة

تدمج إضافة Dodo Payments Framer معالجة المدفوعات بسلاسة في مواقع Framer الخاصة بك. أنشئ تجارب دفع قابلة للتخصيص مع دعم لكل من بيئات الاختبار والإنتاج.

قم بالوصول إلى المكون الإضافي الرسمي لـ Dodo Payments على سوق Framer.
</Card>

## دليل الإعداد

<Steps>
<Step title="Generate API Key">
قم بزيارة [لوحة تحكم Dodo Payments](https://app.dodopayments.com/developer/api-keys) وأنشئ مفتاح API جديد.

<Tabs>

<Tab title="Live Mode">
للاستخدام في الإنتاج مع معالجة المدفوعات الحقيقية.
</Tab>

<Tab title="Test Mode">
مثالية للتطوير والاختبار دون أي رسوم حقيقية.
</Tab>

</Tabs>

<Check>
انسخ مفتاح API الخاص بك بأمان - ستحتاجه للخطوة التالية.
</Check>
</Step>

<Step title="Authenticate Plugin">
1. افتح مكوّن Dodo Payments في Framer
2. انتقل إلى صفحة المصادقة
3. الصق مفتاح API الخاص بك
4. اختر الوضع المناسب (اختبار أو مباشر)
5. انقر **متابعة** للمصادقة

<Warning>
احفظ مفاتيح API الخاصة بك بأمان ولا تشاركها علنًا.
</Warning>
</Step>

<Step title="Manage Products">
بعد المصادقة، سيظهر لك قائمة المنتجات الخاصة بك:

- **المنتجات الموجودة**: تُعرض تلقائيًا
- **لا توجد منتجات**: انقر على زر **زائد (+)** بجوار 'المنتجات'

<Info>
سيتم توجيهك إلى لوحة تحكم Dodo Payments لإنشاء المنتجات مع التسعير والوصف والإعدادات.
</Info>
</Step>

<Step title="Add Buy Buttons">
1. اختر منتجًا من قائمتك المتاحة
2. انقر على الزر `Add Buy Button`
3. سيتم إضافة زر شراء بالنص `Buy {product_name}` إلى اللوحة الخاصة بك

<Warning>
سيتم تعطيل زر إضافة شراء إذا لم تكن تملك الأذونات الصحيحة للمشروع.
</Warning>
</Step>
</Steps>

## التخصيص

<Steps>
<Step title="Select Checkout Experience">
    اختر تجربة الدفع المفضلة لديك:
    - **تراكب (مضمّن)**: تظهر صفحة الدفع كتراكب داخل مشروع Framer الخاص بك، مما يبقي المستخدمين على موقعك.
    - **مستضاف**: يتم إعادة توجيه المستخدمين إلى صفحة دفع مستضافة من Dodo Payments لتجربة سلسة.
</Step>

<Step title="Customize Appearance Options">
    خصص تجربة الدفع من خلال إعدادات المظهر التالية:
    - **نمط الدفع**: اختر بين الثيم الفاتح أو الداكن ليتماشى مع هوية علامتك.
    - **عبارة الزر**: خصص النص (مثال: 'اشتري الآن'، 'شراء'، 'ابدأ الآن').
    - **أنماط الزر**: عدّل الألوان، الخطوط، الأحجام، وغيرها من خصائص التنسيق.
    - **عنوان URL للعودة**: حدد صفحة Framer للعودة إليها بعد إنهاء الدفع.
</Step>
</Steps>

## استكشاف الأخطاء وإصلاحها

<AccordionGroup>
<Accordion title="Authentication Issues">
**المشكلة**: المكوّن لا يُصادق

**الحلول**:
- تأكد من صحة مفتاح API الخاص بك
- احرص على استخدام الوضع الصحيح (اختبار/مباشر)
- تحقق من اتصال الإنترنت لديك
</Accordion>

<Accordion title="Button Creation Issues">
**المشكلة**: زر إضافة شراء معطّل

**الحلول**:
- تأكد من أنك تملك الأذونات المناسبة في مشروع Framer
- احرص على المصادقة بشكل صحيح
- جرّب تحديث المكوّن
</Accordion>

<Accordion title="Product Display Issues">
**المشكلة**: عدم ظهور المنتجات

**الحلول**:
- تأكد من أنك أنشأت منتجات في لوحة تحكم Dodo Payments الخاصة بك
- تحقق من أن مفتاح API لديه الأذونات الصحيحة
- جرّب إعادة المصادقة
</Accordion>
</AccordionGroup>
