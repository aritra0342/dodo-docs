<Tip> سوف تتلقى قائمة من رموز البلدان بصيغة ISO Alpha-2 تمثل البلدان التي **ندعمها حاليًا للدفع**. يمكنك عرض القائمة الكاملة للبلدان المدعومة [هنا](/miscellaneous/list-of-countries-we-accept-payments-from). يتم تحديث هذه القائمة بانتظام مع توسّع نطاقنا. </Tip>

<Tip> ملاحظة: تتضمن الخيارات المتاحة رموز ISO Alpha-2 لـ **جميع البلدان**، وليس فقط تلك التي ندعمها حاليًا. يُرجى الرجوع إلى قائمة البلدان المدعومة لتأكيد التوفر. </Tip>
