<Tip> ستتلقى قائمة بأكواد الدول [ISO Alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) التي تمثل الدول التي **ندعمها حاليًا للمدفوعات**. يمكنك عرض القائمة الكاملة للدول المدعومة [هنا](/miscellaneous/list-of-countries-we-accept-payments-from). يتم تحديث هذه القائمة بانتظام مع توسيع نطاق تغطيتنا. </Tip>

<Tip> ملاحظة: تشمل الخيارات المتاحة أكواد ISO Alpha-2 لـ **جميع الدول**، وليس فقط تلك التي ندعمها حاليًا. يرجى الرجوع إلى قائمة الدول المدعومة لتأكيد التوافر. </Tip>
