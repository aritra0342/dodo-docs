## نظرة عامة

تقوم Dodo Payments بإنشاء فواتير احترافية تلقائيًا لكل معاملة ناجحة، مما يلغي الحاجة إلى إنشاء الفواتير يدويًا ويضمن توثيقًا متسقًا لعملك. يتم تسليم الفواتير على الفور لك ولعملائك عبر البريد الإلكتروني بعد كل عملية دفع.

<Info>
يتم إنشاء الفواتير تلقائيًا لكل من عمليات الشراء لمرة واحدة والمدفوعات المتكررة للاشتراك.
</Info>

## ما الذي يتضمنه الفواتير

تحتوي كل فاتورة مُنشأة على تفاصيل شاملة للمعاملة:

- **معلومات العمل**: اسم عملك، وشعارك، وبيانات الاتصال
- **تفاصيل العميل**: اسم العميل وعنوان البريد الإلكتروني
- **رقم الفاتورة**: معرف فريد للتتبع والمرجع
- **تاريخ المعاملة**: متى تمت معالجة الدفع
- **تفاصيل الدفع**: المبلغ المدفوع، العملة، وطريقة الدفع
- **وصف المنتج/الخدمة**: قائمة مفصلة بالمنتجات أو الاشتراكات المشتراة
- **معلومات الضرائب**: الضرائب المطبقة وأرقام التعريف الضريبي
- **حالة الدفع**: تأكيد الدفع الناجح

<Note>
يتم إنشاء أرقام الفواتير تلقائيًا وتتبع تنسيقًا تسلسليًا لتسهيل التتبع.
</Note>

## كيف يعمل إنشاء الفواتير

<Tabs>
{/* LOCKED_PATTERN_1f557924c92ef240191a271a2acc0963 */}
عندما يُكمل العميل عملية شراء لمرة واحدة:

1. تتم معالجة الدفع بنجاح
2. تُنشأ الفاتورة تلقائيًا مع تفاصيل المعاملة
3. تُرسل الفاتورة عبر البريد الإلكتروني لكل من العميل والتاجر
4. تظهر الفاتورة في قسم **الفواتير** في لوحة التحكم الخاصة بك

تتضمن جميع فواتير الشراء لمرة واحدة تفاصيل المنتج والتسعير وتأكيد الدفع.
</Tab>

{/* LOCKED_PATTERN_0c4b055467985bbff5951606e6691191 */}
بالنسبة لمدفوعات الاشتراك المتكررة:

1. تتم معالجة دفع تجديد الاشتراك تلقائيًا
2. تُنشأ فاتورة لفترة الفوترة
3. تُرسل الفاتورة عبر البريد الإلكتروني للعميل مع تفاصيل الاشتراك
4. تُخزن الفاتورة في لوحة التحكم الخاصة بك للسجلات

تتضمن فواتير الاشتراك فترة الفوترة وتفاصيل خطة الاشتراك وتاريخ التجديد التالي.
</Tab>
</Tabs>

## إدارة الفواتير

### الوصول إلى فواتيرك

انتقل إلى قسم **الفواتير** في لوحة تحكم Dodo Payments لعرض جميع الفواتير المُنشأة. يمكنك:

- **عرض تفاصيل الفاتورة**: انقر على أي فاتورة لرؤية معلومات المعاملة الكاملة
- **تنزيل PDF**: تصدير الفواتير بصيغة PDF لسجلاتك
- **إعادة إرسال إلى العميل**: أعد إرسال الفاتورة عبر البريد الإلكتروني إذا لزم الأمر
- **تصفية والبحث**: ابحث عن فواتير محددة حسب التاريخ، أو العميل، أو رقم الفاتورة

<Tip>
استخدم خيارات البحث والتصفية لتحديد الفواتير بسرعة لأغراض المحاسبة أو دعم العملاء.
</Tip>
