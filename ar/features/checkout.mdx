## العملة التكيفية

تسمح العملة التكيفية للعملاء بالدفع بعملتهم المحلية المفضلة، مما يعزز الثقة ومعدلات التحويل.

### كيف تعمل

1. **تفعيل**: قم بتفعيل العملة التكيفية من إعدادات عملك
2. **اختيار**: يمكن للعملاء تبديل العملات مباشرة عند الدفع
3. **تحويل**: يتم تحويل الأسعار ديناميكيًا باستخدام أسعار الصرف الفورية
4. **عرض**: يتم عرض المبلغ النهائي المستحق بشفافية قبل الدفع

## حساب الضرائب التلقائي

يتم حساب الضرائب تلقائيًا بناءً على موقع الفوترة الخاص بالعميل، مما يضمن الامتثال لمتطلبات ضريبة السلع والخدمات، وضريبة القيمة المضافة، وضريبة المبيعات دون إعداد يدوي.

### كيف يعمل حساب الضرائب

<Steps>
<Step title="كشف الموقع">
  يتم تطبيق قواعد الضرائب بناءً على بلد العميل (والمنطقة حيثما ينطبق).
</Step>

<Step title="التحديثات الديناميكية">
  يتم تحديث مبلغ الضريبة تلقائيًا عندما:
  - يتغير البلد
  - يتم تحديث العنوان
</Step>

<Step title="عرض شفاف">
  يتم عرض تفاصيل الضريبة النهائية بوضوح قبل الدفع.
</Step>
</Steps>

<Tip>
حساب الضرائب مؤتمت بالكامل. لا يتطلب تكوين يدوي للسلع الرقمية القياسية ومنتجات SaaS.
</Tip>

## دعم رقم ضريبة الأعمال

بالنسبة للأعمال المسجلة، يسمح الدفع للعملاء بإدخال رقم ضريبة الأعمال الخاص بهم (مثل رقم ضريبة القيمة المضافة/ضريبة السلع والخدمات).

### ماذا يحدث عند إدخال رقم ضريبة

- يتم التحقق من أهلية الضريبة في الوقت الفعلي
- يتم تطبيق الإعفاءات الضريبية أو قواعد التحويل المعمول بها
- يتم تحديث مبلغ الضريبة على الفور عند الدفع

</Info>

## رموز الخصم

يمكن للعملاء تطبيق رموز الخصم أو الرموز الترويجية التي أنشأتها في لوحة التحكم مباشرة على صفحة الدفع.

### تجربة الدفع

1. يدخل العميل رمز الخصم
2. يتم التحقق من الخصم على الفور
3. يتم عرض السعر المحدث والمدخرات بوضوح

<Frame>
  <img src="/images/checkout/c4.png" alt="إدخال رمز الخصم عند الدفع" style={{ width: '70%' }}/>
</Frame>

### تكامل API

قم بتطبيق رمز خصم مسبقًا أو قم بتمكين حقل إدخال الخصم:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  discount_code: 'WELCOME20', // Pre-apply a code
  feature_flags: {
    allow_discount_code: true // Show discount input field
  },
  return_url: 'https://yoursite.com/return'
});
```

## جمع العناوين الذكي

يدعم الدفع إدخال العناوين بشكل مرن لإكمال أسرع.

### الخيارات المتاحة

| الخيار | الوصف |
|--------|-------------|
| **إكمال عنوان جوجل** | اختيار سريع مع الإكمال التلقائي |
| **الإدخال اليدوي** | تحكم كامل في العناوين الكاملة |
| **اختيار البلد** | يقود منطق الضرائب والامتثال |

<Tip>
جمع العناوين يوازن بين السرعة والدقة والتغطية العالمية لتعظيم التحويل مع ضمان الامتثال.
</Tip>

## سياسة الخصوصية وقبول الشروط

لضمان الشفافية القانونية والامتثال:

- يتم عرض روابط [سياسة الخصوصية](https://dodopayments.com/privacy-policy) و[شروط المشتري](https://dodopayments.com/buyer-terms) بوضوح عند الدفع
- يعترف العملاء صراحةً بهذه قبل إتمام الدفع

<Info>
يساعد ذلك في تلبية متطلبات حماية المستهلك العالمية وخصوصية البيانات بما في ذلك الامتثال للائحة العامة لحماية البيانات.
</Info>

## تكوين جلسة الدفع

تحكم في سلوك الدفع باستخدام واجهة برمجة تطبيقات جلسات الدفع:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  customer: {
    email: 'customer@example.com',
    name: 'Jane Doe'
  },
  billing_currency: 'EUR', // Set specific currency
  discount_code: 'PROMO10',
  feature_flags: {
    allow_discount_code: true
  },
  return_url: 'https://yoursite.com/return',
  metadata: {
    order_ref: 'ORD-12345'
  }
});
```
