## ما هي المدفوعات لمرة واحدة؟

المدفوعات لمرة واحدة هي عمليات شراء ثابتة ومقدمة - مثالية لـ:

- **ترخيص البرمجيات**: فتحات مدى الحياة أو إصدارات رئيسية
- **التنزيلات الرقمية**: كتب إلكترونية، قوالب، رسومات، أو إضافات
- **الخدمات**: تدقيق، حزم توجيه، أو استشارات لمرة واحدة
- **الدورات والتذاكر**: دورات، ورش عمل، أو جلسات مباشرة

## الفوائد الرئيسية

- **إجراءات دفع بسيطة**: لا تجديدات أو منطق تقسيط
- **تلبية فورية**: تسليم التراخيص أو الملفات على الفور
- **أسعار مرنة**: دعم الخصومات والضرائب
- **موجهة للمطورين**: واجهات برمجة تطبيقات واضحة لإنشاء، واسترداد، وwebhooks

## إنشاء منتجات لمرة واحدة

قم بإنشاء منتجات في لوحة تحكم مدفوعات Dodo، ثم قم ببيعها عبر الدفع المستضاف، أو روابط الدفع، أو واجهة برمجة التطبيقات الخاصة بك.

### تكوين المنتج

قم بتكوين الحقول في لوحة التحكم لتعريف كيفية ظهور منتجك وبيعه.

#### تفاصيل المنتج

- **اسم المنتج** (مطلوب): الاسم المعروض في الدفع والفواتير
- **وصف المنتج** (مطلوب): بيان قيمة واضح يراه العملاء قبل الدفع
- **صورة المنتج** (مطلوبة): PNG/JPG/WebP حتى 3 ميغابايت
- **العلامة التجارية**: ربط المنتج بعلامة تجارية معينة للتصميم والبريد الإلكتروني
- **فئة الضريبة** (مطلوبة): اختر الفئة الصحيحة لتحديد قواعد الضريبة

<Tip>
اختر فئة الضريبة الأكثر دقة لضمان تحصيل الضريبة الصحيحة حسب المنطقة.
</Tip>

#### التسعير

- **نوع التسعير**: اختر <b>دفعة واحدة</b>
- **السعر** (مطلوب): رسوم ثابتة مع العملة
- **الخصم المطبق (%)**: نسبة الخصم الاختيارية المعروضة في الدفع والفواتير

<Warning>
يؤثر تغيير تسعير المنتج على عمليات الشراء الجديدة فقط. العملاء الحاليون غير متأثرين.
</Warning>

#### الإعدادات المتقدمة

- **أسعار شاملة الضرائب**: عرض الأسعار شاملة الضرائب المطبقة
- **إنشاء مفاتيح الترخيص**: إصدار مفتاح فريد بعد الشراء. انظر <a href="/features/license-keys">مفاتيح الترخيص</a>
- **تسليم المنتج الرقمي**: تسليم الملفات أو المحتوى تلقائيًا. انظر <a href="/features/digital-product-delivery">تسليم المنتج الرقمي</a>
- **البيانات الوصفية**: إرفاق أزواج مفتاح-قيمة مخصصة للتصنيف الداخلي أو التكاملات. انظر <a href="/api-reference/metadata">البيانات الوصفية</a>

<Tip>
استخدم البيانات الوصفية لتخزين معرفات من نظامك (مثلاً orderId) لمطابقة الأحداث والإيصالات لاحقًا.
</Tip>

## حالات الاستخدام الشائعة

- **أدوات قائمة على الترخيص**: فتح الميزات بمفتاح لمرة واحدة
- **الأصول الرقمية**: سمات، قوالب، مجموعات أيقونات، أو وسائط
- **الخدمات المهنية**: تنفيذ أو إعداد برسوم ثابتة
- **التعليم**: تذاكر الدورات وورش العمل لمرة واحدة

## أمثلة على التكامل

### جلسات الدفع (دفعة واحدة)
إنشاء جلسة دفع مع منتج لمرة واحدة في السلة:

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_one_time', quantity: 1 }
  ]
});
```

## أفضل الممارسات

- **توضيح التسليمات**: أخبر المشترين بالضبط ما يحصلون عليه وكيف يتم تسليمه
- **التسليم الفوري**: استخدم مفاتيح الترخيص أو التسليم الرقمي للوصول الفوري
- **التتبع باستخدام البيانات الوصفية**: تخزين المعرفات الخاصة بك لمصالحة سهلة

<Info>
المدفوعات لمرة واحدة هي أسرع طريق لبيع المنتجات ذات السعر الثابت. ابدأ ببساطة، وحقق التوصيل، وكرر عرضك بناءً على التحويل والتعليقات.
</Info>
