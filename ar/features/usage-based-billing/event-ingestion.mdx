الأحداث هي أساس الفوترة المعتمدة على الاستخدام. أرسل الأحداث عند حدوث إجراءات قابلة للفوترة، وتجمع العدادات هذه الأحداث في رسوم.

توثيق API كامل مع أمثلة ورموز الاستجابة.
</Card>

## هيكل الحدث

{/* LOCKED_PATTERN_4d7f2e9ce45e7323dd2d626df4a7c5b8 */}
معرف فريد. استخدم UUIDs أو اجمع بين معرف العميل + الطابع الزمني + الإجراء.
</ParamField>

<ParamField body="customer_id" type="string" required>
معرف عميل Dodo Payments. يجب أن يكون عميلًا موجودًا وصالحًا.
</ParamField>

<ParamField body="event_name" type="string" required>
نوع الحدث الذي يطابق اسم حدث العداد لديك (مع مراعاة حساسية حالة الأحرف). أمثلة: `api.call`، `image.generated`
</ParamField>
</Accordion>

{/* LOCKED_PATTERN_94ee55474fc7877494948c444fe08c91 */}

<ParamField body="timestamp" type="string">
طابع زمني بتنسيق ISO 8601. الافتراضي هو وقت الخادم إذا تم حذفه. أدرجه للفوترة الدقيقة مع الأحداث المؤجلة/المجمعة.
</ParamField>

<ParamField body="metadata" type="object">
خصائص إضافية للتجميع والتصفية:
- القيم الرقمية: `bytes`، `tokens`، `duration_ms`
- الفلاتر: `endpoint`، `method`، `quality`

```javascript
metadata: {
  endpoint: "/v1/orders",
  method: "POST",
  tokens: 1024
}
```

</ParamField>
</Accordion>

## إرسال الأحداث

<CodeGroup>

```javascript Single Event
await fetch('https://test.dodopayments.com/events/ingest', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${process.env.DODO_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    events: [{
      event_id: "api_call_1234",
      customer_id: "cus_abc123",
      event_name: "api.call",
      metadata: { endpoint: "/v1/orders" }
    }]
  })
});
```

```javascript Batch Events
await fetch('https://test.dodopayments.com/events/ingest', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${process.env.DODO_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    events: [
      { event_id: "call_1", customer_id: "cus_abc", event_name: "api.call" },
      { event_id: "call_2", customer_id: "cus_abc", event_name: "api.call" },
      { event_id: "call_3", customer_id: "cus_abc", event_name: "api.call" }
    ]
  })
});
```

```python Python
import requests

requests.post(
    'https://test.dodopayments.com/events/ingest',
    headers={'Authorization': f'Bearer {api_key}', 'Content-Type': 'application/json'},
    json={'events': [{'event_id': 'call_1', 'customer_id': 'cus_abc', 'event_name': 'api.call'}]}
)
```

```curl cURL
curl -X POST 'https://test.dodopayments.com/events/ingest' \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{"events": [{"event_id": "call_1", "customer_id": "cus_abc", "event_name": "api.call"}]}'
```

</CodeGroup>

<Tip>
نجمع ما يصل إلى 100 حدث لكل طلب لتحسين الأداء.
</Tip>

## استكشاف الأخطاء وإصلاحها

<AccordionGroup>
{/* LOCKED_PATTERN_fc1788ddf0982a3a2fe9f6e81812c541 */}
- يجب أن يطابق اسم الحدث العداد تمامًا (مع مراعاة حساسية حالة الأحرف)
- يجب أن يكون معرف العميل موجودًا
- تحقق من أن عوامل التصفية في العداد لا تستبعد الأحداث
- تحقق من أن الطوابع الزمنية حديثة
</Accordion>

{/* LOCKED_PATTERN_1af6f154c82eaebbb38a748cef2be54a */}
تحقق من أن مفتاح API صحيح واستخدم التنسيق: `Bearer YOUR_API_KEY`
</Accordion>

{/* LOCKED_PATTERN_bfe0427c6207c44242acb102b2467be5 */}
تأكد من وجود جميع الحقول المطلوبة: `event_id`، `customer_id`، `event_name`
</Accordion>

{/* LOCKED_PATTERN_61cd934926cd73f663eac6935ef56a1b */}
- يجب أن تطابق مفاتيح البيانات الوصفية خاصية "Over Property" في العداد تمامًا
- استخدم أرقامًا، لا سلاسل: `tokens: 150` وليس `tokens: "150"`
</Accordion>
</AccordionGroup>
