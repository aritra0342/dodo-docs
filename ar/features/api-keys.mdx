## مقدمة

تتيح لك عملية توليد مفتاح API للمستخدمين المصادقة بشكل آمن ودمج أنظمتهم مع منصة Dodo Payments للوصول السلس إلى وظائف الدفع والمنتجات والاشتراكات.

**ملاحظة :** ستقتصر مفاتيح API التي تم إنشاؤها في وضع الاختبار على بيئة وضع الاختبار والعكس صحيح لوضع التشغيل
