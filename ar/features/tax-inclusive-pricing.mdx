## ما هي الأسعار الشاملة للضرائب؟

تتيح لك الأسعار الشاملة للضرائب تضمين الضرائب المطبقة مباشرة في سعر المنتج المعروض. يرى العملاء مبلغًا نهائيًا واحدًا مسبقًا لتجربة أكثر شفافية وصديقة للتحويل. عند الدفع، يقوم Dodo بحساب الجزء غير الضريبي ومبلغ الضريبة من السعر المعروض بناءً على موقع العميل، ويفصلها بوضوح في الفاتورة.

<Info>
يظل الضريبة النهائية متغيرة حسب موقع العميل وفئة الضريبة. يظهر Checkout والفواتير الجزء المحسوب من الضريبة المستمد من السعر المعروض.
</Info>

## الفوائد الرئيسية

- **وضوح للعملاء**: سعر نهائي واحد معروض مسبقًا
- **امتثال حسب المنطقة**: يتم حساب الضريبة حسب الموقع ويتم فصلها
- **فواتير دقيقة**: تظهر الفواتير المبلغ الصافي والضريبة المشتقة من السعر
- **صديقة للتحويل**: تقليل المفاجآت عند الدفع

## إعداد المنتج

قم بإنشاء أو تعديل منتج في لوحة التحكم الخاصة بك، ثم قم بتمكين الأسعار الشاملة للضرائب تحت إعدادات الأسعار.

#### التسعير

- **السعر** (مطلوب): السعر الأساسي الذي يتم عرضه للعميل.
- **أسعار شاملة للضرائب**: قم بالتبديل لتعتبر السعر المعروض شاملًا للضرائب. سيقوم Dodo بحساب المبلغ الصافي (قبل الضريبة) والجزء الضريبي عند الدفع بناءً على موقع العميل وفئة الضريبة.

<Warning>
تنطبق الإعدادات المحدثة التي تشمل الضريبة فقط على عمليات الشراء المستقبلية. لا يتم تعديل عمليات الشراء الحالية أو الاشتراكات النشطة بأثر رجعي.
</Warning>

<Tip>
استخدم نصوص منتجات متناسقة مثل “Tax inclusive” حتى يتمكن العملاء من فهم الإجماليات بنظرة سريعة.
</Tip>

## أمثلة على التكامل

### إنشاء منتج بأسعار شاملة للضرائب

```typescript
const product = await client.products.create({
  name: 'Pro Plan',
  description: 'All features included',
  price: {
    currency: 'USD',
    price: 10000
    tax_inclusive: true
  }
});
```

### تحديث منتج موجود لتبديل شامل للضرائب

```typescript
await client.products.update(product.id, {
  price: {
    tax_inclusive: true
  }
});
```

## أفضل الممارسات

- **اختبر في بيئة الاختبار** للتحقق.
