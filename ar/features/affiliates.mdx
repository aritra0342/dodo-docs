## المقدمة

توجهك هذه الدليل خلال كيفية تتبع إحالات الشركاء، التعامل مع أحداث العمولة، وزيادة إيراداتك من خلال شراكات موثوقة مع الشركاء عبر مدفوعات دودي. نحن نقدم تكاملات مع منصات الشركاء الرائدة بما في ذلك **أفونسو** و **دب بارتنرز** لمساعدتك في إدارة وتتبع برامج الشركاء الخاصة بك بسلاسة.

## خيارات التكامل

تتكامل مدفوعات دودي مع منصتين رائدتين لإدارة الشركاء:

- **[أفونسو](https://affonso.io)**: منصة شاملة لإدارة الشركاء مع تتبع عمولات آلي، تقارير في الوقت الحقيقي، ولوحات معلومات مخصصة للشركاء.
- **[دب بارتنرز](https://dub.co)**: استخدم إدارة الروابط من دب وتتبع التحويلات لتفعيل برنامج الشركاء الخاص بك مع تحليلات تفصيلية للتخصيص.

اختر المنصة التي تناسب احتياجاتك، أو استخدم كلاهما لحالات استخدام مختلفة.

## تكامل أفونسو

لقد تعاوننا مع أفونسو، منصة رائدة لإدارة الشركاء لتفعيل مبيعاتك. اتبع الخطوات أدناه لربط حساب مدفوعات دودي الخاص بك مع أفونسو وبدء تتبع المبيعات المدفوعة من الشركاء.

<Steps>
  <Step title="الاتصال مع مدفوعات دودي">
    للبدء، قم بربط حساب مدفوعات دودي الخاص بك عن طريق إضافة مفتاح API الخاص بك.

    ### كيفية الحصول على مفتاح API الخاص بك من مدفوعات دودي

    - قم بتسجيل الدخول إلى **لوحة معلومات مدفوعات دودي**
    - انتقل إلى قسم **المطور** في القائمة اليسرى
    - انقر على **مفاتيح API**
    - اختر **إضافة مفتاح API**
    - أعطه اسمًا (مثل "تكامل أفونسو")
    - انقر على **إنشاء** و **انسخ** مفتاح API الخاص بك

    ### أضف مفتاح API الخاص بك إلى أفونسو

    - افتح **لوحة معلومات أفونسو عبر** [affonso.io](https://affonso.io/)
    - انتقل إلى **تفاصيل البرنامج > الاتصال**
    - انقر على بطاقة **تكامل مدفوعات دودي**
    - ألصق مفتاح API الخاص بك من مدفوعات دودي
    - انقر على **الاتصال**

    <Callout type="info">
      سيتم تشفير مفتاح API الخاص بك بشكل آمن بواسطة أفونسو.
    </Callout>
  </Step>

  <Step title="إضافة Webhook إلى مدفوعات دودي">
    بمجرد الاتصال، ستتلقى عنوان URL فريد من أفونسو. هذا يمكّن تتبع المدفوعات وتفعيل عمولات الشركاء.

    ### إضافة عنوان URL لـ Webhook في مدفوعات دودي

    - في لوحة معلومات مدفوعات دودي الخاصة بك، انتقل إلى **المطور → Webhooks**
    - انقر على **إضافة Webhook**
    - ألصق **عنوان URL لـ Webhook أفونسو** (من إعدادات برنامجك)
    - احفظ الـ Webhook

    ### إضافة سر Webhook في أفونسو

    - بعد حفظ الـ Webhook في مدفوعات دودي، **انسخ مفتاح التوقيع** المقدم
    - ألصقه مرة أخرى في أفونسو تحت **إعدادات Webhook مدفوعات دودي**

    <Callout type="info">
      هذا يسمح لأفونسو بالتحقق بشكل آمن من الأحداث الواردة من مدفوعات دودي.
    </Callout>
  </Step>

  <Step title="إضافة سكربت التتبع">
    ضع بكسل أفونسو على موقعك لبدء تتبع زيارات الشركاء والتحويلات.

    ```html
    <!-- Place in <head> tag -->
    <script async defer
    src="https://affonso.io/js/pixel.min.js"
    data-affonso="YOUR_PUBLIC_PROGRAM_ID"
    data-cookie_duration="YOUR_COOKIE_DURATION">
    </script>
    ```

    هذا السكربت:
    - يتتبع زيارات الشركاء
    - يضع ملف تعريف الارتباط مع معرف الإحالة (`affonso_referral`)
    - يمكّن تتبع التسجيلات والمدفوعات

    <Callout type="info">
      ضع هذا على جميع النطاقات التي يزور فيها المستخدمون أو يتحولون (مثل موقع التسويق، التطبيق).
    </Callout>
  </Step>

  <Step title="تتبع تسجيلات المستخدمين">
    للحصول على رؤية أفضل في القمع وتحسين أداء الشركاء، تتبع تسجيلات المستخدمين كعملاء محتملين.

    ```js
    // After successful registration
    window.Affonso.signup(userEmail);
    ```

    هذا سيسجل إحالة جديدة كـ **عميل محتمل** إذا جاء المستخدم عبر رابط الشريك
    - سيتم عرضها في لوحة المعلومات الخاصة بك وإحصائيات الشريك

    <Callout type="info">
      أفضل ممارسة: قم بتفعيل هذا بعد التسجيل الناجح أو التأكيد المزدوج (DOI).
    </Callout>
  </Step>

  <Step title="تمرير بيانات الإحالة إلى مدفوعات دودي">
    لتخصيص العمولات بشكل صحيح، مرر معرف الإحالة من أفونسو إلى مدفوعات دودي أثناء الدفع.

    ### تكامل المدفوعات

    ```js
    import DodoPayments from 'dodopayments';

    const referralId = window.affonso_referral;

    const client = new DodoPayments({
      bearerToken: process.env['DODO_PAYMENTS_API_KEY'],
    });

    async function main() {
      const payment = await client.payments.create({
        payment_link: true,
        billing: { city: 'city', country: 'AF', state: 'state', street: 'street', zipcode: 0 },
        customer: { email: 'email@email.com', name: 'name' },
        product_cart: [{ product_id: 'product_id', quantity: 0 }],
        metadata: {
          affonso_referral: referralId,
        },
      });

      console.log(payment.payment_id);
    }

    main();
    ```

    ### تكامل الاشتراك

    ```js
    import DodoPayments from 'dodopayments';

    const referralId = window.affonso_referral;

    const client = new DodoPayments({
      bearerToken: process.env['DODO_PAYMENTS_API_KEY'],
    });

    async function main() {
      const subscription = await client.subscriptions.create({
        billing: { city: 'city', country: 'IN', state: 'state', street: 'street', zipcode: 89789 },
        customer: { customer_id: 'customer_id' },
        product_id: 'product_id',
        payment_link: true,
        return_url: 'https://example.com/success',
        quantity: 1,
        metadata: {
          affonso_referral: referralId,
        },
      });

      console.log(subscription.subscription_id);
    }

    main();
    ```

    ### روابط الدفع

    ```js
    // Original Dodo Payments Link
    const originalLink = 'https://checkout.dodopayments.com/buy/<YOUR_PRODUCT_ID>';

    // Append referral ID as reference_id
    const linkWithReferral = `${originalLink}?metadata_affonso_referral=${window.affonso_referral}`;
    ```

    <Callout type="info">
      لمزيد من التفاصيل، راجع [وثائق API لمدفوعات دودي](https://docs.dodopayments.com/api-reference/introduction)
    </Callout>
  </Step>

  <Step title="اختبار تكاملك">
    قبل الإطلاق علنًا، اختبر إعدادك باستخدام روابط الشركاء الحية والمعاملات التجريبية.

    **قائمة التحقق للاختبار:**
    - ✅ إنشاء شريك وتوليد رابط تتبع
    - ✅ زيارة موقعك عبر رابط الشريك
    - ✅ تأكد من تحميل سكربت أفونسو وأن ملف تعريف الارتباط `affonso_referral` تم تعيينه
    - ✅ إكمال تسجيل (اختياري)
    - ✅ إكمال الدفع عبر مدفوعات دودي
    - ✅ تأكيد التتبع والتخصيص في لوحة معلومات أفونسو
  </Step>
</Steps>

## تكامل دب بارتنرز

يمكنك أيضًا استخدام [دب بارتنرز](https://dub.co) لتتبع إحالات الشركاء والتحويلات. تتيح لك ميزة تتبع التحويلات من دب تخصيص المبيعات إلى روابط الشركاء الخاصة بك، مما يوفر تحليلات تفصيلية حول أي الشركاء يحققون أكبر إيرادات.

<Info>
للحصول على تعليمات إعداد مفصلة، راجع [دليل تكامل دب](/integrations/dub) الذي يغطي كيفية تتبع أحداث تحويل المبيعات وتخصيص الإيرادات إلى روابط الشركاء الخاصة بك.
</Info>

### إعداد سريع مع دب بارتنرز

1. **تفعيل تتبع التحويلات** في لوحة معلومات دب لروابط الشركاء
2. **التقاط معرف نقرة دب** (`dub_id` ملف تعريف الارتباط) عند إنشاء جلسات الدفع
3. **تخزين معرف النقرة** في بيانات الدفع مع معلومات العميل
4. **تكوين تكامل Webhook** لإرسال بيانات المبيعات إلى دب عند نجاح المدفوعات

يتتبع التكامل تلقائيًا:
- المدفوعات لمرة واحدة
- تفعيل الاشتراكات
- المدفوعات المتكررة للاشتراكات
