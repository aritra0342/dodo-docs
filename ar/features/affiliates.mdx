## خطوات التكامل

لقد تعاوننا مع Affonso، وهي منصة رائدة في إدارة الشركاء لدعم مبيعاتك. اتبع الخطوات أدناه لربط حساب Dodo Payments الخاص بك مع Affonso وبدء تتبع المبيعات المدفوعة من الشركاء.

<Steps>
  <Step title="الاتصال بـ Dodo Payments">
    للبدء، قم بربط حساب Dodo Payments الخاص بك عن طريق إضافة مفتاح API الخاص بك.

    ### كيفية الحصول على مفتاح API الخاص بـ Dodo Payments

    - قم بتسجيل الدخول إلى **لوحة معلومات Dodo Payments** الخاصة بك
    - انتقل إلى قسم **المطور** في القائمة اليسرى
    - انقر على **مفاتيح API**
    - اختر **إضافة مفتاح API**
    - أعطه اسمًا (مثل "تكامل Affonso")
    - انقر على **إنشاء** و**انسخ** مفتاح API الخاص بك

    ### أضف مفتاح API الخاص بك إلى Affonso

    - افتح **لوحة معلومات Affonso عبر** [affonso.io](https://affonso.io/)
    - انتقل إلى **تفاصيل البرنامج > الاتصال**
    - انقر على بطاقة **تكامل Dodo Payments**
    - ألصق مفتاح API الخاص بـ Dodo Payments
    - انقر على **الاتصال**

    <Callout type="info">
      سيتم تشفير مفتاح API الخاص بك بشكل آمن بواسطة Affonso.
    </Callout>
  </Step>

  <Step title="أضف ويب هوك إلى Dodo Payments">
    بمجرد الاتصال، ستتلقى عنوان ويب هوك فريد من Affonso. هذا يمكّن تتبع المدفوعات وتحفيز عمولة الشركاء.

    ### أضف عنوان ويب هوك في Dodo Payments

    - في لوحة معلومات Dodo Payments الخاصة بك، انتقل إلى **المطور → ويب هوكس**
    - انقر على **إضافة ويب هوك**
    - ألصق **عنوان ويب هوك الخاص بـ Affonso** (من إعدادات برنامجك)
    - احفظ الويب هوك

    ### أضف سر ويب هوك في Affonso

    - بعد حفظ الويب هوك في Dodo Payments، **انسخ مفتاح التوقيع** المقدم
    - ألصقه مرة أخرى في Affonso تحت **إعدادات ويب هوك Dodo Payments**

    <Callout type="info">
      هذا يسمح لـ Affonso بالتحقق بشكل آمن من الأحداث الواردة من Dodo Payments.
    </Callout>
  </Step>

  <Step title="أضف نص التتبع">
    ضع بكسل Affonso على موقعك لبدء تتبع زيارات الشركاء والتحويلات.

    ```html
    <!-- Place in <head> tag -->
    <script async defer
    src="https://affonso.io/js/pixel.min.js"
    data-affonso="YOUR_PUBLIC_PROGRAM_ID"
    data-cookie_duration="YOUR_COOKIE_DURATION">
    </script>
    ```

    هذا النص:
    - يتتبع زيارات الشركاء
    - يضع ملف تعريف الارتباط مع معرف الإحالة (`affonso_referral`)
    - يمكّن تتبع التسجيلات والمدفوعات

    <Callout type="info">
      ضع هذا على جميع النطاقات التي يزور المستخدمون أو يتحولون فيها (مثل موقع التسويق، التطبيق).
    </Callout>
  </Step>

  <Step title="تتبع تسجيلات المستخدمين">
    للحصول على رؤية أفضل في القمع وتحسين أداء الشركاء، تتبع تسجيلات المستخدمين كقيادات.

    ```js
    // After successful registration
    window.Affonso.signup(userEmail);
    ```

    هذا سيسجل إحالة جديدة كـ **قيادة** إذا جاء المستخدم عبر رابط شريك
    - سيتم عرضها في لوحة المعلومات الخاصة بك وإحصائيات الشريك

    <Callout type="info">
      أفضل ممارسة: قم بتحفيز هذا بعد التسجيل الناجح أو التأكيد المزدوج (DOI).
    </Callout>
  </Step>

  <Step title="تمرير بيانات الإحالة إلى Dodo Payments">
    لنسب العمولات بشكل صحيح، مرر معرف الإحالة من Affonso إلى Dodo Payments أثناء الدفع.

    ### تكامل المدفوعات

    ```js
    import DodoPayments from 'dodopayments';

    const referralId = window.affonso_referral;

    const client = new DodoPayments({
      bearerToken: process.env['DODO_PAYMENTS_API_KEY'],
    });

    async function main() {
      const payment = await client.payments.create({
        payment_link: true,
        billing: { city: 'city', country: 'AF', state: 'state', street: 'street', zipcode: 0 },
        customer: { email: 'email@email.com', name: 'name' },
        product_cart: [{ product_id: 'product_id', quantity: 0 }],
        metadata: {
          affonso_referral: referralId,
        },
      });

      console.log(payment.payment_id);
    }

    main();
    ```

    ### تكامل الاشتراك

    ```js
    import DodoPayments from 'dodopayments';

    const referralId = window.affonso_referral;

    const client = new DodoPayments({
      bearerToken: process.env['DODO_PAYMENTS_API_KEY'],
    });

    async function main() {
      const subscription = await client.subscriptions.create({
        billing: { city: 'city', country: 'IN', state: 'state', street: 'street', zipcode: 89789 },
        customer: { customer_id: 'customer_id' },
        product_id: 'product_id',
        payment_link: true,
        return_url: 'https://example.com/success',
        quantity: 1,
        metadata: {
          affonso_referral: referralId,
        },
      });

      console.log(subscription.subscription_id);
    }

    main();
    ```

    ### روابط الدفع

    ```js
    // Original Dodo Payments Link
    const originalLink = 'https://checkout.dodopayments.com/buy/<YOUR_PRODUCT_ID>';

    // Append referral ID as reference_id
    const linkWithReferral = `${originalLink}?metadata_affonso_referral=${window.affonso_referral}`;
    ```

    <Callout type="info">
      لمزيد من التفاصيل، راجع [وثائق API الخاصة بـ Dodo Payments](https://docs.dodopayments.com/api-reference/introduction)
    </Callout>
  </Step>

  <Step title="اختبار تكاملك">
    قبل الإطلاق علنًا، اختبر إعدادك باستخدام روابط الشركاء الحية والمعاملات التجريبية.

    **قائمة التحقق للاختبار:**
    - ✅ إنشاء شريك وتوليد رابط تتبع
    - ✅ زيارة موقعك عبر رابط الشريك
    - ✅ التأكد من تحميل نص Affonso وأن ملف تعريف الارتباط `affonso_referral` تم تعيينه
    - ✅ إكمال تسجيل (اختياري)
    - ✅ إكمال الدفع عبر Dodo Payments
    - ✅ تأكيد التتبع والنسبة في لوحة معلومات Affonso الخاصة بك
  </Step>
</Steps>
