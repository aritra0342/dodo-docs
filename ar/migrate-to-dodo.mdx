**هل تنتقل من مزود دفع آخر إلى مدفوعات دودي؟** تجعل أداة الترحيل لدينا من السهل نقل **المنتجات والعملاء وأكواد الخصم دون فقدان أي بيانات.** العملية آمنة وموجهة ويمكن إتمامها في بضع دقائق فقط.

نحن ندعم حاليًا عمليات الترحيل من **Lemon Squeezy وStripe وPolar.sh وPaddle**.

## ابدأ الآن

أولاً، قم بتثبيت أداة الترحيل:

```bash
npm install -g dodo-migrate
```

<Note>
ستحتاج إلى تثبيت Node.js إصدار 18 أو أحدث على جهاز الكمبيوتر الخاص بك.
</Note>

## ترحيل بياناتك

### إذا كنت قادمًا من Lemon Squeezy:

```bash
dodo-migrate lemonsqueezy
```

### إذا كنت قادمًا من Stripe:

```bash
dodo-migrate stripe
```

### إذا كنت قادمًا من Polar.sh:

```bash
dodo-migrate polar
```

### إذا كنت قادمًا من Paddle:

```bash
dodo-migrate paddle
```

ستوجهك الأداة خلال كل شيء خطوة بخطوة. ستحتاج إلى:
- مفتاح API لمزودك (Lemon Squeezy وStripe وPolar.sh وPaddle، إلخ.)
- مفتاح API لمدفوعات دودي  
- ما إذا كنت تريد الاختبار أولاً (موصى به) أو الانتقال مباشرة
- أي علامة تجارية من مدفوعات دودي ترغب في الترحيل إليها

## لماذا تستخدم أداة الترحيل لدينا؟

- **لا فقدان للبيانات** - يتم نقل كل شيء بأمان
- **اختبر أولاً** - جربه في وضع الاختبار قبل الانتقال مباشرة
- **عملية موجهة** - نحن نوجهك خلال كل خطوة
- **استئناف إذا لزم الأمر** - إذا حدث خطأ ما، يمكنك استئناف من حيث توقفت

## الاستخدام المتقدم

إذا كنت تفضل تشغيل الترحيل دون مطالبات، يمكنك تقديم جميع التفاصيل مسبقًا:

### ترحيل Lemon Squeezy:

```bash
dodo-migrate lemonsqueezy \
  --provider-api-key=lsq_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX
```

### ترحيل Stripe:

```bash
dodo-migrate stripe \
  --provider-api-key=sk_test_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX \
  --migrate-types=products,coupons
```

### ترحيل Polar.sh:

```bash
dodo-migrate polar \
  --provider-api-key=polar_org_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX \
  --migrate-types=products,discounts,customers
```

### ترحيل Paddle:

```bash
dodo-migrate paddle \
  --provider-api-key=paddle_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX
```
