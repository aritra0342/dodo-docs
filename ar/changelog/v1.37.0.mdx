## ميزات جديدة 🚀

### لوحة تحكم تحليلية محسّنة

لقد قمنا بتوسيع لوحة التحكم التحليلية مع رؤى شاملة حول الإيرادات والاحتفاظ لمساعدتك في فهم أداء عملك بشكل أفضل.

#### تحليلات تفصيل الإيرادات

تتبع مقاييس الإيرادات المتكررة مع تفصيلات دقيقة:

- **الإيرادات المتكررة السنوية المتوقعة**: توقعات الإيرادات المتكررة السنوية مع تحليل الاتجاهات
- **تفصيل الإيرادات المتكررة الشهرية ومعدل النمو**: تحليل الإيرادات المتكررة الشهرية مع مقاييس النمو من شهر إلى شهر  
- **مخطط الإيرادات المتكررة الجديدة**: تمثيل بصري للإيرادات الناتجة عن اكتساب عملاء جدد
- **مخطط الإيرادات المتكررة من التوسع**: تتبع نمو الإيرادات من ترقيات وتوسعات العملاء الحاليين
- **مخطط الإيرادات المتكررة المفقودة**: مراقبة الإيرادات المفقودة من إلغاء العملاء وتخفيضات الاشتراك
- **مخطط الإيرادات المتكررة الصافية الجديدة**: تأثير الإيرادات الصافية التي تجمع بين الإيرادات الجديدة، والتوسع، والإيرادات المفقودة

#### تحليلات الاحتفاظ

راقب أنماط احتفاظ العملاء والإلغاء:

- **معدل إلغاء العملاء**: نسبة العملاء الذين يلغون اشتراكاتهم
- **معدل إلغاء الإيرادات**: نسبة الإيرادات المفقودة من الإلغاءات والتخفيضات
- **اتجاهات معدل الإلغاء**: تتبع تاريخي لكل من أنماط إلغاء العملاء والإيرادات بمرور الوقت
- **مصفوفة احتفاظ المستخدمين**: تحليل المجموعات يظهر معدلات احتفاظ العملاء عبر فترات زمنية مختلفة

### تقارير 2.0

لقد قمنا بإعادة تصميم قسم التقارير بالكامل مع قدرات تقارير أكثر شمولاً وتفصيلاً. انتقل إلى **الأعمال > التقارير** للوصول إلى مجموعة التقارير الجديدة.

التقارير المتاحة تشمل:

- **تقرير المدفوعات**: سجل تفصيلي لجميع المدفوعات المعالجة مع تتبع حالة المعاملات
- **تقرير الاسترداد**: سجل شامل لجميع أنشطة الاسترداد مع معلومات تفصيلية عن الاسترداد
- **تقرير المدفوعات**: ملخص للمدفوعات التي تمت إلى حسابك
- **تقرير المبيعات حسب المنتج**: رؤى حول أداء المبيعات مع مقاييس الإيرادات ونمو الأعمال
- **تقرير فشل المدفوعات**: ملخص لمحاولات الدفع الفاشلة مع أسباب الفشل
- **تقرير العملاء**: ملخص المدفوعات منظم حسب العميل
- **تقرير المنازعات**: ملخص لمنازعات المدفوعات مع حالة الحل
- **تقرير ملخص الحساب**: نظرة عامة كاملة وملخص للحساب

### خادم MCP عن بُعد من Dodo Payments

يمكنك الآن استخدام خادم MCP عن بُعد المستضاف على [https://mcp.dodopayments.com](https://mcp.dodopayments.com) للتكامل الفوري مع MCP—لا حاجة لإعداد محلي.

</Note>

<Steps>
<Step title="Open the Remote MCP Server">
    قم بزيارة <a href="https://mcp.dodopayments.com" target="_blank">https://mcp.dodopayments.com</a> في متصفحك.
</Step>
<Step title="Copy the MCP JSON configuration">
    على الموقع، انسخ تكوين JSON الخاص بـ MCP المزود لعميلك.
</Step>
<Step title="Configure your AI application">
    الصق التكوين في إعدادات MCP لتطبيقك (على سبيل المثال، Cursor أو Claude).
    <Tip>
    بالنسبة إلى Cursor، يجب أن يبدو التكوين الخاص بك هكذا:
    </Tip>

    ```json
    {
      "mcpServers": {
        "dodopayments_api": {
          "command": "npx",
          "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
        }
      }
    }
    ```

</Step>
<Step title="Complete authentication and environment setup">
    بعد إضافة تكوين JSON، ستظهر نافذة منبثقة تطلب منك:
    - أدخل مفتاح واجهة برمجة تطبيقات Dodo Payments الخاص بك
    - حدد البيئة (اختبار أو مباشر)
    - حدد عميل MCP الخاص بك ضمن خيارات التكوين
</Step>
<Step title="Login and Approve">
    - انقر على زر تسجيل الدخول والموافقة
</Step>
<Step title="Verify successful connection">
    بمجرد اكتمال الإعداد، سترى شاشة النجاح.
    <Check>
    أنت الآن جاهز لاستخدام خادم MCP الخاص بـ Dodo Payments!
    </Check>
</Step>
</Steps>
