## ميزات جديدة 🚀

- **مدفوعات بقيمة صفر دولار**: تمت إضافة دعم لمدفوعات بقيمة 0 دولار ورموز خصم بنسبة 100% للمنتجات لمرة واحدة، مما يتيح تقديم منتجات مجانية وحملات ترويجية.

- **حد السحب المخصص**: تمت إضافة القدرة على تحديد الحد الأدنى لمبالغ السحب لمحافظ الدولار الأمريكي، مما يساعد التجار على تحسين معالجة السحب وتقليل الرسوم للمعاملات الصغيرة. يمكنك الوصول إلى هذه الميزة في `Payouts ▸ USD Wallet ▸ Edit Payout Threshold`.
  
  <Frame>
    <img src="/images/changelog/payout-threshold.png" alt="صورة غلاف حد السحب"/>
  </Frame>

- **توحيد API**: تم تنفيذ رموز وأخطاء API موحدة كجزء من مبادرتنا المستمرة لتحسين API. هذه التحسينات تجعل API أكثر اتساقًا وسهولة في الاستخدام للمطورين. لمزيد من المعلومات التفصيلية، قم بزيارة [وثائق رموز الأخطاء](/api-reference/error-codes).

- **توحيد رموز رفض المعاملات**: تم تقديم رموز ورسائل موحدة لرفض المعاملات، مما يوفر رؤى أوضح حول أسباب الفشل ويميز بين أخطاء المستخدم ومشكلات النظام. تعرف على المزيد في [وثائق فشل المعاملات](/api-reference/transaction-failures).
