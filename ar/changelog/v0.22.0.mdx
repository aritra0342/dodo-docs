## ميزات جديدة 🚀

- **أول دفعة للاشتراك**: الميزة الأكثر طلبًا هنا! الآن، ستكون هناك معاملة واحدة للدفع الأول، تجمع بين التفويض ومبلغ الاشتراك الفعلي.

- **طريقة دفع UPI QR**: تمت إضافة UPI QR كطريقة دفع للسوق الهندية، مما يسمح للعملاء بالدفع باستخدام رموز UPI QR.
    
    <img src="/images/changelog/upi-qr.png" alt="ميزة UPI QR"/>

- **رقم التعريف الضريبي عند الخروج**: يمكن للعملاء الآن إضافة رقم التعريف الضريبي الخاص بهم في صفحة الخروج لأغراض الفوترة والضرائب.

- **إعادة هيكلة لوحة التحكم**: تم إعادة هيكلة لوحة تحكم Dodo Payments بالكامل لتكون أكثر سهولة في الاستخدام وودية للمستخدم.

- **أدلة التوجيه الذاتي**: يمكن للمستخدمين الجدد الآن البدء في استخدام Dodo Payments باستخدام أدلة التوجيه الذاتي.

- **دعم متعدد اللغات**: تمت إضافة دعم لأكثر من 14 لغة في صفحة الخروج، مما يسمح للعملاء بالدفع بلغتهم المفضلة. اللغات المدعومة تشمل:
    - الإنجليزية (en)
    - العربية (ar) // Arabic
    - الكتالونية (ca) // Catalan
    - الصينية (cn) // Chinese
    - الألمانية (de) // German
    - الإسبانية (es) // Spanish
    - الفرنسية (fr) // French
    - العبرية (he) // Hebrew
    - الإيطالية (it) // Italian
    - اليابانية (ja) // Japanese
    - الهولندية (nl) // Dutch
    - البولندية (pl) // Polish
    - البرتغالية (pt) // Portuguese
    - السويدية (sv) // Swedish

- **تجديد تصميم PDF الفاتورة**: تم منح PDF الفواتير تصميمًا جديدًا، أوضح وأكثر فعالية.

- **واجهة مساعدة مركزية**: تمت إضافة واجهة مساعدة مركزية لتبسيط وتحسين الدعم.
    
    <img src="/images/changelog/help-widget.png" alt="واجهة المساعدة"/>

## تحسينات وإصلاحات الأخطاء 🔧

- **تحديد معدل الاستخدام**: تمت إضافة تحديد معدل الاستخدام في واجهة برمجة التطبيقات للفواتير لمنع الإساءة وضمان استقرار النظام.

- **البيانات الوصفية في الروابط الثابتة للدفع**: تمت إضافة دعم للبيانات الوصفية في الروابط الثابتة للدفع. لمزيد من التفاصيل، يرجى الرجوع إلى [التوثيق](/developer-resources/integration-guide#1-static-payment-links).

- **تحسينات وإصلاحات طفيفة**: تحسينات وإصلاحات طفيفة متنوعة لتحسين الأداء العام وتجربة المستخدم.
