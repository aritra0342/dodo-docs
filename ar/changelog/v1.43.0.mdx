## ميزات جديدة 🚀

### ترقية كبيرة للويب هوكس

احصل على السيطرة الكاملة، والرؤية، والأمان مع نظام الويب هوكس المعاد تخيله بالكامل. استكشف القدرات الجديدة أدناه.

<Steps>
{/* LOCKED_PATTERN_c69db7a67c22374bc38018f45df7cd33 */}
    - إنشاء وتحرير وإيقاف مؤقت/استئناف وحذف نقاط نهاية webhook (URLs)
    - تحديد أنواع الأحداث لكل نقطة نهاية
    - تكوين رؤوس مخصصة لكل نقطة نهاية
    - تعيين حدود المعدل لكل نقطة نهاية
</Step>

{/* LOCKED_PATTERN_8cbecd2710208355be8248f15a8550e7 */}
    - عرض وتدوير أسرار التوقيع (مفاتيح HMAC) لكل نقطة نهاية
    - إعادة توليد الأسرار وعرض طوابع وقت آخر تدوير
    - نسخ مقتطفات الشفرة أو الوثائق للتحقق من التوقيع
</Step>

{/* LOCKED_PATTERN_dcecaa97684681ec209947fe21250620 */}
    - عرض سجلات الرسائل لكل نقطة نهاية (الحالة، والطوابع الزمنية)
    - فحص محاولات الرسائل الفردية مع أجسام الطلب/الاستجابة والرؤوس
    - إعادة إرسال الرسائل الفاشلة أو السابقة
    - تصفية السجلات حسب نوع الحدث، أو رمز الحالة، أو النطاق الزمني
</Step>

{/* LOCKED_PATTERN_f556e9022b1846586ac7518d86ce8f6f */}
    - إرسال أحداث اختبار من لوحة التحكم باستخدام حمولات زائفة
    - فحص مخططات الحمولات وعرض الحقول المتوقعة
</Step>

{/* LOCKED_PATTERN_b9949e0bf83fcf83823944e5a02934ca */}
    - تتبع إحصائيات التسليم: أعداد النجاح/الفشل، الكمون، والاتجاهات الأخيرة للأخطاء
    - مراقبة حالة صحة النقاط النهائية (مثل: فاشلة، سليمة)
</Step>
</Steps>

<Check>
يمكنك الآن إدارة وتصحيح وتأمين webhooks بمرونة أكبر — كل ذلك من خلال لوحة التحكم.
</Check>
