## نظرة عامة

نموذج جاهز للإنتاج للاشتراكات باستخدام Next.js 15، React 19، Supabase، Drizzle ORM، وDodo Payments. يأتي مع مصادقة Google OAuth، عملية الدفع للاشتراك، معالجة الويب هوك، مخطط قاعدة البيانات، ولوحة تحكم أساسية.

<Info>
إذا كنت تحتاج فقط معالجات المسارات لتطبيق قائم، راجع الموصلات المخصصة: <a href="/developer-resources/nextjs-adaptor">موصل Next.js</a> و<a href="/developer-resources/express-adaptor">موصل Express</a>.
</Info>

## المتطلبات الأساسية

- Node.js 18+ (أو Bun 1.0+)
- مشروع Supabase (URL، مفتاح Anon، مفتاح دور الخدمة، URL قاعدة البيانات)
- حساب Dodo Payments (مفتاح API، سر الويب هوك)
- عميل Google Cloud OAuth (معرف العميل والسر)

```env
DODO_PAYMENTS_API_KEY=your-dodo-api-key
DODO_WEBHOOK_SECRET=your-webhook-secret
DODO_PAYMENTS_ENVIRONMENT=test_mode
```

<Warning>
لا تدفع الأسرار إلى نظام التحكم بالإصدار. استخدم متغيرات البيئة في بيئات النشر.

## ما هو مدرج

- المصادقة عبر Supabase (تم تكوين Google OAuth)
- إنهاء الاشتراك عبر Dodo Payments
- دالة Supabase Edge لـ webhooks (`dodo-webhook`)
- مخطط Drizzle ORM وترحيلات
- لوحة تحكم بها الفواتير وحالة الاشتراك وميزات الخطة

<Tip>
احتفظ بـ `DODO_PAYMENTS_ENVIRONMENT` كـ `test_mode` حتى تنتهي من اختبارات شاملة للنظام.
</Tip>

## التحقق واستكشاف الأخطاء وإصلاحها

<AccordionGroup>
{/* LOCKED_PATTERN_6b286e9e2bbacd5c7684f8442203f10e */}
