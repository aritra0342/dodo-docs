</Tip>

## البداية السريعة

ابدأ باستخدام أي SDK في دقائق:

<Steps>
{/* LOCKED_PATTERN_776027320500bde6b99bac6bed1cc64d */}
  استخدم مدير الحزم الخاص باللغة التي تعمل بها لتثبيت مجموعة SDK
  
  <Tabs>
  <Tab title="TypeScript/Node.js">

  ```bash
  npm install dodopayments
  ```

  </Tab>
  
  <Tab title="Python">

  ```bash
  pip install dodopayments
  ```

  </Tab>
  
  <Tab title="PHP">

  ```bash
  composer require dodopayments/client
  ```

  </Tab>
  
  <Tab title="Go">

  ```bash
  go get github.com/dodopayments/dodopayments-go
  ```

  </Tab>
  </Tabs>
</Step>

  كوّن العميل باستخدام مفتاح واجهة برمجة التطبيقات الخاص بك
  
  <Tabs>
  <Tab title="TypeScript">

  ```typescript
  import DodoPayments from 'dodopayments';
  const client = new DodoPayments({ bearerToken: 'your_api_key' });
  ```

  </Tab>
  
  <Tab title="Python">

  ```python
  from dodopayments import DodoPayments
  client = DodoPayments(bearer_token="your_api_key")
  ```

  </Tab>
  
  <Tab title="PHP">

  ```php
  use Dodopayments\Client;
  $client = new Client(bearerToken: 'your_api_key');
  ```

  </Tab>
  
  <Tab title="Go">

  ```go
  import "github.com/dodopayments/dodopayments-go"
  client := dodopayments.NewClient(option.WithBearerToken("your_api_key"))
  ```

  </Tab>
  </Tabs>
  
  <Warning>
  قم دائمًا بتخزين مفاتيح API الخاصة بك بأمان باستخدام متغيرات البيئة. لا تقم أبدًا بإدراجها في نظام التحكم بالإصدارات.
  </Warning>
</Step>

  أنشئ جلسة دفع أو عملية دفع
  
  <Check>
  أنت الآن جاهز لمعالجة المدفوعات! قم بزيارة صفحات SDK الفردية للحصول على دلائل وأمثلة مفصلة.
  </Check>
</Step>
</Steps>

## الميزات الرئيسية

تشارك جميع SDKs هذه القدرات الأساسية:

- **أمان النوع**: نوع قوي لضمان الأمان في وقت الترجمة ودعم أفضل لبيئات التطوير المتكاملة
- **معالجة الأخطاء**: معالجة استثناءات شاملة مع رسائل خطأ مفصلة
- **المصادقة**: مصادقة بسيطة باستخدام مفتاح API مع دعم متغيرات البيئة
- **دعم غير متزامن**: أنماط حديثة غير متزامنة حيثما كان ذلك ممكنًا
- **التصفح التلقائي**: التصفح التلقائي للاستجابات القائمة
- **الفوترة بناءً على الاستخدام**: دعم مدمج لتتبع واستيعاب أحداث الاستخدام
- **الاختبار**: دعم كامل لبيئة sandbox للتطوير والاختبار

## React Native SDK

لتطبيقات الهواتف المحمولة، نقدم SDK مخصص لـ React Native:

<Card title="React Native SDK" icon="mobile" href="/developer-resources/react-native-integration">
  قم ببناء تجارب دفع آمنة لتطبيقات iOS وAndroid باستخدام مكونات واجهة مستخدم أصلية وتبسيط جمع بيانات الدفع
</Card>

**الميزات:**
- مكونات واجهة مستخدم أصلية لنظامي Android و iOS
- أمان مبسط لجمع بيانات الدفع الحساسة
- دعم لطرق دفع متعددة
- تكامل سلس مع تطبيقات React Native

<Info>
Apple Pay وGoogle Pay وCash App وUPI غير مدعومة حاليًا في SDK الخاص بـ React Native. من المخطط دعم هذه طرق الدفع في الإصدارات المستقبلية.
</Info>

## واجهة سطر الأوامر

لعمليات العمل المستندة إلى الطرفية والأتمتة:

<Card title="CLI" icon="terminal" href="/developer-resources/sdks/cli">
  واجهة سطر أوامر مولدة تلقائيًا مع دعم لجميع نقاط النهاية الـ 78 لواجهة برمجة التطبيقات

**الميزات:**
- هيكل أوامر قائم على الموارد للاستخدام البديهي
- تنسيقات إخراج متعددة (JSON و YAML و pretty و interactive)
- إكمال shell لـ bash و zsh و fish

```bash
# Quick example
dodopayments payments list --format json | jq '.data[] | {id, amount}'
```
