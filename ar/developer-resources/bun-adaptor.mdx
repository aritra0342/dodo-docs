## التثبيت

<Steps>
<Step title="Install the package">
قم بتشغيل الأمر التالي في جذر المشروع الخاص بك:

```bash
bun add @dodopayments/bun
```

</Step>
<Step title="Set up environment variables">
قم بإنشاء ملف <code>.env</code> في جذر المشروع الخاص بك:

```env expandable
DODO_PAYMENTS_API_KEY=your-api-key
DODO_PAYMENTS_WEBHOOK_KEY=your-webhook-secret
DODO_PAYMENTS_ENVIRONMENT="test_mode" or "live_mode"
DODO_PAYMENTS_RETURN_URL=your-return-url
```

<Warning>
لا تقم أبدًا بارتكاب ملف <code>.env</code> أو الأسرار في نظام التحكم بالإصدارات.
</Warning>
</Step>
</Steps>

<Tabs>
<Tab title="Checkout Handler">
  <Info>
  استخدم هذا المعالج لدمج صفحة الدفع من Dodo Payments في خادم Bun الخاص بك. يدعم تدفقات الدفع الثابتة (GET)، الديناميكية (POST)، وجلسات الدفع (POST).
  </Info>
  <CodeGroup>

    ```typescript Bun Server Handler expandable
    import { Checkout } from '@dodopayments/bun';

    const staticCheckoutHandler = Checkout({
        bearerToken: process.env.DODO_PAYMENTS_API_KEY,
        returnUrl: process.env.DODO_PAYMENTS_RETURN_URL,
        environment: process.env.DODO_PAYMENTS_ENVIRONMENT,
        type: "static"
    });

    const sessionCheckoutHandler = Checkout({
        bearerToken: process.env.DODO_PAYMENTS_API_KEY,
        returnUrl: process.env.DODO_PAYMENTS_RETURN_URL,
        environment: process.env.DODO_PAYMENTS_ENVIRONMENT,
        type: "session"
    });

    const dynamicCheckoutHandler = Checkout({
        bearerToken: process.env.DODO_PAYMENTS_API_KEY,
        returnUrl: process.env.DODO_PAYMENTS_RETURN_URL,
        environment: process.env.DODO_PAYMENTS_ENVIRONMENT,
        type: "dynamic"
    });

    Bun.serve({
      port: 3000,
      fetch(request) {
        const url = new URL(request.url);
        
        if (url.pathname === "/api/checkout") {
          if (request.method === "GET") {
            return staticCheckoutHandler(request);
          }
          if (request.method === "POST") {
            return sessionCheckoutHandler(request);
            // or return dynamicCheckoutHandler(request);
          }
        }
        
        return new Response("Not Found", { status: 404 });
      },
    });
    ```

  </CodeGroup>

## معالج مسار صفحة الدفع

<Info>
تدعم Dodo Payments ثلاثة أنواع من تدفقات الدفع لدمج المدفوعات في موقعك، ويدعم هذا المحول جميع أنواع تدفقات الدفع.
</Info>

- **روابط الدفع الثابتة:** عناوين URL قابلة للمشاركة فوريًا لجمع المدفوعات بسرعة ودون كود.
- **روابط الدفع الديناميكية:** إنشاء روابط دفع برمجيًا بتفاصيل مخصصة باستخدام واجهة برمجة التطبيقات أو SDKs.
- **جلسات الدفع:** إنشاء تجارب دفع آمنة وقابلة للتخصيص مع عربات منتجات وتفاصيل عملاء معدة مسبقًا.

### معلمات الاستعلام
<ParamField query="customer_id" type="string" required>
معرّف العميل لجلسة البوابة (مثًلا <code>?customer_id=cus_123</code>).
</ParamField>
<ParamField query="send_email" type="boolean">
إذا كان مُعيّنًا إلى <code>true</code>، يُرسل بريدًا إلكترونيًا للعميل يحتوي على رابط البوابة.
</ParamField>

<Warning>
يرجع 400 إذا كان <code>customer_id</code> مفقودًا.
</Warning>

## معالج مسار الويب هوك

- **الطريقة:** يتم دعم طلبات POST فقط. الطرق الأخرى ترجع 405.
- **التحقق من التوقيع:** يتحقق من توقيع الويب هوك باستخدام <code>webhookKey</code>. يرجع 401 إذا فشل التحقق.
- **التحقق من الحمولة:** يتم التحقق منها باستخدام Zod. يرجع 400 للحمولات غير الصالحة.
- **معالجة الأخطاء:**
  - 401: توقيع غير صالح
  - 400: حمولة غير صالحة
  - 500: خطأ داخلي أثناء التحقق
- **توجيه الأحداث:** يستدعي معالج الحدث المناسب بناءً على نوع الحمولة.

import { Checkout, CustomerPortal, Webhooks } from "@dodopayments/bun";

const staticCheckoutHandler = Checkout({
  bearerToken: process.env.DODO_PAYMENTS_API_KEY,
  returnUrl: process.env.DODO_PAYMENTS_RETURN_URL,
  environment: process.env.DODO_PAYMENTS_ENVIRONMENT,
  type: "static",
});

const sessionCheckoutHandler = Checkout({
  bearerToken: process.env.DODO_PAYMENTS_API_KEY,
  returnUrl: process.env.DODO_PAYMENTS_RETURN_URL,
  environment: process.env.DODO_PAYMENTS_ENVIRONMENT,
  type: "session",
});

const customerPortalHandler = CustomerPortal({
  bearerToken: process.env.DODO_PAYMENTS_API_KEY,
  environment: process.env.DODO_PAYMENTS_ENVIRONMENT,
});

const webhookHandler = Webhooks({
  webhookKey: process.env.DODO_PAYMENTS_WEBHOOK_KEY,
  onPaymentSucceeded: async (payload) => {
    console.log("Payment succeeded:", payload);
    // Your business logic here
  },
  onSubscriptionActive: async (payload) => {
    console.log("Subscription activated:", payload);
    // Your business logic here
  },
});

Bun.serve({
  port: 3000,
  fetch(request) {
    const url = new URL(request.url);
    
    // Checkout routes
    if (url.pathname === "/api/checkout") {
      if (request.method === "GET") {
        return staticCheckoutHandler(request);
      }
      if (request.method === "POST") {
        return sessionCheckoutHandler(request);
      }

    // Customer portal route
    if (url.pathname === "/api/customer-portal" && request.method === "GET") {
      return customerPortalHandler(request);
    }
    
    // Webhook route
    if (url.pathname === "/api/webhook/dodo-payments" && request.method === "POST") {
      return webhookHandler(request);
    }
    
    return new Response("Not Found", { status: 404 });
  },
});

console.log("Server running on http://localhost:3000");
