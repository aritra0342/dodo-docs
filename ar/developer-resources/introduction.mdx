## عناوين URL للبيئة

استخدم عنوان URL الأساسي المناسب لبيئتك:

- وضع الاختبار: [`https://test.dodopayments.com`](https://test.dodopayments.com)
- وضع الإنتاج: [`https://live.dodopayments.com`](https://live.dodopayments.com)

تعرف على المزيد حول [وضع الاختبار مقابل وضع الإنتاج](/miscellaneous/test-mode-vs-live-mode).

## خطوات الدمج

اتبع هذه الخطوات لدمج Dodo Payments:

<Steps>
  1. [قم بإعداد حسابك](/quickstart) واحصل على مفاتيح API الخاصة بك
  2. اختر طريقة الدمج الخاصة بك:
     - [اتبع البرنامج التعليمي الخاص بنا](/developer-resources/integration-tutorial)
     - [استخدم SDKs الخاصة بنا](/developer-resources/dodo-payments-sdks)
     - [دمج API المباشر](/developer-resources/integration-guide)
  3. [قم بإعداد webhooks](/developer-resources/webhooks/intents/webhook-events-guide) للحصول على تحديثات في الوقت الحقيقي
  4. [اختبر دمجك](/miscellaneous/testing-process)
  5. ابدأ العمل!
</Steps>
