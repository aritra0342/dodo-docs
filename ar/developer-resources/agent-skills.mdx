## نظرة عامة

مهارات الوكيل هي قدرات قابلة لإعادة الاستخدام تعزز ما يمكن لمساعدي الترميز بالذكاء الاصطناعي القيام به عند العمل مع Dodo Payments. إنها توفر معرفة إجرائية وأفضل الممارسات التي تساعد الوكلاء على إنجاز مهام محددة بكفاءة أكبر - من دمج تدفقات الدفع إلى إعداد الفوترة القائمة على الاستخدام.

<Info>
**اعتبر المهارات كملحقات لمساعد الذكاء الاصطناعي الخاص بك.** إنها تعلم وكيلك كيفية تنفيذ ميزات Dodo Payments بشكل صحيح، مع اتباع أنماطنا الموصى بها وأفضل الممارسات.
</Info>

تعمل المهارات مع أي وكيل ذكاء اصطناعي متوافق مع MCP، بما في ذلك:
- **Claude Code** - مساعد الترميز عبر سطر الأوامر من Anthropic
- **OpenCode** - وكيل الترميز مفتوح المصدر
- **Cursor** - محرر رموز موجه بالذكاء الاصطناعي
- **Other MCP clients** - أي وكيل يدعم بروتوكول المهارات

## التثبيت

اختر طريقة التثبيت المفضلة لديك بناءً على مساعد الترميز بالذكاء الاصطناعي.

<Tabs>
<Tab title="Skills CLI">
ثبّت المهارات باستخدام أداة CLI العالمية التي تعمل مع أي وكيل متوافق مع MCP:

```bash
# Install all Dodo Payments skills
npx skills add dodopayments/skills
```

أو ثبّت المهارات الفردية حسب الحاجة:

```bash
# Best practices guide
npx skills add dodopayments/skills/dodo-best-practices

# Webhook integration
npx skills add dodopayments/skills/webhook-integration

# Subscription billing
npx skills add dodopayments/skills/subscription-integration

# Checkout flows
npx skills add dodopayments/skills/checkout-integration

# Usage-based billing
npx skills add dodopayments/skills/usage-based-billing

# BillingSDK components
npx skills add dodopayments/skills/billing-sdk

# License key management
npx skills add dodopayments/skills/license-keys
```

<Tip>
ثبّت فقط المهارات التي تحتاجها للحفاظ على تركيز وكيلك وتقليل العبء السياقي.
</Tip>
</Tab>

<Tab title="Claude Code">
أضف سوق مهارات Dodo Payments إلى Claude Code:

```bash
/plugin marketplace add dodopayments/skills
```

ثم ثبّت الإضافات الفردية:

```bash
# Install best practices
/plugin install dodo-best-practices

# Install webhook integration
/plugin install webhook-integration

# Install subscription integration
/plugin install subscription-integration

# Install checkout integration
/plugin install checkout-integration

# Install usage-based billing
/plugin install usage-based-billing

# Install BillingSDK
/plugin install billing-sdk

# Install license keys
/plugin install license-keys
```

</Tab>

<Tab title="OpenCode">
تكون المهارات متاحة تلقائيًا عند تكوينها في إعدادات OpenCode الخاصة بك. أضف مستودع مهارات Dodo Payments إلى تكوينك:

```json
{
  "skills": {
    "repositories": [
      "dodopayments/skills"
    ]
  }
}
```

</Tab>
</Tabs>

## استخدام المهارات

بمجرد تثبيتها، يمكن لمساعدك بالذكاء الاصطناعي الاستفادة من هذه المهارات عندما تطلب منه تنفيذ ميزات Dodo Payments. فيما يلي بعض المطالبات النموذجية:

### Best Practices

```
Help me integrate Dodo Payments into my Next.js app following best practices
```

### Webhooks

```
Set up webhook handlers for payment and subscription events
```

### Subscriptions

```
Implement a subscription flow with free trial and multiple pricing tiers
```

### Checkout

```
Create a checkout session for my SaaS product
```

### Usage-Based Billing

```
Add usage-based billing to track API calls for my AI product
```

### BillingSDK

```
Add a pricing page using BillingSDK components
```

### License Keys

```
Implement license key validation for my desktop app
```

## كيف تعمل المهارات

توفر المهارات لمساعدك بالذكاء الاصطناعي:
1. **المعرفة الإجرائية** - إرشادات خطوة بخطوة لتنفيذ الميزات
2. **قوالب الكود** - أنماط مقتطفات جاهزة للإنتاج
3. **أفضل الممارسات** - الأساليب الموصى بها للأمان، ومعالجة الأخطاء، وتجربة المستخدم
4. **الوعي بالسياق** - فهم كيفية عمل ميزات Dodo Payments المختلفة معًا

1. **Procedural Knowledge** - Step-by-step instructions for implementing features
2. **Code Templates** - Production-ready code patterns and snippets
3. **Best Practices** - Recommended approaches for security, error handling, and UX
4. **Context Awareness** - Understanding of how different Dodo Payments features work together

عندما تطلب من مساعدك بالذكاء الاصطناعي تنفيذ ميزة، يستخدم المهارة ذات الصلة لـ:
- فهم نهج التنفيذ الصحيح
- توليد كود دقيق يعمل بشكل سليم
- اتباع اتفاقيات Dodo Payments
- التعامل مع الحالات الاستثنائية والأخطاء بشكل مناسب

<Note>
تكمل المهارات [MCP Server](/developer-resources/mcp-server) و[Sentra](/developer-resources/sentra). استخدم MCP لعمليات واجهة برمجة التطبيقات، وSentra للتكامل الموجه، والمهارات لتعزيز قدرات مساعد الذكاء الاصطناعي.
</Note>
