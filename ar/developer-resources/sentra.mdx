## نظرة عامة

تضيف سينترا الذكاء والتنفيذ على قمة مجموعات دودي بايمنتس الأساسية. إنها وكيل ذكاء اصطناعي يدمج ويحلل ويتصرف في الفوترة والمدفوعات. تربط سينترا SDKs وAPIs والمحولات عبر مجموعتك (المصادقة، قاعدة البيانات، إدارة علاقات العملاء، إلخ)، وتظهر رؤى الإيرادات مثل محركات النمو، وتنفذ إجراءات العملاء مثل الاستردادات والترقيات والائتمانات.


<Info>
Sentra لا يزال في مرحلة بيتا. قد تتغير الميزات وواجهات برمجة التطبيقات والتكاملات بسرعة. نرحب بملاحظاتكم وتقارير الأخطاء أثناء تطويرنا!
</Info>

## الإعداد

بمجرد التثبيت، تحتاج إلى تكوين سينترا باستخدام مفتاح API الخاص بدودي بايمنتس وإعدادات البيئة.


<Steps>
{/* LOCKED_PATTERN_414ce797076c31d13ae054f651b366fa */}
1. سجِّل الدخول إلى [لوحة تحكم Dodo Payments](https://dashboard.dodopayments.com).
2. انتقل إلى **الإعدادات** › **API Keys**.
3. انسخ مفتاح API الخاص بك.
</Step>


{/* LOCKED_PATTERN_2760e1f00720e8d27cb7b397504bdc4c */}
1. افتح بيئة التطوير المتكاملة الخاصة بك (VS Code أو Cursor أو Windsurf).
2. افتح لوحة الأوامر (`Cmd+Shift+P` على macOS / `Ctrl+Shift+P` على Windows/Linux).
3. اكتب "Sentra: Configure" واختره.
4. أدخل مفتاح API الخاص بـ Dodo Payments عندما يُطلب منك ذلك.

## استخدام سينترا

### الأوضاع

تقدم سينترا ثلاثة أوضاع تفاعل يمكن الوصول إليها من الواجهة:

- **اسأل**: اطرح أسئلة حول دودي بايمنتس، نماذج الفوترة، استخدام API، وأفضل الممارسات
- **ابنِ**: دمج الفوترة وواجهة المستخدم في تطبيقك بمساعدة موجهة
- **خطط**: تصميم هياكل فوترة قابلة للتوسع مع إرشادات معمارية

### مهام نموذجية

إليك بعض المهام الشائعة التي يمكنك إنجازها باستخدام سينترا:

- **دمج صفحة التسعير**: إعداد صفحة تسعير كاملة مع قوائم المنتجات وإجراءات الدفع
- **تنفيذ Webhook**: تكوين معالجات webhook لأحداث الدفع
- **إضافة تدفق الاشتراك**: إنشاء إدارة الاشتراك مع قدرات الترقية/الخفض
- **إعداد فوترة قائمة على الاستخدام**: تكوين عدادات واستيعاب الأحداث لتتبع الاستخدام
- **إنشاء بوابة العملاء**: بناء بوابة خدمة ذاتية للعملاء لإدارة الفوترة

### إعدادات الموافقة التلقائية

يمكنك تكوين إعدادات الموافقة التلقائية لتبسيط سير العمل الخاص بك:

- **لا شيء**: مراجعة والموافقة على جميع التغييرات يدويًا (موصى به للإنتاج)
- **مخاطر منخفضة**: الموافقة التلقائية على التغييرات البسيطة ومنخفضة المخاطر
- **الجميع**: الموافقة التلقائية على جميع التغييرات (استخدم بحذر)

## نماذج الفوترة المدعومة

تدعم سينترا جميع نماذج الفوترة لدودي بايمنتس:

- **المدفوعات لمرة واحدة**: معاملات الشراء الفردية
- **الاشتراكات**: فوترة متكررة مع فترات متعددة
- **فوترة قائمة على الاستخدام**: فوترة مقاسة بناءً على الاستهلاك
- **فوترة قائمة على النتائج**: نماذج تسعير الدفع مقابل النتيجة

## لمن هي

تعمل سينترا على ربط الهندسة والأعمال، بحيث يمكن لكل فريق شحن العائدات دون احتكاك:

- **المطورون**: شحن العائدات بشكل أسرع داخل بيئة التطوير المتكاملة الخاصة بك
- **المؤسسون ومديرو المنتجات**: نمذجة نماذج الفوترة دون الحاجة إلى العمل العميق في البنية التحتية
- **RevOps والمالية**: استخدام إجراءات محمية دون الحاجة إلى مهندسين
- **الدعم والنجاح**: حل حالات الفوترة بأمان، مع الموافقات حيثما دعت الحاجة
