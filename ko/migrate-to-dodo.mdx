**다른 결제 제공업체에서 Dodo Payments로 전환하시나요?** 우리의 마이그레이션 도구를 사용하면 **데이터 손실 없이 제품, 고객 및 할인 코드를 쉽게 이동할 수 있습니다.** 이 과정은 안전하고 안내가 있으며 몇 분 안에 완료할 수 있습니다.

현재 **Lemon Squeezy, Stripe, Polar.sh, Paddle**에서의 마이그레이션을 지원합니다.

## 시작하기

먼저, 마이그레이션 도구를 설치하세요:

```bash
npm install -g dodo-migrate
```

<Note>
컴퓨터에 Node.js ≥ 18이 설치되어 있어야 합니다.
</Note>

## 데이터 마이그레이션하기

### Lemon Squeezy에서 오는 경우:

```bash
dodo-migrate lemonsqueezy
```

### Stripe에서 오는 경우:

```bash
dodo-migrate stripe
```

### Polar.sh에서 오는 경우:

```bash
dodo-migrate polar
```

### Paddle에서 오는 경우:

```bash
dodo-migrate paddle
```

이 도구는 모든 단계를 안내합니다. 필요 사항은 다음과 같습니다:
- 제공업체의 API 키 (Lemon Squeezy, Stripe, Polar.sh, Paddle 등)
- Dodo Payments API 키  
- 먼저 테스트할지 (추천) 또는 라이브로 진행할지
- 마이그레이션할 Dodo Payments 브랜드

## 왜 우리의 마이그레이션 도구를 사용해야 할까요?

- **데이터 손실 없음** - 모든 것이 안전하게 전송됩니다.
- **먼저 테스트** - 라이브로 진행하기 전에 테스트 모드에서 시도해 보세요.
- **안내된 과정** - 각 단계를 안내합니다.
- **필요 시 재개** - 문제가 발생하면 중단한 지점에서 다시 시작할 수 있습니다.

## 고급 사용법

프롬프트 없이 마이그레이션을 실행하려면 모든 세부정보를 미리 제공할 수 있습니다:

### Lemon Squeezy 마이그레이션:

```bash
dodo-migrate lemonsqueezy \
  --provider-api-key=lsq_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX
```

### Stripe 마이그레이션:

```bash
dodo-migrate stripe \
  --provider-api-key=sk_test_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX \
  --migrate-types=products,coupons
```

### Polar.sh 마이그레이션:

```bash
dodo-migrate polar \
  --provider-api-key=polar_org_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX \
  --migrate-types=products,discounts,customers
```

### Paddle 마이그레이션:

```bash
dodo-migrate paddle \
  --provider-api-key=paddle_XXXXXXXXXXXXXXXX \
  --dodo-api-key=dp_XXXXXXXXXXXXXXXX \
  --mode=test_mode \
  --dodo-brand-id=brand_XXXXXX
```
