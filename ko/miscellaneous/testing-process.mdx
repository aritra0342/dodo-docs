## 테스트 카드 세부정보

Dodo Payments는 다양한 결제 시나리오를 시뮬레이션할 수 있는 테스트 카드 번호 세트를 제공합니다. 이 테스트 카드 번호를 사용하여 실제 결제나 요금을 처리하지 않고 테스트 모드에서 시뮬레이션된 결제를 생성할 수 있습니다.

테스트 카드 번호는 테스트 모드에서만 유효합니다. 실제 결제에 사용하지 마세요.

## 테스트 Afterpay 세부정보

체크아웃 통합을 테스트할 때 Afterpay를 결제 방법으로 선택하고 결제 버튼을 클릭하세요.

### 성공적인 결제
- 성공적인 결제를 위해 유효한 이메일 주소와 배송 주소를 사용하세요.

### 인증 실패
- 사용자가 인증에 실패하는 경우를 테스트하려면 테스트 API 키를 사용하고 리디렉션 페이지를 확인하세요. 리디렉션 페이지에서 Afterpay 모달 창을 닫고 결제가 실패했는지 확인합니다. PaymentIntent는 `requires_action`에서 `requires_payment_method`로 전환됩니다.

**참고**: Afterpay 거래에 필요한 최소 금액은 $50입니다.

테스트 모드에서 수동으로 PaymentIntents를 캡처할 경우, 캡처되지 않은 PaymentIntent는 성공적인 승인이 이루어진 후 10분 후에 자동으로 만료됩니다.

## 애플 페이 테스트

Dodo Payments 테스트 계정을 사용하여 애플 페이를 테스트할 수 있습니다.

<Info>
애플 페이를 테스트하려면 iPad, iPhone 또는 Mac이 필요합니다. Mac에서는 Safari를 사용하세요.
</Info>

<Steps>
<Step title="Dodo Payments 계정에서 애플 페이 활성화">
Dodo Payments 계정 설정에서 애플 페이를 활성화하세요.
</Step>

<Step title="애플 ID 등록">
아직 애플 ID가 없다면, [애플 ID 등록](https://support.apple.com/apple-id)하세요.
</Step>

<Step title="애플 페이에 카드 최소 1장 추가">
[애플 페이에 카드 최소 1장 추가](https://support.apple.com/HT204506)하세요. 애플 페이를 테스트할 때는 실제 카드 정보를 사용하지만 카드가 청구되지는 않습니다.
</Step>

<Step title="체크아웃 열고 애플 페이 선택">
테스트 모드에서 체크아웃을 열고 결제 방법으로 애플 페이를 선택하세요.
</Step>

<Step title="카드를 선택하고 테스트 구매 완료">
애플 페이 지갑에서 카드를 선택하고 테스트 구매를 완료하세요.
</Step>
</Steps>

<Warning>
Apple Pay가 나타나려면 체크아웃이 `https`를 통해 제공되어야 합니다.
</Warning>

## 구글 페이 테스트

Dodo Payments 테스트 계정을 사용하여 구글 페이를 테스트할 수 있습니다.

<Info>
구글 페이를 테스트하려면 구글 크롬 또는 안드로이드 기기를 사용해야 합니다.
</Info>

<Steps>
<Step title="Dodo Payments 계정에서 구글 페이 활성화">
Dodo Payments 계정 설정에서 구글 페이를 활성화하세요.
</Step>

<Step title="구글 계정 등록">
아직 구글 계정이 없다면, [구글 계정 등록](https://accounts.google.com/signup)하세요.
</Step>

<Step title="구글 페이에 카드 최소 1장 추가">
[구글 페이에 카드 최소 1장 추가](https://payments.google.com/)하세요. 구글 페이 작업 시 테스트 카드를 사용할 수 있지만, 지갑에 유효한 카드가 최소 1장은 있어야 합니다.
</Step>

<Step title="테스트 카드용 구글 그룹 가입">
[테스트 카드용 구글 그룹 가입](https://groups.google.com/g/googlepay-test-mode-stub-data)하세요. 이는 구글 페이가 Dodo Payments 테스트 모드와 같은 테스트 환경을 감지할 때 자동으로 테스트 카드를 지갑에 추가합니다.
</Step>

<Step title="체크아웃 열고 구글 페이 선택">
테스트 모드에서 체크아웃을 열고 결제 방법으로 구글 페이를 선택하세요.
</Step>

<Step title="구글 페이 테스트 카드 중 하나 선택하고 구매 완료">
지갑에서 구글 페이 테스트 카드 중 하나를 선택하고 테스트 구매를 완료하세요.
</Step>
</Steps>

<Info>
데스크탑에서 구글 크롬을 사용하는 경우, [크롬 설정](chrome://settings/payments)에서 **사이트가 결제 방법을 확인할 수 있도록 허용** 옵션이 켜져 있는지 확인하세요. 기본적으로 켜져 있습니다.
</Info>

## 기타 결제 방법

아마존 페이 및 캐시 앱과 같은 기타 결제 방법을 테스트하려면 일반적인 테스트 절차를 따르세요.
