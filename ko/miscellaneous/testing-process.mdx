## 테스트 카드 세부정보

Dodo Payments는 다양한 결제 시나리오를 시뮬레이션할 수 있는 테스트 카드 번호 세트를 제공합니다. 이 테스트 카드 번호를 사용하여 실제 결제나 요금을 처리하지 않고 테스트 모드에서 시뮬레이션된 결제를 생성할 수 있습니다.

테스트 거래는 실제 결제를 처리하거나 실제 계정에 영향을 주지 않습니다.
</Card>

{/* LOCKED_PATTERN_5ba03ec58a56547ac9750ef18da96a7f */}
모든 API 엔드포인트는 테스트 모드와 라이브 모드에서 동일하게 작동합니다.

{/* LOCKED_PATTERN_1f50b75a756a498817926e42de6e14a9 */}

<Warning>
테스트 자격 증명은 테스트 모드에서만 작동합니다. 라이브 모드에서 사용하려 하면 결제 실패가 발생합니다.
</Warning>

## 테스트 Afterpay 세부정보

체크아웃 통합을 테스트할 때 Afterpay를 결제 방법으로 선택하고 결제 버튼을 클릭하세요.

### 성공적인 결제
- 성공적인 결제를 위해 유효한 이메일 주소와 배송 주소를 사용하세요.

| 상태 | UPI ID |
| :----- | :----- |
| Success | `success@upi` |
| Failure | `failure@upi` |

**참고**: Afterpay 거래에 필요한 최소 금액은 $50입니다.

- 청구 국가를 `IN`로 설정해야 합니다
- 통화를 `INR`으로 설정해야 합니다
- 비인도 상인의 경우: Adaptive Currency를 활성화해야 합니다

## 애플 페이 테스트

Dodo Payments 테스트 계정을 사용하여 애플 페이를 테스트할 수 있습니다.

<Info>
애플 페이를 테스트하려면 iPad, iPhone 또는 Mac이 필요합니다. Mac에서는 Safari를 사용하세요.
</Info>

<Steps>
<Step title="Dodo Payments 계정에서 애플 페이 활성화">
Dodo Payments 계정 설정에서 애플 페이를 활성화하세요.
</Step>

<Step title="애플 ID 등록">
아직 애플 ID가 없다면, [애플 ID 등록](https://support.apple.com/apple-id)하세요.
</Step>

<Step title="애플 페이에 카드 최소 1장 추가">
[애플 페이에 카드 최소 1장 추가](https://support.apple.com/HT204506)하세요. 애플 페이를 테스트할 때는 실제 카드 정보를 사용하지만 카드가 청구되지는 않습니다.
</Step>

{/* LOCKED_PATTERN_49aff3a3aee1d0bab41e7eade47939d0 */}

<Step title="카드를 선택하고 테스트 구매 완료">

</Steps>

<Steps>

</Warning>

## 유럽 방법 테스트

유럽 결제 방법(iDEAL, Bancontact, EPS, Multibanco)은 샌드박스 모드에서 테스트할 수 있습니다.

<Steps>
{/* LOCKED_PATTERN_540056f13df545529727751bb5b93f77 */}
Dodo Payments 테스트 API 키를 사용하세요.
</Step>

{/* LOCKED_PATTERN_73a737ada9108be9fcbfe7701d196c25 */}
결제 수단에 맞게 청구 주소 국가를 설정하세요:
- `NL`은 iDEAL용
- `BE`은 Bancontact용
- `AT`은 EPS용
- `PT`은 Multibanco용
</Step>

{/* LOCKED_PATTERN_111b236b56d1ef796758b64d1fe1fcd1 */}
유럽 결제 수단은 EUR 통화를 요구합니다.
</Step>

{/* LOCKED_PATTERN_3efbb33511174ca4755c102c30869543 */}
테스트 환경에서 시뮬레이션된 은행 인증 흐름을 따르세요.
</Step>
</Steps>

전체 유럽 결제 방법 문서는 [유럽](/features/payment-methods/europe) 페이지를 참조하세요.

- `dodo wh listen`는 라이브 테스트 웹훅을 로컬 서버로 전달합니다
- `dodo wh trigger`는 22개 이벤트 유형 모두에 대한 모의 페이로드를 보냅니다
</Accordion>
