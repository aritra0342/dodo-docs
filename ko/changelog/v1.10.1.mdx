## 새로운 기능 🚀

- **체크아웃 2.0**: 다음을 포함한 완전히 재설계된 체크아웃 경험을 소개합니다:
    - 향상된 사용자 경험을 위한 간소화되고 현대적인 인터페이스
    - 세션 저장소에 고유 ID로 매개변수를 저장하는 새로운 쿼리 수집기 시스템(예: `sess_1a2b3c4d`)
    - 전체 체크아웃 흐름에서 지속적으로 데이터를 사용할 수 있음
    - 향후 릴리즈에 더 많은 개선 사항 예정!
  
    <Frame><img src="/images/changelog/checkout_2.png" alt="Checkout 2.0"/></Frame>

- **구성 가능한 결제 수단**: 새로운 `allowed_payment_methods` 구성으로 판매자가 체크아웃 시 사용할 수 있는 결제 옵션을 맞춤 설정할 수 있습니다. 구현 세부 정보는 [Allowed Payment Methods](/features/payment-methods#configuring-payment-methods) 문서를 참조하세요.

- **동적 세금 계산**: 고객 청구 주소에 따라 세금률을 실시간으로 계산하여 정적 결제 링크의 가격을 보다 정확하게 제공합니다.

- **주문형 구독**: 유연한 구독 청구를 위한 새로운 API 엔드포인트 (선택된 상인에게 제공됨). [주문형 구독](/features/subscription#enabling-on-demand-subscriptions) 가이드에서 자세히 알아보세요.

- **추가 업데이트**:
    - 성능 최적화
    - 버그 수정
    - 일반 시스템 개선
