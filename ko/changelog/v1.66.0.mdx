## 새로운 기능 🚀

1. **`subscription.updated` Webhook**  
   구독 객체 업데이트마다 발생하는 새로운 웹후크 이벤트로, 구독의 변경 사항에 대한 실시간 알림을 제공합니다.
   
   이 웹후크를 사용하여:
   - 구독 세부정보 변경 시 즉시 알림을 받습니다
   - 애플리케이션 상태를 Dodo Payments와 동기화 상태로 유지합니다
   - API 폴링 없이 구독 업데이트에 대응합니다
   
   <Tip>
   구독을 위해 `subscription.updated`을(를) 구독하여 폴링 필요를 없애고 실시간 구독 변경 사항과 시스템을 동기화 상태로 유지하세요.
   </Tip>
   
   자세히 알아보기: [Subscription Webhooks](/developer-resources/webhooks/intents/subscription)

3. **읽기 전용 API 키**  
   보다 안전한 통합을 위해 읽기 전용 권한으로 API 키를 생성하세요. 새 API 키를 만들 때 **쓰기 액세스 활성화** 선택을 해제하여 읽기 전용 키를 생성합니다.
   
   읽기 전용 키는 다음을 수행할 수 있습니다:
   - 결제, 구독 및 고객 데이터를 가져옵니다
   - 제품 목록을 확인하고 세부 정보를 검색합니다
   - 보고서 및 분석에 접근합니다
   
   읽기 전용 키는 **다음은 할 수 없습니다**:
   - 결제를 생성하거나 수정합니다
   - 구독을 업데이트합니다
   - 제품 구성을 변경합니다
   
   <Tip>
   대시보드 통합, 분석 도구 및 변경 없이 데이터만 볼 필요가 있는 모든 시스템에 대해 "쓰기 액세스 활성화" 선택을 해제하세요.
   </Tip>

4. **UPI AutoPay 및 RuPay 카드 구독 - 전체 출시**  
   UPI AutoPay 및 RuPay 카드 구독이 이제 모든 비즈니스에서 이용 가능합니다. 이를 통해 인도 고객을 위한 반복 결제가 가능합니다:
   
   - **UPI AutoPay**: UPI 약정을 통한 자동 반복 청구
   - **RuPay 카드**: 인도 국내 카드 네트워크용 구독 지원
   
   <Check>
   결과: RBI 준수 약정 처리를 통해 인도 시장에서 확장된 구독 결제 옵션을 제공합니다.
   </Check>
   
   자세히 알아보기: [Subscriptions with RBI-Compliant Mandates](/features/subscription#subscriptions-with-rbi-compliant-mandates)

5. **플랜 변경 미리보기 API**  
   구독 플랜 변경을 커밋하기 전에 미리 볼 수 있는 새로운 API 엔드포인트입니다. 실제 수정 없이 변경 후 구독이 어떻게 보일지와 정확한 청구 금액을 확인하세요.
   

   ```javascript
   const preview = await client.subscriptions.previewChangePlan('sub_123', {
     product_id: 'prod_pro_plan',
     proration_billing_mode: 'prorated_immediately',
     quantity: 1
   });
   
   console.log('Charge amount:', preview.immediate_charge.summary);
   console.log('New plan details:', preview.new_plan);
   ```

   
   이를 사용하여:
   - 고객에게 업그레이드/다운그레이드를 확인하기 전에 정확히 얼마를 지불할지 보여줍니다.
   - 가격 계산기 및 플랜 비교 도구를 만듭니다.
   - 가격 투명성을 개선하여 지원 티켓을 줄입니다.
   
   자세히 알아보기: [플랜 변경 미리보기 API](/api-reference/subscriptions/preview-change-plan)
