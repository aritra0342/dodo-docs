## 새로운 기능 🚀

- **전용 구독 페이지**: 새로운 전용 페이지로 고객 구독을 쉽게 관리하세요. 대시보드에서 `Sales > Subscriptions`로 이동하세요.
  
    <Frame><img src="/images/changelog/subscriptions.png" alt="구독 페이지"/></Frame>

- **향상된 고객 세부정보**: 각 고객의 구독, 결제 및 환불을 한 곳에서 확인하세요. 대시보드에서 `Sales > Customers > arrow icon`로 이동하세요.

    <Frame><img src="/images/changelog/customer_info.png" alt="고객 세부정보"/></Frame>

- **상세한 지급 정보**: 지급 금액, 주기 및 날짜에 대한 더 많은 통찰력을 얻으세요. 대시보드에서 `Business > Payouts`로 방문하세요.

    <Frame><img src="/images/changelog/payouts_update.png" alt="지급 정보"/></Frame>

- **공식 Ruby SDK**: Ruby 애플리케이션에 Dodo Payments를 손쉽게 통합하세요. 자세한 내용은 [Ruby SDK](https://github.com/dodopayments/dodopayments-ruby)를 확인하세요.

- **향상된 API 필터**: 관련 목록 API에 `customer_id`, `status` 등과 같은 필터가 추가되었습니다. 더 많은 정보는 API 문서를 참조하세요.

- **소규모 개선 및 버그 수정**: 다양한 소규모 개선 및 버그 수정이 구현되었습니다.
