## 새로운 기능 🚀

- **새로운 유럽 결제 수단**: 네 가지 새로운 유럽 결제 수단으로 결제 옵션을 확장했습니다:
  - **iDEAL** - 네덜란드
  - **Bancontact** - 벨기에
  - **Przelewy24 (P24)** - 폴란드
  - **EPS** - 오스트리아

  <Frame>
    <img alt="지원되는 유럽 결제 수단" src="/images/transactions/payments/european.png" />
  </Frame>

  <Note>
    이러한 결제 수단은 현재 다음으로 제한됩니다:
      * 유로 (EUR) 통화만
      * 일회성 결제만 (구독에는 사용 불가)

    이러한 결제 수단에 대한 자세한 정보는 [결제 수단](/features/payment-methods) 문서를 방문하세요.
  </Note>

- **구독 청구지 주소 업데이트**: 다음을 통해 구독의 청구지 주소를 업데이트할 수 있는 기능이 추가되었습니다:
  * 고객 포털 인터페이스
  * 패치 구독 API 엔드포인트
