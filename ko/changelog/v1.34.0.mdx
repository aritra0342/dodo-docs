## 새로운 기능 🚀

- **디지털 제품 배송**: 자동화된 이행을 통해 고객이 성공적인 결제 후 즉시 디지털 상품을 받을 수 있도록 하는 간소화된 디지털 제품 배송 시스템. [디지털 제품 배송](/features/digital-product-delivery) 문서에서 자세히 알아보세요.

- **React Native SDK (베타)**: 네이티브 Android 및 iOS 앱에서 사용자 정의 UI 구성 요소 및 화면을 사용하여 안전한 결제 처리를 가능하게 하는 React Native SDK를 베타로 출시했습니다. [React Native SDK](/developer-resources/react-native-integration) 문서에서 자세히 알아보세요.

- **새로운 비례 요금 모드**: 구독 변경 시 즉각적인 전체 요금을 허용하는 `full_immediately` 비례 요금 모드를 도입하여 비즈니스 요구에 맞는 더 유연한 청구 옵션을 제공합니다.

- **구독을 위한 적응형 통화**: 고객의 위치와 현지 통화 선호도에 따라 가격을 자동으로 조정하는 적응형 통화 지원으로 구독 관리 기능을 향상시켰습니다.

- **즉각적인 주문형 처리**: 주문형 요금에 대한 처리 속도를 개선하여 거의 즉각적으로 만들어 거래 시간을 단축하고 더 빠른 결제 확인을 제공합니다.

- **인증 양식 관리**: Dodo Payments 대시보드 인증 섹션에서 제출된 모든 인증 문서를 중앙 집중식 위치에서 검토하고 관리할 수 있는 포괄적인 인증 양식 데이터 보기 기능을 추가했습니다.

- **구독 취소 기능 향상**: 대시보드와 고객 포털 모두에서 사용할 수 있는 청구 기간 종료 시 구독 취소 기능을 구현하여 사용자 경험과 청구 투명성을 개선했습니다.

- **일반 개선 사항**: 전체 플랫폼 안정성과 사용자 경험을 향상시키기 위한 다양한 소규모 수정 및 성능 개선.
