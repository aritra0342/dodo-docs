## 새로운 기능
- **오픈 소스 Atlas**: Dodo Payments 통합을 보여주는 Next.js로 작성된 데모 애플리케이션입니다. [GitHub 리포지토리](https://github.com/dodopayments/dodo-checkout-demo)와 문서는 [여기](/developer-resources/integration-guide)에서 확인하세요.
- **기능 요청 섹션**: 사용자 피드백을 기반으로 제품 로드맵의 우선순위를 정하기 위해 Dodo Payments 대시보드에 기능 요청 섹션을 도입했습니다.
- **웹훅 요청 로그 API**: 웹훅 요청 로그를 나열하고 세부 정보를 제공하는 API를 추가했습니다.
- **정적 결제 링크**: 결제 과정에서 더 많은 유연성을 제공하기 위해 정적 결제 링크에 더 많은 쿼리 매개변수를 추가했습니다. 자세한 내용은 [여기](/developer-resources/integration-guide#1-static-payment-links)를 클릭하세요.
- **구독 목록 API**: 모든 구독을 가져오는 API를 추가했습니다. 자세한 내용은 [API 참조](/api-reference/subscriptions/get-subscriptions-1)를 참조하세요.

## 사소한 개선 사항 및 버그 수정
- 추가 매개변수로 제품 및 구독 API 응답을 향상시켰습니다.
- API 요청에 대한 속도 제한을 구현했으며, 현재 초당 20 요청으로 설정되어 있습니다.
- 문서를 보다 사용자 친화적이고 탐색하기 쉽게 개선했습니다.
- 다양한 사소한 버그 수정 및 개선 사항이 포함되어 있습니다.
