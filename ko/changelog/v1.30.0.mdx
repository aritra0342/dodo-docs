## 새로운 기능 🚀

- **제로 달러 결제**: $0 결제 및 일회성 제품에 대한 100% 할인 코드 지원 추가, 무료 제품 제공 및 프로모션 캠페인 가능.

- **Custom Payout Threshold**: USD 지갑에 대한 최소 지급액을 설정할 수 있는 기능을 추가하여 상인들이 지급 처리 최적화 및 소액 거래 수수료를 줄일 수 있도록 지원합니다. 이 기능은 `Payouts ▸ USD Wallet ▸ Edit Payout Threshold`에서 이용하세요.
  
  <Frame>
    <img src="/images/changelog/payout-threshold.png" alt="Payout Threshold Cover Image"/>
  </Frame>

- **API 표준화**: 지속적인 API 개선 이니셔티브의 일환으로 표준화된 API 오류 코드 및 메시지 구현. 이 개선은 API를 더 일관되게 만들고 개발자 친화적으로 만듭니다. 자세한 정보는 [오류 코드 문서](/api-reference/error-codes)를 방문하세요.

- **거래 거부 표준화**: 통합된 거래 거부 코드 및 메시지 도입, 실패 원인에 대한 명확한 통찰 제공 및 사용자 오류와 시스템 문제 구분. 자세한 내용은 [거래 실패 문서](/api-reference/transaction-failures)를 확인하세요.
