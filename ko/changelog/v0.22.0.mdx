## 새로운 기능 🚀

- **구독의 첫 결제**: 가장 요청이 많았던 기능이 여기 있습니다! 이제 첫 결제는 위임과 실제 구독 금액을 결합한 단일 거래로 진행됩니다.

- **UPI QR 결제 방법**: 인도 시장을 위한 결제 방법으로 UPI QR을 추가하여 고객이 UPI QR 코드를 사용하여 결제할 수 있도록 했습니다.
    
    <img src="/images/changelog/upi-qr.png" alt="UPI QR 기능"/>

- **체크아웃 시 세금 ID**: 고객은 이제 인보이스 및 세금 목적을 위해 체크아웃 페이지에서 세금 ID를 추가할 수 있습니다.

- **대시보드 재구성**: Dodo Payments 대시보드 전체가 더 직관적이고 사용자 친화적으로 재구성되었습니다.

- **셀프 서비스 온보딩 가이드**: 새로운 사용자는 이제 셀프 서비스 온보딩 가이드를 사용하여 Dodo Payments를 시작할 수 있습니다.

- **다국어 지원**: 체크아웃 페이지에서 14개 이상의 언어를 지원하여 고객이 선호하는 언어로 결제할 수 있도록 했습니다. 지원되는 언어는 다음과 같습니다:
    - 영어 (en)
    - العربية (ar) // 아랍어
    - Català (ca) // 카탈로니아어
    - 中文 (cn) // 중국어
    - Deutsch (de) // 독일어
    - Español (es) // 스페인어
    - Français (fr) // 프랑스어
    - עברית (he) // 히브리어
    - Italiano (it) // 이탈리아어
    - 日本語 (ja) // 일본어
    - Nederlands (nl) // 네덜란드어
    - Polski (pl) // 폴란드어
    - Português (pt) // 포르투갈어
    - Svenska (sv) // 스웨덴어

- **인보이스 PDF 개편**: 인보이스 PDF가 새롭고 더 명확하며 효과적인 디자인으로 변경되었습니다.

- **중앙 집중식 도움말 위젯**: 지원을 간소화하고 개선하기 위해 중앙 집중식 도움말 위젯이 추가되었습니다.
    
    <img src="/images/changelog/help-widget.png" alt="도움말 위젯"/>

## 개선 사항 및 버그 수정 🔧

- **요금 제한**: 인보이스 API에 요금 제한을 추가하여 남용을 방지하고 시스템 안정성을 보장합니다.

- **정적 결제 링크의 메타데이터**: 정적 결제 링크에 대한 메타데이터 지원이 추가되었습니다. 자세한 내용은 [문서](/developer-resources/integration-guide#1-static-payment-links)를 참조하세요.

- **소규모 개선 및 수정**: 전반적인 성능 및 사용자 경험을 개선하기 위한 다양한 소규모 개선 및 수정이 이루어졌습니다.
