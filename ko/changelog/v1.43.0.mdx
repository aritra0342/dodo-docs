## 새로운 기능 🚀

### 주요 웹훅 업그레이드

완전히 재구성된 웹훅 시스템으로 완벽한 제어, 가시성 및 보안을 확보하세요. 아래의 새로운 기능을 탐색해 보세요.

<Steps>
{/* LOCKED_PATTERN_c69db7a67c22374bc38018f45df7cd33 */}
    - 웹훅 엔드포인트(URL)를 생성, 편집, 일시중지/다시 시작, 삭제
    - 각 엔드포인트에 대한 이벤트 유형 선택
    - 엔드포인트별 사용자 정의 헤더 구성
    - 각 엔드포인트별 속도 제한 설정
</Step>

{/* LOCKED_PATTERN_8cbecd2710208355be8248f15a8550e7 */}
    - 엔드포인트별 서명 비밀(HMAC 키) 보기 및 회전
    - 비밀 다시 생성 및 마지막 회전 타임스탬프 보기
    - 서명 검증을 위한 코드 스니펫 또는 문서 복사
</Step>

{/* LOCKED_PATTERN_dcecaa97684681ec209947fe21250620 */}
    - 엔드포인트별 메시지 로그(상태, 타임스탬프) 보기
    - 요청/응답 본문과 헤더를 포함한 개별 메시지 시도 검사
    - 실패하거나 이전 메시지 다시 재전송
    - 이벤트 유형, 상태 코드 또는 시간 범위로 로그 필터링
</Step>

{/* LOCKED_PATTERN_f556e9022b1846586ac7518d86ce8f6f */}
    - 포털에서 모의 페이로드로 테스트 이벤트 전송
    - 페이로드 스키마를 검사하고 예상 필드 보기
</Step>

{/* LOCKED_PATTERN_b9949e0bf83fcf83823944e5a02934ca */}
    - 전달 통계 추적: 성공/실패 수, 지연 시간, 최근 오류 추세
    - 엔드포인트 상태 모니터링(예: 실패 중, 정상)
</Step>
</Steps>
