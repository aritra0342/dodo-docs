## 새로운 기능 🚀

### 체크아웃 세션 API
새로운 **체크아웃 세션 API**를 소개합니다 - 일회성 구매와 구독 모두를 위한 완전한 결제 흐름을 제공하는 강력한 호스팅 체크아웃 솔루션입니다. 이 새로운 API는 이전의 동적 링크 접근 방식을 대체하며, 유연성과 사용자 경험을 향상시킵니다.

**주요 이점:**
- **간소화된 통합**: 전체 청구 주소를 미리 제공할 필요가 없습니다 - 누락된 세부정보는 자동으로 처리합니다
- **향상된 사용자 정의**: 결제 방법, 통화, 테마 및 기능 플래그에 대한 완전한 제어
- **향상된 UX**: 체크아웃 마찰을 줄이고 전환율을 개선하기 위해 고객 정보를 미리 채웁니다
- **유연한 구성**: 다중 제품 장바구니, 구독 체험, 할인 코드 등을 지원합니다

**변경 사항:**
- **이전**: 결제 링크를 생성할 때 전체 청구 주소 정보가 필요했습니다
- **현재**: 사용 가능한 고객 정보를 전달하세요 - 체크아웃 중 나머지를 수집합니다
- **마이그레이션**: 추가 처리 없이 간단한 API 엔드포인트 변경

- 결제 내보내기 보고서에 데이터 분석을 위한 메타데이터가 포함되었습니다.
- 이제 API를 통해 구독의 다음 청구일을 직접 업데이트할 수 있습니다.
