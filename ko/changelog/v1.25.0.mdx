## 새로운 기능 🚀

- **Overlay Checkout Beta**: 새로운 오버레이 체크아웃 경험의 베타 출시를 발표하게 되어 기쁩니다. 이 현대적인 TypeScript 라이브러리는 실시간 이벤트 처리 및 사용자 정의 가능한 테마를 통해 웹 애플리케이션에 결제 오버레이를 통합하는 원활한 방법을 제공합니다. 주요 기능은 다음과 같습니다:
  - **실시간 이벤트 처리**: 결제 생성, 성공 및 실패와 같은 체크아웃 이벤트를 수신
  - **TypeScript 지원**: 더 나은 개발자 경험을 위한 완전한 타입 API
  - **사용자 정의 가능한 테마**: 밝은 테마와 어두운 테마 지원
  - **브라우저 호환성**: Chrome, Firefox, Safari, Edge 및 IE11+를 포함한 모든 최신 브라우저에서 작동
  - **PCI 준수**: 내장 경험을 제공하면서 보안 표준 유지

- **새로운 구독 이벤트**: 구독이 업그레이드, 다운그레이드 또는 수정될 때마다 발생하는 `subscription.plan_changed` 웹후크 이벤트에 대한 지원이 추가되었습니다. 이 이벤트는 다음 정보를 제공합니다:
  - 변경 후 완전한 구독 세부 정보
  - 이전 및 새 요금제 정보
  - 비례 청구 조정
  - 업데이트된 다음 청구 날짜

  이 이벤트는 구독 수명 주기에 대한 포괄적인 가시성을 제공하기 위해 기존 구독 이벤트(`subscription.active`, `subscription.on_hold` 등)를 보완합니다.

- **플랫폼 개선**: 플랫폼 전반에 걸쳐 다양한 성능 최적화, 버그 수정 및 소규모 개선.
