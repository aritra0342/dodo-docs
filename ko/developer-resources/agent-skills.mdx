## 개요

Agent Skills는 Dodo Payments와 함께 작업할 때 AI 코딩 어시스턴트의 기능을 향상시키는 재사용 가능한 기능입니다. 체크아웃 흐름 통합부터 사용량 기반 청구 설정까지 특정 작업을 보다 효과적으로 수행하도록 돕는 절차 지식과 모범 사례를 제공합니다.

<Info>
**스킬은 AI 어시스턴트를 위한 플러그인으로 생각하세요.** 이러한 스킬은 권장 패턴과 모범 사례를 따라 Dodo Payments 기능을 정확히 구현하도록 에이전트를 교육합니다.
</Info>

스킬은 다음을 포함한 모든 MCP 호환 AI 에이전트와 함께 작동합니다:
- **Claude Code** - Anthropic의 CLI 코딩 어시스턴트
- **OpenCode** - 오픈 소스 AI 코딩 에이전트
- **Cursor** - AI 중심 코드 에디터
- **기타 MCP 클라이언트** - 스킬 프로토콜을 지원하는 모든 에이전트

## 설치

선호하는 AI 코딩 어시스턴트에 따라 설치 방법을 선택하세요.

<Tabs>
<Tab title="Skills CLI">
모든 MCP 호환 에이전트와 함께 작동하는 범용 스킬 CLI를 사용하여 스킬을 설치하세요:

```bash
# Install all Dodo Payments skills
npx skills add dodopayments/skills
```

또는 필요한 개별 스킬만 설치하세요:

```bash
# Best practices guide
npx skills add dodopayments/skills/dodo-best-practices

# Webhook integration
npx skills add dodopayments/skills/webhook-integration

# Subscription billing
npx skills add dodopayments/skills/subscription-integration

# Checkout flows
npx skills add dodopayments/skills/checkout-integration

# Usage-based billing
npx skills add dodopayments/skills/usage-based-billing

# BillingSDK components
npx skills add dodopayments/skills/billing-sdk

# License key management
npx skills add dodopayments/skills/license-keys
```

<Tip>
필요한 스킬만 설치해 에이전트의 집중도를 유지하고 컨텍스트 부담을 줄이세요.
</Tip>
</Tab>

<Tab title="Claude Code">
Claude Code에 Dodo Payments 스킬 마켓플레이스를 추가하세요:

```bash
/plugin marketplace add dodopayments/skills
```

그런 다음 개별 플러그인을 설치하세요:

```bash
# Install best practices
/plugin install dodo-best-practices

# Install webhook integration
/plugin install webhook-integration

# Install subscription integration
/plugin install subscription-integration

# Install checkout integration
/plugin install checkout-integration

# Install usage-based billing
/plugin install usage-based-billing

# Install BillingSDK
/plugin install billing-sdk

# Install license keys
/plugin install license-keys
```

</Tab>

<Tab title="OpenCode">
OpenCode 설정에서 구성하면 스킬이 자동으로 사용 가능해집니다. Dodo Payments 스킬 저장소를 구성에 추가하세요:

```json
{
  "skills": {
    "repositories": [
      "dodopayments/skills"
    ]
  }
}
```

</Tab>
</Tabs>

## 스킬 사용 방법

설치 후, AI 어시스턴트는 Dodo Payments 기능 구현을 요청할 때 이러한 스킬을 활용할 수 있습니다. 다음은 예시 프롬프트입니다:

### 모범 사례

```
Help me integrate Dodo Payments into my Next.js app following best practices
```

### 웹훅

```
Set up webhook handlers for payment and subscription events
```

### 구독

```
Implement a subscription flow with free trial and multiple pricing tiers
```

### 체크아웃

```
Create a checkout session for my SaaS product
```

### 사용량 기반 청구

```
Add usage-based billing to track API calls for my AI product
```

### BillingSDK

```
Add a pricing page using BillingSDK components
```

### 라이선스 키

```
Implement license key validation for my desktop app
```

## 스킬 작동 방식

스킬은 AI 어시스턴트에 다음을 제공합니다:

1. **절차 지식** - 기능 구현을 위한 단계별 지침
2. **코드 템플릿** - 프로덕션 준비된 코드 패턴 및 스니펫
3. **모범 사례** - 보안, 오류 처리 및 UX에 권장되는 접근 방식
4. **맥락 인식** - 서로 다른 Dodo Payments 기능이 함께 작동하는 방식에 대한 이해

AI 어시스턴트에게 기능 구현을 요청하면 관련 스킬을 사용하여:
- 올바른 구현 방식을 이해하고
- 정확하고 작동하는 코드를 생성하며
- Dodo Payments 컨벤션을 따르고
- 엣지 케이스와 오류를 적절히 처리합니다

<Note>
스킬은 [MCP Server](/developer-resources/mcp-server) 및 [Sentra](/developer-resources/sentra)를 보완합니다. API 작업에는 MCP를, 안내형 통합에는 Sentra를, 향상된 AI 어시스턴트 기능에는 Skills를 사용하세요.
</Note>
