</Note>

## API 엔드포인트

| 환경 | 기본 URL | 결제 확인 엔드포인트 |
| --- | --- | --- |
| **테스트 모드** | `https://test.dodopayments.com` | `https://api.custom.hs.dodopayments.com/payments/<payment_id>/confirm` |
| **실운영 모드** | `https://live.dodopayments.com` | `https://api.hs.dodopayments.com/payments/<payment_id>/confirm` |

## 단계별 카드 결제 흐름

### 1단계: 비밀 키 확보

다음 API를 사용하여 결제를 생성하여 시작하세요:

- **Payments**: [Create Payment API](/api-reference/payments/post-payments)

이 API는 이후 단계에 필요한 **`secret_key`**를 반환합니다.

### 2단계: 결제 확인

`payments confirm` 엔드포인트를 호출하여 결제를 확인하세요:

- **테스트 모드**: `POST https://api.custom.hs.dodopayments.com/payments/<payment_id>/confirm`
- **실운영 모드**: `POST https://api.hs.dodopayments.com/payments/<payment_id>/confirm`

요청 본문에 다음 세부 정보를 포함하세요:

- `client_secret`: 1단계에서 획득한 비밀 키
- `payment_method`: `card`로 설정
- `payment_method_data`: 카드 및 청구 정보
- `browser_info`: 브라우저 관련 데이터(사용자 에이전트, 화면 정보 등)
- `payment_type`: 일회성 결제의 경우 `normal`로 명시적으로 설정

필요한 헤더:

- DodoPayments 지원팀에서 공개 키를 받으세요
- `api-key`: 공개 키 (Dodo Payments 팀에서 받으세요)
- `content-type`: `application/json`

**예시 요청 본문:**

```json
{
  "client_secret": "<secret_key>",
  "payment_method": "card",
  "payment_method_data": {
    "card": {
      "card_number": "<card_number>",

      "card_exp_year": "<exp_year>",
      "card_holder_name": "<cardholder_name>",
      "card_cvc": "<cvc>",
      "card_network": "<network>"
    },
    "billing": {
      "address": {
        "first_name": "<first_name>",
        "last_name": "<last_name>"
      }
    }
  },
  "browser_info": {
    "user_agent": "<user_agent>",
    "accept_header": "<accept_header>",
    "language": "<language>",
    "color_depth": "<color_depth>",
    "screen_height": "<screen_height>",
    "screen_width": "<screen_width>",
    "time_zone": "<time_zone>",

}
```

브라우저 정보는 일반적으로 다음과 같으며(실제 값은 다를 수 있음):

  "language": "en-GB",
  "color_depth": 24,

상인들은 체크아웃 중 고객의 브라우저에서 이러한 세부 정보를 직접 수집해야 합니다.

### 응답 처리

- 응답의 **`status`**가 **`requires_customer_action`**인 경우 추가 인증을 위해 **`next_action`** 필드에 제공된 URL로 고객을 리디렉션하세요.

<Warning>
원시 카드 데이터를 처리할 때 PCI 준수 가이드라인을 반드시 준수하세요.
</Warning>

## Apple Pay 통합

Apple Pay를 사용하려면 `payment_method`을 `wallet`으로 설정하고 Apple Pay 데이터를 `payment_method_data`에 포함하세요.

### 요청 본문 구조

```json
{
  "client_secret": "<secret_key>",
  "payment_method": "wallet",
  "payment_method_data": {
    "wallet": {
      "apple_pay": {
        "payment_data": "<decrypted_payment_data>",
        "payment_method": {
          "display_name": "Visa 1234",
          "network": "Visa",
          "type": "debit"
        },
        "transaction_identifier": "UNIQUE_TRANSACTION_ID_FROM_APPLE"
      }
    }
  },
  "browser_info": {
    "user_agent": "<user_agent>",
    "accept_header": "<accept_header>",
    "language": "<language>",
    "color_depth": "<color_depth>",
    "screen_height": "<screen_height>",
    "screen_width": "<screen_width>",
    "time_zone": "<time_zone>",
    "java_enabled": true,
    "java_script_enabled": true,
    "ip_address": "<ip_address_of_client>",
    "device_model": "<device_model>",
    "os_type": "<os_type>",
    "os_version": "<version>"
  },
  "payment_type": "normal"
}
```

### Apple Pay 필드 참조

| 필드 | 유형 | 필수 | 설명 |
| --- | --- | --- | --- |
| `payment_data` | string | 예 | 이 열거형은 암호화되었거나 복호화된 Apple Pay 결제 데이터를 나타내는 데 사용됩니다. |
| `payment_method.display_name` | string | 예 | Apple Pay 버튼에 표시되는 이름(예: "Visa 1234") |
| `payment_method.network` | string | 예 | 카드 네트워크(예: "Visa", "Mastercard", "Amex") |
| `payment_method.type` | string | 예 | 결제 방법 유형(예: "debit", "credit") |
| `payment_method.card_exp_month` | string | 아니요 | 카드 만료 월(예: "12") |
| `payment_method.card_exp_year` | string | 아니요 | 카드 만료 연도(예: "25") |
| `transaction_identifier` | string | 예 | Apple Pay에서 제공하는 고유 거래 식별자 |

## Google Pay 통합

Google Pay를 사용하려면 `payment_method`을 `wallet`으로 설정하고 Google Pay 데이터를 `payment_method_data`에 포함하세요.

### 요청 본문 구조

```json
{
  "client_secret": "<secret_key>",
  "payment_method": "wallet",
  "payment_method_data": {
    "wallet": {
      "google_pay": {
        "type": "CARD",
        "description": "Visa ending in 1234",
        "info": {
          "card_network": "VISA",
          "card_details": "1234"
        },
        "tokenization_data": {
          "type": "PAYMENT_GATEWAY",
          "token": "<encrypted_payment_token>"
        }
      }
    }
  },
  "browser_info": {
    "user_agent": "<user_agent>",
    "accept_header": "<accept_header>",
    "language": "<language>",
    "color_depth": "<color_depth>",
    "screen_height": "<screen_height>",
    "screen_width": "<screen_width>",
    "time_zone": "<time_zone>",
    "java_enabled": true,
    "java_script_enabled": true,
    "ip_address": "<ip_address_of_client>",
    "device_model": "<device_model>",
    "os_type": "<os_type>",
    "os_version": "<version>"
  },
  "payment_type": "normal"
}
```

### Google Pay 필드 참조

| 필드 | 유형 | 필수 | 설명 |
| --- | --- | --- | --- |
| `type` | string | 예 | 결제 방법 유형(일반적으로 "CARD") |
| `description` | string | 예 | 사용자에게 표시되는 결제 방법 설명 |
| `info.card_network` | string | 예 | 카드 네트워크(예: "VISA", "MASTERCARD", "AMEX") |
| `info.card_details` | string | 예 | 카드의 마지막 네 자리 |
| `info.assurance_details` | object | 아니요 | 인증 보증 관련 세부 정보 |
| `tokenization_data.type` | string | 예 | 토큰화 유형(예: "PAYMENT_GATEWAY") |
| `tokenization_data.token` | string | 예 | Google Pay에서 제공된 암호화된 결제 토큰 |
