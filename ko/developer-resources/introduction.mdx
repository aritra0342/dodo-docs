## 환경 URL

귀하의 환경에 적합한 기본 URL을 사용하십시오:

- 테스트 모드: [`https://test.dodopayments.com`](https://test.dodopayments.com)
- 라이브 모드: [`https://live.dodopayments.com`](https://live.dodopayments.com)

[테스트 모드와 라이브 모드](/miscellaneous/test-mode-vs-live-mode)에 대해 자세히 알아보세요.

## 통합 단계

Dodo Payments를 통합하려면 다음 단계를 따르십시오:

<Steps>
  1. [계정을 설정](/quickstart)하고 API 키를 받으세요.
  2. 통합 방법을 선택하세요:
     - [우리의 튜토리얼을 따르세요](/developer-resources/integration-tutorial)
     - [우리의 SDK를 사용하세요](/developer-resources/dodo-payments-sdks)
     - [직접 API 통합](/developer-resources/integration-guide)
  3. [웹훅을 설정](/developer-resources/webhooks/intents/webhook-events-guide)하여 실시간 업데이트를 받으세요.
  4. [통합을 테스트](/miscellaneous/testing-process)하세요.
  5. 라이브로 전환하세요!
</Steps>
