## Dodo Payments란 무엇인가요?

Dodo Payments는 **올인원 엔진**으로, 전 세계에서 출시, 확장 및 수익화할 수 있도록 설계되었습니다. 새로운 세대의 디지털 SaaS 및 AI 제품을 위해 설계되었으며, **결제에서 청구, 배포까지** 모든 것을 처리합니다 - 추가 엔지니어링 없이도 가능합니다.

<Info>
**SaaS 및 AI 제품을 위해 제작됨**: 최적화된 청구 인프라로 일회성 결제, 구독, 크레딧, 좌석, 토큰 또는 어떤 사용량 지표든 처리하세요.
</Info>

<Tip>
  <strong>Sentra를 만나보세요: AI 기반 청구 도우미</strong><br/>
  Sentra는 IDE 내에서 청구 및 결제를 통합, 분석, 실행하는 AI 에이전트입니다. 원하는 내용을 설명하면 질문하고, 통합을 구성하거나, 청구 아키텍처를 계획할 수 있습니다.

    Sentra로 시작하기 &rarr;
  </a>
</Tip>

### AI 어시스턴트를 위한 MCP 서버

AI 코딩 어시스턴트(Cursor, Claude Desktop, Windsurf, VS Code)를 Dodo Payments에 직접 연결할 수 있는 MCP 서버입니다.

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  Dodo Payments 문서를 통한 의미 기반 검색—기능, API, 모범 사례에 대한 즉시 답변 제공
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  API 작업을 직접 실행—결제 생성, 구독 관리, 환불 처리 등 AI 비서를 통해 수행
</Card>
</CardGroup>

{/* LOCKED_PATTERN_8ee343b87e057252cc4a8a376b190970 */}
두 서버를 MCP 구성(`~/.cursor/mcp.json` 또는 Claude Desktop 구성)에 추가하세요:

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
Node.js 18+ 필요. 모든 지원 클라이언트에 대한 자세한 설정 지침은 [MCP Server 문서](/developer-resources/mcp-server)를 참조하세요.
</Info>
</Accordion>

### 프레임워크 어댑터

{/* LOCKED_PATTERN_01f10564742337453a746085912d93d1 */}

</Card>

```bash
# Install all Dodo Payments skills
npx skills add dodopayments/skills

# Or install individual skills
npx skills add dodopayments/skills/dodo-best-practices
npx skills add dodopayments/skills/webhook-integration
npx skills add dodopayments/skills/subscription-integration
```

<Info>
Claude Code, OpenCode 및 기타 클라이언트 설정 지침은 [Agent Skills 문서](/developer-resources/agent-skills)를 참조하세요.
