## Dodo Payments란 무엇인가요?

Dodo Payments는 **올인원 엔진**으로, 전 세계에서 출시, 확장 및 수익화할 수 있도록 설계되었습니다. 새로운 세대의 디지털 SaaS 및 AI 제품을 위해 설계되었으며, **결제에서 청구, 배포까지** 모든 것을 처리합니다 - 추가 엔지니어링 없이도 가능합니다.

<Info>
**SaaS 및 AI 제품을 위해 구축됨**: 최적화된 청구 인프라를 통해 일회성 결제, 구독, 크레딧, 좌석, 토큰 또는 모든 사용 메트릭을 처리합니다.
</Info>

<Tip>
  <strong>Sentra를 만나보세요: AI 기반 청구 어시스턴트</strong><br/>
  Sentra는 IDE에서 직접 청구 및 결제를 통합, 분석 및 조치를 취하는 AI 에이전트입니다. 질문을 하거나, 통합을 구축하거나, 청구 아키텍처를 계획하세요 - 원하는 것을 설명하기만 하면 됩니다.

  </a>
</Tip>

AI 코딩 어시스턴트(Cursor, Claude Desktop, Windsurf, VS Code)를 Dodo Payments에 직접 연결할 수 있는 MCP 서버입니다.

<CardGroup cols={2}>
<Card title="Dodo Knowledge MCP" icon="book-open" href="/developer-resources/mcp-server#dodo-knowledge-mcp">
  Dodo Payments 문서에 대한 의미론적 검색 - 기능, API 및 모범 사례에 대한 즉각적인 답변을 얻으세요.
</Card>

<Card title="Dodo Payments MCP" icon="credit-card" href="/developer-resources/mcp-server">
  API 작업을 직접 실행 - AI 어시스턴트를 통해 결제 생성, 구독 관리 및 환불 처리
</Card>
</CardGroup>

<Accordion title="Cursor/Claude Desktop을 위한 빠른 MCP 설정">
두 서버를 MCP 구성에 추가하세요 (`~/.cursor/mcp.json` 또는 Claude Desktop 구성):

```json
{
  "mcpServers": {
    "dodo-knowledge": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://knowledge.dodopayments.com/mcp"]
    },
    "dodopayments": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.dodopayments.com/sse"]
    }
  }
}
```

<Info>
Node.js 18+가 필요합니다. 모든 지원 클라이언트에 대한 자세한 설정 지침은 [MCP 서버 문서](/developer-resources/mcp-server)를 참조하세요.
</Info>
</Accordion>
