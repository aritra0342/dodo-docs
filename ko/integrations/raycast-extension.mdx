Dodo Payments 대시보드에 몇 번의 키 입력으로 접근하세요. Raycast를 사용하여 대시보드를 열지 않고도 결제, 구독, 고객, 제품, 할인 코드, 라이센스 키 및 환불을 빠르게 확인할 수 있습니다.

## 설치

<Card title="Raycast Store에서 설치하기 →" icon="puzzle-piece" href="https://www.raycast.com/dodopayments/dodo-payments">
  Raycast Store에서 Dodo Payments 확장 페이지를 엽니다.
</Card>

<Info>
macOS에서 Raycast와 Dodo Payments API 키가 필요합니다.
</Info>

## 설정

<Steps>
<Step title="API 키 가져오기">
1. Dodo Payments 대시보드에 로그인합니다.
2. 설정 › API 키로 이동합니다.
3. API 키를 복사합니다.
</Step>

<Step title="확장 구성하기">
1. Raycast를 엽니다.
2. "Dodo Payments"를 검색하고 원하는 명령을 선택합니다.
3. 요청 시, API 키 필드에 API 키를 입력합니다.
4. API 모드를 선택합니다:
   - 테스트: 개발 및 테스트에 사용합니다.
   - 라이브: 프로덕션 데이터에 사용합니다.
5. 설정을 저장합니다.
</Step>
</Steps>

<Tip>
설정을 검증하기 위해 테스트 모드에서 시작한 후, 라이브로 전환하세요.
</Tip>

<Warning>
API 키를 비밀번호처럼 취급하세요. 공유하거나 평문으로 저장하지 마세요.
</Warning>

## 확장 사용하기

- **검색**: Raycast를 열고 "Dodo Payments"를 입력한 후 명령을 선택합니다.
- **필터**: ID, 이메일, 이름 또는 상태의 일부를 입력하여 결과를 좁힙니다.
- **상세 정보 열기**: 목록 항목에서 Enter를 눌러 더 많은 정보를 확인합니다.

## 문제 해결

<AccordionGroup>
<Accordion title="인증 실패 또는 401/403 오류">
API 키와 모드(테스트 vs 라이브)가 대시보드 데이터와 일치하는지 확인하세요. 필요 시 명령을 다시 열고 키를 다시 입력하세요.
</Accordion>
<Accordion title="결과가 나타나지 않음">
선택한 모드에 데이터가 있는지 확인하세요. 테스트와 라이브 간에 전환해 보세요.
</Accordion>
<Accordion title="요금 제한 또는 시간 초과">
잠시 기다렸다가 다시 시도하세요. 문제가 지속되면 네트워크 연결을 확인하세요.
</Accordion>
</AccordionGroup>
