Dodo Payments 대시보드에 몇 번의 키 입력으로 접근하세요. Raycast를 사용하여 대시보드를 열지 않고도 결제, 구독, 고객, 제품, 할인 코드, 라이센스 키 및 환불을 빠르게 확인할 수 있습니다.

## 설치

{/* LOCKED_PATTERN_689eae9422ae1bb6dca6aba8d59d9dd9 */}
  Raycast 스토어에서 Dodo Payments 확장 페이지를 엽니다.
</Card>

<Info>
  macOS용 Raycast와 Dodo Payments API 키가 필요합니다.
</Info>

## 설정

<Steps>
{/* LOCKED_PATTERN_92afe5a5f9799e8778da662cdecfb3a0 */}
1. Dodo Payments 대시보드에 로그인합니다.
2. 설정 › API 키로 이동합니다.
3. API 키를 복사합니다.
</Step>

{/* LOCKED_PATTERN_62acba3447424cd114a190a47207d390 */}
1. Raycast를 엽니다.
2. "Dodo Payments"를 검색하고 명령을 선택합니다.
3. 프롬프트가 표시되면 API 키 필드에 API 키를 입력합니다.
4. API 모드를 선택합니다:
   - 테스트: 개발 및 테스트용으로 사용합니다.
   - 라이브: 운영 데이터용으로 사용합니다.
5. 설정을 저장합니다.
</Step>
</Steps>

<Tip>
  먼저 테스트 모드에서 설정을 검증한 후 라이브로 전환합니다.
</Tip>

<Warning>
  API 키를 비밀번호처럼 취급하세요. 공유하거나 평문으로 저장하지 마세요.
</Warning>

## 확장 사용하기

- **검색**: Raycast를 열고 "Dodo Payments"를 입력한 후 명령을 선택합니다.
- **필터**: ID, 이메일, 이름 또는 상태의 일부를 입력하여 결과를 좁힙니다.
- **상세 정보 열기**: 목록 항목에서 Enter를 눌러 더 많은 정보를 확인합니다.
