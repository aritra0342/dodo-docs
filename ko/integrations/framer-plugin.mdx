## 개요

Dodo Payments Framer 플러그인은 Framer 웹사이트에 결제 처리를 원활하게 통합합니다. 테스트 및 프로덕션 환경을 지원하는 사용자 정의 가능한 체크아웃 경험을 만드세요.

<Card title="공식 플러그인" icon="link" href="https://www.framer.com/marketplace/plugins/dodo-payments">
Framer 마켓플레이스에서 공식 Dodo Payments 플러그인에 액세스하세요.
</Card>

## 설정 가이드

<Steps>
<Step title="API 키 생성">
[Dodo Payments 대시보드](https://app.dodopayments.com/developer/api-keys)를 방문하여 새 API 키를 생성하세요.

<Tabs>

<Tab title="실제 모드">
실제 결제 처리를 위한 프로덕션 사용.
</Tab>

<Tab title="테스트 모드">
실제 요금이 없는 개발 및 테스트에 적합합니다.
</Tab>

</Tabs>

<Check>
API 키를 안전하게 복사하세요 - 다음 단계에서 필요합니다.
</Check>
</Step>

<Step title="플러그인 인증">
1. Framer에서 Dodo Payments 플러그인을 엽니다.
2. 인증 페이지로 이동합니다.
3. API 키를 붙여넣습니다.
4. 적절한 모드(테스트 또는 실제)를 선택합니다.
5. 인증을 위해 **계속**을 클릭합니다.

<Warning>
API 키를 안전하게 보관하고 공개적으로 공유하지 마세요.
</Warning>
</Step>

<Step title="제품 관리">
인증 후, 제품 목록이 표시됩니다:

- **기존 제품**: 자동으로 표시됩니다.
- **제품 없음**: '제품' 옆의 **플러스 (+)** 버튼을 클릭하세요.

<Info>
가격, 설명 및 설정으로 제품을 생성하기 위해 Dodo Payments 대시보드로 리디렉션됩니다.
</Info>
</Step>

<Step title="구매 버튼 추가하기">
1. 사용 가능한 목록에서 제품을 선택합니다.
2. `Add Buy Button` 버튼을 클릭합니다.
3. 'Buy \{product_name\}'라는 텍스트가 있는 구매 버튼이 캔버스에 추가됩니다.

<Warning>
구매 버튼 추가는 올바른 프로젝트 권한이 없으면 비활성화됩니다.
</Warning>
</Step>
</Steps>

## 사용자 정의

<Steps>
<Step title="체크아웃 경험 선택">
    선호하는 체크아웃 경험을 선택하세요:
    - **오버레이(임베디드)**: 체크아웃이 Framer 프로젝트 내에서 오버레이로 나타나 사용자가 사이트에 머무르게 합니다.
    - **호스팅**: 사용자가 Dodo Payments 호스팅 체크아웃 페이지로 리디렉션되어 간소화된 경험을 제공합니다.
</Step>

<Step title="모양 옵션 사용자 정의">
    다음 모양 설정으로 체크아웃 경험을 사용자 정의하세요:
    - **체크아웃 테마**: 브랜드 미학에 맞게 밝은 테마 또는 어두운 테마 중에서 선택하세요.
    - **버튼 레이블**: 텍스트를 사용자 정의하세요(예: '지금 구매', '구매', '시작하기').
    - **버튼 스타일**: 색상, 글꼴, 크기 및 기타 스타일 속성을 조정하세요.
    - **반환 URL**: 체크아웃 후 사용자가 돌아갈 Framer 페이지를 설정하세요.
</Step>
</Steps>

## 문제 해결

<AccordionGroup>
<Accordion title="인증 문제">
**문제**: 플러그인이 인증되지 않음

**해결책**:
- API 키가 올바른지 확인하세요.
- 올바른 모드(테스트/실제)를 사용하고 있는지 확인하세요.
- 인터넷 연결을 확인하세요.
</Accordion>

<Accordion title="버튼 생성 문제">
**문제**: 구매 버튼 추가가 비활성화됨

**해결책**:
- Framer 프로젝트에서 올바른 권한이 있는지 확인하세요.
- 제대로 인증되었는지 확인하세요.
- 플러그인을 새로 고쳐보세요.
</Accordion>

<Accordion title="제품 표시 문제">
**문제**: 제품이 표시되지 않음

**해결책**:
- Dodo Payments 대시보드에서 제품이 생성되었는지 확인하세요.
- API 키에 올바른 권한이 있는지 확인하세요.
- 재인증을 시도하세요.
</Accordion>
</AccordionGroup>
