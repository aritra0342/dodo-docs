## 소개

이 가이드는 제휴 추천을 추적하고, 커미션 이벤트를 처리하며, Dodo Payments를 통해 신뢰할 수 있는 제휴 파트너십으로 수익을 성장시키는 방법을 안내합니다. 우리는 **Affonso** 및 **Dub Partners**와 같은 주요 제휴 플랫폼과의 통합을 제공하여 제휴 프로그램을 원활하게 관리하고 추적할 수 있도록 돕습니다.

## 통합 옵션

Dodo Payments는 두 개의 주요 제휴 관리 플랫폼과 통합됩니다:

- **[Affonso](https://affonso.io)**: 자동화된 커미션 추적, 실시간 보고 및 전용 제휴 대시보드를 갖춘 종합 제휴 관리 플랫폼입니다.
- **[Dub Partners](https://dub.co)**: Dub의 링크 관리 및 전환 추적을 사용하여 자세한 귀속 분석을 통해 제휴 프로그램을 강화하세요.

귀하의 필요에 가장 적합한 플랫폼을 선택하거나 서로 다른 사용 사례에 대해 두 가지 모두 사용할 수 있습니다.

## Affonso 통합

우리는 귀하의 판매를 지원하기 위해 선도적인 제휴 관리 플랫폼인 Affonso와 파트너십을 맺었습니다. 아래 단계를 따라 Dodo Payments 계정을 Affonso와 연결하고 제휴 기반 판매를 추적하기 시작하세요.

<Steps>
  <Step title="Dodo Payments와 연결하기">
    시작하려면 API 키를 추가하여 Dodo Payments 계정을 연결하세요.

    ### Dodo Payments API 키 가져오는 방법

    - **Dodo Payments 대시보드**에 로그인하세요.
    - 왼쪽 메뉴에서 **개발자** 섹션으로 이동하세요.
    - **API 키**를 클릭하세요.
    - **API 키 추가**를 선택하세요.
    - 이름을 지정하세요 (예: "Affonso 통합")
    - **생성**을 클릭하고 API 키를 **복사**하세요.

    ### Affonso에 API 키 추가하기

    - **[affonso.io](https://affonso.io/)**를 통해 **Affonso 대시보드**를 엽니다.
    - **프로그램 세부정보 > 연결**로 이동하세요.
    - **Dodo Payments 통합** 카드 클릭
    - Dodo Payments API 키를 붙여넣으세요.
    - **연결**을 클릭하세요.

    <Callout type="info">
      귀하의 API 키는 Affonso에 의해 안전하게 암호화됩니다.
    </Callout>
  </Step>

  <Step title="Dodo Payments에 웹훅 추가하기">
    연결되면 Affonso에서 고유한 웹훅 URL을 받게 됩니다. 이는 결제 추적 및 제휴 커미션 트리거를 가능하게 합니다.

    ### Dodo Payments에 웹훅 URL 추가하기

    - Dodo Payments 대시보드에서 **개발자 → 웹훅**으로 이동하세요.
    - **웹훅 추가**를 클릭하세요.
    - **Affonso 웹훅 URL**(프로그램 설정에서)을 붙여넣으세요.
    - 웹훅을 저장하세요.

    ### Affonso에 웹훅 비밀 추가하기

    - Dodo Payments에서 웹훅을 저장한 후, 제공된 **서명 키**를 **복사**하세요.
    - Affonso의 **Dodo Payments 웹훅 설정**에 다시 붙여넣으세요.

    <Callout type="info">
      이는 Affonso가 Dodo Payments에서 오는 이벤트를 안전하게 검증할 수 있도록 합니다.
    </Callout>
  </Step>

  <Step title="추적 스크립트 추가하기">
    제휴 방문 및 전환을 추적하기 위해 Affonso 픽셀을 사이트에 배치하세요.

    ```html
    <!-- Place in <head> tag -->
    <script async defer
    src="https://affonso.io/js/pixel.min.js"
    data-affonso="YOUR_PUBLIC_PROGRAM_ID"
    data-cookie_duration="YOUR_COOKIE_DURATION">
    </script>
    ```

    이 스크립트는:
    - 제휴 방문을 추적합니다.
    - 추천 ID로 쿠키를 설정합니다 (`affonso_referral`).
    - 가입 및 결제 추적을 가능하게 합니다.

    <Callout type="info">
      사용자가 방문하거나 전환하는 모든 도메인에 배치하세요 (예: 마케팅 사이트, 앱).
    </Callout>
  </Step>

  <Step title="사용자 가입 추적하기">
    퍼널 가시성을 확보하고 제휴 성과를 최적화하기 위해 사용자 가입을 리드로 추적하세요.

    ```js
    // After successful registration
    window.Affonso.signup(userEmail);
    ```

    이는:
    - 사용자가 제휴 링크를 통해 왔을 경우 새로운 추천을 **리드**로 기록합니다.
    - 대시보드와 제휴 통계에 표시됩니다.

    <Callout type="info">
      모범 사례: 성공적인 등록 또는 이중 옵트인(DOI) 후에 이 작업을 트리거하세요.
    </Callout>
  </Step>

  <Step title="추천 데이터를 Dodo Payments에 전달하기">
    커미션을 올바르게 귀속시키기 위해 체크아웃 중에 Affonso에서 Dodo Payments로 추천 ID를 전달하세요.

    ### 결제 통합

    ```js
    import DodoPayments from 'dodopayments';

    const referralId = window.affonso_referral;

    const client = new DodoPayments({
      bearerToken: process.env['DODO_PAYMENTS_API_KEY'],
    });

    async function main() {
      const payment = await client.payments.create({
        payment_link: true,
        billing: { city: 'city', country: 'AF', state: 'state', street: 'street', zipcode: 0 },
        customer: { email: 'email@email.com', name: 'name' },
        product_cart: [{ product_id: 'product_id', quantity: 0 }],
        metadata: {
          affonso_referral: referralId,
        },
      });

      console.log(payment.payment_id);
    }

    main();
    ```

    ### 구독 통합

    ```js
    import DodoPayments from 'dodopayments';

    const referralId = window.affonso_referral;

    const client = new DodoPayments({
      bearerToken: process.env['DODO_PAYMENTS_API_KEY'],
    });

    async function main() {
      const subscription = await client.subscriptions.create({
        billing: { city: 'city', country: 'IN', state: 'state', street: 'street', zipcode: 89789 },
        customer: { customer_id: 'customer_id' },
        product_id: 'product_id',
        payment_link: true,
        return_url: 'https://example.com/success',
        quantity: 1,
        metadata: {
          affonso_referral: referralId,
        },
      });

      console.log(subscription.subscription_id);
    }

    main();
    ```

    ### 결제 링크

    ```js
    // Original Dodo Payments Link
    const originalLink = 'https://checkout.dodopayments.com/buy/<YOUR_PRODUCT_ID>';

    // Append referral ID as reference_id
    const linkWithReferral = `${originalLink}?metadata_affonso_referral=${window.affonso_referral}`;
    ```

    <Callout type="info">
      자세한 내용은 [Dodo Payments API 문서](https://docs.dodopayments.com/api-reference/introduction)를 참조하세요.
    </Callout>
  </Step>

  <Step title="통합 테스트하기">
    공개적으로 출시하기 전에 라이브 제휴 링크 및 테스트 거래로 설정을 테스트하세요.

    **테스트 체크리스트:**
    - ✅ 제휴를 생성하고 추적 링크를 생성하세요.
    - ✅ 제휴 링크를 통해 웹사이트를 방문하세요.
    - ✅ Affonso 스크립트가 로드되고 `affonso_referral` 쿠키가 설정되었는지 확인하세요.
    - ✅ 가입을 완료하세요 (선택 사항)
    - ✅ Dodo Payments를 통해 결제를 완료하세요.
    - ✅ Affonso 대시보드에서 추적 및 귀속을 확인하세요.
  </Step>
</Steps>

## Dub Partners 통합

또한 [Dub Partners](https://dub.co)를 사용하여 제휴 추천 및 전환을 추적할 수 있습니다. Dub의 전환 추적 기능을 통해 판매를 제휴 링크에 귀속시킬 수 있으며, 어떤 파트너가 가장 많은 수익을 창출하는지에 대한 자세한 분석을 제공합니다.

<Info>
자세한 설정 지침은 판매 전환 이벤트를 추적하고 제휴 링크에 수익을 귀속시키는 방법을 다룬 [Dub 통합 가이드](/integrations/dub)를 참조하세요.
</Info>

### Dub Partners로 빠른 설정

1. **Dub 대시보드에서 제휴 링크에 대한 전환 추적을 활성화**하세요.
2. 체크아웃 세션을 생성할 때 **Dub 클릭 ID** (`dub_id` 쿠키)을 캡처하세요.
3. 고객 정보와 함께 결제 메타데이터에 **클릭 ID**를 저장하세요.
4. 결제가 성공할 때 판매 데이터를 Dub에 전송하기 위해 **웹훅 통합을 구성**하세요.

이 통합은 자동으로 다음을 추적합니다:
- 일회성 결제
- 구독 활성화
- 반복 구독 결제
