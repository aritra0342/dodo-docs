카드 결제는 온라인 결제의 기반으로, 전 세계적으로 수용되고 고객들에게 신뢰받고 있습니다. Dodo Payments는 내장된 사기 방지 기능 및 PCI 준수를 통해 모든 주요 카드 네트워크를 지원합니다.

## 지원되는 카드 네트워크

### 글로벌 네트워크

| 네트워크 | 범위 |
| :------ | :------- |
| **Visa** | 전 세계 리더, 40억 장 이상의 카드 |
| **Mastercard** | 글로벌 도달, 강력한 보안 기능 |
| **American Express** | 프리미엄 카드 사용자, 높은 소비 |
| **Discover** | 미국 중심, 글로벌 확대 중 |
| **JCB** | 일본에서 선도, 아시아 전역으로 확대 중 |
| **UnionPay** | 중국에서 우세, 80억 장 이상 카드 |
| **Diners Club** | 프리미엄 국제 여행자 |

### 지역 네트워크

| 네트워크 | 지역 |
| :------ | :----- |
| **Interac** | 캐나다의 직불 네트워크 |
| **Cartes Bancaires** | 프랑스의 국가 네트워크 |
| **Korean Local Cards** | 한국 내국 카드 네트워크 |
| **Rupay** | 인도의 국가 네트워크 |

## 구성

다음 값을 `allowed_payment_method_types`에서 사용하십시오:

| 유형 | 설명 |
| :--- | :---------- |
| `credit` | 모든 신용카드 |
| `debit` | 모든 직불카드 |

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  allowed_payment_method_types: ['credit', 'debit'],
  return_url: 'https://example.com/success'
});
```

<Tip>
특정한 사유가 없는 한 `credit`와 `debit`를 모두 포함하십시오. 많은 고객이 직불카드를 선호하며 수수료도 더 낮은 경우가 많습니다.
</Tip>

## 3D 보안 인증

3D Secure (3DS)는 카드 소지자의 신원을 확인하여 사기 및 청구 취소를 줄이는 인증 레이어를 추가합니다.

### 3DS가 트리거되는 경우

3DS는 다음과 같은 경우 자동으로 트리거됩니다:
- 카드 네트워크에서 요구됨
- 지역 규정(예: 유럽의 PSD2)에서 요구됨
- 거래가 고위험으로 표시됨

### 3DS 강제 적용

모든 거래에서 3DS를 요구할 수 있습니다:

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  force_3ds: true,
  return_url: 'https://example.com/success'
});
```

<Note>
모든 거래에 대해 3DS를 활성화하면 사기를 줄일 수 있지만, 인증 중 일부 고객이 이탈할 수 있어 전환율이 다소 감소할 수 있습니다.
</Note>

## 저장된 결제 방법

고객은 더 빠른 결제를 위해 카드를 저장할 수 있습니다.

<CardGroup cols={3}>
<Card title="Tokenized" icon="lock">
원본 카드 번호는 절대 저장되지 않습니다.
</Card>
<Card title="PCI Compliant" icon="shield-check">
Dodo가 모든 규정 준수를 처리합니다.
</Card>
<Card title="Customer-Scoped" icon="user">
카드는 특정 고객에 연결되어 있습니다.
</Card>
</CardGroup>

### 저장 카드 활성화

```javascript
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  show_saved_payment_methods: true,
  customer: { customer_id: 'cus_existing_123' },
  return_url: 'https://example.com/success'
});
```

### 원클릭 구매

```javascript
// Get customer's saved payment methods
const methods = await client.customers.getPaymentMethods('cus_123');

// Use saved card for instant checkout
const session = await client.checkoutSessions.create({
  product_cart: [{ product_id: 'prod_123', quantity: 1 }],
  customer: { customer_id: 'cus_123' },
  payment_method_id: methods[0].payment_method_id,
  confirm: true,
  return_url: 'https://example.com/success'
});
```

## 문제 해결

<AccordionGroup>
<Accordion title="Card declined">
**원인:** 자금 부족, 카드 만료, CVV 오류, 은행 사기 방지.

**해결책:** 고객에게 정보를 확인하거나 다른 카드를 사용해 보라고 요청하십시오.
</Accordion>

<Accordion title="3DS authentication failed">
**원인:** 고객이 이탈했거나, 은행 시스템 이용 불가, 시간 초과.

**해결책:** 다시 시도하거나 고객에게 은행에 문의하도록 요청하십시오.
</Accordion>

<Accordion title="Card not supported">
**원인:** 해당 지역 카드 미지원, 선불 카드 제한.

**해결책:** 주요 네트워크의 다른 카드를 사용해 보도록 고객에게 안내하십시오.
</Accordion>
</AccordionGroup>
