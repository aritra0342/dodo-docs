## **새로운 기능 요청하기**

Dodo Payments와의 경험을 향상시킬 수 있는 기능에 대한 아이디어가 있다면, Featurebase에서 직접 기능 요청을 제출할 수 있습니다.

<img src="/images/feature-request/Feature.png" alt="기능 요청"/>

- **기능 요청 방법**:
    1. 우리의 [Featurebase 포털](https://dodopayments.featurebase.app/)에 방문하세요.
    2. **"아이디어 게시하기"** 또는 **"기능 요청 제출하기"**를 클릭하세요.
    3. 다음 세부정보를 제공하세요:
        - **기능 이름**: 기능 아이디어에 대한 간결한 제목.
        - **기능 설명**: 기능에 대한 간단한 설명, 왜 중요한지 및 비즈니스에 어떻게 도움이 되는지 포함.
    4. 요청을 제출하세요.
- **효과적인 요청을 위한 팁**:
    - 명확하고 간결하게 작성하세요.
    - 기능의 비즈니스 영향을 강조하세요.
    - 가능하다면 예시를 제공하세요.
    - 가치를 보여주는 사용 사례를 포함하세요.

## **기존 기능 요청에 투표하기**

우리는 가장 많은 상인에게 혜택을 주는 기능을 우선시합니다. Featurebase에서 기존 기능 요청을 탐색하고, 가치 있다고 생각하는 요청에 투표할 수 있습니다.

- **투표 방법**:
    1. 우리의 [Featurebase 포털](https://dodopayments.featurebase.app/)에 방문하세요.
    2. 제출된 요청 목록을 탐색하세요.
    3. 지원하는 기능 옆에 있는 **업보트** 버튼 (👍)을 클릭하세요.
- **투표의 이점**:
    - 관심 있는 기능의 개발 우선순위에 영향을 미칠 수 있습니다.
    - 우리 팀이 광범위한 수요가 있는 기능을 식별하는 데 도움을 줍니다.
    - 투표한 기능이 출시될 때 알림을 받을 수 있습니다.

## **제품 로드맵 보기**

Featurebase에서 우리의 공개 로드맵을 통해 기능 개발 상태에 대한 정보를 유지하세요.

- **로드맵 기능**:
    - **계획됨**: 승인되어 개발이 예정된 기능.
    - **진행 중**: 우리 팀이 현재 작업 중인 기능.
    - **완료됨**: 성공적으로 구현되어 라이브 상태인 기능.
- **접근 방법**:
    - 우리의 [Featurebase 포털](https://dodopayments.featurebase.app/)에 방문하세요.
    - 상태별로 기능을 보기 위해 **로드맵** 섹션으로 이동하세요.
    - 기능 카테고리와 현재 상태를 탐색하세요.
    - 상태, 카테고리별로 필터링하거나 특정 기능을 검색하세요.
