## 소개

API 키 생성을 통해 상인들은 Dodo Payments 플랫폼과 안전하게 인증하고 시스템을 통합하여 결제, 제품 및 구독 기능에 원활하게 접근할 수 있습니다.

**참고 :** 테스트 모드에서 생성된 API 키는 테스트 모드 환경에 제한되며, 라이브 모드의 경우도 마찬가지입니다.
