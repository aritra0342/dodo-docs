## 할인 코드란 무엇인가요?

할인 코드는 체크아웃 시 주문 총액을 줄여주는 프로모션 토큰입니다. 다음과 같은 경우에 이상적입니다:

- **계절 캠페인**: 블랙 프라이데이, 제품 출시 또는 기념일
- **획득 제안**: 첫 구매 인센티브 또는 추천 보상
- **유지**: 기존 고객을 위한 재유치 또는 충성도 보상
- **B2B 거래**: 비공식 코드로 계약된 또는 협상된 가격

## 주요 이점

- **유연한 할인**: 비율 또는 고정 금액 할인
- **타겟 제어**: 제품 및 구독 주기로 제한
- **캠페인 관리**: 만료일 및 사용 한도
- **원활한 체크아웃**: 체크아웃 세션을 통한 UI 필드 및 API 지원

## 할인 코드 생성하기

Dodo Payments 대시보드에서 할인 코드를 생성한 후, 호스팅된 체크아웃 또는 API를 통해 적용하세요.

### 대시보드 설정

- **할인 이름** (필수): 내부 및 대시보드 표시 이름
- **코드** (필수): 고객이 체크아웃 시 입력하는 문자열
- **유형 및 금액** (필수): 비율 또는 고정 금액 값을 설정하거나 제공된 버튼을 사용하여 무작위 코드를 생성하세요
- **만료일** (선택 사항): 코드가 무효가 되는 날짜
- **사용 한도** (선택 사항): 모든 고객에 대한 최대 총 사용 횟수
- **제품 제한** (선택 사항): 선택한 제품에만 적용 가능하도록 제한
- **구독 주기 제한** (선택 사항): 할인이 적용되는 청구 주기 수

<Tip>
구독에 대한 소개 가격을 위해 주기 제한을 사용하세요 (예: "3개월 동안 50% 할인").
</Tip>

## 체크아웃 경험

1. 쇼핑객이 체크아웃 필드에 코드를 입력합니다.
2. 적격 할인 적용 및 총액이 즉시 업데이트됩니다.

<Info>
체크아웃 세션에서 `discount_code`를 전달하여 코드를 미리 적용하고, `feature_flags.allow_discount_code`를 설정하여 입력 필드를 표시하세요.
</Info>

## 일반적인 사용 사례

- **소개 제안**: 신제품에 대한 한정 시간 출시 프로모션
- **대량 또는 B2B**: 선택된 제품 세트에 대한 계약 할인
- **유지 전략**: 이탈 방지 워크플로우에서 재유치 코드
- **계절 캠페인**: 휴일 또는 이벤트 기반 프로모션

## 통합 예시

### 체크아웃 세션에서 할인 적용
할인을 미리 적용하고 코드 입력 UI를 표시하세요.

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  discount_code: 'BLACKFRIDAY2024',
  customer: { email: 'user@example.com', name: 'Jane Doe' },
  return_url: 'https://yoursite.com/return'
});
```

### 미리 적용하지 않고 할인 입력 활성화
고객이 사전 입력 없이 체크아웃 시 코드를 입력할 수 있도록 하세요.

```typescript
const session = await client.checkoutSessions.create({
  product_cart: [
    { product_id: 'prod_abc', quantity: 1 }
  ],
  feature_flags: {
    allow_discount_code: true
  },
  return_url: 'https://yoursite.com/return'
});
```

## 모범 사례

- **명확하게 이름 짓기**: 캠페인 이름과 일치하는 인식 가능한 코드를 사용하세요
- **시간 제한**: 긴급성을 유도하고 남용을 방지하기 위해 만료일을 추가하세요
- **신중하게 범위 설정**: 마진 유출을 피하기 위해 특정 제품으로 제한하세요
- **조기 검증**: 체크아웃 확인 전에 코드 적용 가능성을 확인하세요
- **영향 모니터링**: 캠페인별 사용 및 전환을 추적하세요

<Info>
할인 코드는 획득 및 유지를 위한 강력한 수단입니다. 간단하고 잘 명명된 제안으로 시작하고, 철저히 검증하며, 성과에 따라 반복하세요.
</Info>
