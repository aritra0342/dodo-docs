## 소개

[ParityDeals](https://www.paritydeals.com/)는 고객의 지리적 위치에 따라 자동으로 가격을 최적화하는 플랫폼으로, 일반적으로 "구매력 평형" 가격 책정으로 알려져 있습니다.

ParityDeals를 사용하면 연중 내내 [자동 시간 기반 할인](https://www.paritydeals.com/features/time-based-discounts/) 및 [휴일 기반 프로모션](https://www.paritydeals.com/features/holiday-based-discounts/)을 제공할 수 있습니다.

<Info>
Dodo Payments 상점에 ParityDeals 배너를 직접 삽입할 수는 없지만(사용자 정의 스크립트는 지원되지 않음), 여전히 자신의 웹사이트나 애플리케이션에서 ParityDeals를 효과적으로 구현할 수 있습니다.
</Info>

## 작동 방식

ParityDeals가 귀하의 웹사이트에 통합되면, 적격 방문자는 체크아웃 시 사용할 수 있는 위치 기반 할인 코드가 포함된 배너를 보게 됩니다.

<Frame>
  <img src="/images/ppp/1.png" alt="ParityDeals 할인 배너 예시" />
</Frame>

이 할인 코드는 ParityDeals에 의해 자동으로 생성 및 관리되며, 귀하의 **Dodo Payments 대시보드**의 **판매 → 할인** 아래에 표시됩니다.

## 단계별 설정 가이드

### 1. ParityDeals에 가입하기

[app.paritydeals.com](https://app.paritydeals.com/)를 방문하여 계정을 생성하세요.

### 2. 거래 생성하기

로그인 후 거래를 생성하는 옵션을 찾습니다. 드롭다운 메뉴에서 선택하세요:

**→ "Dodo Payments를 통한 거래 생성"**

<Frame>
  <img src="/images/ppp/2.png" alt="Dodo Payments를 통한 거래 생성" />
</Frame>

### 3. Dodo Payments API 키 생성하기

귀하의 **Dodo Payments 대시보드**에 로그인하고 다음으로 이동합니다:

**개발자 → API 키**

**"새 API 생성"**을 클릭하고 "ParityDeals"와 같은 설명적인 이름을 지정합니다.

<Frame>
  <img src="/images/ppp/3.png" alt="Dodo Payments에서 새 API 키 생성하기" />
</Frame>

### 4. ParityDeals에 API 키 연결하기

생성한 API 키를 복사하여 ParityDeals 인터페이스에 붙여넣어 Dodo Payments 계정과의 연결을 설정합니다.

<Frame>
  <img src="/images/ppp/4.png" alt="API 키를 ParityDeals에 연결하기" />
</Frame>

### 5. 제품 선택하기

연결되면 ParityDeals가 귀하의 Dodo Payments 제품 카탈로그를 가져옵니다. 거래와 연결할 제품을 선택하세요.

<Frame>
  <img src="/images/ppp/5.png" alt="ParityDeals를 위한 제품 선택하기" />
</Frame>

### 6. 거래 설정 구성하기

귀하의 웹사이트 URL을 입력하고 할인 배너가 표시될 특정 대상 페이지를 선택할 수 있습니다.

그런 다음 선호하는 거래 유형을 선택하세요:

- 위치 기반 거래
- 휴일 거래
- 시간 기반 플래시 거래

<Frame>
  <img src="/images/ppp/6.png" alt="거래 설정 구성하기" />
</Frame>

### 7. 국가 기반 규칙 설정하기

지역화된 할인을 제공할 국가를 선택하고 각 국가에 대한 할인 비율을 정의합니다. ParityDeals는 필요에 따라 조정할 수 있는 추천 기본값을 제공합니다.

<Frame>
  <img src="/images/ppp/7.png" alt="국가 기반 할인 규칙 설정하기" />
</Frame>

### 8. 배너 사용자 정의하기

할인 배너를 디자인합니다:

- 텍스트 사용자 정의
- 색상 및 레이아웃 선택
- 배너 위치 선택(페이지 상단 또는 하단)

이 배너는 구성한 규칙에 따라 적격 방문자에게 표시됩니다.

<Frame>
  <img src="/images/ppp/8.png" alt="할인 배너 사용자 정의하기" />
</Frame>

### 9. ParityDeals 스크립트를 웹사이트에 추가하기

거래를 저장한 후, ParityDeals는 스크립트를 생성합니다. 이 스크립트를 웹사이트 코드에 추가하세요 (일반적으로 `<head>` 섹션에).

이 스크립트는 방문자의 위치를 자동으로 감지하고 적절한 할인 배너를 표시합니다.

<Frame>
  <img src="/images/ppp/9.png" alt="ParityDeals 통합 스크립트" />
</Frame>

## 모두 설정 완료!

이제 다음을 제공할 준비가 되었습니다:

- **위치 기반 거래** - 구매력에 따라 가격 조정
- **카운트다운/시간 제한 할인** - 한정된 시간 제공으로 긴급성 창출
- **휴일 프로모션** - 계절 쇼핑 트렌드 활용

이 모든 것은 ParityDeals를 통해 자동화되며 Dodo Payments와 원활하게 작동합니다.

자세한 정보는 [ParityDeals 문서](https://www.paritydeals.com/docs/)를 참조하거나 지원 팀에 문의하세요.
