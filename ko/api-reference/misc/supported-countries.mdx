---
<Tip> 다음은 우리가 현재 결제를 지원하는 국가를 나타내는 [ISO Alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) 국가 코드 목록입니다. 전체 지원 국가 목록은 [여기](/miscellaneous/list-of-countries-we-accept-payments-from)에서 확인할 수 있습니다. 이 목록은 커버리지를 확장함에 따라 정기적으로 업데이트됩니다. </Tip>

<Tip> 참고: 이용 가능한 옵션에는 현재 지원하는 국가뿐만 아니라 모든 국가의 ISO Alpha-2 코드가 포함됩니다. 이용 가능 여부는 지원 국가 목록을 참조해 주세요. </Tip>
