## 개요

Dodo Payments API는 API 요청의 성공 또는 실패를 나타내기 위해 표준 HTTP 상태 코드와 사용자 정의 오류 코드를 사용합니다. 오류가 발생하면 API는 적절한 HTTP 상태 코드와 오류에 대한 자세한 정보를 포함하는 JSON 응답을 반환합니다.

각 오류 응답에는 다음이 포함됩니다:
- 오류의 일반적인 범주를 나타내는 HTTP 상태 코드
- 오류의 정확한 성격을 식별하는 특정 오류 코드
- 무엇이 잘못되었는지 설명하는 사람이 읽을 수 있는 오류 메시지
- 해당하는 경우 오류에 대한 추가 세부정보

이러한 오류 코드와 그 의미를 이해하는 것은 다음과 같은 데 중요합니다:
- 통합 문제 디버깅
- 애플리케이션에서 적절한 오류 처리 구현
- 최종 사용자에게 의미 있는 피드백 제공
- 강력한 결제 처리 시스템 유지

다음 섹션에서는 Dodo Payments API를 사용할 때 발생할 수 있는 표준 HTTP 상태 코드 및 특정 오류 코드에 대한 자세한 정보를 제공합니다.

## 표준 API 오류 코드

| 오류 코드 | HTTP 상태 | 설명 |
|------------|-------------|-------------|
| **400** | 잘못된 요청 | 요청이 잘못되었거나 유효하지 않은 매개변수를 포함하고 있습니다. |
| **401** | 인증되지 않음 | 인증에 실패했거나 API 키가 유효하지 않습니다. |
| **403** | 금지됨 | API 키가 요청된 리소스에 접근할 권한이 없습니다. |
| **404** | 찾을 수 없음 | 요청된 리소스가 존재하지 않습니다. |
| **405** | 허용되지 않는 메서드 | 이 엔드포인트에 대해 HTTP 메서드가 지원되지 않습니다. |
| **409** | 충돌 | 요청이 리소스의 현재 상태와 충돌합니다. |
| **422** | 처리할 수 없는 엔터티 | 요청이 잘 형성되었지만 의미론적 오류가 포함되어 있습니다. |
| **429** | 너무 많은 요청 | 비율 제한 초과 |
| **500** | 내부 서버 오류 | 서버에서 예기치 않은 오류가 발생했습니다. |
| **502** | 잘못된 게이트웨이 | 서버가 업스트림 서버로부터 잘못된 응답을 받았습니다. |
| **503** | 서비스 사용 불가 | 서비스가 일시적으로 사용할 수 없습니다. |
| **504** | 게이트웨이 시간 초과 | 서버가 업스트림 응답을 기다리는 동안 시간 초과가 발생했습니다. |

## 오류 응답 형식

오류가 발생하면 API는 다음 구조의 JSON 응답을 반환합니다:

```json
{
  "code": "UNSUPPORTED_COUNTRY",
  "message": "Country AI currently not supported"
}
```

## 오류 코드 참조

- `ACTIVATION_LIMIT_LESS_THAN_CURRENT_AMOUNT`
  - **Trigger:** 라이센스 키 활성화: 새로운 한도가 기존 인스턴스 수보다 작음
  - **Message:** 새로운 활성화 한도는 현재 인스턴스 수보다 작을 수 없습니다.

- `ADDONS_IN_USAGE_BASED_BILLING_NOT_SUPPORTED`
  - **Trigger:** 사용 기반 청구 구독에 애드온 추가 시도
  - **Message:** 구독의 애드온은 사용 기반 청구를 지원하지 않습니다.

- `ADDONS_NOT_ALLOWED_FOR_ON_DEMAND`
  - **Trigger:** 주문형 구독에 애드온 추가 시도
  - **Message:** 주문형 구독에는 애드온이 허용되지 않습니다.

- `BRAND_MISMATCH`
  - **Trigger:** 장바구니 항목이 서로 다른 브랜드에 속함
  - **Message:** 제품 장바구니의 모든 항목은 동일한 브랜드에 속해야 합니다.

- `BRAND_NOT_ENABLED`
  - **Trigger:** 브랜드가 비활성화되었거나 활성 상태가 아님
  - **Message:** 제공된 브랜드가 활성화되지 않았습니다.

- `BRAND_SUBMISSION_NOT_ENABLED`
  - **Trigger:** 브랜드 검증 재제출 기능이 활성화되지 않음
  - **Message:** 브랜드 검증 재제출이 활성화되지 않았습니다.

- `CHARGE_NOT_ALLOWED_FOR_SCHEDULED_CANCELLATION`
  - **Trigger:** 취소 예정인 구독에 대한 요금 청구 시도
  - **Message:** 취소 예정인 구독입니다.

- `CHECKOUT_SESSION_CONSUMED`
  - **Trigger:** 체크아웃 세션이 이미 결제를 생성함
  - **Message:** 체크아웃 세션이 이미 사용되었습니다.

- `DISCOUNT_CODE_ALREADY_EXISTS`
  - **Trigger:** 중복 할인 코드 생성
  - **Message:** 할인 코드가 이미 존재합니다.

- `DISCOUNT_CODE_EXPIRED`
  - **Trigger:** 할인 코드가 `expires_at` 날짜를 초과함
  - **Message:** 할인 코드가 만료되었습니다.

- `DISCOUNT_CODE_USAGE_LIMIT_EXCEEDED`
  - **Trigger:** `usage_limit`에 도달한 후 할인 재사용
  - **Message:** 사용 한도는 times_used보다 작을 수 없습니다 / 할인 코드 사용 한도에 도달했습니다.

- `DISCOUNT_NOT_AVAILABLE_FOR_ON_DEMAND`
  - **Trigger:** 주문형 구독에 코드 적용됨
  - **Message:** 할인 쿠폰은 주문형 구독에 사용할 수 없습니다.

- `DISCOUNT_NOT_AVAILABLE_FOR_PRODUCT`
  - **Trigger:** 관련 없는 제품에 코드 적용됨
  - **Message:** 이 제품에 대한 할인 쿠폰은 사용할 수 없습니다.

- `DUPLICATE_LINE_ITEMS_IN_REQUEST`
  - **Trigger:** 동일한 `item_id`가 `items[]`에 두 번 나타남
  - **Message:** 항목 배열에 중복 item_ids가 지정되었습니다.

- `DUPLICATE_METER_IDS_IN_REQUEST`
  - **Trigger:** 요청에 동일한 미터 ID가 여러 번 나타남
  - **Message:** 중복 미터 ID는 허용되지 않습니다.

- `EXCHANGE_RATE_NOT_FOUND`
  - **Trigger:** `from → to` 통화 쌍에 대한 FX 환율 없음
  - **Message:** 통화에서 통화로 변환할 수 있는 환율을 찾을 수 없습니다.

- `EXISTING_REFUND_REQUEST_PROCESSING`
  - **Trigger:** 이전 환불 요청이 아직 처리 중임
  - **Message:** 상태가 "대기 중"인 환불 요청이 아직 처리 중입니다.

- `INACTIVE_LICENSE_KEY`
  - **Trigger:** 키 상태 ≠ `ACTIVE`
  - **Message:** 라이센스 키가 활성화되지 않았습니다.

- `INACTIVE_SUBSCRIPTION_PLAN_CHANGE_NOT_SUPPORTED`
  - **Trigger:** 비활성 구독에서 계획 변경
  - **Message:** 비활성 구독에 대한 계획 변경은 지원되지 않습니다.

- `INSUFFICIENT_WALLET_FUNDS`
  - **Trigger:** 지갑 잔액 < 차감 금액
  - **Message:** 지갑에 자금이 부족합니다.

- `INTEGER_CONVERSION_FAILURE`
  - **Trigger:** 서버 측에서 실패한 모든 정수 ↔ 문자열/소수 변환
  - **Message:** 정수 변환 실패

- `INTERNAL_SERVER_ERROR`
  - **Trigger:** 포착되지 않은 예외; 서버 측에서 세부 정보를 기록해야 합니다.
  - **Message:** *공식 메시지 없음 (일반 500)*

- `INVALID_DISCOUNT_CODE`
  - **Trigger:** 코드가 존재하지 않음 / 적용할 수 없음
  - **Message:** 유효하지 않은 할인 코드 / 할인 코드를 장바구니의 어떤 제품에도 적용할 수 없습니다.

- `INVALID_PERCENTAGE`
  - **Trigger:** 비율 금액 > 100% (또는 10,000 기준 포인트)
  - **Message:** 비율 금액은 10000을 초과할 수 없습니다 / 할인 코드 금액은 100%를 초과할 수 없습니다.

- `INVALID_QUANTITY`
  - **Trigger:** 사용 기반 가격 책정에 대해 지정된 수량이 유효하지 않음
  - **Message:** 사용 기반 가격 제품에서는 1개 수량만 허용됩니다.

- `INVALID_QUERY_PARAMS`
  - **Trigger:** 상호 배타적 / 잘못된 쿼리 매개변수
  - **Message:** 쿼리 매개변수는 time_frame 또는 (start, end) 중 하나만 포함해야 합니다.

- `INVALID_REQUEST_BODY`
  - **Trigger:** 잘못된 JSON 또는 스키마 위반
  - **Message:** 요청 본문이 유효하지 않습니다. 요청 헤더와 객체를 확인하십시오.

- `INVALID_REQUEST_PARAMETERS`
  - **Trigger:** 의미론적으로 잘못됨 (예: 과거 날짜)
  - **Message:** next_billing_date를 과거 시간으로 변경할 수 없습니다.

- `INVALID_SUGGESTED_PRICE`
  - **Trigger:** PWYW 가격 < 최소 허용 가격
  - **Message:** 제안된 가격은 최소 가격보다 낮을 수 없습니다. 원하는 만큼 지불하는 경우 가격은 최소 수락 금액으로 간주됩니다.

- `INVALID_TAX_ID`
  - **Trigger:** VAT/GST/TIN 유효성 검사 실패
  - **Message:** 세금 ID가 유효하지 않습니다.

- `LICENSE_KEY_LIMIT_REACHED`
  - **Trigger:** 활성화 수 = 한도
  - **Message:** 라이센스 키 활성화 한도에 도달했습니다.

- `LICENSE_KEY_NOT_FOUND`
  - **Trigger:** 인스턴스 ID 또는 키 ID가 유효하지 않음
  - **Message:** 라이센스 키 인스턴스를 찾을 수 없거나 이 라이센스 키에 속하지 않습니다.

- `LINE_ITEM_FULLY_REFUNDED`
  - **Trigger:** 이미 완전히 환불된 항목에 대한 환불 시도
  - **Message:** 항목 {id}는 이미 완전히 환불되었으며 추가로 환불할 수 없습니다.

- `LINE_ITEM_NOT_FOUND`
  - **Trigger:** 항목 ID가 참조된 결제의 일부가 아님
  - **Message:** 항목 {id}가 결제에서 발견되지 않았습니다.

- `LINE_ITEM_PRORATED`
  - **Trigger:** 비례 항목에 대한 환불 또는 업데이트 시도
  - **Message:** 항목 {id}는 비례적이므로 환불할 수 없습니다.

- `LINE_ITEM_REFUND_AMOUNT_TOO_HIGH`
  - **Trigger:** 환불 금액 > 지불 금액 (세금 포함)
  - **Message:** 항목 {id}의 요청된 환불 금액(세금 포함)은 {amount}로, 지불 금액 {amount}를 초과합니다.

- `LINE_ITEM_REFUND_AMOUNT_TOO_LOW`
  - **Trigger:** 환불 금액이 최소 기준 이하
  - **Message:** 항목 {id}의 요청된 환불 금액은 {amount}로 너무 낮습니다.

- `MAXIMUM_KEYS_REACHED`
  - **Trigger:** 메타데이터 / 사용자 정의 필드가 50 쌍을 초과함
  - **Message:** 50개의 키-값 쌍을 초과했습니다.

- `MERCHANT_NOT_LIVE`
  - **Trigger:** 비즈니스가 여전히 테스트/샌드박스 모드에 있음
  - **Message:** 상인이 라이브 상태가 아닙니다.

- `METER_IS_DELETED`
  - **Trigger:** 삭제된 미터 사용 시도
  - **Message:** 미터가 이미 삭제되었습니다.

- `MISSING_ADDON_IDS`
  - **Trigger:** `addon_id` 목록이 비어 있거나 알 수 없는 ID
  - **Message:** 하나 이상의 제품 ID가 존재하지 않습니다: {id}

- `MISSING_METER_IDS`
  - **Trigger:** 미터 ID 목록이 비어 있거나 유효하지 않은 ID 포함
  - **Message:** 하나 이상의 미터 ID가 존재하지 않습니다: {id}

- `MISSING_PRODUCT_INFORMATION`
  - **Trigger:** 제품이 존재하지만 필수 정보가 누락됨
  - **Message:** 제품 {id}가 존재하지만 다른 필수 정보가 누락되었거나 유효하지 않습니다.

- `NEGATIVE_BALANCE_ADJUSTMENT`
  - **Trigger:** 지갑 잔액을 음수로 만들려고 시도함
  - **Message:** 지갑 잔액을 음수로 만드는 것은 허용되지 않습니다.

- `NO_ELIGIBLE_PAYMENT_METHODS`
  - **Trigger:** 필터링 후 남은 것이 없음
  - **Message:** 적격 결제 방법을 찾을 수 없습니다.

- `NO_EXPIRY_ON_SUBSCRIPTION_LICENSE_KEYS`
  - **Trigger:** 구독 기반 키에 만료 설정 시도
  - **Message:** 구독 기반 라이센스 키에 대한 만료 날짜를 설정할 수 없습니다.

- `NOT_FOUND`
  - **Trigger:** 누락된 리소스에 대한 일반 404
  - **Message:** 항목을 찾을 수 없습니다 *(또는 더 구체적)*

- `ON_DEMAND_PLAN_CHANGE_NOT_SUPPORTED`
  - **Trigger:** 주문형에 대한 계획 변경이 허용되지 않음
  - **Message:** 주문형 구독에 대한 계획 변경은 지원되지 않습니다.

- `ON_DEMAND_SUBSCRIPTIONS_NOT_ENABLED`
  - **Trigger:** 비즈니스에 기능 플래그가 꺼져 있음
  - **Message:** 이 비즈니스에 대해 주문형 구독이 활성화되지 않았습니다.

- `ON_DEMAND_USAGE_BASED_BILLING_NOT_SUPPORTED`
  - **Trigger:** 사용 기반 청구와 함께 주문형 사용 시도
  - **Message:** 사용 기반 청구에 대해 주문형 구독이 지원되지 않습니다.

- `PAY_AS_YOU_WANT_AMOUNT_REQUIRED`
  - **Trigger:** PWYW 제품에 대한 가격 누락
  - **Message:** 원하는 만큼 지불하는 제품에 대한 금액은 필수입니다.

- `PAYMENT_ALREADY_REFUNDED`
  - **Trigger:** 중복 환불
  - **Message:** 이 결제는 이미 환불되었습니다.

- `PAYMENT_HAS_BEEN_REFUNDED`
  - **Trigger:** 결제가 완전히 환불됨
  - **Message:** 결제 ID가 완전히 환불되었습니다.

- `PAYMENT_NOT_SUCCEEDED`
  - **Trigger:** 실패한 결제에 대한 환불/처리 시도
  - **Message:** 제공된 결제가 성공하지 않았습니다.

- `PLAN_CHANGE_NOT_ALLOWED_FOR_SCHEDULED_CANCELLATION`
  - **Trigger:** 취소 예정인 구독에서 계획 변경 시도
  - **Message:** 취소 예정인 구독입니다.

- `PREVIOUS_PAYMENT_PENDING`
  - **Trigger:** 이전 결제가 비정상 상태일 때 요금 생성 시도
  - **Message:** 이전 결제가 아직 성공하지 않았으므로 새로운 요금을 생성할 수 없습니다.

- `PRODUCT_CART_EMPTY`
  - **Trigger:** 빈 제품 장바구니 제출됨
  - **Message:** product_cart가 비어 있습니다.

- `PRODUCT_IS_DELETED`
  - **Trigger:** 제품이 소프트 삭제됨
  - **Message:** *메시지 없음*

- `REFUND_AMOUNT_EXCEEDS_PAID_AMOUNT`
  - **Trigger:** 집계 환불 금액 > 지불 금액
  - **Message:** 계산된 환불 금액이 지불 금액보다 큽니다.

- `REFUND_WINDOW_EXPIRED`
  - **Trigger:** 허용된 환불 기간 외부
  - **Message:** 결제 생성 후 {days}일 이내에 환불을 시작할 수 없습니다. support@dodopayments.com에 문의하십시오.

- `REQUEST_AMOUNT_BELOW_MINIMUM`
  - **Trigger:** 금액 < 제품 최소
  - **Message:** 금액은 제품에 대해 지정된 최소 금액보다 작을 수 없습니다.

- `SUBSCRIPTION_EXPIRED`
  - **Trigger:** 청구가 `ends_at`를 초과함
  - **Message:** 구독이 만료되어 새로운 요금을 생성할 수 없습니다.

- `SUBSCRIPTION_INACTIVE`
  - **Trigger:** 상태 ≠ `ACTIVE`
  - **Message:** 구독이 활성 상태가 아닙니다.

- `SUBSCRIPTION_NOT_ON_DEMAND`
  - **Trigger:** 예상된 주문형이지만 고정 간격을 받음
  - **Message:** 구독이 이미 주문형이 아닙니다.

- `SUBSCRIPTION_PAYMENT_RETRY_LIMIT_EXCEEDED`
  - **Trigger:** 구독 결제 재시도가 최대 시도를 초과함
  - **Message:** 이 구독에 대해 최대 10회의 재시도 한도를 초과했습니다.

- `TOO_MANY_REQUESTS`
  - **Trigger:** 429 속도 제한
  - **Message:** *메시지 없음*

- `TOTAL_PAYMENT_AMOUNT_BELOW_MINIMUM_AMOUNT`
  - **Trigger:** 결합된 장바구니 총액 < 게이트웨이 최소
  - **Message:** 결제를 처리하기 위해서는 {display_str}의 최소 금액이 필요합니다.

- `UNABLE_TO_EDIT_PRIMARY_BRAND`
  - **Trigger:** 일반 API를 통해 기본 브랜드 업데이트 시도
  - **Message:** 기본 브랜드는 이 API 엔드포인트를 통해 업데이트할 수 없습니다.

- `UNAUTHORIZED`
  - **Trigger:** API 키 없음 또는 유효하지 않은 토큰 / 범위
  - **Message:** 이 작업을 수행할 권한이 없습니다.

- `UNSUPPORTED_ACTION`
  - **Trigger:** 리소스 유형에 대한 지원되지 않는 작업
  - **Message:** 사용 기반 구독에 대한 계획 변경은 지원되지 않습니다.

- `UNSUPPORTED_BILLING_CURRENCY`
  - **Trigger:** 구독이 USD로 제한됨
  - **Message:** 구독에 대한 비 USD 청구 통화는 지원되지 않습니다.

- `UNSUPPORTED_COUNTRY`
  - **Trigger:** 지원되지 않는 지역
  - **Message:** 국가 {country_name}는 현재 지원되지 않습니다.

- `UNSUPPORTED_CURRENCY`
  - **Trigger:** 제품 또는 애드온 통화가 유효하지 않음
  - **Message:** 통화가 현재 지원되지 않음 / 현재 USD 및 INR 제품만 지원됨 / 애드온 가격에 대해서는 현재 USD 및 INR만 지원됨 / 청구 통화에 대해서는 USD 또는 INR만 요청할 수 있음 / 통화가 지원되지 않음 / 인도 카드 구독에 대한 예상치 못한 통화

- `UNSUPPORTED_DISCOUNT_TYPE`
  - **Trigger:** 고정 금액 할인 등 아직 활성화되지 않음
  - **Message:** 현재는 비율 할인 코드만 지원됩니다.

- `UNSUPPORTED_PAYMENT_CURRENCY`
  - **Trigger:** 커넥터에 대해 차단된 결제 통화
  - **Message:** 이 거래에 대해 INR 거래는 지원되지 않습니다.

- `UNSUPPORTED_TAX_CATEGORY`
  - **Trigger:** 세금 카테고리 문자열이 열거형에 없음
  - **Message:** 카테고리 {category}는 현재 지원되지 않습니다.

- `UNSUCCESSFUL_PAYMENT_ID`
  - **Trigger:** 결제 ID가 실패한 결제를 참조함
  - **Message:** 결제 ID가 실패한 상태입니다.

- `ZERO_AMOUNT_PAYMENT_REFUND_NOT_ALLOWED`
  - **Trigger:** 0 금액 결제 환불 시도
  - **Message:** 0 통화 금액의 결제를 환불할 수 없습니다.
